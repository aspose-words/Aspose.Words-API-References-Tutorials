В этом примере мы расскажем вам, как выделить текст жирным шрифтом с помощью Aspose.Words для .NET. Жирный текст делает его более заметным и выделяет его.

## Шаг 1: Использование генератора документов

Во-первых, мы будем использовать генератор документов, чтобы добавить содержимое в наш документ.

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## Шаг 2: Жирный текст

 Мы можем выделить текст жирным шрифтом, установив конструктор документа`Font.Bold` собственность на`true`.

```csharp
builder.Font.Bold = true;
```

## Шаг 3. Добавьте содержимое в документ.

 Теперь мы можем добавлять содержимое в документ, используя методы конструктора документов, такие как`Writeln`, который добавляет строку текста.

```csharp
builder.Writeln("This text will be bold");
```

## Пример исходного кода для полужирного текста с использованием Aspose.Words для .NET


```csharp
// Используйте конструктор документов, чтобы добавить содержимое в документ.
DocumentBuilder builder = new DocumentBuilder();

// Сделайте текст жирным.
builder.Font.Bold = true;
builder.Writeln("This text will be Bold");  
```

Поздравляем! Теперь вы научились выделять текст жирным шрифтом с помощью Aspose.Words для .NET.


### Часто задаваемые вопросы

#### В: Как сделать текст жирным в Aspose.Words?

 О: Чтобы сделать текст полужирным в Aspose.Words, вы можете использовать`Font.Bold`собственность`Run`объект. Вы можете установить это свойство на`true` выделить жирным шрифтом конкретный текст. Например, вы можете использовать`run.Font.Bold=true` выделить текст внутри`Run` объект.

#### В: Можно ли выделить жирным шрифтом несколько фрагментов текста в одном абзаце?

 О: Да, вы можете выделить несколько фрагментов текста в одном абзаце, используя несколько`Run` объекты. Вы можете создать несколько`Run` объекты и установить`Font.Bold` собственность на`true` для каждого объекта выделить нужные части текста жирным шрифтом. Затем вы можете добавить их в абзац с помощью`Paragraph.AppendChild(run)` метод.

#### В: Можно ли выделить жирным шрифтом текст в таблице или ячейке в Aspose.Words?

 О: Да, вы можете выделить жирным шрифтом текст, который находится в таблице или ячейке в Aspose.Words. Вы можете перейти к нужной ячейке или абзацу с помощью соответствующих методов, а затем применить жирное форматирование с помощью кнопки`Font.Bold`собственность`Run` или`Paragraph` объект.
