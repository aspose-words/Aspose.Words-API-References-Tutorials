В этом примере мы собираемся показать вам, как использовать функцию горизонтального правила с Aspose.Words для .NET. Горизонтальная линейка используется для визуального разделения разделов документа.

## Шаг 1: Использование генератора документов

Во-первых, мы будем использовать генератор документов, чтобы добавить содержимое в наш документ.

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## Шаг 2. Вставка горизонтальной линейки

 Мы можем вставить горизонтальную линейку, используя`InsertHorizontalRule` Метод генератора документов.

```csharp
builder. InsertHorizontalRule();
```

## Пример исходного кода для горизонтального правила с Aspose.Words для .NET

```csharp
// Используйте конструктор документов, чтобы добавить содержимое в документ.
DocumentBuilder builder = new DocumentBuilder();

// Вставьте горизонтальную линейку.
builder.InsertHorizontalRule();
```

Поздравляем! Теперь вы узнали, как использовать функцию горизонтального правила с Aspose.Words для .NET.


### Часто задаваемые вопросы

#### В: Как создать горизонтальную линейку в Markdown?

О: Чтобы создать горизонтальную линейку в Markdown, вы можете использовать один из следующих символов на пустой строке: три звездочки (\***), три тире (\---), или три знака подчеркивания (\___).

#### В: Могу ли я настроить внешний вид горизонтальной линейки в Markdown?

О: В стандартном Markdown нет возможности настроить внешний вид горизонтальных линеек. Однако некоторые расширенные редакторы и расширения Markdown предлагают дополнительные функции настройки.

#### В: Поддерживаются ли горизонтальные линейки всеми редакторами Markdown?

О: Да, большинство популярных редакторов Markdown поддерживают горизонтальные линейки. Однако всегда лучше проверить документацию конкретного поставщика, чтобы убедиться, что он поддерживается.

#### В: Какие еще элементы я могу создать в Markdown?

О: Помимо горизонтальных линеек, в Markdown можно создавать заголовки, абзацы, списки, ссылки, изображения, таблицы и многое другое.
