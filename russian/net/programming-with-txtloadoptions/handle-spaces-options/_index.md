В этом руководстве мы собираемся изучить исходный код C#, предоставленный для функциональности «Управление пространствами с параметрами загрузки TXT» с помощью Aspose.Words для .NET. Эта функция позволяет указать поведение обработки пробелов при загрузке документа TXT.

## Шаг 1. Настройка среды

Прежде чем начать, убедитесь, что вы настроили среду разработки с Aspose.Words для .NET. Убедитесь, что вы добавили необходимые ссылки и импортировали соответствующие пространства имен.

## Шаг 2: Создание текстового документа

```csharp
// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENTS DIRECTORY";

const string textDoc = "Line 1\n" +
                        "Line 2\n" +
                        "Line 3";
```

На этом шаге мы создаем текстовую строку, имитирующую текстовый документ, содержащий строки с начальными и конечными пробелами.

## Шаг 3. Настройка параметров загрузки

```csharp
TxtLoadOptions loadOptions = new TxtLoadOptions
{
     LeadingSpacesOptions = TxtLeadingSpacesOptions.Trim,
     TrailingSpacesOptions = TxtTrailingSpacesOptions.Trim
};
```

 На этом этапе мы настраиваем параметры загрузки документа TXT. Мы создаем новый`TxtLoadOptions` объект и установить`LeadingSpacesOptions` и`TrailingSpacesOptions` свойства к`TxtLeadingSpacesOptions.Trim` и`TxtTrailingSpacesOptions.Trim` соответственно. Это говорит Aspose.Words удалять начальные и конечные пробелы из строк при загрузке документа.

## Шаг 4: Загрузка документа

```csharp
Document doc = new Document(new MemoryStream(Encoding.UTF8.GetBytes(textDoc)), loadOptions);
```

 На этом шаге мы загружаем документ с помощью`Document` метода и передачи потока памяти, содержащего указанную текстовую строку и параметры загрузки.

## Шаг 5: Сохраните документ

```csharp
doc.Save(dataDir + "WorkingWithTxtLoadOptions.HandleSpacesOptions.docx");
```

 На этом последнем шаге мы сохраняем полученный документ в формате .docx, используя`Save` метод и передача пути к выходному файлу.

Теперь вы можете запустить исходный код для загрузки текстового документа, указав параметры обработки пробелов. Полученный документ будет сохранен в указанном каталоге с именем «WorkingWithTxtLoadOptions.HandleSpacesOptions.docx».

### Пример исходного кода для функции управления пространством с параметрами загрузки TXT с помощью Aspose.Words для .NET*

```csharp

            
// Путь к вашему каталогу документов
string dataDir = "YOUR DOCUMENT DIRECTORY";

const string textDoc = "      Line 1 \n" +
					   "    Line 2   \n" +
					   " Line 3       ";

TxtLoadOptions loadOptions = new TxtLoadOptions
{
	LeadingSpacesOptions = TxtLeadingSpacesOptions.Trim,
	TrailingSpacesOptions = TxtTrailingSpacesOptions.Trim
};

Document doc = new Document(new MemoryStream(Encoding.UTF8.GetBytes(textDoc)), loadOptions);

doc.Save(dataDir + "WorkingWithTxtLoadOptions.HandleSpacesOptions.docx")
            
        
```
