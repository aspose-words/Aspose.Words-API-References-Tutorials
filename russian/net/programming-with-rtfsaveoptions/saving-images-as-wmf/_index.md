В этом руководстве мы рассмотрим исходный код C#, предоставленный для функции «Сохранение изображений в формате WMF с параметрами сохранения RTF» с помощью Aspose.Words для .NET. Эта функция позволяет сохранять изображения документов в формате метафайла Windows (WMF) при преобразовании в формат RTF.

## Шаг 1. Настройка среды

Прежде чем начать, убедитесь, что вы настроили среду разработки с Aspose.Words для .NET. Убедитесь, что вы добавили необходимые ссылки и импортировали соответствующие пространства имен.

## Шаг 2: Загрузка документа

```csharp
// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENTS DIRECTORY";

Document doc = new Document(dataDir + "Document.docx");
```

 На этом шаге мы загружаем документ с помощью`Document` Метод и передача пути к файлу DOCX для загрузки.

## Шаг 3. Настройка параметров резервного копирования

```csharp
RtfSaveOptions saveOptions = new RtfSaveOptions { SaveImagesAsWmf = true };
```

 На этом этапе мы настраиваем параметры резервного копирования RTF. Мы создаем новый`RtfSaveOptions` объект и установить`SaveImagesAsWmf` собственность на`true`. Это говорит Aspose.Words сохранять изображения документов в формате WMF при преобразовании в RTF.

## Шаг 4: Сохранение документа

```csharp
doc.Save(dataDir + "WorkingWithRtfSaveOptions.SavingImagesAsWmf.rtf", saveOptions);
```

 На этом последнем шаге мы сохраняем полученный документ в формате RTF, используя`Save` метод и передачу пути к выходному файлу вместе с указанными параметрами сохранения.

Теперь вы можете запускать исходный код для сохранения изображений документов в формате WMF при преобразовании в формат RTF. Полученный документ будет сохранен в указанном каталоге с именем «WorkingWithRtfSaveOptions.SavingImagesAsWmf.rtf».

### Пример исходного кода для функции сохранения изображений WMF с параметрами сохранения RTF с помощью Aspose.Words for .NET".

```csharp

            
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Document.docx");

RtfSaveOptions saveOptions = new RtfSaveOptions { SaveImagesAsWmf = true };

doc.Save(dataDir + "WorkingWithRtfSaveOptions.SavingImagesAsWmf.rtf", saveOptions);
            
        
```
