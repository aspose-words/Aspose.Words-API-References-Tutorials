В этом руководстве мы пошагово проведем вас через процесс установки папок шрифтов с приоритетом при рендеринге документа с использованием Aspose.Words для .NET. Мы объясним прилагаемый исходный код C# и предоставим вам исчерпывающее руководство, которое поможет вам понять и реализовать эту функцию в ваших собственных проектах. В конце этого руководства вы узнаете, как указать несколько папок шрифтов с настраиваемым приоритетом поиска при отображении ваших документов с помощью Aspose.Words для .NET.

## Шаг 1: Определите каталог документов
Во-первых, вам нужно указать путь к каталогу ваших документов. Это место, где вы хотите сохранить отредактированный визуализированный документ. Замените «КАТАЛОГ ВАШИХ ДОКУМЕНТОВ» на соответствующий путь.

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## Шаг 2: Установите папки шрифтов с приоритетом
 Затем вы можете установить папки шрифтов с приоритетом, используя`FontSettings` класс и`SetFontsSources()`метод. Вы можете указать несколько источников шрифтов, используя экземпляры`SystemFontSource` и`FolderFontSource`. В этом примере мы определили два источника шрифтов: источник системного шрифта по умолчанию и папку пользовательского шрифта с приоритетом 1.

```csharp
FontSettings.DefaultInstance.SetFontsSources(new FontSourceBase[]
{
new SystemFontSource(), new FolderFontSource("C:\\MyFonts\\", true, 1)
});
```

## Шаг 3: Загрузите документ для рендеринга
 Теперь вы можете загрузить документ для рендеринга с помощью`Document` сорт. Обязательно укажите правильный путь к документу.

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## Шаг 4: Сохраните визуализированный документ
 Наконец, вы можете сохранить визуализированный документ в файл, используя`Save()` метод`Document` сорт. Обязательно укажите правильный путь и имя файла.

```csharp
doc.Save(dataDir + "WorkingWithFonts.SetFontsFoldersWithPriority.pdf");
```

### Пример исходного кода для установки папок шрифтов с приоритетом с использованием Aspose.Words для .NET 
```csharp
//Путь к вашему каталогу документов
string dataDir = "YOUR DOCUMENT DIRECTORY";

FontSettings.DefaultInstance.SetFontsSources(new FontSourceBase[]
{
	new SystemFontSource(), new FolderFontSource("C:\\MyFonts\\", true,1)
});
Document doc = new Document(dataDir + "Rendering.docx");
doc.Save(dataDir + "WorkingWithFonts.SetFontsFoldersWithPriority.pdf");
```

## Заключение
В этом руководстве мы узнали, как установить папки шрифтов с приоритетом при рендеринге документа с использованием Aspose.Words для .NET. Следуя этому пошаговому руководству, вы можете легко указать несколько папок шрифтов с настраиваемым приоритетом поиска при отображении ваших документов. Aspose.Words предлагает мощный и гибкий API для обработки слов со шрифтами в ваших документах. Обладая этими знаниями, вы можете контролировать и настраивать источники шрифтов, используемые при рендеринге ваших документов, в соответствии с вашими конкретными потребностями.

### Часто задаваемые вопросы

#### Q: Как установить приоритет папок со шрифтами в Aspose.Words?

 О: Чтобы установить приоритет папок со шрифтами в Aspose.Words, вы можете использовать`SetFontsFoldersWithPriority` метод`Fonts` class, указав расположение папок шрифтов и их порядок приоритета.

#### В: Что произойдет, если шрифт находится в нескольких папках с разным приоритетом?

О: Если шрифт присутствует в нескольких папках с разным приоритетом, Aspose.Words будет использовать версию из папки с наивысшим приоритетом при обработке документов.

#### В: Могу ли я указать несколько папок шрифтов с одинаковым приоритетом в Aspose.Words?

О: Да, вы можете указать несколько папок шрифтов с одинаковым приоритетом в Aspose.Words. Aspose.Words будет рассматривать их все с одинаковым приоритетом при поиске шрифтов в ваших документах.

#### Q: Как я могу проверить папки со шрифтами, определенные с приоритетом в Aspose.Words?

 A: Чтобы проверить папки шрифтов, определенные в Aspose.Words с приоритетом, вы можете использовать`GetFolders` метод`Fonts` class, чтобы получить список настроенных папок шрифтов, включая их порядок приоритета.

#### Q: Какой смысл в установке папок шрифтов с приоритетом в Aspose.Words?

A: Установка папок шрифтов с приоритетом в Aspose.Words позволяет вам управлять порядком поиска шрифтов в ваших документах Word. Это поможет вам убедиться, что нужные шрифты используются, и избежать нежелательных проблем с заменой шрифтов.
