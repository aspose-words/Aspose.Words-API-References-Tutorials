## مقدمة

Aspose.Words for .NET هي مكتبة قوية تتيح لك إنشاء مستندات Word ومعالجتها وتحويلها برمجيًا. إحدى المهام الشائعة التي قد تحتاج إلى تنفيذها هي استرداد تباعد الأسطر لخط معين في المستند. في هذا البرنامج التعليمي، سنرشدك خلال العملية خطوة بخطوة، مما يضمن أنه يمكنك بسهولة الحصول على تباعد الأسطر باستخدام Aspose.Words for .NET. 

## المتطلبات الأساسية

قبل أن نتعمق في الكود، تأكد من توفر المتطلبات الأساسية التالية:

1.  Aspose.Words for .NET Library: قم بتنزيل أحدث إصدار وتثبيته من[هنا](https://releases.aspose.com/words/net/).
2. بيئة التطوير: تأكد من أن لديك IDE مثل إعداد Visual Studio.
3. المعرفة الأساسية بـ C#: يفترض هذا البرنامج التعليمي أن لديك فهمًا أساسيًا لبرمجة C#.

## استيراد مساحات الأسماء

أولاً، تحتاج إلى استيراد مساحات الأسماء الضرورية في مشروع C# الخاص بك. ستسمح لك مساحات الأسماء هذه بالوصول إلى وظائف Aspose.Words.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Fonts;
```

دعونا نقسم عملية الحصول على تباعد الأسطر إلى خطوات بسيطة يمكن التحكم فيها.

## الخطوة 1: إنشاء مستند جديد

الخطوة الأولى هي إنشاء مثيل مستند Word جديد باستخدام Aspose.Words لـ .NET.

```csharp
Document doc = new Document();
```

## الخطوة 2: تهيئة DocumentBuilder

بعد ذلك، نحن بحاجة إلى تهيئة`DocumentBuilder` هدف. سيساعدنا هذا الكائن في إنشاء محتوى المستند ومعالجته.

```csharp
DocumentBuilder builder = new DocumentBuilder(doc);
```

## الخطوة 3: تعيين خصائص الخط

الآن، قمنا بتعيين خصائص الخط للنص الذي نريد إدراجه. في هذا المثال، سوف نستخدم الخط "Calibri".

```csharp
builder.Font.Name = "Calibri";
```

## الخطوة 4: كتابة نص في المستند

 باستخدام`DocumentBuilder` الكائن، اكتب بعض النص في المستند. سيستخدم هذا النص خصائص الخط التي قمنا بتعيينها في الخطوة السابقة.

```csharp
builder.Writeln("Sample Text");
```

## الخطوة 5: استرداد كائن الخط

للحصول على تباعد الأسطر، نحتاج إلى الوصول إلى كائن الخط للنص الذي أضفناه للتو. يمكن القيام بذلك عن طريق التنقل عبر بنية المستند إلى تشغيل الفقرة الأولى.

```csharp
Font font = builder.Document.FirstSection.Body.FirstParagraph.Runs[0].Font;
```

## الخطوة 6: احصل على تباعد الأسطر

أخيرًا، نقوم باسترداد تباعد الأسطر من كائن الخط وطباعته على وحدة التحكم.

```csharp
Console.WriteLine($"lineSpacing = {font.LineSpacing}");
```

## خاتمة

وهنا لديك! يعد استرداد تباعد أسطر الخط باستخدام Aspose.Words for .NET أمرًا سهلاً عندما تقوم بتقسيمه إلى هذه الخطوات البسيطة. سواء كنت تقوم بإنشاء مستند جديد أو تعمل مع مستند موجود، فإن Aspose.Words يوفر جميع الأدوات التي تحتاجها لإدارة خصائص الخط بكفاءة.
