في هذا البرنامج التعليمي، سنخبرك بكيفية الحصول على تباعد أسطر الخط في مستند Word باستخدام مكتبة Aspose.Words لـ .NET. يحدد تباعد أسطر الخط المسافة الرأسية بين أسطر النص. سنأخذك خطوة بخطوة لمساعدتك على فهم التعليمات البرمجية وتنفيذها في مشروع .NET الخاص بك.

## المتطلبات الأساسية
قبل البدء، تأكد من أن لديك العناصر التالية:
- معرفة عملية بلغة البرمجة C#
- تم تثبيت مكتبة Aspose.Words الخاصة بـ .NET في مشروعك

## الخطوة 1: إنشاء مستند جديد ومولد المستندات
 أولاً، سنقوم بإنشاء مستند جديد عن طريق إنشاء مثيل لـ`Document` فئة ومنشئ المستندات عن طريق إنشاء مثيل لـ`DocumentBuilder` فصل.

```csharp
// إنشاء مستند جديد
Document doc = new Document();

//إنشاء مولد وثيقة
DocumentBuilder builder = new DocumentBuilder(doc);
```

## الخطوة 2: تكوين الخط
 بعد ذلك، سوف نقوم بتكوين الخط عن طريق تعيين`Name` خاصية مولد الوثيقة.

```csharp
// تكوين الخط
builder.Font.Name = "Calibri";
```

## الخطوة 3: إضافة نص إلى المستند
سنستخدم الآن منشئ المستندات لإضافة نص منسق إلى المستند.

```csharp
// إضافة نص إلى الوثيقة
builder. Writen("qText");
```

## الخطوة 4: الحصول على تباعد الأسطر بين الخطوط
 الآن سوف نقوم بالوصول إلى`Font` كائن الفقرة الأولى من المستند واسترجاع قيمة`LineSpacing` ملكية.

```csharp
// الحصول على تباعد الأسطر من الخط
Font font = builder.Document.FirstSection.Body.FirstParagraph.Runs[0].Font;
Console.WriteLine($"lineSpacing = {font.LineSpacing}");
```

### نموذج التعليمات البرمجية المصدر للحصول على تباعد الأسطر باستخدام Aspose.Words لـ .NET 
```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
builder.Font.Name = "Calibri";
builder.Writeln("qText");
Font font = builder.Document.FirstSection.Body.FirstParagraph.Runs[0].Font;
Console.WriteLine($"lineSpacing = {font.LineSpacing}");
```

في هذا البرنامج التعليمي، رأينا كيفية الحصول على تباعد الأسطر في مستند Word باستخدام Aspose.Words for .NET. يعد تباعد أسطر الخط أمرًا مهمًا للتحكم في التباعد الرأسي بين أسطر النص. لا تتردد في استخدام هذه الميزة لتخصيص مظهر النص في مستنداتك.
