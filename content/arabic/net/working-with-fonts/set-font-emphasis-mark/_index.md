## مقدمة

في البرنامج التعليمي اليوم، سنتعمق في كيفية تعيين علامات التركيز على الخط في مستند Word باستخدام Aspose.Words for .NET. سواء كنت تريد وضع خط تحت نص معين بعلامة فريدة أو ببساطة إبراز كلمات معينة، فإن هذا الدليل يوفر لك كل ما تحتاجه. لذلك، ربط حزام الأمان ودعنا نبدأ!

## استيراد مساحات الأسماء

للعمل مع Aspose.Words لـ .NET، ستحتاج إلى استيراد مساحات الأسماء الضرورية. أضف هذه في أعلى ملف التعليمات البرمجية الخاص بك:

```csharp
using Aspose.Words;
using Aspose.Words.Fonts;
```

الآن، دعونا نقسم العملية إلى خطوات بسيطة. اتبع كل خطوة بعناية لتعيين علامات التركيز على الخط في مستند Word الخاص بك.

## الخطوة 1: تهيئة المستند و DocumentBuilder

أول الأشياء أولاً، تحتاج إلى تهيئة مستند جديد وDocumentBuilder. توفر فئة DocumentBuilder طرقًا لإدراج النص والعناصر الأخرى في المستند.

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";

// تهيئة مستند جديد
Document document = new Document();

// قم بتهيئة DocumentBuilder باستخدام المستند
DocumentBuilder builder = new DocumentBuilder(document);
```

## الخطوة 2: تعيين علامة التأكيد على الخط

بعد أن أصبح DocumentBuilder جاهزًا، يمكنك الآن تعيين علامة تركيز الخط. في هذا المثال، سنستخدم علامة التركيز "UnderSolidCircle".

```csharp
// قم بتعيين علامة التركيز على الخط
builder.Font.EmphasisMark = EmphasisMark.UnderSolidCircle;

// كتابة النص مع علامة التركيز
builder.Write("Emphasis text");
builder.Writeln();
```

## الخطوة 3: مسح التنسيق وإضافة نص عادي

بعد تعيين علامة التركيز، قد ترغب في إضافة بعض النصوص العادية دون أي تركيز. للقيام بذلك، تحتاج إلى مسح التنسيق.

```csharp
// امسح تنسيق الخط
builder.Font.ClearFormatting();

// اكتب نصًا عاديًا
builder.Write("Simple text");
```

## الخطوة 4: احفظ المستند

بمجرد إضافة كل النص والتنسيق الذي تحتاجه، فإن الخطوة الأخيرة هي حفظ المستند. حدد المسار واسم الملف الذي تريد حفظ المستند فيه.

```csharp
// احفظ المستند
document.Save(dataDir + "WorkingWithFonts.SetFontEmphasisMark.docx");
```
