## مقدمة

قد يمثل ضمان عرض مستندات Word بشكل صحيح عبر الأنظمة الأساسية المختلفة تحديًا، خاصة عند التعامل مع توافق الخطوط. تتمثل إحدى طرق الحفاظ على المظهر المتسق في تحديد خط افتراضي عند تحويل مستنداتك إلى PDF أو تنسيقات أخرى. في هذا البرنامج التعليمي، سنستكشف كيفية تعيين خط افتراضي باستخدام Aspose.Words لـ .NET، بحيث تبدو مستنداتك رائعة بغض النظر عن مكان عرضها.

## المتطلبات الأساسية

قبل الغوص في الكود، دعنا نغطي ما ستحتاج إلى متابعته مع هذا البرنامج التعليمي:

- Aspose.Words for .NET: تأكد من تثبيت أحدث إصدار لديك. يمكنك تنزيله[هنا](https://releases.aspose.com/words/net/).
- بيئة التطوير: Visual Studio أو أي بيئة تطوير .NET أخرى.
- المعرفة الأساسية بـ C#: يفترض هذا البرنامج التعليمي أنك مرتاح لبرمجة C#.

## استيراد مساحات الأسماء

للبدء، تحتاج إلى استيراد مساحات الأسماء الضرورية. سيسمح لك ذلك بالوصول إلى الفئات والأساليب المطلوبة للعمل مع Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Fonts;
```

الآن، دعونا نقسم عملية تحديد الخط الافتراضي إلى خطوات سهلة المتابعة.

## الخطوة 1: قم بإعداد دليل المستندات الخاص بك

أولاً، حدد المسار إلى دليل المستندات الخاص بك. هذا هو المكان الذي سيتم فيه تخزين ملفات الإدخال والإخراج.

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## الخطوة 2: قم بتحميل المستند الخاص بك

بعد ذلك، قم بتحميل المستند الذي تريد تقديمه. في هذا المثال، سنستخدم ملفًا باسم "Rendering.docx".

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## الخطوة 3: تكوين إعدادات الخط

 إنشاء مثيل ل`FontSettings` وحدد الخط الافتراضي. إذا تعذر العثور على الخط المحدد أثناء العرض، فسيستخدم Aspose.Words أقرب خط متاح على الجهاز.

```csharp
FontSettings fontSettings = new FontSettings();
fontSettings.SubstitutionSettings.DefaultFontSubstitution.DefaultFontName = "Arial Unicode MS";
```

## الخطوة 4: تطبيق إعدادات الخط على المستند

قم بتعيين إعدادات الخط المكونة للمستند الخاص بك.

```csharp
doc.FontSettings = fontSettings;
```

## الخطوة 5: احفظ المستند

وأخيرا، احفظ المستند بالتنسيق المطلوب. في هذه الحالة، سنقوم بحفظه بصيغة PDF.

```csharp
doc.Save(dataDir + "WorkingWithFonts.SpecifyDefaultFontWhenRendering.pdf");
```

## خاتمة

باتباع هذه الخطوات، يمكنك التأكد من عرض مستندات Word الخاصة بك بخط افتراضي محدد، مع الحفاظ على الاتساق عبر الأنظمة الأساسية المختلفة. يمكن أن يكون هذا مفيدًا بشكل خاص للمستندات التي تتم مشاركتها على نطاق واسع أو التي يتم عرضها على أنظمة ذات خطوط متفاوتة التوفر.


## الأسئلة الشائعة

### لماذا تحديد الخط الافتراضي في Aspose.Words؟
يؤدي تحديد خط افتراضي إلى ضمان ظهور مستندك بشكل متسق عبر الأنظمة الأساسية المختلفة، حتى إذا كانت الخطوط الأصلية غير متوفرة.

### ماذا يحدث إذا لم يتم العثور على الخط الافتراضي أثناء العرض؟
سيستخدم Aspose.Words أقرب خط متوفر على الجهاز للحفاظ على مظهر المستند قدر الإمكان.

### هل يمكنني تحديد خطوط افتراضية متعددة؟
 لا، يمكنك تحديد خط افتراضي واحد فقط. ومع ذلك، يمكنك التعامل مع استبدال الخط لحالات معينة باستخدام`FontSettings` فصل.

### هل يتوافق Aspose.Words for .NET مع كافة إصدارات مستندات Word؟
نعم، يدعم Aspose.Words for .NET نطاقًا واسعًا من تنسيقات مستندات Word، بما في ذلك DOC وDOCX وRTF والمزيد.

### أين يمكنني الحصول على الدعم إذا واجهت مشاكل؟
 يمكنك الحصول على الدعم من مجتمع Aspose والمطورين على الموقع[منتدى دعم Aspose.Words](https://forum.aspose.com/c/words/8).
