## استيراد مساحات الأسماء

أول الأشياء أولاً، لنقم بإعداد مشروعنا واستيراد مساحات الأسماء الضرورية. تعد مساحات الأسماء هذه ضرورية للوصول إلى وظيفة Aspose.Words.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.ReportingServices;
```

## الخطوة 1: إنشاء مستند جديد


 تبدأ الرحلة بإنشاء مستند جديد وتهيئة ملف`DocumentBuilder` . ال`DocumentBuilder` يوفر class مجموعة من الطرق لمعالجة محتوى المستند بسهولة.

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";

// إنشاء مستند جديد
Document doc = new Document();

// تهيئة DocumentBuilder
DocumentBuilder builder = new DocumentBuilder(doc);
```

### توضيح

 في هذه الخطوة، قمنا بإعداد المستند والمنشئ الذي سنستخدمه لإدراج المحتوى الخاص بنا وتنسيقه. ال`dataDir` يحمل المتغير المسار الذي سنحفظ فيه وثيقتنا النهائية.

## الخطوة 2: أدخل مخططًا

 بعد ذلك، سنضيف مخططًا دائريًا إلى وثيقتنا. ال`InsertChart` طريقة`DocumentBuilder` الفصل يجعل هذا الأمر سهلاً للغاية.

```csharp
// أدخل مخططًا دائريًا
Shape shape = builder.InsertChart(ChartType.Pie, 432, 252);

// الوصول إلى كائن المخطط
Chart chart = shape.Chart;
```

### توضيح

هنا، نقوم بإدراج مخطط دائري في وثيقتنا. ال`InsertChart` تتطلب الطريقة نوع المخطط والعرض والارتفاع كمعلمات. بعد إدراج المخطط، يمكننا الوصول إلى كائن المخطط لمعالجته بشكل أكبر.

## الخطوة 3: تخصيص سلسلة المخططات

الآن، سنقوم بمسح أي سلسلة موجودة في المخطط وإضافة سلسلتنا المخصصة. هذه السلسلة سوف تمثل نقاط البيانات لدينا.

```csharp
// مسح سلسلة المخططات الموجودة
chart.Series.Clear();

// إضافة سلسلة جديدة إلى المخطط
ChartSeries series = chart.Series.Add("Aspose Series 1",
    new string[] { "Category 1", "Category 2", "Category 3" },
    new double[] { 2.7, 3.2, 0.8 });
```

### توضيح

في هذه الخطوة، نتأكد من أن المخطط الخاص بنا فارغ عن طريق مسح أي سلسلة موجودة مسبقًا. بعد ذلك، نضيف سلسلة جديدة بفئات وقيم مخصصة، والتي سيتم عرضها في المخطط الدائري الخاص بنا.

## الخطوة 4: تعيين الخيارات الافتراضية لتسميات البيانات

تعتبر تسميات البيانات ضرورية لجعل المخطط الخاص بك غنيًا بالمعلومات. سنقوم بتعيين خيارات لإظهار النسبة المئوية والقيمة وتخصيص الفاصل.

```csharp
// الوصول إلى مجموعة تسميات البيانات
ChartDataLabelCollection labels = series.DataLabels;

// ضبط خيارات تسمية البيانات
labels.ShowPercentage = true;
labels.ShowValue = true;
labels.ShowLeaderLines = false;
labels.Separator = " - ";
```

### توضيح

 هنا، نحن نصل إلى`DataLabels`خاصية سلسلتنا لتخصيص المظهر والمعلومات المعروضة على كل ملصق بيانات. لقد اخترنا إظهار كل من النسبة المئوية والقيمة، وإخفاء الخطوط الرئيسية، وتعيين فاصل مخصص.

## الخطوة 5: احفظ المستند

وأخيرًا، سنقوم بحفظ المستند في الدليل المحدد. تضمن هذه الخطوة كتابة جميع تغييراتنا في ملف.

```csharp
// احفظ المستند
doc.Save(dataDir + "WorkingWithCharts.DefaultOptionsForDataLabels.docx");
```

### توضيح

 في هذه الخطوة الأخيرة، نقوم بحفظ المستند الخاص بنا باستخدام الملف`Save` طريقة. سيتم حفظ المستند في الدليل المحدد بواسطة`dataDir`، بالاسم "WorkingWithCharts.DefaultOptionsForDataLabels.docx".
