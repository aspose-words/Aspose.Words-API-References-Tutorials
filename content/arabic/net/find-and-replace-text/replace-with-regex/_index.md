## استيراد مساحات الأسماء

أول الأشياء أولاً، نحتاج إلى استيراد مساحات الأسماء الضرورية. في ملف C# الخاص بك، أضف ما يلي باستخدام العبارات في الأعلى:

```csharp
using System;
using System.Text.RegularExpressions;
using Aspose.Words;
using Aspose.Words.Replacing;
```

## الخطوة 1: قم بإعداد دليل المستندات الخاص بك

لنبدأ بتحديد المسار إلى دليل المستندات الخاص بك. هذا هو المكان الذي يتم فيه تخزين مستندات Word الخاصة بك والمكان الذي سنحفظ فيه المستند المعدل.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 يستبدل`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي إلى الدليل الخاص بك.

## الخطوة 2: إنشاء مستند جديد

 بعد ذلك، سنقوم بإنشاء مستند جديد و`DocumentBuilder` لإضافة بعض النص الأولي.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

builder.Writeln("sad mad bad");
```

هنا، نقوم بإنشاء مستند جديد وإضافة النص "sad mad bad" إليه. سيكون هذا النص بمثابة بيانات الاختبار الخاصة بنا لاستبدال التعبير العادي.

## الخطوة 3: تحديد خيارات البحث والاستبدال

 لإجراء استبدال regex، نحتاج إلى إعداد بعض الخيارات. ال`FindReplaceOptions`تسمح لنا الفئة بتحديد الطريقة التي يجب أن تتصرف بها عملية البحث والاستبدال.

```csharp
FindReplaceOptions options = new FindReplaceOptions();
```

في الوقت الحالي، نستخدم الخيارات الافتراضية، ولكن يمكنك تخصيصها إذا لزم الأمر.

## الخطوة 4: إجراء استبدال Regex

 الآن يأتي الجزء الممتع! سوف نستخدم`Range.Replace` طريقة لاستبدال كافة تكرارات "sad" أو "mad" بكلمة "bad" باستخدام تعبير عادي.

```csharp
doc.Range.Replace(new Regex("[s|m]ad"), "bad", options);
```

 نمط ريكس`[s|m]ad` يطابق أي كلمة تنتهي بـ "ad" والتي تبدأ إما بـ "s" أو "m". ستحل سلسلة الاستبدال "bad" محل أي تطابقات تم العثور عليها.

## الخطوة 5: احفظ المستند المعدل

وأخيرًا، سنقوم بحفظ المستند المعدل في الدليل المحدد لدينا.

```csharp
doc.Save(dataDir + "FindAndReplace.ReplaceWithRegex.docx");
```

 هذا السطر يحفظ المستند باسم الملف`FindAndReplace.ReplaceWithRegex.docx` في الدليل المحدد بواسطة`dataDir`.
