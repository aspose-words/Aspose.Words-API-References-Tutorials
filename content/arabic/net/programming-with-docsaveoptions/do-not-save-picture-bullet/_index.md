تعد التعداد النقطي للصور ميزة شائعة الاستخدام في مستندات Word لإضافة تعداد نقطي مخصص. ومع ذلك، في بعض الحالات قد يكون من الضروري تعطيل تسجيل التعداد النقطي للصور عند معالجة المستندات باستخدام مكتبة Aspose.Words لـ .NET. في هذا الدليل خطوة بخطوة، سنشرح كيفية استخدام الكود المصدري لـ Aspose.Words C# لـ .NET لتعطيل حفظ الصورة النقطية باستخدام خيارات حفظ DocSaveOptions.

## فهم مكتبة Aspose.Words

قبل الغوص في التعليمات البرمجية، من المهم فهم مكتبة Aspose.Words الخاصة بـ .NET. Aspose.Words هي مكتبة قوية لإنشاء مستندات Word وتحريرها وتحويلها وحمايتها في منصات مختلفة بما في ذلك .NET. فهو يوفر العديد من الميزات لمعالجة المستندات، مثل إدراج النص وتغيير التنسيق وإضافة الأقسام وغير ذلك الكثير.

## الخطوة 1: إعداد دليل المستندات

الخطوة الأولى هي تحديد الدليل الذي توجد به مستنداتك. يجب عليك تحديد مسار الدليل الكامل. على سبيل المثال :

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

تأكد من استبدال "دليل مستنداتك" بالمسار الفعلي لدليل مستنداتك.

## الخطوة 2: تحميل المستند باستخدام صورة نقطية

بعد ذلك، تحتاج إلى تحميل المستند مع الصور النقطية. استخدم فئة المستند لتحميل المستند من ملف. على سبيل المثال :

```csharp
Document doc = new Document(dataDir + "Image bullet points.docx");
```

في هذا المثال نقوم بتحميل المستند من الملف "Image Bullet Points.docx"

  الموجود في دليل المستندات.

## الخطوة 3: تكوين خيارات التسجيل

الآن دعونا نقوم بتكوين خيارات الحفظ لمستندنا. استخدم فئة DocSaveOptions لتحديد إعدادات الحفظ. على سبيل المثال :

```csharp
DocSaveOptions saveOptions = new DocSaveOptions { SavePictureBullet = false };
```

في هذا المثال، قمنا بإنشاء كائن DocSaveOptions جديد وقمنا بتعيين الخاصية SavePictureBullet إلى false لتعطيل حفظ الصور النقطية.

## الخطوة 4: تمكين ميزة "عدم حفظ الصورة النقطية".

لتمكين ميزة "عدم حفظ الصورة النقطية"، قمنا بالفعل بتكوين خيارات الحفظ مع تعيين SavePictureBullet على "خطأ". وهذا يضمن عدم حفظ الصور النقطية في المستند النهائي.

## الخطوة 5: احفظ المستند

وأخيرًا، يمكنك حفظ المستند باستخدام طريقة الحفظ الخاصة بفئة المستند. حدد المسار الكامل للملف واسم الملف المطلوب. على سبيل المثال :

```csharp
doc.Save(dataDir + "WorkingWithDocSaveOptions.DoNotSavePictureBullet.docx", saveOptions);
```

تأكد من استبدال "dataDir" بمسار الدليل إلى مستنداتك.

## مثال على التعليمات البرمجية المصدر لخيارات حفظ DocSaveOptions مع وظيفة "Do Not Save Picture Bullet" باستخدام Aspose.Words for .NET

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// قم بتحميل المستند باستخدام الصور النقطية
Document doc = new Document(dataDir + "Image bullet points.docx");

// قم بتكوين خيارات الحفظ باستخدام ميزة "عدم حفظ الصورة النقطية".
DocSaveOptions saveOptions = new DocSaveOptions { SavePictureBullet = false };

// احفظ المستند بالخيارات المحددة
doc.Save(dataDir + "WorkingWithDocSaveOptions.DoNotSavePictureBullet.docx", saveOptions);
```

## خاتمة

في هذا الدليل، تناولنا كيفية تعطيل حفظ التعداد النقطي للصور في مستند باستخدام مكتبة Aspose.Words لـ .NET. باتباع الخطوات المقدمة واستخدام كود مصدر C# المقدم، يمكنك بسهولة تطبيق هذه الوظيفة في تطبيق C# الخاص بك. يمكن أن يكون تعطيل حفظ التعداد النقطي للصور مفيدًا في بعض المواقف للحفاظ على بنية المستند وتنسيقه دون حفظ التعداد النقطي للصور.
