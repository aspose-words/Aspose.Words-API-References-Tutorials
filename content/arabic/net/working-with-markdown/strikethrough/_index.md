للبدء، ستحتاج إلى استيراد مساحات الأسماء الضرورية. تعتبر هذه العناصر ضرورية للوصول إلى مكتبة Aspose.Words وميزاتها.

```csharp

```

## الخطوة 1: تهيئة DocumentBuilder

 ال`DocumentBuilder` تعد class أداة قوية في Aspose.Words تتيح لك إضافة محتوى إلى مستندك بسهولة.

```csharp
// تهيئة DocumentBuilder.
DocumentBuilder builder = new DocumentBuilder();
```

## الخطوة 2: تعيين خاصية يتوسطه خط

الآن، دعونا نطبق خاصية يتوسطه خط على النص لدينا. وهذا ينطوي على تحديد`StrikeThrough` ملكية`Font` يعترض على`true`.

```csharp
// جعل النص يتوسطه خط.
builder.Font.StrikeThrough = true;
```

## الخطوة 3: كتابة نص يتوسطه خط

 مع مجموعة الخاصية يتوسطه خط، يمكننا الآن إضافة النص الخاص بنا. ال`Writeln` ستضيف الطريقة النص إلى المستند.

```csharp
// كتابة النص مع يتوسطه خط.
builder.Writeln("This text will be StrikeThrough");
```
