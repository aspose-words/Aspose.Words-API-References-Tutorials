## استيراد مساحات الأسماء

أول الأشياء أولاً، فلنستورد مساحات الأسماء الضرورية. يساعد هذا في الوصول إلى الفئات والأساليب المطلوبة للواصلة.

```csharp
using Aspose.Words;
using Aspose.Words.Hyphenation;
```

## الخطوة 1: قم بتحميل المستند الخاص بك

 ستحتاج إلى تحديد الدليل الذي يوجد به المستند الخاص بك. يستبدل`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي إلى المستند الخاص بك.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "German text.docx");
```

## الخطوة 3: تسجيل قواميس الواصلة

 يتطلب Aspose.Words قواميس الواصلة للغات مختلفة. تأكد من حصولك على`.dic`الملفات الخاصة باللغات التي تريد وصلها. قم بتسجيل هذه القواميس باستخدام`Hyphenation.RegisterDictionary` طريقة.

```csharp
Hyphenation.RegisterDictionary("en-US", dataDir + "hyph_en_US.dic");
Hyphenation.RegisterDictionary("de-CH", dataDir + "hyph_de_CH.dic");
```

## الخطوة 4: احفظ المستند

وأخيرًا، احفظ المستند الموصول بالتنسيق المطلوب. وهنا نقوم بحفظه بصيغة PDF.

```csharp
doc.Save(dataDir + "TreatmentByCesure.pdf");
```
