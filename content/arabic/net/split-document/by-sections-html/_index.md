بمجرد الانتهاء من تجهيز هذه العناصر، يمكننا البدء في البرمجة!

## استيراد مساحات الأسماء

أول الأشياء أولاً، فلنستورد مساحات الأسماء الضرورية. سيسمح لنا هذا باستخدام الفئات والأساليب التي توفرها مكتبة Aspose.Words for .NET.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

مع وجود مساحات الأسماء هذه في مكانها الصحيح، تصبح جاهزًا لبدء العمل مع Aspose.Words.

## الخطوة 1: إعداد دليل المستندات

قبل أن نتمكن من معالجة أي مستند، نحتاج إلى تحديد مكان تخزين مستنداتنا. سيكون هذا دليل العمل الخاص بنا.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## الخطوة 2: قم بتحميل مستند Word

 الآن بعد أن قمنا بإعداد الدليل الخاص بنا، نحتاج إلى تحميل مستند Word الذي نريد تقسيمه. ويتم ذلك باستخدام`Document` فئة من Aspose.Words.

```csharp
// قم بتحميل مستند الوورد.
Document doc = new Document(dataDir + "Rendering.docx");
```

## الخطوة 3: تكوين خيارات حفظ HTML

 لتقسيم المستند إلى أقسام، نحتاج إلى تعيين خيارات الحفظ المناسبة. ال`HtmlSaveOptions`تسمح لنا الفئة بتحديد الطريقة التي نريد بها حفظ المستند إلى HTML.

```csharp
// تكوين خيارات حفظ HTML.
HtmlSaveOptions options = new HtmlSaveOptions
{
    DocumentSplitCriteria = DocumentSplitCriteria.SectionBreak
};
```

## الخطوة 4: احفظ المستند بتنسيق HTML

بعد تكوين خيارات الحفظ لدينا، فإن الخطوة الأخيرة هي حفظ المستند كملف HTML. سيؤدي هذا إلى تقسيم المستند إلى أقسام بناءً على المعايير التي وضعناها.

```csharp
// احفظ المستند بتنسيق HTML.
doc.Save(dataDir + "SplitDocument.BySectionsHtml.html", options);
```
