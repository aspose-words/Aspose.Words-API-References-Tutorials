## مقدمة

عند العمل مع الجداول في مستندات Word، قد ترغب في التأكد من عدم تقاطع الصفوف عبر الصفحات، وهو ما قد يكون ضروريًا للحفاظ على سهولة قراءة مستنداتك وتنسيقها. يوفر Aspose.Words for .NET طريقة سهلة لتعطيل فواصل الصفوف عبر الصفحات.

في هذا البرنامج التعليمي، سنرشدك خلال عملية تعطيل فواصل الصفوف عبر الصفحات في مستند Word باستخدام Aspose.Words for .NET.

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من توفر المتطلبات الأساسية التالية:
- تم تثبيت Aspose.Words لمكتبة .NET.
- مستند Word يحتوي على جدول يمتد لعدة صفحات.

## استيراد مساحات الأسماء

أولاً، قم باستيراد مساحات الأسماء الضرورية في مشروعك:

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

## الخطوة 1: قم بتحميل المستند

قم بتحميل المستند الذي يحتوي على الجدول الذي يمتد لعدة صفحات.

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Table spanning two pages.docx");
```

## الخطوة 2: الوصول إلى الجدول

الوصول إلى الجدول الأول في المستند. يفترض هذا أن الجدول الذي تريد تعديله هو الجدول الأول في المستند.

```csharp
Table table = (Table) doc.GetChild(NodeType.Table, 0, true);
```

## الخطوة 3: تعطيل التقسيم عبر الصفحات لجميع الصفوف

 قم بالمرور خلال كل صف في الجدول وقم بتعيين`AllowBreakAcrossPages`الملكية ل`false`. وهذا يضمن عدم تقاطع الصفوف عبر الصفحات.

```csharp
// تعطيل التقسيم عبر الصفحات لجميع الصفوف في الجدول.
foreach (Row row in table.Rows)
    row.RowFormat.AllowBreakAcrossPages = false;
```

## الخطوة 4: احفظ المستند

احفظ المستند المعدل في الدليل المحدد الخاص بك.

```csharp
doc.Save(dataDir + "WorkingWithTables.RowFormatDisableBreakAcrossPages.docx");
```

## خاتمة

في هذا البرنامج التعليمي، أوضحنا كيفية تعطيل فواصل الصفوف عبر الصفحات في مستند Word باستخدام Aspose.Words for .NET. باتباع الخطوات الموضحة أعلاه، يمكنك التأكد من بقاء صفوف الجدول سليمة وعدم تقسيمها عبر الصفحات، مما يحافظ على سهولة قراءة المستند وتنسيقه.

## الأسئلة الشائعة

### هل يمكنني تعطيل فواصل الصفوف عبر الصفحات لصف معين بدلاً من جميع الصفوف؟  
 نعم، يمكنك تعطيل فواصل الصفوف لصفوف معينة عن طريق الوصول إلى الصف المطلوب وتعيينه`AllowBreakAcrossPages`الملكية ل`false`.

### هل تعمل هذه الطريقة مع الجداول التي تحتوي على خلايا مدمجة؟  
 نعم، تعمل هذه الطريقة مع الجداول التي تحتوي على خلايا مدمجة. الملكية`AllowBreakAcrossPages` ينطبق على الصف بأكمله، بغض النظر عن دمج الخلايا.

### هل ستعمل هذه الطريقة إذا كان الجدول متداخلاً داخل جدول آخر؟  
نعم، يمكنك الوصول إلى الجداول المتداخلة وتعديلها بنفس الطريقة. تأكد من الإشارة بشكل صحيح إلى الجدول المتداخل من خلال فهرسه أو خصائصه الأخرى.

### كيف يمكنني التحقق مما إذا كان الصف يسمح بتقسيم الصفحات؟  
 يمكنك التحقق مما إذا كان الصف يسمح بتقسيم الصفحات عن طريق الوصول إلى ملف`AllowBreakAcrossPages` ملكية`RowFormat` والتحقق من قيمتها.

### هل هناك طريقة لتطبيق هذا الإعداد على كافة الجداول في المستند؟  
نعم، يمكنك تكرار جميع الجداول في المستند وتطبيق هذا الإعداد على كل منها.
