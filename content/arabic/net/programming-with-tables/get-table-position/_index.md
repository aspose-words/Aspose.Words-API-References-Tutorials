## استيراد مساحات الأسماء

أول الأشياء أولاً، فلنستورد مساحات الأسماء الضرورية. هذا يشبه إعداد صندوق الأدوات الخاص بك قبل بدء المشروع.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Tables;
```

## الخطوة 1: قم بتحميل المستند الخاص بك

حسنًا، لنقم بتحميل مستند Word الخاص بك. هذا هو المكان الذي ستشير فيه إلى الملف الذي تريد العمل معه.

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";

// قم بتحميل المستند
Document doc = new Document(dataDir + "Tables.docx");
```

## الخطوة 2: الوصول إلى الجدول الأول

الآن، دعونا نضع أيدينا على الجدول الأول في الوثيقة. فكر في هذا كأنك تصطاد أول قطعة حلوى من الجرة.

```csharp
// الوصول إلى الجدول الأول في المستند
Table table = (Table)doc.GetChild(NodeType.Table, 0, true);
```

## الخطوة 3: التحقق من التفاف نص الجدول

يمكن لف الجداول في Word حول النص بطرق مختلفة. دعونا نرى كيف يتم تغليف طاولتنا.

```csharp
// تحقق من تعيين التفاف نص الجدول على "حول"
if (table.TextWrapping == TextWrapping.Around)
{
    // إذا كانت ملفوفة، احصل على المحاذاة الأفقية والرأسية النسبية
    Console.WriteLine(table.RelativeHorizontalAlignment);
    Console.WriteLine(table.RelativeVerticalAlignment);
}
else
{
    // إذا لم تكن ملفوفة، احصل على المحاذاة القياسية
    Console.WriteLine(table.Alignment);
}
```

## الخطوة 4: قم بتشغيل التعليمات البرمجية الخاصة بك

بعد أن تم إعداد كل شيء، حان الوقت لتشغيل التعليمات البرمجية الخاصة بك. افتح وحدة التحكم الخاصة بك وشاهد السحر يتكشف! ستحصل إما على المحاذاة النسبية إذا كان الجدول ملتفًا أو المحاذاة القياسية إذا لم تكن كذلك.

## الخطوة 5: تحليل الإخراج

بمجرد تشغيل التعليمات البرمجية الخاصة بك، سترى تفاصيل موضع الجدول مطبوعة في وحدة التحكم. تعد هذه المعلومات مفيدة للغاية لمحاذاة المحتوى الخاص بك أو تصحيح مشكلات التخطيط.
