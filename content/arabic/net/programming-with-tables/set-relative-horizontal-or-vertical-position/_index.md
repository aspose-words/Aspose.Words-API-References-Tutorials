## استيراد مساحات الأسماء

أول الأشياء أولاً، تحتاج إلى استيراد مساحات الأسماء الضرورية. يعد هذا أمرًا ضروريًا للوصول إلى وظائف Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

## الخطوة 1: قم بتحميل المستند الخاص بك

للبدء، ستحتاج إلى تحميل مستند Word الخاص بك في البرنامج. وإليك كيف يمكنك القيام بذلك:

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Table wrapped by text.docx");
```

يقوم مقتطف الكود هذا بإعداد المسار إلى دليل المستند الخاص بك ويقوم بتحميل المستند المحدد الذي تريد العمل عليه. تأكد من صحة مسار المستند لتجنب أي مشكلات في التحميل.

## الخطوة 2: الوصول إلى الجدول

بعد ذلك، نحتاج إلى الوصول إلى الجدول داخل المستند. عادةً، قد ترغب في العمل مع الجدول الأول في قسم النص الأساسي.

```csharp
Table table = doc.FirstSection.Body.Tables[0];
```

يقوم هذا السطر من التعليمات البرمجية بإحضار الجدول الأول من نص المستند. إذا كان المستند الخاص بك يحتوي على جداول متعددة، فيمكنك ضبط الفهرس وفقًا لذلك.

## الخطوة 3: ضبط الوضع الأفقي

الآن، دعونا نضبط الموضع الأفقي للجدول بالنسبة لعنصر معين. في هذا المثال، سنقوم بوضعه بالنسبة للعمود.

```csharp
table.HorizontalAnchor = RelativeHorizontalPosition.Column;
```

 من خلال تعيين`HorizontalAnchor` ل`RelativeHorizontalPosition.Column`، فأنت تطلب من الجدول محاذاة نفسه أفقيًا بالنسبة للعمود الموجود فيه.

## الخطوة 4: ضبط الوضع الرأسي

كما هو الحال مع الوضع الأفقي، يمكنك أيضًا ضبط الوضع الرأسي. هنا، نقوم بوضعه بالنسبة للصفحة.

```csharp
table.VerticalAnchor = RelativeVerticalPosition.Page;
```

 وضع`VerticalAnchor` ل`RelativeVerticalPosition.Page` يضمن محاذاة الجدول عموديًا وفقًا للصفحة.

## الخطوة 5: احفظ المستند الخاص بك

وأخيرًا، احفظ تغييراتك في مستند جديد. هذه خطوة حاسمة للتأكد من الحفاظ على تغييراتك.

```csharp
doc.Save(dataDir + "WorkingWithTables.SetFloatingTablePosition.docx");
```

يحفظ هذا الأمر المستند المعدل باسم جديد، مما يضمن عدم الكتابة فوق الملف الأصلي.
