## استيراد مساحات الأسماء

أولاً، ستحتاج إلى استيراد مساحات الأسماء الضرورية. سيسمح لك هذا بالوصول إلى فئات وطرق Aspose.Words. إليك كيفية القيام بذلك:

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Tables;
```

الآن بعد أن قمنا بإعداد كل شيء، دعنا نقسم العملية إلى خطوات سهلة المتابعة.

## الخطوة 1: قم بتحميل المستند الخاص بك

الخطوة الأولى هي تحميل مستند Word الخاص بك. يجب أن يحتوي هذا المستند على الجداول التي تريد دمجها. إليك الكود لتحميل مستند:

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Tables.docx");
```

 في هذا المثال، استبدل`"YOUR DOCUMENT DIRECTORY"` مع المسار إلى المستند الخاص بك.

## الخطوة 2: تحديد الجداول

 بعد ذلك، عليك تحديد الجداول التي تريد دمجها. يتيح لك Aspose.Words الحصول على الجداول من مستند باستخدام الملف`GetChild` طريقة. إليك الطريقة:

```csharp
Table firstTable = (Table) doc.GetChild(NodeType.Table, 0, true);
Table secondTable = (Table) doc.GetChild(NodeType.Table, 1, true);
```

في هذا الكود نقوم بإحضار الجدولين الأول والثاني من المستند.

## الخطوة 3: إلحاق صفوف من الجدول الثاني بالجدول الأول

الآن حان الوقت لدمج الصفوف. سنقوم بإلحاق كافة الصفوف من الجدول الثاني إلى الجدول الأول. يتم ذلك باستخدام حلقة بسيطة:

```csharp
// إلحاق كافة الصفوف من الجدول الثاني إلى الجدول الأول
while (secondTable.HasChildNodes)
    firstTable.Rows.Add(secondTable.FirstRow);
```

تستمر هذه الحلقة حتى تتم إضافة كافة الصفوف من الجدول الثاني إلى الجدول الأول.

## الخطوة 4: إزالة الجدول الثاني

 بعد إلحاق الصفوف، لم تعد هناك حاجة إلى الجدول الثاني. يمكنك إزالته باستخدام`Remove` طريقة:

```csharp
secondTable.Remove();
```

## الخطوة 5: احفظ المستند

وأخيرا، احفظ الوثيقة المعدلة. تضمن هذه الخطوة كتابة تغييراتك على الملف:

```csharp
doc.Save(dataDir + "WorkingWithTables.CombineRows.docx");
```
