## مقدمة

هل سبق لك أن وجدت نفسك تعمل مع جدول كبير في مستند Word وتمنيت أن تتمكن من تقسيمه إلى جدولين أصغر وأكثر قابلية للإدارة؟ حسنًا، سنتعمق اليوم في كيفية تحقيق ذلك باستخدام Aspose.Words for .NET. سواء كنت تتعامل مع جداول بيانات شاملة أو هياكل مستندات معقدة، فإن تقسيم الجداول يمكن أن يساعد في تحسين إمكانية القراءة والتنظيم. دعنا نستكشف العملية خطوة بخطوة لتقسيم جدول باستخدام Aspose.Words for .NET.

## المتطلبات الأساسية

قبل أن ننتقل إلى البرنامج التعليمي، تأكد من أن لديك ما يلي:

1.  Aspose.Words لمكتبة .NET: تأكد من تنزيل وتثبيت مكتبة Aspose.Words لـ .NET. يمكنك الحصول عليه من[صفحة الإصدارات Aspose](https://releases.aspose.com/words/net/).
2. بيئة التطوير: قم بإعداد بيئة تطوير تدعم .NET Framework، مثل Visual Studio.
3. نموذج مستند: تحضير مستند Word (`Tables.docx`) مع جدول واحد على الأقل لتطبيق عملية التقسيم.

## استيراد مساحات الأسماء

أولاً، قم باستيراد مساحات الأسماء الضرورية لمشروعك. يتيح لك هذا الوصول إلى الفئات والأساليب التي يوفرها Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

## الخطوة 1: قم بتحميل المستند

لنبدأ بتحميل المستند الذي يحتوي على الجدول الذي تريد تقسيمه. تأكد من تحديد المسار الصحيح للمستند الخاص بك.

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Tables.docx");
```

## الخطوة 2: تحديد الجدول المراد تقسيمه

بعد ذلك، قم بتحديد واسترجاع الجدول الذي ترغب في تقسيمه. في هذا المثال، سنستهدف الجدول الأول في المستند.

```csharp
Table firstTable = (Table)doc.GetChild(NodeType.Table, 0, true);
```

## الخطوة 3: حدد الصف المراد تقسيمه

حدد الصف الذي تريد تقسيم الجدول فيه. هنا، نقوم بتقسيم الجدول في الصف الثالث (ضمنا).

```csharp
Row row = firstTable.Rows[2];
```

## الخطوة 4: إنشاء حاوية جدول جديدة

قم بإنشاء حاوية جدول جديدة لاستيعاب الصفوف التي سيتم نقلها من الجدول الأصلي.

```csharp
Table table = (Table)firstTable.Clone(false);
```

## الخطوة 5: أدخل حاوية الجدول الجديدة

قم بإدراج حاوية الجدول الجديد مباشرة بعد الجدول الأصلي في المستند.

```csharp
firstTable.ParentNode.InsertAfter(table, firstTable);
```

## الخطوة 6: إضافة فقرة المخزن المؤقت

أضف فقرة عازلة بين الجدولين للتأكد من بقائهما منفصلين.

```csharp
firstTable.ParentNode.InsertAfter(new Paragraph(doc), firstTable);
```

## الخطوة 7: نقل الصفوف إلى الجدول الجديد

نقل الصفوف من الجدول الأصلي إلى حاوية الجدول الجديد. تستمر هذه الحلقة حتى يتم نقل الصف المحدد (ضمناً).

```csharp
Row currentRow;
do
{
    currentRow = firstTable.LastRow;
    table.PrependChild(currentRow);
} while (currentRow != row);
```

## الخطوة 8: احفظ المستند

وأخيرًا، احفظ المستند المعدل مع تقسيم الجداول.

```csharp
doc.Save(dataDir + "WorkingWithTables.SplitTable.docx");
```
