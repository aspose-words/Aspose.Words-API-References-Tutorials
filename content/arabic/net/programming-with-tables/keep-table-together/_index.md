## استيراد مساحات الأسماء

أول الأشياء أولاً، فلنستورد مساحات الأسماء الضرورية. سيتيح لنا هذا الوصول إلى الفئات والأساليب التي نحتاجها من Aspose.Words لـ .NET.

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

دعونا نقسم العملية إلى خطوات سهلة وسهلة الهضم. سنبدأ بتحميل المستند الخاص بنا وننتهي بحفظ المستند المحدث حيث يظل الجدول معًا.

## الخطوة 1: قم بتحميل المستند

 للعمل مع مستند Word، نحتاج أولاً إلى تحميله. سوف نستخدم`Document` فئة لهذا.

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Table spanning two pages.docx");
```

## الخطوة 2: الوصول إلى الجدول

بعد ذلك، نحتاج إلى الحصول على الطاولة التي نريد الاحتفاظ بها معًا. سنفترض أنه الجدول الأول في المستند.

```csharp
Table table = (Table) doc.GetChild(NodeType.Table, 0, true);
```

## الخطوة 3: قم بتعيين KeepWithNext للفقرات

 لمنع الجدول من التكسر عبر الصفحات، نحتاج إلى تعيين`KeepWithNext` خاصية لكل فقرة في الجدول، باستثناء الفقرات الأخيرة في الصف الأخير.

```csharp
foreach (Cell cell in table.GetChildNodes(NodeType.Cell, true))
{
    cell.EnsureMinimum();
    foreach (Paragraph para in cell.Paragraphs)
    {
        if (!(cell.ParentRow.IsLastRow && para.IsEndOfCell))
            para.ParagraphFormat.KeepWithNext = true;
    }
}
```

## الخطوة 4: احفظ المستند

وأخيرا، نقوم بحفظ الوثيقة المحدثة. سيؤدي هذا إلى تطبيق التغييرات التي أجريناها والتأكد من بقاء الجدول معًا في صفحة واحدة.

```csharp
doc.Save(dataDir + "WorkingWithTables.KeepTableTogether.docx");
```
