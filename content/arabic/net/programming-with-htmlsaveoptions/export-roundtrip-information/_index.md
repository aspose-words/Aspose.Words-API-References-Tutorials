## استيراد مساحات الأسماء

أول الأشياء أولاً، نحتاج إلى استيراد مساحات الأسماء اللازمة للبدء في استخدام Aspose.Words لـ .NET.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

الآن، دعونا نقسم العملية إلى خطوات يمكن التحكم فيها. ستكون كل خطوة مصحوبة بتفسيرات مفصلة لضمان عدم تفويت أي شيء.

## الخطوة 1: قم بإعداد دليل المستندات الخاص بك

أولاً، تحتاج إلى إعداد المسار إلى دليل المستندات الخاص بك. هذا هو المكان الذي يتم فيه تخزين مستند Word الخاص بك والمكان الذي سيتم فيه حفظ ملف HTML.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## الخطوة 2: قم بتحميل مستند Word

بعد ذلك، قم بتحميل مستند Word الذي تريد تحويله. في هذا البرنامج التعليمي، سنستخدم مستندًا باسم "Rendering.docx".

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## الخطوة 3: تكوين خيارات حفظ HTML

الآن، هذا هو المكان الذي يحدث فيه السحر. نحتاج إلى إعداد خيارات حفظ HTML، وتحديدًا تمكين خاصية ExportRoundtripInformation. وهذا يضمن الحفاظ على كافة معلومات رحلة الذهاب والإياب أثناء التحويل.

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions { ExportRoundtripInformation = true };
```

## الخطوة 4: احفظ المستند بتنسيق HTML

وأخيرًا، احفظ المستند كملف HTML باستخدام خيارات الحفظ التي تم تكوينها. تضمن هذه الخطوة احتفاظ المستند بجميع تنسيقاته وبياناته عند تحويله إلى HTML وإعادته إلى Word.

```csharp
doc.Save(dataDir + "WorkingWithHtmlSaveOptions.ExportRoundtripInformation.html", saveOptions);
```
