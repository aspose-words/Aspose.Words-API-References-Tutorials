## الخطوة 1: إعداد دليل المستندات

أول الأشياء أولاً، فلنقم بإعداد المسار إلى دليل المستندات الخاص بنا. هذا هو المكان الذي يوجد فيه ملف Word الخاص بك.

### الخطوة 1.1: تحديد دليل البيانات

في مشروع C# الخاص بك، حدد المسار إلى دليل المستندات الخاص بك. استبدل "دليل المستندات الخاص بك" بالمسار الفعلي.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

### الخطوة 1.2: قم بتحميل المستند

الآن، لنقم بتحميل المستند في الكود الخاص بنا. هذا هو المكان الذي يلعب فيه Aspose.Words for .NET دوره.

```csharp
Document doc = new Document(dataDir + "Properties.docx");
```

## الخطوة 2: إزالة خصائص المستند المخصصة

بعد تحميل المستند، حان الوقت لإزالة تلك الخصائص المخصصة المزعجة. وإليك كيفية القيام بذلك خطوة بخطوة.

### الخطوة 2.1: الوصول إلى خصائص المستند المخصصة

أولاً، قم بالوصول إلى خصائص المستند المخصصة للمستند الذي تم تحميله.

```csharp
var customProperties = doc.CustomDocumentProperties;
```

### الخطوة 2.2: إزالة الخاصية المحددة

بعد ذلك، قم بإزالة الخاصية المخصصة المحددة حسب اسمها. في هذا المثال، نقوم بإزالة "التاريخ المعتمد".

```csharp
customProperties.Remove("Authorized Date");
```

## الخطوة 3: حفظ المستند

بعد إزالة الخاصية المخصصة، الخطوة الأخيرة هي حفظ المستند. وهذا يضمن تطبيق التغييرات الخاصة بك.

### الخطوة 3.1: تحديد مسار الحفظ

حدد المكان الذي تريد حفظ المستند المعدل فيه.

```csharp
string savePath = dataDir + "ModifiedProperties.docx";
```

### الخطوة 3.2: احفظ المستند

وأخيرا، احفظ المستند مع التغييرات.

```csharp
doc.Save(savePath);
```
