## استيراد مساحات الأسماء

قبل كتابة التعليمات البرمجية الخاصة بك، تحتاج إلى استيراد مساحات الأسماء الضرورية. أضف الأسطر التالية إلى أعلى ملف C# الخاص بك:

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

## الخطوة 1: إعداد مشروعك

أولاً، ستحتاج إلى إعداد مشروعك في Visual Studio (أو IDE المفضل لديك). 

1. إنشاء مشروع C# جديد: افتح Visual Studio وقم بإنشاء مشروع تطبيق وحدة تحكم C# جديد.
2. تثبيت Aspose.Words لـ .NET: استخدم NuGet Package Manager لتثبيت Aspose.Words لـ .NET. انقر بزر الماوس الأيمن على مشروعك في Solution Explorer، وحدد "إدارة حزم NuGet"، وابحث عن "Aspose.Words"، وقم بتثبيته.

## الخطوة 2: قم بتحميل المستند الخاص بك

بعد ذلك، ستقوم بتحميل مستند Word الذي يحتوي على أحرف التحكم القديمة.

1. تحديد مسار المستند: قم بتعيين المسار إلى دليل المستند الخاص بك.
   
   ```csharp
   string dataDir = "YOUR DOCUMENT DIRECTORY";
   ```

2.  قم بتحميل المستند: استخدم`Document` فئة لتحميل المستند الخاص بك.

   ```csharp
   Document doc = new Document(dataDir + "Legacy control character.doc");
   ```

## الخطوة 3: تكوين خيارات الحفظ

الآن، دعونا نقوم بتكوين خيارات الحفظ للحفاظ على أحرف التحكم القديمة سليمة.

1.  إنشاء خيارات الحفظ: تهيئة مثيل لـ`OoxmlSaveOptions` وتعيين`KeepLegacyControlChars`الملكية ل`true`.

   ```csharp
   OoxmlSaveOptions saveOptions = new OoxmlSaveOptions(SaveFormat.FlatOpc)
   {
       KeepLegacyControlChars = true
   };
   ```

## الخطوة 4: احفظ المستند

وأخيرًا، احفظ المستند باستخدام خيارات الحفظ التي تم تكوينها.

1.  احفظ المستند: استخدم`Save` طريقة`Document` class لحفظ المستند بخيارات الحفظ المحددة.

   ```csharp
   doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.KeepLegacyControlChars.docx", saveOptions);
   ```
