## مقدمة

يمكن أن يكون تقليل حجم ملفات PDF أمرًا ضروريًا للتخزين الفعال والمشاركة السريعة. إحدى الطرق الفعالة للقيام بذلك هي تعطيل الخطوط المضمنة، خاصة عندما تكون الخطوط القياسية متوفرة بالفعل في معظم الأنظمة. في هذا البرنامج التعليمي، سنستكشف كيفية تقليل حجم ملف PDF عن طريق تعطيل الخطوط المضمنة باستخدام Aspose.Words for .NET. سنتعرف على كل خطوة للتأكد من أنه يمكنك تنفيذ ذلك بسهولة في مشاريعك الخاصة.

## المتطلبات الأساسية

قبل الغوص في الكود، تأكد من أن لديك ما يلي:

-  Aspose.Words for .NET: إذا لم تكن قد قمت بذلك بالفعل، فقم بتنزيله وتثبيته من[رابط التحميل](https://releases.aspose.com/words/net/).
- بيئة تطوير .NET: يعد Visual Studio خيارًا شائعًا.
- نموذج مستند Word: جهز ملف DOCX الذي تريد تحويله إلى PDF.

## استيراد مساحات الأسماء

للبدء، تأكد من استيراد مساحات الأسماء الضرورية إلى مشروعك. يتيح لك هذا الوصول إلى الفئات والأساليب المطلوبة لمهمتنا.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

دعونا نقسم العملية إلى خطوات بسيطة يمكن التحكم فيها. سترشدك كل خطوة خلال المهمة، مما يضمن فهمك لما يحدث في كل نقطة.

## الخطوة 1: تهيئة المستند الخاص بك

أولاً، نحتاج إلى تحميل مستند Word الذي تريد تحويله إلى PDF. هذا هو المكان الذي تبدأ رحلتك.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Rendering.docx");
```

 هنا،`dataDir` هو عنصر نائب للدليل الذي يوجد به المستند الخاص بك. يستبدل`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي

## الخطوة 2: تكوين خيارات حفظ PDF

بعد ذلك، سنقوم بإعداد خيارات حفظ PDF. هذا هو المكان الذي نحدد فيه أننا لا نريد تضمين خطوط Windows القياسية.

```csharp
// سيتم حفظ ملف PDF الناتج دون تضمين خطوط Windows القياسية.
PdfSaveOptions saveOptions = new PdfSaveOptions
{
    FontEmbeddingMode = PdfFontEmbeddingMode.EmbedNone
};
```

 عن طريق الإعداد`FontEmbeddingMode` ل`EmbedNone`، فإننا نطلب من Aspose.Words عدم تضمين هذه الخطوط في ملف PDF، مما يقلل من حجم الملف.

## الخطوة 3: احفظ المستند بصيغة PDF

أخيرًا، نقوم بحفظ المستند كملف PDF باستخدام خيارات الحفظ التي تم تكوينها. هذه هي لحظة الحقيقة حيث يتحول ملف DOCX الخاص بك إلى ملف PDF مضغوط.

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.DisableEmbedWindowsFonts.pdf", saveOptions);
```

 يستبدل`"YOUR DOCUMENT DIRECTORY"` مع مسار الدليل الفعلي الخاص بك مرة أخرى. سيتم الآن حفظ ملف PDF الناتج في الدليل المحدد بدون الخطوط القياسية المضمنة.

## خاتمة
