## Импортировать пространства имен

Прежде чем мы начнем кодировать, нам нужно импортировать необходимые пространства имен. Это похоже на сбор всех наших инструментов перед началом проекта.

```csharp
using Aspose.Words;
using Aspose.Words.Layout;
```

Эти строки кода важны, поскольку они реализуют функциональные возможности Aspose.Words, которые мы будем использовать.

## Шаг 1. Настройка документа

Прежде всего, давайте создадим новый документ Word. Это похоже на закладку фундамента перед постройкой дома.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

Здесь мы определяем каталог, в котором будет сохранен наш документ, создаем новый документ и инициализируем DocumentBuilder. DocumentBuilder — наш основной инструмент для добавления контента в документ.

## Шаг 2. Настройка форматирования абзацев

Далее нам нужно настроить параметры форматирования абзаца. Думайте об этом как о настройке вашего рабочего пространства, чтобы все идеально подходило.

```csharp
ParagraphFormat paragraphFormat = builder.ParagraphFormat;
paragraphFormat.AddSpaceBetweenFarEastAndAlpha = true;
paragraphFormat.AddSpaceBetweenFarEastAndDigit = true;
```

 Установив`AddSpaceBetweenFarEastAndAlpha` и`AddSpaceBetweenFarEastAndDigit` к`true`, мы сообщаем Aspose.Words автоматически регулировать расстояние между азиатскими символами и латинскими буквами или цифрами.

## Шаг 3. Добавление текста в документ

Теперь, когда форматирование установлено, давайте добавим текст, чтобы увидеть эти настройки в действии.

```csharp
builder.Writeln("Automatically adjust space between Asian and Latin text");
builder.Writeln("Automatically adjust space between Asian text and numbers");
```

Здесь мы добавляем в документ две строки текста. Первая строка содержит как азиатские символы, так и латинский текст, а вторая строка — азиатские символы и цифры. Это поможет нам четко увидеть корректировку интервалов.

## Шаг 4: Сохранение документа

Наконец, нам нужно сохранить наш документ. Это все равно, что внести последние штрихи в свой проект и нажать кнопку «Сохранить».

```csharp
doc.Save(dataDir + "DocumentFormatting.SpaceBetweenAsianAndLatinText.docx");
```

С помощью этой строки кода мы сохраняем наш документ в указанном каталоге с описательным именем. И вуаля! Ваш документ готов с идеальной регулировкой интервалов между азиатским и латинским текстом.
