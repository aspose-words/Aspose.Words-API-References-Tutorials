## Импортировать пространства имен

Прежде всего, давайте импортируем необходимые пространства имен. Этот шаг имеет решающее значение для обеспечения бесперебойной работы нашего кода.

Давайте разобьем процесс на небольшие этапы, чтобы сделать его проще простого. Готовый? Давайте погрузимся!

## Шаг 1. Создайте новый документ

Для начала нам нужно создать новый экземпляр документа. Этот документ станет нашей площадкой для применения типизированного доступа.

```csharp
Document doc = new Document();
```

## Шаг 2. Доступ к первому разделу

Каждый документ структурирован по разделам. Нам нужен доступ к первому разделу, чтобы глубже погрузиться в его элементы.

```csharp
Section section = doc.FirstSection;
```

## Шаг 3: Получите тело раздела

В теле раздела находится содержимое. Давайте возьмемся за это.

```csharp
Body body = section.Body;
```

## Шаг 4. Доступ к коллекции таблиц

Теперь давайте быстро получим доступ ко всем таблицам внутри тела. Именно здесь проявляется типизированный доступ, предоставляющий простой способ доступа к нашим таблицам.

```csharp
TableCollection tables = body.Tables;
```

## Шаг 5. Перебор таблиц

У нас есть таблицы, но что, если мы захотим ими манипулировать? Итерация — это ключ. Давайте пройдемся по каждой таблице.

```csharp
foreach (Table table in tables)
{
    // Здесь мы будем манипулировать строками
}
```

## Шаг 6: Удалите первую строку

В каждой таблице давайте быстро получим доступ к первой строке и удалим ее. Именно здесь типизированный доступ упрощает нашу жизнь.

```csharp
table.FirstRow?.Remove();
```

## Шаг 7: удалите последнюю строку

Аналогичным образом мы можем получить доступ к последней строке и удалить ее. На этом наши основные манипуляции закончены.

```csharp
table.LastRow?.Remove();
```
