## Импортировать пространства имен

Для начала вам необходимо импортировать необходимые пространства имен в ваш проект C#:

```csharp
using System;
using System.Collections.Generic;
using Aspose.Words;

```

Давайте разобьем процесс настройки обратного вызова с предупреждением на выполнимые шаги.

## Шаг 1. Установите каталог документов

Сначала вам нужно указать путь к каталогу ваших документов. Здесь хранится ваш документ Word.

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## Шаг 2. Настройте параметры загрузки с обратным вызовом с предупреждением

 Далее настройте параметры загрузки документа. Это предполагает создание`LoadOptions` объект и установка его`WarningCallback` свойство.

```csharp
LoadOptions loadOptions = new LoadOptions
{
    WarningCallback = new DocumentLoadingWarningCallback()
};
```

## Шаг 3. Загрузите документ с помощью функции обратного вызова

 Теперь загрузите документ, используя`LoadOptions` объект, настроенный с обратным вызовом предупреждения.

```csharp
Document doc = new Document(dataDir + "Document.docx", loadOptions);
```

## Шаг 4. Реализация класса обратного вызова с предупреждением

 Создайте класс, реализующий`IWarningCallback` интерфейс. Этот класс будет определять, как обрабатываются предупреждения во время обработки документа.

```csharp
private class DocumentLoadingWarningCallback : IWarningCallback
{
    public void Warning(WarningInfo info)
    {
        Console.WriteLine($"Warning: {info.WarningType}");
        Console.WriteLine($"\tSource: {info.Source}");
        Console.WriteLine($"\tDescription: {info.Description}");
        mWarnings.Add(info);
    }

    public List<WarningInfo> GetWarnings()
    {
        return mWarnings;
    }

    private readonly List<WarningInfo> mWarnings = new List<WarningInfo>();
}
```
