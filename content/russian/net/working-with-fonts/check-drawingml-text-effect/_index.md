## Введение

Добро пожаловать на еще одно подробное руководство по работе с Aspose.Words для .NET! Сегодня мы окунемся в увлекательный мир текстовых эффектов DrawingML. Если вы хотите улучшить свои документы Word с помощью теней, отражений или 3D-эффектов, это руководство покажет вам, как проверить наличие этих текстовых эффектов в ваших документах с помощью Aspose.Words для .NET. Давайте начнем!

## Предварительные условия

Прежде чем мы перейдем к руководству, вам необходимо выполнить несколько предварительных условий:

-  Библиотека Aspose.Words for .NET: убедитесь, что у вас установлена библиотека Aspose.Words for .NET. Вы можете скачать его с сайта[Страница релизов Aspose](https://releases.aspose.com/words/net/).
- Среда разработки: у вас должна быть настроена среда разработки, например Visual Studio.
- Базовые знания C#: некоторое знакомство с программированием на C# будет полезно.

## Импортировать пространства имен

Во-первых, вам необходимо импортировать необходимые пространства имен. Эти пространства имен предоставят вам доступ к классам и методам, необходимым для управления документами Word и проверки текстовых эффектов DrawingML.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Drawing;
```

## Пошаговое руководство по проверке текстовых эффектов DrawingML

Теперь давайте разобьем процесс на несколько этапов, чтобы было легче следовать ему.

## Шаг 1. Загрузите документ

Первый шаг — загрузить документ Word, который вы хотите проверить на наличие текстовых эффектов DrawingML. 

```csharp
// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "DrawingML text effects.docx");
```

Этот фрагмент кода загружает документ с именем «DrawingML text effect.docx» из указанного вами каталога.

## Шаг 2. Доступ к коллекции пробегов

Далее нам нужно получить доступ к коллекции прогонов в первом пункте документа. Прогоны — это части текста с одинаковым форматированием.

```csharp
RunCollection runs = doc.FirstSection.Body.FirstParagraph.Runs;
```

Эта строка кода извлекает строки из первого абзаца первого раздела документа.

## Шаг 3. Получите шрифт первого запуска

Теперь мы получим свойства шрифта первого запуска в коллекции запусков. Это позволяет нам проверять различные текстовые эффекты DrawingML, примененные к тексту.

```csharp
Font runFont = runs[0].Font;
```

## Шаг 4. Проверьте наличие текстовых эффектов DrawingML

Наконец, мы можем проверить наличие различных текстовых эффектов DrawingML, таких как тень, 3D-эффект, отражение, контур и заливка.

```csharp
Console.WriteLine(runFont.HasDmlEffect(TextDmlEffect.Shadow));
Console.WriteLine(runFont.HasDmlEffect(TextDmlEffect.Effect3D));
Console.WriteLine(runFont.HasDmlEffect(TextDmlEffect.Reflection));
Console.WriteLine(runFont.HasDmlEffect(TextDmlEffect.Outline));
Console.WriteLine(runFont.HasDmlEffect(TextDmlEffect.Fill));
```

 Эти строки кода будут распечатаны`true` или`false` в зависимости от того, применяется ли каждый конкретный текстовый эффект DrawingML к шрифту выполнения.
