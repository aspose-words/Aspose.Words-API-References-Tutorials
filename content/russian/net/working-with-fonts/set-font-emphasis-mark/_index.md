## Импортировать пространства имен

Для работы с Aspose.Words for .NET вам необходимо импортировать необходимые пространства имен. Добавьте их в начало файла кода:

```csharp
using Aspose.Words;
using Aspose.Words.Fonts;
```

Теперь давайте разобьем процесс на простые шаги. Внимательно следуйте каждому шагу, чтобы установить знаки выделения шрифта в документе Word.

## Шаг 1. Инициализируйте документ и DocumentBuilder

Прежде всего, вам необходимо инициализировать новый документ и DocumentBuilder. Класс DocumentBuilder предоставляет методы для вставки текста и других элементов в документ.

```csharp
// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Инициализировать новый документ
Document document = new Document();

// Инициализируйте DocumentBuilder с документом
DocumentBuilder builder = new DocumentBuilder(document);
```

## Шаг 2. Установите знак подчеркивания шрифта

Когда DocumentBuilder готов, вы можете установить знак выделения шрифта. В этом примере мы будем использовать знак подчеркивания «UnderSolidCircle».

```csharp
// Установите знак выделения шрифта
builder.Font.EmphasisMark = EmphasisMark.UnderSolidCircle;

// Напишите текст со знаком ударения
builder.Write("Emphasis text");
builder.Writeln();
```

## Шаг 3. Очистите форматирование и добавьте обычный текст

После установки знака выделения вы можете добавить обычный текст без выделения. Для этого вам необходимо очистить форматирование.

```csharp
// Очистить форматирование шрифта
builder.Font.ClearFormatting();

// Пишите обычный текст
builder.Write("Simple text");
```

## Шаг 4. Сохраните документ

После того, как вы добавили весь необходимый текст и форматирование, последним шагом будет сохранение документа. Укажите путь и имя файла, в котором вы хотите сохранить документ.

```csharp
// Сохраните документ
document.Save(dataDir + "WorkingWithFonts.SetFontEmphasisMark.docx");
```
