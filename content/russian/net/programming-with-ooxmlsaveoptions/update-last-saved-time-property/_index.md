В этом руководстве мы рассмотрим предоставленный исходный код C# для обновления свойства времени последнего сохранения при сохранении документа с помощью Aspose.Words для .NET. Эта функция позволяет автоматически обновлять свойство времени последнего сохранения созданного документа.

## Шаг 1. Настройка среды

Прежде чем начать, убедитесь, что вы настроили свою среду разработки с помощью Aspose.Words для .NET. Убедитесь, что вы добавили необходимые ссылки и импортировали соответствующие пространства имен.

## Шаг 2: Загрузка документа

```csharp
// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENTS DIRECTORY";

Document doc = new Document(dataDir + "Document.docx");
```

 На этом этапе мы загружаем документ, используя`Document` метод и передав путь к файлу DOCX для загрузки.

## Шаг 3. Настройка параметров резервного копирования OOXML

```csharp
OoxmlSaveOptions saveOptions = new OoxmlSaveOptions { UpdateLastSavedTimeProperty = true };
```

 На этом этапе мы настраиваем параметры сохранения OOXML, используя команду`OoxmlSaveOptions` сорт. Мы включаем автоматическое обновление свойства времени последнего сохранения, установив`UpdateLastSavedTimeProperty` к`true`.

## Шаг 4. Сохраните документ с обновленным свойством.

```csharp
doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.UpdateLastSavedTimeProperty.docx", saveOptions);
```

 На этом последнем шаге мы сохраняем документ, используя`Save` метод и передавая путь к выходному файлу с помощью`.docx` расширение вместе с указанными параметрами сохранения.

Теперь вы можете запустить исходный код для автоматического обновления свойства времени последнего сохранения при сохранении документа. Полученный файл будет сохранен в указанном каталоге с именем «WorkingWithOoxmlSaveOptions.UpdateLastSavedTimeProperty.docx».

### Пример исходного кода для обновления свойства времени последнего сохранения с использованием Aspose.Words для .NET 

```csharp

// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENT DIRECTORY"; 

Document doc = new Document(dataDir + "Document.docx");

OoxmlSaveOptions saveOptions = new OoxmlSaveOptions { UpdateLastSavedTimeProperty = true };

doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.UpdateLastSavedTimeProperty.docx", saveOptions);
            
        
```

## Заключение

В этом руководстве мы рассмотрели функцию автоматического обновления свойства времени последнего сохранения при сохранении документа с помощью Aspose.Words для .NET. Включив эту функцию с параметрами сохранения OOXML, вы можете гарантировать, что свойство времени последнего сохранения автоматически обновляется в сгенерированном документе.

Обновление свойства времени последнего сохранения может быть полезно для отслеживания изменений и версий документа. Он также отслеживает дату последнего сохранения документа, что может быть полезно в различных сценариях.

Aspose.Words для .NET упрощает автоматическое обновление свойства «Время последнего резервного копирования», предоставляя гибкие и мощные возможности резервного копирования. Вы можете интегрировать эту функцию в свои проекты, чтобы гарантировать, что созданные документы содержат точную резервную информацию.
