## Введение

Хотите работать с документами Word программно? Aspose.Words for .NET — мощный инструмент, который поможет вам добиться именно этого. Сегодня я расскажу вам, как перечислить свойства документа Word с помощью Aspose.Words для .NET. Независимо от того, являетесь ли вы новичком или имеете некоторый опыт, это руководство разберет все шаг за шагом в разговорной и простой для понимания форме.

## Предварительные условия

Прежде чем мы углубимся в руководство, вам нужно сделать несколько вещей:

-  Aspose.Words для .NET: вы можете[скачай это здесь](https://releases.aspose.com/words/net/).
- Среда разработки: рекомендуется использовать Visual Studio, но вы можете использовать любую среду разработки C#.
- Базовые знания C#. Фундаментальное понимание C# поможет вам в дальнейшем.

Теперь давайте приступим!

## Шаг 1: Настройка вашего проекта

Прежде всего, вам необходимо настроить проект в Visual Studio.

1. Создайте новый проект. Откройте Visual Studio и создайте новый проект консольного приложения.
2. Установите Aspose.Words для .NET: используйте диспетчер пакетов NuGet для установки Aspose.Words для .NET. Щелкните правой кнопкой мыши свой проект в обозревателе решений, выберите «Управление пакетами NuGet» и найдите «Aspose.Words». Установите пакет.

## Шаг 2. Импортируйте пространства имен

Для работы с Aspose.Words вам необходимо импортировать необходимые пространства имен. Добавьте следующее в начало файла Program.cs:

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Properties;
```

## Шаг 3. Загрузите документ

Далее давайте загрузим документ Word, с которым вы хотите работать. В этом примере мы будем использовать документ с именем «Properties.docx», расположенный в каталоге вашего проекта.

1. Определите путь к документу: укажите путь к документу.
2.  Загрузите документ: используйте Aspose.Words`Document` класс для загрузки документа.

Вот код:

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Properties.docx");
```

## Шаг 4. Отображение имени документа

После загрузки документа вы можете отобразить его имя. Aspose.Words предоставляет для этого свойство:

```csharp
Console.WriteLine("1. Document name: {0}", doc.OriginalFileName);
```

## Шаг 5. Перечислите встроенные свойства

Встроенные свойства — это свойства метаданных, предварительно определенные в Microsoft Word. К ним относятся название, автор и многое другое.

1.  Доступ к встроенным свойствам: используйте`BuiltInDocumentProperties` коллекция.
2. Циклическое перебор свойств: перебирайте свойства и отображайте их имена и значения.

Вот код:

```csharp
Console.WriteLine("2. Built-in Properties");

foreach (DocumentProperty prop in doc.BuiltInDocumentProperties)
    Console.WriteLine("{0} : {1}", prop.Name, prop.Value);
```

## Шаг 6. Перечислите пользовательские свойства

Пользовательские свойства — это определяемые пользователем свойства метаданных. Это может быть все, что вы хотите добавить в свой документ.

1.  Доступ к пользовательским свойствам: используйте`CustomDocumentProperties` коллекция.
2. Циклическое перебор свойств: перебирайте свойства и отображайте их имена и значения.

Вот код:

```csharp
Console.WriteLine("3. Custom Properties");

foreach (DocumentProperty prop in doc.CustomDocumentProperties)
    Console.WriteLine("{0} : {1}", prop.Name, prop.Value);
```
