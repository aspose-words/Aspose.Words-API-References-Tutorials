## Импортировать пространства имен

Прежде всего, давайте настроим наш проект и импортируем необходимые пространства имен. Эти пространства имен имеют решающее значение для доступа к функциональности Aspose.Words.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.ReportingServices;
```

## Шаг 1. Создайте новый документ


 Путешествие начинается с создания нового документа и инициализации`DocumentBuilder` .`DocumentBuilder` Класс предоставляет набор методов для легкого управления содержимым документа.

```csharp
// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Создать новый документ
Document doc = new Document();

// Инициализировать DocumentBuilder
DocumentBuilder builder = new DocumentBuilder(doc);
```

### Объяснение

 На этом этапе мы настроили документ и конструктор, который будем использовать для вставки и форматирования нашего контента.`dataDir` Переменная содержит путь, по которому мы сохраним наш окончательный документ.

## Шаг 2. Вставьте диаграмму

 Далее мы добавим в наш документ круговую диаграмму.`InsertChart` метод`DocumentBuilder` class делает это очень простым.

```csharp
// Вставка круговой диаграммы
Shape shape = builder.InsertChart(ChartType.Pie, 432, 252);

// Доступ к объекту диаграммы
Chart chart = shape.Chart;
```

### Объяснение

Здесь мы вставляем круговую диаграмму в наш документ.`InsertChart` Метод требует тип диаграммы, ширину и высоту в качестве параметров. После вставки диаграммы мы получаем доступ к объекту диаграммы для дальнейшего манипулирования им.

## Шаг 3. Настройте серию диаграмм

Теперь мы очистим все существующие серии на диаграмме и добавим собственные серии. Эта серия будет представлять наши точки данных.

```csharp
// Очистить существующую серию диаграмм
chart.Series.Clear();

// Добавить новую серию в диаграмму
ChartSeries series = chart.Series.Add("Aspose Series 1",
    new string[] { "Category 1", "Category 2", "Category 3" },
    new double[] { 2.7, 3.2, 0.8 });
```

### Объяснение

На этом этапе мы убеждаемся, что наша диаграмма пуста, удаляя все ранее существовавшие ряды. Затем мы добавляем новую серию с настраиваемыми категориями и значениями, которые будут отображаться на нашей круговой диаграмме.

## Шаг 4. Установите параметры по умолчанию для меток данных

Метки данных имеют решающее значение для информативности диаграммы. Мы установим параметры для отображения процентов, значения и настроим разделитель.

```csharp
// Доступ к коллекции меток данных
ChartDataLabelCollection labels = series.DataLabels;

// Установите параметры метки данных
labels.ShowPercentage = true;
labels.ShowValue = true;
labels.ShowLeaderLines = false;
labels.Separator = " - ";
```

### Объяснение

 Здесь мы получаем доступ к`DataLabels`свойство нашей серии для настройки внешнего вида и информации, отображаемой на каждой метке данных. Мы решили показать как процент, так и значение, скрыть выносные линии и установить собственный разделитель.

## Шаг 5: Сохраните документ

Наконец, мы сохраним наш документ в указанном каталоге. Этот шаг гарантирует, что все наши изменения будут записаны в файл.

```csharp
// Сохраните документ
doc.Save(dataDir + "WorkingWithCharts.DefaultOptionsForDataLabels.docx");
```

### Объяснение

 На этом последнем шаге мы сохраняем наш документ, используя`Save` метод. Документ будет сохранен в каталоге, указанном`dataDir`с именем «WorkingWithCharts.DefaultOptionsForDataLabels.docx».
