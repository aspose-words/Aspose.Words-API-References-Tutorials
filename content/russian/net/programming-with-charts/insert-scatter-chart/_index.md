## Введение

В этом уроке вы узнаете, как использовать Aspose.Words для .NET для вставки точечной диаграммы в документ Word. Точечные диаграммы — это мощные визуальные инструменты, которые могут эффективно отображать точки данных на основе двух переменных, делая ваши документы более привлекательными и информативными.

## Предварительные условия

Прежде чем мы углубимся в создание точечных диаграмм с помощью Aspose.Words для .NET, убедитесь, что у вас есть следующие предварительные условия:

1.  Установка Aspose.Words для .NET: Загрузите и установите Aspose.Words для .NET с сайта[здесь](https://releases.aspose.com/words/net/).
   
2. Базовые знания C#: знание языка программирования C# и платформы .NET будет полезным.

## Импортировать пространства имен

Для начала вам необходимо импортировать необходимые пространства имен в ваш проект C#:

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.Saving;
```

Теперь давайте разберем процесс вставки точечной диаграммы в документ Word с помощью Aspose.Words для .NET:

## Шаг 1. Инициализируйте документ и DocumentBuilder

 Сначала инициализируйте новый экземпляр`Document` класс и`DocumentBuilder` class, чтобы начать создавать документ.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## Шаг 2. Вставьте точечную диаграмму

 Использовать`InsertChart` метод`DocumentBuilder` класс для вставки точечной диаграммы в документ.

```csharp
Shape shape = builder.InsertChart(ChartType.Scatter, 432, 252);
Chart chart = shape.Chart;
```

## Шаг 3. Добавьте ряд данных на диаграмму

Теперь добавьте ряды данных в точечную диаграмму. В этом примере показано добавление ряда с конкретными точками данных.

```csharp
chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 });
```

## Шаг 4. Сохраните документ

 Наконец, сохраните измененный документ в нужном месте, используя команду`Save` метод`Document` сорт.

```csharp
doc.Save(dataDir + "WorkingWithCharts.InsertScatterChart.docx");
```
