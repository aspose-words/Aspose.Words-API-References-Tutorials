В этом руководстве объясняется, как использовать Aspose.Words для .NET для вставки точечной диаграммы в документ. В предоставленном исходном коде показано, как создать диаграмму, добавить ряд данных и сохранить документ.

## Шаг 1: Настройте проект

Убедитесь, что у вас есть следующие предварительные условия:

- Установлена библиотека Aspose.Words for .NET. Вы можете загрузить его с помощью диспетчера пакетов NuGet для его установки.
- Путь к каталогу документа, в котором будет сохранен выходной документ.

## Шаг 2. Создайте новый документ и вставьте диаграмму

 Создать новый`Document` объект и`DocumentBuilder` для построения документа.

```csharp
// Путь к вашему каталогу документов
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 Далее используйте`InsertChart` метод`DocumentBuilder` чтобы вставить точечную диаграмму в документ.

```csharp
Shape shape = builder.InsertChart(ChartType.Scatter, 432, 252);
Chart chart = shape.Chart;
```

## Шаг 3. Добавьте ряд данных на диаграмму.

Добавьте ряд данных на диаграмму. В этом примере мы добавим два набора координат X и Y.

```csharp
chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 });
```

## Шаг 4: Сохраните документ

 Наконец, сохраните документ в указанный каталог, используя`Save` метод`Document` объект.

```csharp
doc.Save(dataDir + "WorkingWithCharts.InsertScatterChart.docx");
```

На этом реализация вставки точечной диаграммы с использованием Aspose.Words for .NET завершена.

### Пример исходного кода для вставки точечной диаграммы с использованием Aspose.Words для .NET 

```csharp
	//Путь к вашему каталогу документов
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Scatter, 432, 252);
	Chart chart = shape.Chart;
	chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 });
	doc.Save(dataDir + "WorkingWithCharts.InsertScatterChart.docx");
```

### Часто задаваемые вопросы

#### Q1. Что такое точечная диаграмма?
Точечная диаграмма — это тип диаграммы, которая отображает взаимосвязь между двумя числовыми переменными. Он состоит из ряда точек, нанесенных на координатную сетку, с одной переменной, представленной на оси X, и другой переменной, представленной на оси Y. Точечные диаграммы используются для выявления закономерностей, корреляций или тенденций между двумя наборами точек данных.

#### Q2. Могу ли я добавить несколько серий в точечную диаграмму?
Да, вы можете добавить несколько серий в точечную диаграмму, используя Aspose.Words для .NET. Каждая серия представляет собой набор точек данных с соответствующими координатами X и Y. Добавляя несколько рядов, вы можете сравнивать и анализировать различные наборы данных в одной и той же точечной диаграмме, обеспечивая комплексное представление ваших данных.

#### Q3. Могу ли я настроить внешний вид точечной диаграммы?
Да, используя Aspose.Words для .NET, вы можете настроить различные аспекты внешнего вида точечной диаграммы. Вы можете изменить такие свойства, как цвет ряда, форма маркера, метки осей и форматирование области диаграммы. Библиотека предоставляет богатый набор API-интерфейсов для управления визуальными элементами диаграммы и создания индивидуального вида, который соответствует вашим потребностям.

#### Q4. Можно ли сохранить документ со вставленной точечной диаграммой в разных форматах?
Да, Aspose.Words для .NET позволяет сохранять документ со вставленной точечной диаграммой в различные форматы, такие как DOCX, PDF, HTML и другие. Вы можете выбрать желаемый формат вывода в зависимости от ваших требований и использовать`Save` метод`Document` объект для сохранения документа. Вставленная точечная диаграмма будет сохранена в сохраненном документе.

#### Q5. Могу ли я изменить данные и внешний вид точечной диаграммы после ее вставки?
Да, после вставки точечной диаграммы в документ вы можете изменить ее данные и внешний вид с помощью API-интерфейсов, предоставляемых Aspose.Words для .NET. Вы можете обновить данные серии с новыми координатами X и Y, изменить формы и цвета маркеров, настроить свойства оси и применить параметры форматирования для создания динамических и интерактивных диаграмм в ваших документах Word.
