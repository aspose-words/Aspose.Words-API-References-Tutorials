В этом руководстве объясняется, как использовать Aspose.Words для .NET для вставки точечной диаграммы в документ. Предоставленный исходный код демонстрирует, как создать диаграмму, добавить данные серии и сохранить документ.

## Шаг 1. Настройте проект

Убедитесь, что у вас есть следующие предварительные условия:

- Установлена библиотека Aspose.Words для .NET. Вы можете загрузить его, используя для установки диспетчер пакетов NuGet.
- Путь к каталогу документов, в котором будет сохранен выходной документ.

## Шаг 2. Создайте новый документ и вставьте диаграмму.

 Создать новый`Document` объект и`DocumentBuilder` для построения документа.

```csharp
// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 Далее используйте`InsertChart` метод`DocumentBuilder` чтобы вставить точечную диаграмму в документ.

```csharp
Shape shape = builder.InsertChart(ChartType.Scatter, 432, 252);
Chart chart = shape.Chart;
```

## Шаг 3. Добавьте данные серии на диаграмму

Добавьте данные серии на диаграмму. В этом примере мы добавим два набора координат X и Y.

```csharp
chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 });
```

## Шаг 4. Сохраните документ.

 Наконец, сохраните документ в указанном каталоге, используя команду`Save` метод`Document` объект.

```csharp
doc.Save(dataDir + "WorkingWithCharts.InsertScatterChart.docx");
```

На этом реализация вставки точечной диаграммы с помощью Aspose.Words для .NET завершена.

### Пример исходного кода для вставки точечной диаграммы с использованием Aspose.Words для .NET 

```csharp
	//Путь к каталогу ваших документов
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Scatter, 432, 252);
	Chart chart = shape.Chart;
	chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 });
	doc.Save(dataDir + "WorkingWithCharts.InsertScatterChart.docx");
```

### Часто задаваемые вопросы

#### Вопрос 1. Что такое диаграмма рассеяния?
Точечная диаграмма — это тип диаграммы, которая отображает взаимосвязь между двумя числовыми переменными. Он состоит из ряда точек, нанесенных на координатную сетку, причем одна переменная представлена на оси X, а другая переменная представлена на оси Y. Точечные диаграммы используются для выявления закономерностей, корреляций или тенденций между двумя наборами точек данных.

#### В2. Могу ли я добавить несколько серий в точечную диаграмму?
Да, вы можете добавить несколько рядов в точечную диаграмму, используя Aspose.Words для .NET. Каждая серия представляет собой набор точек данных с соответствующими координатами X и Y. Добавляя несколько рядов, вы можете сравнивать и анализировать различные наборы данных в одной диаграмме рассеяния, обеспечивая комплексное представление ваших данных.

#### Вопрос 3. Могу ли я настроить внешний вид точечной диаграммы?
Да, используя Aspose.Words для .NET, вы можете настроить различные аспекты внешнего вида точечной диаграммы. Вы можете изменить такие свойства, как цвет ряда, форму маркера, метки осей и форматирование области диаграммы. Библиотека предоставляет богатый набор API-интерфейсов для управления визуальными элементами диаграммы и создания индивидуального внешнего вида, соответствующего вашим потребностям.

#### Вопрос 4. Могу ли я сохранить документ со вставленной точечной диаграммой в разных форматах?
Да, Aspose.Words for .NET позволяет сохранять документ со вставленной точечной диаграммой в различных форматах, таких как DOCX, PDF, HTML и других. Вы можете выбрать желаемый формат вывода в соответствии с вашими требованиями и использовать`Save` метод`Document` объект для сохранения документа. Вставленная точечная диаграмма сохранится в сохраненном документе.

#### Вопрос 5. Могу ли я изменить данные и внешний вид точечной диаграммы после ее вставки?
Да, после вставки точечной диаграммы в документ вы можете изменить ее данные и внешний вид с помощью API, предоставляемых Aspose.Words для .NET. Вы можете обновить данные серии, добавив новые координаты X и Y, изменить форму и цвет маркеров, настроить свойства осей и применить параметры форматирования для создания динамических и интерактивных диаграмм в документах Word.
