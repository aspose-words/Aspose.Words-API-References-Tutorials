## Введение

Диаграммы — мощный инструмент для визуализации данных. Если вам нужно создавать профессиональные документы с динамическими диаграммами, Aspose.Words for .NET — бесценная библиотека. Эта статья проведет вас через процесс определения свойств оси XY на диаграмме с использованием Aspose.Words для .NET, разбивая каждый шаг, чтобы обеспечить ясность и простоту понимания.

## Предварительные условия

Прежде чем погрузиться в кодирование, необходимо выполнить несколько предварительных условий:

1. Aspose.Words для .NET: убедитесь, что у вас есть библиотека Aspose.Words для .NET. Ты можешь[скачай это здесь](https://releases.aspose.com/words/net/).
2. Среда разработки: вам нужна интегрированная среда разработки (IDE), такая как Visual Studio.
3. .NET Framework: убедитесь, что ваша среда разработки настроена для разработки .NET.
4. Базовые знания C#. В этом руководстве предполагается, что у вас есть базовые знания программирования на C#.

## Импортировать пространства имен

Для начала вам необходимо импортировать необходимые пространства имен в ваш проект. Это гарантирует, что у вас есть доступ ко всем классам и методам, необходимым для создания документов и диаграмм и управления ими.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.Drawing.Charts;
```

Мы разобьем процесс на простые шаги, каждый из которых посвящен определенной части определения свойств оси XY на диаграмме.

## Шаг 1. Инициализируйте документ и DocumentBuilder

 Сначала вам необходимо инициализировать новый документ и`DocumentBuilder` объект.`DocumentBuilder` помогает в вставке содержимого в документ.

```csharp
// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## Шаг 2. Вставьте диаграмму

Далее вы вставите диаграмму в документ. В этом примере мы будем использовать диаграмму областей. При необходимости вы можете настроить размеры диаграммы.

```csharp
// Вставить диаграмму
Shape shape = builder.InsertChart(ChartType.Area, 432, 252);
Chart chart = shape.Chart;
```

## Шаг 3. Очистите серию по умолчанию и добавьте пользовательские данные

По умолчанию диаграмма будет иметь несколько заранее определенных серий. Мы очистим их и добавим наши собственные серии данных.

```csharp
chart.Series.Clear();
chart.Series.Add("Aspose Series 1",
	new DateTime[]
	{
		new DateTime(2002, 01, 01), new DateTime(2002, 06, 01), new DateTime(2002, 07, 01),
		new DateTime(2002, 08, 01), new DateTime(2002, 09, 01)
	},
	new double[] { 640, 320, 280, 120, 150 });
```

## Шаг 4. Определите свойства оси X

Теперь пришло время определить свойства оси X. Сюда входит установка типа категории, настройка пересечения осей, а также настройка делений и меток.

```csharp
ChartAxis xAxis = chart.AxisX;
xAxis.CategoryType = AxisCategoryType.Category;
xAxis.Crosses = AxisCrosses.Custom;
xAxis.CrossesAt = 3; //Измеряется в единицах отображения по оси Y (сотни).
xAxis.ReverseOrder = true;
xAxis.MajorTickMark = AxisTickMark.Cross;
xAxis.MinorTickMark = AxisTickMark.Outside;
xAxis.TickLabelOffset = 200;
```

## Шаг 5. Определите свойства оси Y

Аналогичным образом вы установите свойства для оси Y. Сюда входит настройка положения галочки, основных и второстепенных единиц измерения, единиц отображения и масштабирования.

```csharp
ChartAxis yAxis = chart.AxisY;
yAxis.TickLabelPosition = AxisTickLabelPosition.High;
yAxis.MajorUnit = 100;
yAxis.MinorUnit = 50;
yAxis.DisplayUnit.Unit = AxisBuiltInUnit.Hundreds;
yAxis.Scaling.Minimum = new AxisBound(100);
yAxis.Scaling.Maximum = new AxisBound(700);
```

## Шаг 6: Сохраните документ

Наконец, сохраните документ в указанном вами каталоге. Это создаст документ Word с настроенной диаграммой.

```csharp
doc.Save(dataDir + "WorkingWithCharts.DefineXYAxisProperties.docx");
```
