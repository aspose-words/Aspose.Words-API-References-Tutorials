В этом руководстве объясняется, как использовать Aspose.Words для .NET для настройки отдельных серий диаграмм на диаграмме. Предоставленный исходный код демонстрирует, как создать диаграмму, получить доступ к определенным рядам и изменить их свойства.

## Шаг 1. Настройте проект

Убедитесь, что у вас есть следующие предварительные условия:

- Установлена библиотека Aspose.Words для .NET. Вы можете загрузить его, используя для установки диспетчер пакетов NuGet.
- Путь к каталогу документов, в котором будет сохранен выходной документ.

## Шаг 2. Создайте новый документ и вставьте диаграмму.

 Создать новый`Document` объект и`DocumentBuilder` для построения документа.

```csharp
// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 Далее используйте`InsertChart` метод`DocumentBuilder` чтобы вставить линейную диаграмму в документ.

```csharp
Shape shape = builder.InsertChart(ChartType.Line, 432, 252);
Chart chart = shape.Chart;
```

## Шаг 3. Доступ к сериям диаграмм и их настройка.

 Чтобы изменить одну серию диаграмм, вам необходимо получить доступ к`ChartSeries` объекты диаграммы.

```csharp
ChartSeries series0 = chart.Series[0];
ChartSeries series1 = chart.Series[1];

series0.Name = "Chart Series Name 1";
series1.Name = "Chart Series Name 2";

series0.Smooth = true;
series1.Smooth = true;

series0.InvertIfNegative = true;
series0.Marker.Symbol = MarkerSymbol.Circle;
series0.Marker.Size = 15;

series1.Marker.Symbol = MarkerSymbol.Star;
series1.Marker.Size = 10;
```

## Шаг 4. Сохраните документ.

 Наконец, сохраните документ в указанном каталоге, используя команду`Save` метод`Document` объект.

```csharp
doc.Save(dataDir + "WorkingWithCharts.SingleChartSeries.docx");
```

На этом реализация настройки отдельной серии диаграмм с использованием Aspose.Words для .NET завершена.

### Пример исходного кода для серии одиночных диаграмм с использованием Aspose.Words для .NET 

```csharp
	//Путь к каталогу ваших документов
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Line, 432, 252);
	Chart chart = shape.Chart;
	ChartSeries series0 = chart.Series[0];
	ChartSeries series1 = chart.Series[1];
	series0.Name = "Chart Series Name 1";
	series1.Name = "Chart Series Name 2";
	// Вы также можете указать, должна ли линия, соединяющая точки на диаграмме, сглаживаться с помощью сплайнов Catmull-Rom.
	series0.Smooth = true;
	series1.Smooth = true;
	// Указывает, должен ли родительский элемент по умолчанию инвертировать свои цвета, если значение отрицательное.
	series0.InvertIfNegative = true;
	series0.Marker.Symbol = MarkerSymbol.Circle;
	series0.Marker.Size = 15;
	series1.Marker.Symbol = MarkerSymbol.Star;
	series1.Marker.Size = 10;
	doc.Save(dataDir + "WorkingWithCharts.SingleChartSeries.docx");
```

Aspose.Words для .NET предоставляет мощные функции для управления диаграммами в документах Word. Имея доступ к отдельным сериям диаграмм, вы можете применять определенные изменения, чтобы настроить их внешний вид и поведение. Это позволяет вам изменить имя серии, включить сглаживание линии диаграммы, настроить маркеры для точек данных, инвертировать цвета для отрицательных значений и многое другое, чтобы улучшить визуальное представление вашей диаграммы.

С помощью Aspose.Words для .NET вы можете легко получать доступ к свойствам серий диаграмм и изменять их, что позволяет создавать визуально привлекательные и информативные диаграммы в документах Word.

### Часто задаваемые вопросы

#### Вопрос 1. Могу ли я настроить несколько серий диаграмм на диаграмме?
 Да, вы можете настроить несколько серий диаграмм в диаграмме, используя Aspose.Words для .NET. Получив доступ к`ChartSeries`объектов на диаграмме вы можете выбирать и изменять несколько серий на основе их индексов или определенных критериев. Используйте цикл или отдельные назначения, чтобы изменить нужные свойства для каждой серии диаграмм. Таким образом, вы можете применять различные настройки к нескольким рядам на одной диаграмме.

#### В2. Как изменить название серии диаграмм?
 Чтобы изменить имя серии диаграмм на диаграмме с помощью Aspose.Words for .NET, вам необходимо получить доступ к`Name` собственность`ChartSeries` объект и присвойте ему желаемое имя. Имя серии обычно отображается в легенде диаграммы или метках данных, обеспечивая описательную метку для серии. Изменяя имя серии, вы можете предоставить осмысленные имена, отражающие данные, представленные каждой серией.

#### Вопрос 3. Что такое сглаживание рядов диаграмм?
Сглаживание серий диаграмм — это метод визуального улучшения, позволяющий создать плавную линию, соединяющую точки на диаграмме. Он применяет алгоритм сглаживания, такой как сплайны Катмулла-Рома, для интерполяции между точками данных и создания визуально приятной кривой. Чтобы включить сглаживание рядов на диаграмме с помощью Aspose.Words для .NET, откройте`Smooth` собственность`ChartSeries` объект и установите его в`true`. Сглаживание может быть полезно для отображения тенденций или закономерностей в данных с нерегулярными колебаниями.

#### Вопрос 4. Как настроить маркеры для точек данных в серии диаграмм?
 Чтобы настроить маркеры для точек данных в серии диаграмм с помощью Aspose.Words for .NET, вам необходимо получить доступ к`Marker` собственность`ChartSeries` объект и изменить его свойства, такие как`Symbol` и`Size`. Маркеры — это визуальные индикаторы, размещаемые на диаграмме для обозначения отдельных точек данных. Вы можете выбирать из множества встроенных символов-маркеров и регулировать их размер, чтобы выделить или отличить определенные точки данных в серии.

#### Вопрос 5. Могу ли я инвертировать цвета отрицательных значений в серии диаграмм?
 Да, вы можете инвертировать цвета отрицательных значений в серии диаграмм, используя Aspose.Words для .NET. Установив`InvertIfNegative` собственность`ChartSeries` Возражать`true`, цвета точек данных с отрицательными значениями будут инвертированы, что делает их визуально отличными от положительных значений. Эта функция может быть полезна при сравнении положительных и отрицательных значений в серии диаграмм, обеспечивая четкое различие между ними.
