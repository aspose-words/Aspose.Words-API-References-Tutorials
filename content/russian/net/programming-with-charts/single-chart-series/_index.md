В этом руководстве объясняется, как использовать Aspose.Words для .NET для настройки отдельных рядов диаграмм на диаграмме. В предоставленном исходном коде показано, как создать диаграмму, получить доступ к определенным рядам и изменить их свойства.

## Шаг 1: Настройте проект

Убедитесь, что у вас есть следующие предварительные условия:

- Установлена библиотека Aspose.Words for .NET. Вы можете загрузить его с помощью диспетчера пакетов NuGet для его установки.
- Путь к каталогу документа, в котором будет сохранен выходной документ.

## Шаг 2. Создайте новый документ и вставьте диаграмму

 Создать новый`Document` объект и`DocumentBuilder` для построения документа.

```csharp
// Путь к вашему каталогу документов
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 Далее используйте`InsertChart` метод`DocumentBuilder` чтобы вставить линейную диаграмму в документ.

```csharp
Shape shape = builder.InsertChart(ChartType.Line, 432, 252);
Chart chart = shape.Chart;
```

## Шаг 3. Получите доступ к серии диаграмм и настройте ее

 Чтобы изменить отдельные серии диаграмм, вам необходимо получить доступ к`ChartSeries` объекты диаграммы.

```csharp
ChartSeries series0 = chart.Series[0];
ChartSeries series1 = chart.Series[1];

series0.Name = "Chart Series Name 1";
series1.Name = "Chart Series Name 2";

series0.Smooth = true;
series1.Smooth = true;

series0.InvertIfNegative = true;
series0.Marker.Symbol = MarkerSymbol.Circle;
series0.Marker.Size = 15;

series1.Marker.Symbol = MarkerSymbol.Star;
series1.Marker.Size = 10;
```

## Шаг 4: Сохраните документ

 Наконец, сохраните документ в указанный каталог, используя`Save` метод`Document` объект.

```csharp
doc.Save(dataDir + "WorkingWithCharts.SingleChartSeries.docx");
```

На этом реализация настройки одной серии диаграмм с помощью Aspose.Words for .NET завершена.

### Пример исходного кода для серии одиночных диаграмм с использованием Aspose.Words для .NET 

```csharp
	//Путь к вашему каталогу документов
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Line, 432, 252);
	Chart chart = shape.Chart;
	ChartSeries series0 = chart.Series[0];
	ChartSeries series1 = chart.Series[1];
	series0.Name = "Chart Series Name 1";
	series1.Name = "Chart Series Name 2";
	// Вы также можете указать, будет ли линия, соединяющая точки на графике, сглаживаться с помощью сплайнов Катмулла-Рома.
	series0.Smooth = true;
	series1.Smooth = true;
	// Указывает, должен ли родительский элемент по умолчанию инвертировать свои цвета, если значение отрицательное.
	series0.InvertIfNegative = true;
	series0.Marker.Symbol = MarkerSymbol.Circle;
	series0.Marker.Size = 15;
	series1.Marker.Symbol = MarkerSymbol.Star;
	series1.Marker.Size = 10;
	doc.Save(dataDir + "WorkingWithCharts.SingleChartSeries.docx");
```

## Заключение

В этом руководстве вы узнали, как настроить один ряд диаграмм на диаграмме с помощью Aspose.Words для .NET. Следуя пошаговому руководству и используя предоставленный исходный код, вы можете создать новый документ, вставить линейную диаграмму, получить доступ к определенным сериям диаграмм и изменить их свойства для достижения желаемой настройки.

Aspose.Words для .NET предоставляет мощные функции для работы с диаграммами в документах Word. Получая доступ к отдельным сериям диаграмм, вы можете вносить определенные изменения, чтобы настроить их внешний вид и поведение. Это позволяет вам изменить имя ряда, включить сглаживание линии диаграммы, настроить маркеры для точек данных, инвертировать цвета для отрицательных значений и многое другое, чтобы улучшить визуальное представление вашей диаграммы.

Настройка отдельной серии диаграмм дает вам возможность выделять определенные данные или выделять определенные тенденции на диаграмме. С помощью Aspose.Words для .NET вы можете легко получить доступ к свойствам серии диаграмм и изменить их, что позволит вам создавать визуально привлекательные и информативные диаграммы в ваших документах Word.

### Часто задаваемые вопросы

#### Q1. Можно ли настроить несколько рядов диаграмм на диаграмме?
 Да, вы можете настроить несколько рядов диаграмм на диаграмме, используя Aspose.Words для .NET. Получив доступ к`ChartSeries`объекты на диаграмме, вы можете выбрать и изменить несколько рядов на основе их индексов или определенных критериев. Используйте цикл или отдельные назначения, чтобы изменить нужные свойства для каждой серии диаграммы. Таким образом, вы можете применить разные настройки к нескольким рядам на одной и той же диаграмме.

#### Q2. Как изменить название серии диаграмм?
 Чтобы изменить имя серии диаграмм на диаграмме с помощью Aspose.Words for .NET, вам необходимо получить доступ к`Name` собственность`ChartSeries` объекта и присвойте ему нужное имя. Имя ряда обычно отображается в легенде диаграммы или метках данных, предоставляя описательную метку для ряда. Изменяя имя серии, вы можете предоставить осмысленные имена, отражающие данные, представленные каждой серией.

#### Q3. Что такое сглаживание рядов диаграммы?
Сглаживание рядов диаграмм — это метод визуального улучшения, позволяющий создать плавную линию, соединяющую точки на диаграмме. Он применяет алгоритм сглаживания, такой как сплайны Катмулла-Рома, для интерполяции между точками данных и создания визуально приятной кривой. Чтобы включить сглаживание рядов на диаграмме с помощью Aspose.Words for .NET, откройте`Smooth` собственность`ChartSeries` объект и установить его на`true`. Сглаживание может быть полезно для отображения трендов или закономерностей в данных с нерегулярными колебаниями.

#### Q4. Как настроить маркеры для точек данных в серии диаграмм?
 Чтобы настроить маркеры для точек данных в серии диаграмм с помощью Aspose.Words for .NET, вам необходимо получить доступ к`Marker` собственность`ChartSeries` объекта и изменить его свойства, такие как`Symbol` и`Size`. Маркеры — это визуальные индикаторы, размещаемые на диаграмме для представления отдельных точек данных. Вы можете выбирать из множества встроенных символов маркеров и настраивать их размер, чтобы выделять или различать определенные точки данных в серии.

#### Q5. Можно ли инвертировать цвета для отрицательных значений в серии диаграмм?
 Да, вы можете инвертировать цвета для отрицательных значений в серии диаграмм, используя Aspose.Words для .NET. Установив`InvertIfNegative` собственность`ChartSeries` Возражать`true`, цвета точек данных с отрицательными значениями будут инвертированы, что сделает их визуально отличными от положительных значений. Эта функция может быть полезна при сравнении положительных и отрицательных значений в серии диаграмм, обеспечивая четкое различие между ними.
