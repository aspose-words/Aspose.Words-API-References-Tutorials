## Введение

Создание диаграмм в документах может стать мощным способом визуализации данных. При работе с данными временных рядов добавление значений даты и времени к оси диаграммы имеет решающее значение для ясности. В этом уроке мы покажем вам процесс добавления значений даты и времени на ось диаграммы с помощью Aspose.Words для .NET. Это пошаговое руководство поможет вам настроить среду, написать код и понять каждую часть процесса. Давайте погрузимся!

## Импортировать пространства имен

Для начала убедитесь, что в ваш проект импортированы необходимые пространства имен. Этот шаг имеет решающее значение для доступа к классам и методам Aspose.Words.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.Drawing.Charts;
```

## Шаг 1. Настройте каталог документов

Во-первых, вам необходимо определить каталог, в котором будет сохранен ваш документ. Это важно для организации ваших файлов и обеспечения правильной работы вашего кода.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Шаг 2. Создайте новый документ и DocumentBuilder

 Далее создайте новый экземпляр`Document` класс и`DocumentBuilder` объект. Эти объекты помогут вам создавать документ и манипулировать им.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## Шаг 3. Вставьте диаграмму в документ

 Теперь вставьте диаграмму в документ, используя`DocumentBuilder` объект. В этом примере мы используем гистограмму, но вы можете выбрать и другие типы.

```csharp
Shape shape = builder.InsertChart(ChartType.Column, 432, 252);
Chart chart = shape.Chart;
```

## Шаг 4. Очистите существующую серию

Очистите все существующие серии на диаграмме, чтобы начать с чистого листа. Этот шаг важен для пользовательских данных.

```csharp
chart.Series.Clear();
```

## Шаг 5. Добавьте значения даты и времени в серию

Добавьте значения даты и времени в серию диаграмм. Этот шаг включает в себя создание массивов для дат и соответствующих значений.

```csharp
chart.Series.Add("Aspose Series 1",
    new[]
    {
        new DateTime(2017, 11, 06), new DateTime(2017, 11, 09), new DateTime(2017, 11, 15),
        new DateTime(2017, 11, 21), new DateTime(2017, 11, 25), new DateTime(2017, 11, 29)
    },
    new double[] { 1.2, 0.3, 2.1, 2.9, 4.2, 5.3 });
```

## Шаг 6: Настройте ось X

Установите масштаб и отметки для оси X. Это гарантирует, что ваши даты отображаются правильно и через соответствующие промежутки времени.

```csharp
ChartAxis xAxis = chart.AxisX;
xAxis.Scaling.Minimum = new AxisBound(new DateTime(2017, 11, 05).ToOADate());
xAxis.Scaling.Maximum = new AxisBound(new DateTime(2017, 12, 03).ToOADate());
xAxis.MajorUnit = 7;
xAxis.MinorUnit = 1;
xAxis.MajorTickMark = AxisTickMark.Cross;
xAxis.MinorTickMark = AxisTickMark.Outside;
```

## Шаг 7: Сохраните документ

Наконец, сохраните документ в указанном каталоге. Этот шаг завершает процесс, и теперь ваш документ должен содержать диаграмму со значениями даты и времени на оси X.

```csharp
doc.Save(dataDir + "WorkingWithCharts.DateTimeValuesToAxis.docx");
```

Да, Aspose.Words поддерживает различные типы диаграмм, включая линейные, столбчатые, круговые и другие.

Вы можете настроить внешний вид, открыв свойства диаграммы и задав стили, цвета и многое другое.

### Что делать, если мне нужно динамически обновлять данные диаграммы?

Вы можете обновлять данные диаграммы динамически, программно управляя свойствами рядов и осей в соответствии с вашими требованиями.
