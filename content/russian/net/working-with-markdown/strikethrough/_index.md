В этом примере мы покажем вам, как применить стиль зачеркивания текста с помощью Aspose.Words для .NET. Зачеркнутый текст используется для обозначения того, что текст удален или больше не действителен.

## Шаг 1. Использование генератора документов

Сначала мы воспользуемся генератором документов, чтобы добавить контент в наш документ.

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## Шаг 2. Примените стиль зачеркнутого текста

Мы включим стиль зачеркивания текста, установив`StrikeThrough` собственность`Font` Возражать`true`.

```csharp
builder.Font.StrikeThrough = true;
```

## Шаг 3. Добавьте зачеркнутый текст

 Теперь мы можем добавить зачеркнутый текст, используя генератор документов.`Writeln` метод.

```csharp
builder.Writeln("This text will be StrikeThrough");
```


### Пример исходного кода для зачеркивания текста с помощью Aspose.Words для .NET

```csharp
// Используйте конструктор документов, чтобы добавить содержимое в документ.
DocumentBuilder builder = new DocumentBuilder();

// Сделайте текст зачеркнутым.
builder.Font.StrikeThrough = true;
builder.Writeln("This text will be StrikeThrough");
```

Поздравляем! Теперь вы узнали, как применить стиль зачеркивания текста с помощью Aspose.Words для .NET.

### Часто задаваемые вопросы

#### Вопрос: Как добавить зачеркнутый текст в Aspose.Words?

 О: Чтобы добавить зачеркнутый текст в Aspose.Words, вы можете использовать`Font.StrikeThrough` собственность`Run` объект. Вы можете установить это свойство на`true` чтобы добавить зачеркнутый текст к конкретному тексту. Например, вы можете использовать`run.Font.StrikeThrough=true` чтобы добавить зачеркнутый текст в`Run` объект.

#### Вопрос: Можно ли добавить зачеркивание к нескольким фрагментам текста в одном абзаце?

 О: Да, вы можете добавить зачеркнутый текст к нескольким частям текста в одном абзаце, используя несколько`Run` объекты. Вы можете создать несколько`Run` объекты и установите`Font.StrikeThrough`собственность`true` для каждого объекта добавить зачеркнутый текст к нужным частям текста. Затем вы можете добавить их в абзац, используя`Paragraph.AppendChild(run)` метод.

#### Вопрос: Могу ли я добавить зачеркнутый текст к тексту, который находится в таблице или ячейке в Aspose.Words?

 О: Да, вы можете добавить зачеркнутый текст к тексту, который находится в таблице или ячейке в Aspose.Words. Вы можете перейти к нужной ячейке или абзацу, используя соответствующие методы, а затем применить форматирование текста с помощью зачеркивания.`Font.StrikeThrough` собственность`Run` или`Paragraph` объект.
