В этом примере мы объясним, как использовать функцию отступа в коде с Aspose.Words для .NET. Код с отступом используется для визуального представления блоков кода с определенным форматированием.

## Шаг 1. Использование генератора документов

Сначала мы воспользуемся генератором документов, чтобы добавить контент в наш документ.

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## Шаг 2. Добавьте стиль для кода с отступом

 Мы добавим собственный стиль для кода с отступом, используя`Styles.Add` метод`Document` объект. В этом примере мы создаем стиль IndentedCode для кода с отступом.

```csharp
Style indentedCode = builder.Document.Styles.Add(StyleType.Paragraph, "IndentedCode");
builder.ParagraphFormat.Style = indentedCode;
```

## Шаг 3. Добавьте код с отступом

Теперь мы можем добавить блок кода с отступом, используя собственный стиль «IndentedCode».

```csharp
builder.Writeln("This is an indented code block");
```

### Пример исходного кода для кода с отступом с помощью Aspose.Words для .NET

```csharp
// Используйте конструктор документов, чтобы добавить содержимое в документ.
DocumentBuilder builder = new DocumentBuilder();

Style indentedCode = builder.Document.Styles.Add(StyleType.Paragraph, "IndentedCode");
builder.ParagraphFormat.Style = indentedCode;
builder.Writeln("This is an indented code");
```

Поздравляем! Теперь вы узнали, как использовать функцию отступа в коде с Aspose.Words для .NET.


### Часто задаваемые вопросы

#### Вопрос: Что такое код с отступом в Markdown?

Ответ: Код с отступом в Markdown — это метод форматирования, используемый для отображения кода в документе Markdown. Он состоит из отступов каждой строки кода с помощью пробелов или табуляции.

#### Вопрос: Как использовать код с отступом в Markdown?

О: Чтобы использовать код с отступом в Markdown, сделайте отступ в каждой строке кода пробелами или табуляциями.

#### Вопрос: Каковы преимущества кода с отступами в Markdown?

Ответ: Код с отступами в Markdown улучшает читаемость кода и облегчает его понимание читателями.

#### Вопрос: В чем разница между кодом с отступом и блоками кода в Markdown?

О: Код с отступом используется для небольших фрагментов кода, вставленных в текст, а блоки кода используются для отображения более крупных фрагментов кода в отдельном форматировании.

#### Вопрос: Поддерживается ли код с отступом в Markdown всеми редакторами Markdown?

О: Поддержка кода с отступами в Markdown может различаться в разных редакторах Markdown. Чтобы быть уверенным, проверьте документацию вашего издателя.
