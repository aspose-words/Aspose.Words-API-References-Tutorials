В этом примере мы расскажем вам, как выделить текст полужирным шрифтом с помощью Aspose.Words для .NET. Выделение текста жирным шрифтом делает его более заметным и выделяет его.

## Шаг 1. Использование генератора документов

Сначала мы воспользуемся генератором документов, чтобы добавить контент в наш документ.

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## Шаг 2. Жирный текст

 Мы можем выделить текст жирным шрифтом, установив в конструкторе документов`Font.Bold`собственность`true`.

```csharp
builder.Font.Bold = true;
```

## Шаг 3. Добавьте содержимое в документ

 Теперь мы можем добавлять контент в документ, используя методы построителя документов, такие как`Writeln`, который добавляет строку текста.

```csharp
builder.Writeln("This text will be bold");
```

## Пример исходного кода для жирного текста с использованием Aspose.Words для .NET


```csharp
// Используйте конструктор документов, чтобы добавить содержимое в документ.
DocumentBuilder builder = new DocumentBuilder();

// Сделайте текст жирным.
builder.Font.Bold = true;
builder.Writeln("This text will be Bold");  
```

Поздравляем! Теперь вы узнали, как выделять жирный текст с помощью Aspose.Words для .NET.


### Часто задаваемые вопросы

#### Вопрос: Как сделать текст жирным в Aspose.Words?

 О: Чтобы сделать текст жирным в Aspose.Words, вы можете использовать`Font.Bold` собственность`Run` объект. Вы можете установить это свойство на`true` выделить конкретный текст жирным шрифтом. Например, вы можете использовать`run.Font.Bold=true` чтобы выделить текст внутри`Run` объект.

#### Вопрос: Можно ли выделить жирным шрифтом несколько фрагментов текста в одном абзаце?

О: Да, вы можете выделить несколько фрагментов текста в одном абзаце, используя несколько`Run` объекты. Вы можете создать несколько`Run` объекты и установите`Font.Bold`собственность`true` для каждого объекта выделить нужные части текста жирным шрифтом. Затем вы можете добавить их в абзац, используя`Paragraph.AppendChild(run)` метод.

#### Вопрос: Могу ли я выделить жирным шрифтом текст, находящийся в таблице или ячейке в Aspose.Words?

 О: Да, вы можете выделить жирным шрифтом текст в таблице или ячейке Aspose.Words. Вы можете перейти к нужной ячейке или абзацу, используя соответствующие методы, а затем применить жирное форматирование, используя`Font.Bold` собственность`Run` или`Paragraph` объект.
