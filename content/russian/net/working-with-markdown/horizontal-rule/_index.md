В этом примере мы собираемся показать вам, как использовать функцию горизонтального правила с Aspose.Words для .NET. Горизонтальное правило используется для визуального разделения разделов документа.

## Шаг 1. Использование генератора документов

Сначала мы воспользуемся генератором документов, чтобы добавить контент в наш документ.

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## Шаг 2. Вставка горизонтальной линейки

 Мы можем вставить горизонтальную линейку, используя`InsertHorizontalRule` метод генератора документов.

```csharp
builder. InsertHorizontalRule();
```

## Пример исходного кода для горизонтального правила с помощью Aspose.Words для .NET

```csharp
// Используйте конструктор документов, чтобы добавить содержимое в документ.
DocumentBuilder builder = new DocumentBuilder();

// Вставьте горизонтальное правило.
builder.InsertHorizontalRule();
```

Поздравляем! Теперь вы узнали, как использовать функцию горизонтального правила в Aspose.Words для .NET.


### Часто задаваемые вопросы

#### Вопрос: Как создать горизонтальную линейку в Markdown?

О: Чтобы создать горизонтальную линейку в Markdown, вы можете использовать в пустой строке один из следующих символов: три звездочки (\***), три тире (\---) или три знака подчеркивания (\___).

#### Вопрос: Могу ли я настроить внешний вид горизонтальной линейки в Markdown?

О: В стандартном Markdown нет возможности настроить внешний вид горизонтальных линеек. Однако некоторые продвинутые редакторы и расширения Markdown предлагают дополнительные функции настройки.

#### Вопрос: Поддерживаются ли горизонтальные линейки всеми редакторами Markdown?

О: Да, большинство популярных редакторов Markdown поддерживают горизонтальные линейки. Однако всегда лучше проверить документацию вашего конкретного поставщика, чтобы убедиться, что она поддерживается.

#### Вопрос: Какие еще элементы я могу создать в Markdown?

О: Помимо горизонтальных линеек, в Markdown вы можете создавать заголовки, абзацы, списки, ссылки, изображения, таблицы и многое другое.
