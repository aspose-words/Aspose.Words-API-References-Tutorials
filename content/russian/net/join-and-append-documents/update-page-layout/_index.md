## Импортировать пространства имен

Чтобы начать работу с Aspose.Words для .NET, вам необходимо импортировать необходимые пространства имен в ваш проект C#. Вот как вы можете это сделать:

```csharp
using Aspose.Words;
using Aspose.Words.Layout;
```

Эти пространства имен предоставят вам доступ к классам и методам, которые вам понадобятся для работы с документами Word и управления их макетами.

Теперь, когда мы рассмотрели все необходимые условия, давайте перейдем к самому процессу. Разобьем его на ряд простых шагов:

## Шаг 1. Загрузите документ

Сначала вам нужно загрузить документ Word, с которым вы хотите работать. Это включает в себя указание пути к вашему документу и создание`Document` объект.

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Загрузите документ
Document doc = new Document(dataDir + "input.docx");
```

 Вот замените`"YOUR DOCUMENT DIRECTORY"` с фактическим путем, где ваш`input.docx` файл сохранен.

## Шаг 2. Сохраните документ с исходным макетом.

Прежде чем вносить какие-либо изменения, рекомендуется сохранить документ в формате PDF или любом другом формате, чтобы кэшировать его первоначальный макет.

```csharp
// Сохраните документ в PDF
doc.Save(dataDir + "Document.UpdatePageLayout.1.pdf");
```

Сохранение таким образом гарантирует, что исходный макет будет кэширован и его можно будет использовать в качестве образца для последующих обновлений.

## Шаг 3. Измените документ

Теперь, когда мы кэшировали исходный макет, давайте изменим документ. На этом шаге показано, как изменить размер шрифта документа, ориентацию страницы и поля.

```csharp
// Изменить документ
doc.Styles["Normal"].Font.Size = 6;
doc.Sections[0].PageSetup.Orientation = Aspose.Words.Orientation.Landscape;
doc.Sections[0].PageSetup.Margins = Margins.Mirrored;
```

В этом примере:
- Изменяем размер шрифта стиля «Обычный» на 6 пунктов.
- Ориентацию страницы выставляем «Альбомная».
- Настраиваем поля страницы на Зеркальные.

## Шаг 4. Обновите макет страницы.

После внесения изменений вам необходимо вручную обновить макет страницы, чтобы отразить изменения. Это гарантирует, что кэшированный макет будет перестроен с учетом ваших новых настроек.

```csharp
// Обновите макет страницы
doc.UpdatePageLayout();
```

Этот шаг имеет решающее значение, поскольку без него ваши изменения могут не быть точно отражены в конечном результате.

## Шаг 5. Сохраните измененный документ

Наконец, снова сохраните документ в новый PDF-файл, чтобы увидеть обновленный макет.

```csharp
// Сохраните документ с обновленным макетом.
doc.Save(dataDir + "Document.UpdatePageLayout.2.pdf");
```

Эта последняя операция сохранения зафиксирует внесенные вами изменения и применит обновленный макет к новому PDF-файлу.
