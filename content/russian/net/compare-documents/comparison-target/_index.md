Ниже приведено пошаговое руководство, объясняющее приведенный ниже исходный код C#, в котором используется цель сравнения в функциях текстовых документов Aspose.Words для .NET.

## Шаг 1: Введение

Функция сравнения целевого объекта Aspose.Words for .NET позволяет сравнивать два документа и генерировать новый документ, содержащий изменения, внесенные в целевой документ. Это может быть полезно для отслеживания изменений, внесенных между различными версиями документа.

## Шаг 2. Настройка среды

Прежде чем начать, вам необходимо настроить среду разработки для работы с Aspose.Words for .NET. Убедитесь, что у вас установлена библиотека Aspose.Words и имеется подходящий проект C# для встраивания кода.

## Шаг 3. Добавьте необходимые сборки

Чтобы использовать функцию цели сравнения Aspose.Words for .NET, вы должны добавить необходимые сборки в свой проект. Убедитесь, что в вашем проекте есть правильные ссылки на Aspose.Words.

```csharp
using Aspose.Words;
```

## Шаг 4: Инициализация документа

На этом этапе мы инициализируем два документа для сравнения. Вы должны указать путь к каталогу, в котором находятся ваши документы, а также имя исходного документа.

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// Инициализация документа A для сравнения.
Document docA = new Document(dataDir + "DocumentA.docx");

// Клонируйте документ А, чтобы создать идентичную копию документа Б.
Document docB = docA.Clone();
```

## Шаг 5. Настройка параметров сравнения

На этом этапе мы настроим параметры сравнения, чтобы указать поведение сравнения. Параметры включают в себя возможность игнорировать форматирование, а также цель сравнения, которой является опция «Показать изменения в» в диалоговом окне «Сравнить документы» Microsoft Word.

```csharp
CompareOptions options = new CompareOptions { IgnoreFormatting = true, Target = ComparisonTargetType.New };
```

## Шаг 6: Сравнение документов

Теперь мы сравним документы и сгенерируем результат в новом документе.

```csharp
docA.Compare(docB, "user", DateTime.Now, options);
```

`Compare`Метод сравнивает документ A с документом B и сохраняет изменения в документе A. Для справки вы можете указать имя пользователя и дату сравнения.

### Пример исходного кода для цели сравнения с использованием Aspose.Words для .NET


```csharp
            
Document docA = new Document(MyDir + "Document.docx");
Document docB = docA.Clone();

// Относится к параметру Microsoft Word «Показать изменения» в диалоговом окне «Сравнение документов».
CompareOptions options = new CompareOptions { IgnoreFormatting = true, Target = ComparisonTargetType.New };

docA.Compare(docB, "user", DateTime.Now, options);
            
        
```

## Заключение

В этой статье мы рассмотрели функцию цели сравнения в Aspose.Words для .NET. Эта функция позволяет сравнить два документа и создать новый документ, содержащий внесенные изменения. Вы можете использовать эти знания для отслеживания изменений между различными версиями ваших документов.

### Часто задаваемые вопросы

#### Вопрос: Какова цель использования цели сравнения в Aspose.Words для .NET?

О: Цель сравнения в Aspose.Words для .NET позволяет сравнивать два документа и генерировать новый документ, содержащий изменения, внесенные в целевой документ. Эта функция полезна для отслеживания изменений, внесенных между разными версиями документа, и визуализации различий в отдельном документе.

#### Вопрос: Как использовать цель сравнения в Aspose.Words для .NET?

О: Чтобы использовать объект сравнения в Aspose.Words для .NET, выполните следующие действия:
1. Настройте среду разработки с помощью библиотеки Aspose.Words.
2. Добавьте необходимые сборки в свой проект, ссылаясь на Aspose.Words.
3.  Инициализируйте документы, которые вы хотите сравнить, используя`Document` класс или`DocumentBuilder` класс.
4.  Настройте параметры сравнения, создав`CompareOptions` объекта и установки свойств, таких как`IgnoreFormatting` и`Target` (например,`ComparisonTargetType.New` для цели сравнения).
5.  Использовать`Compare` метод для одного документа, передавая другой документ и`CompareOptions` объект в качестве параметров. Этот метод сравнит документы и сохранит изменения в первом документе.

####  Вопрос: Какова цель`Target` property in the `CompareOptions` class?

 А:`Target` недвижимость в`CompareOptions` Класс позволяет указать цель сравнения, что аналогично параметру «Показать изменения» в диалоговом окне «Сравнить документы» Microsoft Word. Целью можно установить`ComparisonTargetType.New` чтобы показать изменения в новом документе,`ComparisonTargetType.Current` чтобы показать изменения в текущем документе, или`ComparisonTargetType.Formatting` чтобы показать только изменения форматирования.
