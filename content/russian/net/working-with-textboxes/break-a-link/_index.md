Aspose.Words for .NET — это мощная библиотека, которая предлагает различные функции для программной обработки слов с документами Microsoft Word. Одной из его полезных функций является возможность разрывать прямые ссылки в документе Word. В этом руководстве мы рассмотрим исходный код на C#, который демонстрирует, как разорвать прямую ссылку в документе Word с помощью Aspose.Words для .NET.

## Шаг 1. Предварительный просмотр исходного кода C#

Предоставленный исходный код C# посвящен функции «Разорвать ссылку» Aspose.Words для .NET. Он показывает, как разорвать ссылку в форме TextBox внутри документа. В коде представлены различные сценарии разрыва ссылок и даны четкие инструкции, как добиться желаемых результатов.

## Шаг 2. Настройка документа и создание формы TextBox.

 Для начала нам нужно настроить документ и создать фигуру TextBox. Следующий код инициализирует новый экземпляр`Document` класс и создает форму текстового поля:

```csharp
Document doc = new Document();
Shape shape = new Shape(doc, ShapeType.TextBox);
TextBox textBox = shape.TextBox;
```

## Шаг 3. Разрыв ссылки вперед в TextBox

 Чтобы разорвать прямую ссылку в TextBox, мы можем использовать`BreakForwardLink()` метод. Этот метод разрывает ссылку на следующую фигуру в последовательности. Следующий код показывает, как разорвать прямую ссылку:

```csharp
textBox.BreakForwardLink();
```

## Шаг 4. Разорвите прямую ссылку, установив нулевое значение.

 В качестве альтернативы мы можем разорвать прямую ссылку, установив для TextBox`Next`собственность`null`. Это эффективно удаляет связь со следующей фигурой. Следующий код демонстрирует этот подход:

```csharp
textBox. Next = null;
```

## Шаг 5. Разорвите ссылку, ведущую к TextBox.

 В некоторых случаях нам нужно разорвать ссылку, ведущую к форме TextBox. Мы можем добиться этого, вызвав`BreakForwardLink()` метод на`Previous` form, что разрывает ссылку на TextBox. Вот пример того, как разорвать такую ссылку:

```csharp
textBox.Previous?.BreakForwardLink();
```

### Пример исходного кода для разрыва связи с Aspose.Words для .NET

```csharp
Document doc = new Document();
Shape shape = new Shape(doc, ShapeType.TextBox);
TextBox textBox = shape.TextBox;

// Разорвать прямую ссылку.
textBox.BreakForwardLink();

// Разорвите прямую ссылку, установив нулевое значение.
textBox. Next = null;

// Разорвите ссылку, ведущую к этому текстовому полю.
textBox.Previous?.BreakForwardLink();
```
