В этом руководстве мы рассмотрим исходный код C#, предоставленный для функции «Получить диапазон страниц JPEG» с помощью Aspose.Words для .NET. Эта функция позволяет конвертировать определенный диапазон страниц документа в изображения в формате JPEG.

## Шаг 1. Настройка среды

Прежде чем начать, убедитесь, что вы настроили свою среду разработки с помощью Aspose.Words для .NET. Убедитесь, что вы добавили необходимые ссылки и импортировали соответствующие пространства имен.

## Шаг 2: Загрузка документа

```csharp
// Путь к каталогу ваших документов
string dataDir = "YOUR DOCUMENTS DIRECTORY";

Document doc = new Document(dataDir + "Rendering.docx");
```

 На этом этапе мы загружаем документ, используя`Document` метод и передав путь к файлу DOCX для загрузки.

## Шаг 3. Настройте параметры резервного копирования образа

```csharp
ImageSaveOptions options = new ImageSaveOptions(SaveFormat.Jpeg);
options. PageSet = new PageSet(0);
options. ImageBrightness = 0.3f;
options. ImageContrast = 0.7f;
options. HorizontalResolution = 72f;
```

 На этом этапе мы настраиваем параметры резервного копирования изображений. Мы создаем новый`ImageSaveOptions` объект, определяющий желаемый формат сохранения, здесь «Jpeg» для формата JPEG. Мы также устанавливаем диапазон страниц для конвертации с помощью`PageSet`объект. Наконец, мы настраиваем яркость и контрастность изображения с помощью`ImageBrightness`и`ImageContrast` свойства соответственно. Мы также изменяем горизонтальное разрешение с помощью`HorizontalResolution` свойство.

## Шаг 4. Резервное копирование изображений

```csharp
doc.Save(dataDir + "WorkingWithImageSaveOptions.GetJpegPageRange.jpeg", options);
```

 На этом последнем шаге мы сохраняем изображения указанного диапазона страниц в формате JPEG, используя метод`Save` метод и передавая путь к выходному файлу вместе с указанными параметрами сохранения.

Теперь вы можете запустить исходный код для преобразования определенного диапазона страниц вашего документа в изображения JPEG. Полученный файл будет сохранен в указанном каталоге с именем «WorkingWithImageSaveOptions.GetJpegPageRange.jpeg».

### Пример исходного кода для получения диапазона страниц Jpeg с использованием Aspose.Words For .NET

```csharp 
 // Путь к каталогу ваших документов
 string dataDir = "YOUR DOCUMENT DIRECTORY"; 

Document doc = new Document(dataDir + "Rendering.docx");

ImageSaveOptions options = new ImageSaveOptions(SaveFormat.Jpeg);

// Установите для параметра «PageSet» значение «0», чтобы преобразовать только первую страницу документа.
options.PageSet = new PageSet(0);

// Измените яркость и контрастность изображения.
// Оба имеют шкалу 0–1 и по умолчанию равны 0,5.
options.ImageBrightness = 0.3f;
options.ImageContrast = 0.7f;

// Измените горизонтальное разрешение.
// Значение по умолчанию для этих свойств — 96,0 для разрешения 96 точек на дюйм.
options.HorizontalResolution = 72f;

doc.Save(dataDir + "WorkingWithImageSaveOptions.GetJpegPageRange.jpeg", options);
            
        
```
