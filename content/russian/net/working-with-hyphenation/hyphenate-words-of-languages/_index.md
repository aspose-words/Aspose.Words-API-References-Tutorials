## Импортировать пространства имен

Прежде всего, давайте импортируем необходимые пространства имен. Это помогает получить доступ к классам и методам, необходимым для расстановки переносов.

```csharp
using Aspose.Words;
using Aspose.Words.Hyphenation;
```

## Шаг 1. Загрузите документ

 Вам нужно будет указать каталог, в котором находится ваш документ. Заменять`"YOUR DOCUMENT DIRECTORY"` с фактическим путем к вашему документу.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "German text.docx");
```

## Шаг 3. Зарегистрируйте словари расстановки переносов

 Aspose.Words требует наличия словарей расстановки переносов для разных языков. Убедитесь, что у вас есть`.dic`файлы для языков, на которых вы хотите расставить переносы. Зарегистрируйте эти словари, используя`Hyphenation.RegisterDictionary` метод.

```csharp
Hyphenation.RegisterDictionary("en-US", dataDir + "hyph_en_US.dic");
Hyphenation.RegisterDictionary("de-CH", dataDir + "hyph_de_CH.dic");
```

## Шаг 4. Сохраните документ

Наконец, сохраните документ с переносами в нужном формате. Здесь мы сохраняем его в формате PDF.

```csharp
doc.Save(dataDir + "TreatmentByCesure.pdf");
```
