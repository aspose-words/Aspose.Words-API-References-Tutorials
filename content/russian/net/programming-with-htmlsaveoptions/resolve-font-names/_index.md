## Импортировать пространства имен

Прежде чем мы начнем кодирование, убедитесь, что вы импортировали необходимые пространства имен в свой проект. Это крайне важно для доступа к функциям Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

## Шаг 1. Настройка каталога документов

Прежде всего, давайте настроим путь к каталогу ваших документов. Здесь находится ваш документ Word и где вы сохраните результат.

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Объяснение:
 Здесь,`dataDir` содержит путь к каталогу ваших документов. Заменять`"YOUR DOCUMENT DIRECTORY"` с фактическим путем в вашей системе.

## Шаг 2. Загрузка документа Word

Далее нам нужно загрузить документ Word, который мы хотим обработать. В этом документе должны быть шрифты, которые вы хотите разрешить.

```csharp
Document doc = new Document(dataDir + "Missing font.docx");
```

Объяснение:
 Мы создаем`Document` объект и загрузите документ Word с именем «Отсутствует шрифт.docx» из нашего`dataDir`.

## Шаг 3. Настройка параметров сохранения HTML

Теперь давайте настроим параметры сохранения документа в формате HTML. Здесь мы обеспечим правильное разрешение имен шрифтов.

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.Html)
{
    PrettyFormat = true,
    ResolveFontNames = true
};
```

Объяснение:
 Мы создаем экземпляр`HtmlSaveOptions` с`SaveFormat.Html` .`PrettyFormat` опция делает HTML-вывод более читабельным и`ResolveFontNames` гарантирует, что имена шрифтов будут разрешены.

## Шаг 4. Сохранение документа в формате HTML

Наконец, мы сохраняем документ как файл HTML, используя настроенные параметры сохранения.

```csharp
doc.Save(dataDir + "WorkingWithHtmlSaveOptions.ResolveFontNames.html", saveOptions);
```

Объяснение:
 Мы называем`Save` метод на`Document` объект, указав путь вывода и настроенные нами параметры сохранения. Это создаст HTML-файл с разрешенными именами шрифтов.
