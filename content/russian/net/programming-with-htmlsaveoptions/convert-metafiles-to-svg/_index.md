## Импортировать пространства имен

Перво-наперво, давайте импортировать. В вашем проекте C# вам потребуется импортировать необходимые пространства имен. Это крайне важно для доступа к функциям Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

Теперь, когда у нас есть необходимые условия и пространства имен, давайте углубимся в пошаговое руководство по преобразованию метафайлов в SVG.

## Шаг 1. Инициализируйте документ и DocumentBuilder

 Хорошо, давайте начнем с создания нового документа Word и инициализации`DocumentBuilder` объект. Этот конструктор поможет нам добавить контент в наш документ.

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 Здесь мы инициализируем новый документ и построитель документов.`dataDir` Переменная содержит путь к каталогу ваших документов, в котором вы сохраните файлы.

## Шаг 2. Добавьте текст в документ

 Далее давайте добавим текст в наш документ. Мы будем использовать`Write` метод`DocumentBuilder` чтобы вставить текст.

```csharp
builder.Write("Here is an SVG image: ");
```

Эта строка добавляет в документ текст «Вот изображение SVG:». Всегда полезно предоставить некоторый контекст или описание изображения SVG, которое вы собираетесь вставить.

## Шаг 3. Вставьте изображение SVG

 Теперь самое интересное! Мы вставим изображение SVG в наш документ, используя`InsertHtml` метод.

```csharp
builder.InsertHtml(
    @"<svg height='210' width='500'>
    <polygon points='100,10 40,198 190,78 10,78 160,198' 
    style='fill:lime;stroke:purple;stroke-width:5;fill-rule:evenodd;' />
</svg> ");
```

Этот фрагмент вставляет в документ изображение SVG. Код SVG определяет простой многоугольник с указанными точками, цветами и стилями. Не стесняйтесь настраивать код SVG в соответствии с вашими требованиями.

## Шаг 4. Определите HtmlSaveOptions.

 Чтобы гарантировать сохранение наших метафайлов в формате SVG, мы определим`HtmlSaveOptions` и установите`MetafileFormat`собственность`HtmlMetafileFormat.Svg`.

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions
{
    MetafileFormat = HtmlMetafileFormat.Svg
};
```

Это указывает Aspose.Words сохранять любые метафайлы в документе как SVG при экспорте в HTML.

## Шаг 5: Сохраните документ

 Наконец, давайте сохраним наш документ. Мы будем использовать`Save` метод`Document` class, передайте путь к каталогу и сохраните параметры.

```csharp
doc.Save(dataDir + "WorkingWithHtmlSaveOptions.ConvertMetafilesToSvg.html", saveOptions);
```

 Эта строка сохраняет документ в указанный каталог с именем файла`WorkingWithHtmlSaveOptions.ConvertMetafilesToSvg.html` .`saveOptions` убедитесь, что метафайлы преобразованы в SVG.
