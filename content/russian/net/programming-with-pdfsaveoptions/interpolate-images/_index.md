## Импортировать пространства имен

Для начала вам необходимо импортировать необходимые пространства имен в ваш проект C#. Вот как:

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

Эти пространства имен предоставляют вам доступ к функциям Aspose.Words и опциям сохранения для экспорта вашего документа.

## Шаг 1. Настройте путь к документу

Прежде всего, вам необходимо определить путь, по которому хранятся ваши документы. Здесь вы загрузите документ Word и сохраните результат в формате PDF.

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Заменять`"YOUR DOCUMENT DIRECTORY"` с фактическим путем, где находятся ваши файлы. Это помогает Aspose.Words найти исходный документ и указать место, где вы хотите сохранить PDF-файл.

## Шаг 2. Загрузите документ Word

 Теперь, когда вы установили путь к документу, загрузите документ Word в экземпляр`Document` сорт.

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

 Здесь,`"Rendering.docx"` — это имя вашего файла Word. Убедитесь, что этот файл существует в указанном каталоге.

## Шаг 3. Настройте параметры сохранения PDF-файла

Чтобы обеспечить интерполяцию изображений, необходимо настроить`PdfSaveOptions`. Этот класс позволяет вам установить различные параметры сохранения вашего документа в формате PDF. В частности, вы хотите включить интерполяцию изображений.

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions 
{ 
	InterpolateImages = true
};
```

`InterpolateImages` свойство установлено на`true` чтобы гарантировать, что изображения в вашем PDF-файле интерполируются, улучшая их качество.

## Шаг 4. Сохраните документ в формате PDF.

 Когда параметры настроены, пришло время сохранить документ в формате PDF. Использовать`Save` метод`Document` class, указав путь и параметры сохранения.

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.InterpolateImages.pdf", saveOptions);
```

 Здесь,`"WorkingWithPdfSaveOptions.InterpolateImages.pdf"` — это имя, которое вы хотите для выходного PDF-файла. Этот файл будет содержать ваши изображения улучшенного качества за счет интерполяции.
