```csharp
using Aspose.Words;

```

บรรทัดโค้ดเหล่านี้มีความสำคัญเนื่องจากจะนำฟังก์ชันการทำงานของ Aspose.Words ที่เราจะใช้เข้ามา

## ขั้นตอนที่ 1: การตั้งค่าเอกสารของคุณ

ก่อนอื่น มาตั้งค่าเอกสาร Word ใหม่กันก่อน ก็เหมือนกับการวางรากฐานก่อนสร้างบ้าน

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

ที่นี่ เรากำหนดไดเร็กทอรีที่จะบันทึกเอกสารของเรา สร้างเอกสารใหม่ และเริ่มต้น DocumentBuilder DocumentBuilder เป็นเครื่องมือหลักของเราในการเพิ่มเนื้อหาลงในเอกสาร

## ขั้นตอนที่ 2: การกำหนดค่าการจัดรูปแบบย่อหน้า

ต่อไปเราต้องปรับการตั้งค่าการจัดรูปแบบย่อหน้า ให้คิดว่านี่คือการปรับแต่งพื้นที่ทำงานของคุณเพื่อทำให้ทุกอย่างลงตัว

```csharp
ParagraphFormat paragraphFormat = builder.ParagraphFormat;
paragraphFormat.AddSpaceBetweenFarEastAndAlpha = true;
paragraphFormat.AddSpaceBetweenFarEastAndDigit = true;
```

 โดยการตั้งค่า`AddSpaceBetweenFarEastAndAlpha`และ`AddSpaceBetweenFarEastAndDigit` ถึง`true`เราบอกให้ Aspose.Words ปรับระยะห่างระหว่างอักขระเอเชียและตัวอักษรละตินหรือตัวเลขโดยอัตโนมัติ

## ขั้นตอนที่ 3: การเพิ่มข้อความลงในเอกสาร

เมื่อตั้งค่าการจัดรูปแบบแล้ว ให้เพิ่มข้อความเพื่อดูการปรับเปลี่ยนเหล่านี้

```csharp
builder.Writeln("Automatically adjust space between Asian and Latin text");
builder.Writeln("Automatically adjust space between Asian text and numbers");
```

ที่นี่ เราเพิ่มข้อความสองบรรทัดลงในเอกสาร บรรทัดแรกมีทั้งอักขระเอเชียและข้อความละติน ในขณะที่บรรทัดที่สองประกอบด้วยอักขระและตัวเลขเอเชีย ซึ่งจะช่วยให้เรามองเห็นการปรับระยะห่างได้ชัดเจน

## ขั้นตอนที่ 4: บันทึกเอกสาร

สุดท้ายเราจำเป็นต้องบันทึกเอกสารของเรา เหมือนกับการใส่ใจขั้นสุดท้ายกับโปรเจ็กต์ของคุณแล้วกดปุ่มบันทึก

```csharp
doc.Save(dataDir + "DocumentFormatting.SpaceBetweenAsianAndLatinText.docx");
```

ด้วยโค้ดบรรทัดนี้ เราจะบันทึกเอกสารของเราในไดเร็กทอรีที่ระบุพร้อมชื่อที่สื่อความหมาย แล้วก็เอาล่ะ! เอกสารของคุณพร้อมด้วยการปรับระยะห่างระหว่างข้อความเอเชียและละตินอย่างสมบูรณ์แบบ
