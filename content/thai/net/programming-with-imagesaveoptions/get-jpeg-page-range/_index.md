## นำเข้าเนมสเปซ

หากต้องการใช้ Aspose.Words สำหรับ .NET คุณจะต้องนำเข้าเนมสเปซที่จำเป็นที่จุดเริ่มต้นของโค้ด สิ่งนี้ทำให้แน่ใจได้ว่าคุณจะสามารถเข้าถึงคลาสและวิธีการทั้งหมดที่จำเป็นสำหรับการจัดการเอกสาร

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

## ขั้นตอนที่ 1: โหลดเอกสารของคุณ

ขั้นแรก เราต้องโหลดเอกสาร Word ที่เราต้องการแปลง สมมติว่าเอกสารของเรามีชื่อ`Rendering.docx` และอยู่ในไดเร็กทอรีที่ระบุโดยตัวยึดตำแหน่ง`YOUR DOCUMENT DIRECTORY`.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Rendering.docx");
```

 รหัสนี้เริ่มต้นเส้นทางไปยังเอกสารของคุณและโหลดลงใน Aspose.Words`Document` วัตถุ.

## ขั้นตอนที่ 2: ตั้งค่า ImageSaveOptions

 ต่อไปเราจะตั้งค่า`ImageSaveOptions` เพื่อระบุว่าเราต้องการสร้าง JPEG ของเราอย่างไร ซึ่งรวมถึงการตั้งค่าช่วงหน้า ความสว่างของภาพ คอนทราสต์ และความละเอียด

```csharp
ImageSaveOptions options = new ImageSaveOptions(SaveFormat.Jpeg);
options.PageSet = new PageSet(0); // แปลงเฉพาะหน้าแรก
options.ImageBrightness = 0.3f;   // ตั้งค่าความสว่าง
options.ImageContrast = 0.7f;     // ตั้งค่าคอนทราสต์
options.HorizontalResolution = 72f; // ตั้งค่าความละเอียด
```

## ขั้นตอนที่ 3: บันทึกเอกสารเป็น JPEG

สุดท้ายนี้ เราจะบันทึกเอกสารเป็นไฟล์ JPEG โดยใช้การตั้งค่าที่เรากำหนดไว้

```csharp
doc.Save(dataDir + "WorkingWithImageSaveOptions.GetJpegPageRange.jpeg", options);
```

 รหัสนี้จะบันทึกหน้าแรกของ`Rendering.docx` เป็นภาพ JPEG ที่มีการตั้งค่าความสว่าง คอนทราสต์ และความละเอียดที่ระบุ

## คำถามที่พบบ่อย

### ฉันสามารถแปลงหลายหน้าพร้อมกันได้หรือไม่
 ได้ คุณสามารถระบุช่วงของหน้าได้โดยใช้`PageSet` ทรัพย์สินใน`ImageSaveOptions`.

### ฉันจะปรับคุณภาพของภาพได้อย่างไร?
 คุณสามารถปรับคุณภาพของ JPEG ได้โดยใช้`JpegQuality` ทรัพย์สินใน`ImageSaveOptions`.

### ฉันสามารถบันทึกเป็นรูปแบบภาพอื่นได้หรือไม่?
 ใช่ Aspose.Words รองรับรูปแบบภาพที่หลากหลาย เช่น PNG, BMP และ TIFF เปลี่ยน`SaveFormat` ใน`ImageSaveOptions` ตามนั้น

### มีวิธีดูตัวอย่างภาพก่อนบันทึกหรือไม่?
คุณจะต้องใช้กลไกการแสดงตัวอย่างแยกต่างหาก เนื่องจาก Aspose.Words ไม่มีคุณลักษณะการแสดงตัวอย่างในตัว
