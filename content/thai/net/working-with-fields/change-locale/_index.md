## นำเข้าเนมสเปซ

ก่อนที่เราจะเริ่มเขียนโค้ด เราจำเป็นต้องนำเข้าเนมสเปซที่จำเป็นก่อน สิ่งเหล่านี้เปรียบเสมือนส่วนผสมในสูตรที่ช่วยให้ทุกอย่างทำงานได้อย่างราบรื่น

```csharp
using System.Globalization;
using System.Threading;
using Aspose.Words;
using Aspose.Words.Fields;
```

การเปลี่ยนภาษาในเอกสาร Word เป็นกระบวนการที่ไม่ซับซ้อน มาทำลายมันทีละขั้นตอนกัน

## ขั้นตอนที่ 1: ตั้งค่าเอกสารของคุณ

ก่อนอื่น มาตั้งค่าเอกสารและตัวสร้างเอกสารของเรากัน เหมือนกับการจัดพื้นที่ทำงานก่อนเริ่มทำอาหาร

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## ขั้นตอนที่ 2: แทรกเขตข้อมูลผสาน

ตอนนี้ เราจะแทรกเขตข้อมูลผสานสำหรับวันที่ นี่คือจุดที่สถานที่จะเข้ามามีบทบาท

```csharp
builder.InsertField("MERGEFIELD Date");
```

## ขั้นตอนที่ 3: บันทึกวัฒนธรรมปัจจุบัน

ก่อนที่เราจะเปลี่ยนสถานที่ เราจำเป็นต้องรักษาวัฒนธรรมในปัจจุบันเอาไว้ ให้คิดว่านี่เป็นการบุ๊กมาร์กสถานที่ของคุณก่อนที่จะไปยังบทอื่น

```csharp
CultureInfo currentCulture = Thread.CurrentThread.CurrentCulture;
```

## ขั้นตอนที่ 4: เปลี่ยนสถานที่

ต่อไป เราจะเปลี่ยนวัฒนธรรมของเธรดปัจจุบันเป็นภาษาเยอรมัน ("de-DE") นี่เหมือนกับการเปลี่ยนการตั้งค่าภาษาในโทรศัพท์ของคุณ

```csharp
Thread.CurrentThread.CurrentCulture = new CultureInfo("de-DE");
```

## ขั้นตอนที่ 5: ดำเนินการจดหมายเวียน

ตอนนี้ เราดำเนินการจดหมายเวียนด้วยวันที่ปัจจุบัน ซึ่งจะใช้สถานที่ใหม่กับรูปแบบวันที่

```csharp
doc.MailMerge.Execute(new[] { "Date" }, new object[] { DateTime.Now });
```

## ขั้นตอนที่ 6: ฟื้นฟูวัฒนธรรมดั้งเดิม

หลังจากดำเนินการจดหมายเวียน เราจะฟื้นฟูวัฒนธรรมดั้งเดิม นี่เหมือนกับการเปลี่ยนกลับไปใช้การตั้งค่าภาษาที่คุณต้องการ

```csharp
Thread.CurrentThread.CurrentCulture = currentCulture;
```

## ขั้นตอนที่ 7: บันทึกเอกสาร

สุดท้าย ให้บันทึกเอกสารลงในไดเร็กทอรีที่คุณระบุ

```csharp
doc.Save(dataDir + "WorkingWithFields.ChangeLocale.docx");
```
