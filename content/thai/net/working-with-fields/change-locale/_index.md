## นำเข้าเนมสเปซ

ก่อนที่เราจะเริ่มเขียนโค้ด เราจะต้องนำเข้าเนมสเปซที่จำเป็นก่อน ซึ่งเปรียบเสมือนส่วนผสมในสูตรอาหาร เพื่อให้แน่ใจว่าทุกอย่างทำงานได้อย่างราบรื่น

```csharp
using System.Globalization;
using System.Threading;
using Aspose.Words;
using Aspose.Words.Fields;
```

การเปลี่ยนตำแหน่งในเอกสาร Word เป็นกระบวนการที่ตรงไปตรงมา มาแบ่งขั้นตอนทีละขั้นตอนกัน

## ขั้นตอนที่ 1: ตั้งค่าเอกสารของคุณ

ขั้นแรกเลย มาตั้งค่าเอกสารและเครื่องมือสร้างเอกสารกันก่อน ซึ่งก็เหมือนกับการตั้งค่าพื้นที่ทำงานก่อนเริ่มทำอาหาร

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## ขั้นตอนที่ 2: แทรกเขตข้อมูลผสาน

ตอนนี้เราจะแทรกฟิลด์ผสานสำหรับวันที่ นี่คือจุดที่ตำแหน่งจะเข้ามามีบทบาท

```csharp
builder.InsertField("MERGEFIELD Date");
```

## ขั้นตอนที่ 3: บันทึกวัฒนธรรมปัจจุบัน

ก่อนที่เราจะเปลี่ยนตำแหน่ง เราก็ต้องบันทึกวัฒนธรรมปัจจุบันเสียก่อน ลองนึกถึงการทำบุ๊กมาร์กสถานที่ของคุณก่อนจะไปยังบทอื่น

```csharp
CultureInfo currentCulture = Thread.CurrentThread.CurrentCulture;
```

## ขั้นตอนที่ 4: เปลี่ยนตำแหน่ง

ต่อไปเราจะเปลี่ยนภาษาในเธรดปัจจุบันเป็นภาษาเยอรมัน ("de-DE") ซึ่งก็เหมือนกับการเปลี่ยนการตั้งค่าภาษาในโทรศัพท์ของคุณ

```csharp
Thread.CurrentThread.CurrentCulture = new CultureInfo("de-DE");
```

## ขั้นตอนที่ 5: ดำเนินการผสานจดหมาย

ตอนนี้เราจะดำเนินการผสานจดหมายด้วยวันที่ปัจจุบัน ซึ่งจะใช้ตำแหน่งใหม่กับรูปแบบวันที่

```csharp
doc.MailMerge.Execute(new[] { "Date" }, new object[] { DateTime.Now });
```

## ขั้นตอนที่ 6: ฟื้นฟูวัฒนธรรมดั้งเดิม

หลังจากดำเนินการผสานจดหมายแล้ว เราจะคืนค่าภาษาเดิม ซึ่งก็เหมือนกับการสลับกลับไปยังการตั้งค่าภาษาที่คุณต้องการ

```csharp
Thread.CurrentThread.CurrentCulture = currentCulture;
```

## ขั้นตอนที่ 7: บันทึกเอกสาร

สุดท้ายให้บันทึกเอกสารไปยังไดเร็กทอรีที่คุณระบุ

```csharp
doc.Save(dataDir + "WorkingWithFields.ChangeLocale.docx");
```
