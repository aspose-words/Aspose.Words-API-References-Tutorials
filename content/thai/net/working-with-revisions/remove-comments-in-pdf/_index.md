## นำเข้าเนมสเปซ

ก่อนอื่น เราต้องนำเข้าเนมสเปซที่จำเป็นก่อน สิ่งนี้ช่วยให้เราสามารถใช้คลาสและวิธีการที่ได้รับจาก Aspose.Words

```csharp
using Aspose.Words;
using Aspose.Words.Layout;
```

เนมสเปซเหล่านี้ทำให้เราสามารถเข้าถึงการจัดการเอกสารและตัวเลือกเค้าโครงที่เราต้องการ

## ขั้นตอนที่ 1: โหลดเอกสาร

เริ่มต้นด้วยการโหลดเอกสารที่มีความคิดเห็น เอกสารนี้ควรเก็บไว้ในไดเร็กทอรีที่คุณสามารถเข้าถึงได้


```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Revisions.docx");
```

 ในตัวอย่างนี้ ให้แทนที่`"YOUR DOCUMENT DIRECTORY"` ด้วยเส้นทางจริงไปยังไดเร็กทอรีเอกสารของคุณ เรากำลังโหลดเอกสารชื่อ`Revisions.docx`.

## ขั้นตอนที่ 2: ซ่อนความคิดเห็นใน PDF

ต่อไป เราต้องซ่อนความคิดเห็นเพื่อไม่ให้ปรากฏในเอกสารเวอร์ชัน PDF Aspose.Words ทำให้สิ่งนี้ตรงไปตรงมาอย่างไม่น่าเชื่อ

```csharp
// ซ่อนความคิดเห็นในรูปแบบ PDF
doc.LayoutOptions.CommentDisplayMode = CommentDisplayMode.Hide;
```

บรรทัดโค้ดนี้บอกให้ Aspose.Words ซ่อนความคิดเห็นเมื่อแสดงผลเอกสาร

## ขั้นตอนที่ 3: บันทึกเอกสารเป็น PDF

สุดท้าย เราจะบันทึกเอกสารที่แก้ไขเป็น PDF ขั้นตอนนี้ช่วยให้แน่ใจว่าความคิดเห็นของเราจะถูกลบออกจากไฟล์เอาต์พุต


```csharp
doc.Save(dataDir + "WorkingWithRevisions.RemoveCommentsInPdf.pdf");
```

ที่นี่ เราบันทึกเอกสารลงในไดเร็กทอรีเดียวกันด้วยชื่อใหม่ ซึ่งระบุว่าความคิดเห็นได้ถูกลบออกแล้วในเวอร์ชัน PDF
