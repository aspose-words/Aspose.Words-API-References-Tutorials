## การแนะนำ

เคยพบว่าตัวเองต้องปวดหัวกับการแก้ไขเอกสารมากมาย โดยสงสัยว่าใครย้ายอะไรและเมื่อใด คุณไม่ได้อยู่คนเดียว การจัดการการแก้ไขเอกสารอาจเป็นงานที่น่าเบื่อ โดยเฉพาะอย่างยิ่งเมื่อต้องจัดการกับเอกสารจำนวนมาก แต่ไม่ต้องกังวล! ด้วย Aspose.Words สำหรับ .NET คุณสามารถระบุและจัดการการแก้ไขเหล่านี้ได้อย่างง่ายดาย ในคู่มือนี้ เราจะแนะนำคุณตลอดกระบวนการทีละขั้นตอนเกี่ยวกับวิธีการรับประเภทการแก้ไขของคำในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET รัดเข็มขัดแล้วดำดิ่งลงไปได้เลย!

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะจัดการกับโค้ดบางส่วน คุณจะต้องมีบางสิ่งก่อน:

1.  Aspose.Words for .NET Library: หากคุณยังไม่ได้ดาวน์โหลด ให้ดาวน์โหลดจาก[ที่นี่](https://releases.aspose.com/words/net/).
2. สภาพแวดล้อมการพัฒนา: Visual Studio หรือ IDE ที่รองรับ .NET อื่นๆ
3. ความรู้พื้นฐานของ C#: ความเข้าใจในภาษาการเขียนโปรแกรม C# จะเป็นประโยชน์
4.  เอกสาร Word พร้อมการแก้ไข: ตรวจสอบให้แน่ใจว่าคุณมี`.docx`ไฟล์ที่มีการเปลี่ยนแปลงที่ติดตามเพื่อทดสอบโค้ด

## นำเข้าเนมสเปซ

ในการเริ่มต้น คุณจะต้องนำเข้าเนมสเปซที่จำเป็นในโปรเจ็กต์ C# ของคุณ สิ่งนี้จะช่วยให้คุณสามารถเข้าถึงฟังก์ชันการทำงานที่ Aspose.Words สำหรับ .NET มอบให้

```csharp
using Aspose.Words;
using Aspose.Words.Revision;
using System;
```

เรามาแบ่งตัวอย่างออกเป็นหลายขั้นตอนเพื่อความเข้าใจและการนำไปปฏิบัติที่ดีขึ้น

## ขั้นตอนที่ 1: ตั้งค่าไดเร็กทอรีเอกสารของคุณ

ก่อนอื่น คุณต้องกำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือที่ที่เอกสาร Word ของคุณพร้อมการแก้ไขจะตั้งอยู่

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` ด้วยเส้นทางจริงไปยังโฟลเดอร์เอกสารของคุณ

## ขั้นตอนที่ 2: โหลดเอกสาร Word ของคุณ

ถัดไป คุณต้องโหลดเอกสาร Word ลงในโครงการของคุณ เอกสารนี้ควรมีการแก้ไขที่คุณต้องการวิเคราะห์

```csharp
Document doc = new Document(dataDir + "Revisions.docx");
```

 ตรวจสอบให้แน่ใจว่าไฟล์`Revisions.docx` มีอยู่ในไดเร็กทอรีที่ระบุ

## ขั้นตอนที่ 3: เข้าถึงคอลเลกชันย่อหน้า

เมื่อโหลดเอกสารของคุณแล้ว คุณต้องเข้าถึงย่อหน้าต่างๆ ภายในส่วนแรกของเนื้อหาของเอกสาร ซึ่งจะช่วยให้คุณวนซ้ำแต่ละย่อหน้าเพื่อตรวจสอบการแก้ไข

```csharp
ParagraphCollection paragraphs = doc.FirstSection.Body.Paragraphs;
```

## ขั้นตอนที่ 4: วนซ้ำทุกย่อหน้าและตรวจสอบการแก้ไข

นี่คือจุดที่ความมหัศจรรย์เกิดขึ้น คุณจะวนซ้ำแต่ละย่อหน้าและตรวจสอบว่าได้ถูกย้ายแล้ว (ลบหรือแทรก)

```csharp
for (int i = 0; i < paragraphs.Count; i++)
{
    if (paragraphs[i].IsMoveFromRevision)
        Console.WriteLine("Paragraph {0} has been moved (deleted).", i);
    if (paragraphs[i].IsMoveToRevision)
        Console.WriteLine("Paragraph {0} has been moved (inserted).", i);
}
```

 การวนซ้ำนี้จะผ่านแต่ละย่อหน้าและใช้`IsMoveFromRevision`และ`IsMoveToRevision` คุณสมบัติเพื่อตรวจสอบว่าย่อหน้าถูกย้าย (ลบ) หรือย้าย (แทรก)
