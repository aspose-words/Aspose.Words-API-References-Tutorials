## การแนะนำ

เคยพบว่าตัวเองจำเป็นต้องรักษาความปลอดภัยเอกสารด้วยรหัสผ่านหรือไม่? คุณไม่ได้อยู่คนเดียว ด้วยการเพิ่มขึ้นของเอกสารดิจิทัล การปกป้องข้อมูลที่ละเอียดอ่อนจึงมีความสำคัญมากกว่าที่เคย Aspose.Words สำหรับ .NET นำเสนอวิธีที่ราบรื่นในการเข้ารหัสเอกสารของคุณด้วยรหัสผ่าน ลองนึกภาพว่ามันเป็นการล็อคไดอารี่ของคุณ เฉพาะผู้ที่มีกุญแจ (หรือรหัสผ่านในกรณีนี้) เท่านั้นที่สามารถเข้าไปดูข้างในได้ มาดูกันว่าคุณจะบรรลุเป้าหมายนี้ได้อย่างไรทีละขั้นตอน

## นำเข้าเนมสเปซ

ก่อนที่เราจะเริ่ม คุณจะต้องนำเข้าเนมสเปซที่จำเป็นก่อน คิดว่าเนมสเปซเป็นชุดเครื่องมือที่คุณต้องการสำหรับโปรเจ็กต์ DIY ของคุณ

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

## ขั้นตอนที่ 1: สร้างเอกสาร

ก่อนอื่น มาสร้างเอกสารใหม่กันก่อน ก็เหมือนกับการเตรียมกระดาษเปล่าให้พร้อม

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

### คำอธิบาย

- dataDir: ตัวแปรนี้เก็บเส้นทางที่จะบันทึกเอกสารของคุณ
- Document doc = new Document(): บรรทัดนี้เริ่มต้นเอกสารใหม่
- DocumentBuilder builder = new DocumentBuilder(doc): DocumentBuilder เป็นเครื่องมือที่มีประโยชน์สำหรับการเพิ่มเนื้อหาลงในเอกสารของคุณ

## ขั้นตอนที่ 2: เพิ่มเนื้อหา

ตอนนี้เรามีแผ่นเปล่าแล้ว มาเขียนอะไรบางอย่างลงไปกันดีกว่า แล้วคำว่า “สวัสดีชาวโลก!” ธรรมดาๆ ล่ะ? คลาสสิค.

```csharp
builder.Write("Hello world!");
```

### คำอธิบาย

- builder.Write("Hello world!"): บรรทัดนี้จะเพิ่มข้อความ "Hello world!" ไปยังเอกสารของคุณ

## ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการบันทึก

มาถึงส่วนสำคัญ—การกำหนดค่าตัวเลือกการบันทึกเพื่อรวมการป้องกันด้วยรหัสผ่าน นี่คือที่ที่คุณจะตัดสินใจเกี่ยวกับความแข็งแกร่งของล็อคของคุณ

```csharp
DocSaveOptions saveOptions = new DocSaveOptions { Password = "password" };
```

### คำอธิบาย

- DocSaveOptions saveOptions = new DocSaveOptions: เตรียมใช้งานอินสแตนซ์ใหม่ของคลาส DocSaveOptions
- รหัสผ่าน = "รหัสผ่าน": ตั้งรหัสผ่านสำหรับเอกสาร แทนที่ "รหัสผ่าน" ด้วยรหัสผ่านที่คุณต้องการ

## ขั้นตอนที่ 4: บันทึกเอกสาร

สุดท้ายมาบันทึกเอกสารของเราด้วยตัวเลือกที่ระบุ เหมือนกับการเก็บไดอารี่ที่ล็อคไว้ในที่ปลอดภัย

```csharp
doc.Save(dataDir + "WorkingWithDocSaveOptions.EncryptDocumentWithPassword.docx", saveOptions);
```

### คำอธิบาย

- doc.Save: บันทึกเอกสารไปยังเส้นทางที่ระบุด้วยตัวเลือกการบันทึกที่กำหนดไว้
- dataDir + "WorkingWithDocSaveOptions.EncryptDocumentWithPassword.docx": สร้างเส้นทางแบบเต็มและชื่อไฟล์สำหรับเอกสาร
