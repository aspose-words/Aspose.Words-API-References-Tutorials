ในบทช่วยสอนนี้ เราจะสำรวจซอร์สโค้ด C# ที่ให้มาสำหรับฟีเจอร์ "การบันทึกรูปภาพเป็น WMF พร้อมตัวเลือกการบันทึก RTF" ด้วย Aspose.Words สำหรับ .NET คุณสมบัตินี้ช่วยให้คุณบันทึกรูปภาพเอกสารในรูปแบบ Windows Metafile (WMF) เมื่อแปลงเป็นรูปแบบ RTF

## ขั้นตอนที่ 1: การตั้งค่าสภาพแวดล้อม

ก่อนที่คุณจะเริ่มต้น ตรวจสอบให้แน่ใจว่าคุณได้ตั้งค่าสภาพแวดล้อมการพัฒนาของคุณด้วย Aspose.Words สำหรับ .NET ตรวจสอบให้แน่ใจว่าคุณได้เพิ่มข้อมูลอ้างอิงที่จำเป็นและนำเข้าเนมสเปซที่เหมาะสมแล้ว

## ขั้นตอนที่ 2: กำลังโหลดเอกสาร

```csharp
// พาธไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

Document doc = new Document(dataDir + "Document.docx");
```

 ในขั้นตอนนี้ เราโหลดเอกสารโดยใช้`Document` วิธีการและส่งเส้นทางไปยังไฟล์ DOCX ที่จะโหลด

## ขั้นตอนที่ 3: การกำหนดค่าตัวเลือกการสำรองข้อมูล

```csharp
RtfSaveOptions saveOptions = new RtfSaveOptions { SaveImagesAsWmf = true };
```

 ในขั้นตอนนี้ เราจะกำหนดค่าตัวเลือกการสำรองข้อมูล RTF เราสร้างใหม่`RtfSaveOptions` วัตถุและตั้งค่า`SaveImagesAsWmf`ทรัพย์สินเพื่อ`true`. ซึ่งจะเป็นการบอกให้ Aspose.Words บันทึกรูปภาพเอกสารเป็น WMF เมื่อแปลงเป็น RTF

## ขั้นตอนที่ 4: บันทึกเอกสาร

```csharp
doc.Save(dataDir + "WorkingWithRtfSaveOptions.SavingImagesAsWmf.rtf", saveOptions);
```

 ในขั้นตอนสุดท้ายนี้ เราจะบันทึกเอกสารผลลัพธ์ในรูปแบบ RTF โดยใช้นามสกุลไฟล์`Save` วิธีการและส่งเส้นทางไปยังไฟล์เอาต์พุตพร้อมกับตัวเลือกการบันทึกที่ระบุ

ตอนนี้คุณสามารถเรียกใช้ซอร์สโค้ดเพื่อบันทึกรูปภาพเอกสารในรูปแบบ WMF ในขณะที่แปลงเป็นรูปแบบ RTF เอกสารผลลัพธ์จะถูกบันทึกในไดเร็กทอรีที่ระบุโดยใช้ชื่อ "WorkingWithRtfSaveOptions.SavingImagesAsWmf.rtf"

### ซอร์สโค้ดตัวอย่างสำหรับการทำงานของการบันทึกภาพ WMF ด้วยตัวเลือกการบันทึก RTF ด้วย Aspose.Words สำหรับ .NET"

```csharp

            
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Document.docx");

RtfSaveOptions saveOptions = new RtfSaveOptions { SaveImagesAsWmf = true };

doc.Save(dataDir + "WorkingWithRtfSaveOptions.SavingImagesAsWmf.rtf", saveOptions);
            
        
```
