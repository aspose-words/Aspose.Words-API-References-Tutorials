ก่อนที่เราจะพูดถึงโค้ด โปรดตรวจสอบให้แน่ใจว่าคุณมีทุกสิ่งที่คุณต้องการ:

1.  Aspose.Words สำหรับ .NET: หากคุณยังไม่ได้ดาวน์โหลด ให้ดาวน์โหลดไลบรารีจาก[ที่นี่](https://releases.aspose.com/words/net/).

4. เอกสาร Word สองฉบับ: ตรวจสอบให้แน่ใจว่าคุณมีเอกสาร Word สองฉบับที่พร้อมสำหรับการรวม

## นำเข้าเนมสเปซ

ก่อนอื่น เราต้องนำเข้าเนมสเปซที่จำเป็นก่อน สิ่งเหล่านี้จะทำให้เราสามารถเข้าถึงฟังก์ชัน Aspose.Words ได้

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

ตอนนี้ เรามาแบ่งกระบวนการออกเป็นขั้นตอนง่ายๆ ที่เข้าใจง่าย

## ขั้นตอนที่ 1: ตั้งค่าโครงการของคุณ

ก่อนที่เราจะเจาะลึกโค้ด ตรวจสอบให้แน่ใจว่าโครงการของคุณได้รับการตั้งค่าอย่างถูกต้อง นี่คือรายการตรวจสอบด่วน:

1. สร้างโครงการใหม่: เปิด Visual Studio และสร้างโครงการแอปคอนโซลใหม่
2.  เพิ่มการอ้างอิง Aspose.Words: ดาวน์โหลดและเพิ่มไลบรารี Aspose.Words ในโครงการของคุณ คุณสามารถทำได้ผ่าน NuGet Package Manager โดยค้นหา`Aspose.Words`.

```csharp
Install-Package Aspose.Words
```

## ขั้นตอนที่ 2: กำหนดไดเร็กทอรีเอกสาร

ต่อไป เรามากำหนดไดเร็กทอรีที่เก็บเอกสารของคุณกัน นี่คือที่ที่ Aspose.Words จะดึงข้อมูลและบันทึกไฟล์ของคุณ

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางสู่เอกสารของคุณจริง

## ขั้นตอนที่ 3: โหลดเอกสารต้นฉบับ

ตอนนี้ มาโหลดเอกสารที่คุณต้องการผนวกกัน นี่คือเอกสารต้นฉบับของคุณ

```csharp
Document srcDoc = new Document(dataDir + "Document source.docx");
```

 ที่นี่เรากำลังสร้างสิ่งใหม่`Document` object และโหลดไฟล์ชื่อ "Document source.docx" จากไดเร็กทอรีของคุณ

## ขั้นตอนที่ 4: โหลดเอกสารปลายทาง

ในทำนองเดียวกัน โหลดเอกสารที่คุณต้องการผนวกเอกสารต้นฉบับ นี่คือเอกสารปลายทางของคุณ

```csharp
Document dstDoc = new Document(dataDir + "Northwind traders.docx");
```

 อีกครั้งเราสร้างใหม่`Document` object และโหลดไฟล์ชื่อ "Northwind Trader.docx" จากไดเร็กทอรีของคุณ

## ขั้นตอนที่ 5: ผนวกเอกสารต้นฉบับ

 นี่คือจุดที่ความมหัศจรรย์เกิดขึ้น! เราจะผนวกเอกสารต้นฉบับเข้ากับเอกสารปลายทางโดยใช้`AppendDocument` วิธี.

```csharp
dstDoc.AppendDocument(srcDoc, ImportFormatMode.KeepSourceFormatting);
```

 ที่`AppendDocument` วิธีการใช้พารามิเตอร์สองตัว:
1. เอกสารต้นฉบับ: เอกสารที่คุณต้องการผนวก
2.  โหมดรูปแบบการนำเข้า: พารามิเตอร์นี้จะกำหนดวิธีจัดการการจัดรูปแบบ นี่เราใช้อยู่.`KeepSourceFormatting` เพื่อรักษารูปแบบของเอกสารต้นฉบับ

## ขั้นตอนที่ 6: บันทึกเอกสารรวม

สุดท้าย ให้บันทึกเอกสารที่รวมกันลงในไดเร็กทอรีของคุณ

```csharp
dstDoc.Save(dataDir + "JoinAndAppendDocuments.SimpleAppendDocument.docx");
```

บรรทัดโค้ดนี้จะบันทึกเอกสารที่ผสานด้วยชื่อใหม่ เพื่อให้มั่นใจว่าไฟล์ต้นฉบับของคุณจะไม่เปลี่ยนแปลง
