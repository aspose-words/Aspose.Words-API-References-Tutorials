## การแนะนำ

คุณเคยจำเป็นต้องฝังวัตถุ OLE เช่น งานนำเสนอ PowerPoint หรือสเปรดชีต Excel ลงในเอกสาร Word แต่ต้องการให้ปรากฏเป็นไอคอนเล็กๆ ที่เรียบร้อยแทนที่จะเป็นวัตถุเต็มหรือไม่ คุณอยู่ในสถานที่ที่เหมาะสม! ในบทช่วยสอนนี้ เราจะอธิบายวิธีการแทรกวัตถุ OLE เป็นไอคอนในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET เมื่อสิ้นสุดคู่มือนี้ คุณจะสามารถรวมออบเจ็กต์ OLE เข้ากับเอกสารของคุณได้อย่างราบรื่น ทำให้มีการโต้ตอบและดึงดูดสายตามากขึ้น

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเจาะลึกรายละเอียดที่สำคัญ เรามาพูดถึงสิ่งที่คุณต้องการกันดีกว่า:

1.  Aspose.Words for .NET: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง Aspose.Words for .NET แล้ว หากคุณยังไม่ได้ติดตั้ง คุณสามารถดาวน์โหลดได้จาก[กำหนดหน้าการเผยแพร่](https://releases.aspose.com/words/net/).
2. สภาพแวดล้อมการพัฒนา: คุณต้องมีสภาพแวดล้อมการพัฒนาแบบรวม (IDE) เช่น Visual Studio
3. ความรู้พื้นฐานของ C#: ความเข้าใจพื้นฐานเกี่ยวกับการเขียนโปรแกรม C# จะเป็นประโยชน์

## นำเข้าเนมสเปซ

ขั้นแรก คุณต้องนำเข้าเนมสเปซที่จำเป็น นี่เป็นสิ่งจำเป็นสำหรับการเข้าถึงฟังก์ชันไลบรารี Aspose.Words

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
```

## ขั้นตอนที่ 1: สร้างเอกสารใหม่

ในการเริ่มต้น คุณต้องสร้างอินสแตนซ์เอกสาร Word ใหม่

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

ข้อมูลโค้ดนี้เตรียมใช้งานเอกสาร Word ใหม่และวัตถุ DocumentBuilder ซึ่งใช้ในการสร้างเนื้อหาเอกสาร

## ขั้นตอนที่ 2: แทรกวัตถุ OLE เป็นไอคอน

 ตอนนี้ เรามาแทรกวัตถุ OLE เป็นไอคอนกัน ที่`InsertOleObjectAsIcon` วิธีการของคลาส DocumentBuilder ใช้เพื่อจุดประสงค์นี้

```csharp
builder.InsertOleObjectAsIcon("path_to_your_presentation.pptx", false, "path_to_your_icon.ico", "My embedded file");
```

เรามาแจกแจงวิธีการนี้:
- `"path_to_your_presentation.pptx"`: นี่คือเส้นทางไปยังวัตถุ OLE ที่คุณต้องการฝัง
- `false` : พารามิเตอร์บูลีนนี้ระบุว่าจะแสดงวัตถุ OLE เป็นไอคอนหรือไม่ เนื่องจากเราต้องการไอคอน เราจึงตั้งค่าเป็น`false`.
- `"path_to_your_icon.ico"`: นี่คือเส้นทางไปยังไฟล์ไอคอนที่คุณต้องการใช้สำหรับวัตถุ OLE
- `"My embedded file"`: นี่คือป้ายกำกับที่จะปรากฏใต้ไอคอน

## ขั้นตอนที่ 3: บันทึกเอกสาร

สุดท้ายคุณต้องบันทึกเอกสาร เลือกไดเร็กทอรีที่คุณต้องการบันทึกไฟล์ของคุณ

```csharp
doc.Save(dataDir + "WorkingWithOleObjectsAndActiveX.InsertOleObjectAsIcon.docx");
```

บรรทัดโค้ดนี้จะบันทึกเอกสารไปยังเส้นทางที่ระบุ
