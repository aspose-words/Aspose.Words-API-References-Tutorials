## การแนะนำ

ในบทช่วยสอนนี้ เรากำลังเจาะลึกฟีเจอร์สุดเจ๋งของ Aspose.Words สำหรับ .NET: การแทรกออบเจ็กต์ OLE (Object Linking and Embedding) เป็นไอคอนโดยใช้สตรีม ไม่ว่าคุณจะฝังงานนำเสนอ PowerPoint, สเปรดชีต Excel หรือไฟล์ประเภทอื่นๆ คู่มือนี้จะแสดงวิธีการฝังอย่างชัดเจน พร้อมที่จะเริ่มต้นหรือยัง? ไปกันเลย!

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะพูดถึงโค้ด มีบางสิ่งที่คุณต้องการ:

-  Aspose.Words สำหรับ .NET: หากคุณยังไม่ได้ดำเนินการ[ดาวน์โหลด](https://releases.aspose.com/words/net/) และติดตั้ง Aspose.Words สำหรับ .NET
- สภาพแวดล้อมการพัฒนา: Visual Studio หรือสภาพแวดล้อมการพัฒนา C# อื่น ๆ
- ไฟล์อินพุต: ไฟล์ที่คุณต้องการฝัง (เช่น งานนำเสนอ PowerPoint) และรูปภาพไอคอน

## นำเข้าเนมสเปซ

ในการเริ่มต้น ตรวจสอบให้แน่ใจว่าคุณได้นำเข้าเนมสเปซที่จำเป็นในโครงการของคุณแล้ว:

```csharp
using System;
using System.IO;
using Aspose.Words;
using Aspose.Words.Drawing;
```

เรามาแจกแจงกระบวนการทีละขั้นตอนเพื่อให้ง่ายต่อการติดตาม

## ขั้นตอนที่ 1: สร้างเอกสารใหม่

ขั้นแรก เราจะสร้างเอกสารใหม่และตัวสร้างเอกสารเพื่อใช้งาน

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 คิดถึง`Document` เป็นผืนผ้าใบว่างเปล่าของคุณและ`DocumentBuilder` เป็นพู่กันของคุณ เรากำลังจัดเตรียมเครื่องมือเพื่อเริ่มสร้างสรรค์ผลงานชิ้นเอกของเรา

## ขั้นตอนที่ 2: เตรียมสตรีม

ต่อไป เราต้องเตรียมสตรีมหน่วยความจำที่มีไฟล์ที่เราต้องการฝัง ในตัวอย่างนี้ เราจะฝังงานนำเสนอ PowerPoint

```csharp
using (MemoryStream stream = new MemoryStream(File.ReadAllBytes("Path_to_your_directory/Presentation.pptx")))
{
```

ขั้นตอนนี้เหมือนกับการลงสีลงบนแปรง เรากำลังเตรียมไฟล์ของเราให้พร้อมที่จะฝัง

## ขั้นตอนที่ 3: แทรกวัตถุ OLE เป็นไอคอน

ตอนนี้ เราจะใช้เครื่องมือสร้างเอกสารเพื่อแทรกวัตถุ OLE ลงในเอกสาร เราจะระบุสตรีมไฟล์, ProgID สำหรับประเภทไฟล์ (ในกรณีนี้คือ "แพ็คเกจ") เส้นทางไปยังรูปภาพไอคอน และป้ายกำกับสำหรับไฟล์ที่ฝัง

```csharp
builder.InsertOleObjectAsIcon(stream, "Package", "Path_to_your_directory/Logo icon.ico", "My embedded file");
}
```

นี่คือจุดที่ความมหัศจรรย์เกิดขึ้น! เรากำลังฝังไฟล์ของเราและแสดงเป็นไอคอนภายในเอกสาร

## ขั้นตอนที่ 4: บันทึกเอกสาร

สุดท้าย เราบันทึกเอกสารไปยังเส้นทางที่ระบุ

```csharp
doc.Save(dataDir + "WorkingWithOleObjectsAndActiveX.InsertOleObjectAsIconUsingStream.docx");
```

ขั้นตอนนี้เหมือนกับการนำภาพวาดที่เสร็จแล้วใส่กรอบแล้วแขวนไว้บนผนัง เอกสารของคุณพร้อมใช้งานแล้ว!

## บทสรุป

และคุณก็ได้แล้ว! คุณได้ฝังวัตถุ OLE เป็นไอคอนในเอกสาร Word เรียบร้อยแล้วโดยใช้ Aspose.Words สำหรับ .NET คุณสมบัติอันทรงพลังนี้สามารถช่วยให้คุณสร้างเอกสารแบบไดนามิกและโต้ตอบได้อย่างง่ายดาย ไม่ว่าคุณจะฝังงานนำเสนอ สเปรดชีต หรือไฟล์อื่นๆ Aspose.Words ก็ทำให้เป็นเรื่องง่าย ลองใช้เลย และดูความแตกต่างที่สามารถสร้างได้ในเอกสารของคุณ!
