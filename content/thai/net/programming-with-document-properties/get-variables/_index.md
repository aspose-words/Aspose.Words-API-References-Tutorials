## นำเข้าเนมสเปซ

ขั้นแรกเลย เรามาทำการนำเข้าเนมสเปซที่จำเป็นกันก่อน ซึ่งก็เหมือนกับการเตรียมฉากก่อนการแสดงใหญ่

using System;

## ขั้นตอนที่ 1: การตั้งค่าโครงการของคุณ

3. ตั้งชื่อโครงการของคุณและคลิก "สร้าง"

### การติดตั้ง Aspose.Words

ตอนนี้เรามานำไลบรารี Aspose.Words เข้าสู่โปรเจ็กต์ของเรากัน

1. เปิดตัวจัดการแพ็กเกจ NuGet ใน Visual Studio
2. ค้นหา "Aspose.Words" และติดตั้ง

## ขั้นตอนที่ 2: การโหลดเอกสาร Word

 มาโหลดเอกสาร Word ลงในแอปพลิเคชันของเรากัน เราจะใช้โปรแกรมง่ายๆ`Document` คัดค้านที่จะทำเช่นนี้

1. สร้างตัวแปรสตริงใหม่เพื่อเก็บเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ:
    ```csharp
    string dataDir = "YOUR DOCUMENT DIRECTORY";
    ```

2.  สร้าง`Document` วัตถุและโหลดเอกสาร:
    ```csharp
    Document doc = new Document(dataDir + "Document.docx");
    ```

ขั้นตอนนี้เหมือนกับการเปิดหนังสือ เราแค่บอกโปรแกรมว่าจะค้นหาเอกสารได้จากที่ใด จากนั้นจึงเปิดเอกสารนั้นขึ้นมาเพื่อที่เราจะเริ่มทำงานกับเอกสารนั้นได้

## ขั้นตอนที่ 3: การแยกตัวแปรเอกสาร

ตอนนี้เรามาแยกตัวแปรจากเอกสารของเรา

1. สร้างตัวแปรสตริงเพื่อเก็บตัวแปร:
    ```csharp
    string variables = "";
    ```

2. วนซ้ำผ่านตัวแปรเอกสารและเชื่อมโยงเข้าในตัวแปรสตริง:
    ```csharp
    foreach (KeyValuePair<string, string> entry in doc.Variables)
    {
        string name = entry.Key;
        string value = entry.Value;
        if (variables == "")
        {
            variables = "Name: " + name + ", " + "Value: {1}" + value;
        }
        else
        {
            variables = variables + " Name: " + name + ", " + "Value: {1}" + value;
        }
    }
    ```

3. พิมพ์ตัวแปร:
    ```csharp
    Console.WriteLine("\nDocument has the following variables: " + variables);
    ```


ลองนึกถึงขั้นตอนนี้เหมือนกับการพลิกดูหนังสือและจดบันทึกสิ่งสำคัญทั้งหมดลงไป เราจะดูเอกสารโดยแยกคู่คีย์-ค่าออกมาและจัดเก็บไว้ในสตริงเพื่อแสดงในภายหลัง
