## นำเข้าเนมสเปซ

ก่อนอื่น มานำเข้าเนมสเปซที่จำเป็นกันก่อน นี่เป็นการปูทางสำหรับทุกสิ่งที่เรากำลังจะทำ

```csharp
using System;
using Aspose.Words;
```

## ขั้นตอนที่ 1: ตั้งค่าไดเร็กทอรีเอกสารของคุณ

### 1.1 กำหนดเส้นทาง

เราต้องบอกโปรแกรมของเราว่าจะหาเอกสารที่เรากำลังดำเนินการได้จากที่ไหน นี่คือที่ที่เรากำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

### 1.2 โหลดเอกสาร

ต่อไปเราจะโหลดเอกสารลงในโปรแกรมของเรา ซึ่งทำได้ง่ายเพียงแค่ชี้ไปที่ไฟล์ที่เราต้องการจัดการ

```csharp
Document doc = new Document(dataDir + "Properties.docx");
```

## ขั้นตอนที่ 2: ลบข้อมูลส่วนบุคคล

### 2.1 เปิดใช้งานคุณสมบัติ

Aspose.Words ทำให้ง่ายต่อการดึงข้อมูลส่วนบุคคลออกจากเอกสารของคุณ สิ่งที่คุณต้องทำคือโค้ดหนึ่งบรรทัด

```csharp
doc.RemovePersonalInformation = true;
```

### 2.2 บันทึกเอกสาร

ตอนนี้เราได้ทำความสะอาดเอกสารของเราแล้ว มาบันทึกกันดีกว่า เพื่อให้แน่ใจว่าการเปลี่ยนแปลงทั้งหมดของเราจะถูกนำไปใช้และเอกสารพร้อมใช้งาน

```csharp
doc.Save(dataDir + "DocumentPropertiesAndVariables.RemovePersonalInformation.docx");
```
