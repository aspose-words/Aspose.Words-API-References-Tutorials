ในบทช่วยสอนนี้ เราจะแนะนำคุณเกี่ยวกับซอร์สโค้ด C# เพื่อลบคุณสมบัติแบบกำหนดเองออกจากเอกสารด้วย Aspose.Words สำหรับ .NET คุณลักษณะนี้ช่วยให้คุณสามารถลบคุณสมบัติแบบกำหนดเองเฉพาะออกจากเอกสารได้

## ขั้นตอนที่ 1: การตั้งค่าโครงการ

ในการเริ่มต้น ให้สร้างโปรเจ็กต์ C# ใหม่ใน IDE ที่คุณชื่นชอบ ตรวจสอบให้แน่ใจว่าไลบรารี Aspose.Words สำหรับ .NET ถูกอ้างอิงในโปรเจ็กต์ของคุณ

## ขั้นตอนที่ 2: กำลังโหลดเอกสาร

ในขั้นตอนนี้ เราจะโหลดเอกสาร Word ที่เราต้องการลบคุณสมบัติที่กำหนดเอง ใช้รหัสต่อไปนี้เพื่อโหลดเอกสาร:

```csharp
// พาธไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document doc = new Document(dataDir + "Properties.docx");
```

 แทนที่`"YOUR DOCUMENTS DIRECTORY"` ด้วยเส้นทางจริงของไดเร็กทอรีที่เอกสารของคุณตั้งอยู่

## ขั้นตอนที่ 3: การลบคุณสมบัติที่กำหนดเอง

ตอนนี้เรามาลบคุณสมบัติแบบกำหนดเองเฉพาะออกจากเอกสารกัน ใช้รหัสต่อไปนี้:

```csharp
doc.CustomDocumentProperties.Remove("Authorized Date");
```

รหัสนี้จะลบคุณสมบัติแบบกำหนดเอง "วันที่ได้รับอนุญาต" ออกจากเอกสาร คุณสามารถแทนที่ "วันที่ได้รับอนุญาต" ด้วยชื่อของคุณสมบัติที่กำหนดเองที่คุณต้องการลบ

### ตัวอย่างซอร์สโค้ดสำหรับการลบคุณสมบัติเอกสารแบบกำหนดเองโดยใช้ Aspose.Words สำหรับ .NET

```csharp

	// เส้นทางไปยังไดเร็กทอรีเอกสาร
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Properties.docx");
	doc.CustomDocumentProperties.Remove("Authorized Date");
	
```

 อย่าลืมระบุเส้นทางเอกสารที่ถูกต้องใน`dataDir` ตัวแปร.

ตอนนี้คุณได้เรียนรู้วิธีลบคุณสมบัติแบบกำหนดเองออกจากเอกสารโดยใช้ Aspose.Words สำหรับ .NET แล้ว ด้วยการทำตามคำแนะนำทีละขั้นตอนที่ให้ไว้ในบทช่วยสอนนี้ คุณสามารถลบคุณสมบัติที่กำหนดเองออกจากเอกสารของคุณเองได้อย่างง่ายดาย
