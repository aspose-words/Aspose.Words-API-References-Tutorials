## นำเข้าเนมสเปซ

หากต้องการใช้ Aspose.Words สำหรับ .NET คุณจะต้องนำเข้าเนมสเปซที่จำเป็น การดำเนินการนี้จะช่วยให้โค้ดของเราทำงานได้อย่างราบรื่น

```csharp
using System;
using Aspose.Words;
```

## ขั้นตอนที่ 1: การตั้งค่าไดเรกทอรีเอกสาร

ขั้นแรก ให้ตั้งค่าเส้นทางไปยังไดเร็กทอรีเอกสารของเราก่อน นี่คือที่ที่ไฟล์ Word ของคุณอยู่

### ขั้นตอนที่ 1.1: กำหนดไดเรกทอรีข้อมูล

ในโปรเจ็กต์ C# ของคุณ ให้กำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ แทนที่ "YOUR DOCUMENT DIRECTORY" ด้วยเส้นทางจริง

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

### ขั้นตอนที่ 1.2: โหลดเอกสาร

ตอนนี้เรามาโหลดเอกสารลงในโค้ดกัน นี่คือจุดที่ Aspose.Words สำหรับ .NET เข้ามามีบทบาท

```csharp
Document doc = new Document(dataDir + "Properties.docx");
```

## ขั้นตอนที่ 2: การลบคุณสมบัติเอกสารที่กำหนดเอง

เมื่อโหลดเอกสารเสร็จแล้ว ก็ถึงเวลาที่จะลบคุณสมบัติที่กำหนดเองที่น่ารำคาญเหล่านั้น ต่อไปนี้คือวิธีดำเนินการทีละขั้นตอน

### ขั้นตอนที่ 2.1: เข้าถึงคุณสมบัติเอกสารที่กำหนดเอง

ขั้นแรก ให้เข้าถึงคุณสมบัติเอกสารแบบกำหนดเองของเอกสารที่โหลด

```csharp
var customProperties = doc.CustomDocumentProperties;
```

### ขั้นตอนที่ 2.2: ลบคุณสมบัติเฉพาะ

ขั้นตอนต่อไปคือลบคุณสมบัติที่กำหนดเองตามชื่อ ในตัวอย่างนี้ เราจะลบ "วันที่ได้รับอนุญาต"

```csharp
customProperties.Remove("Authorized Date");
```

## ขั้นตอนที่ 3: การบันทึกเอกสาร

หลังจากลบคุณสมบัติที่กำหนดเองแล้ว ขั้นตอนสุดท้ายคือการบันทึกเอกสาร วิธีนี้จะช่วยให้มั่นใจว่าการเปลี่ยนแปลงของคุณจะถูกนำไปใช้

### ขั้นตอนที่ 3.1: กำหนดเส้นทางการบันทึก

กำหนดว่าคุณต้องการบันทึกเอกสารที่แก้ไขไว้ที่ไหน

```csharp
string savePath = dataDir + "ModifiedProperties.docx";
```

### ขั้นตอนที่ 3.2: บันทึกเอกสาร

สุดท้ายให้บันทึกเอกสารที่มีการเปลี่ยนแปลง

```csharp
doc.Save(savePath);
```
