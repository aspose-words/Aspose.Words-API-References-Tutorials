## นำเข้าเนมสเปซ

หากต้องการใช้ Aspose.Words สำหรับ .NET คุณจะต้องนำเข้าเนมสเปซที่จำเป็น นี่เป็นการปูทางให้โค้ดของเราทำงานได้อย่างราบรื่น

```csharp
using System;
using Aspose.Words;
```

## ขั้นตอนที่ 1: การตั้งค่าไดเร็กทอรีเอกสาร

ก่อนอื่น เรามาตั้งค่าเส้นทางไปยังไดเร็กทอรีเอกสารของเรากันก่อน นี่คือที่ตั้งไฟล์ Word ของคุณ

### ขั้นตอนที่ 1.1: กำหนดไดเร็กทอรีข้อมูล

ในโปรเจ็กต์ C# ของคุณ ให้กำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ แทนที่ "ไดเรกทอรีเอกสารของคุณ" ด้วยเส้นทางจริง

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

### ขั้นตอนที่ 1.2: โหลดเอกสาร

ตอนนี้เรามาโหลดเอกสารลงในโค้ดของเรากัน นี่คือจุดที่ Aspose.Words สำหรับ .NET เข้ามามีบทบาท

```csharp
Document doc = new Document(dataDir + "Properties.docx");
```

## ขั้นตอนที่ 2: การลบคุณสมบัติเอกสารที่กำหนดเอง

เมื่อโหลดเอกสารแล้ว ก็ถึงเวลาที่จะลบคุณสมบัติแบบกำหนดเองที่น่ารำคาญเหล่านั้นออก นี่คือวิธีที่คุณทำทีละขั้นตอน

### ขั้นตอนที่ 2.1: เข้าถึงคุณสมบัติเอกสารแบบกำหนดเอง

ขั้นแรก ให้เข้าถึงคุณสมบัติเอกสารแบบกำหนดเองของเอกสารที่โหลด

```csharp
var customProperties = doc.CustomDocumentProperties;
```

### ขั้นตอนที่ 2.2: ลบคุณสมบัติเฉพาะ

จากนั้น ลบคุณสมบัติแบบกำหนดเองเฉพาะตามชื่อ ในตัวอย่างนี้ เรากำลังลบ "วันที่ได้รับอนุญาต"

```csharp
customProperties.Remove("Authorized Date");
```

## ขั้นตอนที่ 3: บันทึกเอกสาร

หลังจากลบคุณสมบัติแบบกำหนดเองแล้ว ขั้นตอนสุดท้ายคือการบันทึกเอกสาร เพื่อให้แน่ใจว่าการเปลี่ยนแปลงของคุณจะถูกนำไปใช้

### ขั้นตอนที่ 3.1: กำหนดเส้นทางการบันทึก

กำหนดตำแหน่งที่คุณต้องการบันทึกเอกสารที่แก้ไข

```csharp
string savePath = dataDir + "ModifiedProperties.docx";
```

### ขั้นตอนที่ 3.2: บันทึกเอกสาร

สุดท้าย ให้บันทึกเอกสารพร้อมกับการเปลี่ยนแปลง

```csharp
doc.Save(savePath);
```
