## การแนะนำ

คุณเคยสงสัยบ้างไหมว่าจะรักษาสัดส่วนของรูปภาพและรูปร่างที่สมบูรณ์แบบในเอกสาร Word ของคุณได้อย่างไร? บางครั้ง คุณต้องแน่ใจว่ารูปภาพและรูปร่างของคุณไม่บิดเบี้ยวเมื่อปรับขนาด นี่คือจุดที่การล็อคอัตราส่วนภาพมีประโยชน์ ในบทช่วยสอนนี้ เราจะสำรวจวิธีตั้งค่าอัตราส่วนกว้างยาวสำหรับรูปร่างในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET เราจะแบ่งขั้นตอนออกเป็นขั้นตอนง่ายๆ เพื่อให้แน่ใจว่าคุณสามารถใช้ทักษะเหล่านี้กับโครงการของคุณได้อย่างมั่นใจ

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเจาะลึกโค้ด มาดูสิ่งที่คุณต้องเริ่มต้นกันก่อน:

- Aspose.Words สำหรับ .NET Library: คุณต้องติดตั้ง Aspose.Words สำหรับ .NET หากคุณยังไม่ได้คุณสามารถทำได้[ดาวน์โหลดได้ที่นี่](https://releases.aspose.com/words/net/).
- สภาพแวดล้อมการพัฒนา: ตรวจสอบให้แน่ใจว่าคุณได้ตั้งค่าสภาพแวดล้อมการพัฒนา .NET แล้ว Visual Studio เป็นตัวเลือกยอดนิยม
- ความรู้พื้นฐานเกี่ยวกับ C#: ความคุ้นเคยกับการเขียนโปรแกรม C# บ้างจะเป็นประโยชน์

## นำเข้าเนมสเปซ

ก่อนอื่น มานำเข้าเนมสเปซที่จำเป็นกันก่อน เนมสเปซเหล่านี้จะช่วยให้เราสามารถเข้าถึงคลาสและวิธีการที่เราต้องใช้ในการทำงานกับเอกสารและรูปร่าง Word

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
```

## ขั้นตอนที่ 1: ตั้งค่าไดเร็กทอรีเอกสารของคุณ

 ก่อนที่เราจะเริ่มจัดการรูปร่าง เราจำเป็นต้องตั้งค่าไดเร็กทอรีที่จะจัดเก็บเอกสารของเรา เพื่อความเรียบง่าย เราจะใช้ตัวยึดตำแหน่ง`YOUR DOCUMENT DIRECTORY`- แทนที่สิ่งนี้ด้วยเส้นทางจริงไปยังไดเร็กทอรีเอกสารของคุณ

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## ขั้นตอนที่ 2: สร้างเอกสารใหม่

ต่อไป เราจะสร้างเอกสาร Word ใหม่โดยใช้ Aspose.Words เอกสารนี้จะทำหน้าที่เป็นพื้นที่สำหรับเพิ่มรูปร่างและรูปภาพ

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 ที่นี่เราสร้างอินสแตนซ์ของ`Document` คลาสและใช้`DocumentBuilder` เพื่อช่วยเราสร้างเนื้อหาเอกสาร

## ขั้นตอนที่ 3: แทรกรูปภาพ

 ตอนนี้ เรามาแทรกรูปภาพลงในเอกสารของเรากันดีกว่า เราจะใช้`InsertImage` วิธีการของ`DocumentBuilder`ระดับ. ตรวจสอบให้แน่ใจว่าคุณมีรูปภาพในไดเร็กทอรีที่คุณระบุ

```csharp
Shape shape = builder.InsertImage(dataDir + "Transparent background logo.png");
```

 แทนที่`dataDir + "Transparent background logo.png"` พร้อมเส้นทางไปยังไฟล์ภาพของคุณ

## ขั้นตอนที่ 4: ล็อคอัตราส่วนภาพ

เมื่อแทรกรูปภาพแล้ว เราก็สามารถล็อกอัตราส่วนภาพได้ การล็อคอัตราส่วนภาพทำให้มั่นใจได้ว่าสัดส่วนของภาพจะคงที่เมื่อทำการปรับขนาด

```csharp
shape.AspectRatioLocked = true;
```

 การตั้งค่า`AspectRatioLocked` ถึง`true` ช่วยให้มั่นใจได้ว่ารูปภาพจะคงอัตราส่วนเดิมไว้

## ขั้นตอนที่ 5: บันทึกเอกสาร

สุดท้าย เราจะบันทึกเอกสารลงในไดเร็กทอรีที่ระบุ ขั้นตอนนี้เขียนการเปลี่ยนแปลงทั้งหมดที่เราทำกับไฟล์เอกสาร

```csharp
doc.Save(dataDir + "WorkingWithShapes.AspectRatioLocked.docx");
```

## บทสรุป

ยินดีด้วย! คุณได้เรียนรู้วิธีการตั้งค่าอัตราส่วนกว้างยาวสำหรับรูปร่างในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET เรียบร้อยแล้ว เมื่อทำตามขั้นตอนเหล่านี้ คุณจะมั่นใจได้ว่ารูปภาพและรูปร่างของคุณคงสัดส่วนไว้ ทำให้เอกสารของคุณดูเป็นมืออาชีพและสวยงาม คุณสามารถทดลองใช้รูปภาพและรูปร่างต่างๆ เพื่อดูว่าฟีเจอร์การล็อกอัตราส่วนภาพทำงานอย่างไรในสถานการณ์ต่างๆ

## คำถามที่พบบ่อย

### ฉันสามารถปลดล็อคอัตราส่วนภาพหลังจากล็อคได้หรือไม่?
ใช่ คุณสามารถปลดล็อคอัตราส่วนภาพได้โดยการตั้งค่า`shape.AspectRatioLocked = false`.

### จะเกิดอะไรขึ้นหากฉันปรับขนาดรูปภาพโดยล็อคอัตราส่วนภาพไว้
รูปภาพจะปรับขนาดตามสัดส่วน โดยคงอัตราส่วนความกว้างต่อความสูงเดิมไว้

### ฉันสามารถใช้สิ่งนี้กับรูปร่างอื่นนอกเหนือจากรูปภาพได้หรือไม่
อย่างแน่นอน! คุณลักษณะการล็อกอัตราส่วนภาพสามารถใช้กับรูปร่างใดก็ได้ รวมถึงสี่เหลี่ยม วงกลม และอื่นๆ

### Aspose.Words สำหรับ .NET เข้ากันได้กับ .NET Core หรือไม่
ใช่ Aspose.Words สำหรับ .NET รองรับทั้ง .NET Framework และ .NET Core
