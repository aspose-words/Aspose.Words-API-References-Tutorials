## นำเข้าเนมสเปซ

ก่อนที่จะข้ามไปยังส่วนการเขียนโค้ด ตรวจสอบให้แน่ใจว่าคุณได้นำเข้าเนมสเปซที่จำเป็นในโปรเจ็กต์ของคุณแล้ว นี่เป็นสิ่งสำคัญในการใช้ฟังก์ชันการทำงานของ Aspose.Words

```csharp
using System;
using Aspose.Words;
using Aspose.Words.PageSetup;
```

## ขั้นตอนที่ 1: โหลดเอกสารของคุณ

ก่อนอื่นคุณต้องโหลดเอกสารของคุณ นี่คือรากฐานที่คุณจะสร้างการตั้งค่าเพจของคุณ

 สร้างอินสแตนซ์ใหม่ของ`Document` คลาสและโหลดเอกสารของคุณจากไดเร็กทอรีที่ระบุ

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Document.docx");
```

## ขั้นตอนที่ 2: ตั้งค่าโหมดเค้าโครง

โหมดเค้าโครงจะกำหนดวิธีการจัดเรียงข้อความบนเพจ ในตัวอย่างนี้ เราจะใช้โหมดเค้าโครงตาราง สิ่งนี้มีประโยชน์อย่างยิ่งเมื่อต้องจัดการกับเอกสารในภาษาเอเชีย

```csharp
// ตั้งค่าโหมดเค้าโครงสำหรับส่วนที่อนุญาตให้กำหนดลักษณะการทำงานของตารางเอกสาร
doc.FirstSection.PageSetup.LayoutMode = SectionLayoutMode.Grid;
```

## ขั้นตอนที่ 3: กำหนดอักขระต่อบรรทัด

ต่อไป เรามากำหนดจำนวนอักขระต่อบรรทัดกัน ซึ่งช่วยในการรักษาความสม่ำเสมอในลักษณะที่ปรากฏของเอกสารของคุณ

```csharp
doc.FirstSection.PageSetup.CharactersPerLine = 30;
```

## ขั้นตอนที่ 4: กำหนดบรรทัดต่อหน้า

เช่นเดียวกับอักขระต่อบรรทัด การกำหนดจำนวนบรรทัดต่อหน้าช่วยให้แน่ใจว่าเอกสารของคุณมีรูปลักษณ์ที่สอดคล้องกัน

```csharp
doc.FirstSection.PageSetup.LinesPerPage = 10;
```

## ขั้นตอนที่ 5: บันทึกเอกสารของคุณ

หลังจากตั้งค่าเพจแล้ว ขั้นตอนสุดท้ายคือการบันทึกเอกสาร เพื่อให้แน่ใจว่าการตั้งค่าทั้งหมดของคุณจะถูกนำไปใช้และบันทึกอย่างถูกต้อง

```csharp
doc.Save(dataDir + "WorkingWithDocumentOptionsAndSettings.DocumentPageSetup.docx");
```
