## การแนะนำ

สวัสดี! พร้อมที่จะดำดิ่งสู่โลกของ Aspose.Words สำหรับ .NET แล้วหรือยัง? วันนี้ เราจะมาพูดถึงฟีเจอร์ที่มีประโยชน์สุดๆ นั่นก็คือ การรวมตารางในแนวนอน นี่อาจฟังดูเป็นเทคนิคเล็กน้อย แต่ไม่ต้องกังวล ฉันคอยช่วยเหลือคุณ เมื่อสิ้นสุดบทช่วยสอนนี้ คุณจะเชี่ยวชาญการรวมเซลล์ในเอกสาร Word โดยทางโปรแกรม เอาล่ะ เรามาพับแขนเสื้อของเราแล้วเริ่มต้นกันเลย!

เมื่อคุณจัดการสิ่งเหล่านี้เรียบร้อยแล้ว คุณก็พร้อมลุยได้เลย!

## นำเข้าเนมสเปซ

ก่อนที่จะเจาะลึกโค้ด เราต้องแน่ใจว่าเราได้นำเข้าเนมสเปซที่จำเป็นแล้ว ในโปรเจ็กต์ C# ของคุณ อย่าลืมรวม:

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

เอาล่ะ เรามาแจกแจงขั้นตอนการรวมเซลล์ตารางในแนวนอนในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET กัน

## ขั้นตอนที่ 1: การตั้งค่าเอกสารของคุณ

 ก่อนอื่น เราต้องสร้างเอกสาร Word ใหม่และเริ่มต้นไฟล์`DocumentBuilder`-

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 ข้อมูลโค้ดนี้จะตั้งค่าเอกสารใหม่และเตรียมการ`DocumentBuilder` สำหรับการดำเนินการ

## ขั้นตอนที่ 2: การแทรกเซลล์แรก

ต่อไป เราเริ่มต้นด้วยการแทรกเซลล์แรกและทำเครื่องหมายเพื่อผสานแนวนอน:

```csharp
builder.InsertCell();
builder.CellFormat.HorizontalMerge = CellMerge.First;
builder.Write("Text in merged cells.");
```

 ที่นี่ เราแทรกเซลล์ใหม่และตั้งค่า`HorizontalMerge`ทรัพย์สินเพื่อ`CellMerge.First`ซึ่งบ่งชี้ว่าเซลล์นี้เป็นจุดเริ่มต้นของลำดับเซลล์ที่ผสาน

## ขั้นตอนที่ 3: การแทรกเซลล์ที่ผสาน

ตอนนี้เราแทรกเซลล์ที่จะรวมเข้ากับเซลล์ก่อนหน้า:

```csharp
builder.InsertCell();
builder.CellFormat.HorizontalMerge = CellMerge.Previous;
builder.EndRow();
```

 เซลล์นี้ถูกตั้งค่าให้ผสานกับเซลล์ก่อนหน้าโดยใช้`CellMerge.Previous` - สังเกตว่าเราจบแถวด้วยอย่างไร`builder.EndRow()`.

## ขั้นตอนที่ 4: การแทรกเซลล์ที่ไม่ได้ผสาน

เพื่อแสดงให้เห็นความแตกต่าง ให้แทรกเซลล์ที่ยังไม่ได้ผสานสองสามเซลล์:

```csharp
builder.InsertCell();
builder.CellFormat.HorizontalMerge = CellMerge.None;
builder.Write("Text in one cell.");
builder.InsertCell();
builder.Write("Text in another cell.");
builder.EndRow();
```

ที่นี่ เราแทรกสองเซลล์โดยไม่มีการรวมแนวนอน นี่จะแสดงพฤติกรรมของเซลล์เมื่อไม่ได้เป็นส่วนหนึ่งของลำดับที่ผสานกัน

## ขั้นตอนที่ 5: จบตาราง

ในที่สุด เราก็จบตารางและบันทึกเอกสาร:

```csharp
builder.EndTable();
doc.Save(dataDir + "WorkingWithTables.HorizontalMerge.docx");
```

ข้อมูลโค้ดนี้จะทำให้ตารางสมบูรณ์และบันทึกเอกสารลงในไดเร็กทอรีที่ระบุ
