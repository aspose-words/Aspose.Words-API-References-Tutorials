## การแนะนำ

เคยรู้สึกติดอยู่กับการวางตำแหน่งตารางตามที่คุณต้องการในเอกสาร Word ของคุณหรือไม่? คุณไม่ได้อยู่คนเดียว ไม่ว่าคุณจะสร้างรายงานแบบมืออาชีพหรือโบรชัวร์ที่มีสไตล์ การจัดตารางให้ตรงกันสามารถสร้างโลกแห่งความแตกต่างได้ นั่นคือจุดที่ Aspose.Words สำหรับ .NET มีประโยชน์ บทช่วยสอนนี้จะแนะนำคุณทีละขั้นตอนเกี่ยวกับวิธีตั้งค่าตำแหน่งแนวนอนหรือแนวตั้งสัมพัทธ์สำหรับตารางในเอกสาร Word ของคุณ มาดำน้ำกันเถอะ!

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเริ่ม ตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

1.  Aspose.Words for .NET: หากคุณยังไม่มี คุณสามารถดาวน์โหลดได้[ที่นี่](https://releases.aspose.com/words/net/).
2. สภาพแวดล้อมการพัฒนา: Visual Studio หรือ IDE ที่รองรับ .NET อื่นๆ
3. ความรู้พื้นฐานของ C#: บทช่วยสอนนี้ถือว่าคุณคุ้นเคยกับพื้นฐานของการเขียนโปรแกรม C#

## นำเข้าเนมสเปซ

ก่อนอื่น คุณต้องนำเข้าเนมสเปซที่จำเป็นก่อน นี่เป็นสิ่งจำเป็นสำหรับการเข้าถึงฟังก์ชัน Aspose.Words

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

## ขั้นตอนที่ 1: โหลดเอกสารของคุณ

ในการเริ่มต้น คุณจะต้องโหลดเอกสาร Word ของคุณลงในโปรแกรม ต่อไปนี้คือวิธีที่คุณสามารถทำได้:

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Table wrapped by text.docx");
```

ข้อมูลโค้ดนี้จะกำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณและโหลดเอกสารเฉพาะที่คุณต้องการใช้งาน ตรวจสอบให้แน่ใจว่าเส้นทางเอกสารของคุณถูกต้องเพื่อหลีกเลี่ยงปัญหาในการโหลด

## ขั้นตอนที่ 2: เข้าถึงตาราง

ต่อไปเราต้องเข้าถึงตารางภายในเอกสาร โดยทั่วไปแล้ว คุณจะต้องการทำงานกับตารางแรกในส่วนเนื้อหา

```csharp
Table table = doc.FirstSection.Body.Tables[0];
```

โค้ดบรรทัดนี้จะดึงข้อมูลตารางแรกจากเนื้อหาของเอกสาร หากเอกสารของคุณมีหลายตาราง คุณสามารถปรับดัชนีให้เหมาะสมได้

## ขั้นตอนที่ 3: ตั้งค่าตำแหน่งแนวนอน

ตอนนี้ เรามาตั้งค่าตำแหน่งแนวนอนของตารางโดยสัมพันธ์กับองค์ประกอบเฉพาะกัน ในตัวอย่างนี้ เราจะวางตำแหน่งให้สัมพันธ์กับคอลัมน์

```csharp
table.HorizontalAnchor = RelativeHorizontalPosition.Column;
```

 โดยการตั้งค่า`HorizontalAnchor` ถึง`RelativeHorizontalPosition.Column`คุณกำลังบอกให้ตารางจัดตำแหน่งในแนวนอนโดยสัมพันธ์กับคอลัมน์ที่อยู่

## ขั้นตอนที่ 4: ตั้งค่าตำแหน่งแนวตั้ง

เช่นเดียวกับการวางตำแหน่งแนวนอน คุณสามารถกำหนดตำแหน่งแนวตั้งได้ ที่นี่เราวางตำแหน่งให้สัมพันธ์กับหน้า

```csharp
table.VerticalAnchor = RelativeVerticalPosition.Page;
```

 การตั้งค่า`VerticalAnchor` ถึง`RelativeVerticalPosition.Page` รับรองว่าตารางจะจัดแนวตั้งตามหน้า

## ขั้นตอนที่ 5: บันทึกเอกสารของคุณ

สุดท้าย ให้บันทึกการเปลี่ยนแปลงของคุณลงในเอกสารใหม่ นี่เป็นขั้นตอนสำคัญเพื่อให้แน่ใจว่าการเปลี่ยนแปลงของคุณจะถูกรักษาไว้

```csharp
doc.Save(dataDir + "WorkingWithTables.SetFloatingTablePosition.docx");
```

คำสั่งนี้จะบันทึกเอกสารที่แก้ไขด้วยชื่อใหม่ เพื่อให้มั่นใจว่าคุณจะไม่เขียนทับไฟล์ต้นฉบับ
