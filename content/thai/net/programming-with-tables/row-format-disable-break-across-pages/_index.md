## การแนะนำ

เมื่อทำงานกับตารางในเอกสาร Word คุณอาจต้องการตรวจสอบให้แน่ใจว่าแถวไม่แยกข้ามหน้า ซึ่งอาจจำเป็นสำหรับการรักษาความสามารถในการอ่านและการจัดรูปแบบของเอกสารของคุณ Aspose.Words สำหรับ .NET มอบวิธีง่ายๆ ในการปิดใช้งานการแบ่งแถวระหว่างหน้าต่างๆ

ในบทช่วยสอนนี้ เราจะแนะนำคุณตลอดขั้นตอนการปิดใช้งานตัวแบ่งแถวในหน้าต่างๆ ในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเริ่มต้น ตรวจสอบให้แน่ใจว่าคุณมีข้อกำหนดเบื้องต้นดังต่อไปนี้:
- ติดตั้ง Aspose.Words สำหรับไลบรารี .NET แล้ว
- เอกสาร Word ที่มีตารางซึ่งครอบคลุมหลายหน้า

## นำเข้าเนมสเปซ

ขั้นแรก นำเข้าเนมสเปซที่จำเป็นในโครงการของคุณ:

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

## ขั้นตอนที่ 1: โหลดเอกสาร

โหลดเอกสารที่มีตารางซึ่งครอบคลุมหลายหน้า

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Table spanning two pages.docx");
```

## ขั้นตอนที่ 2: เข้าถึงตาราง

เข้าถึงตารางแรกในเอกสาร ซึ่งจะถือว่าตารางที่คุณต้องการแก้ไขเป็นตารางแรกในเอกสาร

```csharp
Table table = (Table) doc.GetChild(NodeType.Table, 0, true);
```

## ขั้นตอนที่ 3: ปิดการใช้งานการแบ่งข้ามหน้าสำหรับทุกแถว

 วนซ้ำแต่ละแถวในตารางแล้วตั้งค่า`AllowBreakAcrossPages`ทรัพย์สินเพื่อ`false`- เพื่อให้แน่ใจว่าแถวจะไม่แตกข้ามหน้าต่างๆ

```csharp
// ปิดใช้การแยกหน้าสำหรับแถวทั้งหมดในตาราง
foreach (Row row in table.Rows)
    row.RowFormat.AllowBreakAcrossPages = false;
```

## ขั้นตอนที่ 4: บันทึกเอกสาร

บันทึกเอกสารที่แก้ไขไปยังไดเร็กทอรีที่คุณระบุ

```csharp
doc.Save(dataDir + "WorkingWithTables.RowFormatDisableBreakAcrossPages.docx");
```

## บทสรุป

ในบทช่วยสอนนี้ เราได้สาธิตวิธีปิดการใช้งานตัวแบ่งแถวในหน้าต่างๆ ในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET ด้วยการทำตามขั้นตอนที่อธิบายไว้ข้างต้น คุณสามารถมั่นใจได้ว่าแถวในตารางของคุณยังคงสภาพเดิมและไม่แยกออกเป็นหน้าต่างๆ โดยยังคงรักษาความสามารถในการอ่านและการจัดรูปแบบของเอกสารไว้

## คำถามที่พบบ่อย

### ฉันสามารถปิดการใช้งานการแบ่งแถวข้ามหน้าสำหรับแถวใดแถวหนึ่งแทนที่จะใช้ทุกแถวได้หรือไม่  
 ได้ คุณสามารถปิดใช้งานการแบ่งแถวสำหรับแถวที่ต้องการได้โดยการเข้าถึงแถวที่ต้องการและตั้งค่า`AllowBreakAcrossPages`ทรัพย์สินเพื่อ`false`.

### วิธีนี้ใช้ได้กับตารางที่มีเซลล์ที่ผสานแล้วหรือไม่  
 ใช่ วิธีนี้ใช้ได้กับตารางที่มีเซลล์ที่ผสานกัน ทรัพย์สิน`AllowBreakAcrossPages` ใช้กับทั้งแถว โดยไม่คำนึงถึงการรวมเซลล์

### วิธีนี้จะได้ผลหรือไม่หากตารางซ้อนอยู่ในตารางอื่น  
ใช่ คุณสามารถเข้าถึงและแก้ไขตารางที่ซ้อนกันได้ในลักษณะเดียวกัน ตรวจสอบให้แน่ใจว่าคุณอ้างอิงตารางที่ซ้อนกันอย่างถูกต้องตามดัชนีหรือคุณสมบัติอื่นๆ

### ฉันจะตรวจสอบได้อย่างไรว่าแถวอนุญาตให้แบ่งหน้าต่างๆ ได้  
 คุณสามารถตรวจสอบว่าแถวอนุญาตให้แบ่งหน้าต่างๆ ได้หรือไม่โดยเข้าไปที่`AllowBreakAcrossPages` ทรัพย์สินของ`RowFormat` และตรวจสอบมูลค่าของมัน

### มีวิธีใช้การตั้งค่านี้กับตารางทั้งหมดในเอกสารหรือไม่?  
ได้ คุณสามารถวนซ้ำตารางทั้งหมดในเอกสารและใช้การตั้งค่านี้กับแต่ละตารางได้
