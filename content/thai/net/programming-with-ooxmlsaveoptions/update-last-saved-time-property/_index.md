## การแนะนำ

เคยสงสัยบ้างไหมว่าจะติดตามคุณสมบัติเวลาที่บันทึกไว้ล่าสุดในเอกสาร Word ของคุณโดยทางโปรแกรมได้อย่างไร? หากคุณกำลังจัดการกับเอกสารหลายฉบับและจำเป็นต้องรักษาข้อมูลเมตาของเอกสารเหล่านั้น การอัปเดตคุณสมบัติเวลาที่บันทึกไว้ล่าสุดอาจมีประโยชน์มาก วันนี้ ฉันจะแนะนำคุณตลอดกระบวนการนี้โดยใช้ Aspose.Words สำหรับ .NET ดังนั้น รัดเข็มขัดแล้วดำดิ่งลงไปได้เลย!

## นำเข้าเนมสเปซ

ขั้นแรก ตรวจสอบให้แน่ใจว่าได้นำเข้าเนมสเปซที่จำเป็นลงในโปรเจ็กต์ของคุณ ซึ่งจะช่วยให้คุณสามารถเข้าถึงคลาสและวิธีการที่จำเป็นสำหรับการจัดการเอกสาร Word

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

ตอนนี้ เรามาแบ่งกระบวนการออกเป็นขั้นตอนง่ายๆ กัน แต่ละขั้นตอนจะแนะนำคุณตลอดกระบวนการอัปเดตคุณสมบัติเวลาที่บันทึกไว้ล่าสุดในเอกสาร Word ของคุณ

## ขั้นตอนที่ 1: ตั้งค่าไดเร็กทอรีเอกสารของคุณ

ขั้นแรก คุณต้องระบุเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือที่จัดเก็บเอกสารที่มีอยู่ของคุณและที่ที่เอกสารที่อัปเดตจะถูกบันทึก

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงไปยังไดเร็กทอรีของคุณ

## ขั้นตอนที่ 2: โหลดเอกสาร Word ของคุณ

 จากนั้นโหลดเอกสาร Word ที่คุณต้องการอัปเดต คุณสามารถทำได้โดยการสร้างอินสแตนซ์ของ`Document` คลาสและผ่านเส้นทางของเอกสารของคุณ

```csharp
Document doc = new Document(dataDir + "Document.docx");
```

 ตรวจสอบให้แน่ใจว่าเอกสารชื่อ`Document.docx` มีอยู่ในไดเร็กทอรีที่ระบุ

## ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการบันทึก

 ตอนนี้สร้างอินสแตนซ์ของ`OoxmlSaveOptions` ระดับ. คลาสนี้อนุญาตให้คุณระบุตัวเลือกสำหรับการบันทึกเอกสารของคุณในรูปแบบ Office Open XML (OOXML) ที่นี่คุณจะตั้งค่า`UpdateLastSavedTimeProperty` ถึง`true`.

```csharp
OoxmlSaveOptions saveOptions = new OoxmlSaveOptions
{
    UpdateLastSavedTimeProperty = true
};
```

สิ่งนี้จะบอก Aspose.Words ให้อัปเดตคุณสมบัติเวลาที่บันทึกไว้ล่าสุดของเอกสาร

## ขั้นตอนที่ 4: บันทึกเอกสารที่อัปเดต

 สุดท้ายให้บันทึกเอกสารโดยใช้ไฟล์`Save` วิธีการของ`Document` คลาส โดยส่งผ่านเส้นทางที่คุณต้องการบันทึกเอกสารที่อัพเดตและตัวเลือกการบันทึก

```csharp
doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.UpdateLastSavedTimeProperty.docx", saveOptions);
```

การดำเนินการนี้จะบันทึกเอกสารด้วยคุณสมบัติเวลาที่บันทึกไว้ล่าสุดที่อัปเดต
