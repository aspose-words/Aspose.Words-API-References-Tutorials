## การแนะนำ

เคยสงสัยไหมว่าจะติดตามคุณสมบัติเวลาที่บันทึกไว้ล่าสุดในเอกสาร Word ของคุณผ่านโปรแกรมได้อย่างไร หากคุณต้องจัดการกับเอกสารหลายฉบับและจำเป็นต้องดูแลข้อมูลเมตาของเอกสารเหล่านั้น การอัปเดตคุณสมบัติเวลาที่บันทึกไว้ล่าสุดอาจมีประโยชน์มาก วันนี้ ฉันจะอธิบายกระบวนการนี้ให้คุณฟังโดยใช้ Aspose.Words สำหรับ .NET ดังนั้น เตรียมตัวให้พร้อมแล้วมาเริ่มกันเลย!

3. ความรู้พื้นฐานเกี่ยวกับ C#: การทำความเข้าใจพื้นฐานของการเขียนโปรแกรม C# จะเป็นประโยชน์

## นำเข้าเนมสเปซ

ในการเริ่มต้น โปรดแน่ใจว่าได้นำเข้าเนมสเปซที่จำเป็นลงในโปรเจ็กต์ของคุณ ซึ่งจะทำให้คุณสามารถเข้าถึงคลาสและวิธีการที่จำเป็นสำหรับการจัดการเอกสาร Word ได้

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

ตอนนี้เรามาแบ่งกระบวนการออกเป็นขั้นตอนง่ายๆ กัน แต่ละขั้นตอนจะแนะนำคุณตลอดกระบวนการอัปเดตคุณสมบัติเวลาที่บันทึกไว้ล่าสุดในเอกสาร Word ของคุณ

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ

ขั้นแรก คุณต้องระบุเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือที่ที่คุณเก็บเอกสารที่มีอยู่และที่ที่จะบันทึกเอกสารที่อัปเดต

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงไปยังไดเร็กทอรีของคุณ

## ขั้นตอนที่ 2: โหลดเอกสาร Word ของคุณ

 ขั้นตอนต่อไปคือโหลดเอกสาร Word ที่คุณต้องการอัปเดต คุณสามารถทำได้โดยสร้างอินสแตนซ์ของเอกสาร`Document` คลาสและการส่งผ่านเส้นทางของเอกสารของคุณ

```csharp
Document doc = new Document(dataDir + "Document.docx");
```

 ให้แน่ใจว่าเอกสารที่ตั้งชื่อ`Document.docx` มีอยู่ในไดเร็กทอรีที่ระบุ

## ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการบันทึก

 ตอนนี้สร้างอินสแตนซ์ของ`OoxmlSaveOptions` คลาส คลาสนี้ช่วยให้คุณสามารถระบุตัวเลือกสำหรับการบันทึกเอกสารของคุณในรูปแบบ Office Open XML (OOXML) ที่นี่ คุณจะตั้งค่า`UpdateLastSavedTimeProperty` ถึง`true`.

```csharp
OoxmlSaveOptions saveOptions = new OoxmlSaveOptions
{
    UpdateLastSavedTimeProperty = true
};
```

นี่จะบอก Aspose.Words ให้อัปเดตคุณสมบัติเวลาที่บันทึกล่าสุดของเอกสาร

## ขั้นตอนที่ 4: บันทึกเอกสารที่อัปเดต

 สุดท้ายให้บันทึกเอกสารโดยใช้`Save` วิธีการของ`Document` คลาส โดยผ่านเส้นทางที่คุณต้องการบันทึกเอกสารที่อัพเดต และตัวเลือกการบันทึก

```csharp
doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.UpdateLastSavedTimeProperty.docx", saveOptions);
```

การดำเนินการนี้จะบันทึกเอกสารพร้อมคุณสมบัติเวลาที่บันทึกล่าสุดที่อัปเดต
