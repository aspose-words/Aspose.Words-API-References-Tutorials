## นำเข้าเนมสเปซ

ก่อนอื่น มานำเข้าเนมสเปซที่จำเป็นกันก่อน นี่คือจุดเริ่มต้นของความมหัศจรรย์ทั้งหมด

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
using System;
```

## ขั้นตอนที่ 1: การตั้งค่าไดเร็กทอรีเอกสาร

เอาล่ะ เริ่มต้นด้วยการตั้งค่าเส้นทางไปยังไดเร็กทอรีเอกสารของเรา นี่คือที่ที่เอกสาร Word ของคุณจะถูกบันทึกไว้

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` ด้วยเส้นทางจริงที่คุณต้องการบันทึกเอกสารของคุณ

## ขั้นตอนที่ 2: สร้างเอกสารใหม่

 ต่อไปเราต้องสร้างอินสแตนซ์ใหม่ของ`Document` ชั้นเรียน ให้คิดว่านี่เป็นผืนผ้าใบเปล่าๆ ที่คุณสามารถเริ่มเพิ่มเนื้อหาได้

```csharp
Document doc = new Document();
```

## ขั้นตอนที่ 3: การเริ่มต้น DocumentBuilder

 ที่`DocumentBuilder` ก็เหมือนกับพู่กันของคุณ ช่วยให้คุณเพิ่มเนื้อหาลงในเอกสารของคุณ มาเริ่มกันเลย

```csharp
DocumentBuilder builder = new DocumentBuilder(doc);
```

## ขั้นตอนที่ 4: การเขียนเนื้อหาไปยังหน้าแรก

มาเพิ่มเนื้อหาบางส่วนในหน้าแรกกัน นี่คือที่ที่คุณสามารถสร้างสรรค์ได้

```csharp
builder.Writeln("This is page 1.");
```

## ขั้นตอนที่ 5: การแทรกตัวแบ่งหน้า

 ตอนนี้ส่วนที่สนุกมา เราจำเป็นต้องแทรกตัวแบ่งหน้าเพื่อย้ายไปยังหน้าถัดไป มันง่ายเหมือนการโทร`InsertBreak` วิธี.

```csharp
builder.InsertBreak(BreakType.PageBreak);
```

## ขั้นตอนที่ 6: การเขียนเนื้อหาไปยังหน้าสอง

หลังจากแทรกตัวแบ่งหน้าแล้ว ให้เพิ่มเนื้อหาบางส่วนลงในหน้าที่สอง

```csharp
builder.Writeln("This is page 2.");
```

## ขั้นตอนที่ 7: การแทรกตัวแบ่งหน้าอื่น

ลองแทรกตัวแบ่งหน้าอื่นเพื่อย้ายไปยังหน้าที่สาม

```csharp
builder.InsertBreak(BreakType.PageBreak);
```

## ขั้นตอนที่ 8: การเขียนเนื้อหาไปยังหน้าที่สาม

สุดท้ายนี้ มาเพิ่มเนื้อหาบางส่วนในหน้าที่สามกัน

```csharp
builder.Writeln("This is page 3.");
```

## ขั้นตอนที่ 9: บันทึกเอกสาร

สุดท้ายแต่ไม่ท้ายสุด เราต้องบันทึกเอกสารของเรา นี่คือที่ที่การทำงานหนักทั้งหมดของคุณมารวมกัน

```csharp
doc.Save(dataDir + "AddContentUsingDocumentBuilder.InsertBreak.docx");
```
