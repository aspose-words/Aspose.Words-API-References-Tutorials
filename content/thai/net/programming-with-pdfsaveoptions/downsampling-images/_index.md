ในบทช่วยสอนนี้ เราจะแนะนำคุณตลอดขั้นตอนในการลดขนาดเอกสาร PDF ด้วยการลดขนาดรูปภาพเมื่อแปลงเป็น PDF ด้วย Aspose.Words สำหรับ .NET ซึ่งจะช่วยลดขนาดของไฟล์ PDF ที่สร้างขึ้น ทำตามขั้นตอนด้านล่าง:

## ขั้นตอนที่ 1: กำลังโหลดเอกสาร

เริ่มต้นด้วยการอัปโหลดเอกสารที่คุณต้องการแปลงเป็น PDF:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document doc = new Document(dataDir + "Rendering.docx");
```

อย่าลืมระบุเส้นทางที่ถูกต้องไปยังเอกสารของคุณ

## ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการบันทึก PDF

สร้างอินสแตนซ์ของคลาส PdfSaveOptions และตั้งค่าตัวเลือกการลดขนาดรูปภาพ:

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions
{
DownsampleOptions = { Resolution = 36, ResolutionThreshold = 128 }
};
```

 ที่`Resolution` คุณสมบัติระบุความละเอียดเป้าหมายของภาพและ`ResolutionThreshold`คุณสมบัติระบุความละเอียดขั้นต่ำด้านล่างซึ่งรูปภาพจะไม่ถูกลดขนาดลง

## ขั้นตอนที่ 3: แปลงเอกสารเป็น PDF

 ใช้`Save` วิธีการแปลงเอกสารเป็น PDF โดยระบุตัวเลือกการบันทึก:

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.DownsamplingImages.pdf", saveOptions);
```

ตรวจสอบให้แน่ใจว่าได้ระบุเส้นทางที่ถูกต้องเพื่อบันทึก PDF ที่แปลงแล้ว

### ตัวอย่างซอร์สโค้ดสำหรับการลดขนาดรูปภาพโดยใช้ Aspose.Words สำหรับ .NET

```csharp

	// เส้นทางไปยังไดเร็กทอรีเอกสาร
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Rendering.docx");

	// เราสามารถกำหนดเกณฑ์ขั้นต่ำสำหรับการสุ่มตัวอย่างได้
	// ค่านี้จะป้องกันรูปภาพที่สองในเอกสารอินพุตจากการสุ่มตัวอย่าง
	PdfSaveOptions saveOptions = new PdfSaveOptions
	{
		DownsampleOptions = { Resolution = 36, ResolutionThreshold = 128 }
	};

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.DownsamplingImages.pdf", saveOptions);

```

ด้วยการทำตามขั้นตอนเหล่านี้ คุณสามารถลดความละเอียดของภาพได้อย่างง่ายดายเมื่อแปลงเป็น PDF ด้วย Aspose.Words สำหรับ .NET

## บทสรุป

ในบทช่วยสอนนี้ เราได้อธิบายวิธีลดขนาดของเอกสาร PDF ด้วยการสุ่มตัวอย่างรูปภาพเมื่อแปลงเป็น PDF โดยใช้ Aspose.Words สำหรับ .NET ด้วยการทำตามขั้นตอนที่อธิบายไว้ คุณสามารถลดความละเอียดของภาพและขนาดของไฟล์ PDF ที่สร้างขึ้นได้อย่างง่ายดาย อย่าลืมระบุเส้นทางที่ถูกต้องไปยังเอกสารของคุณและกำหนดค่าตัวเลือกการสุ่มตัวอย่างรูปภาพตามความจำเป็น การลดขนาดไฟล์ PDF ช่วยให้แชร์ จัดเก็บ และโหลดไฟล์บนแพลตฟอร์มต่างๆ ได้ง่ายขึ้น เพลิดเพลินกับคุณประโยชน์ของการลดขนาดเอกสาร PDF ด้วยการสุ่มตัวอย่างภาพโดยใช้ Aspose.Words สำหรับ .NET

 กำหนดเส้นทางไดเร็กทอรีที่มีเอกสารของคุณอยู่โดยการแทนที่`"YOUR DOCUMENTS DIRECTORY"` ด้วยเส้นทางจริงของไดเร็กทอรีเอกสารของคุณ

 โหลดเอกสารที่คุณต้องการแปลงเป็น PDF โดยใช้`Document` และระบุเส้นทางไปยังเอกสารในไดเร็กทอรีเอกสารที่ระบุ

 กำหนดค่าตัวเลือกบันทึกเป็น PDF โดยการสร้างอินสแตนซ์ของ`PdfSaveOptions` และการตั้งค่าตัวเลือกการสุ่มตัวอย่างภาพโดยใช้`DownsampleOptions` คุณสมบัติ. คุณสามารถระบุความละเอียดเป้าหมายของรูปภาพได้โดยใช้`Resolution` คุณสมบัติและตั้งค่าเกณฑ์ความละเอียดขั้นต่ำซึ่งรูปภาพจะไม่ถูกลดขนาดลงโดยใช้`ResolutionThreshold` คุณสมบัติ.

 บันทึกเอกสารในรูปแบบ PDF โดยใช้ไฟล์`Save` วิธีการของ`Document` คลาสที่ระบุเส้นทางและตัวเลือกการบันทึก

ตอบ: ข้อดีของการลดขนาดเอกสาร PDF ด้วยการสุ่มตัวอย่างรูปภาพคือ:

การเพิ่มประสิทธิภาพพื้นที่จัดเก็บข้อมูล: การลดขนาดไฟล์ PDF ช่วยเพิ่มประสิทธิภาพการใช้พื้นที่จัดเก็บข้อมูล โดยเฉพาะอย่างยิ่งเมื่อคุณมีไฟล์ PDF จำนวนมากที่มีภาพความละเอียดสูง
