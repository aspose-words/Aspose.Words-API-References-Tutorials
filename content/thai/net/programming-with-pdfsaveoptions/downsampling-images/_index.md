## การแนะนำ

PDF เป็นเนื้อหาหลักในโลกดิจิทัล ซึ่งใช้สำหรับทุกสิ่งตั้งแต่การแชร์เอกสารไปจนถึงการสร้าง eBook อย่างไรก็ตาม บางครั้งขนาดอาจเป็นอุปสรรค โดยเฉพาะอย่างยิ่งเมื่อต้องรับมือกับเนื้อหาที่มีรูปภาพมากมาย นี่คือที่มาของภาพการสุ่มตัวอย่างต่ำ ด้วยการลดความละเอียดของรูปภาพภายใน PDF คุณสามารถลดขนาดไฟล์ได้อย่างมากโดยไม่กระทบต่อคุณภาพมากเกินไป ในบทช่วยสอนนี้ เราจะอธิบายขั้นตอนต่าง ๆ เพื่อให้บรรลุเป้าหมายนี้โดยใช้ Aspose.Words สำหรับ .NET

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะพูดถึงโค้ด โปรดตรวจสอบให้แน่ใจว่าคุณมีทุกสิ่งที่คุณต้องการ:

1.  Aspose.Words สำหรับ .NET: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้งไลบรารี Aspose.Words แล้ว ถ้าไม่คุณสามารถดาวน์โหลดได้[ที่นี่](https://releases.aspose.com/words/net/).
2. สภาพแวดล้อมการพัฒนา: สภาพแวดล้อมการพัฒนา .NET เช่น Visual Studio
3. ความรู้พื้นฐานของ C#: การทำความเข้าใจพื้นฐานของการเขียนโปรแกรม C# จะเป็นประโยชน์
4.  เอกสารตัวอย่าง: เอกสาร Word (เช่น`Rendering.docx`) พร้อมรูปภาพที่จะแปลงเป็น PDF

## นำเข้าเนมสเปซ

ก่อนอื่น คุณต้องนำเข้าเนมสเปซที่จำเป็นก่อน เพิ่มสิ่งเหล่านี้ที่ด้านบนของไฟล์โค้ดของคุณ:

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

ตอนนี้ เรามาแบ่งกระบวนการออกเป็นขั้นตอนที่สามารถจัดการได้

## ขั้นตอนที่ 1: โหลดเอกสาร

ขั้นตอนแรกคือการโหลดเอกสาร Word ของคุณ นี่คือที่ที่คุณระบุเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Rendering.docx");
```

ในขั้นตอนนี้ เรากำลังโหลดเอกสาร Word จากไดเร็กทอรีที่ระบุ ตรวจสอบให้แน่ใจว่าได้เปลี่ยน`"YOUR DOCUMENT DIRECTORY"`พร้อมเส้นทางจริงที่เอกสารของคุณตั้งอยู่

## ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการลดขนาดตัวอย่าง

ต่อไป เราต้องกำหนดค่าตัวเลือกการลดขนาดตัวอย่าง สิ่งนี้เกี่ยวข้องกับการตั้งค่าความละเอียดและเกณฑ์ความละเอียดสำหรับรูปภาพ

```csharp
// เราสามารถกำหนดเกณฑ์ขั้นต่ำสำหรับการสุ่มตัวอย่างได้
// ค่านี้จะป้องกันรูปภาพที่สองในเอกสารอินพุตจากการสุ่มตัวอย่าง
PdfSaveOptions saveOptions = new PdfSaveOptions
{
    DownsampleOptions = { Resolution = 36, ResolutionThreshold = 128 }
};
```

 ที่นี่ เรากำลังสร้างอินสแตนซ์ใหม่ของ`PdfSaveOptions` และการตั้งค่า`Resolution` ถึง 36 DPI และ`ResolutionThreshold` ถึง 128 จุดต่อนิ้ว ซึ่งหมายความว่ารูปภาพใดๆ ที่มีความละเอียดสูงกว่า 128 DPI จะถูกลดขนาดลงเป็น 36 DPI

## ขั้นตอนที่ 3: บันทึกเอกสารเป็น PDF

สุดท้ายนี้ เราจะบันทึกเอกสารเป็น PDF พร้อมตัวเลือกที่กำหนดค่าไว้

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.DownsamplingImages.pdf", saveOptions);
```

ในขั้นตอนสุดท้ายนี้ เรากำลังบันทึกเอกสารเป็น PDF ในไดเรกทอรีเดียวกันกับตัวเลือกการสุ่มตัวอย่างที่ระบุ

## บทสรุป

และคุณก็ได้แล้ว! คุณได้ลดขนาด PDF ของคุณสำเร็จด้วยการลดขนาดรูปภาพโดยใช้ Aspose.Words สำหรับ .NET สิ่งนี้ไม่เพียงทำให้ PDF ของคุณสามารถจัดการได้มากขึ้น แต่ยังช่วยในการอัพโหลด ดาวน์โหลด และประสบการณ์การรับชมที่ราบรื่นยิ่งขึ้นอีกด้วย

## คำถามที่พบบ่อย

### การลดขนาดตัวอย่างคืออะไร?
การสุ่มตัวอย่างเป็นกระบวนการลดความละเอียดของภาพ ซึ่งจะช่วยลดขนาดไฟล์ของเอกสารที่มีภาพเหล่านั้น

### การสุ่มตัวอย่างต่ำจะส่งผลต่อคุณภาพของภาพหรือไม่
ใช่ การสุ่มตัวอย่างต่ำจะลดคุณภาพของภาพ อย่างไรก็ตามผลกระทบจะขึ้นอยู่กับระดับของการลดความละเอียด เป็นการแลกเปลี่ยนระหว่างขนาดไฟล์และคุณภาพของภาพ

### ฉันสามารถเลือกรูปภาพที่จะลดขนาดตัวอย่างได้หรือไม่
 ใช่โดยการตั้งค่า`ResolutionThreshold`คุณสามารถควบคุมได้ว่ารูปภาพใดจะถูกลดขนาดลงตามความละเอียดดั้งเดิมของรูปภาพนั้น

### ความละเอียดที่เหมาะสมที่สุดสำหรับการลดขนาดตัวอย่างคืออะไร
ความละเอียดที่เหมาะสมที่สุดนั้นขึ้นอยู่กับความต้องการเฉพาะของคุณ โดยทั่วไป 72 DPI จะใช้สำหรับรูปภาพบนเว็บ ในขณะที่ความละเอียดสูงกว่าจะใช้สำหรับคุณภาพการพิมพ์
