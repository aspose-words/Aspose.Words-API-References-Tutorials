ในบทช่วยสอนนี้ เราจะแนะนำคุณตลอดขั้นตอนในการแสดงชื่อเอกสารในแถบหัวเรื่องหน้าต่างด้วย Aspose.Words สำหรับ .NET คุณสมบัตินี้ช่วยให้คุณแสดงชื่อเอกสารในแถบชื่อเรื่องของหน้าต่างเมื่อคุณเปิดเอกสาร PDF ที่สร้างขึ้น ทำตามขั้นตอนด้านล่าง:

## ขั้นตอนที่ 1: กำลังโหลดเอกสาร

เริ่มต้นด้วยการอัปโหลดเอกสารที่คุณต้องการแปลงเป็น PDF:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document doc = new Document(dataDir + "Rendering.docx");
```

อย่าลืมระบุเส้นทางที่ถูกต้องไปยังเอกสารของคุณ

## ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการบันทึก PDF

สร้างอินสแตนซ์ของคลาส PdfSaveOptions และเปิดใช้งานการแสดงชื่อเอกสารในแถบหัวเรื่องของหน้าต่าง:

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions { DisplayDocTitle = true };
```

ตัวเลือกนี้ทำให้สามารถแสดงชื่อเอกสารในแถบชื่อเรื่องของหน้าต่างเมื่อแปลงเป็น PDF

## ขั้นตอนที่ 3: แปลงเอกสารเป็น PDF

 ใช้`Save` วิธีการแปลงเอกสารเป็น PDF โดยระบุตัวเลือกการแปลง:

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.DisplayDocTitleInWindowTitlebar.pdf", saveOptions);
```

ตรวจสอบให้แน่ใจว่าได้ระบุเส้นทางที่ถูกต้องเพื่อบันทึก PDF ที่แปลงแล้ว

### ตัวอย่างซอร์สโค้ดสำหรับแสดงชื่อเอกสารในแถบหัวเรื่องหน้าต่างโดยใช้ Aspose.Words สำหรับ .NET

นี่คือซอร์สโค้ดแบบเต็มเพื่อแสดงชื่อเอกสารในแถบหัวเรื่องหน้าต่างในเอกสาร PDF ด้วย Aspose.Words สำหรับ .NET:

```csharp

	// เส้นทางไปยังไดเร็กทอรีเอกสาร
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Rendering.docx");

	PdfSaveOptions saveOptions = new PdfSaveOptions { DisplayDocTitle = true };

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.DisplayDocTitleInWindowTitlebar.pdf", saveOptions);
        
```
เมื่อทำตามขั้นตอนเหล่านี้ คุณจะสามารถแสดงชื่อเอกสารในแถบชื่อเรื่องของหน้าต่างได้อย่างง่ายดายเมื่อแปลงเป็น PDF ด้วย Aspose.Words สำหรับ .NET
