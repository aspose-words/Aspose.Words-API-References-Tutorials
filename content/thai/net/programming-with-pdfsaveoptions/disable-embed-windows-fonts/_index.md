ในบทช่วยสอนนี้ เราจะแนะนำคุณตลอดขั้นตอนในการลดขนาด PDF ด้วยการปิดใช้งานการฝังแบบอักษร Windows ในเอกสาร PDF ด้วย Aspose.Words สำหรับ .NET เมื่อปิดใช้งานการฝังแบบอักษร คุณสามารถลดขนาดของไฟล์ PDF ที่สร้างขึ้นได้ ทำตามขั้นตอนด้านล่าง:

## ขั้นตอนที่ 1: กำลังโหลดเอกสาร

เริ่มต้นด้วยการอัปโหลดเอกสารที่คุณต้องการแปลงเป็น PDF:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document doc = new Document(dataDir + "Rendering.docx");
```

อย่าลืมระบุเส้นทางที่ถูกต้องไปยังเอกสารของคุณ

## ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการบันทึก PDF

สร้างอินสแตนซ์ของคลาส PdfSaveOptions และระบุวิธีการฝังแบบอักษร:

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions { FontEmbeddingMode = PdfFontEmbeddingMode.EmbedNone };
```

ตัวเลือกนี้ช่วยให้คุณสามารถปิดใช้งานการรวมแบบอักษร Windows ในไฟล์ PDF ที่สร้างขึ้นได้

## ขั้นตอนที่ 3: แปลงเอกสารเป็น PDF

 ใช้`Save` วิธีการแปลงเอกสารเป็น PDF โดยระบุตัวเลือกการแปลง:

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.DisableEmbedWindowsFonts.pdf", saveOptions);
```

ตรวจสอบให้แน่ใจว่าได้ระบุเส้นทางที่ถูกต้องเพื่อบันทึก PDF ที่แปลงแล้ว

### ตัวอย่างซอร์สโค้ดสำหรับการปิดใช้งานแบบอักษร Windows แบบฝังโดยใช้ Aspose.Words สำหรับ .NET

นี่คือซอร์สโค้ดแบบเต็มเพื่อปิดใช้งานการฝังแบบอักษร Windows ในเอกสาร PDF ด้วย Aspose.Words สำหรับ .NET:

```csharp

	// เส้นทางไปยังไดเร็กทอรีเอกสาร
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Rendering.docx");

	// ผลลัพธ์ PDF จะถูกบันทึกโดยไม่ต้องฝังแบบอักษร Windows มาตรฐาน
	PdfSaveOptions saveOptions = new PdfSaveOptions { FontEmbeddingMode = PdfFontEmbeddingMode.EmbedNone };
	
	doc.Save(dataDir + "WorkingWithPdfSaveOptions.DisableEmbedWindowsFonts.pdf", saveOptions);

```
ด้วยการทำตามขั้นตอนเหล่านี้ คุณสามารถปิดใช้งานการฝังแบบอักษร Windows ในเอกสาร PDF ด้วย Aspose.Words for .NET ได้อย่างง่ายดาย
