## ขั้นตอนที่ 2: กำลังโหลดเอกสาร Word

### ขั้นตอนที่ 2.1: ระบุไดเร็กทอรีเอกสาร
กำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือที่ตั้งเอกสาร Word ของคุณ

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงไปยังไดเร็กทอรีของคุณ

### ขั้นตอนที่ 2.2: โหลดเอกสาร
โหลดเอกสาร Word ของคุณลงในโครงการ

```csharp
Document doc = new Document(dataDir + "Content-ID.docx");
```

## ขั้นตอนที่ 3: การกำหนดค่าตัวเลือกการบันทึก HTML

 สร้างอินสแตนซ์ของ`HtmlSaveOptions` เพื่อปรับแต่งวิธีการบันทึกเอกสารของคุณเป็น MHTML

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.Mhtml)
{
    PrettyFormat = true,
    ExportCidUrlsForMhtmlResources = true
};
```

- `SaveFormat.Mhtml` ระบุว่ารูปแบบเอาต์พุตเป็น MHTML
- `PrettyFormat = true` ช่วยให้มั่นใจได้ว่าเอาต์พุตได้รับการจัดรูปแบบอย่างเรียบร้อย
- `ExportCidUrlsForMhtmlResources = true` เปิดใช้งานการส่งออก Cid URL สำหรับทรัพยากร MHTML

### ขั้นตอนที่ 4: บันทึกเอกสารเป็น MHTML

ขั้นตอนที่ 4.1: บันทึกเอกสาร
บันทึกเอกสารของคุณเป็นไฟล์ MHTML โดยใช้ตัวเลือกที่กำหนดค่าไว้

```csharp
doc.Save(dataDir + "WorkingWithHtmlSaveOptions.ExportCidUrlsForMhtmlResources.mhtml", saveOptions);
```
