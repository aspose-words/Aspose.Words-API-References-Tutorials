## การแนะนำ

คุณพร้อมที่จะเรียนรู้ศิลปะในการส่งออก URL ของ Cid สำหรับทรัพยากร MHTML โดยใช้ Aspose.Words สำหรับ .NET หรือยัง ไม่ว่าคุณจะเป็นนักพัฒนาที่มีประสบการณ์หรือเพิ่งเริ่มต้น คู่มือฉบับสมบูรณ์นี้จะแนะนำคุณในทุกขั้นตอน เมื่ออ่านบทความนี้จบ คุณจะเข้าใจอย่างแจ่มชัดว่าต้องจัดการทรัพยากร MHTML ในเอกสาร Word ของคุณอย่างมีประสิทธิภาพอย่างไร มาเริ่มกันเลย!

## ขั้นตอนที่ 2: การโหลดเอกสาร Word

### ขั้นตอนที่ 2.1: ระบุไดเรกทอรีเอกสาร
กำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือตำแหน่งที่เอกสาร Word ของคุณตั้งอยู่

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงไปยังไดเร็กทอรีของคุณ

### ขั้นตอนที่ 2.2: โหลดเอกสาร
โหลดเอกสาร Word ของคุณลงในโครงการ

```csharp
Document doc = new Document(dataDir + "Content-ID.docx");
```

## ขั้นตอนที่ 3: การกำหนดค่าตัวเลือกการบันทึก HTML

 สร้างอินสแตนซ์ของ`HtmlSaveOptions` เพื่อปรับแต่งวิธีการบันทึกเอกสารของคุณเป็น MHTML

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.Mhtml)
{
    PrettyFormat = true,
    ExportCidUrlsForMhtmlResources = true
};
```

- `SaveFormat.Mhtml` ระบุว่ารูปแบบเอาต์พุตเป็น MHTML
- `PrettyFormat = true` ทำให้แน่ใจว่าผลลัพธ์ได้รับการจัดรูปแบบเรียบร้อย
- `ExportCidUrlsForMhtmlResources = true` เปิดใช้งานการส่งออก URL ของ Cid สำหรับทรัพยากร MHTML

### ขั้นตอนที่ 4: บันทึกเอกสารเป็น MHTML

ขั้นตอนที่ 4.1: บันทึกเอกสาร
บันทึกเอกสารของคุณเป็นไฟล์ MHTML โดยใช้ตัวเลือกที่กำหนดค่าไว้

```csharp
doc.Save(dataDir + "WorkingWithHtmlSaveOptions.ExportCidUrlsForMhtmlResources.mhtml", saveOptions);
```
