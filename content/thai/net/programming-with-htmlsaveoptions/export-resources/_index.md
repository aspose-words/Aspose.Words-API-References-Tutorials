## นำเข้าเนมสเปซ

หากต้องการใช้ Aspose.Words สำหรับ .NET คุณต้องรวมเนมสเปซที่เกี่ยวข้องในโครงการของคุณ นี่คือวิธีการ:

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

เนมสเปซเหล่านี้มีความสำคัญอย่างยิ่งต่อการเข้าถึงคลาสและวิธีการของ Aspose.Words ที่เราจะใช้ในบทช่วยสอนของเรา

เรามาแจกแจงขั้นตอนการส่งออกทรัพยากรเมื่อบันทึกเอกสาร Word เป็น HTML เราจะดำเนินการทีละขั้นตอน ดังนั้นจึงง่ายต่อการติดตาม

## ขั้นตอนที่ 1: ตั้งค่าไดเร็กทอรีเอกสารของคุณ

ก่อนอื่น คุณต้องระบุเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือตำแหน่งของเอกสาร Word และตำแหน่งที่จะบันทึกไฟล์ HTML

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงไปยังไดเร็กทอรีของคุณ

## ขั้นตอนที่ 2: โหลดเอกสาร Word

 ต่อไป มาโหลดเอกสาร Word ที่คุณต้องการแปลงเป็น HTML สำหรับบทช่วยสอนนี้ เราจะใช้เอกสารชื่อ`Rendering.docx`.

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

บรรทัดโค้ดนี้โหลดเอกสารจากไดเร็กทอรีที่ระบุ

## ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการบันทึก HTML

หากต้องการส่งออกทรัพยากร เช่น CSS และแบบอักษร คุณต้องกำหนดค่า`HtmlSaveOptions`- ขั้นตอนนี้มีความสำคัญอย่างยิ่งในการตรวจสอบให้แน่ใจว่าเอาต์พุต HTML ของคุณมีโครงสร้างที่ดีและรวมทรัพยากรที่จำเป็น

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions
{
    CssStyleSheetType = CssStyleSheetType.External,
    ExportFontResources = true,
    ResourceFolder = dataDir + "Resources",
    ResourceFolderAlias = "http://example.com/resources"
};
```

เรามาดูรายละเอียดว่าแต่ละตัวเลือกทำอะไรได้บ้าง:
- `CssStyleSheetType = CssStyleSheetType.External`: ตัวเลือกนี้ระบุว่าควรบันทึกสไตล์ CSS ไว้ในสไตล์ชีตภายนอก
- `ExportFontResources = true`: สิ่งนี้ทำให้สามารถส่งออกทรัพยากรแบบอักษรได้
- `ResourceFolder = dataDir + "Resources"`: ระบุโฟลเดอร์ในเครื่องที่จะบันทึกทรัพยากร (เช่น แบบอักษรและไฟล์ CSS)
- `ResourceFolderAlias = "http://example.com/resources"`: ตั้งชื่อแทนสำหรับโฟลเดอร์ทรัพยากรซึ่งจะใช้ในไฟล์ HTML

## ขั้นตอนที่ 4: บันทึกเอกสารเป็น HTML

เมื่อกำหนดค่าตัวเลือกการบันทึกแล้ว ขั้นตอนสุดท้ายคือการบันทึกเอกสารเป็นไฟล์ HTML นี่คือวิธีการ:

```csharp
doc.Save(dataDir + "WorkingWithHtmlSaveOptions.ExportResources.html", saveOptions);
```

บรรทัดโค้ดนี้จะบันทึกเอกสารในรูปแบบ HTML พร้อมกับทรัพยากรที่ส่งออก

## คำถามที่พบบ่อย

### ฉันสามารถส่งออกรูปภาพพร้อมกับแบบอักษรและ CSS ได้หรือไม่
 ใช่คุณสามารถ! Aspose.Words สำหรับ .NET รองรับการส่งออกรูปภาพเช่นกัน เพียงตรวจสอบให้แน่ใจว่าได้กำหนดค่า`HtmlSaveOptions` ตามนั้น

### มีวิธีฝัง CSS แทนการใช้สไตล์ชีตภายนอกหรือไม่?
 อย่างแน่นอน. คุณสามารถตั้งค่าได้`CssStyleSheetType` ถึง`CssStyleSheetType.Embedded` หากคุณต้องการสไตล์แบบฝัง

### ฉันจะปรับแต่งชื่อไฟล์ HTML เอาต์พุตได้อย่างไร
 คุณสามารถระบุชื่อไฟล์ใด ๆ ที่คุณต้องการใน`doc.Save` วิธี. ตัวอย่างเช่น,`doc.Save(dataDir + "CustomFileName.html", saveOptions);`.

 ใช่ รองรับรูปแบบต่างๆ รวมถึง PDF, DOCX, TXT และอื่นๆ ตรวจสอบ[เอกสารประกอบ](https://reference.aspose.com/words/net/) สำหรับรายการทั้งหมด
