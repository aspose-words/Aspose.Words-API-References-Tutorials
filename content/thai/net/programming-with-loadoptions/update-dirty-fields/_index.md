เมื่อประมวลผลคำด้วยเอกสาร Word ในแอปพลิเคชัน C# อาจจำเป็นต้องอัปเดตฟิลด์สกปรกเพื่อแสดงค่าล่าสุด ด้วยไลบรารี Aspose.Words สำหรับ .NET คุณสามารถอัปเดตฟิลด์สกปรกในการโหลดเอกสารได้อย่างง่ายดายโดยใช้ LoadOptions ในคำแนะนำทีละขั้นตอนนี้ เราจะแนะนำวิธีใช้ Aspose.Words สำหรับซอร์สโค้ด .NET C# เพื่อโหลดเอกสารโดยการอัปเดตฟิลด์สกปรกโดยใช้ LoadOptions

## ทำความเข้าใจกับไลบรารี Aspose.Words

ก่อนที่จะเจาะลึกโค้ด สิ่งสำคัญคือต้องทำความเข้าใจไลบรารี Aspose.Words สำหรับ .NET Aspose.Words เป็นไลบรารีที่มีประสิทธิภาพในการสร้าง แก้ไข แปลง และปกป้องเอกสาร Word ในแพลตฟอร์มต่างๆ รวมถึง .NET มันมีฟีเจอร์มากมายสำหรับการจัดการเอกสาร เช่น การแทรกข้อความ การเปลี่ยนการจัดรูปแบบ การเพิ่มส่วน และอื่นๆ อีกมากมาย

## การกำหนดค่าตัวเลือกการโหลด

ขั้นตอนแรกคือการกำหนดค่าตัวเลือกการโหลดสำหรับเอกสารของเรา ใช้คลาส LoadOptions เพื่อระบุพารามิเตอร์การโหลด ในกรณีของเรา เราจำเป็นต้องตั้งค่าคุณสมบัติ UpdateDirtyFields ให้เป็นจริงเพื่ออัปเดตฟิลด์สกปรก ต่อไปนี้เป็นวิธีดำเนินการ:

```csharp
LoadOptions loadOptions = new LoadOptions { UpdateDirtyFields = true };
```

เราสร้างออบเจ็กต์ LoadOptions ใหม่และตั้งค่าคุณสมบัติ UpdateDirtyFields ให้เป็นจริงเพื่ออัปเดตฟิลด์สกปรกเมื่อโหลดเอกสาร

## กำลังโหลดเอกสารอัปเดตฟิลด์สกปรก

ตอนนี้เราได้กำหนดค่าตัวเลือกการโหลดแล้ว เราสามารถโหลดเอกสารโดยใช้คลาสเอกสารและระบุตัวเลือกการโหลดได้ นี่คือตัวอย่าง:

```csharp
Document doc = new Document(dataDir + "Dirty field.docx", loadOptions);
```

ในตัวอย่างนี้ เราโหลดเอกสาร "Dirty field.docx" ที่อยู่ในไดเร็กทอรีเอกสารโดยใช้ตัวเลือกการโหลดที่ระบุ

## ตัวอย่างซอร์สโค้ดสำหรับ LoadOptions พร้อมฟังก์ชัน "อัปเดตฟิลด์สกปรก" โดยใช้ Aspose.Words สำหรับ .NET

```csharp
// พาธไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// กำหนดค่าตัวเลือกการโหลดด้วยคุณสมบัติ "อัปเดตฟิลด์สกปรก"
LoadOptions loadOptions = new LoadOptions { UpdateDirtyFields = true };

// โหลดเอกสารโดยการอัปเดตฟิลด์สกปรก
Document doc = new Document(dataDir + "Dirty field.docx", loadOptions);

// บันทึกเอกสาร
doc.Save(dataDir + "WorkingWithLoadOptions.UpdateDirtyFields.docx");
```
