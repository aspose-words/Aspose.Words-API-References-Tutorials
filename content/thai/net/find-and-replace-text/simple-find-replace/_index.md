## นำเข้าเนมสเปซ

ก่อนที่เราจะพูดถึงเนื้อหาสำคัญ คุณต้องนำเข้าเนมสเปซที่จำเป็นลงในโปรเจ็กต์ของคุณก่อน นี่คือวิธีการ:

```csharp
using Aspose.Words;
using Aspose.Words.Replacing;
```

ตอนนี้ เรามาแจกแจงขั้นตอนการค้นหาและแทนที่ข้อความในเอกสาร Word ทีละขั้นตอน แต่ละขั้นตอนจะง่ายและง่ายต่อการปฏิบัติตาม

## ขั้นตอนที่ 1: การตั้งค่าไดเร็กทอรีเอกสารของคุณ

ก่อนอื่น เรามาตั้งค่าเส้นทางไปยังไดเร็กทอรีเอกสารของคุณกันก่อน นี่คือที่ที่เอกสาร Word ของคุณจะถูกบันทึกหลังการแทนที่ข้อความ

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## ขั้นตอนที่ 2: การสร้างเอกสารใหม่

ต่อไป คุณจะสร้างเอกสาร Word ใหม่โดยใช้ Aspose.Words เอกสารนี้จะถูกจัดการเพื่อแสดงฟังก์ชันการค้นหาและแทนที่

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 ที่นี่เราสร้าง`Document` วัตถุและก`DocumentBuilder` วัตถุ. ที่`DocumentBuilder` ช่วยให้เราเขียนข้อความลงในเอกสารของเรา

## ขั้นตอนที่ 3: การเขียนข้อความเริ่มต้น

 ตอนนี้ เรามาเขียนข้อความลงในเอกสารที่เราจะแทนที่ในภายหลัง เราใช้`DocumentBuilder` เพื่อแทรกข้อความ "สวัสดี_CustomerName_-

```csharp
builder.Writeln("Hello _CustomerName_,");
```

เพื่อให้แน่ใจว่าทุกอย่างทำงานได้อย่างถูกต้อง เราจะพิมพ์ข้อความเอกสารต้นฉบับไปยังคอนโซล

```csharp
Console.WriteLine("Original document text: " + doc.Range.Text);
```

## ขั้นตอนที่ 4: การแทนที่ข้อความ

นี่คือจุดที่ความมหัศจรรย์เกิดขึ้น! เราจะแทนที่ "_CustomerName_ " กับ "เจมส์ บอนด์" โดยใช้`Replace` วิธี. 

```csharp
doc.Range.Replace("_CustomerName_", "James Bond", new FindReplaceOptions(FindReplaceDirection.Forward));
```

 ที่นี่,`FindReplaceOptions`ช่วยให้เราสามารถระบุทิศทางของการดำเนินการค้นหาและแทนที่ได้ เราใช้อยู่`FindReplaceDirection.Forward` เพื่อแทนที่ข้อความตั้งแต่ต้นจนจบเอกสาร

## ขั้นตอนที่ 5: ตรวจสอบการเปลี่ยน

เพื่อตรวจสอบว่าการเปลี่ยนทดแทนได้ผล ให้พิมพ์ข้อความเอกสารที่แก้ไขไปยังคอนโซล

```csharp
Console.WriteLine("Document text after replace: " + doc.Range.Text);
```

คุณควรจะเห็นว่า "_CustomerName_" ถูกแทนที่ด้วย "เจมส์ บอนด์"

## ขั้นตอนที่ 6: บันทึกเอกสาร

สุดท้าย ให้บันทึกเอกสารที่แก้ไขไปยังไดเร็กทอรีที่คุณระบุ

```csharp
doc.Save(dataDir + "FindAndReplace.SimpleFindReplace.docx");
```
