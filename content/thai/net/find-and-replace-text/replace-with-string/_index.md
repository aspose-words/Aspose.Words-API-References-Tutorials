ในบทความนี้ เราจะสำรวจซอร์สโค้ด C# ด้านบนเพื่อทำความเข้าใจวิธีใช้ฟังก์ชันแทนที่ด้วยสตริงในไลบรารี Aspose.Words สำหรับ .NET คุณลักษณะนี้ช่วยให้คุณสามารถแทนที่ข้อความตามสตริงอักขระเฉพาะในเอกสาร Word

## ข้อกำหนดเบื้องต้น

- ความรู้พื้นฐานของภาษา C#
- สภาพแวดล้อมการพัฒนา .NET ที่ติดตั้งไลบรารี Aspose.Words

## ขั้นตอนที่ 1: สร้างเอกสารใหม่

 ก่อนที่เราจะเริ่มใช้การแทนที่สตริง เราจำเป็นต้องสร้างเอกสารใหม่โดยใช้ Aspose.Words สำหรับ .NET ซึ่งสามารถทำได้โดยการยกตัวอย่าง a`Document` วัตถุ:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
```

## ขั้นตอนที่ 2: แทรกข้อความลงในเอกสาร

 เมื่อได้เอกสารแล้ว เราก็สามารถแทรกข้อความโดยใช้ a`DocumentBuilder` วัตถุ วัตถุ ในตัวอย่างของเรา เราใช้`Writeln` วิธีแทรกวลี “เศร้า บ้า แย่”:

```csharp
DocumentBuilder builder = new DocumentBuilder(doc);
builder.Writeln("sad mad bad");
```

## ขั้นตอนที่ 3: แทนที่ด้วยสตริง

 เราใช้`Range.Replace`วิธีการแทนที่ข้อความด้วยสตริง ในตัวอย่างของเรา เราแทนที่คำว่า "sad" ทุกรายการด้วย "bad" โดยใช้ the`FindReplaceOptions` ตัวเลือกด้วย`FindReplaceDirection.Forward` ทิศทางการค้นหา:

```csharp
doc.Range.Replace("sad", "bad", new FindReplaceOptions(FindReplaceDirection.Forward));
```

## ขั้นตอนที่ 4: บันทึกเอกสารที่แก้ไข

สุดท้าย เราจะบันทึกเอกสารที่แก้ไขไปยังไดเร็กทอรีที่ระบุโดยใช้`Save` วิธี:

```csharp
doc.Save(dataDir + "FindAndReplace.ReplaceWithString.docx");
```

### ตัวอย่างซอร์สโค้ดสำหรับการแทนที่ด้วยสตริงโดยใช้ Aspose.Words สำหรับ .NET

นี่คือตัวอย่างซอร์สโค้ดแบบเต็มเพื่อแสดงการใช้การแทนที่ด้วยสตริงอักขระด้วย Aspose.Words สำหรับ .NET:

```csharp

	// เส้นทางไปยังไดเร็กทอรีเอกสาร
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	
	builder.Writeln("sad mad bad");

	doc.Range.Replace("sad", "bad", new FindReplaceOptions(FindReplaceDirection.Forward));

	doc.Save(dataDir + "FindAndReplace.ReplaceWithString.docx");
  
```
