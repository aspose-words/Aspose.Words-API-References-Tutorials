## การแนะนำ

เคยพบว่าตัวเองประสบปัญหาในการจัดการแบบอักษรในเอกสาร Word ของคุณหรือไม่? หากคุณเป็นนักพัฒนา .NET Aspose.Words สำหรับ .NET พร้อมช่วยเหลือคุณแล้ว! ไลบรารีอันทรงพลังนี้ไม่เพียงช่วยให้คุณสร้างและจัดการเอกสาร Word โดยทางโปรแกรมเท่านั้น แต่ยังมอบความสามารถในการจัดการแบบอักษรที่ครอบคลุมอีกด้วย ในคู่มือนี้ เราจะอธิบายบทช่วยสอนทีละขั้นตอนเกี่ยวกับวิธีรับรายการแบบอักษรที่พร้อมใช้งานโดยใช้ Aspose.Words สำหรับ .NET เราจะแบ่งขั้นตอนออกเป็นขั้นตอนย่อยๆ เพื่อให้แน่ใจว่าคุณสามารถปฏิบัติตามได้อย่างง่ายดาย ดังนั้น มาดำดิ่งและทำให้การจัดการแบบอักษรเป็นเรื่องง่าย!

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเริ่ม มีบางสิ่งที่คุณต้องการ:

-  Aspose.Words for .NET: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้งไลบรารี Aspose.Words for .NET แล้ว คุณสามารถดาวน์โหลดได้จาก[ที่นี่](https://releases.aspose.com/words/net/).
- Visual Studio: ตัวอย่างนี้ใช้ Visual Studio เป็นสภาพแวดล้อมการพัฒนา
- .NET Framework: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง .NET Framework บนเครื่องของคุณ
- Document Directory: เส้นทางไดเร็กทอรีที่เก็บเอกสารของคุณ

## นำเข้าเนมสเปซ

ขั้นแรก นำเข้าเนมสเปซที่จำเป็นลงในโปรเจ็กต์ของคุณ:

```csharp
using System;
using System.Collections.Generic;
using Aspose.Words;
using Aspose.Words.Fonts;
```

## ขั้นตอนที่ 1: เริ่มต้นการตั้งค่าแบบอักษร

ขั้นตอนแรกคือการเริ่มต้นการตั้งค่าแบบอักษร ซึ่งจะช่วยให้คุณสามารถจัดการแหล่งแบบอักษรสำหรับเอกสารของคุณได้

```csharp
FontSettings fontSettings = new FontSettings();
List<FontSourceBase> fontSources = new List<FontSourceBase>(fontSettings.GetFontsSources());
```

- FontSettings: คลาสนี้ใช้เพื่อระบุการตั้งค่าสำหรับการทดแทนแบบอักษรและแหล่งที่มาของแบบอักษร
- แหล่งแบบอักษร: เราสร้างรายการแหล่งแบบอักษรที่มีอยู่จากการตั้งค่าแบบอักษรปัจจุบัน

## ขั้นตอนที่ 2: กำหนดไดเรกทอรีเอกสาร

ถัดไป ระบุเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือที่ที่ Aspose.Words จะค้นหาแบบอักษร

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

-  dataDir: ตัวแปรสตริงนี้เก็บเส้นทางไปยังไดเร็กทอรีที่มีแบบอักษรของคุณอยู่ แทนที่`"YOUR DOCUMENT DIRECTORY"` กับเส้นทางที่แท้จริง

## ขั้นตอนที่ 3: เพิ่มโฟลเดอร์แบบอักษรที่กำหนดเอง

ตอนนี้ให้เพิ่มแหล่งที่มาของโฟลเดอร์ใหม่เพื่อสั่งให้ Aspose.Words ค้นหาแบบอักษรในโฟลเดอร์นี้

```csharp
FolderFontSource folderFontSource = new FolderFontSource(dataDir, true);
```

- FolderFontSource: คลาสนี้แสดงถึงแหล่งแบบอักษรของโฟลเดอร์ พารามิเตอร์ที่สอง (`true`) ระบุว่าจะค้นหาแบบอักษรซ้ำในโฟลเดอร์ย่อยหรือไม่

## ขั้นตอนที่ 4: อัปเดตแหล่งที่มาของแบบอักษร

เพิ่มโฟลเดอร์ฟอนต์แบบกำหนดเองลงในรายการแหล่งฟอนต์ที่มีอยู่และอัพเดตการตั้งค่าฟอนต์

```csharp
fontSources.Add(folderFontSource);
FontSourceBase[] updatedFontSources = fontSources.ToArray();
```

- fontSources.Add(folderFontSource): เพิ่มโฟลเดอร์ฟอนต์แบบกำหนดเองให้กับแหล่งฟอนต์ที่มีอยู่
- updatedFontSources: แปลงรายการแหล่งแบบอักษรเป็นอาร์เรย์

## ขั้นตอนที่ 5: ดึงและแสดงแบบอักษร

สุดท้าย ดึงข้อมูลแบบอักษรที่มีอยู่และแสดงรายละเอียด

```csharp
foreach (PhysicalFontInfo fontInfo in updatedFontSources[0].GetAvailableFonts())
{
    Console.WriteLine("FontFamilyName : " + fontInfo.FontFamilyName);
    Console.WriteLine("FullFontName  : " + fontInfo.FullFontName);
    Console.WriteLine("Version  : " + fontInfo.Version);
    Console.WriteLine("FilePath : " + fontInfo.FilePath);
}
```

- GetAvailableFonts(): ดึงรายการแบบอักษรที่มีอยู่จากแหล่งแบบอักษรแรกในรายการที่อัปเดต
-  fontInfo: ตัวอย่างของ`PhysicalFontInfo` มีรายละเอียดเกี่ยวกับแบบอักษรแต่ละแบบ

## บทสรุป

ยินดีด้วย! คุณได้รับรายการแบบอักษรที่มีอยู่โดยใช้ Aspose.Words สำหรับ .NET สำเร็จแล้ว บทช่วยสอนนี้จะแนะนำคุณในแต่ละขั้นตอน ตั้งแต่การเริ่มต้นการตั้งค่าแบบอักษรไปจนถึงการแสดงรายละเอียดแบบอักษร ด้วยความรู้นี้ คุณสามารถจัดการแบบอักษรในเอกสาร Word ของคุณได้อย่างง่ายดาย โปรดจำไว้ว่า Aspose.Words สำหรับ .NET เป็นเครื่องมืออันทรงพลังที่สามารถเพิ่มความสามารถในการประมวลผลเอกสารของคุณได้อย่างมาก ดังนั้น ลองสำรวจคุณสมบัติเพิ่มเติมเพื่อทำให้กระบวนการพัฒนาของคุณมีประสิทธิภาพมากยิ่งขึ้น
