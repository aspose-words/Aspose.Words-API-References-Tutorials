ในบทช่วยสอนนี้ เราจะแสดงวิธีการตั้งค่าไดเรกทอรีแบบอักษรใน Aspose.Words สำหรับ .NET คุณจะได้เรียนรู้วิธีระบุไดเรกทอรีที่มีแบบอักษรที่ใช้ในเอกสาร Word ของคุณ

## ข้อกำหนดเบื้องต้น
ก่อนที่คุณจะเริ่มต้น ตรวจสอบให้แน่ใจว่าคุณมีรายการต่อไปนี้:
- ความรู้การทำงานของภาษาการเขียนโปรแกรม C #
- ไลบรารี Aspose.Words สำหรับ .NET ที่ติดตั้งในโครงการของคุณ

## ขั้นตอนที่ 1: กำหนดไดเร็กทอรีเอกสาร
 เริ่มต้นด้วยการตั้งค่าเส้นทางไดเรกทอรีไปยังตำแหน่งของเอกสาร Word ของคุณ แทนที่`"YOUR DOCUMENT DIRECTORY"` ในโค้ดด้วยเส้นทางที่เหมาะสม

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## ขั้นตอนที่ 2: ตั้งค่าไดเรกทอรีแบบอักษร
 สร้างอินสแตนซ์ของ`FontSettings` คลาสและใช้`SetFontsFolder` วิธีการระบุไดเรกทอรีที่มีแบบอักษร แทนที่`"Fonts"` ด้วยชื่อของไดเร็กทอรีฟอนต์จริง

```csharp
FontSettings fontSettings = new FontSettings();
fontSettings.SetFontsFolder(dataDir + "Fonts", false);
```

## ขั้นตอนที่ 3: โหลดเอกสารด้วยการตั้งค่าแบบอักษร
 ใช้`LoadOptions` คลาสเพื่อระบุการตั้งค่าแบบอักษรใน`FontSettings` ตัวเลือก. จากนั้นใช้`Document` คลาสเพื่อโหลดเอกสารโดยใช้ตัวเลือกเหล่านี้

```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.FontSettings = fontSettings;
Document doc = new Document(dataDir + "Rendering.docx", loadOptions);

```

### ตัวอย่างซอร์สโค้ดสำหรับ Set Fonts Folder โดยใช้ Aspose.Words สำหรับ .NET 

```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENT DIRECTORY";

FontSettings fontSettings = new FontSettings();
fontSettings.SetFontsFolder(dataDir + "Fonts", false);
LoadOptions loadOptions = new LoadOptions();
loadOptions.FontSettings = fontSettings;
Document doc = new Document(dataDir + "Rendering.docx", loadOptions);

```

## บทสรุป
ขอแสดงความยินดี! ตอนนี้คุณรู้วิธีการตั้งค่าไดเรกทอรีแบบอักษรใน Aspose.Words สำหรับ .NET แล้ว คุณสามารถใช้คุณสมบัตินี้เพื่อให้แน่ใจว่ามีฟอนต์ที่ใช้ในเอกสารของคุณพร้อมใช้งาน และเพื่อให้มั่นใจถึงความสอดคล้องในการแสดงฟอนต์

### คำถามที่พบบ่อย

#### ถาม: ฉันจะตั้งค่าโฟลเดอร์แบบอักษรแบบกำหนดเองใน Aspose.Words ได้อย่างไร

 ตอบ: หากต้องการตั้งค่าโฟลเดอร์แบบอักษรแบบกำหนดเองใน Aspose.Words คุณสามารถใช้ไฟล์`FontsFolder` ชั้นเรียนและ`SetFontsFolders` วิธีการระบุเส้นทางไปยังโฟลเดอร์ที่มีแบบอักษรของคุณ

#### ถาม: ฉันสามารถตั้งค่าโฟลเดอร์ฟอนต์หลายโฟลเดอร์ใน Aspose.Words ได้หรือไม่

 ตอบ: ได้ คุณสามารถตั้งค่าโฟลเดอร์ฟอนต์ได้หลายโฟลเดอร์ใน Aspose.Words โดยการเรียก`SetFontsFolders` วิธีการหลายครั้งด้วยเส้นทางของโฟลเดอร์แบบอักษรต่างๆ ที่คุณต้องการใช้

#### ถาม: จะเกิดอะไรขึ้นหากแบบอักษรที่ใช้ในเอกสารไม่ปรากฏในโฟลเดอร์แบบอักษรที่กำหนด

ตอบ: หากไม่มีแบบอักษรที่ใช้ในเอกสารในโฟลเดอร์แบบอักษรที่กำหนดไว้ใน Aspose.Words จะใช้แบบอักษรทดแทนแทน เพื่อให้แน่ใจว่าข้อความในเอกสารจะแสดงอย่างถูกต้องเสมอ แม้ว่าจะไม่มีแบบอักษรต้นฉบับก็ตาม

#### ถาม: โฟลเดอร์ฟอนต์ที่กำหนดไว้ใน Aspose.Words มีลำดับความสำคัญมากกว่าฟอนต์ที่ติดตั้งบนระบบหรือไม่

ตอบ: ใช่ โฟลเดอร์ฟอนต์ที่กำหนดไว้ใน Aspose.Words จะมีความสำคัญเหนือกว่าฟอนต์ที่ติดตั้งบนระบบ ซึ่งหมายความว่าหากมีแบบอักษรที่มีชื่อเดียวกันทั้งในโฟลเดอร์แบบอักษรที่กำหนดและในแบบอักษรของระบบ เวอร์ชันในโฟลเดอร์แบบอักษรจะถูกนำมาใช้เมื่อประมวลผลเอกสาร Word
