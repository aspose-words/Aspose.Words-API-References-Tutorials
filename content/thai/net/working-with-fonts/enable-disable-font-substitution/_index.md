## การแนะนำ

เคยพบว่าตัวเองอยู่ในสถานการณ์ที่แบบอักษรที่คุณเลือกอย่างพิถีพิถันในเอกสาร Word ถูกแทนที่ด้วยเมื่อดูบนคอมพิวเตอร์เครื่องอื่นหรือไม่? น่ารำคาญใช่มั้ย? สิ่งนี้เกิดขึ้นเนื่องจากการทดแทนแบบอักษร ซึ่งเป็นกระบวนการที่ระบบจะแทนที่แบบอักษรที่หายไปด้วยแบบอักษรที่มีอยู่ แต่ไม่ต้องกังวล! ด้วย Aspose.Words สำหรับ .NET คุณสามารถจัดการและควบคุมการทดแทนแบบอักษรได้อย่างง่ายดาย ในบทช่วยสอนนี้ เราจะอธิบายขั้นตอนต่างๆ เพื่อเปิดหรือปิดการแทนที่แบบอักษรในเอกสาร Word ของคุณ เพื่อให้มั่นใจว่าเอกสารของคุณจะมีลักษณะตามที่คุณต้องการเสมอ

## ข้อกำหนดเบื้องต้น

ก่อนที่จะดำเนินการตามขั้นตอนต่างๆ โปรดตรวจสอบให้แน่ใจว่าคุณมีทุกสิ่งที่คุณต้องการ:

-  Aspose.Words สำหรับ .NET: ดาวน์โหลดเวอร์ชันล่าสุด[ที่นี่](https://releases.aspose.com/words/net/).
- Visual Studio: เวอร์ชันใดก็ได้ที่รองรับ .NET
- ความรู้พื้นฐานของ C#: ข้อมูลนี้จะช่วยให้คุณปฏิบัติตามตัวอย่างการเขียนโค้ดได้

## นำเข้าเนมสเปซ

ในการเริ่มต้น ตรวจสอบให้แน่ใจว่าคุณได้นำเข้าเนมสเปซที่จำเป็นในโปรเจ็กต์ของคุณ เพิ่มสิ่งเหล่านี้ที่ด้านบนของไฟล์ C# ของคุณ:

```csharp
using Aspose.Words;
using Aspose.Words.Fonts;
```

ตอนนี้ เรามาแบ่งกระบวนการออกเป็นขั้นตอนง่ายๆ ที่จัดการได้

## ขั้นตอนที่ 1: ตั้งค่าโครงการของคุณ

ขั้นแรก ให้ตั้งค่าโครงการใหม่ใน Visual Studio และเพิ่มการอ้างอิงไปยังไลบรารี Aspose.Words สำหรับ .NET หากยังไม่มี ให้ดาวน์โหลดจาก[เว็บไซต์กำหนด](https://releases.aspose.com/words/net/).

## ขั้นตอนที่ 2: โหลดเอกสารของคุณ

จากนั้น โหลดเอกสารที่คุณต้องการใช้งาน นี่คือวิธีการ:

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Rendering.docx");
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` ด้วยเส้นทางจริงไปยังไดเร็กทอรีเอกสารของคุณ รหัสนี้จะโหลดเอกสารลงในหน่วยความจำเพื่อให้คุณสามารถจัดการได้

## ขั้นตอนที่ 3: กำหนดการตั้งค่าแบบอักษร

 ตอนนี้เรามาสร้างก`FontSettings` วัตถุเพื่อจัดการการตั้งค่าการทดแทนแบบอักษร:

```csharp
FontSettings fontSettings = new FontSettings();
```

## ขั้นตอนที่ 4: ตั้งค่าการทดแทนแบบอักษรเริ่มต้น

ตั้งค่าการทดแทนแบบอักษรเริ่มต้นเป็นแบบอักษรที่คุณเลือก แบบอักษรนี้จะถูกใช้หากไม่มีแบบอักษรดั้งเดิม:

```csharp
fontSettings.SubstitutionSettings.DefaultFontSubstitution.DefaultFontName = "Arial";
```

ในตัวอย่างนี้ เราใช้ Arial เป็นแบบอักษรเริ่มต้น

## ขั้นตอนที่ 5: ปิดใช้งานการทดแทนข้อมูลแบบอักษร

หากต้องการปิดใช้งานการทดแทนข้อมูลแบบอักษร ซึ่งจะทำให้ระบบไม่สามารถแทนที่แบบอักษรที่หายไปด้วยแบบอักษรที่มีอยู่ ให้ใช้รหัสต่อไปนี้:

```csharp
fontSettings.SubstitutionSettings.FontInfoSubstitution.Enabled = false;
```

## ขั้นตอนที่ 6: ใช้การตั้งค่าแบบอักษรกับเอกสาร

ตอนนี้ ใช้การตั้งค่าเหล่านี้กับเอกสารของคุณ:

```csharp
doc.FontSettings = fontSettings;
```

## ขั้นตอนที่ 7: บันทึกเอกสารของคุณ

สุดท้าย ให้บันทึกเอกสารที่คุณแก้ไข คุณสามารถบันทึกในรูปแบบใดก็ได้ที่คุณต้องการ สำหรับบทช่วยสอนนี้ เราจะบันทึกเป็น PDF:

```csharp
doc.Save(dataDir + "WorkingWithFonts.EnableDisableFontSubstitution.pdf");
```

## บทสรุป

และคุณก็ได้แล้ว! เมื่อทำตามขั้นตอนเหล่านี้ คุณจะสามารถควบคุมการทดแทนแบบอักษรในเอกสาร Word ของคุณได้อย่างง่ายดายโดยใช้ Aspose.Words for .NET สิ่งนี้ทำให้มั่นใจได้ว่าเอกสารของคุณคงรูปลักษณ์และการใช้งานตามที่ตั้งใจไว้ ไม่ว่าจะดูจากที่ไหนก็ตาม

## คำถามที่พบบ่อย

### ฉันสามารถใช้แบบอักษรอื่นที่ไม่ใช่ Arial เพื่อทดแทนได้หรือไม่

 อย่างแน่นอน! คุณสามารถระบุแบบอักษรที่มีอยู่ในระบบของคุณโดยการเปลี่ยนชื่อแบบอักษรใน`DefaultFontName` คุณสมบัติ.

### จะเกิดอะไรขึ้นถ้าแบบอักษรเริ่มต้นที่ระบุไม่พร้อมใช้งาน?

หากไม่มีแบบอักษรเริ่มต้น Aspose.Words จะใช้กลไกทางเลือกของระบบเพื่อค้นหาการแทนที่ที่เหมาะสม

### ฉันสามารถเปิดใช้งานการทดแทนแบบอักษรอีกครั้งหลังจากปิดใช้งานได้หรือไม่

 ใช่ คุณสามารถสลับได้`Enabled` ทรัพย์สินของ`FontInfoSubstitution` กลับไปที่`true` หากคุณต้องการเปิดใช้งานการทดแทนแบบอักษรอีกครั้ง

### มีวิธีตรวจสอบว่าแบบอักษรใดถูกแทนที่หรือไม่?

ใช่ Aspose.Words จัดเตรียมวิธีการบันทึกและติดตามการแทนที่แบบอักษร ช่วยให้คุณเห็นว่าแบบอักษรใดที่ถูกแทนที่

### ฉันสามารถใช้วิธีนี้กับเอกสารรูปแบบอื่นนอกเหนือจาก DOCX ได้หรือไม่

อย่างแน่นอน! Aspose.Words รองรับรูปแบบต่างๆ และคุณสามารถใช้การตั้งค่าแบบอักษรเหล่านี้กับรูปแบบที่รองรับได้
