## นำเข้าเนมสเปซ

ก่อนอื่น มาตั้งค่าโปรเจ็กต์ของเราและนำเข้าเนมสเปซที่จำเป็นกัน เนมสเปซเหล่านี้มีความสำคัญอย่างยิ่งต่อการเข้าถึงฟังก์ชันการทำงานของ Aspose.Words

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.ReportingServices;
```

## ขั้นตอนที่ 1: สร้างเอกสารใหม่


 การเดินทางเริ่มต้นด้วยการสร้างเอกสารใหม่และการเริ่มต้น`DocumentBuilder` - ที่`DocumentBuilder` คลาสจัดเตรียมชุดของวิธีการในการจัดการเนื้อหาเอกสารได้อย่างง่ายดาย

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENT DIRECTORY";

// สร้างเอกสารใหม่
Document doc = new Document();

// เริ่มต้น DocumentBuilder
DocumentBuilder builder = new DocumentBuilder(doc);
```

### คำอธิบาย

 ในขั้นตอนนี้ เราได้ตั้งค่าเอกสารและตัวสร้างที่เราจะใช้ในการแทรกและจัดรูปแบบเนื้อหาของเรา ที่`dataDir` ตัวแปรถือเป็นเส้นทางที่เราจะบันทึกเอกสารสุดท้ายของเรา

## ขั้นตอนที่ 2: แทรกแผนภูมิ

 ต่อไป เราจะเพิ่มแผนภูมิวงกลมลงในเอกสารของเรา ที่`InsertChart` วิธีการของ`DocumentBuilder` ชั้นเรียนทำให้สิ่งนี้ง่ายมาก

```csharp
// แทรกแผนภูมิวงกลม
Shape shape = builder.InsertChart(ChartType.Pie, 432, 252);

// เข้าถึงวัตถุแผนภูมิ
Chart chart = shape.Chart;
```

### คำอธิบาย

ที่นี่ เรากำลังแทรกแผนภูมิวงกลมลงในเอกสารของเรา ที่`InsertChart` วิธีการต้องใช้ประเภทแผนภูมิ ความกว้าง และความสูงเป็นพารามิเตอร์ หลังจากแทรกแผนภูมิแล้ว เราจะเข้าถึงวัตถุแผนภูมิเพื่อจัดการเพิ่มเติม

## ขั้นตอนที่ 3: ปรับแต่งซีรี่ส์แผนภูมิ

ตอนนี้ เราจะล้างซีรี่ส์ที่มีอยู่ในแผนภูมิและเพิ่มซีรี่ส์ที่กำหนดเองของเรา ชุดนี้จะเป็นตัวแทนของจุดข้อมูลของเรา

```csharp
// ล้างชุดแผนภูมิที่มีอยู่
chart.Series.Clear();

// เพิ่มซีรี่ส์ใหม่ลงในแผนภูมิ
ChartSeries series = chart.Series.Add("Aspose Series 1",
    new string[] { "Category 1", "Category 2", "Category 3" },
    new double[] { 2.7, 3.2, 0.8 });
```

### คำอธิบาย

ในขั้นตอนนี้ เรากำลังตรวจสอบให้แน่ใจว่าแผนภูมิของเราว่างเปล่าโดยการล้างชุดข้อมูลที่มีอยู่แล้ว จากนั้น เราจะเพิ่มซีรี่ส์ใหม่พร้อมหมวดหมู่และค่าที่กำหนดเอง ซึ่งจะแสดงในแผนภูมิวงกลมของเรา

## ขั้นตอนที่ 4: ตั้งค่าตัวเลือกเริ่มต้นสำหรับป้ายกำกับข้อมูล

ป้ายข้อมูลมีความสำคัญอย่างยิ่งในการทำให้แผนภูมิของคุณมีข้อมูลที่เป็นประโยชน์ เราจะตั้งค่าตัวเลือกเพื่อแสดงเปอร์เซ็นต์ ค่า และปรับแต่งตัวคั่น

```csharp
// เข้าถึงคอลเลกชันป้ายกำกับข้อมูล
ChartDataLabelCollection labels = series.DataLabels;

// ตั้งค่าตัวเลือกป้ายกำกับข้อมูล
labels.ShowPercentage = true;
labels.ShowValue = true;
labels.ShowLeaderLines = false;
labels.Separator = " - ";
```

### คำอธิบาย

 ที่นี่เรากำลังเข้าถึง`DataLabels`คุณสมบัติของซีรี่ส์ของเราเพื่อปรับแต่งรูปลักษณ์และข้อมูลที่แสดงบนป้ายกำกับข้อมูลแต่ละอัน เราได้เลือกที่จะแสดงทั้งเปอร์เซ็นต์และมูลค่า ซ่อนเส้นตัวนำ และตั้งค่าตัวคั่นที่กำหนดเอง

## ขั้นตอนที่ 5: บันทึกเอกสาร

สุดท้าย เราจะบันทึกเอกสารของเราลงในไดเร็กทอรีที่ระบุ ขั้นตอนนี้ช่วยให้แน่ใจว่าการเปลี่ยนแปลงทั้งหมดของเราถูกเขียนลงในไฟล์

```csharp
// บันทึกเอกสาร
doc.Save(dataDir + "WorkingWithCharts.DefaultOptionsForDataLabels.docx");
```

### คำอธิบาย

 ในขั้นตอนสุดท้ายนี้ เราจะบันทึกเอกสารของเราโดยใช้`Save` วิธี. เอกสารจะถูกบันทึกไว้ในไดเร็กทอรีที่ระบุโดย`dataDir`โดยมีชื่อว่า "WorkingWithCharts.DefaultOptionsForDataLabels.docx"
