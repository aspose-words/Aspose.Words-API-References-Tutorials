บทช่วยสอนนี้จะอธิบายวิธีใช้ Aspose.Words สำหรับ .NET เพื่อตั้งค่ารูปแบบตัวเลขสำหรับแกนในแผนภูมิ ซอร์สโค้ดที่ให้มาสาธิตวิธีการสร้างแผนภูมิ เพิ่มข้อมูลชุด และจัดรูปแบบป้ายกำกับแกน

## ขั้นตอนที่ 1: ตั้งค่าโครงการ

ตรวจสอบให้แน่ใจว่าคุณมีข้อกำหนดเบื้องต้นต่อไปนี้:

- ติดตั้ง Aspose.Words สำหรับไลบรารี .NET แล้ว คุณสามารถดาวน์โหลดได้โดยใช้ตัวจัดการแพ็คเกจ NuGet เพื่อติดตั้ง
- เส้นทางไดเร็กทอรีเอกสารที่เอกสารเอาต์พุตจะถูกบันทึก

## ขั้นตอนที่ 2: สร้างเอกสารใหม่และแทรกแผนภูมิ

 สร้างใหม่`Document` วัตถุและก`DocumentBuilder` เพื่อสร้างเอกสาร

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 ต่อไปให้ใช้`InsertChart` วิธีการของ`DocumentBuilder` เพื่อแทรกแผนภูมิคอลัมน์ลงในเอกสาร

```csharp
Shape shape = builder.InsertChart(ChartType.Column, 432, 252);
Chart chart = shape.Chart;
```

## ขั้นตอนที่ 3: เพิ่มข้อมูลซีรีส์ลงในแผนภูมิ

เพิ่มข้อมูลซีรีส์ลงในแผนภูมิ ในตัวอย่างนี้ เราจะเพิ่มห้ารายการด้วยค่าที่สอดคล้องกัน

```csharp
chart.Series.Clear();
chart.Series.Add("Aspose Series 1",
    new string[] { "Item 1", "Item 2", "Item 3", "Item 4", "Item 5" },
    new double[] { 1900000, 850000, 2100000, 600000, 1500000 });
```

## ขั้นตอนที่ 4: จัดรูปแบบป้ายกำกับแกน

 หากต้องการตั้งค่ารูปแบบตัวเลขสำหรับป้ายกำกับแกน Y ให้เข้าไปที่`AxisY` คุณสมบัติของแผนภูมิและตั้งค่า`NumberFormat.FormatCode` คุณสมบัติตามรูปแบบที่ต้องการ ในตัวอย่างนี้ เราตั้งค่ารูปแบบเป็น "#,##0" เพื่อแสดงตัวเลขที่มีตัวคั่นหลักพัน

```csharp
chart.AxisY.NumberFormat.FormatCode = "#,##0";
```

## ขั้นตอนที่ 5: บันทึกเอกสาร

 สุดท้าย ให้บันทึกเอกสารลงในไดเร็กทอรีที่ระบุโดยใช้`Save` วิธีการของ`Document` วัตถุ วัตถุ

```csharp
doc.Save(dataDir + "WorkingWithCharts.NumberFormatForAxis.docx");
```

การดำเนินการตั้งค่ารูปแบบตัวเลขสำหรับแกนโดยใช้ Aspose.Words สำหรับ .NET เสร็จสมบูรณ์

### ตัวอย่างซอร์สโค้ดสำหรับรูปแบบตัวเลขสำหรับแกนโดยใช้ Aspose.Words สำหรับ .NET 

```csharp
	// เส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Column, 432, 252);
	Chart chart = shape.Chart;
	chart.Series.Clear();
	chart.Series.Add("Aspose Series 1",
		new string[] { "Item 1", "Item 2", "Item 3", "Item 4", "Item 5" },
		new double[] { 1900000, 850000, 2100000, 600000, 1500000 });
	chart.AxisY.NumberFormat.FormatCode = "#,##0";
	doc.Save(dataDir + "WorkingWithCharts.NumberFormatForAxis.docx");
```

## บทสรุป

ในบทช่วยสอนนี้ คุณได้เรียนรู้วิธีตั้งค่ารูปแบบตัวเลขสำหรับแกนในแผนภูมิโดยใช้ Aspose.Words สำหรับ .NET ด้วยการทำตามคำแนะนำทีละขั้นตอนและใช้ซอร์สโค้ดที่ให้มา คุณสามารถสร้างเอกสารใหม่ แทรกแผนภูมิคอลัมน์ เพิ่มข้อมูลชุด และจัดรูปแบบป้ายกำกับแกนเพื่อแสดงตัวเลขในรูปแบบเฉพาะได้

Aspose.Words สำหรับ .NET มีคุณสมบัติที่มีประสิทธิภาพในการปรับแต่งรูปลักษณ์ของแผนภูมิในเอกสาร Word ด้วยการตั้งค่ารูปแบบตัวเลขสำหรับป้ายกำกับแกน คุณสามารถควบคุมวิธีการแสดงตัวเลขได้ รวมถึงตัวเลือกต่างๆ เช่น ตำแหน่งทศนิยม ตัวคั่นหลักพัน สัญลักษณ์สกุลเงิน และอื่นๆ ช่วยให้คุณสามารถนำเสนอข้อมูลตัวเลขได้อย่างชัดเจนและมีความหมาย

ด้วย Aspose.Words สำหรับ .NET คุณจะมีความยืดหยุ่นในการจัดรูปแบบแง่มุมต่างๆ ของแผนภูมิ รวมถึงป้ายกำกับแกนด้วย ด้วยการตั้งค่ารูปแบบตัวเลขสำหรับแกน คุณสามารถรับประกันความสอดคล้องและปรับปรุงความสามารถในการอ่านแผนภูมิ ทำให้ผู้ใช้สามารถตีความค่าที่แสดงได้ง่ายขึ้น

### คำถามที่พบบ่อย

#### ไตรมาสที่ 1 รูปแบบตัวเลขสำหรับแกนในแผนภูมิคืออะไร
รูปแบบตัวเลขสำหรับแกนในแผนภูมิหมายถึงการจัดรูปแบบที่ใช้กับค่าตัวเลขที่แสดงบนแกน ช่วยให้คุณสามารถควบคุมวิธีการแสดงตัวเลข รวมถึงตัวเลือกต่างๆ เช่น ตำแหน่งทศนิยม ตัวคั่นหลักพัน สัญลักษณ์สกุลเงิน เครื่องหมายเปอร์เซ็นต์ และอื่นๆ ด้วยการตั้งค่ารูปแบบตัวเลข คุณสามารถปรับแต่งลักษณะที่ปรากฏของข้อมูลตัวเลขในแผนภูมิให้เหมาะกับความต้องการเฉพาะของคุณได้

#### ไตรมาสที่ 2 ฉันจะกำหนดรูปแบบตัวเลขสำหรับป้ายกำกับแกนได้อย่างไร
 หากต้องการตั้งค่ารูปแบบตัวเลขสำหรับป้ายกำกับแกนในแผนภูมิโดยใช้ Aspose.Words สำหรับ .NET คุณสามารถเข้าถึง`AxisY` คุณสมบัติของแผนภูมิและตั้งค่า`NumberFormat.FormatCode`คุณสมบัติเป็นโค้ดรูปแบบที่ต้องการ โค้ดรูปแบบเป็นไปตามไวยากรณ์ของรูปแบบการจัดรูปแบบตัวเลขมาตรฐาน และกำหนดวิธีการแสดงตัวเลข ตัวอย่างเช่น คุณสามารถใช้ "#,##0.00" เพื่อแสดงตัวเลขที่มีทศนิยมสองตำแหน่งและตัวคั่นหลักพัน

#### ไตรมาสที่ 3 ฉันสามารถตั้งค่ารูปแบบตัวเลขที่แตกต่างกันสำหรับฉลากแกน X และแกน Y ได้หรือไม่
ได้ คุณสามารถตั้งค่ารูปแบบตัวเลขที่แตกต่างกันสำหรับป้ายกำกับแกน X และแกน Y ได้โดยใช้ Aspose.Words สำหรับ .NET เข้าถึงแกนที่เกี่ยวข้อง (`AxisX` สำหรับแกน X หรือ`AxisY` สำหรับแกน Y) ของแผนภูมิและแก้ไข`NumberFormat.FormatCode` คุณสมบัติแยกกันสำหรับแต่ละแกน ซึ่งจะทำให้คุณสามารถใช้รูปแบบตัวเลขที่แตกต่างกันกับฉลากบนแต่ละแกนได้ตามความต้องการเฉพาะของคุณ

#### ไตรมาสที่ 4 รหัสรูปแบบตัวเลขทั่วไปใดบ้างที่ฉันสามารถใช้ได้
Aspose.Words สำหรับ .NET รองรับโค้ดรูปแบบตัวเลขที่หลากหลาย ซึ่งคุณสามารถใช้จัดรูปแบบป้ายกำกับแกนในแผนภูมิได้ โค้ดรูปแบบทั่วไปบางส่วนได้แก่:

- `0` หรือ`#` - แสดงตัวเลขที่ไม่มีทศนิยม
- `0.00` หรือ`#.00` - แสดงตัวเลขที่มีทศนิยมสองตำแหน่ง
- `#,##0` แสดงตัวเลขพร้อมตัวคั่นหลักพัน
- `"€"0.00` - แสดงตัวเลขที่มีสัญลักษณ์สกุลเงินยูโรและทศนิยมสองตำแหน่ง
- `"%"0` - แสดงตัวเลขเป็นเปอร์เซ็นต์

 คุณสามารถค้นหาข้อมูลเพิ่มเติมเกี่ยวกับหมายเลขได้[รหัสรูปแบบ](https://reference.aspose.com/words/net/aspose.words.drawing.charts/chartnumberformat/formatcode/) ในการอ้างอิง API ของ Aspose.Words สำหรับ .NET

#### คำถามที่ 5 ฉันสามารถปรับแต่งคุณสมบัติอื่นๆ ของป้ายกำกับแกนได้หรือไม่
ใช่ Aspose.Words สำหรับ .NET มีคุณสมบัติที่หลากหลายเพื่อปรับแต่งรูปลักษณ์และการทำงานของป้ายกำกับแกน นอกจากรูปแบบตัวเลขแล้ว คุณยังสามารถแก้ไขคุณสมบัติต่างๆ เช่น แบบอักษร ขนาด สี การวางแนว การจัดตำแหน่ง และอื่นๆ อีกมากมาย สิ่งนี้ช่วยให้คุณปรับแต่งป้ายกำกับแกนให้ตรงกับสไตล์และความต้องการการนำเสนอที่คุณต้องการได้อย่างเต็มที่
