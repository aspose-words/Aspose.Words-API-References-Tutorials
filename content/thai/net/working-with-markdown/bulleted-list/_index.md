ในบทช่วยสอนนี้ เราจะบอกวิธีสร้างรายการสัญลักษณ์แสดงหัวข้อย่อยด้วย Aspose.Words สำหรับ .NET รายการสัญลักษณ์แสดงหัวข้อย่อยใช้เพื่อแสดงรายการโดยไม่ต้องใช้ลำดับเลข

## ขั้นตอนที่ 1: การใช้ตัวสร้างเอกสาร

ขั้นแรก เราจะใช้เครื่องมือสร้างเอกสารเพื่อเพิ่มเนื้อหาลงในเอกสารของเรา

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## ขั้นตอนที่ 2: การใช้รายการสัญลักษณ์แสดงหัวข้อย่อยเริ่มต้น

 เราสามารถใช้รายการสัญลักษณ์แสดงหัวข้อย่อยเริ่มต้นโดยใช้เครื่องมือสร้างเอกสาร`ApplyBulletDefault` วิธี.

```csharp
builder.ListFormat.ApplyBulletDefault();
```

## ขั้นตอนที่ 3: การปรับแต่งรูปแบบสัญลักษณ์แสดงหัวข้อย่อย

เราสามารถปรับแต่งรูปแบบสัญลักษณ์แสดงหัวข้อย่อยได้โดยการเข้าถึงคุณสมบัติของ`ListFormat.List.ListLevels[0]`. ในตัวอย่างนี้ เราใช้เครื่องหมายขีด "-" เป็นสัญลักษณ์แสดงหัวข้อย่อย

```csharp
builder.ListFormat.List.ListLevels[0].NumberFormat = "-";
```

## ขั้นตอนที่ 4: การเพิ่มรายการลงในรายการ

 ตอนนี้เราสามารถเพิ่มรายการในรายการสัญลักษณ์แสดงหัวข้อย่อยโดยใช้ตัวสร้างเอกสาร`Writeln` วิธี.

```csharp
builder. Writen("Element 1");
builder. Writen("Element 2");
```

## ขั้นตอนที่ 5: การลบการเยื้องออกจากรายการ

 หากเราต้องการสร้างรายการย่อย เราสามารถเพิ่มการเยื้องได้โดยใช้`ListFormat.ListIndent()` วิธี. ในตัวอย่างนี้ เรากำลังเพิ่มรายการย่อยให้กับรายการ 2a และ 2b

```csharp
builder.ListFormat.ListIndent();
builder. Writeln("Element 2a");
builder.Writeln("Element 2b");
```
### ตัวอย่างซอร์สโค้ดสำหรับรายการสัญลักษณ์แสดงหัวข้อย่อยโดยใช้ Aspose.Words สำหรับ .NET


```csharp
// ใช้ตัวสร้างเอกสารเพื่อเพิ่มเนื้อหาลงในเอกสาร
DocumentBuilder builder = new DocumentBuilder();

builder.ListFormat.ApplyBulletDefault();
builder.ListFormat.List.ListLevels[0].NumberFormat = "-";

builder.Writeln("Item 1");
builder.Writeln("Item 2");

builder.ListFormat.ListIndent();

builder.Writeln("Item 2a");
builder.Writeln("Item 2b");
```

ขอแสดงความยินดี! ตอนนี้คุณได้เรียนรู้วิธีสร้างรายการสัญลักษณ์แสดงหัวข้อย่อยด้วย Aspose.Words สำหรับ .NET แล้ว

### คำถามที่พบบ่อย

#### ถาม: จะสร้างรายการหัวข้อย่อยใน Markdown ได้อย่างไร

ตอบ: หากต้องการสร้างรายการหัวข้อย่อยใน Markdown ให้เริ่มแต่ละรายการด้วยสัญลักษณ์หัวข้อย่อย (`-`, `*` , หรือ`+`) ตามด้วยช่องว่าง

#### ถาม: คุณสามารถซ้อนรายการหัวข้อย่อยใน Markdown ได้หรือไม่

ตอบ: ได้ คุณสามารถซ้อนรายการสัญลักษณ์แสดงหัวข้อย่อยใน Markdown ได้โดยการเพิ่มช่องว่างออฟเซ็ตสี่ช่องไว้ด้านหน้าแต่ละรายการในรายการที่ซ้อนกัน

#### ถาม: จะปรับแต่งสัญลักษณ์หัวข้อย่อยได้อย่างไร

ตอบ: ใน Markdown มาตรฐาน จะมีการกำหนดสัญลักษณ์หัวข้อย่อยไว้ล่วงหน้า อย่างไรก็ตาม เครื่องมือแก้ไข Markdown บางตัวอนุญาตให้คุณปรับแต่งได้โดยใช้ส่วนขยายเฉพาะ

#### ถาม: รายการสัญลักษณ์แสดงหัวข้อย่อยใน Markdown รองรับการเยื้องหรือไม่

ตอบ: ใช่ รายการสัญลักษณ์แสดงหัวข้อย่อยใน Markdown รองรับการเยื้อง คุณสามารถเพิ่มกะซ้ายได้โดยใช้ช่องว่างหรือแท็บ

#### ถาม: สามารถเพิ่มลิงก์หรือข้อความอินไลน์ลงในรายการได้หรือไม่

ตอบ: ได้ คุณสามารถเพิ่มลิงก์หรือข้อความอินไลน์เพื่อแสดงรายการโดยใช้ไวยากรณ์ Markdown ที่เหมาะสมได้
