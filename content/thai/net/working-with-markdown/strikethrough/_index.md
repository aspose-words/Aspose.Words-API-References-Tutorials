ในตัวอย่างนี้ เราจะอธิบายวิธีการใช้รูปแบบข้อความขีดทับโดยใช้ Aspose.Words สำหรับ .NET ข้อความขีดทับใช้เพื่อระบุว่าข้อความถูกลบหรือไม่ถูกต้องอีกต่อไป

## ขั้นตอนที่ 1: การใช้ตัวสร้างเอกสาร

ขั้นแรก เราจะใช้เครื่องมือสร้างเอกสารเพื่อเพิ่มเนื้อหาลงในเอกสารของเรา

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## ขั้นตอนที่ 2: ใช้รูปแบบข้อความขีดทับ

 เราจะเปิดใช้งานรูปแบบข้อความขีดทับโดยการตั้งค่า`StrikeThrough` ทรัพย์สินของ`Font` วัตถุประสงค์`true`.

```csharp
builder.Font.StrikeThrough = true;
```

## ขั้นตอนที่ 3: เพิ่มข้อความขีดทับ

 ขณะนี้เราสามารถเพิ่มข้อความขีดทับโดยใช้เครื่องมือสร้างเอกสารได้`Writeln` วิธี.

```csharp
builder.Writeln("This text will be StrikeThrough");
```


### ตัวอย่างซอร์สโค้ดสำหรับข้อความขีดทับด้วย Aspose.Words สำหรับ .NET

```csharp
// ใช้ตัวสร้างเอกสารเพื่อเพิ่มเนื้อหาลงในเอกสาร
DocumentBuilder builder = new DocumentBuilder();

// ทำให้ข้อความขีดทับ
builder.Font.StrikeThrough = true;
builder.Writeln("This text will be StrikeThrough");
```

ขอแสดงความยินดี! ตอนนี้ คุณได้เรียนรู้วิธีใช้ลักษณะข้อความขีดทับกับ Aspose.Words สำหรับ .NET แล้ว

### คำถามที่พบบ่อย

#### ถาม: ฉันจะเพิ่มข้อความขีดทับใน Aspose.Words ได้อย่างไร

 ตอบ: หากต้องการเพิ่มข้อความขีดทับใน Aspose.Words คุณสามารถใช้`Font.StrikeThrough` ทรัพย์สินของ`Run`วัตถุ. คุณสามารถตั้งค่าคุณสมบัตินี้เป็น`true` เพื่อเพิ่มข้อความขีดทับให้กับข้อความเฉพาะ ตัวอย่างเช่นคุณสามารถใช้`run.Font.StrikeThrough=true` เพื่อเพิ่มข้อความขีดทับลงใน`Run` วัตถุ.

#### ถาม: เป็นไปได้ไหมที่จะเพิ่มข้อความขีดทับให้กับข้อความหลาย ๆ ส่วนในย่อหน้าเดียวกัน

 ตอบ: ได้ คุณสามารถเพิ่มข้อความขีดทับลงในข้อความหลายส่วนในย่อหน้าเดียวได้โดยใช้หลายย่อหน้า`Run` วัตถุ คุณสามารถสร้างได้หลายรายการ`Run` วัตถุและตั้งค่า`Font.StrikeThrough`ทรัพย์สินเพื่อ`true` สำหรับแต่ละวัตถุเพื่อเพิ่มข้อความขีดทับในส่วนข้อความที่ต้องการ จากนั้นคุณสามารถเพิ่มลงในย่อหน้าได้โดยใช้`Paragraph.AppendChild(run)` วิธี.

#### ถาม: ฉันสามารถเพิ่มข้อความขีดทับให้กับข้อความที่อยู่ในตารางหรือเซลล์ใน Aspose.Words ได้หรือไม่

 ตอบ: ได้ คุณสามารถเพิ่มข้อความขีดทับให้กับข้อความที่อยู่ในตารางหรือเซลล์ใน Aspose.Words ได้ คุณสามารถข้ามไปยังเซลล์หรือย่อหน้าที่คุณต้องการโดยใช้วิธีการที่เหมาะสม จากนั้นใช้การจัดรูปแบบข้อความขีดทับโดยใช้`Font.StrikeThrough` ทรัพย์สินของ`Run` หรือ`Paragraph` วัตถุ.
