## นำเข้าเนมสเปซ

ขั้นแรก ให้เรานำเข้าเนมสเปซที่จำเป็นก่อน วิธีนี้จะช่วยให้เราเข้าถึงฟังก์ชัน Aspose.Words ได้โดยไม่ต้องอ้างอิงเส้นทางเนมสเปซแบบเต็มอยู่ตลอดเวลา

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Tables;
```

ตอนนี้เรามาดูกระบวนการในการทำข้อความให้เป็นตัวหนาในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET กัน

## ขั้นตอนที่ 1: เริ่มต้นใช้งาน DocumentBuilder

 การ`DocumentBuilder` คลาสนี้ช่วยให้เพิ่มเนื้อหาลงในเอกสารของคุณได้อย่างรวดเร็วและง่ายดาย มาเริ่มต้นกันเลย

```csharp
// ใช้เครื่องมือสร้างเอกสารเพื่อเพิ่มเนื้อหาลงในเอกสาร
DocumentBuilder builder = new DocumentBuilder();
```

## ขั้นตอนที่ 2: ทำให้ข้อความเป็นตัวหนา

 ตอนนี้มาถึงส่วนที่สนุกแล้ว - การทำให้ข้อความเป็นตัวหนา เราจะตั้งค่า`Bold` ทรัพย์สินของ`Font` คัดค้าน`true` และเขียนข้อความตัวหนาของเรา

```csharp
// ทำให้ข้อความเป็นตัวหนา
builder.Font.Bold = true;
builder.Writeln("This text will be Bold");
```

## คำถามที่พบบ่อย

### ฉันสามารถทำให้ข้อความบางส่วนเป็นตัวหนาได้ไหม?
 ใช่ คุณสามารถทำได้ ใช้`DocumentBuilder` เพื่อจัดรูปแบบส่วนเฉพาะของข้อความของคุณ

 ใช่ คุณสามารถทำได้ ตัวอย่างเช่น คุณสามารถทำให้ข้อความเป็นตัวหนาและตัวเอียงพร้อมกันได้โดยการตั้งค่าทั้งสองอย่าง`builder.Font.Bold` และ`builder.Font.Italic` ถึง`true`.
