ในตัวอย่างนี้ เราจะบอกวิธีทำให้ข้อความเป็นตัวหนาด้วย Aspose.Words สำหรับ .NET ข้อความที่เป็นตัวหนาทำให้มองเห็นได้ชัดเจนยิ่งขึ้นและเพิ่มความโดดเด่นยิ่งขึ้น

## ขั้นตอนที่ 1: การใช้ตัวสร้างเอกสาร

ขั้นแรก เราจะใช้เครื่องมือสร้างเอกสารเพื่อเพิ่มเนื้อหาลงในเอกสารของเรา

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## ขั้นตอนที่ 2: ข้อความตัวหนา

 เราสามารถทำให้ข้อความเป็นตัวหนาได้โดยตั้งค่าตัวสร้างเอกสาร`Font.Bold`ทรัพย์สินเพื่อ`true`.

```csharp
builder.Font.Bold = true;
```

## ขั้นตอนที่ 3: เพิ่มเนื้อหาลงในเอกสาร

 ตอนนี้เราสามารถเพิ่มเนื้อหาลงในเอกสารโดยใช้วิธีการสร้างเอกสาร เช่น`Writeln`ซึ่งจะเพิ่มบรรทัดข้อความ

```csharp
builder.Writeln("This text will be bold");
```

## ตัวอย่างซอร์สโค้ดสำหรับข้อความตัวหนาโดยใช้ Aspose.Words สำหรับ .NET


```csharp
// ใช้ตัวสร้างเอกสารเพื่อเพิ่มเนื้อหาลงในเอกสาร
DocumentBuilder builder = new DocumentBuilder();

// ทำให้ข้อความเป็นตัวหนา
builder.Font.Bold = true;
builder.Writeln("This text will be Bold");  
```

ขอแสดงความยินดี! ตอนนี้คุณได้เรียนรู้วิธีสร้างข้อความตัวหนาด้วย Aspose.Words for .NET แล้ว


### คำถามที่พบบ่อย

#### ถาม: ฉันจะทำให้ข้อความเป็นตัวหนาใน Aspose.Words ได้อย่างไร

 ตอบ: หากต้องการทำให้ข้อความเป็นตัวหนาใน Aspose.Words คุณสามารถใช้`Font.Bold` ทรัพย์สินของ`Run` วัตถุ. คุณสามารถตั้งค่าคุณสมบัตินี้เป็น`true` เพื่อทำให้ข้อความเฉพาะเจาะจงเป็นตัวหนา ตัวอย่างเช่นคุณสามารถใช้`run.Font.Bold=true` เพื่อทำให้ข้อความภายในตัวหนา`Run` วัตถุ.

#### ถาม: เป็นไปได้ไหมที่ทำให้ข้อความหลายส่วนในย่อหน้าเดียวกันเป็นตัวหนา

ตอบ: ได้ คุณสามารถทำให้ข้อความหลายชิ้นเป็นตัวหนาในย่อหน้าเดียวได้โดยใช้หลายข้อความ`Run` วัตถุ คุณสามารถสร้างได้หลายรายการ`Run` วัตถุและตั้งค่า`Font.Bold`ทรัพย์สินเพื่อ`true` เพื่อให้แต่ละวัตถุเป็นตัวหนาในส่วนที่ต้องการของข้อความ จากนั้นคุณสามารถเพิ่มลงในย่อหน้าได้โดยใช้`Paragraph.AppendChild(run)` วิธี.

#### ถาม: ฉันสามารถทำให้ข้อความที่อยู่ในตารางหรือเซลล์ใน Aspose.Words เป็นตัวหนาได้หรือไม่

 ตอบ: ได้ คุณสามารถส่งข้อความตัวหนาที่อยู่ในตารางหรือเซลล์ใน Aspose.Words ได้ คุณสามารถนำทางไปยังเซลล์หรือย่อหน้าที่คุณต้องการโดยใช้วิธีการที่เหมาะสม จากนั้นใช้การจัดรูปแบบตัวหนาโดยใช้`Font.Bold` ทรัพย์สินของ`Run` หรือ`Paragraph` วัตถุ.
