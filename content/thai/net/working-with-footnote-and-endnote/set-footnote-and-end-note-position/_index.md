ในบทช่วยสอนทีละขั้นตอนนี้ เราจะแนะนำคุณเกี่ยวกับวิธีใช้ Aspose.Words สำหรับ .NET เพื่อกำหนดตำแหน่งของเชิงอรรถและอ้างอิงท้ายเรื่องในเอกสาร Word เราจะอธิบายซอร์สโค้ด C# ที่ให้มา และแสดงวิธีนำไปใช้ในโครงการของคุณเอง

 ในการเริ่มต้น ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง Aspose.Words สำหรับ .NET และตั้งค่าในสภาพแวดล้อมการพัฒนาของคุณ หากคุณยังไม่ได้ดำเนินการ ให้ดาวน์โหลดและติดตั้งไลบรารีจาก[Aspose.Releases]https://releases.aspose.com/words/net/

## ขั้นตอนที่ 1: การเริ่มต้นวัตถุเอกสาร

 ขั้นแรกให้เริ่มต้น`Document` วัตถุโดยระบุเส้นทางไปยังเอกสารต้นฉบับของคุณ:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";     
Document doc = new Document(dataDir + "Document.docx");
```

## ขั้นตอนที่ 2: การตั้งค่าเชิงอรรถและตำแหน่งอ้างอิงท้ายเรื่อง

 ต่อไปให้เข้าไปที่`FootnoteOptions` และ`EndnoteOptions`คุณสมบัติของเอกสารเพื่อกำหนดตำแหน่งของเชิงอรรถและอ้างอิงท้ายเรื่อง ในตัวอย่างนี้ เราตั้งค่าตำแหน่งของเชิงอรรถให้อยู่ใต้ข้อความ และตำแหน่งของอ้างอิงท้ายเรื่องให้อยู่ที่ส่วนท้ายของส่วน:

```csharp
doc.FootnoteOptions.Position = FootnotePosition.BeneathText;
doc.EndnoteOptions.Position = EndnotePosition.EndOfSection;
```

## ขั้นตอนที่ 3: บันทึกเอกสาร

สุดท้าย ให้บันทึกเอกสารที่แก้ไข:

```csharp
doc.Save(dataDir + "WorkingWithFootnotes.SetFootnoteAndEndNotePosition.docx");
```

แค่นั้นแหละ! คุณได้ตั้งค่าตำแหน่งของเชิงอรรถและอ้างอิงท้ายเรื่องในเอกสาร Word เรียบร้อยแล้วโดยใช้ Aspose.Words สำหรับ .NET

### ตัวอย่างซอร์สโค้ดสำหรับตั้งค่าตำแหน่งเชิงอรรถและอ้างอิงท้ายเรื่องโดยใช้ Aspose.Words สำหรับ .NET

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";     
Document doc = new Document(dataDir + "Document.docx");

doc.FootnoteOptions.Position = FootnotePosition.BeneathText;
doc.EndnoteOptions.Position = EndnotePosition.EndOfSection;

doc.Save(dataDir + "WorkingWithFootnotes.SetFootnoteAndEndNotePosition.docx");
```

คุณสามารถใช้โค้ดนี้ในโครงการของคุณเองและแก้ไขได้ตามความต้องการเฉพาะของคุณ

### คำถามที่พบบ่อย

#### ถาม: ฉันจะวางตำแหน่งเชิงอรรถและอ้างอิงท้ายเรื่องใน Aspose.Words ได้อย่างไร

 ตอบ: หากต้องการวางเชิงอรรถและอ้างอิงท้ายเรื่องใน Aspose.Words คุณต้องใช้`FootnoteOptions` ชั้นเรียนและ`Position` คุณสมบัติ. คุณสามารถตั้งค่าคุณสมบัตินี้เป็นค่าใดก็ได้ที่คุณต้องการ เช่น`BottomOfPage` (ที่ด้านล่างของหน้า) หรือ`EndOfSection` (ในตอนท้ายของส่วน)

#### ถาม: เป็นไปได้หรือไม่ที่จะปรับแต่งตำแหน่งของเชิงอรรถและอ้างอิงท้ายเรื่องสำหรับแต่ละหน้าหรือส่วนของเอกสาร

ตอบ: ได้ คุณสามารถปรับแต่งตำแหน่งของเชิงอรรถและอ้างอิงท้ายเรื่องสำหรับแต่ละหน้าหรือส่วนของเอกสารได้ คุณสามารถใช้ส่วน Aspose.Words และวิธีการจัดการหน้าเพื่อกำหนดตำแหน่งเฉพาะสำหรับเชิงอรรถและอ้างอิงท้ายเรื่อง

#### ถาม: ฉันจะลบเชิงอรรถหรืออ้างอิงท้ายเรื่องออกจากเอกสารได้อย่างไร

 ตอบ: หากต้องการลบเชิงอรรถหรืออ้างอิงท้ายเรื่องออกจากเอกสารใน Aspose.Words คุณสามารถใช้วิธีการที่เหมาะสม เช่น`RemoveAllFootnotes` เพื่อลบเชิงอรรถทั้งหมดหรือ`RemoveAllEndnotes` เพื่อลบอ้างอิงท้ายเรื่องทั้งหมด อย่าลืมบันทึกเอกสารหลังจากดำเนินการเหล่านี้

#### ถาม: สามารถวางเชิงอรรถและอ้างอิงท้ายเรื่องไว้นอกระยะขอบหน้าได้หรือไม่

ไม่ ตามค่าเริ่มต้นเชิงอรรถและอ้างอิงท้ายเรื่องไม่สามารถวางนอกระยะขอบหน้าใน Aspose.Words อย่างไรก็ตาม คุณสามารถปรับระยะขอบของเอกสารเพื่อเพิ่มพื้นที่สำหรับเชิงอรรถและอ้างอิงท้ายเรื่องได้ ถ้าจำเป็น

#### ถาม: เชิงอรรถและอ้างอิงท้ายเรื่องสามารถปรับแต่งด้วยแบบอักษรหรือสไตล์การจัดรูปแบบเฉพาะได้หรือไม่

ตอบ: ได้ คุณสามารถปรับแต่งเชิงอรรถและอ้างอิงท้ายเรื่องด้วยแบบอักษรหรือรูปแบบการจัดรูปแบบเฉพาะใน Aspose.Words คุณสามารถใช้วิธีการและคุณสมบัติที่มีอยู่เพื่อใช้ลักษณะแบบอักษร สี ขนาดแบบอักษร ฯลฯ เชิงอรรถและอ้างอิงท้ายเรื่อง
