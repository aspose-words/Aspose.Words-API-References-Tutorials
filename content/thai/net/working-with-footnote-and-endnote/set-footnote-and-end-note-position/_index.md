## การแนะนำ

หากคุณทำงานกับเอกสาร Word และต้องการจัดการเชิงอรรถและเชิงท้ายอย่างมีประสิทธิภาพ Aspose.Words สำหรับ .NET คือไลบรารีที่คุณต้องการ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการตั้งค่าตำแหน่งเชิงอรรถและเชิงท้ายในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET เราจะอธิบายแต่ละขั้นตอนเพื่อให้ทำตามและนำไปใช้ได้ง่าย

## ข้อกำหนดเบื้องต้น

ก่อนจะเริ่มบทช่วยสอนนี้ ให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

-  Aspose.Words สำหรับไลบรารี .NET: คุณสามารถดาวน์โหลดได้จาก[ที่นี่](https://releases.aspose.com/words/net/).
- Visual Studio: เวอร์ชันล่าสุดใดๆ ก็ทำงานได้ดี
- ความรู้พื้นฐานเกี่ยวกับ C#: การทำความเข้าใจพื้นฐานจะช่วยให้คุณทำตามได้อย่างง่ายดาย

## นำเข้าเนมสเปซ

ขั้นแรก นำเข้าเนมสเปซที่จำเป็นในโครงการ C# ของคุณ:

```csharp
using System;
using Aspose.Words;
```

## ขั้นตอนที่ 1: โหลดเอกสาร Word

ในการเริ่มต้น คุณต้องโหลดเอกสาร Word ของคุณลงในอ็อบเจ็กต์ Aspose.Words Document ซึ่งจะช่วยให้คุณจัดการเนื้อหาของเอกสารได้

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Document.docx");
```

ในโค้ดนี้ให้แทนที่`"YOUR DOCUMENT DIRECTORY"`ด้วยเส้นทางจริงที่เอกสารของคุณตั้งอยู่

## ขั้นตอนที่ 2: กำหนดตำแหน่งเชิงอรรถ

ขั้นต่อไป คุณจะกำหนดตำแหน่งของเชิงอรรถ Aspose.Words สำหรับ .NET ช่วยให้คุณวางตำแหน่งเชิงอรรถได้ที่ด้านล่างของหน้าหรือใต้ข้อความ

```csharp
doc.FootnoteOptions.Position = FootnotePosition.BeneathText;
```

 ที่นี่ เราได้ตั้งค่าเชิงอรรถให้ปรากฏอยู่ใต้ข้อความ หากคุณต้องการให้เชิงอรรถอยู่ด้านล่างสุดของหน้า ให้ใช้`FootnotePosition.BottomOfPage`.

## ขั้นตอนที่ 3: กำหนดตำแหน่งเชิงอรรถ

ในทำนองเดียวกัน คุณสามารถกำหนดตำแหน่งของเชิงอรรถได้ โดยสามารถวางเชิงอรรถไว้ที่ท้ายส่วนหรือท้ายเอกสารก็ได้

```csharp
doc.EndnoteOptions.Position = EndnotePosition.EndOfSection;
```

 ในตัวอย่างนี้ เชิงอรรถจะถูกวางไว้ที่ท้ายของแต่ละหัวข้อ หากต้องการวางไว้ที่ท้ายเอกสาร ให้ใช้`EndnotePosition.EndOfDocument`.

## ขั้นตอนที่ 4: บันทึกเอกสาร

สุดท้าย ให้บันทึกเอกสารเพื่อใช้การเปลี่ยนแปลง ตรวจสอบให้แน่ใจว่าคุณได้ระบุเส้นทางไฟล์และชื่อที่ถูกต้องสำหรับเอกสารเอาต์พุต

```csharp
doc.Save(dataDir + "WorkingWithFootnotes.SetFootnoteAndEndNotePosition.docx");
```

บรรทัดนี้จะบันทึกเอกสารที่แก้ไขแล้วไปยังไดเร็กทอรีที่คุณระบุ

## บทสรุป

การกำหนดตำแหน่งเชิงอรรถและเชิงท้ายในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET เป็นเรื่องง่ายเมื่อคุณทราบขั้นตอนต่างๆ แล้ว โดยปฏิบัติตามคำแนะนำนี้ คุณสามารถปรับแต่งเอกสารให้เหมาะกับความต้องการของคุณ และทำให้มั่นใจว่าเชิงอรรถและเชิงท้ายจะอยู่ในตำแหน่งที่คุณต้องการ

## คำถามที่พบบ่อย

### ฉันสามารถกำหนดตำแหน่งที่แตกต่างกันสำหรับเชิงอรรถหรือเชิงท้ายแต่ละรายการได้หรือไม่

ไม่ Aspose.Words สำหรับ .NET จะกำหนดตำแหน่งเชิงอรรถและเชิงท้ายทั้งหมดในเอกสารอย่างสม่ำเสมอ

### Aspose.Words สำหรับ .NET เข้ากันได้กับเอกสาร Word ทุกเวอร์ชันหรือไม่
