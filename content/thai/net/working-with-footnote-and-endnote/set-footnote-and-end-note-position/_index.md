## การแนะนำ

หากคุณกำลังทำงานกับเอกสาร Word และต้องการจัดการเชิงอรรถและอ้างอิงท้ายเรื่องอย่างมีประสิทธิภาพ Aspose.Words สำหรับ .NET ก็คือไลบรารี่ที่เหมาะกับคุณ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการตั้งค่าเชิงอรรถและตำแหน่งอ้างอิงท้ายเรื่องในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET เราจะแจกแจงแต่ละขั้นตอนเพื่อให้ง่ายต่อการปฏิบัติตามและนำไปใช้

## ข้อกำหนดเบื้องต้น

ก่อนที่จะเข้าสู่บทช่วยสอน ตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

-  Aspose.Words สำหรับ .NET Library: คุณสามารถดาวน์โหลดได้จาก[ที่นี่](https://releases.aspose.com/words/net/).
- Visual Studio: เวอร์ชันล่าสุดใด ๆ จะทำงานได้ดี
- ความรู้พื้นฐานของ C#: การทำความเข้าใจพื้นฐานจะช่วยให้คุณปฏิบัติตามได้อย่างง่ายดาย

## นำเข้าเนมสเปซ

ขั้นแรก นำเข้าเนมสเปซที่จำเป็นในโปรเจ็กต์ C# ของคุณ:

```csharp
using System;
using Aspose.Words;
```

## ขั้นตอนที่ 1: โหลดเอกสาร Word

ในการเริ่มต้น คุณต้องโหลดเอกสาร Word ของคุณลงในวัตถุ Aspose.Words Document ซึ่งจะช่วยให้คุณสามารถจัดการเนื้อหาของเอกสารได้

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Document.docx");
```

 ในโค้ดนี้ให้แทนที่`"YOUR DOCUMENT DIRECTORY"`พร้อมเส้นทางจริงที่เอกสารของคุณตั้งอยู่

## ขั้นตอนที่ 2: ตั้งค่าตำแหน่งเชิงอรรถ

ต่อไป คุณจะกำหนดตำแหน่งของเชิงอรรถ Aspose.Words สำหรับ .NET ช่วยให้คุณสามารถวางเชิงอรรถที่ด้านล่างของหน้าหรือใต้ข้อความได้

```csharp
doc.FootnoteOptions.Position = FootnotePosition.BeneathText;
```

 ที่นี่ เราได้ตั้งค่าเชิงอรรถให้ปรากฏใต้ข้อความ หากคุณต้องการที่ด้านล่างของหน้าให้ใช้`FootnotePosition.BottomOfPage`.

## ขั้นตอนที่ 3: ตั้งค่าตำแหน่งอ้างอิงท้ายเรื่อง

ในทำนองเดียวกัน คุณสามารถกำหนดตำแหน่งของอ้างอิงท้ายเรื่องได้ อ้างอิงท้ายเรื่องสามารถวางตำแหน่งไว้ที่ส่วนท้ายของส่วนหรือส่วนท้ายของเอกสารก็ได้

```csharp
doc.EndnoteOptions.Position = EndnotePosition.EndOfSection;
```

 ในตัวอย่างนี้ อ้างอิงท้ายเรื่องจะถูกวางไว้ที่ส่วนท้ายของแต่ละส่วน หากต้องการวางไว้ที่ส่วนท้ายของเอกสาร ให้ใช้`EndnotePosition.EndOfDocument`.

## ขั้นตอนที่ 4: บันทึกเอกสาร

สุดท้าย ให้บันทึกเอกสารเพื่อใช้การเปลี่ยนแปลง ตรวจสอบให้แน่ใจว่าคุณระบุเส้นทางไฟล์และชื่อที่ถูกต้องสำหรับเอกสารเอาต์พุต

```csharp
doc.Save(dataDir + "WorkingWithFootnotes.SetFootnoteAndEndNotePosition.docx");
```

บรรทัดนี้จะบันทึกเอกสารที่แก้ไขไปยังไดเร็กทอรีที่คุณระบุ

## บทสรุป

การตั้งค่าตำแหน่งเชิงอรรถและอ้างอิงท้ายเรื่องในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET จะตรงไปตรงมาเมื่อคุณทราบขั้นตอนต่างๆ ด้วยการทำตามคำแนะนำนี้ คุณสามารถปรับแต่งเอกสารของคุณให้เหมาะกับความต้องการของคุณได้ เพื่อให้แน่ใจว่าเชิงอรรถและอ้างอิงท้ายเรื่องอยู่ในตำแหน่งที่คุณต้องการ
