คุณต้องการปรับปรุงเอกสาร Word ของคุณด้วยการจัดการอ้างอิงท้ายเรื่องอย่างมีประสิทธิภาพหรือไม่? ไม่ต้องมองอีกต่อไป! ในบทช่วยสอนนี้ เราจะแนะนำคุณตลอดขั้นตอนการตั้งค่าตัวเลือกอ้างอิงท้ายเรื่องในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET ในตอนท้ายของคู่มือนี้ คุณจะเป็นมืออาชีพในการปรับแต่งอ้างอิงท้ายเรื่องให้เหมาะกับความต้องการของเอกสารของคุณ

## นำเข้าเนมสเปซ

ในการเริ่มต้น คุณจะต้องนำเข้าเนมสเปซที่จำเป็น เนมสเปซเหล่านี้ให้การเข้าถึงคลาสและวิธีการที่จำเป็นสำหรับการจัดการเอกสาร Word

```csharp
using Aspose.Words;
using Aspose.Words.Notes;
```

## ขั้นตอนที่ 1: โหลดเอกสาร

 ขั้นแรก ให้โหลดเอกสารในตำแหน่งที่เราต้องการตั้งค่าตัวเลือกอ้างอิงท้ายเรื่อง เราจะใช้`Document` จากไลบรารี Aspose.Words เพื่อทำสิ่งนี้ให้สำเร็จ

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Document.docx");
```

## ขั้นตอนที่ 2: เริ่มต้น DocumentBuilder

 ต่อไปเราจะเริ่มต้น`DocumentBuilder`ระดับ. คลาสนี้มีวิธีง่ายๆ ในการเพิ่มเนื้อหาลงในเอกสาร

```csharp
DocumentBuilder builder = new DocumentBuilder(doc);
```

## ขั้นตอนที่ 3: เพิ่มข้อความและแทรกอ้างอิงท้ายเรื่อง

 ตอนนี้ เรามาเพิ่มข้อความลงในเอกสารและแทรกอ้างอิงท้ายเรื่อง ที่`InsertFootnote` วิธีการของ`DocumentBuilder` คลาสช่วยให้เราสามารถเพิ่มอ้างอิงท้ายเรื่องลงในเอกสารได้

```csharp
builder.Write("Some text");
builder.InsertFootnote(FootnoteType.Endnote, "Footnote text.");
```

## ขั้นตอนที่ 4: เข้าถึงและตั้งค่าตัวเลือกอ้างอิงท้ายเรื่อง

 ในการปรับแต่งตัวเลือกอ้างอิงท้ายเรื่อง เราจำเป็นต้องเข้าถึง`EndnoteOptions` ทรัพย์สินของ`Document` ระดับ. จากนั้นเราสามารถตั้งค่าตัวเลือกต่างๆ เช่น กฎการรีสตาร์ทและตำแหน่ง

```csharp
EndnoteOptions option = doc.EndnoteOptions;
option.RestartRule = FootnoteNumberingRule.RestartPage;
option.Position = EndnotePosition.EndOfSection;
```

## ขั้นตอนที่ 5: บันทึกเอกสาร

 สุดท้ายนี้ มาบันทึกเอกสารด้วยตัวเลือกอ้างอิงท้ายเรื่องที่อัปเดตกัน ที่`Save` วิธีการของ`Document` คลาสอนุญาตให้เราบันทึกเอกสารไปยังไดเร็กทอรีที่ระบุ

```csharp
doc.Save(dataDir + "WorkingWithFootnotes.SetEndnoteOptions.docx");
```
