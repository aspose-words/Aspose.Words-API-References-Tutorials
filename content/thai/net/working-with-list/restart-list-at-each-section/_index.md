## การแนะนำ

การสร้างเอกสารที่มีโครงสร้างและจัดระเบียบอย่างดีบางครั้งอาจรู้สึกเหมือนเป็นการไขปริศนาที่ซับซ้อน ปริศนาชิ้นหนึ่งคือการจัดการรายการอย่างมีประสิทธิภาพ โดยเฉพาะอย่างยิ่งเมื่อคุณต้องการให้พวกเขารีสตาร์ทในแต่ละส่วน ด้วย Aspose.Words สำหรับ .NET คุณสามารถทำสิ่งนี้ให้สำเร็จได้อย่างราบรื่น มาดูวิธีที่คุณสามารถเริ่มรายการใหม่ในแต่ละส่วนในเอกสาร Word ของคุณโดยใช้ Aspose.Words สำหรับ .NET กันดีกว่า

## นำเข้าเนมสเปซ

ก่อนที่จะเขียนโค้ด ตรวจสอบให้แน่ใจว่าคุณนำเข้าเนมสเปซที่จำเป็น:

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Lists;
```

ตอนนี้ เรามาแบ่งกระบวนการออกเป็นหลายขั้นตอนเพื่อให้ง่ายต่อการปฏิบัติตาม

## ขั้นตอนที่ 1: เริ่มต้นเอกสาร

ขั้นแรก คุณจะต้องสร้างอินสแตนซ์เอกสารใหม่

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
```

## ขั้นตอนที่ 2: เพิ่มรายการลำดับเลข

ถัดไป เพิ่มรายการลำดับเลขลงในเอกสาร รายการนี้จะเป็นไปตามรูปแบบการกำหนดหมายเลขเริ่มต้น

```csharp
doc.Lists.Add(ListTemplate.NumberDefault);
```

## ขั้นตอนที่ 3: เข้าถึงรายการและตั้งค่าคุณสมบัติการรีสตาร์ท

ดึงข้อมูลรายการที่คุณเพิ่งสร้างขึ้นและตั้งค่า`IsRestartAtEachSection`ทรัพย์สินเพื่อ`true`- เพื่อให้แน่ใจว่ารายการจะรีสตาร์ทการกำหนดหมายเลขในแต่ละส่วนใหม่

```csharp
List list = doc.Lists[0];
list.IsRestartAtEachSection = true;
```

## ขั้นตอนที่ 4: สร้างตัวสร้างเอกสารและเชื่อมโยงรายการ

 สร้างก`DocumentBuilder` เพื่อแทรกเนื้อหาลงในเอกสารและเชื่อมโยงกับรายการ

```csharp
DocumentBuilder builder = new DocumentBuilder(doc);
builder.ListFormat.List = list;
```

## ขั้นตอนที่ 5: เพิ่มรายการและแทรกตัวแบ่งส่วน

ตอนนี้เพิ่มรายการลงในรายการ เพื่อแสดงให้เห็นฟังก์ชันการรีสตาร์ท เราจะแทรกตัวแบ่งส่วนหลังรายการจำนวนหนึ่ง

```csharp
for (int i = 1; i < 45; i++)
{
    builder.Writeln($"List item {i}");

    if (i == 15)
        builder.InsertBreak(BreakType.SectionBreakNewPage);
}
```

## ขั้นตอนที่ 6: บันทึกเอกสาร

สุดท้าย ให้บันทึกเอกสารพร้อมตัวเลือกที่เหมาะสมเพื่อให้แน่ใจว่าเป็นไปตามข้อกำหนด

```csharp
OoxmlSaveOptions options = new OoxmlSaveOptions { Compliance = OoxmlCompliance.Iso29500_2008_Transitional };
doc.Save(dataDir + "ResetListAtEachSection.docx", options);
```

และคุณก็ได้แล้ว! ด้วยการทำตามขั้นตอนเหล่านี้ คุณสามารถรีสตาร์ทรายการในแต่ละส่วนในเอกสาร Word ของคุณได้อย่างง่ายดายโดยใช้ Aspose.Words for .NET คุณลักษณะนี้มีประโยชน์อย่างเหลือเชื่อสำหรับการสร้างเอกสารที่มีโครงสร้างที่ดีซึ่งต้องมีส่วนแยกกันโดยมีการกำหนดหมายเลขรายการของตัวเอง ด้วย Aspose.Words การจัดการงานดังกล่าวกลายเป็นเรื่องง่าย ช่วยให้คุณมุ่งเน้นไปที่การสร้างเนื้อหาคุณภาพสูงได้
