## นำเข้าเนมสเปซ

ก่อนอื่น เราต้องนำเข้าเนมสเปซที่จำเป็นก่อน เปิดโครงการ C# ของคุณและเพิ่มสิ่งต่อไปนี้โดยใช้คำสั่ง:

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Lists;
```

นี่เป็นการปูทางสำหรับการทำงานกับ Aspose.Words ในโปรเจ็กต์ของคุณ

## ขั้นตอนที่ 1: การตั้งค่าเอกสารและ DocumentBuilder

 เริ่มต้นด้วยการสร้างเอกสารใหม่และก`DocumentBuilder` คัดค้านที่จะทำงานร่วมกับมัน

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## ขั้นตอนที่ 2: การสร้างรายการลำดับเลข

 ตอนนี้ เราจะสร้างรายการลำดับเลขตามเทมเพลตรายการ Microsoft Word รายการใดรายการหนึ่งและนำไปใช้กับ`DocumentBuilder`'ย่อหน้าปัจจุบัน

```csharp
builder.ListFormat.List = doc.Lists.Add(ListTemplate.NumberArabicDot);
```

## ขั้นตอนที่ 3: การใช้หลายระดับรายการ

Aspose.Words ช่วยให้คุณสามารถระบุระดับรายการได้สูงสุดเก้าระดับ ลองใช้ทั้งหมดเพื่อดูว่ามันทำงานอย่างไร

```csharp
for (int i = 0; i < 9; i++)
{
    builder.ListFormat.ListLevelNumber = i;
    builder.Writeln("Level " + i);
}
```

ในวงจรนี้ เรากำลังตั้งค่าระดับรายการสำหรับแต่ละย่อหน้า และเขียนบรรทัดข้อความที่ระบุระดับ

## ขั้นตอนที่ 4: การสร้างรายการสัญลักษณ์แสดงหัวข้อย่อย

ต่อไป เรามาเปลี่ยนเกียร์และสร้างรายการสัญลักษณ์แสดงหัวข้อย่อย คราวนี้ เราจะใช้เทมเพลตรายการอื่น

```csharp
builder.ListFormat.List = doc.Lists.Add(ListTemplate.BulletDiamonds);
```

## ขั้นตอนที่ 5: การใช้หลายระดับกับรายการสัญลักษณ์แสดงหัวข้อย่อย

เช่นเดียวกับรายการลำดับเลข เราจะใช้หลายระดับกับรายการสัญลักษณ์แสดงหัวข้อย่อยของเรา

```csharp
for (int i = 0; i < 9; i++)
{
    builder.ListFormat.ListLevelNumber = i;
    builder.Writeln("Level " + i);
}
```

## ขั้นตอนที่ 6: การหยุดการจัดรูปแบบรายการ

สุดท้ายนี้ เรามาดูกันว่าเราจะหยุดการจัดรูปแบบรายการให้กลับไปเป็นข้อความปกติได้อย่างไร

```csharp
builder.ListFormat.List = null;
```

## ขั้นตอนที่ 7: บันทึกเอกสาร

หลังจากการทำงานหนักทั้งหมด ก็ถึงเวลาบันทึกเอกสารของเรา มาบันทึกด้วยชื่อที่มีความหมายกันเถอะ

```csharp
builder.Document.Save(dataDir + "WorkingWithList.SpecifyListLevel.docx");
```

แค่นั้นแหละ! คุณเพิ่งสร้างเอกสารที่มีโครงสร้างรายการที่ซับซ้อนโดยใช้ Aspose.Words สำหรับ .NET
