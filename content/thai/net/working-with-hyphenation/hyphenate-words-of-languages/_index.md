## นำเข้าเนมสเปซ

ก่อนอื่น มานำเข้าเนมสเปซที่จำเป็นกันก่อน ซึ่งจะช่วยในการเข้าถึงคลาสและวิธีการที่จำเป็นสำหรับการยัติภังค์

```csharp
using Aspose.Words;
using Aspose.Words.Hyphenation;
```

## ขั้นตอนที่ 1: โหลดเอกสารของคุณ

 คุณจะต้องระบุไดเร็กทอรีที่มีเอกสารของคุณอยู่ แทนที่`"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงไปยังเอกสารของคุณ

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "German text.docx");
```

## ขั้นตอนที่ 3: ลงทะเบียนพจนานุกรมการใส่ยัติภังค์

 Aspose.Words ต้องใช้พจนานุกรมการใส่ยัติภังค์สำหรับภาษาต่างๆ ตรวจสอบให้แน่ใจว่าคุณมี`.dic`ไฟล์สำหรับภาษาที่คุณต้องการใส่ยัติภังค์ ลงทะเบียนพจนานุกรมเหล่านี้โดยใช้`Hyphenation.RegisterDictionary` วิธี.

```csharp
Hyphenation.RegisterDictionary("en-US", dataDir + "hyph_en_US.dic");
Hyphenation.RegisterDictionary("de-CH", dataDir + "hyph_de_CH.dic");
```

## ขั้นตอนที่ 4: บันทึกเอกสาร

สุดท้าย ให้บันทึกเอกสารที่มีการใส่ยัติภังค์ในรูปแบบที่ต้องการ ที่นี่เราจะบันทึกเป็น PDF

```csharp
doc.Save(dataDir + "TreatmentByCesure.pdf");
```
