## นำเข้าเนมสเปซ

เริ่มต้นด้วยการนำเข้าเนมสเปซที่จำเป็นในโปรเจ็กต์ C# ของคุณ:
```csharp
using System;
using System.IO;
using Aspose.Words;
```

## ขั้นตอนที่ 1: แปลง Docx เป็น Byte Array

เมื่อต้องการแปลงไฟล์ Docx เป็นอาร์เรย์ไบต์ ให้ทำตามขั้นตอนเหล่านี้:
```csharp
// โหลดไฟล์ Docx จากดิสก์หรือสตรีม
Document doc = new Document("input.docx");

// บันทึกเอกสารลงใน MemoryStream
MemoryStream outStream = new MemoryStream();
doc.Save(outStream, SaveFormat.Docx);

// แปลง MemoryStream เป็นอาร์เรย์ไบต์
byte[] docBytes = outStream.ToArray();
```

## ขั้นตอนที่ 2: แปลง Byte Array กลับเป็นเอกสาร

หากต้องการแปลงอาร์เรย์ไบต์กลับเป็นออบเจ็กต์ Document:
```csharp
// แปลงอาร์เรย์ไบต์กลับเป็น MemoryStream
MemoryStream inStream = new MemoryStream(docBytes);

// โหลดเอกสารจาก MemoryStream
Document docFromBytes = new Document(inStream);
```
