using Aspose.Words;

## ขั้นตอนที่ 1: ตั้งค่าโครงการของคุณ

ก่อนที่จะเขียนโค้ดใดๆ เราจำเป็นต้องตั้งค่าโปรเจ็กต์ของเราใน Visual Studio ก่อน

### สร้างโครงการใหม่

เปิด Visual Studio และสร้างโครงการ C# Console App ใหม่ ตั้งชื่อสิ่งที่มีความหมายเช่น "WordToPdfConverter"

### ติดตั้ง Aspose.Words สำหรับ .NET

คุณต้องเพิ่มไลบรารี Aspose.Words สำหรับ .NET ให้กับโปรเจ็กต์ของคุณ คุณสามารถทำได้ผ่านทาง NuGet Package Manager 

1. คลิกขวาที่โครงการของคุณใน Solution Explorer
2. เลือก "จัดการแพ็คเกจ NuGet"
3.  ค้นหา`Aspose.Words` และติดตั้งมัน

## ขั้นตอนที่ 2: โหลดเอกสาร Word

ตอนนี้ มาโหลดเอกสาร Word ที่คุณต้องการแปลงเป็น PDF กัน

### กำหนดไดเร็กทอรีเอกสาร

ขั้นแรก ระบุไดเร็กทอรีที่มีเอกสาร Word ของคุณอยู่

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` ด้วยเส้นทางจริงไปยังเอกสาร Word ของคุณ

### โหลดเอกสาร

ถัดไป โหลดเอกสารโดยใช้ Aspose.Words

```csharp
Document doc = new Document(dataDir + "Document.docx");
```

 รหัสบรรทัดนี้เริ่มต้นใหม่`Document` วัตถุด้วยไฟล์ Word ของคุณ

## ขั้นตอนที่ 3: บันทึกเอกสารเป็น PDF

สุดท้าย เราจะบันทึกเอกสารที่โหลดเป็น PDF

นี่คือสายเวทย์มนตร์ที่ใช้งานได้ทั้งหมด:

```csharp
doc.Save(dataDir + "ConvertedDocument.pdf");
```

 วิธีนี้จะบันทึกเอกสารในไดเร็กทอรีที่ระบุเป็นไฟล์ PDF ที่ตั้งชื่อ`ConvertedDocument.pdf`.
