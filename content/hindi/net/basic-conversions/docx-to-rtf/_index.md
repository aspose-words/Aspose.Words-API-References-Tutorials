## चरण 2: दस्तावेज़ निर्देशिका आरंभ करें

इसके बाद, हमें वह निर्देशिका निर्दिष्ट करनी होगी जहाँ आपकी DOCX फ़ाइल स्थित है। इसका उपयोग DOCX फ़ाइल को पढ़ने और परिवर्तित RTF फ़ाइल को सहेजने के लिए किया जाएगा।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 3: DOCX दस्तावेज़ लोड करें

 अब, DOCX दस्तावेज़ लोड करते हैं। हम एक का उपयोग करते हैं`Stream` दस्तावेज़ को केवल पढ़ने के लिए मोड में खोलने के लिए.

```csharp
using (Stream stream = File.OpenRead(dataDir + "Document.docx"))
{
    Document doc = new Document(stream);
    // जब उपयोग ब्लॉक से बाहर निकला जाएगा तो स्ट्रीम स्वचालित रूप से बंद हो जाएगी।
}
```

## चरण 4: DOCX को RTF में बदलें

 दस्तावेज़ लोड करने के बाद, इसे RTF फ़ॉर्मेट में बदलने का समय आ गया है। हम दस्तावेज़ को RTF फ़ॉर्मेट में सेव करते हैं।`MemoryStream`.

```csharp
MemoryStream dstStream = new MemoryStream();
doc.Save(dstStream, SaveFormat.Rtf);
```

## चरण 5: RTF दस्तावेज़ सहेजें

अंत में, हमें परिवर्तित RTF दस्तावेज़ को निर्दिष्ट निर्देशिका में सहेजना होगा। सहेजने से पहले स्ट्रीम स्थिति को शून्य पर वापस लाना सुनिश्चित करें।

```csharp
dstStream.Position = 0;
File.WriteAllBytes(dataDir + "BaseConversions.DocxToRtf.rtf", dstStream.ToArray());
```
