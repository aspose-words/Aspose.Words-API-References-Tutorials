## परिचय

एक आश्चर्यजनक रिपोर्ट या प्रस्ताव बनाने की कल्पना करें जहां छवियां आपके पाठ के पूरक के लिए पूरी तरह से स्थित हों। .NET के लिए Aspose.Words के साथ, आप इसे आसानी से प्राप्त कर सकते हैं। यह लाइब्रेरी दस्तावेज़ हेरफेर के लिए शक्तिशाली सुविधाएँ प्रदान करती है, जिससे यह डेवलपर्स के लिए एक आसान समाधान बन जाती है। इस ट्यूटोरियल में, हम DocumentBuilder क्लास का उपयोग करके एक फ़्लोटिंग छवि डालने पर ध्यान केंद्रित करेंगे। चाहे आप एक अनुभवी डेवलपर हों या अभी शुरुआत कर रहे हों, यह मार्गदर्शिका आपको प्रत्येक चरण में मार्गदर्शन करेगी।

## आवश्यक शर्तें

इससे पहले कि हम आगे बढ़ें, आइए सुनिश्चित करें कि आरंभ करने के लिए आपके पास वह सब कुछ है जो आपको चाहिए:

1.  .NET के लिए Aspose.Words: आप लाइब्रेरी को यहां से डाउनलोड कर सकते हैं[Aspose ने पृष्ठ जारी किया](https://releases.aspose.com/words/net/).
2. विजुअल स्टूडियो: कोई भी संस्करण जो .NET विकास का समर्थन करता है।
3. C# का बुनियादी ज्ञान: C# प्रोग्रामिंग की मूल बातें समझना मददगार होगा।
4. छवि फ़ाइल: एक छवि फ़ाइल जिसे आप सम्मिलित करना चाहते हैं, जैसे कोई लोगो या चित्र।

## नामस्थान आयात करें

अपने प्रोजेक्ट में Aspose.Words का उपयोग करने के लिए, आपको आवश्यक नामस्थान आयात करने की आवश्यकता है। यह आपकी C# फ़ाइल के शीर्ष पर निम्नलिखित पंक्तियाँ जोड़कर किया जाता है:

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
```

इन पूर्वावश्यकताओं और नामस्थानों के साथ, हम अपना ट्यूटोरियल शुरू करने के लिए तैयार हैं।

आइए किसी Word दस्तावेज़ में फ़्लोटिंग छवि डालने की प्रक्रिया को प्रबंधनीय चरणों में विभाजित करें। यह सुनिश्चित करने के लिए कि आप बिना किसी रुकावट के आगे बढ़ सकें, प्रत्येक चरण को विस्तार से समझाया जाएगा।

## चरण 1: अपना प्रोजेक्ट सेट करें

सबसे पहले, विजुअल स्टूडियो में एक नया C# प्रोजेक्ट बनाएं। सरलता के लिए आप एक कंसोल ऐप चुन सकते हैं।

1. विज़ुअल स्टूडियो खोलें और एक नया प्रोजेक्ट बनाएं।
2. "कंसोल ऐप (.NET कोर)" चुनें और "अगला" पर क्लिक करें।
3. अपने प्रोजेक्ट को नाम दें और इसे सहेजने के लिए एक स्थान चुनें। "बनाएँ" पर क्लिक करें।
4. NuGet पैकेज मैनेजर के माध्यम से .NET के लिए Aspose.Words इंस्टॉल करें। सॉल्यूशन एक्सप्लोरर में अपने प्रोजेक्ट पर राइट-क्लिक करें, "न्यूगेट पैकेज प्रबंधित करें" चुनें और "Apose.Words" खोजें। नवीनतम संस्करण स्थापित करें.

## चरण 2: दस्तावेज़ और दस्तावेज़बिल्डर को प्रारंभ करें

अब जब आपका प्रोजेक्ट सेट हो गया है, तो आइए दस्तावेज़ और दस्तावेज़बिल्डर ऑब्जेक्ट को प्रारंभ करें।

1.  का एक नया उदाहरण बनाएं`Document` कक्षा:

```csharp
Document doc = new Document();
```

2. एक DocumentBuilder ऑब्जेक्ट प्रारंभ करें:

```csharp
DocumentBuilder builder = new DocumentBuilder(doc);
```

`Document` ऑब्जेक्ट Word दस्तावेज़ का प्रतिनिधित्व करता है, और`DocumentBuilder` इसमें सामग्री जोड़ने में मदद मिलती है।

## चरण 3: छवि पथ को परिभाषित करें

इसके बाद, अपनी छवि फ़ाइल का पथ निर्दिष्ट करें। सुनिश्चित करें कि आपकी छवि आपके प्रोजेक्ट की निर्देशिका से पहुंच योग्य है।

छवि निर्देशिका और छवि फ़ाइल नाम परिभाषित करें:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
string imagePath = dataDir + "Transparent background logo.png";
```

 प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` वास्तविक पथ के साथ जहां आपकी छवि संग्रहीत है।

## चरण 4: फ़्लोटिंग छवि डालें

सब कुछ सेट हो जाने के बाद, आइए दस्तावेज़ में फ़्लोटिंग छवि डालें।

 उपयोग`InsertImage` की विधि`DocumentBuilder` छवि सम्मिलित करने के लिए कक्षा:

```csharp
builder.InsertImage(imagePath,
   RelativeHorizontalPosition.Margin,
   100,
   RelativeVerticalPosition.Margin,
   100,
   200,
   100,
   WrapType.Square);
```

यहां प्रत्येक पैरामीटर का अर्थ बताया गया है:
- `imagePath`आपकी छवि फ़ाइल का पथ.
- `RelativeHorizontalPosition.Margin`: मार्जिन के सापेक्ष क्षैतिज स्थिति.
- `100`: मार्जिन से क्षैतिज ऑफसेट (अंकों में)।
- `RelativeVerticalPosition.Margin`: मार्जिन के सापेक्ष ऊर्ध्वाधर स्थिति.
- `100`: मार्जिन से ऊर्ध्वाधर ऑफसेट (अंकों में)।
- `200`: छवि की चौड़ाई (अंकों में)।
- `100`: छवि की ऊंचाई (अंकों में)।
- `WrapType.Square`: छवि के चारों ओर टेक्स्ट रैपिंग शैली।

## चरण 5: दस्तावेज़ सहेजें

अंत में, दस्तावेज़ को अपने इच्छित स्थान पर सहेजें।

1. आउटपुट फ़ाइल पथ निर्दिष्ट करें:

```csharp
string outputPath = dataDir + "AddContentUsingDocumentBuilder.InsertFloatingImage.docx";
```

2. दस्तावेज़ सहेजें:

```csharp
doc.Save(outputPath);
```

फ़्लोटिंग छवि वाला आपका वर्ड दस्तावेज़ अब तैयार है!

## निष्कर्ष

.NET के लिए Aspose.Words का उपयोग करके Word दस्तावेज़ में एक फ़्लोटिंग छवि सम्मिलित करना प्रबंधनीय चरणों में विभाजित होने पर एक सीधी प्रक्रिया है। इस गाइड का पालन करके, आप अपने दस्तावेज़ों में पेशेवर दिखने वाली छवियां जोड़ सकते हैं, जिससे उनकी दृश्य अपील बढ़ सकती है। Aspose.Words एक मजबूत एपीआई प्रदान करता है जो दस्तावेज़ में हेरफेर को आसान बनाता है, चाहे आप रिपोर्ट, प्रस्ताव या किसी अन्य दस्तावेज़ प्रकार पर काम कर रहे हों।

## अक्सर पूछे जाने वाले प्रश्न

### क्या मैं .NET के लिए Aspose.Words का उपयोग करके एकाधिक छवियां सम्मिलित कर सकता हूं?

 हाँ, आप इसे दोहराकर एकाधिक छवियाँ सम्मिलित कर सकते हैं`InsertImage` वांछित मापदंडों के साथ प्रत्येक छवि के लिए विधि।

### मैं छवि की स्थिति कैसे बदलूं?

 आप समायोजित कर सकते हैं`RelativeHorizontalPosition`, `RelativeVerticalPosition`, और आवश्यकतानुसार छवि की स्थिति के लिए ऑफसेट पैरामीटर।

### छवियों के लिए अन्य कौन से रैप प्रकार उपलब्ध हैं?

 Aspose.Words विभिन्न रैप प्रकारों का समर्थन करता है जैसे कि`Inline`, `TopBottom`, `Tight`, `Through`, और अधिक। आप वह चुन सकते हैं जो आपके दस्तावेज़ लेआउट के लिए सबसे उपयुक्त हो।

### क्या मैं विभिन्न छवि प्रारूपों का उपयोग कर सकता हूँ?

हां, Aspose.Words JPEG, PNG, BMP और GIF सहित छवि प्रारूपों की एक विस्तृत श्रृंखला का समर्थन करता है।

### मैं .NET के लिए Aspose.Words का निःशुल्क परीक्षण कैसे प्राप्त कर सकता हूँ?

 आप नि:शुल्क परीक्षण प्राप्त कर सकते हैं[निःशुल्क परीक्षण पृष्ठ का अनुमान लगाएं](https://releases.aspose.com/).
