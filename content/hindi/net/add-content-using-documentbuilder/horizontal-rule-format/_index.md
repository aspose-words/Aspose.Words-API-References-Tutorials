इस व्यापक उदाहरण में, आप सीखेंगे कि .NET के लिए Aspose.Words का उपयोग करके किसी Word दस्तावेज़ में क्षैतिज नियम को कैसे प्रारूपित किया जाए। हम प्रक्रिया में आपका मार्गदर्शन करेंगे और आपको आवश्यक C# कोड स्निपेट प्रदान करेंगे। इस गाइड के अंत तक, आप क्षैतिज नियम के संरेखण, चौड़ाई, ऊंचाई, रंग और अन्य गुणों को अनुकूलित करने में सक्षम होंगे।

## आवश्यक शर्तें
शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित शर्तें हैं:
- आपके सिस्टम पर .NET लाइब्रेरी के लिए Aspose.Words इंस्टॉल किया गया है।

## चरण 1: एक डॉक्यूमेंटबिल्डर बनाएं और एक क्षैतिज नियम डालें
प्रारंभ करने के लिए, एक DocumentBuilder ऑब्जेक्ट बनाएं और एक क्षैतिज नियम सम्मिलित करने के लिए InsertHrizontalRule विधि का उपयोग करें:

```csharp
DocumentBuilder builder = new DocumentBuilder();
Shape shape = builder.InsertHorizontalRule();
```

## चरण 2: क्षैतिज नियम प्रारूप तक पहुंचें
इसके बाद, फ़ॉर्मेटिंग विकल्पों को पुनः प्राप्त करने के लिए शेप ऑब्जेक्ट की HorizontalRuleFormat प्रॉपर्टी तक पहुंचें:

```csharp
HorizontalRuleFormat horizontalRuleFormat = shape.HorizontalRuleFormat;
```

## चरण 3: फ़ॉर्मेटिंग विकल्पों को अनुकूलित करें
अब, आप क्षैतिज नियम के लिए विभिन्न स्वरूपण विकल्पों को अनुकूलित कर सकते हैं। उदाहरण के लिए, आप संरेखण, चौड़ाई, ऊंचाई, रंग और छायांकन को समायोजित कर सकते हैं:

```csharp
horizontalRuleFormat.Alignment = HorizontalRuleAlignment.Center;
horizontalRuleFormat.WidthPercent = 70;
horizontalRuleFormat.Height = 3;
horizontalRuleFormat.Color = Color.Blue;
horizontalRuleFormat.NoShade = true;
```

## चरण 4: दस्तावेज़ सहेजें
क्षैतिज नियम को फ़ॉर्मेट करने के बाद, दस्तावेज़ ऑब्जेक्ट की सेव विधि का उपयोग करके दस्तावेज़ को फ़ाइल में सहेजें:

```csharp
builder.Document.Save(ArtifactsDir + "AddContentUsingDocumentBuilder.HorizontalRuleFormat.docx");
```

### .NET के लिए Aspose.Words का उपयोग करके क्षैतिज नियम प्रारूप के लिए उदाहरण स्रोत कोड
.NET के लिए Aspose.Words का उपयोग करके क्षैतिज नियम को स्वरूपित करने के लिए संपूर्ण स्रोत कोड यहां दिया गया है:

```csharp
DocumentBuilder builder = new DocumentBuilder();

Shape shape = builder.InsertHorizontalRule();

HorizontalRuleFormat horizontalRuleFormat = shape.HorizontalRuleFormat;
horizontalRuleFormat.Alignment = HorizontalRuleAlignment.Center;
horizontalRuleFormat.WidthPercent = 70;
horizontalRuleFormat.Height = 3;
horizontalRuleFormat.Color = Color.Blue;
horizontalRuleFormat.NoShade = true;

builder.Document.Save(ArtifactsDir + "AddContentUsingDocumentBuilder.HorizontalRuleFormat.docx");
```

अपनी विशिष्ट आवश्यकताओं के अनुसार कोड को समायोजित करना और आवश्यकतानुसार अतिरिक्त कार्यक्षमता के साथ इसे बढ़ाना याद रखें।

## निष्कर्ष
बधाई हो! आपने सफलतापूर्वक सीख लिया है कि .NET के लिए Aspose.Words का उपयोग करके किसी Word दस्तावेज़ में क्षैतिज नियम को कैसे प्रारूपित किया जाए। चरण-दर-चरण मार्गदर्शिका का पालन करके और दिए गए स्रोत कोड का उपयोग करके, अब आप अपने दस्तावेज़ के दृश्य लेआउट को बढ़ाने के लिए क्षैतिज नियमों की उपस्थिति को अनुकूलित कर सकते हैं।

अपने क्षैतिज नियमों के लिए वांछित शैली और प्रभाव प्राप्त करने के लिए विभिन्न स्वरूपण विकल्पों के साथ प्रयोग करें।

### वर्ड दस्तावेज़ में क्षैतिज नियम प्रारूप के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: क्या मैं क्षैतिज नियम पर अलग-अलग रंग लगा सकता हूँ?

उत्तर: बिल्कुल! .NET के लिए Aspose.Words के साथ, आप Color प्रॉपर्टी को वांछित रंग मान पर सेट करके क्षैतिज नियम के रंग को आसानी से अनुकूलित कर सकते हैं। यह आपको अपने दस्तावेज़ के समग्र डिज़ाइन के साथ क्षैतिज नियम का मिलान करने की अनुमति देता है।

#### प्रश्न: क्या क्षैतिज नियम की चौड़ाई और ऊंचाई को समायोजित करना संभव है?

उत्तर: हां, क्षैतिज नियम की चौड़ाई और ऊंचाई पर आपका पूरा नियंत्रण है। WidthPercent और Height गुणों को संशोधित करके, आप क्षैतिज नियम के लिए वांछित आयाम प्राप्त कर सकते हैं।

#### प्रश्न: क्या मैं दस्तावेज़ के भीतर क्षैतिज नियम का संरेखण बदल सकता हूँ?

उत्तर: निश्चित रूप से! .NET के लिए Aspose.Words आपको संरेखण संपत्ति का उपयोग करके क्षैतिज नियम के संरेखण को निर्दिष्ट करने में सक्षम बनाता है। आप सेंटर, लेफ्ट, राइट और जस्टिफाइड जैसे विभिन्न विकल्पों में से चुन सकते हैं।

#### प्रश्न: क्या मैं क्षैतिज नियम पर छायांकन या पृष्ठभूमि रंग लागू कर सकता हूँ?

उ: हाँ, आप क्षैतिज नियम में छायांकन या पृष्ठभूमि रंग जोड़ सकते हैं। डिफ़ॉल्ट रूप से, NoShade प्रॉपर्टी सत्य पर सेट है, लेकिन आप इसे गलत पर सेट कर सकते हैं और उचित तरीकों का उपयोग करके शेडिंग को परिभाषित कर सकते हैं।

#### प्रश्न: क्या मैं एक ही दस्तावेज़ में एकाधिक क्षैतिज नियम सम्मिलित कर सकता हूँ?

उत्तर: बिल्कुल! आप .NET के लिए Aspose.Words का उपयोग करके किसी Word दस्तावेज़ में एकाधिक क्षैतिज नियम सम्मिलित कर सकते हैं। जितनी आवश्यकता हो उतने क्षैतिज नियम जोड़ने के लिए बस ट्यूटोरियल में दिए गए चरणों को आवश्यकतानुसार दोहराएं।
