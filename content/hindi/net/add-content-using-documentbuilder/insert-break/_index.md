## नामस्थान आयात करें

सबसे पहले, आइए आवश्यक नेमस्पेस को आयात करें। यहीं से सारा जादू शुरू होता है।

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
using System;
```

## चरण 1: दस्तावेज़ निर्देशिका सेट अप करना

ठीक है, चलिए अपने डॉक्यूमेंट डायरेक्टरी का पथ सेट करके शुरू करते हैं। यहीं पर आपका वर्ड डॉक्यूमेंट सेव होगा।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` उस वास्तविक पथ के साथ जहाँ आप अपना दस्तावेज़ सहेजना चाहते हैं.

## चरण 2: नया दस्तावेज़ बनाना

 इसके बाद, हमें एक नया उदाहरण बनाना होगा`Document` क्लास। इसे अपने खाली कैनवास के रूप में सोचें जहाँ आप अपनी सामग्री जोड़ना शुरू कर सकते हैं।

```csharp
Document doc = new Document();
```

## चरण 3: डॉक्यूमेंटबिल्डर को आरंभ करना

`DocumentBuilder` यह आपके पेंटब्रश की तरह है। यह आपके दस्तावेज़ में सामग्री जोड़ने में आपकी मदद करता है। आइए इसे आरंभ करें।

```csharp
DocumentBuilder builder = new DocumentBuilder(doc);
```

## चरण 4: पहले पृष्ठ के लिए सामग्री लिखना

चलिए पहले पेज पर कुछ सामग्री जोड़ते हैं। यहीं पर आप रचनात्मक हो सकते हैं।

```csharp
builder.Writeln("This is page 1.");
```

## चरण 5: पेज ब्रेक सम्मिलित करना

 अब आता है मज़ेदार हिस्सा। हमें अगले पेज पर जाने के लिए पेज ब्रेक डालना होगा। यह कॉल करने जितना ही सरल है।`InsertBreak` तरीका।

```csharp
builder.InsertBreak(BreakType.PageBreak);
```

## चरण 6: दूसरे पेज के लिए सामग्री लिखना

पेज ब्रेक डालने के बाद, आइए दूसरे पेज पर कुछ सामग्री जोड़ें।

```csharp
builder.Writeln("This is page 2.");
```

## चरण 7: दूसरा पेज ब्रेक सम्मिलित करना

आइए तीसरे पेज पर जाने के लिए एक और पेज ब्रेक डालें।

```csharp
builder.InsertBreak(BreakType.PageBreak);
```

## चरण 8: तीसरे पेज के लिए सामग्री लिखना

अंत में, आइए तीसरे पृष्ठ पर कुछ सामग्री जोड़ें।

```csharp
builder.Writeln("This is page 3.");
```

## चरण 9: दस्तावेज़ को सहेजना

आखिरी लेकिन सबसे महत्वपूर्ण बात, हमें अपने दस्तावेज़ को सहेजना होगा। यहीं पर आपकी सारी मेहनत एक साथ आती है।

```csharp
doc.Save(dataDir + "AddContentUsingDocumentBuilder.InsertBreak.docx");
```
