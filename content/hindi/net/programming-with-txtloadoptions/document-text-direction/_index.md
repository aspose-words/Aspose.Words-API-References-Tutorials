इस ट्यूटोरियल में, हम .NET के लिए Aspose.Words के साथ "दस्तावेज़ टेक्स्ट दिशा" सुविधा के लिए प्रदान किए गए C# स्रोत कोड का पता लगाएंगे। यह सुविधा आपको दस्तावेज़ में पाठ की दिशा निर्दिष्ट करने देती है, जो विशेष रूप से उन भाषाओं के लिए उपयोगी है जो दाईं ओर से बाईं ओर लिखी जाती हैं, जैसे कि हिब्रू या अरबी।

## चरण 1: वातावरण की स्थापना

आरंभ करने से पहले, सुनिश्चित करें कि आपने .NET के लिए Aspose.Words के साथ अपना विकास वातावरण सेट अप कर लिया है। सुनिश्चित करें कि आपने आवश्यक संदर्भ जोड़ दिए हैं और उचित नामस्थान आयात कर लिए हैं।

## चरण 2: अपलोड विकल्प कॉन्फ़िगर करना

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

TxtLoadOptions loadOptions = new TxtLoadOptions { DocumentDirection = DocumentDirection. Auto };
```

 इस चरण में, हम दस्तावेज़ लोडिंग विकल्पों को कॉन्फ़िगर करते हैं। हम एक नया दस्तावेज़ बनाते हैं।`TxtLoadOptions` ऑब्जेक्ट और सेट करें`DocumentDirection`संपत्ति को`DocumentDirection.Auto`यह मान Aspose.Words को दस्तावेज़ की सामग्री के आधार पर पाठ की दिशा स्वचालित रूप से निर्धारित करने के लिए कहता है।

## चरण 3: दस्तावेज़ लोड करना

```csharp
Document doc = new Document(dataDir + "Hebrew text.txt", loadOptions);
```

 इस चरण में, हम दस्तावेज़ को लोड करते हैं`Document` विधि और लोड करने के लिए पाठ फ़ाइल के पथ को पास करना। हम निर्दिष्ट लोडिंग विकल्पों का भी उपयोग करते हैं।

## चरण 4: पैराग्राफ़ में बदलाव करें और टेक्स्ट की दिशा प्रदर्शित करें

```csharp
Paragraph paragraph = doc.FirstSection.Body.FirstParagraph;
Console.WriteLine(paragraph.ParagraphFormat.Bidi);
```

 इस चरण में, हम दस्तावेज़ के पहले पैराग्राफ तक पहुँचते हैं`FirstSection` और`Body` गुण। इसके बाद, हम पहुँचते हैं`ParagraphFormat.Bidi` पैराग्राफ़ की टेक्स्ट दिशा प्राप्त करने के लिए प्रॉपर्टी का उपयोग करें। फिर हम इस मान को कंसोल में प्रदर्शित करते हैं।

## चरण 5: दस्तावेज़ सहेजें

```csharp
doc.Save(dataDir + "WorkingWithTxtLoadOptions.DocumentTextDirection.docx");
```

 इस अंतिम चरण में, हम परिणामी दस्तावेज़ को .docx प्रारूप में सहेजते हैं`Save` विधि और आउटपुट फ़ाइल के लिए पथ पारित करना।

अब आप टेक्स्ट डॉक्यूमेंट को लोड करने और टेक्स्ट की दिशा निर्धारित करने के लिए सोर्स कोड चला सकते हैं। परिणामी दस्तावेज़ "WorkingWithTxtLoadOptions.DocumentTextDirection.docx" नाम से निर्दिष्ट निर्देशिका में सहेजा जाएगा।

### .NET के लिए Aspose.Words के साथ दस्तावेज़ पाठ दिशा कार्यक्षमता के लिए नमूना स्रोत कोड।


```csharp

            
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";

TxtLoadOptions loadOptions = new TxtLoadOptions { DocumentDirection = DocumentDirection.Auto };

Document doc = new Document(dataDir + "Hebrew text.txt", loadOptions);

Paragraph paragraph = doc.FirstSection.Body.FirstParagraph;
Console.WriteLine(paragraph.ParagraphFormat.Bidi);

doc.Save(dataDir + "WorkingWithTxtLoadOptions.DocumentTextDirection.docx");
            
        
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने .NET के लिए Aspose.Words में दस्तावेज़ टेक्स्ट दिशा सुविधा का पता लगाया। हमने सीखा कि दस्तावेज़ में टेक्स्ट की दिशा कैसे निर्दिष्ट करें, खासकर उन भाषाओं के लिए जो दाईं से बाईं ओर लिखी जाती हैं, जैसे कि हिब्रू या अरबी।

यह सुविधा यह सुनिश्चित करने के लिए आवश्यक है कि बहुभाषी दस्तावेज़ों में पाठ सही ढंग से प्रदर्शित हो। उपयुक्त लोडिंग विकल्पों का उपयोग करके, Aspose.Words स्वचालित रूप से पाठ की दिशा का पता लगा सकता है और इसे दस्तावेज़ पर लागू कर सकता है।

Aspose.Words के साथ, आप अपने दस्तावेज़ों में पाठ की दिशा को आसानी से बदल सकते हैं, जिससे उपयोगकर्ताओं को एक सहज और सहज पढ़ने का अनुभव मिल सकता है।

यह ध्यान रखना महत्वपूर्ण है कि यह सुविधा विशेष रूप से तब उपयोगी होती है जब ऐसी भाषाओं के साथ वर्ड प्रोसेसिंग की जाती है जिसके लिए विशिष्ट टेक्स्ट दिशा की आवश्यकता होती है। Aspose.Words आपके दस्तावेज़ों में टेक्स्ट की दिशा को प्रबंधित करने के लिए शक्तिशाली उपकरण प्रदान करके इस कार्य को आसान बनाता है।

अपने दस्तावेज़ों में वांछित परिणाम प्राप्त करने के लिए उपयुक्त लोडिंग विकल्पों का उपयोग करना याद रखें, जैसे स्वचालित पाठ दिशा सेट करना।

Aspose.Words for .NET दस्तावेज़ हेरफेर और निर्माण के लिए कई उन्नत सुविधाएँ प्रदान करता है। Aspose.Words द्वारा प्रदान किए गए दस्तावेज़ों और उदाहरणों की और अधिक खोज करके, आप इस शक्तिशाली लाइब्रेरी की क्षमताओं का पूरी तरह से दोहन करने में सक्षम होंगे।
