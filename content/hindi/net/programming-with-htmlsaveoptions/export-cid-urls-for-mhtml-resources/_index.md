## चरण 3: HTML सेव विकल्प कॉन्फ़िगर करना

 इसका एक उदाहरण बनाएं`HtmlSaveOptions` अपने दस्तावेज़ को MHTML के रूप में सहेजने के तरीके को अनुकूलित करने के लिए क्लिक करें.

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.Mhtml)
{
    PrettyFormat = true,
    ExportCidUrlsForMhtmlResources = true
};
```

- `SaveFormat.Mhtml` निर्दिष्ट करता है कि आउटपुट स्वरूप MHTML है.
- `PrettyFormat = true` यह सुनिश्चित करता है कि आउटपुट सुव्यवस्थित रूप से स्वरूपित हो।
- `ExportCidUrlsForMhtmlResources = true` MHTML संसाधनों के लिए Cid URL के निर्यात को सक्षम करता है।

### चरण 4: दस्तावेज़ को MHTML के रूप में सहेजना
