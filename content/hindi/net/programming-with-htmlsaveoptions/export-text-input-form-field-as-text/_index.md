## परिचय

तो, आप .NET के लिए Aspose.Words की दुनिया में गोता लगा रहे हैं? शानदार विकल्प! यदि आप यह जानना चाहते हैं कि टेक्स्ट इनपुट फ़ॉर्म फ़ील्ड को टेक्स्ट के रूप में कैसे निर्यात किया जाए, तो आप सही जगह पर हैं। चाहे आप अभी शुरुआत कर रहे हों या अपने कौशल को निखार रहे हों, यह गाइड आपको वह सब कुछ बताएगी जो आपको जानना चाहिए। चलिए शुरू करते हैं, है न?

## आवश्यक शर्तें

इससे पहले कि हम बारीकियों में उतरें, आइए यह सुनिश्चित करें कि आपके पास सुचारू रूप से आगे बढ़ने के लिए आवश्यक सभी चीजें हैं:

-  Aspose.Words for .NET: यहां से नवीनतम संस्करण डाउनलोड करें और इंस्टॉल करें[यहाँ](https://releases.aspose.com/words/net/).
- IDE: विजुअल स्टूडियो या कोई भी C# विकास वातावरण।
- बुनियादी C# ज्ञान: बुनियादी C# सिंटैक्स और ऑब्जेक्ट-ओरिएंटेड प्रोग्रामिंग अवधारणाओं की समझ।
- दस्तावेज़: एक नमूना वर्ड दस्तावेज़ (`Rendering.docx`) टेक्स्ट इनपुट फॉर्म फ़ील्ड के साथ।

## नामस्थान आयात करें

सबसे पहले, आपको आवश्यक नेमस्पेस आयात करने की आवश्यकता है। ये बिल्डिंग ब्लॉक की तरह हैं जो सब कुछ सहजता से काम करते हैं।

```csharp
using System;
using System.IO;
using Aspose.Words;
using Aspose.Words.Saving;
```

ठीक है, अब जब हमारे पास नामस्थान तैयार हैं, तो चलिए कार्य शुरू करते हैं!

## चरण 1: प्रोजेक्ट सेट अप करें

कोड में प्रवेश करने से पहले, आइए सुनिश्चित करें कि हमारा प्रोजेक्ट सही ढंग से सेट किया गया है।

## प्रोजेक्ट बनाना

1. विज़ुअल स्टूडियो खोलें: विज़ुअल स्टूडियो या अपने पसंदीदा C# विकास वातावरण को खोलकर शुरुआत करें।
2.  नया प्रोजेक्ट बनाएं: यहां जाएं`File > New > Project` । चुनना`Console App (.NET Core)` या कोई अन्य प्रासंगिक परियोजना प्रकार।
3.  अपने प्रोजेक्ट को नाम दें: अपने प्रोजेक्ट को एक सार्थक नाम दें, कुछ इस तरह`AsposeWordsExportExample`.

## Aspose.Words जोड़ना

1.  NuGet पैकेज प्रबंधित करें: समाधान एक्सप्लोरर में अपने प्रोजेक्ट पर राइट-क्लिक करें और चुनें`Manage NuGet Packages`.
2.  Aspose.Words के लिए खोजें: NuGet पैकेज मैनेजर में, खोजें`Aspose.Words`.
3.  Aspose.Words स्थापित करें: पर क्लिक करें`Install` अपने प्रोजेक्ट में Aspose.Words लाइब्रेरी जोड़ने के लिए.

## चरण 2: वर्ड दस्तावेज़ लोड करें

अब जबकि हमारा प्रोजेक्ट सेट हो गया है, आइए Word दस्तावेज़ को लोड करें जिसमें टेक्स्ट इनपुट फ़ॉर्म फ़ील्ड शामिल हैं।

1. दस्तावेज़ निर्देशिका निर्दिष्ट करें: उस निर्देशिका का पथ परिभाषित करें जहाँ आपका दस्तावेज़ संग्रहीत है।
2.  दस्तावेज़ लोड करें: का उपयोग करें`Document` अपने वर्ड दस्तावेज़ को लोड करने के लिए क्लास का उपयोग करें।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Rendering.docx");
```

## चरण 3: निर्यात निर्देशिका तैयार करें

निर्यात करने से पहले, आइए सुनिश्चित करें कि हमारी निर्यात निर्देशिका तैयार है। यहीं पर हमारी HTML फ़ाइल और छवियाँ सहेजी जाएँगी।

1. निर्यात निर्देशिका निर्धारित करें: वह पथ निर्दिष्ट करें जहां निर्यात की गई फ़ाइलें सहेजी जाएंगी.
2. निर्देशिका की जाँच करें और उसे साफ़ करें: सुनिश्चित करें कि निर्देशिका मौजूद है और खाली है।

```csharp
string imagesDir = Path.Combine(dataDir, "Images");

if (Directory.Exists(imagesDir))
    Directory.Delete(imagesDir, true);

Directory.CreateDirectory(imagesDir);
```

## चरण 4: सहेजें विकल्प कॉन्फ़िगर करें

यहाँ पर जादू होता है। हमें टेक्स्ट इनपुट फॉर्म फ़ील्ड को सादे टेक्स्ट के रूप में निर्यात करने के लिए अपने सेव ऑप्शन को सेट करना होगा।

1.  सहेजें विकल्प बनाएँ: एक नया प्रारंभ करें`HtmlSaveOptions` वस्तु।
2.  निर्यात टेक्स्ट विकल्प सेट करें: कॉन्फ़िगर करें`ExportTextInputFormFieldAsText`संपत्ति को`true`.
3. छवियाँ फ़ोल्डर सेट करें: वह फ़ोल्डर निर्धारित करें जहाँ छवियाँ सहेजी जाएँगी।

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.Html)
{
    ExportTextInputFormFieldAsText = true,
    ImagesFolder = imagesDir
};
```

## चरण 5: दस्तावेज़ को HTML के रूप में सहेजें

अंत में, आइए हमारे कॉन्फ़िगर किए गए सेव विकल्पों का उपयोग करके Word दस्तावेज़ को HTML फ़ाइल के रूप में सेव करें।

1. आउटपुट पथ निर्धारित करें: वह पथ निर्दिष्ट करें जहां HTML फ़ाइल सहेजी जाएगी।
2.  दस्तावेज़ सहेजें: का उपयोग करें`Save` की विधि`Document`दस्तावेज़ निर्यात करने के लिए क्लास का उपयोग करें.

```csharp
doc.Save(dataDir + "ExportedDocument.html", saveOptions);
```
