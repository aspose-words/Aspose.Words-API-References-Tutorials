इस ट्यूटोरियल में, हम आपको बताएंगे कि .NET के लिए Aspose.Words की इन्सर्ट डॉक्यूमेंट व्हेन रिप्लेसिंग सुविधा का उपयोग करते समय किसी दस्तावेज़ को दूसरे दस्तावेज़ में कैसे सम्मिलित किया जाए। स्रोत कोड को समझने और दस्तावेज़ प्रविष्टि करने के लिए नीचे दिए गए चरणों का पालन करें।

## चरण 1: मुख्य दस्तावेज़ लोड हो रहा है

आरंभ करने के लिए, अपने दस्तावेज़ों के लिए निर्देशिका निर्दिष्ट करें और मुख्य दस्तावेज़ को दस्तावेज़ ऑब्जेक्ट में लोड करें। ऐसे:

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document mainDoc = new Document(MyDir + "Document insert 1.docx");
```

## चरण 2: खोज कॉन्फ़िगर करें और विकल्प बदलें

अब हम एक दस्तावेज़ को दूसरे दस्तावेज़ में सम्मिलित करने के लिए खोज दिशा और प्रतिस्थापन कॉलबैक निर्दिष्ट करके खोजने और बदलने के विकल्पों को कॉन्फ़िगर करेंगे। ऐसे:

```csharp
// खोज कॉन्फ़िगर करें और विकल्प बदलें.
FindReplaceOptions options = new FindReplaceOptions
{
Direction = FindReplaceDirection.Backward,
ReplacingCallback = new InsertDocumentAtReplaceHandler()
};
```

## चरण 3: प्रतिस्थापन विधि को कॉल करना

अब हम कॉन्फ़िगर किए गए विकल्पों का उपयोग करके निर्दिष्ट टेक्स्ट को एक खाली स्ट्रिंग के साथ ढूंढने और बदलने के लिए रिप्लेस विधि को कॉल करेंगे। ऐसे:

```csharp
mainDoc.Range.Replace(new Regex("\\[MY_DOCUMENT\\]"), "", options);
mainDoc.Save(dataDir + "CloneAndCombineDocuments.InsertDocumentAtReplace.docx");
```

### .NET के लिए Aspose.Words का उपयोग करके रिप्लेस पर दस्तावेज़ सम्मिलित करने के लिए उदाहरण स्रोत कोड

.NET के लिए Aspose.Words को प्रतिस्थापित करते समय सम्मिलित दस्तावेज़ सुविधा के लिए संपूर्ण स्रोत कोड यहां दिया गया है:

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document mainDoc = new Document(MyDir + "Document insertion 1.docx");

// ढूँढें और बदलें विकल्प सेट करें।
FindReplaceOptions options = new FindReplaceOptions
{
	Direction = FindReplaceDirection.Backward, 
	ReplacingCallback = new InsertDocumentAtReplaceHandler()
};

// प्रतिस्थापित विधि को कॉल करें.
mainDoc.Range.Replace(new Regex("\\[MY_DOCUMENT\\]"), "", options);
mainDoc.Save(dataDir + "CloneAndCombineDocuments.InsertDocumentAtReplace.docx");
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने पता लगाया कि .NET के लिए Aspose.Words के इन्सर्ट डॉक्यूमेंट व्हेन रिप्लेसिंग फीचर का उपयोग करके रिप्लेस के दौरान किसी दस्तावेज़ को दूसरे दस्तावेज़ में कैसे सम्मिलित किया जाए। ढूंढने और बदलने के विकल्पों को कॉन्फ़िगर करके और आवश्यक डेटा प्रदान करके, आप विशिष्ट प्लेसहोल्डर्स को अन्य दस्तावेज़ टेम्पलेट्स या अनुभागों की सामग्री के साथ प्रतिस्थापित करके दस्तावेज़ों को गतिशील रूप से इकट्ठा कर सकते हैं। .NET के लिए Aspose.Words जटिल दस्तावेज़ हेरफेर कार्यों को प्रबंधित करने का एक शक्तिशाली और लचीला तरीका प्रदान करता है, जो इसे दस्तावेज़ निर्माण और सामग्री प्रविष्टि परिदृश्यों को स्वचालित करने के लिए एक मूल्यवान उपकरण बनाता है।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: प्रतिस्थापन के दौरान किसी दस्तावेज़ को दूसरे दस्तावेज़ में सम्मिलित करने का क्या उद्देश्य है?

उ: प्रतिस्थापन के दौरान किसी दस्तावेज़ को दूसरे दस्तावेज़ में सम्मिलित करने से आप किसी विशिष्ट प्लेसहोल्डर को एक अलग दस्तावेज़ की सामग्री के साथ गतिशील रूप से प्रतिस्थापित कर सकते हैं। यह सुविधा विशेष रूप से तब उपयोगी होती है जब आप विभिन्न पूर्व-निर्धारित दस्तावेज़ टेम्पलेट्स या अनुभागों को विशिष्ट प्लेसहोल्डर्स में संयोजित करके एक बड़े दस्तावेज़ को इकट्ठा करना चाहते हैं।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके रिप्लेस के दौरान किसी दस्तावेज़ को दूसरे दस्तावेज़ में कैसे सम्मिलित करूँ?

उ: .NET के लिए Aspose.Words का उपयोग करके प्रतिस्थापित करने के दौरान किसी दस्तावेज़ को दूसरे दस्तावेज़ में सम्मिलित करने के लिए, इन चरणों का पालन करें:
1. प्लेसहोल्डर्स वाले मुख्य दस्तावेज़ को दस्तावेज़ ऑब्जेक्ट में लोड करें।
2. दस्तावेज़ प्रविष्टि को संभालने के लिए खोज दिशा और कॉलबैक को प्रतिस्थापित करने सहित खोजने और बदलने के विकल्पों को कॉन्फ़िगर करें।
3. कॉन्फ़िगर किए गए विकल्पों का उपयोग करके, प्लेसहोल्डर्स को एक खाली स्ट्रिंग के साथ प्रतिस्थापित करते हुए, उचित खोज पैटर्न के साथ रिप्लेस विधि को कॉल करें।

#### प्रश्न: क्या मैं प्रतिस्थापन के दौरान सम्मिलन व्यवहार को अनुकूलित कर सकता हूँ?

उत्तर: हाँ, आप कस्टम रिप्लेसिंगकॉलबैक लागू करके रिप्लेस के दौरान सम्मिलन व्यवहार को अनुकूलित कर सकते हैं। IReplacingCallback इंटरफ़ेस से इनहेरिट करके, आप यह नियंत्रित कर सकते हैं कि प्लेसहोल्डर्स को प्रतिस्थापित करते समय आपकी विशिष्ट आवश्यकताओं के आधार पर दस्तावेज़ कैसे डाले और मर्ज किए जाते हैं।

#### प्रश्न: क्या मैं अनेक प्लेसहोल्डर्स को विभिन्न दस्तावेज़ों से बदल सकता हूँ?

उ: हां, आप प्रत्येक प्लेसहोल्डर के लिए उपयुक्त खोज पैटर्न निर्दिष्ट करके और सम्मिलित किए जाने वाले संबंधित दस्तावेज़ प्रदान करके एकाधिक प्लेसहोल्डर्स को विभिन्न दस्तावेज़ों से बदल सकते हैं।
