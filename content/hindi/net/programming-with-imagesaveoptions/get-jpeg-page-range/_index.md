इस ट्यूटोरियल में, हम .NET के लिए Aspose.Words के साथ "JPEG पेजों की रेंज प्राप्त करें" सुविधा के लिए प्रदान किए गए C# स्रोत कोड का पता लगाएंगे। यह सुविधा आपको दस्तावेज़ के पृष्ठों की एक विशिष्ट श्रेणी को JPEG प्रारूप में छवियों में बदलने की अनुमति देती है।

## चरण 1: वातावरण स्थापित करना

शुरू करने से पहले, सुनिश्चित करें कि आपने .NET के लिए Aspose.Words के साथ अपना विकास वातावरण स्थापित कर लिया है। सुनिश्चित करें कि आपने आवश्यक संदर्भ जोड़ दिए हैं और उचित नामस्थान आयात कर लिए हैं।

## चरण 2: दस्तावेज़ लोड करना

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

Document doc = new Document(dataDir + "Rendering.docx");
```

 इस चरण में, हम का उपयोग करके दस्तावेज़ को लोड करते हैं`Document` विधि और लोड करने के लिए DOCX फ़ाइल का पथ पास करना।

## चरण 3: छवि बैकअप विकल्प कॉन्फ़िगर करें

```csharp
ImageSaveOptions options = new ImageSaveOptions(SaveFormat.Jpeg);
options. PageSet = new PageSet(0);
options. ImageBrightness = 0.3f;
options. ImageContrast = 0.7f;
options. HorizontalResolution = 72f;
```

 इस चरण में, हम छवियों के लिए बैकअप विकल्प कॉन्फ़िगर करते हैं। हम एक नया बनाते हैं`ImageSaveOptions` वांछित सेव फॉर्मेट निर्दिष्ट करने वाली वस्तु, यहां JPEG फॉर्मेट के लिए "Jpeg"। हम इसका उपयोग करके रूपांतरित करने के लिए पृष्ठों की श्रेणी भी निर्धारित करते हैं`PageSet`वस्तु। अंत में, हम इसका उपयोग करके छवि की चमक और कंट्रास्ट को समायोजित करते हैं`ImageBrightness` और`ImageContrast` गुण, क्रमशः. हम इसका उपयोग करके क्षैतिज रिज़ॉल्यूशन भी बदलते हैं`HorizontalResolution` संपत्ति।

## चरण 4: छवियों का बैकअप लेना

```csharp
doc.Save(dataDir + "WorkingWithImageSaveOptions.GetJpegPageRange.jpeg", options);
```

 इस अंतिम चरण में, हम निर्दिष्ट पृष्ठ श्रेणी की छवियों को JPEG प्रारूप में सहेजते हैं`Save` निर्दिष्ट सेव विकल्पों के साथ, विधि और आउटपुट फ़ाइल के लिए पथ पास करना।

अब आप अपने दस्तावेज़ में पृष्ठों की एक विशिष्ट श्रेणी को JPEG छवियों में बदलने के लिए स्रोत कोड चला सकते हैं। परिणामी फ़ाइल निर्दिष्ट निर्देशिका में "WorkingWithImageSaveOptions.GetJpegPageRange.jpeg" नाम से सहेजी जाएगी।

### .NET के लिए Aspose.Words का उपयोग करके Jpeg पेज रेंज प्राप्त करने के लिए नमूना स्रोत कोड

```csharp 
 // आपकी दस्तावेज़ निर्देशिका का पथ
 string dataDir = "YOUR DOCUMENT DIRECTORY"; 

Document doc = new Document(dataDir + "Rendering.docx");

ImageSaveOptions options = new ImageSaveOptions(SaveFormat.Jpeg);

// किसी दस्तावेज़ के केवल पहले पृष्ठ को परिवर्तित करने के लिए "पेजसेट" को "0" पर सेट करें।
options.PageSet = new PageSet(0);

// छवि की चमक और कंट्रास्ट बदलें।
// दोनों 0-1 पैमाने पर हैं और डिफ़ॉल्ट रूप से 0.5 पर हैं।
options.ImageBrightness = 0.3f;
options.ImageContrast = 0.7f;

// क्षैतिज रिज़ॉल्यूशन बदलें.
// 96dpi के रिज़ॉल्यूशन के लिए इन गुणों का डिफ़ॉल्ट मान 96.0 है।
options.HorizontalResolution = 72f;

doc.Save(dataDir + "WorkingWithImageSaveOptions.GetJpegPageRange.jpeg", options);
            
        
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने .NET के लिए Aspose.Words के साथ JPEG पेज रेंज प्राप्त करने की कार्यक्षमता का पता लगाया। हमने सीखा कि सेव विकल्पों को अनुकूलित करते हुए किसी दस्तावेज़ के पृष्ठों की एक विशिष्ट श्रेणी को JPEG प्रारूप में छवियों में कैसे परिवर्तित किया जाए।

यह सुविधा तब उपयोगी होती है जब आप किसी दस्तावेज़ से विशिष्ट पृष्ठ निकालना चाहते हैं और उन्हें JPEG छवियों के रूप में सहेजना चाहते हैं। वैयक्तिकृत परिणाम प्राप्त करने के लिए आप छवियों की चमक, कंट्रास्ट और क्षैतिज रिज़ॉल्यूशन को भी समायोजित कर सकते हैं।

.NET के लिए Aspose.Words दस्तावेज़ हेरफेर और निर्माण के लिए उन्नत सुविधाओं की एक विस्तृत श्रृंखला प्रदान करता है। JPEG पेज रेंज प्राप्त करना आपके लिए उपलब्ध कई शक्तिशाली उपकरणों में से एक है।

अपने दस्तावेज़ों से उच्च गुणवत्ता वाली JPEG छवियाँ प्राप्त करने के लिए .NET परियोजनाओं के लिए इस सुविधा को अपने Aspose.Words में एकीकृत करने के लिए स्वतंत्र महसूस करें।
