## परिचय

दस्तावेज़ प्रसंस्करण की दुनिया में, द्विदिशात्मक (बीडी) पाठ को प्रबंधित करना अक्सर थोड़ा मुश्किल हो सकता है। यह विशेष रूप से उन भाषाओं के साथ काम करते समय सच है जिनमें अलग-अलग पाठ दिशाएँ हैं, जैसे कि अरबी या हिब्रू। सौभाग्य से, .NET के लिए Aspose.Words ऐसे परिदृश्यों को संभालना आसान बनाता है। इस ट्यूटोरियल में, हम Aspose.Words for .NET का उपयोग करके Word दस्तावेज़ में बीडी चिह्न जोड़ने का तरीका बताएंगे।

## आवश्यक शर्तें

इससे पहले कि हम कोड में उतरें, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1. Aspose.Words for .NET: आपके पास Aspose.Words for .NET इंस्टॉल होना चाहिए। आप इसे यहाँ से डाउनलोड कर सकते हैं[Aspose डाउनलोड पृष्ठ](https://releases.aspose.com/words/net/).
2. .NET फ्रेमवर्क या .NET कोर: सुनिश्चित करें कि आपके पास उदाहरणों को चलाने के लिए एक संगत .NET वातावरण स्थापित है।
3. C# का बुनियादी ज्ञान: C# प्रोग्रामिंग भाषा और .NET में बुनियादी संचालन से परिचित होना।

## नामस्थान आयात करें

आरंभ करने के लिए, आपको आवश्यक नामस्थान आयात करने होंगे। यहां बताया गया है कि आप उन्हें अपने प्रोजेक्ट में कैसे शामिल कर सकते हैं:

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

आइए वर्ड डॉक्यूमेंट में बीड़ी मार्क जोड़ने की प्रक्रिया को स्पष्ट चरणों में विभाजित करें। प्रत्येक चरण आपको कोड और उसके उद्देश्य के बारे में बताएगा।

## चरण 1: अपना दस्तावेज़ सेट करें

 एक नया उदाहरण बनाकर शुरू करें`Document` कक्षा और एक`DocumentBuilder` दस्तावेज़ में सामग्री जोड़ने के लिए.

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// दस्तावेज़ बनाएँ और सामग्री जोड़ें
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 इस चरण में, आप एक नया Word दस्तावेज़ आरंभ करते हैं और एक सेटअप करते हैं`DocumentBuilder` सामग्री प्रविष्टि को सुविधाजनक बनाने के लिए।

## चरण 2: अपने दस्तावेज़ में सामग्री जोड़ें

इसके बाद, अपने दस्तावेज़ में कुछ टेक्स्ट जोड़ें। यहाँ, हम बिडी टेक्स्ट हैंडलिंग को दर्शाने के लिए अलग-अलग भाषाओं में टेक्स्ट जोड़ेंगे।

```csharp
builder.Writeln("Hello world!");
builder.ParagraphFormat.Bidi = true;
builder.Writeln("שלום עולם!");
builder.Writeln("مرحبا بالعالم!");
```

यहाँ, हम पहले एक मानक अंग्रेजी वाक्यांश जोड़ते हैं। फिर, हम बाद के पाठ के लिए बिडी पाठ स्वरूपण सक्षम करते हैं, जो हिब्रू और अरबी में लिखा गया है। यह दर्शाता है कि द्विदिश पाठ को कैसे शामिल किया जाए।

## चरण 3: बीड़ी चिह्नों के लिए सेव विकल्प कॉन्फ़िगर करें

 यह सुनिश्चित करने के लिए कि दस्तावेज़ में बीड़ी चिह्न सही ढंग से सहेजे गए हैं, आपको कॉन्फ़िगर करने की आवश्यकता है`TxtSaveOptions` और सक्षम करें`AddBidiMarks` विकल्प।

```csharp
// बीड़ी के निशान जोड़ें
TxtSaveOptions saveOptions = new TxtSaveOptions { AddBidiMarks = true };
doc.Save(dataDir + "WorkingWithTxtSaveOptions.AddBidiMarks.txt", saveOptions);
```

 इस चरण में, हम एक उदाहरण बनाते हैं`TxtSaveOptions` और सेट करें`AddBidiMarks`संपत्ति को`true`यह सुनिश्चित करता है कि दस्तावेज़ को टेक्स्ट फ़ाइल के रूप में सहेजते समय बीड़ी चिह्न शामिल किए जाएं।

## निष्कर्ष

अपने वर्ड दस्तावेज़ों में बिडी चिह्न जोड़ना बहुभाषी सामग्री से निपटने के दौरान एक महत्वपूर्ण कदम हो सकता है जिसमें विभिन्न पाठ दिशाओं वाली भाषाएँ शामिल हैं। .NET के लिए Aspose.Words के साथ, यह प्रक्रिया सीधी और कुशल है। ऊपर बताए गए चरणों का पालन करके, आप यह सुनिश्चित कर सकते हैं कि आपके दस्तावेज़ बिडी पाठ को सही ढंग से प्रस्तुत करते हैं, जिससे पठनीयता और सटीकता बढ़ती है।

## अक्सर पूछे जाने वाले प्रश्न

### बीड़ी मार्क क्या हैं और वे महत्वपूर्ण क्यों हैं?
बीड़ी चिह्न विशेष वर्ण होते हैं जिनका उपयोग दस्तावेज़ों में पाठ की दिशा को नियंत्रित करने के लिए किया जाता है। वे अरबी और हिब्रू जैसी भाषाओं को सही ढंग से प्रदर्शित करने के लिए आवश्यक हैं जो दाईं से बाईं ओर पढ़ी जाती हैं।

### क्या मैं अन्य प्रकार के पाठ दिशा मुद्दों को संभालने के लिए .NET के लिए Aspose.Words का उपयोग कर सकता हूं?
हां, .NET के लिए Aspose.Words विभिन्न पाठ दिशा और स्वरूपण आवश्यकताओं के लिए व्यापक समर्थन प्रदान करता है, जिसमें दाएं से बाएं और बाएं से दाएं भाषाएं शामिल हैं।

### क्या किसी दस्तावेज़ के केवल विशिष्ट भागों पर ही बीड़ी फ़ॉर्मेटिंग लागू करना संभव है?
हां, आप आवश्यकतानुसार अपने दस्तावेज़ के विशिष्ट पैराग्राफ़ या अनुभागों पर बीड़ी फ़ॉर्मेटिंग लागू कर सकते हैं।

### मैं बीड़ी मार्क के साथ दस्तावेज़ को किस प्रारूप में सहेज सकता हूँ?
दिए गए उदाहरण में, दस्तावेज़ को टेक्स्ट फ़ाइल के रूप में सहेजा गया है। हालाँकि, Aspose.Words बिडी चिह्नों को संरक्षित करते हुए दस्तावेज़ों को विभिन्न प्रारूपों में सहेजने का भी समर्थन करता है।

### मैं Aspose.Words for .NET के बारे में अधिक जानकारी कहां पा सकता हूं?
 आप Aspose.Words for .NET के बारे में अधिक जानकारी यहाँ से प्राप्त कर सकते हैं।[Aspose दस्तावेज़ीकरण](https://reference.aspose.com/words/net/) और पहुँच[सहयता मंच](https://forum.aspose.com/c/words/8) अतिरिक्त सहायता के लिए.
