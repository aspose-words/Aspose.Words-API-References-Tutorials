.NET के लिए Aspose.Words C# एप्लिकेशन में Word दस्तावेज़ बनाने, संपादित करने और हेरफेर करने के लिए एक शक्तिशाली लाइब्रेरी है। Aspose.Words द्वारा दी जाने वाली सुविधाओं में से एक दस्तावेज़ में बीड़ी (द्वि-दिशात्मक) चिह्न जोड़ने की क्षमता है। इस गाइड में, हम आपको बताएंगे कि दस्तावेज़ में बीड़ी चिह्न जोड़ने के लिए .NET के लिए Aspose.Words के C# स्रोत कोड का उपयोग कैसे करें।

## Aspose.Words लाइब्रेरी को समझना

कोड में गोता लगाने से पहले, .NET के लिए Aspose.Words लाइब्रेरी को समझना महत्वपूर्ण है। Aspose.Words एक लोकप्रिय लाइब्रेरी है जो Word दस्तावेज़ों के साथ Word प्रोसेसिंग को आसान और कुशल बनाती है। यह Word दस्तावेज़ों को बनाने, संपादित करने और उनमें हेरफेर करने के लिए सुविधाओं की एक विस्तृत श्रृंखला प्रदान करता है, जिसमें बीड़ी चिह्न जोड़ना भी शामिल है।

## दस्तावेज़ बनाना और सामग्री जोड़ना

पहला कदम एक नया दस्तावेज़ बनाना और उसमें सामग्री जोड़ना है। नया दस्तावेज़ उदाहरण बनाने के लिए दस्तावेज़ वर्ग का उपयोग करें। फिर दस्तावेज़ में टेक्स्ट जोड़ने के लिए DocumentBuilder क्लास का उपयोग करें। यहाँ एक उदाहरण है :

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

builder. Writen("Hello world!");
builder.ParagraphFormat.Bidi = true;
builder. Writen("שלום עולם!");
builder. Writen("مرحبا بالعالم!");
```

इस उदाहरण में, हम एक नया दस्तावेज़ बनाते हैं और टेक्स्ट जोड़ने के लिए DocumentBuilder का उपयोग करते हैं। हमने विभिन्न भाषाओं में सामग्री जोड़ने को प्रदर्शित करने के लिए पाठ की तीन पंक्तियाँ जोड़ी हैं: एक अंग्रेजी में, एक हिब्रू में और एक अरबी में।

## बीड़ी के निशान जोड़ दिये

एक बार सामग्री जुड़ जाने के बाद, अब हम दस्तावेज़ में बीड़ी के निशान जोड़ सकते हैं। इसके लिए, हम TxtSaveOptions क्लास का उपयोग करते हैं और AddBidiMarks प्रॉपर्टी को सत्य पर सेट करते हैं। ऐसे:

```csharp
TxtSaveOptions saveOptions = new TxtSaveOptions { AddBidiMarks = true };
doc.Save(dataDir + "WorkingWithTxtSaveOptions.AddBidiMarks.txt", saveOptions);
```

इस उदाहरण में, हम TxtSaveOptions का एक उदाहरण बनाते हैं और AddBidiMarks प्रॉपर्टी को सत्य पर सेट करते हैं। इसके बाद, हम दस्तावेज़ को बीड़ी चिह्नों के साथ सहेजने के लिए दस्तावेज़ वर्ग की सेव विधि का उपयोग करते हैं।

### .NET के लिए Aspose.Words के साथ "बीड़ी के निशान जोड़ें" कार्यक्षमता के लिए उदाहरण स्रोत कोड

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// दस्तावेज़ बनाएं और सामग्री जोड़ें
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

builder. Writen("Hello world!");
builder.ParagraphFormat.Bidi = true;
builder. Writen("שלום עולם!");
builder. Writen("مرحبا بالعالم!");

// बीड़ी के निशान जोड़ें
TxtSaveOptions saveOptions = new TxtSaveOptions { AddBidiMarks = true

  };
doc.Save(dataDir + "WorkingWithTxtSaveOptions.AddBidiMarks.txt", saveOptions);
```

## निष्कर्ष

इस गाइड में, हमने बताया है कि दिए गए C# स्रोत कोड का उपयोग करके किसी Word दस्तावेज़ में बीड़ी चिह्न जोड़ने के लिए .NET के लिए Aspose.Words का उपयोग कैसे करें। दिए गए चरणों का पालन करके, आप आसानी से अपने C# एप्लिकेशन में अपने Word दस्तावेज़ों में बीड़ी चिह्न जोड़ सकते हैं। Aspose.Words टेक्स्ट फ़ॉर्मेटिंग और भाषा प्रबंधन के साथ वर्ड प्रोसेसिंग के लिए जबरदस्त लचीलापन और शक्ति प्रदान करता है, जिससे आप पेशेवर रूप से बहुभाषी दस्तावेज़ बना सकते हैं।

### अक्सर पूछे जाने वाले प्रश्नों

#### प्रश्न: .NET के लिए Aspose.Words क्या है?
.NET के लिए Aspose.Words C# एप्लिकेशन में Word दस्तावेज़ बनाने, संपादित करने और हेरफेर करने के लिए एक शक्तिशाली लाइब्रेरी है। यह वर्ड दस्तावेज़ों के साथ वर्ड प्रोसेसिंग के लिए कई सुविधाएँ प्रदान करता है, जिसमें बीड़ी (द्विदिशात्मक) चिह्न जोड़ना भी शामिल है।

#### प्रश्न: Aspose.Words for .NET क्या सुविधाएँ प्रदान करता है?
.NET के लिए Aspose.Words Word दस्तावेज़ों को बनाने, संपादित करने और उनमें हेरफेर करने के लिए सुविधाओं की एक विस्तृत श्रृंखला प्रदान करता है। इनमें से कुछ सुविधाओं में दस्तावेज़ बनाना, सामग्री जोड़ना, टेक्स्ट को फ़ॉर्मेट करना, तालिकाओं को प्रबंधित करना, दस्तावेज़ों को मर्ज करना और विभाजित करना, दस्तावेज़ों को परिवर्तित करना और बहुत कुछ शामिल हैं।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके किसी Word दस्तावेज़ में बीड़ी चिह्न कैसे जोड़ सकता हूँ?
आप इन चरणों का पालन करके किसी Word दस्तावेज़ में बीड़ी चिह्न जोड़ सकते हैं:

 का उपयोग करके एक नया दस्तावेज़ बनाएं`Document` कक्षा।

 उपयोग`DocumentBuilder` दस्तावेज़ में सामग्री जोड़ने के लिए क्लास।

 एक बार जब आप सामग्री जोड़ लें, तो इसका उपयोग करें`TxtSaveOptions` कक्षा और सेट करें`AddBidiMarks`संपत्ति को`true`.

 का उपयोग करके दस्तावेज़ को बीड़ी मार्क के साथ सहेजें`Save` की विधि`Document` कक्षा।

#### प्रश्न: क्या Aspose.Words बीड़ी मार्क्स जोड़ने के लिए कई भाषाओं का समर्थन करता है?
हां, Aspose.Words बीड़ी चिह्न जोड़ने के लिए कई भाषाओं का समर्थन करता है। आप .NET के लिए Aspose.Words का उपयोग करके विभिन्न भाषाओं, जैसे अंग्रेजी, हिब्रू और अरबी में पाठ में बीड़ी चिह्न जोड़ सकते हैं।

#### प्रश्न: क्या दस्तावेज़ को बीड़ी मार्क के साथ सहेजने के लिए कोई अतिरिक्त विकल्प हैं?
 हां, आप दस्तावेज़ को बीड़ी चिह्नों के साथ सहेजते समय अन्य विकल्प निर्दिष्ट कर सकते हैं`TxtSaveOptions` कक्षा। उदाहरण के लिए, आप दस्तावेज़ सहेजने का प्रारूप, एन्कोडिंग विकल्प आदि सेट कर सकते हैं।
