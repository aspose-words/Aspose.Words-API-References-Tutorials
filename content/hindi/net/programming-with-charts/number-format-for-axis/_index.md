यह ट्यूटोरियल बताता है कि चार्ट में अक्ष के लिए संख्या प्रारूप सेट करने के लिए Aspose.Words for .NET का उपयोग कैसे करें। प्रदान किया गया स्रोत कोड दर्शाता है कि चार्ट कैसे बनाएं, श्रृंखला डेटा कैसे जोड़ें और अक्ष लेबल को प्रारूपित करें।

## चरण 1: प्रोजेक्ट सेट अप करें

सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ हैं:

- Aspose.Words for .NET लाइब्रेरी स्थापित है। आप इसे स्थापित करने के लिए NuGet पैकेज मैनेजर का उपयोग करके डाउनलोड कर सकते हैं।
- दस्तावेज़ निर्देशिका पथ जहाँ आउटपुट दस्तावेज़ सहेजा जाएगा.

## चरण 2: एक नया दस्तावेज़ बनाएँ और एक चार्ट डालें

 कोई नया बनाएं`Document` वस्तु और एक`DocumentBuilder` दस्तावेज़ बनाने के लिए.

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 इसके बाद, का उपयोग करें`InsertChart` की विधि`DocumentBuilder` दस्तावेज़ में कॉलम चार्ट सम्मिलित करने के लिए.

```csharp
Shape shape = builder.InsertChart(ChartType.Column, 432, 252);
Chart chart = shape.Chart;
```

## चरण 3: चार्ट में श्रृंखला डेटा जोड़ें

चार्ट में श्रृंखला डेटा जोड़ें। इस उदाहरण में, हम पाँच आइटम को उनके संगत मानों के साथ जोड़ेंगे।

```csharp
chart.Series.Clear();
chart.Series.Add("Aspose Series 1",
    new string[] { "Item 1", "Item 2", "Item 3", "Item 4", "Item 5" },
    new double[] { 1900000, 850000, 2100000, 600000, 1500000 });
```

## चरण 4: अक्ष लेबल को प्रारूपित करें

 Y-अक्ष लेबल के लिए संख्या प्रारूप सेट करने के लिए, तक पहुँचें`AxisY` चार्ट की संपत्ति और सेट करें`NumberFormat.FormatCode` प्रॉपर्टी को वांछित प्रारूप में बदलें। इस उदाहरण में, हमने संख्याओं को हज़ारों विभाजकों के साथ प्रदर्शित करने के लिए प्रारूप को "#,##0" पर सेट किया है।

```csharp
chart.AxisY.NumberFormat.FormatCode = "#,##0";
```

## चरण 5: दस्तावेज़ सहेजें

 अंत में, दस्तावेज़ को निर्दिष्ट निर्देशिका में सहेजें`Save` की विधि`Document` वस्तु।

```csharp
doc.Save(dataDir + "WorkingWithCharts.NumberFormatForAxis.docx");
```

इससे .NET के लिए Aspose.Words का उपयोग करके अक्ष के लिए संख्या प्रारूप सेट करने का कार्यान्वयन पूरा हो जाता है।

### .NET के लिए Aspose.Words का उपयोग करके अक्ष के लिए संख्या प्रारूप के लिए उदाहरण स्रोत कोड 

```csharp
	// आपके दस्तावेज़ निर्देशिका का पथ
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Column, 432, 252);
	Chart chart = shape.Chart;
	chart.Series.Clear();
	chart.Series.Add("Aspose Series 1",
		new string[] { "Item 1", "Item 2", "Item 3", "Item 4", "Item 5" },
		new double[] { 1900000, 850000, 2100000, 600000, 1500000 });
	chart.AxisY.NumberFormat.FormatCode = "#,##0";
	doc.Save(dataDir + "WorkingWithCharts.NumberFormatForAxis.docx");
```

## निष्कर्ष

इस ट्यूटोरियल में, आपने सीखा है कि Aspose.Words for .NET का उपयोग करके चार्ट में अक्ष के लिए संख्या प्रारूप कैसे सेट करें। चरण-दर-चरण मार्गदर्शिका का पालन करके और प्रदान किए गए स्रोत कोड का उपयोग करके, आप एक नया दस्तावेज़ बना सकते हैं, एक कॉलम चार्ट सम्मिलित कर सकते हैं, श्रृंखला डेटा जोड़ सकते हैं, और संख्याओं को एक विशिष्ट प्रारूप में प्रदर्शित करने के लिए अक्ष लेबल को प्रारूपित कर सकते हैं।

Aspose.Words for .NET, Word दस्तावेज़ों में चार्ट की उपस्थिति को अनुकूलित करने के लिए शक्तिशाली सुविधाएँ प्रदान करता है। अक्ष लेबल के लिए संख्या प्रारूप सेट करके, आप नियंत्रित कर सकते हैं कि संख्याएँ कैसे प्रदर्शित की जाएँ, जिसमें दशमलव स्थान, हज़ार विभाजक, मुद्रा चिह्न और बहुत कुछ जैसे विकल्प शामिल हैं। यह आपको संख्यात्मक डेटा को स्पष्ट और सार्थक तरीके से प्रस्तुत करने की अनुमति देता है।

.NET के लिए Aspose.Words के साथ, आपके पास चार्ट के विभिन्न पहलुओं को प्रारूपित करने की सुविधा है, जिसमें अक्ष लेबल भी शामिल हैं। अक्ष के लिए संख्या प्रारूप सेट करके, आप स्थिरता सुनिश्चित कर सकते हैं और चार्ट की पठनीयता में सुधार कर सकते हैं, जिससे उपयोगकर्ताओं के लिए दर्शाए गए मानों की व्याख्या करना आसान हो जाता है।

### पूछे जाने वाले प्रश्न

#### प्रश्न 1. चार्ट में अक्ष का संख्या प्रारूप क्या है?
चार्ट में अक्ष के लिए संख्या प्रारूप अक्ष पर प्रदर्शित संख्यात्मक मानों पर लागू स्वरूपण को संदर्भित करता है। यह आपको संख्याओं को प्रस्तुत करने के तरीके को नियंत्रित करने की अनुमति देता है, जिसमें दशमलव स्थान, हज़ार विभाजक, मुद्रा चिह्न, प्रतिशत चिह्न और बहुत कुछ जैसे विकल्प शामिल हैं। संख्या प्रारूप सेट करके, आप अपनी विशिष्ट आवश्यकताओं के अनुरूप चार्ट में संख्यात्मक डेटा की उपस्थिति को अनुकूलित कर सकते हैं।

#### प्रश्न 2. मैं अक्ष लेबल के लिए संख्या प्रारूप कैसे सेट कर सकता हूं?
 .NET के लिए Aspose.Words का उपयोग करके चार्ट में अक्ष लेबल के लिए संख्या प्रारूप सेट करने के लिए, आप इसका उपयोग कर सकते हैं`AxisY` चार्ट की संपत्ति और सेट करें`NumberFormat.FormatCode`प्रॉपर्टी को वांछित फ़ॉर्मेट कोड में बदलें। फ़ॉर्मेट कोड मानक संख्यात्मक फ़ॉर्मेटिंग पैटर्न के सिंटैक्स का अनुसरण करता है और यह निर्धारित करता है कि संख्याएँ कैसे प्रदर्शित की जाएँगी। उदाहरण के लिए, आप दो दशमलव स्थानों और हज़ारों विभाजकों वाली संख्याओं को प्रदर्शित करने के लिए "#,##0.00" का उपयोग कर सकते हैं।

#### प्रश्न 3. क्या मैं X-अक्ष और Y-अक्ष लेबल के लिए अलग-अलग संख्या प्रारूप सेट कर सकता हूँ?
हां, आप .NET के लिए Aspose.Words का उपयोग करके X-अक्ष और Y-अक्ष लेबल के लिए अलग-अलग संख्या प्रारूप सेट कर सकते हैं। संबंधित अक्ष तक पहुंचें (`AxisX` X-अक्ष के लिए या`AxisY` चार्ट के Y-अक्ष के लिए) और संशोधित करें`NumberFormat.FormatCode` प्रत्येक अक्ष के लिए अलग-अलग संपत्ति। यह आपको अपनी विशिष्ट आवश्यकताओं के आधार पर प्रत्येक अक्ष पर लेबल पर अलग-अलग संख्या प्रारूप लागू करने की अनुमति देता है।

#### प्रश्न 4. कुछ सामान्य संख्या प्रारूप कोड क्या हैं जिनका मैं उपयोग कर सकता हूँ?
Aspose.Words for .NET संख्या प्रारूप कोड की एक विस्तृत श्रृंखला का समर्थन करता है जिसका उपयोग आप चार्ट में अक्ष लेबल को प्रारूपित करने के लिए कर सकते हैं। कुछ सामान्य प्रारूप कोड में शामिल हैं:

- `0` या`#` - संख्या को दशमलव स्थान के बिना प्रदर्शित करता है।
- `0.00` या`#.00` - संख्या को दो दशमलव स्थानों के साथ प्रदर्शित करता है।
- `#,##0` संख्या को हजारों विभाजकों के साथ प्रदर्शित करता है।
- `"€"0.00` - यूरो मुद्रा प्रतीक और दो दशमलव स्थानों के साथ संख्या प्रदर्शित करता है।
- `"%"0` - संख्या को प्रतिशत के रूप में प्रदर्शित करता है.

 आप संख्या के बारे में अधिक जानकारी पा सकते हैं[प्रारूप कोड](https://reference.aspose.com/words/net/aspose.words.drawing.charts/chartnumberformat/formatcode/) .NET के लिए Aspose.Words के API संदर्भ में।

#### प्रश्न 5. क्या मैं अक्ष लेबल के अन्य गुणों को अनुकूलित कर सकता हूँ?
हां, Aspose.Words for .NET अक्ष लेबल की उपस्थिति और व्यवहार को अनुकूलित करने के लिए गुणों की एक विस्तृत श्रृंखला प्रदान करता है। संख्या प्रारूप के अलावा, आप फ़ॉन्ट, आकार, रंग, अभिविन्यास, संरेखण और अधिक जैसे गुणों को संशोधित कर सकते हैं। यह आपको अपनी इच्छित शैली और प्रस्तुति आवश्यकताओं से मेल खाने के लिए अक्ष लेबल को पूरी तरह से अनुकूलित करने की अनुमति देता है।
