यह ट्यूटोरियल बताता है कि चार्ट में डेटा लेबल के लिए डिफ़ॉल्ट विकल्प सेट करने के लिए .NET के लिए Aspose.Words का उपयोग कैसे करें। प्रदान किया गया कोड दर्शाता है कि Aspose.Words का उपयोग करके चार्ट कैसे बनाएं, डेटा श्रृंखला जोड़ें और डेटा लेबल को कस्टमाइज़ करें।

## चरण 1: प्रोजेक्ट सेट अप करें

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित आवश्यकताएं पूरी हैं:

- Aspose.Words for .NET लाइब्रेरी स्थापित है। आप इसे स्थापित करने के लिए NuGet पैकेज मैनेजर का उपयोग करके डाउनलोड कर सकते हैं।
- दस्तावेज़ निर्देशिका पथ जहाँ आउटपुट दस्तावेज़ सहेजा जाएगा.

## चरण 2: एक नया दस्तावेज़ बनाएँ और एक चार्ट डालें

 सबसे पहले, आइए एक नया बनाएं`Document` वस्तु और एक`DocumentBuilder` दस्तावेज़ बनाने के लिए.

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 इसके बाद, हम दस्तावेज़ में एक चार्ट सम्मिलित करते हैं`InsertChart` की विधि`DocumentBuilder`इस उदाहरण में, हम एक पाई चार्ट डालेंगे।

```csharp
Shape shape = builder.InsertChart(ChartType.Pie, 432, 252);
Chart chart = shape.Chart;
```

## चरण 3: चार्ट में डेटा श्रृंखला जोड़ें

अब, चार्ट में डेटा सीरीज़ जोड़ते हैं। इस उदाहरण में, हम तीन श्रेणियाँ और उनके संगत मान जोड़ेंगे।

```csharp
chart.Series.Clear();
ChartSeries series = chart.Series.Add("Aspose Series 1",
    new string[] { "Category 1", "Category 2", "Category 3" },
    new double[] { 2.7, 3.2, 0.8 });
```

## चरण 4: डेटा लेबल अनुकूलित करें

 चार्ट में डेटा लेबल को अनुकूलित करने के लिए, हमें एक्सेस करने की आवश्यकता है`ChartDataLabelCollection` श्रृंखला से संबद्ध वस्तु.

```csharp
ChartDataLabelCollection labels = series.DataLabels;
```

 फिर हम इसके विभिन्न गुणों को संशोधित कर सकते हैं`labels`डेटा लेबल के लिए वांछित विकल्प सेट करने के लिए ऑब्जेक्ट। इस उदाहरण में, हम प्रतिशत और मान दिखाना सक्षम करेंगे, लीडर लाइन अक्षम करेंगे, और एक कस्टम विभाजक सेट करेंगे।

```csharp
labels.ShowPercentage = true;
labels.ShowValue = true;
labels.ShowLeaderLines = false;
labels.Separator = " - ";
```

## चरण 5: दस्तावेज़ सहेजें

 अंत में, हम दस्तावेज़ को निर्दिष्ट निर्देशिका में सहेजते हैं`Save` की विधि`Document` वस्तु।

```csharp
doc.Save(dataDir + "WorkingWithCharts.DefaultOptionsForDataLabels.docx");
```

इससे .NET के लिए Aspose.Words का उपयोग करके चार्ट में डेटा लेबल के लिए डिफ़ॉल्ट विकल्प सेट करने का कार्यान्वयन पूरा हो जाता है।

### .NET के लिए Aspose.Words का उपयोग करके डेटा लेबल के लिए डिफ़ॉल्ट विकल्पों के लिए उदाहरण स्रोत कोड 

```csharp
	// आपके दस्तावेज़ निर्देशिका का पथ
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Pie, 432, 252);
	Chart chart = shape.Chart;
	chart.Series.Clear();
	ChartSeries series = chart.Series.Add("Aspose Series 1",
		new string[] { "Category 1", "Category 2", "Category 3" },
		new double[] { 2.7, 3.2, 0.8 });
	ChartDataLabelCollection labels = series.DataLabels;
	labels.ShowPercentage = true;
	labels.ShowValue = true;
	labels.ShowLeaderLines = false;
	labels.Separator = " - ";
	doc.Save(dataDir + "WorkingWithCharts.DefaultOptionsForDataLabels.docx");
```

## निष्कर्ष

इस ट्यूटोरियल में, आपने सीखा है कि Aspose.Words for .NET का उपयोग करके चार्ट में डेटा लेबल के लिए डिफ़ॉल्ट विकल्प कैसे सेट करें। चरण-दर-चरण मार्गदर्शिका का पालन करके, आप एक चार्ट बना सकते हैं, डेटा श्रृंखला जोड़ सकते हैं, और अपनी विशिष्ट आवश्यकताओं को पूरा करने के लिए डेटा लेबल को अनुकूलित कर सकते हैं। Aspose.Words for .NET Word दस्तावेज़ों में चार्ट के साथ Words Processing के लिए एक शक्तिशाली API प्रदान करता है, जिससे आप विभिन्न चार्ट तत्वों में हेरफेर कर सकते हैं और वांछित उपस्थिति और कार्यक्षमता प्राप्त कर सकते हैं।

 के गुण सेट करके`ChartDataLabelCollection`चार्ट श्रृंखला से जुड़े ऑब्जेक्ट के लिए, आप डेटा लेबल के प्रदर्शन को नियंत्रित कर सकते हैं, जिसमें प्रतिशत, मान, लीडर लाइन और कस्टम विभाजक दिखाने जैसे विकल्प शामिल हैं। यह लचीलापन आपको डेटा को प्रभावी ढंग से प्रस्तुत करने और अपने चार्ट के दृश्य प्रतिनिधित्व को बढ़ाने में सक्षम बनाता है।

### पूछे जाने वाले प्रश्न

#### प्रश्न 1. .NET के लिए Aspose.Words क्या है?
Aspose.Words for .NET एक लाइब्रेरी है जो डेवलपर्स को .NET अनुप्रयोगों का उपयोग करके प्रोग्रामेटिक रूप से Word दस्तावेज़ बनाने, हेरफेर करने और सहेजने में सक्षम बनाती है। यह चार्ट सहित दस्तावेज़ तत्वों के साथ Words प्रोसेसिंग के लिए कई प्रकार की सुविधाएँ प्रदान करता है।

#### प्रश्न 2. मैं .NET के लिए Aspose.Words कैसे स्थापित कर सकता हूँ?
आप Visual Studio में NuGet पैकेज मैनेजर का उपयोग करके इसे डाउनलोड करके .NET के लिए Aspose.Words इंस्टॉल कर सकते हैं। बस NuGet पैकेज मैनेजर में "Aspose.Words" खोजें और इसे अपने प्रोजेक्ट में इंस्टॉल करें।

#### प्रश्न 3. क्या मैं .NET के लिए Aspose.Words का उपयोग करके चार्ट के अन्य पहलुओं को अनुकूलित कर सकता हूं?
हां, Aspose.Words for .NET आपको चार्ट के विभिन्न पहलुओं को अनुकूलित करने की अनुमति देता है, जैसे कि चार्ट प्रकार, अक्ष लेबल, लीजेंड, प्लॉट क्षेत्र, और बहुत कुछ। आप वांछित उपस्थिति और व्यवहार को प्राप्त करने के लिए चार्ट ऑब्जेक्ट के विभिन्न गुणों तक पहुंच सकते हैं और उन्हें संशोधित कर सकते हैं।

#### प्रश्न 4. क्या मैं चार्ट को विभिन्न प्रारूपों में सहेज सकता हूँ?
 हां, Aspose.Words for .NET चार्ट वाले दस्तावेज़ को विभिन्न प्रारूपों में सहेजने का समर्थन करता है, जिसमें DOCX, PDF, HTML, और बहुत कुछ शामिल है। आप अपनी आवश्यकताओं के आधार पर उपयुक्त प्रारूप चुन सकते हैं और इसका उपयोग कर सकते हैं।`Save` की विधि`Document` दस्तावेज़ को सहेजने के लिए ऑब्जेक्ट का उपयोग करें.

#### प्रश्न 5. क्या मैं इन तकनीकों को अन्य चार्ट प्रकारों पर लागू कर सकता हूँ?
हां, इस ट्यूटोरियल में वर्णित तकनीकों को .NET के लिए Aspose.Words द्वारा समर्थित अन्य चार्ट प्रकारों पर लागू किया जा सकता है। कुंजी उस चार्ट प्रकार के लिए विशिष्ट प्रासंगिक ऑब्जेक्ट्स और गुणों तक पहुंचना है जिसके साथ आप Words प्रोसेसिंग कर रहे हैं।
