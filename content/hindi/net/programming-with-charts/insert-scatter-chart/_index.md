यह ट्यूटोरियल बताता है कि दस्तावेज़ में स्कैटर चार्ट डालने के लिए .NET के लिए Aspose.Words का उपयोग कैसे करें। प्रदान किया गया स्रोत कोड दर्शाता है कि चार्ट कैसे बनाया जाता है, श्रृंखला डेटा कैसे जोड़ा जाता है और दस्तावेज़ को कैसे सहेजा जाता है।

## चरण 1: प्रोजेक्ट सेट अप करें

सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ हैं:

- Aspose.Words for .NET लाइब्रेरी स्थापित है। आप इसे स्थापित करने के लिए NuGet पैकेज मैनेजर का उपयोग करके डाउनलोड कर सकते हैं।
- दस्तावेज़ निर्देशिका पथ जहाँ आउटपुट दस्तावेज़ सहेजा जाएगा.

## चरण 2: एक नया दस्तावेज़ बनाएँ और एक चार्ट डालें

 कोई नया बनाएं`Document` वस्तु और एक`DocumentBuilder` दस्तावेज़ बनाने के लिए.

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 इसके बाद, का उपयोग करें`InsertChart` की विधि`DocumentBuilder` दस्तावेज़ में स्कैटर चार्ट सम्मिलित करने के लिए.

```csharp
Shape shape = builder.InsertChart(ChartType.Scatter, 432, 252);
Chart chart = shape.Chart;
```

## चरण 3: चार्ट में श्रृंखला डेटा जोड़ें

चार्ट में श्रृंखला डेटा जोड़ें। इस उदाहरण में, हम X और Y निर्देशांक के दो सेट जोड़ेंगे।

```csharp
chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 });
```

## चरण 4: दस्तावेज़ सहेजें

 अंत में, दस्तावेज़ को निर्दिष्ट निर्देशिका में सहेजें`Save` की विधि`Document` वस्तु।

```csharp
doc.Save(dataDir + "WorkingWithCharts.InsertScatterChart.docx");
```

इससे .NET के लिए Aspose.Words का उपयोग करके स्कैटर चार्ट सम्मिलित करने का कार्यान्वयन पूरा हो जाता है।

### .NET के लिए Aspose.Words का उपयोग करके स्कैटर चार्ट सम्मिलित करने के लिए उदाहरण स्रोत कोड 

```csharp
	// आपके दस्तावेज़ निर्देशिका का पथ
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Scatter, 432, 252);
	Chart chart = shape.Chart;
	chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 });
	doc.Save(dataDir + "WorkingWithCharts.InsertScatterChart.docx");
```

## निष्कर्ष

इस ट्यूटोरियल में, आपने सीखा है कि .NET के लिए Aspose.Words का उपयोग करके Word दस्तावेज़ में स्कैटर चार्ट कैसे डालें। चरण-दर-चरण मार्गदर्शिका का पालन करके और दिए गए स्रोत कोड का उपयोग करके, आप एक नया दस्तावेज़ बना सकते हैं, एक स्कैटर चार्ट डाल सकते हैं, X और Y निर्देशांक के साथ श्रृंखला डेटा जोड़ सकते हैं, और चार्ट के साथ दस्तावेज़ को सहेज सकते हैं।

Aspose.Words for .NET वर्ड डॉक्यूमेंट में चार्ट के साथ वर्ड प्रोसेसिंग के लिए एक व्यापक API प्रदान करता है। स्कैटर चार्ट दो संख्यात्मक चर के साथ डेटा को विज़ुअलाइज़ करने और उसका विश्लेषण करने के लिए उपयोगी होते हैं। Aspose.Words for .NET के साथ, आप आसानी से स्कैटर चार्ट बना सकते हैं जो X और Y मानों के बीच संबंध दर्शाते हैं और डेटा में पैटर्न या रुझान की पहचान करते हैं।

.NET के लिए Aspose.Words का उपयोग करके, आप स्कैटर चार्ट के साथ दस्तावेज़ बनाने की प्रक्रिया को स्वचालित कर सकते हैं, जिससे मैन्युअल दस्तावेज़ निर्माण में समय और प्रयास की बचत होती है। लाइब्रेरी स्कैटर चार्ट सहित चार्ट प्रकारों की एक विस्तृत श्रृंखला प्रदान करती है, और आपकी आवश्यकताओं के अनुसार चार्ट की उपस्थिति को अनुकूलित करने के लिए विभिन्न अनुकूलन विकल्प प्रदान करती है।

### पूछे जाने वाले प्रश्न

#### प्रश्न 1. स्कैटर चार्ट क्या है?
स्कैटर चार्ट एक प्रकार का चार्ट है जो दो संख्यात्मक चरों के बीच संबंध प्रदर्शित करता है। इसमें निर्देशांक ग्रिड पर प्लॉट किए गए बिंदुओं की एक श्रृंखला होती है, जिसमें एक चर X-अक्ष पर और दूसरा चर Y-अक्ष पर दर्शाया जाता है। स्कैटर चार्ट का उपयोग डेटा बिंदुओं के दो सेटों के बीच पैटर्न, सहसंबंध या रुझानों की पहचान करने के लिए किया जाता है।

#### प्रश्न 2. क्या मैं स्कैटर चार्ट में एकाधिक श्रृंखलाएं जोड़ सकता हूं?
हां, आप .NET के लिए Aspose.Words का उपयोग करके स्कैटर चार्ट में कई श्रृंखलाएं जोड़ सकते हैं। प्रत्येक श्रृंखला अपने संबंधित X और Y निर्देशांक के साथ डेटा बिंदुओं के एक सेट का प्रतिनिधित्व करती है। कई श्रृंखलाओं को जोड़कर, आप एक ही स्कैटर चार्ट के भीतर विभिन्न डेटासेट की तुलना और विश्लेषण कर सकते हैं, जिससे आपके डेटा का एक व्यापक दृश्य मिलता है।

#### प्रश्न 3. क्या मैं स्कैटर चार्ट के स्वरूप को अनुकूलित कर सकता हूँ?
हां, .NET के लिए Aspose.Words का उपयोग करके, आप स्कैटर चार्ट की उपस्थिति के विभिन्न पहलुओं को अनुकूलित कर सकते हैं। आप श्रृंखला रंग, मार्कर आकार, अक्ष लेबल और चार्ट क्षेत्र स्वरूपण जैसे गुणों को संशोधित कर सकते हैं। लाइब्रेरी चार्ट के दृश्य तत्वों को नियंत्रित करने और आपकी आवश्यकताओं के अनुरूप एक अनुकूलित रूप बनाने के लिए API का एक समृद्ध सेट प्रदान करती है।

#### प्रश्न 4. क्या मैं सम्मिलित स्कैटर चार्ट वाले दस्तावेज़ को विभिन्न प्रारूपों में सहेज सकता हूँ?
हां, .NET के लिए Aspose.Words आपको सम्मिलित स्कैटर चार्ट के साथ दस्तावेज़ को विभिन्न स्वरूपों में सहेजने की अनुमति देता है, जैसे कि DOCX, PDF, HTML, और अधिक। आप अपनी आवश्यकताओं के आधार पर वांछित आउटपुट प्रारूप चुन सकते हैं और इसका उपयोग कर सकते हैं`Save` की विधि`Document` दस्तावेज़ को सहेजने के लिए ऑब्जेक्ट का उपयोग करें। सम्मिलित स्कैटर चार्ट सहेजे गए दस्तावेज़ में संरक्षित रहेगा।

#### प्रश्न 5. क्या मैं स्कैटर चार्ट डालने के बाद उसके डेटा और स्वरूप को संशोधित कर सकता हूँ?
हां, दस्तावेज़ में स्कैटर चार्ट डालने के बाद, आप .NET के लिए Aspose.Words द्वारा प्रदान किए गए API का उपयोग करके इसके डेटा और उपस्थिति को संशोधित कर सकते हैं। आप नए X और Y निर्देशांक के साथ श्रृंखला डेटा को अपडेट कर सकते हैं, मार्कर आकार और रंग बदल सकते हैं, अक्ष गुणों को अनुकूलित कर सकते हैं, और अपने Word दस्तावेज़ों में गतिशील और इंटरैक्टिव चार्ट बनाने के लिए स्वरूपण विकल्प लागू कर सकते हैं।
