यह ट्यूटोरियल बताता है कि दस्तावेज़ में स्कैटर चार्ट डालने के लिए .NET के लिए Aspose.Words का उपयोग कैसे करें। प्रदान किया गया स्रोत कोड दर्शाता है कि चार्ट कैसे बनाएं, श्रृंखला डेटा कैसे जोड़ें और दस्तावेज़ को कैसे सहेजें।

## चरण 1: प्रोजेक्ट सेट करें

सुनिश्चित करें कि आपके पास निम्नलिखित आवश्यकताएँ हैं:

- .NET लाइब्रेरी के लिए Aspose.Words स्थापित। आप इसे इंस्टॉल करने के लिए NuGet पैकेज मैनेजर का उपयोग करके इसे डाउनलोड कर सकते हैं।
- एक दस्तावेज़ निर्देशिका पथ जहां आउटपुट दस्तावेज़ सहेजा जाएगा।

## चरण 2: एक नया दस्तावेज़ बनाएं और एक चार्ट डालें

 कोई नया बनाएं`Document` वस्तु और ए`DocumentBuilder` दस्तावेज़ बनाने के लिए.

```csharp
// आपकी दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 अगला, का उपयोग करें`InsertChart` की विधि`DocumentBuilder` दस्तावेज़ में स्कैटर चार्ट सम्मिलित करने के लिए।

```csharp
Shape shape = builder.InsertChart(ChartType.Scatter, 432, 252);
Chart chart = shape.Chart;
```

## चरण 3: चार्ट में श्रृंखला डेटा जोड़ें

चार्ट में श्रृंखला डेटा जोड़ें. इस उदाहरण में, हम X और Y निर्देशांक के दो सेट जोड़ेंगे।

```csharp
chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 });
```

## चरण 4: दस्तावेज़ सहेजें

 अंत में, का उपयोग करके दस्तावेज़ को निर्दिष्ट निर्देशिका में सहेजें`Save` की विधि`Document` वस्तु।

```csharp
doc.Save(dataDir + "WorkingWithCharts.InsertScatterChart.docx");
```

यह .NET के लिए Aspose.Words का उपयोग करके स्कैटर चार्ट सम्मिलित करने का कार्यान्वयन पूरा करता है।

### .NET के लिए Aspose.Words का उपयोग करके इन्सर्ट स्कैटर चार्ट के लिए उदाहरण स्रोत कोड 

```csharp
	// आपकी दस्तावेज़ निर्देशिका का पथ
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Scatter, 432, 252);
	Chart chart = shape.Chart;
	chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 });
	doc.Save(dataDir + "WorkingWithCharts.InsertScatterChart.docx");
```

## निष्कर्ष

इस ट्यूटोरियल में, आपने सीखा कि .NET के लिए Aspose.Words का उपयोग करके किसी Word दस्तावेज़ में स्कैटर चार्ट कैसे सम्मिलित किया जाए। चरण-दर-चरण मार्गदर्शिका का पालन करके और दिए गए स्रोत कोड का उपयोग करके, आप एक नया दस्तावेज़ बना सकते हैं, एक स्कैटर चार्ट सम्मिलित कर सकते हैं, एक्स और वाई निर्देशांक के साथ श्रृंखला डेटा जोड़ सकते हैं, और दस्तावेज़ को चार्ट के साथ सहेज सकते हैं।

.NET के लिए Aspose.Words Word दस्तावेज़ों में चार्ट के साथ वर्ड प्रोसेसिंग के लिए एक व्यापक एपीआई प्रदान करता है। स्कैटर चार्ट दो संख्यात्मक चर के साथ डेटा को देखने और विश्लेषण करने के लिए उपयोगी होते हैं। .NET के लिए Aspose.Words के साथ, आप आसानी से स्कैटर चार्ट बना सकते हैं जो X और Y मानों के बीच संबंध दर्शाते हैं और डेटा में पैटर्न या रुझान की पहचान करते हैं।

.NET के लिए Aspose.Words का उपयोग करके, आप स्कैटर चार्ट के साथ दस्तावेज़ बनाने की प्रक्रिया को स्वचालित कर सकते हैं, जिससे मैन्युअल दस्तावेज़ निर्माण में समय और प्रयास की बचत होती है। लाइब्रेरी स्कैटर चार्ट सहित चार्ट प्रकारों की एक विस्तृत श्रृंखला प्रदान करती है, और आपकी आवश्यकताओं के अनुसार चार्ट की उपस्थिति को तैयार करने के लिए विभिन्न अनुकूलन विकल्प प्रदान करती है।

### पूछे जाने वाले प्रश्न

#### Q1. स्कैटर चार्ट क्या है?
स्कैटर चार्ट एक प्रकार का चार्ट है जो दो संख्यात्मक चर के बीच संबंध प्रदर्शित करता है। इसमें एक समन्वय ग्रिड पर अंकित बिंदुओं की एक श्रृंखला होती है, जिसमें एक चर एक्स-अक्ष पर और दूसरा चर वाई-अक्ष पर दर्शाया जाता है। स्कैटर चार्ट का उपयोग डेटा बिंदुओं के दो सेटों के बीच पैटर्न, सहसंबंध या रुझान की पहचान करने के लिए किया जाता है।

#### Q2. क्या मैं स्कैटर चार्ट में अनेक श्रृंखलाएँ जोड़ सकता हूँ?
हाँ, आप .NET के लिए Aspose.Words का उपयोग करके स्कैटर चार्ट में कई श्रृंखलाएँ जोड़ सकते हैं। प्रत्येक श्रृंखला अपने संबंधित X और Y निर्देशांक के साथ डेटा बिंदुओं के एक सेट का प्रतिनिधित्व करती है। एकाधिक श्रृंखला जोड़कर, आप एक ही स्कैटर चार्ट के भीतर विभिन्न डेटासेट की तुलना और विश्लेषण कर सकते हैं, जो आपके डेटा का एक व्यापक दृश्य प्रदान करता है।

#### Q3. क्या मैं स्कैटर चार्ट के स्वरूप को अनुकूलित कर सकता हूँ?
हां, .NET के लिए Aspose.Words का उपयोग करके, आप स्कैटर चार्ट की उपस्थिति के विभिन्न पहलुओं को अनुकूलित कर सकते हैं। आप श्रृंखला रंग, मार्कर आकार, अक्ष लेबल और चार्ट क्षेत्र स्वरूपण जैसे गुणों को संशोधित कर सकते हैं। लाइब्रेरी चार्ट के दृश्य तत्वों को नियंत्रित करने और आपकी आवश्यकताओं के अनुरूप एक अनुकूलित लुक बनाने के लिए एपीआई का एक समृद्ध सेट प्रदान करती है।

#### Q4. क्या मैं सम्मिलित स्कैटर चार्ट के साथ दस्तावेज़ को विभिन्न प्रारूपों में सहेज सकता हूँ?
हां, .NET के लिए Aspose.Words आपको दस्तावेज़ को DOCX, PDF, HTML और अन्य जैसे विभिन्न स्वरूपों में सम्मिलित स्कैटर चार्ट के साथ सहेजने की अनुमति देता है। आप अपनी आवश्यकताओं के आधार पर वांछित आउटपुट स्वरूप चुन सकते हैं और इसका उपयोग कर सकते हैं`Save` की विधि`Document` दस्तावेज़ को सहेजने के लिए ऑब्जेक्ट। सम्मिलित स्कैटर चार्ट सहेजे गए दस्तावेज़ में संरक्षित किया जाएगा।

#### Q5. क्या मैं स्कैटर चार्ट डालने के बाद उसके डेटा और स्वरूप को संशोधित कर सकता हूँ?
हां, दस्तावेज़ में स्कैटर चार्ट डालने के बाद, आप .NET के लिए Aspose.Words द्वारा प्रदान की गई एपीआई का उपयोग करके इसके डेटा और स्वरूप को संशोधित कर सकते हैं। आप श्रृंखला डेटा को नए X और Y निर्देशांक के साथ अपडेट कर सकते हैं, मार्कर आकार और रंग बदल सकते हैं, अक्ष गुणों को अनुकूलित कर सकते हैं, और अपने Word दस्तावेज़ों में गतिशील और इंटरैक्टिव चार्ट बनाने के लिए फ़ॉर्मेटिंग विकल्प लागू कर सकते हैं।
