यह ट्यूटोरियल बताता है कि चार्ट में डेटा लेबल की संख्या को फ़ॉर्मेट करने के लिए Aspose.Words for .NET का उपयोग कैसे करें। प्रदान किया गया स्रोत कोड दर्शाता है कि चार्ट कैसे बनाएँ, श्रृंखला डेटा कैसे जोड़ें और डेटा लेबल के संख्या प्रारूप को कैसे अनुकूलित करें।

## चरण 1: प्रोजेक्ट सेट अप करें

सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ हैं:

- Aspose.Words for .NET लाइब्रेरी स्थापित है। आप इसे स्थापित करने के लिए NuGet पैकेज मैनेजर का उपयोग करके डाउनलोड कर सकते हैं।
- दस्तावेज़ निर्देशिका पथ जहाँ आउटपुट दस्तावेज़ सहेजा जाएगा.

## चरण 2: एक नया दस्तावेज़ बनाएँ और एक चार्ट डालें

 कोई नया बनाएं`Document` वस्तु और एक`DocumentBuilder` दस्तावेज़ बनाने के लिए.

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 इसके बाद, दस्तावेज़ में एक चार्ट डालें`InsertChart` की विधि`DocumentBuilder`इस उदाहरण में, हम एक लाइन चार्ट डालेंगे।

```csharp
Shape shape = builder.InsertChart(ChartType.Line, 432, 252);
Chart chart = shape.Chart;
chart.Title.Text = "Data Labels With Different Number Format";
```

## चरण 3: चार्ट में श्रृंखला डेटा जोड़ें

चार्ट में श्रृंखला डेटा जोड़ें। इस उदाहरण में, हम तीन श्रेणियाँ और उनके संगत मान जोड़ेंगे।

```csharp
chart.Series.Clear();
ChartSeries series1 = chart.Series.Add("Aspose Series 1", 
    new string[] { "Category 1", "Category 2", "Category 3" }, 
    new double[] { 2.5, 1.5, 3.5 });
series1.HasDataLabels = true;
```

## चरण 4: डेटा लेबल का संख्या प्रारूप अनुकूलित करें

 डेटा लेबल की संख्या को प्रारूपित करने के लिए, तक पहुँचें`DataLabels` श्रृंखला से संबंधित संग्रह।

```csharp
series1.DataLabels.ShowValue = true;
series1.DataLabels[0].NumberFormat.FormatCode = "\"$\"#,##0.00";
series1.DataLabels[1].NumberFormat.FormatCode = "dd/mm/yyyy";
series1.DataLabels[2].NumberFormat.FormatCode = "0.00%";
```

इस उदाहरण में, हमने प्रत्येक डेटा लेबल के लिए अलग-अलग संख्या प्रारूप निर्धारित किए हैं। पहला डेटा लेबल मुद्रा के रूप में, दूसरा दिनांक के रूप में और तीसरा प्रतिशत के रूप में प्रारूपित है।

## चरण 5: दस्तावेज़ सहेजें

 अंत में, दस्तावेज़ को निर्दिष्ट निर्देशिका में सहेजें`Save` की विधि`Document` वस्तु।

```csharp
doc.Save(dataDir + "WorkingWithCharts.FormatNumberOfDataLabel.docx");
```

इससे .NET के लिए Aspose.Words का उपयोग करके चार्ट में डेटा लेबल की संख्या को प्रारूपित करने का कार्यान्वयन पूरा हो जाता है।

### .NET के लिए Aspose.Words का उपयोग करके डेटा लेबल की संख्या प्रारूपित करने के लिए उदाहरण स्रोत कोड 

```csharp
	// आपके दस्तावेज़ निर्देशिका का पथ
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Line, 432, 252);
	Chart chart = shape.Chart;
	chart.Title.Text = "Data Labels With Different Number Format";
	// डिफ़ॉल्ट रूप से उत्पन्न श्रृंखला को हटाएँ.
	chart.Series.Clear();
	ChartSeries series1 = chart.Series.Add("Aspose Series 1", 
		new string[] { "Category 1", "Category 2", "Category 3" }, 
		new double[] { 2.5, 1.5, 3.5 });
	series1.HasDataLabels = true;
	series1.DataLabels.ShowValue = true;
	series1.DataLabels[0].NumberFormat.FormatCode = "\"$\"#,##0.00";
	series1.DataLabels[1].NumberFormat.FormatCode = "dd/mm/yyyy";
	series1.DataLabels[2].NumberFormat.FormatCode = "0.00%";
	// या आप स्रोत सेल से लिंक करने के लिए प्रारूप कोड सेट कर सकते हैं,
	//इस स्थिति में NumberFormat को सामान्य पर रीसेट कर दिया जाएगा और स्रोत सेल से विरासत में प्राप्त किया जाएगा।
	series1.DataLabels[2].NumberFormat.IsLinkedToSource = true;
	doc.Save(dataDir + "WorkingWithCharts.FormatNumberOfDataLabel.docx");
```

## निष्कर्ष

इस ट्यूटोरियल में, आपने सीखा है कि .NET के लिए Aspose.Words का उपयोग करके चार्ट में डेटा लेबल की संख्या को कैसे फ़ॉर्मेट किया जाए। चरण-दर-चरण मार्गदर्शिका का पालन करके और दिए गए स्रोत कोड का उपयोग करके, आप एक चार्ट बना सकते हैं, श्रृंखला डेटा जोड़ सकते हैं, और अपनी आवश्यकताओं के अनुसार डेटा लेबल के संख्या प्रारूप को अनुकूलित कर सकते हैं।

 Aspose.Words for .NET वर्ड डॉक्यूमेंट में चार्ट के साथ वर्ड प्रोसेसिंग के लिए एक व्यापक API प्रदान करता है, जिससे आप डेटा लेबल सहित चार्ट के विभिन्न पहलुओं में हेरफेर कर सकते हैं।`DataLabels` किसी श्रृंखला से संबद्ध संग्रह के लिए, आप व्यक्तिगत डेटा लेबल के संख्या प्रारूप को अनुकूलित कर सकते हैं।

API आपको मानों के प्रदर्शन को नियंत्रित करने, प्रत्येक डेटा लेबल के लिए अलग-अलग संख्या प्रारूप सेट करने और संख्या प्रारूप को स्रोत सेल से लिंक करने की अनुमति देता है। यह लचीलापन आपको चार्ट में संख्यात्मक डेटा को वांछित स्वरूपण, जैसे मुद्रा चिह्न, दिनांक प्रारूप और प्रतिशत मानों के साथ प्रस्तुत करने में सक्षम बनाता है।

.NET के लिए Aspose.Words का उपयोग करके, आप अपने .NET अनुप्रयोगों में शक्तिशाली चार्टिंग क्षमताओं को शामिल कर सकते हैं और पूरी तरह से प्रारूपित चार्ट और डेटा लेबल के साथ पेशेवर दिखने वाले दस्तावेज़ उत्पन्न कर सकते हैं।

### पूछे जाने वाले प्रश्न

#### प्रश्न 1. .NET के लिए Aspose.Words क्या है?
Aspose.Words for .NET एक सुविधा संपन्न दस्तावेज़ प्रसंस्करण लाइब्रेरी है जो डेवलपर्स को .NET अनुप्रयोगों में प्रोग्रामेटिक रूप से Word दस्तावेज़ बनाने, हेरफेर करने और सहेजने में सक्षम बनाती है। यह चार्ट और डेटा लेबल सहित दस्तावेज़ तत्वों के साथ Words प्रोसेसिंग के लिए कई प्रकार की सुविधाएँ प्रदान करता है।

#### प्रश्न 2. मैं .NET के लिए Aspose.Words कैसे स्थापित कर सकता हूँ?
आप Visual Studio में NuGet पैकेज मैनेजर का उपयोग करके इसे डाउनलोड करके .NET के लिए Aspose.Words इंस्टॉल कर सकते हैं। बस NuGet पैकेज मैनेजर में "Aspose.Words" खोजें और इसे अपने प्रोजेक्ट में इंस्टॉल करें।

#### प्रश्न 3. क्या मैं .NET के लिए Aspose.Words का उपयोग करके चार्ट के अन्य पहलुओं को प्रारूपित कर सकता हूं?
हां, .NET के लिए Aspose.Words चार्ट के विभिन्न पहलुओं को फ़ॉर्मेट करने के लिए व्यापक क्षमताएं प्रदान करता है। डेटा लेबल के अलावा, आप चार्ट प्रकार, श्रृंखला डेटा, अक्ष गुण, किंवदंती, शीर्षक, प्लॉट क्षेत्र और चार्ट के कई अन्य तत्वों को अनुकूलित कर सकते हैं। API चार्ट की उपस्थिति और फ़ॉर्मेटिंग पर बारीक नियंत्रण प्रदान करता है।

#### प्रश्न 4. क्या मैं एक ही श्रृंखला में विभिन्न डेटा लेबलों पर अलग-अलग संख्या प्रारूप लागू कर सकता हूँ?
हां, .NET के लिए Aspose.Words आपको एक ही श्रृंखला के भीतर अलग-अलग डेटा लेबल पर अलग-अलग संख्या प्रारूप लागू करने की अनुमति देता है।`DataLabels` किसी श्रृंखला से संबद्ध संग्रह के लिए, आप सेट कर सकते हैं`FormatCode` प्रत्येक डेटा लेबल की प्रॉपर्टी का उपयोग वांछित संख्या प्रारूप निर्दिष्ट करने के लिए करें। यह आपको एक ही चार्ट के भीतर विभिन्न प्रारूपों में संख्यात्मक मान प्रस्तुत करने की अनुमति देता है।

#### प्रश्न 5. क्या मैं डेटा लेबल के लिए कस्टम संख्या प्रारूप का उपयोग कर सकता हूँ?
 हां, Aspose.Words for .NET डेटा लेबल के लिए कस्टम नंबर फ़ॉर्मेट का समर्थन करता है। आप वांछित संख्या फ़ॉर्मेट को सेट करके निर्दिष्ट कर सकते हैं`FormatCode` डेटा लेबल की प्रॉपर्टी को कस्टम फ़ॉर्मेट कोड में बदलें। इससे आपको कई तरह के नंबर फ़ॉर्मेट लागू करने की सुविधा मिलती है, जैसे कि करेंसी सिंबल, डेट फ़ॉर्मेट, प्रतिशत मान और बहुत कुछ।

#### प्रश्न 6. क्या मैं चार्ट को विभिन्न प्रारूपों में स्वरूपित डेटा लेबल के साथ सहेज सकता हूं?
हां, .NET के लिए Aspose.Words आपको विभिन्न प्रारूपों, जैसे DOCX, PDF, HTML, और अधिक में स्वरूपित डेटा लेबल के साथ चार्ट युक्त दस्तावेज़ को सहेजने की अनुमति देता है। आप अपनी आवश्यकताओं के आधार पर उपयुक्त प्रारूप चुन सकते हैं और इसका उपयोग कर सकते हैं।`Save` की विधि`Document` दस्तावेज़ को सहेजने के लिए ऑब्जेक्ट का उपयोग करें। स्वरूपित डेटा लेबल सहेजे गए दस्तावेज़ में संरक्षित रहेंगे।
