यह ट्यूटोरियल बताता है कि दस्तावेज़ में एक साधारण कॉलम चार्ट डालने के लिए .NET के लिए Aspose.Words का उपयोग कैसे करें। प्रदान किया गया स्रोत कोड दर्शाता है कि चार्ट कैसे बनाएं, श्रृंखला डेटा कैसे जोड़ें और दस्तावेज़ को कैसे सहेजें।

## चरण 1: प्रोजेक्ट सेट अप करें

सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ हैं:

- Aspose.Words for .NET लाइब्रेरी स्थापित है। आप इसे स्थापित करने के लिए NuGet पैकेज मैनेजर का उपयोग करके डाउनलोड कर सकते हैं।
- दस्तावेज़ निर्देशिका पथ जहाँ आउटपुट दस्तावेज़ सहेजा जाएगा.

## चरण 2: एक नया दस्तावेज़ बनाएँ और एक चार्ट डालें

 कोई नया बनाएं`Document` वस्तु और एक`DocumentBuilder` दस्तावेज़ बनाने के लिए.

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 इसके बाद, का उपयोग करें`InsertChart` की विधि`DocumentBuilder` दस्तावेज़ में कॉलम चार्ट डालने के लिए। आप अपनी आवश्यकताओं के अनुसार विभिन्न चार्ट प्रकार और आकार निर्दिष्ट कर सकते हैं।

```csharp
Shape shape = builder.InsertChart(ChartType.Column, 432, 252);
Chart chart = shape.Chart;
```

## चरण 3: चार्ट में श्रृंखला डेटा जोड़ें

चार्ट में श्रृंखला डेटा जोड़ें। इस उदाहरण में, हम दो श्रेणियों वाली कई श्रृंखलाएँ जोड़ेंगे।

```csharp
ChartSeriesCollection seriesColl = chart.Series;
seriesColl.Clear();

string[] categories = new string[] { "Category 1", "Category 2" };

seriesColl.Add("Aspose Series 1", categories, new double[] { 1, 2 });
seriesColl.Add("Aspose Series 2", categories, new double[] { 3, 4 });
seriesColl.Add("Aspose Series 3", categories, new double[] { 5, 6 });
seriesColl.Add("Aspose Series 4", categories, new double[] { 7, 8 });
seriesColl.Add("Aspose Series 5", categories, new double[] { 9, 10 });
```

## चरण 4: दस्तावेज़ सहेजें

 अंत में, दस्तावेज़ को निर्दिष्ट निर्देशिका में सहेजें`Save` की विधि`Document` वस्तु।

```csharp
doc.Save(dataDir + "WorkingWithCharts.InsertSimpleColumnChart.docx");
```

इससे .NET के लिए Aspose.Words का उपयोग करके एक सरल कॉलम चार्ट सम्मिलित करने का कार्यान्वयन पूरा हो जाता है।

### .NET के लिए Aspose.Words का उपयोग करके सरल कॉलम चार्ट सम्मिलित करने के लिए उदाहरण स्रोत कोड 

```csharp
	// आपके दस्तावेज़ निर्देशिका का पथ
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	// आप विभिन्न चार्ट प्रकार और आकार निर्दिष्ट कर सकते हैं.
	Shape shape = builder.InsertChart(ChartType.Column, 432, 252);
	Chart chart = shape.Chart;
	ChartSeriesCollection seriesColl = chart.Series;
	Console.WriteLine(seriesColl.Count);
	// डिफ़ॉल्ट रूप से उत्पन्न श्रृंखला को हटाएँ.
	seriesColl.Clear();
	// श्रेणी नाम सरणी बनाएँ, इस ट्यूटोरियल में हमारे पास दो श्रेणियाँ हैं।
	string[] categories = new string[] { "Category 1", "Category 2" };
	// कृपया ध्यान दें, डेटा सारणी खाली नहीं होनी चाहिए तथा सारणी का आकार समान होना चाहिए।
	seriesColl.Add("Aspose Series 1", categories, new double[] { 1, 2 });
	seriesColl.Add("Aspose Series 2", categories, new double[] { 3, 4 });
	seriesColl.Add("Aspose Series 3", categories, new double[] { 5, 6 });
	seriesColl.Add("Aspose Series 4", categories, new double[] { 7, 8 });
	seriesColl.Add("Aspose Series 5", categories, new double[] { 9, 10 });
	doc.Save(dataDir + "WorkingWithCharts.InsertSimpleColumnChart.docx");
```

## निष्कर्ष

इस ट्यूटोरियल में, आपने सीखा है कि Aspose.Words for .NET का उपयोग करके Word दस्तावेज़ में एक सरल कॉलम चार्ट कैसे डालें। चरण-दर-चरण मार्गदर्शिका का पालन करके और दिए गए स्रोत कोड का उपयोग करके, आप एक नया दस्तावेज़ बना सकते हैं, एक कॉलम चार्ट डाल सकते हैं, श्रेणियों और संबंधित मानों के साथ कई श्रृंखलाएँ जोड़ सकते हैं, और चार्ट के साथ दस्तावेज़ को सहेज सकते हैं।

Aspose.Words for .NET वर्ड डॉक्यूमेंट में चार्ट के साथ वर्ड प्रोसेसिंग के लिए एक शक्तिशाली और लचीला API प्रदान करता है। सरल कॉलम चार्ट विभिन्न श्रेणियों में डेटा का प्रतिनिधित्व और तुलना करने का एक प्रभावी तरीका है। Aspose.Words for .NET के साथ, आप आसानी से कस्टम डेटा के साथ कॉलम चार्ट बना सकते हैं, दृश्य तुलना के लिए कई श्रृंखलाएँ जोड़ सकते हैं, और अपनी आवश्यकताओं के अनुसार चार्ट की उपस्थिति को अनुकूलित कर सकते हैं।

.NET के लिए Aspose.Words का उपयोग करके, आप कॉलम चार्ट के साथ दस्तावेज़ बनाने की प्रक्रिया को स्वचालित कर सकते हैं, जिससे मैन्युअल दस्तावेज़ निर्माण में समय और प्रयास की बचत होती है। लाइब्रेरी सरल कॉलम चार्ट सहित चार्ट प्रकारों की एक विस्तृत श्रृंखला प्रदान करती है, और आपकी आवश्यकताओं के अनुरूप चार्ट की उपस्थिति को अनुकूलित करने के लिए विभिन्न अनुकूलन विकल्प प्रदान करती है।

### पूछे जाने वाले प्रश्न

#### प्रश्न 1. कॉलम चार्ट क्या है?
कॉलम चार्ट एक प्रकार का चार्ट है जो अलग-अलग ऊंचाई के ऊर्ध्वाधर बार का उपयोग करके डेटा प्रदर्शित करता है। प्रत्येक कॉलम एक श्रेणी का प्रतिनिधित्व करता है, और कॉलम की ऊंचाई उस श्रेणी के मान से मेल खाती है। कॉलम चार्ट का उपयोग आमतौर पर विभिन्न श्रेणियों में डेटा की तुलना करने या समय के साथ परिवर्तनों को ट्रैक करने के लिए किया जाता है।

#### प्रश्न 2. क्या मैं कॉलम चार्ट में एकाधिक श्रृंखलाएं जोड़ सकता हूं?
हां, .NET के लिए Aspose.Words का उपयोग करके, आप कॉलम चार्ट में कई श्रृंखलाएँ जोड़ सकते हैं। प्रत्येक श्रृंखला अपने संबंधित श्रेणियों और मूल्यों के साथ डेटा बिंदुओं के एक सेट का प्रतिनिधित्व करती है। कई श्रृंखलाएँ जोड़कर, आप एक ही कॉलम चार्ट के भीतर विभिन्न डेटासेट की तुलना और विश्लेषण कर सकते हैं, जिससे आपके डेटा का एक व्यापक दृश्य मिलता है।

#### प्रश्न 3. क्या मैं कॉलम चार्ट के स्वरूप को अनुकूलित कर सकता हूँ?
हां, Aspose.Words for .NET आपको कॉलम चार्ट के स्वरूप के विभिन्न पहलुओं को अनुकूलित करने की अनुमति देता है। आप श्रृंखला रंग, अक्ष लेबल, डेटा लेबल और चार्ट क्षेत्र स्वरूपण जैसे गुणों को संशोधित कर सकते हैं। लाइब्रेरी चार्ट के दृश्य तत्वों को नियंत्रित करने और आपकी आवश्यकताओं के अनुरूप एक अनुकूलित रूप बनाने के लिए API का एक समृद्ध सेट प्रदान करती है।

#### प्रश्न 4. क्या मैं सम्मिलित कॉलम चार्ट वाले दस्तावेज़ को विभिन्न प्रारूपों में सहेज सकता हूँ?
 हां, Aspose.Words for .NET आपको सम्मिलित कॉलम चार्ट के साथ दस्तावेज़ को विभिन्न प्रारूपों में सहेजने की अनुमति देता है, जैसे कि DOCX, PDF, HTML, और अधिक। आप अपनी आवश्यकताओं के आधार पर वांछित आउटपुट प्रारूप चुन सकते हैं और इसका उपयोग कर सकते हैं।`Save` की विधि`Document` दस्तावेज़ को सहेजने के लिए ऑब्जेक्ट का उपयोग करें। सम्मिलित कॉलम चार्ट सहेजे गए दस्तावेज़ में संरक्षित रहेगा।

#### प्रश्न 5. क्या मैं कॉलम चार्ट डालने के बाद उसके डेटा और स्वरूप को संशोधित कर सकता हूँ?
हां, दस्तावेज़ में कॉलम चार्ट डालने के बाद, आप .NET के लिए Aspose.Words द्वारा प्रदान किए गए API का उपयोग करके इसके डेटा और उपस्थिति को संशोधित कर सकते हैं। आप नई श्रेणियों और मूल्यों के साथ श्रृंखला डेटा को अपडेट कर सकते हैं, कॉलम के रंग और स्वरूपण को बदल सकते हैं, अक्ष गुणों को अनुकूलित कर सकते हैं, और अपने Word दस्तावेज़ों में गतिशील और नेत्रहीन आकर्षक चार्ट बनाने के लिए विभिन्न स्वरूपण विकल्प लागू कर सकते हैं।
