नीचे C# स्रोत कोड को समझाने के लिए चरण-दर-चरण मार्गदर्शिका दी गई है, जो .NET के लिए Aspose.Words की "गेट मर्ज फ़ील्ड नाम" सुविधा का उपयोग करती है। वांछित परिणाम प्राप्त करने के लिए प्रत्येक चरण का सावधानीपूर्वक पालन करना सुनिश्चित करें।

## चरण 1: दस्तावेज़ निर्देशिका सेटअप

दिए गए कोड में, आपको अपने दस्तावेज़ों की निर्देशिका निर्दिष्ट करनी होगी। "आपकी दस्तावेज़ निर्देशिका" मान को अपने दस्तावेज़ निर्देशिका के लिए उपयुक्त पथ से बदलें।

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## चरण 2: दस्तावेज़ लोड करना

पहला चरण उस दस्तावेज़ को लोड करना है जहां आप मर्ज फ़ील्ड नाम प्राप्त करना चाहते हैं।

```csharp
Document doc = new Document(dataDir + "YOUR DOCUMENT FILE");
```

"आपकी दस्तावेज़ फ़ाइल" को अपनी फ़ाइल के नाम से प्रतिस्थापित करना सुनिश्चित करें।

## चरण 3: मर्ज फ़ील्ड नाम प्राप्त करें

 हम उपयोग करते हैं`GetFieldNames()` दस्तावेज़ में मौजूद मर्ज फ़ील्ड के नामों वाली एक सरणी प्राप्त करने की विधि।

```csharp
string[] fieldNames = doc.MailMerge.GetFieldNames();
```

`fieldNames` वेरिएबल में अब मर्ज फ़ील्ड के नाम शामिल हैं.

### .NET के लिए Aspose.Words के साथ मर्ज फ़ील्ड नाम प्राप्त करने के लिए स्रोत कोड उदाहरण

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// दस्तावेज़ लोड करें.
Document doc = new Document(dataDir + "YOUR DOCUMENT FILE");

// मर्ज फ़ील्ड नाम प्राप्त करें.
string[] fieldNames = doc.MailMerge.GetFieldNames();

// मर्ज फ़ील्ड की संख्या प्रदर्शित करें.
Console.WriteLine("\nDocument contains " + fieldNames.Length + " merge fields.");
```

 इस उदाहरण में, हमने एक दस्तावेज़ लोड किया, मर्ज फ़ील्ड नाम प्राप्त किए`GetFieldNames()` विधि, और दस्तावेज़ में मौजूद मर्ज फ़ील्ड की संख्या प्रदर्शित की।

यह .NET के लिए Aspose.Words के साथ "गेट मर्ज फील्ड नेम" सुविधा का उपयोग करने पर हमारी मार्गदर्शिका का समापन करता है।

### पूछे जाने वाले प्रश्न

#### प्रश्न 1: Aspose.Words में मेल मर्ज क्या है?

Aspose.Words में मेल मर्ज एक बाहरी स्रोत (जैसे एक्सेल स्प्रेडशीट या डेटाबेस) से डेटा को एक टेम्पलेट वर्ड दस्तावेज़ के साथ मर्ज करने की प्रक्रिया है, ताकि दस्तावेज़ों को व्यक्तिगत बनाया जा सके। यह पत्रों, रिपोर्टों और अन्य समान दस्तावेज़ों के स्वचालित निर्माण की सुविधा प्रदान करता है।

#### प्रश्न 2: मैं वर्ड दस्तावेज़ में उपलब्ध मेल मर्ज फ़ील्ड की सूची कैसे प्राप्त कर सकता हूँ?

किसी Word दस्तावेज़ में उपलब्ध मेल मर्ज फ़ील्ड की सूची प्राप्त करने के लिए, आप इन चरणों का पालन कर सकते हैं:

1. Aspose.Words नामस्थान से Document और MailMergeFieldNames वर्गों को आयात करें।
2. अपना वर्ड दस्तावेज़ लोड करके एक दस्तावेज़ उदाहरण बनाएँ.
3. उपलब्ध मेल मर्ज फ़ील्ड की सूची प्राप्त करने के लिए दस्तावेज़ ऑब्जेक्ट की GetMailMergeFieldNames विधि का उपयोग करें।

इस प्रक्रिया को समझाने के लिए नमूना कोड यहां दिया गया है:

```csharp
// आवश्यक नामस्थान आयात करें
using Aspose.Words;
using Aspose.Words.MailMerging;

// मौजूदा दस्तावेज़ लोड करें
Document document = new Document("FilePath");

// मेल मर्ज फ़ील्ड की सूची प्राप्त करें
MailMergeFieldNames fieldNames = document.MailMerge.GetFieldNames();

// उपलब्ध मेल मर्ज फ़ील्ड के माध्यम से चक्र करें
foreach (string fieldName in fieldNames)
{
     // फ़ील्ड नाम के साथ कुछ करें
     Console.WriteLine(fieldName);
}
```
### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: Aspose.Words में मेल मर्ज क्या है?

उत्तर: Aspose.Words में मेल मर्ज एक बाहरी स्रोत (जैसे एक्सेल स्प्रेडशीट या डेटाबेस) से डेटा को एक टेम्पलेट वर्ड दस्तावेज़ के साथ मर्ज करने की प्रक्रिया है, ताकि दस्तावेज़ों को व्यक्तिगत बनाया जा सके। यह पत्रों, रिपोर्टों और अन्य समान दस्तावेज़ों के स्वचालित निर्माण की सुविधा प्रदान करता है।

#### प्रश्न: मैं वर्ड दस्तावेज़ में उपलब्ध मेल मर्ज फ़ील्ड की सूची कैसे प्राप्त कर सकता हूँ?

उत्तर: किसी Word दस्तावेज़ में उपलब्ध मेल मर्ज फ़ील्ड की सूची प्राप्त करने के लिए, आप इन चरणों का पालन कर सकते हैं:

#### प्रश्न: क्या मैं किसी बाह्य डेटा स्रोत जैसे एक्सेल स्प्रेडशीट से मेल मर्ज फ़ील्ड प्राप्त कर सकता हूँ?

उत्तर: हाँ, आप एक्सेल स्प्रेडशीट जैसे बाहरी डेटा स्रोत से मेल मर्ज फ़ील्ड प्राप्त कर सकते हैं। इसके लिए, आप डेटा स्रोत के साथ कनेक्शन स्थापित करने और उपलब्ध फ़ील्ड के नाम प्राप्त करने के लिए Aspose.Words की डेटा बाइंडिंग सुविधाओं का उपयोग कर सकते हैं।

#### प्रश्न: क्या कुछ मानदंडों के आधार पर मेल मर्ज फ़ील्ड को फ़िल्टर करना संभव है?

उत्तर: हां, कुछ मानदंडों के आधार पर मेल मर्ज फ़ील्ड को फ़िल्टर करना संभव है। आप मेल मर्ज फ़ील्ड को फ़िल्टर करने के लिए रेगुलर एक्सप्रेशन या विशिष्ट शर्तों का उपयोग कर सकते हैं और केवल वे फ़ील्ड प्राप्त कर सकते हैं जो आपके विशिष्ट मानदंडों को पूरा करते हैं।

#### प्रश्न: मैं Aspose.Words में मेल मर्ज फ़ील्ड में कैसे बदलाव कर सकता हूँ?

उत्तर: Aspose.Words में मेल मर्ज फ़ील्ड में हेरफेर करने के लिए, आप Document और MailMergeField ऑब्जेक्ट द्वारा प्रदान की गई विधियों और गुणों का उपयोग कर सकते हैं। आप मेल मर्ज फ़ील्ड जोड़ सकते हैं, हटा सकते हैं या अपडेट कर सकते हैं, साथ ही फ़ील्ड से जुड़े मानों को पुनः प्राप्त और संपादित कर सकते हैं।
