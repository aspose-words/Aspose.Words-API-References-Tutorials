.NET कार्यक्षमता के लिए Aspose.Words का उपयोग करके हाइपरलिंक को बदलने के लिए निम्नलिखित C# स्रोत कोड को समझाने के लिए चरण-दर-चरण मार्गदर्शिका यहां दी गई है। सुनिश्चित करें कि आपने इस कोड का उपयोग करने से पहले Aspose.Words लाइब्रेरी को अपने प्रोजेक्ट में शामिल कर लिया है।

## चरण 1: दस्तावेज़ निर्देशिका पथ सेट करें

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR_DIRECTORY_OF_DOCUMENTS";
```

 अपने दस्तावेज़ निर्देशिका के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें`Hyperlinks.docx` फ़ाइल।

## चरण 2: हाइपरलिंक वाले दस्तावेज़ को लोड करें

```csharp
Document doc = new Document(dataDir + "Hyperlinks.docx");
```

 यहां हम इसका एक उदाहरण बना रहे हैं`Document` निर्दिष्ट फ़ाइल से कक्षा।

## चरण 3: हाइपरलिंक ढूंढने के लिए फ़ील्ड ब्राउज़ करें

```csharp
foreach(Field field in doc.Range.Fields)
{
     if (field.Type == FieldType.FieldHyperlink)
     {
         FieldHyperlink hyperlink = (FieldHyperlink)field;

         // कुछ हाइपरलिंक स्थानीय हो सकते हैं (दस्तावेज़ के अंदर बुकमार्क के लिंक), हम उन्हें अनदेखा कर देते हैं।
         if (hyperlink.SubAddress != null)
             keep on going;

         hyperlink.Address = "http://www.aspose.com";
         hyperlink.Result = "Aspose - The .NET & Java component editor";
     }
}
```

 यह लूप दस्तावेज़ में सभी फ़ील्ड के माध्यम से प्रकार के फ़ील्ड की तलाश में जाता है`FieldType.FieldHyperlink` . एक बार जब इस प्रकार का कोई फ़ील्ड मिल जाता है, तो हम जाँच करके जाँचते हैं कि क्या यह एक स्थानीय लिंक है`SubAddress` संपत्ति। यदि नहीं, तो हम लिंक पता बदल देते हैं`"http://www.aspose.com"` और परिणाम के साथ`"Aspose - The .NET & Java Component Editor"`.

## चरण 4: संशोधित दस्तावेज़ सहेजें

```csharp
doc.Save(dataDir + "WorkingWithFields.ReplaceHyperlinks.docx");
```

अंत में, हम संशोधित दस्तावेज़ को प्रतिस्थापित हाइपरलिंक के साथ एक निर्दिष्ट फ़ाइल में सहेजते हैं।

### .NET के लिए हाइपरलिंक को Aspose.Words से बदलने के लिए उदाहरण स्रोत कोड

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR_DIRECTORY_OF_DOCUMENTS";

Document doc = new Document(dataDir + "Hyperlinks.docx");

foreach(Field field in doc.Range.Fields)
{
     if (field.Type == FieldType.FieldHyperlink)
     {
         FieldHyperlink hyperlink = (FieldHyperlink)field;

         // कुछ हाइपरलिंक स्थानीय हो सकते हैं (दस्तावेज़ के अंदर बुकमार्क के लिंक), हम उन्हें अनदेखा कर देते हैं।
         if (hyperlink.SubAddress != null)
             keep on going;

         hyperlink.Address = "http://www.aspose.com";
         hyperlink.Result = "Aspose - The .NET & Java component editor";
     }
}

doc.Save(dataDir + "WorkingWithFields.ReplaceHyperlinks.docx");
```

यह .NET के लिए Aspose.Words का उपयोग करके किसी दस्तावेज़ में हाइपरलिंक को बदलने के लिए नमूना स्रोत कोड है।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके किसी Word दस्तावेज़ में हाइपरलिंक्स को कैसे बदल सकता हूँ?

 उ: .NET के लिए Aspose.Words का उपयोग करके किसी Word दस्तावेज़ में हाइपरलिंक को बदलने के लिए, आप इसका उपयोग कर सकते हैं`Document.Range.Replace`खोजे जाने वाले पाठ और प्रतिस्थापन पाठ को निर्दिष्ट करने की विधि। खोज और पैरामीटर बदलने के लिए उपयुक्त विकल्पों का उपयोग करना सुनिश्चित करें।

#### प्रश्न: क्या .NET के लिए किसी Word दस्तावेज़ में केवल कुछ हाइपरलिंक्स को Aspose.Words से बदलना संभव है?

उत्तर: हाँ, किसी Word दस्तावेज़ में केवल कुछ हाइपरलिंक्स को .NET के लिए Aspose.Words से बदलना संभव है। आप विशिष्ट मानदंडों, जैसे लिंक यूआरएल, लिंक टेक्स्ट, या किसी अन्य प्रासंगिक संपत्ति का उपयोग करके प्रतिस्थापित किए जाने वाले हाइपरलिंक को फ़िल्टर कर सकते हैं। फिर आप प्रतिस्थापन को केवल मेल खाने वाले हाइपरलिंक पर लागू कर सकते हैं।

#### प्रश्न: .NET के लिए Aspose.Words से प्रतिस्थापित करते समय मैं हेडर, फ़ूटर या फ़ुटनोट में हाइपरलिंक को कैसे अनदेखा कर सकता हूँ?

उ: .NET के लिए Aspose.Words के साथ प्रतिस्थापित करते समय हेडर, फ़ूटर या फ़ुटनोट में हाइपरलिंक को अनदेखा करने के लिए, आप उन्नत खोज विकल्पों का उपयोग कर सकते हैं और उचित खोज सीमाएँ निर्दिष्ट कर सकते हैं। उदाहरण के लिए, आप खोज को दस्तावेज़ के प्रमुख अनुभागों तक सीमित कर सकते हैं और शीर्षलेख, पादलेख या फ़ुटनोट को बाहर कर सकते हैं।

#### प्रश्न: क्या दस्तावेज़ के अन्य भागों में हाइपरलिंक को आंतरिक लिंक से बदलना संभव है?

 उ: हाँ, .NET के लिए Aspose.Words के साथ हाइपरलिंक को दस्तावेज़ के अन्य भागों के आंतरिक लिंक से बदलना संभव है। आप आंतरिक लिंक बनाने के लिए एंकर या टेक्स्ट आईडी का उपयोग कर सकते हैं और फिर उनका उपयोग करके उन्हें बदल सकते हैं`Document.Range.Replace` उचित विकल्पों के साथ विधि.

#### प्रश्न: क्या .NET के लिए हाइपरलिंक को Aspose.Words से बदलने से रंग या शैली जैसे लिंक गुण सुरक्षित रहते हैं?

उत्तर: हां, .NET के लिए हाइपरलिंक को Aspose.Words से प्रतिस्थापित करते समय, रंग या शैली जैसे लिंक गुण बरकरार रहते हैं। सुसंगत परिणाम प्राप्त करने के लिए आप प्रतिस्थापन पाठ में समान स्वरूपण गुण निर्दिष्ट कर सकते हैं।
