Aspose.Words for .NET एक शक्तिशाली लाइब्रेरी है जो C# एप्लीकेशन में Word दस्तावेज़ बनाने, संपादित करने और हेरफेर करने के लिए है। Aspose.Words द्वारा प्रदान की जाने वाली कार्यक्षमताओं में, Word दस्तावेज़ की विषय-सूची में उपयोग किए जाने वाले टैब को संशोधित करने की संभावना है। इस गाइड में, हम आपको दिखाएंगे कि दस्तावेज़ की विषय-सूची में टैब बदलने के लिए Aspose.Words for .NET के C# स्रोत कोड का उपयोग कैसे करें।

## Aspose.Words लाइब्रेरी को समझना

कोड में गोता लगाने से पहले, .NET के लिए Aspose.Words लाइब्रेरी को समझना महत्वपूर्ण है। Aspose.Words एक लोकप्रिय लाइब्रेरी है जो Word दस्तावेज़ों के साथ Word प्रोसेसिंग को आसान और कुशल बनाती है। यह Word दस्तावेज़ों को बनाने, संपादित करने और हेरफेर करने के लिए कई प्रकार की सुविधाएँ प्रदान करता है, जिसमें सामग्री टैब की तालिका बदलना भी शामिल है।

## विषय-सूची युक्त दस्तावेज़ लोड करना

पहला चरण उस Word दस्तावेज़ को लोड करना है जिसमें वह सामग्री तालिका है जिसे आप संशोधित करना चाहते हैं। स्रोत फ़ाइल से दस्तावेज़ लोड करने के लिए Document वर्ग का उपयोग करें। यहाँ एक उदाहरण दिया गया है:

```csharp
Document doc = new Document(dataDir + "Table of contents.docx");
```

इस उदाहरण में, हम दस्तावेज़ निर्देशिका में स्थित दस्तावेज़ "Table of content.docx" को लोड करते हैं।

## विषय-सूची में टैब बदलना

एक बार दस्तावेज़ लोड हो जाने के बाद, हम दस्तावेज़ के प्रत्येक पैराग्राफ़ को देखते हैं और जाँचते हैं कि क्या यह सामग्री तालिका (TOC) परिणाम शैलियों का उपयोग करके स्वरूपित है। यदि ऐसा है, तो हम पृष्ठ संख्याओं को संरेखित करने के लिए उपयोग किए जाने वाले टैब को संशोधित करते हैं। यहाँ बताया गया है कि कैसे:

```csharp
foreach(Paragraph para in doc.GetChildNodes(NodeType.Paragraph, true))
{
     if (para.ParagraphFormat.Style.StyleIdentifier >= StyleIdentifier.Toc1 &&
         para.ParagraphFormat.Style.StyleIdentifier <= StyleIdentifier.Toc9)
     {
         TabStop tab = para.ParagraphFormat.TabStops[0];
         para.ParagraphFormat.TabStops.RemoveByPosition(tab.Position);
         para.ParagraphFormat.TabStops.Add(tab.Position - 50, tab.Alignment, tab.Leader);
     }
}
```

इस उदाहरण में, हम दस्तावेज़ में प्रत्येक पैराग्राफ़ को लूप करने के लिए लूप का उपयोग कर रहे हैं। फिर हम जाँचते हैं कि पैराग्राफ़ को टेबल ऑफ़ कॉन्टेंट रिजल्ट (TOC) शैलियों का उपयोग करके फ़ॉर्मेट किया गया है या नहीं। यदि ऐसा है, तो हम इस पैराग्राफ़ में उपयोग किए गए पहले टैब तक पहुँचते हैं और पुराने टैब को हटाकर और संशोधित स्थिति के साथ एक नया टैब जोड़कर इसे संशोधित करते हैं।

## संशोधित दस्तावेज़ सहेजें

एक बार जब आप विषय-सूची में टैब में आवश्यक परिवर्तन कर लेते हैं, तो आप Document वर्ग की Save विधि का उपयोग करके संशोधित दस्तावेज़ को सहेज सकते हैं। यहाँ एक उदाहरण दिया गया है:

```csharp
doc.Save(dataDir + "WorkingWithTableOfContent.ChangeTocTabStops.docx");
```

इस उदाहरण में, हम संशोधित दस्तावेज़ को "WorkingWithTableOfContent.ChangeTocTabStops.docx" के रूप में सहेजते हैं।

#### प्रश्न: Aspose.Words for .NET में "Word Document में Toc Tab Stops बदलें" कार्यक्षमता का उद्देश्य क्या है?

उत्तर: Aspose.Words for .NET में "वर्ड डॉक्यूमेंट में टॉक टैब स्टॉप बदलें" कार्यक्षमता आपको वर्ड डॉक्यूमेंट की विषय-सूची में उपयोग किए जाने वाले टैब स्टॉप को संशोधित करने की अनुमति देती है। यह आपको विषय-सूची के भीतर पृष्ठ संख्याओं और संबंधित शीर्षकों के संरेखण और स्थिति को अनुकूलित करने में सक्षम बनाता है।

#### प्रश्न: .NET के लिए Aspose.Words क्या है?

उत्तर: Aspose.Words for .NET एक शक्तिशाली लाइब्रेरी है जिसे .NET अनुप्रयोगों में Word दस्तावेज़ों के साथ Words प्रोसेसिंग के लिए डिज़ाइन किया गया है। यह C# या अन्य .NET भाषाओं का उपयोग करके Word दस्तावेज़ों को प्रोग्रामेटिक रूप से बनाने, संपादित करने, हेरफेर करने और परिवर्तित करने के लिए व्यापक सुविधाएँ प्रदान करता है।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके सामग्री तालिका वाले Word दस्तावेज़ को कैसे लोड करूं?

 उत्तर: Aspose.Words for .NET का उपयोग करके सामग्री तालिका वाले Word दस्तावेज़ को लोड करने के लिए, आप इसका उपयोग कर सकते हैं`Document` क्लास और उसके कंस्ट्रक्टर। दस्तावेज़ का फ़ाइल पथ प्रदान करके, आप इसे एक में लोड कर सकते हैं`Document` वस्तु। यहाँ एक उदाहरण है:

```csharp
Document doc = new Document(dataDir + "Table of contents.docx");
```

यह कोड स्निपेट निर्दिष्ट निर्देशिका में स्थित दस्तावेज़ "Table of content.docx" को लोड करता है।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके सामग्री तालिका में प्रयुक्त टैब्स को कैसे बदल सकता हूँ?

उत्तर: एक बार दस्तावेज़ लोड हो जाने के बाद, आप दस्तावेज़ के प्रत्येक पैराग्राफ़ को फिर से देख सकते हैं और जाँच सकते हैं कि क्या यह सामग्री तालिका (TOC) परिणाम शैलियों का उपयोग करके स्वरूपित है। यदि कोई पैराग्राफ़ TOC शैली के रूप में स्वरूपित है, तो आप पृष्ठ संख्याओं को संरेखित करने के लिए उपयोग किए जाने वाले टैब को संशोधित कर सकते हैं। .NET के लिए Aspose.Words में, आप एक्सेस कर सकते हैं`ParagraphFormat` टैब स्टॉप को पुनः प्राप्त करने और संशोधित करने के लिए प्रत्येक पैराग्राफ की संपत्ति। यहाँ एक उदाहरण है:

```csharp
foreach(Paragraph para in doc.GetChildNodes(NodeType.Paragraph, true))
{
    if (para.ParagraphFormat.Style.StyleIdentifier >= StyleIdentifier.Toc1 &&
        para.ParagraphFormat.Style.StyleIdentifier <= StyleIdentifier.Toc9)
    {
        TabStop tab = para.ParagraphFormat.TabStops[0];
        para.ParagraphFormat.TabStops.RemoveByPosition(tab.Position);
        para.ParagraphFormat.TabStops.Add(tab.Position - 50, tab.Alignment, tab.Leader);
    }
}
```

इस कोड में, लूप दस्तावेज़ में प्रत्येक पैराग्राफ़ के माध्यम से पुनरावृत्त होता है। यदि किसी पैराग्राफ़ में TOC शैली है, तो यह उस पैराग्राफ़ में उपयोग किए गए पहले टैब स्टॉप तक पहुँचता है, उसे हटाता है, और संशोधित स्थिति के साथ एक नया टैब स्टॉप जोड़ता है।

उत्तर: हाँ, आप Aspose.Words for .NET का उपयोग करके विषय-सूची में कई स्तरों के लिए टैब बदल सकते हैं। प्रत्येक पैराग्राफ़ को दोहराकर और TOC शैली की जाँच करके, आप प्रत्येक स्तर के लिए टैब को व्यक्तिगत रूप से संशोधित कर सकते हैं। आप विषय-सूची के वांछित स्तर तक पहुँच सकते हैं और टैब स्टॉप को तदनुसार समायोजित कर सकते हैं।

#### प्रश्न: मैं Aspose.Words for .NET का उपयोग करके सामग्री तालिका में टैब बदलने के बाद संशोधित दस्तावेज़ को कैसे सहेजूँ?

 उत्तर: विषय-सूची के टैब में आवश्यक परिवर्तन करने के बाद, आप संशोधित दस्तावेज़ को निम्न का उपयोग करके सहेज सकते हैं:`Save` की विधि`Document` क्लास। आउटपुट दस्तावेज़ के लिए वांछित फ़ाइल पथ और नाम को पैरामीटर के रूप में प्रदान करें`Save` विधि। यहाँ एक उदाहरण है:

