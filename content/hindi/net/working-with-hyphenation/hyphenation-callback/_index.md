## चरण 1: हाइफ़नेशन कॉलबैक पंजीकृत करें

आरंभ करने के लिए, हमें अपना हाइफ़नेशन कॉलबैक पंजीकृत करना होगा। यहीं पर हम Aspose.Words को हमारे कस्टम हाइफ़नेशन तर्क का उपयोग करने के लिए कहते हैं।

```csharp
try
{
    // हाइफ़नेशन कॉलबैक पंजीकृत करें.
    Hyphenation.Callback = new CustomHyphenationCallback();
}
catch (Exception e)
{
    Console.WriteLine($"Error registering hyphenation callback: {e.Message}");
}
```

 यहां, हम अपने कस्टम कॉलबैक का एक उदाहरण बना रहे हैं और उसे असाइन कर रहे हैं`Hyphenation.Callback`.

इसके बाद, हमें उस निर्देशिका को परिभाषित करने की आवश्यकता है जहां हमारे दस्तावेज़ संग्रहीत हैं। यह महत्वपूर्ण है क्योंकि हम इस पथ से दस्तावेज़ों को लोड और सहेजेंगे।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ों के वास्तविक पथ के साथ।

दस्तावेज़ को लोड करने के बाद, हम प्रक्रिया में हाइफ़नेशन कॉलबैक लागू करके इसे एक नई फ़ाइल में सहेजते हैं।

document.Save(dataDir + "TreatmentByCesureWithRecall.pdf");

यह पंक्ति हाइफ़नेशन लागू करके दस्तावेज़ को पीडीएफ के रूप में सहेजती है।

## चरण 5: लुप्त हाइफ़नेशन डिक्शनरी अपवाद को संभालें

कभी-कभी, आपको ऐसी समस्या का सामना करना पड़ सकता है जहां हाइफ़नेशन शब्दकोश गायब है। आइए इसे संभालें.

```csharp
catch (Exception e) when (e.Message.StartsWith("Missing hyphenation dictionary"))
{
    Console.WriteLine(e.Message);
}
finally
{
    Hyphenation.Callback = null;
}
```

इस ब्लॉक में, हम लुप्त शब्दकोशों से संबंधित विशिष्ट अपवाद को पकड़ते हैं और संदेश को प्रिंट करते हैं।

## चरण 6: कस्टम हाइफ़नेशन कॉलबैक क्लास लागू करें

 अब, आइए इसे लागू करें`CustomHyphenationCallback` वह वर्ग जो हाइफ़नेशन शब्दकोशों के अनुरोध को संभालता है।

```csharp
public class CustomHyphenationCallback : IHyphenationCallback
{
    public void RequestDictionary(string language)
    {
        string dictionaryFolder = MyDir;
        string dictionaryFullFileName;
        switch (language)
        {
            case "en-US":
                dictionaryFullFileName = Path.Combine(dictionaryFolder, "hyph_en_US.dic");
                break;
            case "de-CH":
                dictionaryFullFileName = Path.Combine(dictionaryFolder, "hyph_de_CH.dic");
                break;
            default:
                throw new Exception($"Missing hyphenation dictionary for {language}.");
        }
        // अनुरोधित भाषा के लिए शब्दकोश पंजीकृत करें।
        Hyphenation.RegisterDictionary(language, dictionaryFullFileName);
    }
}
```

 इस वर्ग में,`RequestDictionary` जब भी हाइफ़नेशन डिक्शनरी की आवश्यकता होती है तो विधि को कॉल किया जाता है। यह भाषा की जाँच करता है और उपयुक्त शब्दकोश को पंजीकृत करता है।

आखिर तुमने इसे हासिल कर ही लिया है! आपने अभी सीखा कि .NET के लिए Aspose.Words में हाइफ़नेशन कॉलबैक कैसे लागू किया जाए। इन चरणों का पालन करके, आप यह सुनिश्चित कर सकते हैं कि भाषा की परवाह किए बिना आपके दस्तावेज़ खूबसूरती से प्रारूपित हैं। चाहे आप अंग्रेजी, जर्मन, या किसी अन्य भाषा से निपट रहे हों, यह विधि आपको आसानी से हाइफ़नेशन को संभालने की अनुमति देती है।

## पूछे जाने वाले प्रश्न

### .NET के लिए Aspose.Words क्या है?
.NET के लिए Aspose.Words एक शक्तिशाली दस्तावेज़ हेरफेर लाइब्रेरी है जो डेवलपर्स को दस्तावेज़ों को प्रोग्रामेटिक रूप से बनाने, संशोधित करने और परिवर्तित करने की अनुमति देता है।

### दस्तावेज़ स्वरूपण में हाइफ़नेशन क्यों महत्वपूर्ण है?
हाइफ़नेशन शब्दों को उपयुक्त स्थानों पर तोड़कर टेक्स्ट लेआउट में सुधार करता है, जिससे दस्तावेज़ अधिक पठनीय और देखने में आकर्षक लगता है।

### क्या मैं Aspose.Words का निःशुल्क उपयोग कर सकता हूँ?
 Aspose.Words निःशुल्क परीक्षण प्रदान करता है। आप ये पा सकते हैं[यहाँ](https://releases.aspose.com/).

### मैं हाइफ़नेशन शब्दकोश कैसे प्राप्त कर सकता हूँ?
आप विभिन्न ऑनलाइन संसाधनों से हाइफ़नेशन शब्दकोश डाउनलोड कर सकते हैं या यदि आवश्यक हो तो अपना स्वयं का शब्दकोश बना सकते हैं।

### यदि हाइफ़नेशन शब्दकोश गुम हो तो क्या होगा?
 यदि कोई शब्दकोष गुम है, तो`RequestDictionary` विधि एक अपवाद फेंकती है, जिसे आप उपयोगकर्ता को सूचित करने या फ़ॉलबैक प्रदान करने के लिए संभाल सकते हैं।
