इस ट्यूटोरियल में, हम आपको दिखाएंगे कि .NET के लिए Aspose.Words में फॉन्ट डायरेक्टरी कैसे सेट करें। आप सीखेंगे कि अपने वर्ड दस्तावेज़ में उपयोग किए गए फ़ॉन्ट वाली निर्देशिका को कैसे निर्दिष्ट करें।

## आवश्यक शर्तें
शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित चीज़ें हैं:
- C# प्रोग्रामिंग भाषा का कार्यसाधक ज्ञान
- आपके प्रोजेक्ट में .NET के लिए Aspose.Words लाइब्रेरी स्थापित है

## चरण 1: दस्तावेज़ निर्देशिका को परिभाषित करें
अपने Word दस्तावेज़ के स्थान पर निर्देशिका पथ सेट करके प्रारंभ करें। प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` उचित पथ के साथ कोड में।

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## चरण 2: फ़ॉन्ट निर्देशिका सेट करें
 का एक उदाहरण बनाएं`FontSettings` कक्षा और उपयोग करें`SetFontsFolder` फ़ॉन्ट युक्त निर्देशिका को निर्दिष्ट करने की विधि। प्रतिस्थापित करें`"Fonts"` वास्तविक फ़ॉन्ट निर्देशिका के नाम के साथ।

```csharp
FontSettings fontSettings = new FontSettings();
fontSettings.SetFontsFolder(dataDir + "Fonts", false);
```

## चरण 3: दस्तावेज़ को फ़ॉन्ट सेटिंग्स के साथ लोड करें
 उपयोग`LoadOptions` वर्ग में फ़ॉन्ट सेटिंग्स निर्दिष्ट करने के लिए`FontSettings` विकल्प। फिर उपयोग करें`Document` इन विकल्पों का उपयोग करके दस्तावेज़ को लोड करने के लिए क्लास।

```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.FontSettings = fontSettings;
Document doc = new Document(dataDir + "Rendering.docx", loadOptions);

```

### .NET के लिए Aspose.Words का उपयोग करके सेट फ़ॉन्ट्स फ़ोल्डर के लिए नमूना स्रोत कोड 

```csharp

// आपकी दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";

FontSettings fontSettings = new FontSettings();
fontSettings.SetFontsFolder(dataDir + "Fonts", false);
LoadOptions loadOptions = new LoadOptions();
loadOptions.FontSettings = fontSettings;
Document doc = new Document(dataDir + "Rendering.docx", loadOptions);

```

## निष्कर्ष
बधाई हो! अब आप जानते हैं कि .NET के लिए Aspose.Words में फ़ॉन्ट निर्देशिका कैसे सेट करें। आप इस सुविधा का उपयोग अपने दस्तावेज़ में उपयोग किए गए फ़ॉन्ट की उपलब्धता सुनिश्चित करने और फ़ॉन्ट के प्रदर्शन में स्थिरता सुनिश्चित करने के लिए कर सकते हैं।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मैं Aspose.Words में एक कस्टम फ़ॉन्ट फ़ोल्डर कैसे सेट कर सकता हूं?

 उ: Aspose.Words में एक कस्टम फ़ॉन्ट फ़ोल्डर सेट करने के लिए, आप इसका उपयोग कर सकते हैं`FontsFolder` कक्षा और`SetFontsFolders` आपके फ़ॉन्ट वाले फ़ोल्डर का पथ निर्दिष्ट करने वाली विधि।

#### प्रश्न: क्या मैं Aspose.Words में एकाधिक फ़ॉन्ट फ़ोल्डर सेट कर सकता हूँ?

 उ: हां, आप कॉल करके Aspose.Words में एकाधिक फ़ॉन्ट फ़ोल्डर सेट कर सकते हैं`SetFontsFolders` आप जिन विभिन्न फ़ॉन्ट फ़ोल्डरों का उपयोग करना चाहते हैं उनके पथों के साथ कई बार विधि का उपयोग करें।

#### प्रश्न: यदि दस्तावेज़ में प्रयुक्त फ़ॉन्ट परिभाषित फ़ॉन्ट फ़ोल्डरों में मौजूद नहीं है तो क्या होगा?

उ: यदि दस्तावेज़ में उपयोग किया गया फ़ॉन्ट Aspose.Words में परिभाषित फ़ॉन्ट फ़ोल्डरों में मौजूद नहीं है, तो उसके स्थान पर एक वैकल्पिक फ़ॉन्ट का उपयोग किया जाएगा। यह सुनिश्चित करता है कि दस्तावेज़ में पाठ हमेशा सही ढंग से प्रदर्शित किया जाएगा, भले ही मूल फ़ॉन्ट उपलब्ध न हो।

#### प्रश्न: क्या Aspose.Words में परिभाषित फ़ॉन्ट फ़ोल्डरों को सिस्टम पर स्थापित फ़ॉन्ट्स पर प्राथमिकता दी जाती है?

उ: हाँ, Aspose.Words में परिभाषित फ़ॉन्ट फ़ोल्डर सिस्टम पर स्थापित फ़ॉन्ट पर प्राथमिकता लेते हैं। इसका मतलब यह है कि यदि एक ही नाम वाला फ़ॉन्ट परिभाषित फ़ॉन्ट फ़ोल्डर और सिस्टम फ़ॉन्ट दोनों में मौजूद है, तो Word दस्तावेज़ों को संसाधित करते समय फ़ॉन्ट फ़ोल्डर में संस्करण का उपयोग किया जाएगा।
