इस ट्यूटोरियल में, हम आपको Aspose.Words for .NET का उपयोग करके दस्तावेज़ रेंडर करते समय डिफ़ॉल्ट फ़ॉन्ट निर्दिष्ट करने की चरण-दर-चरण प्रक्रिया से अवगत कराएँगे। हम बंडल किए गए C# स्रोत कोड की व्याख्या करेंगे और आपको अपने स्वयं के प्रोजेक्ट में इस सुविधा को समझने और लागू करने में मदद करने के लिए एक व्यापक मार्गदर्शिका प्रदान करेंगे। इस ट्यूटोरियल के अंत तक, आप जान जाएँगे कि Aspose.Words for .NET का उपयोग करके अपने दस्तावेज़ों को रेंडर करते समय उपयोग करने के लिए डिफ़ॉल्ट फ़ॉन्ट कैसे निर्दिष्ट करें।

## चरण 1: दस्तावेज़ निर्देशिका निर्धारित करें
सबसे पहले, आपको अपने दस्तावेज़ निर्देशिका का पथ सेट करना होगा। यह वह स्थान है जहाँ आप अपने संपादित रेंडर किए गए दस्तावेज़ को सहेजना चाहते हैं। "आपके दस्तावेज़ निर्देशिका" को उचित पथ से बदलें।

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## चरण 2: दस्तावेज़ को रेंडर करने के लिए लोड करें
 इसके बाद, आपको दस्तावेज़ को रेंडर करने के लिए लोड करना होगा`Document` क्लास. सही दस्तावेज़ पथ निर्दिष्ट करना सुनिश्चित करें.

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## चरण 3: डिफ़ॉल्ट फ़ॉन्ट सेट करें
 अब आप रेंडरिंग के समय उपयोग करने के लिए डिफ़ॉल्ट फ़ॉन्ट निर्दिष्ट कर सकते हैं, इसके लिए आपको फ़ॉन्ट का एक उदाहरण बनाना होगा।`FontSettings` वर्ग और सेटिंग`DefaultFontName` की संपत्ति`DefaultFontSubstitution` इस पर आपत्ति जताना`DefaultFontSubstitution` वस्तु`SubstitutionSettings` का`FontSettings`.

```csharp
FontSettings fontSettings = new FontSettings();
fontSettings.SubstitutionSettings.DefaultFontSubstitution.DefaultFontName = "Arial Unicode MS";
doc.FontSettings = fontSettings;
```

## चरण 4: प्रस्तुत दस्तावेज़ को सहेजें
 अंत में, आप रेंडर किए गए दस्तावेज़ को फ़ाइल में सहेज सकते हैं`Save()` की विधि`Document` क्लास में सही पथ और फ़ाइल नाम निर्दिष्ट करना सुनिश्चित करें।

```csharp
doc.Save(dataDir + "WorkingWithFonts.SpecifyDefaultFontWhenRendering.pdf");
```

### .NET के लिए Aspose.Words का उपयोग करके रेंडरिंग करते समय डिफ़ॉल्ट फ़ॉन्ट निर्दिष्ट करने के लिए नमूना स्रोत कोड 

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Rendering.docx");
FontSettings fontSettings = new FontSettings();
// यदि यहाँ परिभाषित डिफ़ॉल्ट फ़ॉन्ट रेंडरिंग के दौरान नहीं मिल पाता है तो
// इसके स्थान पर मशीन पर सबसे निकटतम फ़ॉन्ट का उपयोग किया जाता है।
fontSettings.SubstitutionSettings.DefaultFontSubstitution.DefaultFontName = "Arial Unicode MS";
doc.FontSettings = fontSettings;
doc.Save(dataDir + "WorkingWithFonts.SpecifyDefaultFontWhenRendering.pdf");
```

## निष्कर्ष
इस ट्यूटोरियल में, हमने सीखा कि .NET के लिए Aspose.Words का उपयोग करके दस्तावेज़ को रेंडर करते समय डिफ़ॉल्ट फ़ॉन्ट कैसे निर्दिष्ट करें। इस चरण-दर-चरण मार्गदर्शिका का पालन करके, आप अपने दस्तावेज़ों को रेंडर करते समय उपयोग करने के लिए आसानी से एक डिफ़ॉल्ट फ़ॉन्ट सेट कर सकते हैं। Aspose.Words आपके दस्तावेज़ों में फ़ॉन्ट के साथ Words Processing के लिए एक शक्तिशाली और लचीला API प्रदान करता है। इस ज्ञान के साथ, आप अपने दस्तावेज़ों के रेंडरिंग को अपनी विशिष्ट आवश्यकताओं के अनुसार नियंत्रित और अनुकूलित कर सकते हैं।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: Aspose.Words में PDF में कनवर्ट करते समय मैं डिफ़ॉल्ट फ़ॉन्ट कैसे निर्दिष्ट कर सकता हूँ?

 उत्तर: Aspose.Words में PDF में कनवर्ट करते समय डिफ़ॉल्ट फ़ॉन्ट निर्दिष्ट करने के लिए, आप इसका उपयोग कर सकते हैं`PdfOptions` वर्ग और सेट`DefaultFontName` संपत्ति को इच्छित फ़ॉन्ट के नाम में जोड़ें.

#### प्रश्न: यदि पीडीएफ में कनवर्ट करते समय डिफ़ॉल्ट फ़ॉन्ट उपलब्ध न हो तो क्या होगा?

उत्तर: यदि PDF में कनवर्ट करते समय निर्दिष्ट डिफ़ॉल्ट फ़ॉन्ट उपलब्ध नहीं है, तो Aspose.Words कनवर्ट किए गए दस्तावेज़ में टेक्स्ट प्रदर्शित करने के लिए प्रतिस्थापन फ़ॉन्ट का उपयोग करेगा। इससे मूल फ़ॉन्ट से दिखने में थोड़ा अंतर आ सकता है।

#### प्रश्न: क्या मैं अन्य आउटपुट प्रारूपों, जैसे DOCX या HTML के लिए डिफ़ॉल्ट फ़ॉन्ट निर्दिष्ट कर सकता हूँ?

उत्तर: हां, आप उपयुक्त रूपांतरण विकल्पों का उपयोग करके और प्रत्येक प्रारूप के लिए संगत गुण सेट करके DOCX या HTML जैसे अन्य आउटपुट प्रारूपों के लिए डिफ़ॉल्ट फ़ॉन्ट निर्दिष्ट कर सकते हैं।

#### प्रश्न: मैं Aspose.Words में निर्दिष्ट डिफ़ॉल्ट फ़ॉन्ट की जांच कैसे कर सकता हूं?

 उत्तर: Aspose.Words में निर्दिष्ट डिफ़ॉल्ट फ़ॉन्ट की जांच करने के लिए, आप इसका उपयोग कर सकते हैं`DefaultFontName` की संपत्ति`PdfOptions` क्लास पर जाएँ और कॉन्फ़िगर किए गए फ़ॉन्ट का नाम पुनः प्राप्त करें.

#### प्रश्न: क्या दस्तावेज़ के प्रत्येक अनुभाग के लिए अलग डिफ़ॉल्ट फ़ॉन्ट निर्दिष्ट करना संभव है?

उत्तर: हां, प्रत्येक अनुभाग के लिए विशिष्ट स्वरूपण विकल्पों का उपयोग करके दस्तावेज़ के प्रत्येक अनुभाग के लिए एक अलग डिफ़ॉल्ट फ़ॉन्ट निर्दिष्ट करना संभव है। हालाँकि, इसके लिए Aspose.Words सुविधाओं का उपयोग करके दस्तावेज़ के अधिक उन्नत हेरफेर की आवश्यकता होगी।
