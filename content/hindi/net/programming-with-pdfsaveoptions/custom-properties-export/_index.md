इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.Words का उपयोग करके PDF दस्तावेज़ में दस्तावेज़ के कस्टम गुणों को निर्यात करने के चरणों के बारे में बताएँगे। कस्टम गुणों को निर्यात करने से आप जेनरेट किए गए PDF दस्तावेज़ में अतिरिक्त जानकारी शामिल कर सकते हैं। नीचे दिए गए चरणों का पालन करें:

## चरण 1: दस्तावेज़ बनाना और कस्टम गुण जोड़ना

दस्तावेज़ वर्ग का एक उदाहरण बनाकर प्रारंभ करें:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
```

## चरण 2: कस्टम गुण जोड़ें
 इसके बाद, वांछित कस्टम गुण जोड़ें। उदाहरण के लिए, "Aspose" मान के साथ "Company" गुण जोड़ने के लिए, का उपयोग करें`Add` CustomDocumentProperties संग्रह की विधि:

```csharp
doc.CustomDocumentProperties.Add("Company", "Aspose");
```

आप आवश्यकतानुसार कई कस्टम गुण जोड़ सकते हैं.

## चरण 3: PDF निर्यात विकल्प सेट करें

PdfSaveOptions वर्ग का एक उदाहरण बनाएं और निर्दिष्ट करें कि कस्टम गुणों को कैसे निर्यात किया जाए:

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions { CustomPropertiesExport = PdfCustomPropertiesExport.Standard };
```

यह विकल्प पीडीएफ में रूपांतरण करते समय कस्टम गुणों के निर्यात को नियंत्रित करता है।

## चरण 4: दस्तावेज़ को पीडीएफ में बदलें

 उपयोग`Save` रूपांतरण विकल्पों को निर्दिष्ट करते हुए दस्तावेज़ को पीडीएफ में परिवर्तित करने की विधि:

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.CustomPropertiesExport.pdf", saveOptions);
```

परिवर्तित पीडीएफ को सहेजने के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें।

### .NET के लिए Aspose.Words का उपयोग करके कस्टम गुण निर्यात के लिए उदाहरण स्रोत कोड

.NET के लिए Aspose.Words का उपयोग करके किसी दस्तावेज़ से कस्टम गुण निर्यात करने के लिए यहां पूर्ण स्रोत कोड दिया गया है:


```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document();
	doc.CustomDocumentProperties.Add("Company", "Aspose");

	PdfSaveOptions saveOptions = new PdfSaveOptions { CustomPropertiesExport = PdfCustomPropertiesExport.Standard };

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.CustomPropertiesExport.pdf", saveOptions);

```

इन चरणों का पालन करके, आप .NET के लिए Aspose.Words के साथ PDF में कनवर्ट करते समय आसानी से दस्तावेज़ के कस्टम गुणों को निर्यात कर सकते हैं।


## निष्कर्ष

इस ट्यूटोरियल में, हमने बताया कि Aspose.Words for .NET का उपयोग करके किसी दस्तावेज़ से कस्टम गुणों को PDF दस्तावेज़ में कैसे निर्यात किया जाए। वर्णित चरणों का पालन करके, आप दस्तावेज़ के कस्टम गुणों को निर्यात करके उत्पन्न PDF दस्तावेज़ में आसानी से अतिरिक्त जानकारी शामिल कर सकते हैं। कस्टम गुणों को निर्यात करके अपने PDF दस्तावेज़ों को वैयक्तिकृत और समृद्ध करने के लिए Aspose.Words for .NET की सुविधाओं का लाभ उठाएँ।

### अक्सर पूछे जाने वाले प्रश्नों

#### प्रश्न: कस्टम गुणों को PDF दस्तावेज़ में निर्यात करना क्या है?
उत्तर: कस्टम प्रॉपर्टी को PDF दस्तावेज़ में निर्यात करने से जेनरेट किए गए PDF दस्तावेज़ में अतिरिक्त जानकारी शामिल की जा सकती है। कस्टम प्रॉपर्टी आपके दस्तावेज़ के लिए विशिष्ट मेटाडेटा हैं, जैसे टैग, कीवर्ड या क्रेडेंशियल। इन कस्टम प्रॉपर्टी को निर्यात करके, आप उन्हें PDF दस्तावेज़ देखते समय उपयोगकर्ताओं के लिए उपलब्ध करा सकते हैं।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके किसी दस्तावेज़ के कस्टम गुणों को PDF दस्तावेज़ में कैसे निर्यात कर सकता हूँ?
उत्तर: .NET के लिए Aspose.Words का उपयोग करके किसी दस्तावेज़ के कस्टम गुणों को PDF दस्तावेज़ में निर्यात करने के लिए, इन चरणों का पालन करें:

 इसका एक उदाहरण बनाएं`Document` कक्षा।

 का उपयोग करके वांछित कस्टम गुण जोड़ें`CustomDocumentProperties` संग्रह। उदाहरण के लिए, का उपयोग करें`Add` "Aspose" मान के साथ "Company" प्रॉपर्टी जोड़ने की विधि.

 इसका एक उदाहरण बनाएं`PdfSaveOptions` क्लास और निर्दिष्ट करें कि कस्टम गुणों का उपयोग करके कैसे निर्यात किया जाए`CustomPropertiesExport` संपत्ति.`PdfCustomPropertiesExport.Standard` value डिफ़ॉल्ट सेटिंग्स के अनुसार कस्टम गुण निर्यात करता है।

 उपयोग`Save` की विधि`Document` दस्तावेज़ को पीडीएफ में बदलने के लिए रूपांतरण विकल्प निर्दिष्ट करने हेतु क्लास का उपयोग करें।

#### प्रश्न: मैं पीडीएफ दस्तावेज़ के कस्टम गुणों तक कैसे पहुंच सकता हूं?
उत्तर: PDF दस्तावेज़ के कस्टम गुणों तक पहुँचने के लिए, आप एक संगत PDF रीडर का उपयोग कर सकते हैं जो दस्तावेज़ गुणों को देखने का समर्थन करता है। Adobe Acrobat Reader जैसे अधिकांश सामान्य PDF रीडर, PDF दस्तावेज़ के मेटाडेटा और गुणों तक पहुँच प्रदान करते हैं। आप आमतौर पर इन विकल्पों को "फ़ाइल" मेनू के अंतर्गत या दस्तावेज़ पर राइट-क्लिक करके और "गुण" चुनकर पा सकते हैं।
