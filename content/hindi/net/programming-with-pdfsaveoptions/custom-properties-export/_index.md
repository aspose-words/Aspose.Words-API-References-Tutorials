## परिचय

PDF दस्तावेज़ में कस्टम प्रॉपर्टीज़ को एक्सपोर्ट करना विभिन्न व्यावसायिक आवश्यकताओं के लिए अविश्वसनीय रूप से उपयोगी हो सकता है। चाहे आप बेहतर खोज क्षमता के लिए मेटाडेटा प्रबंधित कर रहे हों या अपने दस्तावेज़ों में सीधे महत्वपूर्ण जानकारी एम्बेड कर रहे हों, Aspose.Words for .NET इस प्रक्रिया को सहज बनाता है। यह ट्यूटोरियल आपको Word दस्तावेज़ बनाने, कस्टम प्रॉपर्टीज़ जोड़ने और इन प्रॉपर्टीज़ को बरकरार रखते हुए उन्हें PDF में एक्सपोर्ट करने में मार्गदर्शन करेगा।

## आवश्यक शर्तें

कोड में आगे बढ़ने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

-  Aspose.Words for .NET इंस्टॉल करें। अगर आपने इसे अभी तक इंस्टॉल नहीं किया है, तो आप इसे डाउनलोड कर सकते हैं[यहाँ](https://releases.aspose.com/words/net/).
- विजुअल स्टूडियो जैसा विकास वातावरण.
- C# प्रोग्रामिंग का बुनियादी ज्ञान.

## नामस्थान आयात करें

सबसे पहले, आपको अपने प्रोजेक्ट में आवश्यक नेमस्पेस को आयात करना होगा। इन नेमस्पेस में वर्ड दस्तावेज़ों में हेरफेर करने और उन्हें PDF के रूप में निर्यात करने के लिए आवश्यक क्लास और विधियाँ होती हैं।

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

आइये इस प्रक्रिया को सरल एवं प्रबंधनीय चरणों में विभाजित करें।

## चरण 1: दस्तावेज़ को आरंभ करें

शुरू करने के लिए, आपको एक नया दस्तावेज़ ऑब्जेक्ट बनाना होगा। यह ऑब्जेक्ट कस्टम प्रॉपर्टी जोड़ने और PDF में निर्यात करने के लिए आधार के रूप में काम करेगा।

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
```

## चरण 2: कस्टम गुण जोड़ें

इसके बाद, आप अपने दस्तावेज़ में कस्टम गुण जोड़ेंगे। इन गुणों में कंपनी का नाम, लेखक या कोई अन्य प्रासंगिक जानकारी जैसे मेटाडेटा शामिल हो सकते हैं।

```csharp
doc.CustomDocumentProperties.Add("Company", "Aspose");
```

## चरण 3: पीडीएफ सेव विकल्प कॉन्फ़िगर करें

 अब, पीडीएफ सेव विकल्पों को कॉन्फ़िगर करें ताकि यह सुनिश्चित हो सके कि दस्तावेज़ निर्यात करते समय कस्टम गुण शामिल हों।`PdfSaveOptions` यह क्लास दस्तावेज़ को पीडीएफ के रूप में सहेजने के तरीके को नियंत्रित करने के लिए विभिन्न सेटिंग्स प्रदान करता है।

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions
{
    CustomPropertiesExport = PdfCustomPropertiesExport.Standard
};
```

## चरण 4: दस्तावेज़ को PDF के रूप में सहेजें

 अंत में, दस्तावेज़ को निर्दिष्ट निर्देशिका में पीडीएफ के रूप में सहेजें।`Save` विधि सभी पिछले चरणों को जोड़ती है और कस्टम गुणों को शामिल करते हुए एक पीडीएफ तैयार करती है।

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.CustomPropertiesExport.pdf", saveOptions);
```

## निष्कर्ष

.NET के लिए Aspose.Words का उपयोग करके PDF दस्तावेज़ में कस्टम गुण निर्यात करना एक सीधी प्रक्रिया है जो आपकी दस्तावेज़ प्रबंधन क्षमताओं को बहुत बढ़ा सकती है। इन चरणों का पालन करके, आप यह सुनिश्चित कर सकते हैं कि महत्वपूर्ण मेटाडेटा संरक्षित और सुलभ है, जिससे आपके डिजिटल दस्तावेज़ों की दक्षता और संगठन में सुधार होता है।

## अक्सर पूछे जाने वाले प्रश्न

### पीडीएफ दस्तावेज़ में कस्टम गुण क्या हैं?
कस्टम गुण, दस्तावेज़ में जोड़े गए मेटाडेटा होते हैं, जिनमें लेखक, कंपनी का नाम या कोई अन्य प्रासंगिक डेटा जैसी जानकारी शामिल हो सकती है, जिसे दस्तावेज़ में एम्बेड किया जाना आवश्यक होता है।

### कस्टम प्रॉपर्टीज़ निर्यात करने के लिए मुझे .NET के लिए Aspose.Words का उपयोग क्यों करना चाहिए?
.NET के लिए Aspose.Words, Word दस्तावेज़ों में हेरफेर करने और उन्हें PDF के रूप में निर्यात करने के लिए एक मजबूत और उपयोग में आसान API प्रदान करता है, जो यह सुनिश्चित करता है कि कस्टम गुण संरक्षित और सुलभ हैं।

### क्या मैं किसी दस्तावेज़ में एकाधिक कस्टम गुण जोड़ सकता हूँ?
 हां, आप कॉल करके किसी दस्तावेज़ में कई कस्टम गुण जोड़ सकते हैं`Add`प्रत्येक संपत्ति के लिए विधि जिसे आप शामिल करना चाहते हैं।

### .NET के लिए Aspose.Words का उपयोग करके मैं अन्य किन प्रारूपों में निर्यात कर सकता हूँ?
.NET के लिए Aspose.Words विभिन्न प्रारूपों में निर्यात का समर्थन करता है, जिसमें DOCX, HTML, EPUB, और कई अन्य शामिल हैं।

 सहायता के लिए आप यहां जा सकते हैं[Aspose.Words समर्थन मंच](https://forum.aspose.com/c/words/8) सहायता के लिए.
