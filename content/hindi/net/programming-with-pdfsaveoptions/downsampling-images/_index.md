इस ट्यूटोरियल में, हम आपको Aspose.Words for .NET के साथ PDF में कनवर्ट करते समय डाउनसैंपलिंग इमेज के साथ PDF दस्तावेज़ का आकार कम करने के चरणों के बारे में बताएँगे। इससे जेनरेट की गई PDF फ़ाइल का आकार कम हो जाता है। नीचे दिए गए चरणों का पालन करें:

## चरण 1: दस्तावेज़ लोड करना

उस दस्तावेज़ को अपलोड करके शुरू करें जिसे आप पीडीएफ में बदलना चाहते हैं:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document doc = new Document(dataDir + "Rendering.docx");
```

अपने दस्तावेज़ का सही पथ निर्दिष्ट करना सुनिश्चित करें.

## चरण 2: PDF सेव विकल्प कॉन्फ़िगर करें

PdfSaveOptions वर्ग का एक उदाहरण बनाएं और छवि डाउनस्केलिंग विकल्प सेट करें:

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions
{
DownsampleOptions = { Resolution = 36, ResolutionThreshold = 128 }
};
```

`Resolution` संपत्ति छवियों के लक्ष्य रिज़ॉल्यूशन को निर्दिष्ट करती है और`ResolutionThreshold`गुण न्यूनतम रिज़ॉल्यूशन निर्दिष्ट करता है जिसके नीचे छवियों को छोटा नहीं किया जाएगा।

## चरण 3: दस्तावेज़ को पीडीएफ में बदलें

 उपयोग`Save` दस्तावेज़ को पीडीएफ में परिवर्तित करने की विधि जिसमें सहेजने के विकल्प निर्दिष्ट किए गए हैं:

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.DownsamplingImages.pdf", saveOptions);
```

परिवर्तित पीडीएफ को सहेजने के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें।

### .NET के लिए Aspose.Words का उपयोग करके छवियों के डाउनसैंपलिंग के लिए उदाहरण स्रोत कोड

```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Rendering.docx");

	// हम डाउनसैंपलिंग के लिए न्यूनतम सीमा निर्धारित कर सकते हैं।
	// यह मान इनपुट दस्तावेज़ में दूसरी छवि को डाउनसैंपल होने से रोकेगा।
	PdfSaveOptions saveOptions = new PdfSaveOptions
	{
		DownsampleOptions = { Resolution = 36, ResolutionThreshold = 128 }
	};

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.DownsamplingImages.pdf", saveOptions);

```

इन चरणों का पालन करके, आप Aspose.Words for .NET के साथ PDF में कनवर्ट करते समय आसानी से छवि रिज़ॉल्यूशन को कम कर सकते हैं।

## निष्कर्ष

इस ट्यूटोरियल में, हमने बताया है कि .NET के लिए Aspose.Words का उपयोग करके PDF में कनवर्ट करते समय इमेज सैंपलिंग के साथ PDF दस्तावेज़ का आकार कैसे कम किया जाए। वर्णित चरणों का पालन करके, आप आसानी से छवियों के रिज़ॉल्यूशन और जेनरेट की गई PDF फ़ाइल के आकार को कम कर सकते हैं। अपने दस्तावेज़ के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें और आवश्यकतानुसार इमेज सैंपलिंग विकल्पों को कॉन्फ़िगर करें। PDF फ़ाइल का आकार कम करने से फ़ाइल को अलग-अलग प्लेटफ़ॉर्म पर साझा करना, संग्रहीत करना और तेज़ी से लोड करना आसान हो जाता है। .NET के लिए Aspose.Words का उपयोग करके इमेज सैंपलिंग के साथ PDF दस्तावेज़ का आकार कम करने के लाभों का आनंद लें।

### अक्सर पूछे जाने वाले प्रश्नों

#### प्रश्न: इमेज सैंपलिंग से पीडीएफ दस्तावेज़ का आकार कैसे कम होता है?
उत्तर: इमेज सैंपलिंग के साथ पीडीएफ दस्तावेज़ का आकार कम करने का मतलब है पीडीएफ में कनवर्ट करते समय छवियों के रिज़ॉल्यूशन को कम करके जेनरेट की गई पीडीएफ फ़ाइल का आकार कम करना। यह स्टोरेज स्पेस के उपयोग को अनुकूलित करता है और पीडीएफ फ़ाइल को साझा करना और स्थानांतरित करना आसान बनाता है।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके छवि नमूनाकरण के साथ पीडीएफ दस्तावेज़ का आकार कैसे कम कर सकता हूं?
उत्तर: .NET के लिए Aspose.Words का उपयोग करके छवि नमूनाकरण के साथ PDF दस्तावेज़ का आकार कम करने के लिए, इन चरणों का पालन करें:

 वह निर्देशिका पथ सेट करें जहाँ आपके दस्तावेज़ स्थित हैं`"YOUR DOCUMENTS DIRECTORY"` अपने दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

 उस दस्तावेज़ को लोड करें जिसे आप पीडीएफ में बदलना चाहते हैं`Document` क्लास में जाएं और निर्दिष्ट दस्तावेज़ निर्देशिका में दस्तावेज़ का पथ निर्दिष्ट करें।

 का एक उदाहरण बनाकर PDF के रूप में सहेजें विकल्प कॉन्फ़िगर करें`PdfSaveOptions` वर्ग और छवि नमूनाकरण विकल्पों का उपयोग करके सेटिंग`DownsampleOptions` प्रॉपर्टी। आप छवियों के लक्ष्य रिज़ॉल्यूशन को निर्दिष्ट कर सकते हैं`Resolution` प्रॉपर्टी पर क्लिक करें और एक न्यूनतम रिज़ॉल्यूशन सीमा निर्धारित करें जिसके ऊपर छवियों को स्केल डाउन नहीं किया जाएगा`ResolutionThreshold` संपत्ति।

 दस्तावेज़ को PDF प्रारूप में सहेजें`Save` की विधि`Document` पथ निर्दिष्ट करने और विकल्प सहेजने वाला वर्ग.

#### प्रश्न: छवि नमूनाकरण के साथ पीडीएफ दस्तावेज़ का आकार कम करने के क्या लाभ हैं?
उत्तर: छवि नमूनाकरण के साथ पीडीएफ दस्तावेज़ का आकार कम करने के लाभ हैं:

पीडीएफ फाइल का आकार कम होना: इमेज सैंपलिंग से पीडीएफ डॉक्यूमेंट में इमेज का रिज़ॉल्यूशन कम हो जाता है, जिससे पीडीएफ फाइल का आकार काफी कम हो जाता है। इससे फाइल को शेयर करना और ट्रांसफर करना आसान हो जाता है, खासकर ईमेल या ऑनलाइन के ज़रिए।

भंडारण स्थान का अनुकूलन: पीडीएफ फाइल का आकार कम करने से भंडारण स्थान के उपयोग को अनुकूलित करने में मदद मिलती है, खासकर तब जब आपके पास उच्च रिज़ॉल्यूशन वाली छवियों वाली कई पीडीएफ फाइलें हों।

प्रदर्शन में सुधार: छोटी पीडीएफ फाइलें तेजी से लोड होती हैं और उन्हें विभिन्न डिवाइसों पर तेजी से खोला और देखा जा सकता है।
