यह आलेख .NET के लिए Aspose.Words के साथ मेटाफ़ाइल आकार सुविधा के लिए सेट आउटलाइन विकल्पों का उपयोग करने के तरीके पर चरण-दर-चरण मार्गदर्शिका प्रदान करता है। हम कोड के प्रत्येक भाग को विस्तार से समझाएंगे। इस ट्यूटोरियल के अंत में, आप यह समझ पाएंगे कि किसी दस्तावेज़ में रूपरेखा विकल्प कैसे सेट करें और संबंधित रूपरेखा विकल्पों के साथ एक पीडीएफ कैसे तैयार करें।

शुरू करने से पहले, सुनिश्चित करें कि आपने अपने प्रोजेक्ट में .NET लाइब्रेरी के लिए Aspose.Words को स्थापित और कॉन्फ़िगर किया है। आप Aspose वेबसाइट पर लाइब्रेरी और इंस्टॉलेशन निर्देश पा सकते हैं।

## चरण 1: दस्तावेज़ निर्देशिका को परिभाषित करें

 आरंभ करने के लिए, आपको उस निर्देशिका का पथ परिभाषित करना होगा जहां आपके दस्तावेज़ स्थित हैं। प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: दस्तावेज़ अपलोड करें

इसके बाद, हमें वह दस्तावेज़ लोड करना होगा जिसे हम संसाधित करना चाहते हैं। इस उदाहरण में, हम मानते हैं कि दस्तावेज़ को "रेंडरिंग.docx" कहा जाता है और वह निर्दिष्ट दस्तावेज़ निर्देशिका में स्थित है।

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## चरण 3: योजना विकल्पों के साथ पीडीएफ विकल्पों के रूप में सहेजें को कॉन्फ़िगर करें

 उत्पन्न पीडीएफ में रूपरेखा विकल्प सेट करने के लिए, हमें कॉन्फ़िगर करने की आवश्यकता है`PdfSaveOptions` वस्तु। हम शीर्षक रूपरेखा स्तरों की संख्या निर्धारित कर सकते हैं (`HeadingsOutlineLevels`) और विस्तारित रूपरेखा स्तरों की संख्या (`ExpandedOutlineLevels`).

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.OutlineOptions.HeadingsOutlineLevels = 3;
saveOptions.OutlineOptions.ExpandedOutlineLevels = 1;
```

## चरण 4: रूपरेखा विकल्पों के साथ दस्तावेज़ को पीडीएफ के रूप में सहेजें

अंत में, हम पहले से कॉन्फ़िगर किए गए सेव विकल्पों का उपयोग करके दस्तावेज़ को पीडीएफ प्रारूप में सहेज सकते हैं।

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.SetOutlineOptions.pdf", saveOptions);
```

बस इतना ही ! आपने दस्तावेज़ में रूपरेखा विकल्प सफलतापूर्वक सेट कर दिए हैं और .NET के लिए Aspose.Words का उपयोग करके संबंधित रूपरेखा विकल्पों के साथ एक पीडीएफ तैयार कर लिया है।

### .NET के लिए Aspose.Words के साथ योजना विकल्पों को मेटाफ़ाइल आकार में सेट करने के लिए उदाहरण स्रोत कोड


```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Rendering.docx");

	PdfSaveOptions saveOptions = new PdfSaveOptions();
	saveOptions.OutlineOptions.HeadingsOutlineLevels = 3;
	saveOptions.OutlineOptions.ExpandedOutlineLevels = 1;

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.SetOutlineOptions.pdf", saveOptions);
   
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने समझाया कि .NET के लिए Aspose.Words का उपयोग करके पीडीएफ दस्तावेज़ में रूपरेखा विकल्प कैसे सेट करें। वर्णित चरणों का उपयोग करके, आप आसानी से अपने दस्तावेज़ में शीर्षक और रूपरेखा स्तर निर्दिष्ट कर सकते हैं और संबंधित रूपरेखा विकल्पों के साथ एक पीडीएफ फाइल तैयार कर सकते हैं। .NET के लिए Aspose.Words का उपयोग करके अपने पीडीएफ दस्तावेजों में संरचना और नेविगेशन को बेहतर बनाने के लिए रूपरेखा विकल्प का लाभ उठाएं।

### अक्सर पूछे जाने वाले प्रश्नों

#### प्रश्न: पीडीएफ दस्तावेज़ में रूपरेखा विकल्प क्या है?
उ: पीडीएफ दस्तावेज़ में रूपरेखा विकल्प दस्तावेज़ सामग्री की पदानुक्रमित संरचना को संदर्भित करता है। यह आपको सामग्री की एक इंटरैक्टिव तालिका बनाने की अनुमति देता है और दस्तावेज़ में नेविगेशन की सुविधा प्रदान करता है। रूपरेखा विकल्प रूपरेखा में शामिल करने के लिए शीर्षक और उपशीर्षक स्तर और उत्पन्न रूपरेखा में प्रदर्शित करने के लिए विवरण का स्तर निर्धारित करते हैं।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके पीडीएफ दस्तावेज़ में रूपरेखा विकल्प कैसे सेट कर सकता हूं?
उ: .NET के लिए Aspose.Words का उपयोग करके PDF दस्तावेज़ में रूपरेखा विकल्प सेट करने के लिए, इन चरणों का पालन करें:

 प्रतिस्थापित करके वह निर्देशिका पथ सेट करें जहां आपके दस्तावेज़ स्थित हैं`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ।

 उस दस्तावेज़ को लोड करें जिसे आप पीडीएफ में कनवर्ट करना चाहते हैं`Document` वर्ग और निर्दिष्ट दस्तावेज़ निर्देशिका में दस्तावेज़ का पथ निर्दिष्ट करें।

 का एक उदाहरण बनाकर पीडीएफ विकल्प के रूप में सहेजें को कॉन्फ़िगर करें`PdfSaveOptions` कक्षा और का उपयोग करना`OutlineOptions` रूपरेखा विकल्प सेट करने के लिए संपत्ति। आप इसका उपयोग करके रूपरेखा में शामिल करने के लिए शीर्षक स्तरों की संख्या निर्दिष्ट कर सकते हैं`HeadingsOutlineLevels` संपत्ति और विस्तारित रूपरेखा स्तरों की संख्या का उपयोग करना`ExpandedOutlineLevels` संपत्ति।

 का उपयोग करके दस्तावेज़ को पीडीएफ प्रारूप में सहेजें`Save` की विधि`Document` वर्ग पथ और बचत विकल्पों को निर्दिष्ट करता है।

#### प्रश्न: पीडीएफ दस्तावेज़ में योजना का विकल्प क्या है?
उ: पीडीएफ दस्तावेज़ में रूपरेखा विकल्प आपको सामग्री की एक पदानुक्रमित संरचना बनाने की अनुमति देता है, जिससे दस्तावेज़ को नेविगेट करना और विभिन्न अनुभागों तक पहुंच आसान हो जाती है। यह उपयोगकर्ताओं को सामग्री या रूपरेखा की तालिका में प्रविष्टियों पर क्लिक करके दस्तावेज़ के विशिष्ट भागों पर तुरंत जाने की अनुमति देता है। रूपरेखा विकल्प समग्र दस्तावेज़ संरचना का अवलोकन प्रदान करके पढ़ने के अनुभव को भी बढ़ाता है।
