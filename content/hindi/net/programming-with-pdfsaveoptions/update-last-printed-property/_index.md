यह आलेख .NET के लिए Aspose.Words के साथ पीडीएफ दस्तावेज़ अद्यतन सुविधा में "अंतिम मुद्रण" संपत्ति का उपयोग करने के तरीके पर चरण-दर-चरण मार्गदर्शिका प्रदान करता है। हम कोड के प्रत्येक भाग को विस्तार से समझाएंगे। इस ट्यूटोरियल के अंत में, आप यह समझ पाएंगे कि पीडीएफ में कनवर्ट करते समय "अंतिम मुद्रित" संपत्ति को अपडेट करने के विकल्प को कैसे कॉन्फ़िगर किया जाए।

शुरू करने से पहले, सुनिश्चित करें कि आपने अपने प्रोजेक्ट में .NET लाइब्रेरी के लिए Aspose.Words को स्थापित और कॉन्फ़िगर किया है। आप Aspose वेबसाइट पर लाइब्रेरी और इंस्टॉलेशन निर्देश पा सकते हैं।

## चरण 1: दस्तावेज़ निर्देशिका को परिभाषित करें

 आरंभ करने के लिए, आपको उस निर्देशिका का पथ परिभाषित करना होगा जहां आपके दस्तावेज़ स्थित हैं। प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: दस्तावेज़ अपलोड करें

इसके बाद, हमें वह दस्तावेज़ लोड करना होगा जिसे हम संसाधित करना चाहते हैं। इस उदाहरण में, हम मानते हैं कि दस्तावेज़ को "रेंडरिंग.docx" कहा जाता है और वह निर्दिष्ट दस्तावेज़ निर्देशिका में स्थित है।

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## चरण 3: अद्यतन "अंतिम मुद्रित" संपत्ति के साथ पीडीएफ विकल्प के रूप में सहेजें कॉन्फ़िगर करें

 पीडीएफ में कनवर्ट करते समय "अंतिम मुद्रित" संपत्ति को अपडेट करने में सक्षम करने के लिए, हमें कॉन्फ़िगर करने की आवश्यकता है`PdfSaveOptions` ऑब्जेक्ट करें और सेट करें`UpdateLastPrintedProperty`संपत्ति को`true`.

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions { UpdateLastPrintedProperty = true };
```

## चरण 4: दस्तावेज़ को "अंतिम मुद्रित" संपत्ति के अद्यतन के साथ पीडीएफ के रूप में सहेजें

अंत में, हम पहले से कॉन्फ़िगर किए गए सेव विकल्पों का उपयोग करके दस्तावेज़ को पीडीएफ प्रारूप में सहेज सकते हैं।

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.UpdateIfLastPrinted.pdf", saveOptions);
```

बस इतना ही ! आपने .NET के लिए Aspose.Words का उपयोग करके किसी दस्तावेज़ को PDF में कनवर्ट करते समय "अंतिम मुद्रित" प्रॉपर्टी को अपडेट करना सफलतापूर्वक सक्षम कर दिया है।

### .NET के लिए Aspose.Words के साथ "अंतिम मुद्रित" प्रॉपर्टी को अपडेट करने के लिए उदाहरण स्रोत कोड


```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Rendering.docx");

	PdfSaveOptions saveOptions = new PdfSaveOptions { UpdateLastPrintedProperty = true };

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.UpdateIfLastPrinted.pdf", saveOptions);

```
## निष्कर्ष

इस ट्यूटोरियल में, हमने बताया कि .NET के लिए Aspose.Words का उपयोग करके एक पीडीएफ दस्तावेज़ में "अंतिम मुद्रित" संपत्ति को कैसे अपडेट किया जाए। दिए गए चरणों का पालन करके, आप दस्तावेज़ को पीडीएफ में परिवर्तित करते समय "अंतिम मुद्रित" संपत्ति को अपडेट करने के विकल्प को आसानी से कॉन्फ़िगर कर सकते हैं। दस्तावेज़ के उपयोग और संबंधित जानकारी पर नज़र रखने के लिए इस सुविधा का उपयोग करें।

### अक्सर पूछे जाने वाले प्रश्नों

#### प्रश्न: पीडीएफ दस्तावेज़ में "अंतिम मुद्रित" संपत्ति क्या है?
उ: पीडीएफ दस्तावेज़ में "अंतिम मुद्रित" संपत्ति उस दिनांक और समय को संदर्भित करती है जब दस्तावेज़ अंतिम बार मुद्रित किया गया था। यह संपत्ति दस्तावेज़ के उपयोग और प्रबंधन के बारे में जानकारी ट्रैक करने के लिए उपयोगी हो सकती है।

#### प्रश्न: मैं .NET के लिए Aspose.Words के साथ PDF दस्तावेज़ में "अंतिम मुद्रित" संपत्ति को कैसे अपडेट कर सकता हूं?
उ: .NET के लिए Aspose.Words के साथ PDF दस्तावेज़ में "अंतिम मुद्रित" संपत्ति को अपडेट करने के लिए, इन चरणों का पालन करें:

 का एक उदाहरण बनाएं`Document` Word दस्तावेज़ के लिए पथ निर्दिष्ट करने वाला वर्ग।

 का एक उदाहरण बनाएं`PdfSaveOptions` कक्षा और सेट करें`UpdateLastPrintedProperty`संपत्ति को`true` "अंतिम मुद्रित" संपत्ति को अद्यतन करने में सक्षम करने के लिए।

 उपयोग`Save` की विधि`Document`सेव विकल्पों को निर्दिष्ट करके दस्तावेज़ को पीडीएफ प्रारूप में सहेजने के लिए क्लास।

#### प्रश्न: मैं कैसे जांच सकता हूं कि जेनरेट किए गए पीडीएफ दस्तावेज़ में "अंतिम मुद्रित" संपत्ति अपडेट की गई है या नहीं?
उ: आप एडोब एक्रोबैट रीडर जैसे संगत पीडीएफ व्यूअर के साथ पीडीएफ फाइल खोलकर और दस्तावेज़ की जानकारी देखकर जांच कर सकते हैं कि जेनरेट किए गए पीडीएफ दस्तावेज़ में "अंतिम मुद्रित" संपत्ति अपडेट की गई है या नहीं। अंतिम मुद्रण की तारीख और समय पीडीएफ दस्तावेज़ के निर्माण की तारीख और समय के अनुरूप होना चाहिए।
