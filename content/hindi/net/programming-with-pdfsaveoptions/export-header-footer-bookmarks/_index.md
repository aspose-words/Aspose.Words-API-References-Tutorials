यह लेख Aspose.Words for .NET के साथ वर्ड डॉक्यूमेंट हेडर फ़ुटर बुकमार्क को पीडीएफ डॉक्यूमेंट फीचर में एक्सपोर्ट करने के बारे में चरण-दर-चरण गाइड प्रदान करता है। हम कोड के प्रत्येक भाग को विस्तार से समझाएंगे। इस ट्यूटोरियल के अंत में, आप समझ पाएंगे कि किसी डॉक्यूमेंट के हेडर और फ़ुटर से बुकमार्क कैसे एक्सपोर्ट करें और उचित बुकमार्क के साथ एक पीडीएफ जेनरेट करें।

शुरू करने से पहले, सुनिश्चित करें कि आपने अपने प्रोजेक्ट में Aspose.Words for .NET लाइब्रेरी को इंस्टॉल और कॉन्फ़िगर किया है। आप Aspose वेबसाइट पर लाइब्रेरी और इंस्टॉलेशन निर्देश पा सकते हैं।

## चरण 1: दस्तावेज़ निर्देशिका निर्धारित करें

 शुरू करने के लिए, आपको उस निर्देशिका का पथ परिभाषित करना होगा जहाँ आपके दस्तावेज़ स्थित हैं।`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: दस्तावेज़ अपलोड करें

इसके बाद, हमें उस दस्तावेज़ को लोड करना होगा जिसे हम प्रोसेस करना चाहते हैं। इस उदाहरण में, हम मानते हैं कि दस्तावेज़ का नाम "बुकमार्क इन हेडर और फ़ुटर.docx" है और यह निर्दिष्ट दस्तावेज़ निर्देशिका में स्थित है।

```csharp
Document doc = new Document(dataDir + "Bookmarks in headers and footers.docx");
```

## चरण 3: PDF के रूप में सहेजें विकल्प कॉन्फ़िगर करें

 हेडर और फ़ुटर बुकमार्क्स को निर्यात करने के लिए, हमें कॉन्फ़िगर करने की आवश्यकता है`PdfSaveOptions` इस उदाहरण में, हमने डिफ़ॉल्ट बुकमार्क आउटलाइन स्तर को 1 पर और हेडर और फ़ुटर बुकमार्क निर्यात मोड को "प्रथम" पर सेट किया है।

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.OutlineOptions.DefaultBookmarksOutlineLevel = 1;
saveOptions.HeaderFooterBookmarksExportMode = HeaderFooterBookmarksExportMode.First;
```

## चरण 4: दस्तावेज़ को हेडर और फ़ुटर बुकमार्क के साथ PDF के रूप में सहेजें

अंत में, हम पहले से कॉन्फ़िगर किए गए सेव विकल्पों का उपयोग करके दस्तावेज़ को पीडीएफ प्रारूप में सेव कर सकते हैं।

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.ExportHeaderFooterBookmarks.pdf", saveOptions);
```

बस इतना ही! आपने एक दस्तावेज़ से हेडर और फ़ुटर बुकमार्क्स को सफलतापूर्वक निर्यात किया है और .NET के लिए Aspose.Words का उपयोग करके उपयुक्त बुकमार्क्स के साथ एक पीडीएफ तैयार किया है।

### .NET के लिए Aspose.Words के साथ हेडर और फ़ुटर बुकमार्क्स निर्यात करने के लिए नमूना स्रोत कोड

```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Bookmarks in headers and footers.docx");

	PdfSaveOptions saveOptions = new PdfSaveOptions();
	saveOptions.OutlineOptions.DefaultBookmarksOutlineLevel = 1;
	saveOptions.HeaderFooterBookmarksExportMode = HeaderFooterBookmarksExportMode.First;

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.ExportHeaderFooterBookmarks.pdf", saveOptions);

```

## निष्कर्ष

इस ट्यूटोरियल में, हमने बताया कि Aspose.Words for .NET का उपयोग करके Word दस्तावेज़ से हेडर और फ़ुटर बुकमार्क को PDF दस्तावेज़ में कैसे निर्यात किया जाए। निर्यात किए गए बुकमार्क जेनरेट किए गए PDF दस्तावेज़ में संबंधित हेडर और फ़ुटर के लिए आसान नेविगेशन और त्वरित संदर्भ की अनुमति देते हैं। किसी दस्तावेज़ से हेडर और फ़ुटर बुकमार्क निर्यात करने और Aspose.Words for .NET का उपयोग करके उपयुक्त बुकमार्क के साथ PDF जेनरेट करने के लिए वर्णित चरणों का पालन करें। अपने दस्तावेज़ों के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें और आवश्यकतानुसार सहेजने के विकल्प कॉन्फ़िगर करें।

### अक्सर पूछे जाने वाले प्रश्नों

### प्रश्न: हेडर और फुटर बुकमार्क्स को वर्ड दस्तावेज़ से पीडीएफ दस्तावेज़ में निर्यात करना क्या है?
उत्तर: वर्ड डॉक्यूमेंट से पीडीएफ डॉक्यूमेंट में हेडर और फूटर बुकमार्क एक्सपोर्ट करना एक ऐसी सुविधा है, जो मूल वर्ड डॉक्यूमेंट के हेडर और फूटर से पीडीएफ डॉक्यूमेंट में बुकमार्क रखने और बनाने की सुविधा देती है। यह उपयोगकर्ताओं को हेडर और फूटर से संबंधित बुकमार्क का उपयोग करके पीडीएफ डॉक्यूमेंट में तेज़ी से और आसानी से नेविगेट करने की अनुमति देता है।

### प्रश्न: मैं किसी Word दस्तावेज़ से हेडर और फ़ुटर बुकमार्क्स को PDF दस्तावेज़ में निर्यात करने के लिए Aspose.Words for .NET का उपयोग कैसे कर सकता हूँ?
उत्तर: Aspose.Words for .NET का उपयोग करके किसी Word दस्तावेज़ से शीर्षलेख और पादलेख बुकमार्क को PDF दस्तावेज़ में निर्यात करने के लिए, इन चरणों का पालन करें:

 वह निर्देशिका पथ सेट करें जहाँ आपके दस्तावेज़ स्थित हैं`"YOUR DOCUMENT DIRECTORY"` अपने दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

 उस दस्तावेज़ को लोड करें जिसे आप संसाधित करना चाहते हैं`Document` क्लास पर जाएँ और निर्दिष्ट दस्तावेज़ निर्देशिका में Word दस्तावेज़ का पथ निर्दिष्ट करें।

 का एक उदाहरण बनाकर PDF के रूप में सहेजें विकल्प कॉन्फ़िगर करें`PdfSaveOptions` क्लास में जाकर उचित हेडर और फूटर बुकमार्क विकल्प सेट करना।

 दस्तावेज़ को PDF प्रारूप में सहेजें`Save` की विधि`Document` पथ निर्दिष्ट करने और विकल्प सहेजने वाला वर्ग.

### प्रश्न: हेडर और फुटर बुकमार्क्स को पीडीएफ दस्तावेज़ में निर्यात करने के क्या लाभ हैं?
उत्तर: हेडर और फुटर बुकमार्क्स को पीडीएफ दस्तावेज़ में निर्यात करने के लाभ ये हैं:

आसान नेविगेशन: बुकमार्क उपयोगकर्ताओं को विशिष्ट हेडर और फ़ुटर का संदर्भ देकर आसानी से पीडीएफ दस्तावेज़ को नेविगेट करने की अनुमति देता है।

त्वरित संदर्भ: बुकमार्क उपयोगकर्ताओं को हेडर और फ़ुटर के आधार पर पीडीएफ दस्तावेज़ के प्रासंगिक अनुभागों को शीघ्रता से खोजने की अनुमति देता है।
