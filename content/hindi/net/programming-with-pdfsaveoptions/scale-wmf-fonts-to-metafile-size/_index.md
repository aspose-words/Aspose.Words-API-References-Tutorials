यह लेख .NET के लिए Aspose.Words के साथ मेटाफ़ाइल आकार सुविधा के लिए स्केल wmf फ़ॉन्ट के साथ पीडीएफ आकार को कम करने के बारे में चरण-दर-चरण मार्गदर्शिका प्रदान करता है। हम कोड के प्रत्येक भाग को विस्तार से समझाएंगे। इस ट्यूटोरियल के अंत में, आप यह समझ पाएंगे कि पीडीएफ में कनवर्ट करते समय WMF फ़ॉन्ट स्केलिंग को कैसे सक्षम या अक्षम किया जाए।

शुरू करने से पहले, सुनिश्चित करें कि आपने अपने प्रोजेक्ट में Aspose.Words for .NET लाइब्रेरी को इंस्टॉल और कॉन्फ़िगर किया है। आप Aspose वेबसाइट पर लाइब्रेरी और इंस्टॉलेशन निर्देश पा सकते हैं।

## चरण 1: दस्तावेज़ निर्देशिका निर्धारित करें

 शुरू करने के लिए, आपको उस निर्देशिका का पथ परिभाषित करना होगा जहाँ आपके दस्तावेज़ स्थित हैं।`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: दस्तावेज़ अपलोड करें

इसके बाद, हमें उस दस्तावेज़ को लोड करना होगा जिसे हम प्रोसेस करना चाहते हैं। इस उदाहरण में, हम मानते हैं कि दस्तावेज़ को "WMF with text.docx" कहा जाता है और यह निर्दिष्ट दस्तावेज़ निर्देशिका में स्थित है।

```csharp
Document doc = new Document(dataDir + "WMF with text.docx");
```

## चरण 3: मेटाफ़ाइल रेंडरिंग विकल्प कॉन्फ़िगर करें

 मेटाफ़ाइल आकार में WMF फ़ॉन्ट स्केलिंग को सक्षम या अक्षम करने के लिए, हमें कॉन्फ़िगर करने की आवश्यकता है`MetafileRenderingOptions` ऑब्जेक्ट. इस उदाहरण में, हम फ़ॉन्ट स्केलिंग को सेट करके अक्षम करते हैं`ScaleWmfFontsToMetafileSize`संपत्ति को`false`.

```csharp
MetafileRenderingOptions metafileRenderingOptions = new MetafileRenderingOptions
{
     ScaleWmfFontsToMetafileSize=false
};
```

## चरण 4: मेटाफ़ाइल रेंडरिंग विकल्पों के साथ PDF के रूप में सहेजें विकल्प कॉन्फ़िगर करें

अंत में, हम पहले से कॉन्फ़िगर किए गए मेटाफ़ाइल रेंडरिंग विकल्पों का उपयोग करके सेव-टू-पीडीएफ विकल्पों को कॉन्फ़िगर कर सकते हैं।

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions { MetafileRenderingOptions = metafileRenderingOptions };
```

## चरण 5: मेटाफ़ाइल रेंडरिंग विकल्पों के साथ दस्तावेज़ को PDF के रूप में सहेजें

पहले से कॉन्फ़िगर किए गए सहेजें विकल्पों का उपयोग करके दस्तावेज़ को PDF प्रारूप में सहेजें।

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.ScaleWmfFontsToMetafileSize.pdf", saveOptions);
```

बस इतना ही! आपने रूपांतरण करते समय मेटाफ़ाइल आकार में WMF फ़ॉन्ट स्केलिंग को सफलतापूर्वक सक्षम या अक्षम कर दिया है

.NET के लिए Aspose.Words का उपयोग कर एक पीडीएफ दस्तावेज़.

### .NET के लिए Aspose.Words के साथ WMF फ़ॉन्ट को मेटाफ़ाइल आकार में स्केल करने के लिए उदाहरण स्रोत कोड

```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "WMF with text.docx");

	MetafileRenderingOptions metafileRenderingOptions = new MetafileRenderingOptions
	{
		ScaleWmfFontsToMetafileSize = false
	};

	//यदि Aspose.Words कुछ मेटाफ़ाइल रिकॉर्ड को वेक्टर ग्राफ़िक्स में सही ढंग से प्रस्तुत नहीं कर सकता है
	// फिर Aspose.Words इस मेटाफ़ाइल को बिटमैप में प्रस्तुत करता है।
	PdfSaveOptions saveOptions = new PdfSaveOptions { MetafileRenderingOptions = metafileRenderingOptions };

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.ScaleWmfFontsToMetafileSize.pdf", saveOptions);
	
        
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने बताया कि .NET के लिए Aspose.Words का उपयोग करके PDF दस्तावेज़ में WMF फ़ॉन्ट का आकार मेटाफ़ाइल आकार में बदलने को कैसे सक्षम या अक्षम किया जाए। वर्णित चरणों का पालन करके, आप आसानी से नियंत्रित कर सकते हैं कि PDF दस्तावेज़ में कनवर्ट करते समय WMF फ़ॉन्ट का आकार मेटाफ़ाइल आकार से मेल खाने के लिए बदला जाना चाहिए या नहीं। यह आपको जेनरेट की गई PDF फ़ाइल का आकार कम करने और रेंडरिंग प्रदर्शन को बेहतर बनाने में मदद कर सकता है। अपने दस्तावेज़ों के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें और आवश्यकतानुसार मेटाफ़ाइल रेंडरिंग विकल्पों को कॉन्फ़िगर करें।

### अक्सर पूछे जाने वाले प्रश्नों

#### प्रश्न: PDF दस्तावेज़ में WMF फ़ॉन्ट को मेटाफ़ाइल आकार में बदलने का क्या अर्थ है?
उत्तर: PDF दस्तावेज़ में WMF फ़ॉन्ट को मेटाफ़ाइल आकार में बदलना एक ऐसी सुविधा है जो नियंत्रित करती है कि PDF दस्तावेज़ में कनवर्ट करते समय WMF फ़ॉन्ट को मेटाफ़ाइल आकार से मेल खाने के लिए स्केल किया जाना चाहिए या नहीं। जब यह सुविधा सक्षम होती है, तो WMF फ़ॉन्ट को मेटाफ़ाइल के आकार से मेल खाने के लिए स्केल किया जाता है, जिससे जेनरेट किए गए PDF दस्तावेज़ का आकार कम हो सकता है।

#### प्रश्न: मैं PDF दस्तावेज़ में WMF फ़ॉन्ट को मेटाफ़ाइल आकार में बदलने को सक्षम या अक्षम करने के लिए .NET के लिए Aspose.Words का उपयोग कैसे कर सकता हूं?
उत्तर: .NET के लिए Aspose.Words का उपयोग करके PDF दस्तावेज़ में WMF फ़ॉन्ट का आकार बदलकर मेटाफ़ाइल आकार में बदलने को सक्षम या अक्षम करने के लिए, इन चरणों का पालन करें:

 वह निर्देशिका पथ सेट करें जहाँ आपके दस्तावेज़ स्थित हैं`"YOUR DOCUMENT DIRECTORY"` अपने दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

 उस दस्तावेज़ को लोड करें जिसे आप संसाधित करना चाहते हैं`Document` क्लास पर जाएँ और निर्दिष्ट दस्तावेज़ निर्देशिका में Word दस्तावेज़ का पथ निर्दिष्ट करें।

 मेटाफ़ाइल रेंडरिंग विकल्पों को कॉन्फ़िगर करने के लिए इसका एक उदाहरण बनाएँ`MetafileRenderingOptions` वर्ग और सेटिंग`ScaleWmfFontsToMetafileSize`संपत्ति को`true` WMF फ़ॉन्ट को मेटाफ़ाइल आकार में स्केल करने में सक्षम करने के लिए, या`false` इस सुविधा को अक्षम करने के लिए.

 का एक उदाहरण बनाकर पीडीएफ के रूप में सहेजें विकल्प को कॉन्फ़िगर करें`PdfSaveOptions` क्लास और पहले से कॉन्फ़िगर किए गए मेटाफ़ाइल रेंडरिंग विकल्पों का उपयोग करना।

 दस्तावेज़ को PDF प्रारूप में सहेजें`Save` की विधि`Document` पथ निर्दिष्ट करने और विकल्प सहेजने वाला वर्ग.

#### प्रश्न: PDF दस्तावेज़ में WMF फ़ॉन्ट को मेटाफ़ाइल आकार में बदलने के क्या लाभ हैं?
उत्तर: PDF दस्तावेज़ में WMF फ़ॉन्ट को मेटाफ़ाइल आकार में बदलने के लाभ ये हैं:

पीडीएफ फाइल आकार में कमी: WMF फ़ॉन्ट को मेटाफ़ाइल आकार में बदलने से, फ़ॉन्ट आकार को मेटाफ़ाइल आवश्यकताओं के अनुसार अनुकूलित करके उत्पन्न पीडीएफ दस्तावेज़ के आकार को कम किया जा सकता है।

बेहतर प्रदर्शन: WMF फ़ॉन्ट के आकार को मेटाफ़ाइल के आयामों के अनुसार समायोजित करके, PDF दस्तावेज़ का रेंडरिंग तेज़ और अधिक कुशल हो सकता है।
