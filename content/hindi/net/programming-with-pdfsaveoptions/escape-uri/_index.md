यह लेख .NET के लिए Aspose.Words के साथ PDF दस्तावेज़ में URI को एस्केप करने के तरीके पर चरण-दर-चरण मार्गदर्शिका प्रदान करता है। हम कोड के प्रत्येक भाग को विस्तार से समझाएँगे। इस ट्यूटोरियल के अंत में, आप समझ पाएँगे कि दस्तावेज़ में एस्केप किए गए Uri के साथ हाइपरलिंक कैसे डालें।

शुरू करने से पहले, सुनिश्चित करें कि आपने अपने प्रोजेक्ट में Aspose.Words for .NET लाइब्रेरी को इंस्टॉल और कॉन्फ़िगर किया है। आप Aspose वेबसाइट पर लाइब्रेरी और इंस्टॉलेशन निर्देश पा सकते हैं।

## चरण 1: दस्तावेज़ निर्देशिका निर्धारित करें

 शुरू करने के लिए, आपको उस निर्देशिका का पथ परिभाषित करना होगा जहाँ आपके दस्तावेज़ स्थित हैं।`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: एक दस्तावेज़ और एक दस्तावेज़ बिल्डर बनाएँ

 इसके बाद, हमें एक नया बनाना होगा`Document` वस्तु और एक`DocumentBuilder` दस्तावेज़ बनाने के लिए ऑब्जेक्ट.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## चरण 3: एस्केप्ड यूआरआई के साथ हाइपरलिंक डालें

 उपयोग`InsertHyperlink` की विधि`DocumentBuilder` दस्तावेज़ में हाइपरलिंक डालने के लिए ऑब्जेक्ट। Uri को एस्केप करने के लिए इसका उपयोग करना होगा`Uri.EscapeUriString` प्रारूप त्रुटियों से बचने के लिए फ़ंक्शन.

```csharp
builder.InsertHyperlink("Testlink",
     Uri.EscapeUriString("https://www.google.com/search?q=%2Fthe%20test"), गलत);
builder. Writen();
builder.InsertHyperlink(Uri.EscapeUriString("https://www.google.com/search?q=%2Fthe%20test"),
     Uri.EscapeUriString("https://www.google.com/search?q=%2Fthe%20test"), गलत);
```

## चरण 4: दस्तावेज़ को PDF के रूप में सहेजें

 अंत में, हम दस्तावेज़ को PDF के रूप में सहेज सकते हैं`Save` की विधि`Document` आउटपुट फ़ाइल नाम निर्दिष्ट करें.

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.EscapeUri.pdf");
```

बस इतना ही! आपने Aspose.Words for .NET का उपयोग करके दस्तावेज़ में एस्केप्ड Uri के साथ हाइपरलिंक सफलतापूर्वक सम्मिलित कर लिया है।

### .NET के लिए Aspose.Words के साथ Uri एस्केपिंग के लिए नमूना स्रोत कोड


```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	
	builder.InsertHyperlink("Testlink", 
		"https://www.google.com/search?q=%2Fthe%20test", गलत);
	builder.Writeln();
	builder.InsertHyperlink("https:// www.google.com/search?q=%2Fthe%20test",
		"https://www.google.com/search?q=%2Fthe%20test", गलत);

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.EscapeUri.pdf");    
  
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने .NET के लिए Aspose.Words का उपयोग करके PDF दस्तावेज़ में URI को एस्केप करने का तरीका बताया है। URI को एस्केप करके, आप फ़ॉर्मेटिंग त्रुटियों से बच सकते हैं और यह सुनिश्चित कर सकते हैं कि हाइपरलिंक्स को PDF दस्तावेज़ में सही ढंग से व्याख्यायित और प्रदर्शित किया गया है। अपने PDF दस्तावेज़ में एस्केप किए गए URI के साथ हाइपरलिंक्स डालने के लिए बताए गए चरणों का पालन करें। एस्केप करना सुनिश्चित करें।

### अक्सर पूछे जाने वाले प्रश्नों

#### प्रश्न: पीडीएफ दस्तावेज़ में एस्केप यूआरआई क्या है और यह महत्वपूर्ण क्यों है?
उत्तर: PDF दस्तावेज़ में एस्केप URI का तात्पर्य URL में विशेष वर्णों को एस्केप अनुक्रमों में परिवर्तित करने की विधि से है, ताकि फ़ॉर्मेट त्रुटियों से बचा जा सके। यह महत्वपूर्ण है क्योंकि URL में विशेष वर्ण URL की संरचना को बाधित कर सकते हैं और गलत व्याख्या या गलत रेंडरिंग का कारण बन सकते हैं। विशेष वर्णों को एस्केप करके, हम गारंटी देते हैं कि URL की सही व्याख्या की गई है और PDF दस्तावेज़ में प्रदर्शित किया गया है।

#### प्रश्न: मैं PDF दस्तावेज़ में URIs से बचने के लिए .NET के लिए Aspose.Words का उपयोग कैसे कर सकता हूं?
उत्तर: .NET के लिए Aspose.Words का उपयोग करके PDF दस्तावेज़ में URI से बचने के लिए, इन चरणों का पालन करें:

 वह निर्देशिका पथ सेट करें जहाँ आपके दस्तावेज़ स्थित हैं`"YOUR DOCUMENT DIRECTORY"` अपने दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

 एक नया उदाहरण बनाएँ`Document` कक्षा और एक`DocumentBuilder` दस्तावेज़ बनाने के लिए ऑब्जेक्ट.

 उपयोग`InsertHyperlink` की विधि`DocumentBuilder` दस्तावेज़ में हाइपरलिंक डालने के लिए ऑब्जेक्ट का उपयोग करें। URI से बचने के लिए सुनिश्चित करें`Uri.EscapeUriString` प्रारूप त्रुटियों से बचने के लिए फ़ंक्शन.

 उपयोग`Save` की विधि`Document` आउटपुट फ़ाइल का नाम निर्दिष्ट करके दस्तावेज़ को पीडीएफ प्रारूप में सहेजने के लिए ऑब्जेक्ट का उपयोग करें।

#### प्रश्न: PDF दस्तावेज़ में URIs को एस्केप करने के क्या लाभ हैं?
उत्तर: PDF दस्तावेज़ में URI एस्केपिंग के लाभ ये हैं:

प्रारूप त्रुटियों की रोकथाम: URI एस्केपिंग, URL में विशेष वर्णों के कारण होने वाली प्रारूप त्रुटियों को रोकने में मदद करता है, तथा यह सुनिश्चित करता है कि URL की सही व्याख्या की गई है और उसे PDF दस्तावेज़ में प्रदर्शित किया गया है।

पीडीएफ रीडर्स के साथ संगतता: एस्केप्ड यूआरआई आमतौर पर पीडीएफ रीडर्स द्वारा अच्छी तरह से समर्थित होते हैं, जिससे बेहतर संगतता और एक सुसंगत उपयोगकर्ता अनुभव सुनिश्चित होता है।

#### प्रश्न: URI में कौन से विशेष वर्णों को एस्केप किया जाना चाहिए?
 उत्तर: URI में जिन विशेष वर्णों को एस्केप किया जाना चाहिए वे हैं: स्पेस, <, >, ", #, %, {, },|, \, ^, ~, [, ], `, ;, /, ?, :, @, =, &, $, +, ,, [, ], and !.
