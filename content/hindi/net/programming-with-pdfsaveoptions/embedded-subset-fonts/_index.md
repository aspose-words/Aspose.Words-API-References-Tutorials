यह आलेख .NET के लिए Aspose.Words के साथ फ़ॉन्ट सबसेट एम्बेडिंग सुविधा का उपयोग करने के तरीके पर चरण-दर-चरण मार्गदर्शिका प्रदान करता है। हम कोड के प्रत्येक भाग को विस्तार से समझाएंगे। इस ट्यूटोरियल के अंत में, आप यह समझ पाएंगे कि किसी दस्तावेज़ में फ़ॉन्ट के सबसेट को कैसे एम्बेड किया जाए और दस्तावेज़ में उपयोग किए गए केवल ग्लिफ़ युक्त एक पीडीएफ कैसे तैयार किया जाए।

शुरू करने से पहले, सुनिश्चित करें कि आपने अपने प्रोजेक्ट में .NET लाइब्रेरी के लिए Aspose.Words को स्थापित और कॉन्फ़िगर किया है। आप Aspose वेबसाइट पर लाइब्रेरी और इंस्टॉलेशन निर्देश पा सकते हैं।

## चरण 1: दस्तावेज़ निर्देशिका को परिभाषित करें

 आरंभ करने के लिए, आपको उस निर्देशिका का पथ परिभाषित करना होगा जहां आपके दस्तावेज़ स्थित हैं। प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: दस्तावेज़ अपलोड करें

इसके बाद, हमें वह दस्तावेज़ लोड करना होगा जिसे हम संसाधित करना चाहते हैं। इस उदाहरण में, हम मानते हैं कि दस्तावेज़ को "रेंडरिंग.docx" कहा जाता है और वह निर्दिष्ट दस्तावेज़ निर्देशिका में स्थित है।

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## चरण 3: पीडीएफ विकल्प के रूप में सहेजें कॉन्फ़िगर करें

 दस्तावेज़ में उपयोग किए गए फ़ॉन्ट के केवल सबसेट युक्त एक पीडीएफ बनाने के लिए, हमें कॉन्फ़िगर करने की आवश्यकता है`PdfSaveOptions` के साथ वस्तु`EmbedFullFonts` संपत्ति के लिए सेट`false`.

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions { EmbedFullFonts = false };
```

## चरण 4: फ़ॉन्ट उपसमुच्चय के साथ दस्तावेज़ को पीडीएफ के रूप में सहेजें

 अंत में, हम फ़ॉन्ट उपसमुच्चय का उपयोग करके दस्तावेज़ को पीडीएफ के रूप में सहेज सकते हैं। आउटपुट फ़ाइल नाम और निर्दिष्ट करें`saveOptions` ऑब्जेक्ट जिसे हमने पिछले चरण में कॉन्फ़िगर किया था।

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.EmbeddSubsetFonts.pdf", saveOptions);
```

बस इतना ही ! आपने दस्तावेज़ में फ़ॉन्ट के सबसेट को सफलतापूर्वक एम्बेड कर दिया है और .NET के लिए Aspose.Words के साथ दस्तावेज़ में उपयोग किए गए केवल ग्लिफ़ युक्त एक पीडीएफ तैयार किया है।

### .NET के लिए Aspose.Words के साथ फ़ॉन्ट उपसमुच्चय एम्बेड करने के लिए नमूना स्रोत कोड

```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Rendering.docx");

	// आउटपुट पीडीएफ में दस्तावेज़ में फ़ॉन्ट के सबसेट होंगे।
	// दस्तावेज़ में उपयोग किए गए केवल ग्लिफ़ ही पीडीएफ फ़ॉन्ट में शामिल हैं।
	PdfSaveOptions saveOptions = new PdfSaveOptions { EmbedFullFonts = false };
	
	doc.Save(dataDir + "WorkingWithPdfSaveOptions.EmbeddSubsetFonts.pdf", saveOptions);

```

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा कि .NET के लिए Aspose.Words का उपयोग करके एक पीडीएफ दस्तावेज़ में फ़ॉन्ट सबसेट को कैसे एम्बेड किया जाए। फ़ॉन्ट के सबसेट को एंबेड करने से केवल वास्तव में उपयोग किए गए वर्णों का उपयोग करके दस्तावेज़ के स्वरूप को संरक्षित करते हुए पीडीएफ फ़ाइल के आकार को कम करने में मदद मिलती है। यह पीडीएफ देखने और प्रिंट करते समय बेहतर अनुकूलता और प्रदर्शन सुनिश्चित करता है। एम्बेडेड फ़ॉन्ट सबसेट के साथ अपने पीडीएफ दस्तावेज़ों की पीढ़ी को अनुकूलित करने के लिए .NET के लिए Aspose.Words की सुविधाओं को और अधिक जानने के लिए स्वतंत्र महसूस करें।

### अक्सर पूछे जाने वाले प्रश्नों

#### प्रश्न: पीडीएफ दस्तावेज़ में फ़ॉन्ट उपसमुच्चय एम्बेड करना क्या है?
उ: पीडीएफ दस्तावेज़ में फ़ॉन्ट उपसमुच्चय को एम्बेड करना सभी पूर्ण फ़ॉन्ट को शामिल करने के बजाय दस्तावेज़ में उपयोग किए गए केवल ग्लिफ़ को शामिल करने की प्रक्रिया है। यह दस्तावेज़ में वास्तव में उपयोग किए गए वर्णों को प्रदर्शित करने के लिए आवश्यक केवल फ़ॉन्ट डेटा को शामिल करके पीडीएफ फ़ाइल के आकार को कम करता है।

#### प्रश्न: पूर्ण फ़ॉन्ट एम्बेड करने और फ़ॉन्ट के सबसेट एम्बेड करने के बीच क्या अंतर है?
उ: पूर्ण फ़ॉन्ट एम्बेडिंग का अर्थ है दस्तावेज़ में उपयोग किए गए सभी फ़ॉन्ट को पीडीएफ फ़ाइल में शामिल करना, जो यह सुनिश्चित करता है कि दस्तावेज़ बिल्कुल उसी तरह प्रदर्शित किया जाएगा जैसा इसे डिज़ाइन किया गया था, लेकिन पीडीएफ फ़ाइल का आकार बढ़ सकता है। इसके विपरीत, एम्बेडिंग फ़ॉन्ट उपसमुच्चय में केवल दस्तावेज़ में उपयोग किए गए ग्लिफ़ शामिल होते हैं, जिससे पीडीएफ फ़ाइल का आकार कम हो जाता है, लेकिन बाद में अतिरिक्त वर्ण जोड़े जाने पर दस्तावेज़ के स्वरूप को बिल्कुल दोहराने की क्षमता सीमित हो जाती है।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके एक पीडीएफ दस्तावेज़ में फ़ॉन्ट सबसेट कैसे एम्बेड कर सकता हूं?
उ: .NET के लिए Aspose.Words का उपयोग करके एक पीडीएफ दस्तावेज़ में फ़ॉन्ट सबसेट एम्बेड करने के लिए, इन चरणों का पालन करें:

 प्रतिस्थापित करके दस्तावेज़ निर्देशिका पथ सेट करें`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ।

 उस दस्तावेज़ को लोड करें जिसे आप का उपयोग करके संसाधित करना चाहते हैं`Document` कक्षा और दस्तावेज़ पथ।

 का एक उदाहरण बनाकर पीडीएफ सेव विकल्पों को कॉन्फ़िगर करें`PdfSaveOptions` क्लास और सेटिंग`EmbedFullFonts`संपत्ति को`false`यह सुनिश्चित करता है कि दस्तावेज़ में उपयोग किए गए केवल फ़ॉन्ट उपसमूह ही पीडीएफ फ़ाइल में शामिल किए जाएंगे।

 का उपयोग करके एम्बेड किए गए फ़ॉन्ट सबसेट के साथ दस्तावेज़ को पीडीएफ प्रारूप में सहेजें`Save` की विधि`Document` ऑब्जेक्ट, आउटपुट फ़ाइल का नाम और पहले से कॉन्फ़िगर किए गए सेव विकल्प निर्दिष्ट करता है।

#### प्रश्न: पीडीएफ दस्तावेज़ में फ़ॉन्ट उपसमुच्चय एम्बेड करने के क्या लाभ हैं?
उ: पीडीएफ दस्तावेज़ में फ़ॉन्ट उपसमुच्चय एम्बेड करने के लाभ हैं:

पीडीएफ फ़ाइल का आकार कम होना: दस्तावेज़ में उपयोग किए गए केवल ग्लिफ़ को शामिल करने से, पूर्ण फ़ॉन्ट एम्बेड करने की तुलना में पीडीएफ फ़ाइल का आकार कम हो जाता है।

दस्तावेज़ की उपस्थिति का संरक्षण: पीडीएफ फ़ाइल में शामिल फ़ॉन्ट के सबसेट केवल वास्तव में उपयोग किए गए वर्णों का उपयोग करके दस्तावेज़ की उपस्थिति को पुन: उत्पन्न करना संभव बनाते हैं।

लाइसेंस के प्रतिबंधों के साथ संगतता: उन मामलों में फ़ॉन्ट के एम्बेडिंग उपसमूह को प्राथमिकता दी जा सकती है जहां लाइसेंसिंग प्रतिबंधों के कारण पूर्ण फ़ॉन्ट को कानूनी रूप से एम्बेड नहीं किया जा सकता है।
