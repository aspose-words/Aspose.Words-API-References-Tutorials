यह लेख Aspose.Words for .NET के साथ फ़ॉन्ट सबसेट एम्बेडिंग सुविधा का उपयोग करने के तरीके पर चरण-दर-चरण मार्गदर्शिका प्रदान करता है। हम कोड के प्रत्येक भाग को विस्तार से समझाएंगे। इस ट्यूटोरियल के अंत में, आप समझ पाएंगे कि दस्तावेज़ में फ़ॉन्ट के सबसेट को कैसे एम्बेड किया जाए और दस्तावेज़ में उपयोग किए गए केवल ग्लिफ़ वाले PDF को कैसे बनाया जाए।

शुरू करने से पहले, सुनिश्चित करें कि आपने अपने प्रोजेक्ट में Aspose.Words for .NET लाइब्रेरी को इंस्टॉल और कॉन्फ़िगर किया है। आप Aspose वेबसाइट पर लाइब्रेरी और इंस्टॉलेशन निर्देश पा सकते हैं।

## चरण 1: दस्तावेज़ निर्देशिका निर्धारित करें

 शुरू करने के लिए, आपको उस निर्देशिका का पथ परिभाषित करना होगा जहाँ आपके दस्तावेज़ स्थित हैं।`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: दस्तावेज़ अपलोड करें

इसके बाद, हमें उस दस्तावेज़ को लोड करना होगा जिसे हम प्रोसेस करना चाहते हैं। इस उदाहरण में, हम मानते हैं कि दस्तावेज़ का नाम "Rendering.docx" है और यह निर्दिष्ट दस्तावेज़ निर्देशिका में स्थित है।

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## चरण 3: PDF के रूप में सहेजें विकल्प कॉन्फ़िगर करें

 दस्तावेज़ में प्रयुक्त फ़ॉन्ट के केवल उपसमूहों वाले PDF बनाने के लिए, हमें कॉन्फ़िगर करने की आवश्यकता है`PdfSaveOptions` वस्तु के साथ`EmbedFullFonts` संपत्ति सेट करने के लिए`false`.

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions { EmbedFullFonts = false };
```

## चरण 4: फ़ॉन्ट उपसमूहों के साथ दस्तावेज़ को PDF के रूप में सहेजें

 अंत में, हम फ़ॉन्ट उपसमूहों का उपयोग करके दस्तावेज़ को PDF के रूप में सहेज सकते हैं। आउटपुट फ़ाइल नाम और निर्दिष्ट करें`saveOptions` ऑब्जेक्ट जिसे हमने पिछले चरण में कॉन्फ़िगर किया था।

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.EmbeddSubsetFonts.pdf", saveOptions);
```

बस इतना ही! आपने एक दस्तावेज़ में फ़ॉन्ट के सबसेट को सफलतापूर्वक एम्बेड किया है और Aspose.Words for .NET के साथ दस्तावेज़ में उपयोग किए गए केवल ग्लिफ़ युक्त एक पीडीएफ तैयार किया है।

### .NET के लिए Aspose.Words के साथ फ़ॉन्ट उपसमूह एम्बेड करने के लिए नमूना स्रोत कोड

```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Rendering.docx");

	// आउटपुट पीडीएफ में दस्तावेज़ के फ़ॉन्ट्स के उपसमूह शामिल होंगे।
	// दस्तावेज़ में प्रयुक्त ग्लिफ़ को ही पीडीएफ फ़ॉन्ट में शामिल किया गया है।
	PdfSaveOptions saveOptions = new PdfSaveOptions { EmbedFullFonts = false };
	
	doc.Save(dataDir + "WorkingWithPdfSaveOptions.EmbeddSubsetFonts.pdf", saveOptions);

```

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा कि Aspose.Words for .NET का उपयोग करके PDF दस्तावेज़ में फ़ॉन्ट सबसेट कैसे एम्बेड करें। फ़ॉन्ट के सबसेट एम्बेड करने से PDF फ़ाइल का आकार कम करने में मदद मिलती है जबकि दस्तावेज़ के लुक को केवल वास्तव में उपयोग किए गए वर्णों का उपयोग करके संरक्षित किया जाता है। यह PDF को देखने और प्रिंट करते समय बेहतर संगतता और प्रदर्शन सुनिश्चित करता है। एम्बेडेड फ़ॉन्ट सबसेट के साथ अपने PDF दस्तावेज़ों की पीढ़ी को अनुकूलित करने के लिए Aspose.Words for .NET की विशेषताओं को और अधिक जानने के लिए स्वतंत्र महसूस करें।

### अक्सर पूछे जाने वाले प्रश्नों

#### प्रश्न: पीडीएफ दस्तावेज़ में फ़ॉन्ट उपसमूह एम्बेड करना क्या है?
उत्तर: पीडीएफ दस्तावेज़ में फ़ॉन्ट उपसमूह एम्बेड करना दस्तावेज़ में उपयोग किए गए केवल ग्लिफ़ को शामिल करने की प्रक्रिया है, न कि सभी पूर्ण फ़ॉन्ट को शामिल करना। यह दस्तावेज़ में वास्तव में उपयोग किए गए वर्णों को प्रदर्शित करने के लिए आवश्यक केवल फ़ॉन्ट डेटा को शामिल करके पीडीएफ फ़ाइल के आकार को कम करता है।

#### प्रश्न: पूर्ण फ़ॉन्ट एम्बेड करने और फ़ॉन्ट के उपसमूह एम्बेड करने में क्या अंतर है?
उत्तर: पूर्ण फ़ॉन्ट एम्बेडिंग का अर्थ है PDF फ़ाइल में दस्तावेज़ में उपयोग किए गए सभी फ़ॉन्ट शामिल करना, जो यह सुनिश्चित करता है कि दस्तावेज़ बिल्कुल वैसा ही प्रदर्शित होगा जैसा कि इसे डिज़ाइन किया गया था, लेकिन PDF फ़ाइल का आकार बढ़ा सकता है। इसके विपरीत, फ़ॉन्ट उपसमूहों को एम्बेडिंग में केवल दस्तावेज़ में उपयोग किए गए ग्लिफ़ शामिल होते हैं, जिससे PDF फ़ाइल का आकार कम हो जाता है, लेकिन बाद में अतिरिक्त वर्ण जोड़े जाने पर दस्तावेज़ के रूप को बिल्कुल वैसा ही दोहराने की क्षमता सीमित हो जाती है।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके PDF दस्तावेज़ में फ़ॉन्ट उपसमूह कैसे एम्बेड कर सकता हूं?
उत्तर: .NET के लिए Aspose.Words का उपयोग करके PDF दस्तावेज़ में फ़ॉन्ट उपसमूह एम्बेड करने के लिए, इन चरणों का पालन करें:

 प्रतिस्थापित करके दस्तावेज़ निर्देशिका पथ सेट करें`"YOUR DOCUMENT DIRECTORY"` अपने दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

 उस दस्तावेज़ को लोड करें जिसे आप संसाधित करना चाहते हैं`Document` क्लास और दस्तावेज़ पथ.

 पीडीएफ सेव विकल्पों को कॉन्फ़िगर करने के लिए इसका एक उदाहरण बनाएं`PdfSaveOptions` वर्ग और सेटिंग`EmbedFullFonts`संपत्ति को`false`यह सुनिश्चित करता है कि दस्तावेज़ में उपयोग किए गए फ़ॉन्ट उपसमूह ही पीडीएफ फ़ाइल में शामिल किए जाएंगे।

 दस्तावेज़ को PDF प्रारूप में सहेजें, फ़ॉन्ट उपसमूहों को एम्बेड करके`Save` की विधि`Document` ऑब्जेक्ट, आउटपुट फ़ाइल का नाम और पहले से कॉन्फ़िगर किए गए सेव विकल्प निर्दिष्ट करता है।

#### प्रश्न: पीडीएफ दस्तावेज़ में फ़ॉन्ट उपसमूह एम्बेड करने के क्या लाभ हैं?
उत्तर: पीडीएफ दस्तावेज़ में फ़ॉन्ट उपसमूह एम्बेड करने के लाभ ये हैं:

पीडीएफ फाइल का आकार कम करना: दस्तावेज़ में प्रयुक्त केवल ग्लिफ़ को शामिल करने से, पूर्ण फ़ॉन्ट एम्बेड करने की तुलना में पीडीएफ फ़ाइल का आकार कम हो जाता है।

दस्तावेज़ के स्वरूप का संरक्षण: पीडीएफ फाइल में शामिल फ़ॉन्ट के उपसमूह केवल वास्तव में प्रयुक्त वर्णों का उपयोग करके दस्तावेज़ के स्वरूप को पुन: प्रस्तुत करना संभव बनाते हैं।

लाइसेंस के प्रतिबंधों के साथ संगतता: उन मामलों में फ़ॉन्ट के उपसमूहों को एम्बेड करना बेहतर हो सकता है, जहां लाइसेंस प्रतिबंधों के कारण पूर्ण फ़ॉन्ट को कानूनी रूप से एम्बेड नहीं किया जा सकता है।
