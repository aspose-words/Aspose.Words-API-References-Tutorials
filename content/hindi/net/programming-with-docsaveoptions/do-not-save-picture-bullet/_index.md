पिक्चर बुलेट्स Word दस्तावेज़ों में कस्टम बुलेट्स जोड़ने के लिए आमतौर पर उपयोग की जाने वाली सुविधा है। हालाँकि, कुछ मामलों में .NET के लिए Aspose.Words लाइब्रेरी का उपयोग करके दस्तावेज़ों में हेरफेर करते समय छवि बुलेट पंजीकरण को अक्षम करना आवश्यक हो सकता है। इस चरण-दर-चरण मार्गदर्शिका में, हम बताएंगे कि DocSaveOptions सेव विकल्पों का उपयोग करके छवि बुलेट सेविंग को अक्षम करने के लिए .NET के लिए Aspose.Words C# स्रोत कोड का उपयोग कैसे करें।

## Aspose.Words लाइब्रेरी को समझना

कोड में गोता लगाने से पहले, .NET के लिए Aspose.Words लाइब्रेरी को समझना महत्वपूर्ण है। Aspose.Words .NET सहित विभिन्न प्लेटफार्मों में Word दस्तावेज़ों को बनाने, संपादित करने, परिवर्तित करने और सुरक्षित करने के लिए एक शक्तिशाली लाइब्रेरी है। यह दस्तावेज़ों में हेरफेर करने के लिए कई सुविधाएँ प्रदान करता है, जैसे टेक्स्ट सम्मिलित करना, फ़ॉर्मेटिंग बदलना, अनुभाग जोड़ना और बहुत कुछ।

## चरण 1: दस्तावेज़ निर्देशिका सेट करना

पहला कदम उस निर्देशिका को परिभाषित करना है जहां आपके दस्तावेज़ स्थित हैं। आपको संपूर्ण निर्देशिका पथ निर्दिष्ट करना होगा. उदाहरण के लिए :

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

अपने दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ "आपकी दस्तावेज़ निर्देशिका" को बदलना सुनिश्चित करें।

## चरण 2: दस्तावेज़ को इमेज बुलेट्स के साथ लोड करना

इसके बाद, आपको दस्तावेज़ को छवि बुलेट के साथ लोड करना होगा। किसी फ़ाइल से दस्तावेज़ लोड करने के लिए दस्तावेज़ वर्ग का उपयोग करें। उदाहरण के लिए :

```csharp
Document doc = new Document(dataDir + "Image bullet points.docx");
```

इस उदाहरण में हम दस्तावेज़ को "इमेज बुलेट पॉइंट्स.docx" फ़ाइल से लोड कर रहे हैं।

  दस्तावेज़ निर्देशिका में स्थित है।

## चरण 3: रिकॉर्डिंग विकल्प कॉन्फ़िगर करें

आइए अब अपने दस्तावेज़ के लिए सेव विकल्पों को कॉन्फ़िगर करें। सेव सेटिंग्स निर्दिष्ट करने के लिए DocSaveOptions क्लास का उपयोग करें। उदाहरण के लिए :

```csharp
DocSaveOptions saveOptions = new DocSaveOptions { SavePictureBullet = false };
```

इस उदाहरण में, हम एक नया DocSaveOptions ऑब्जेक्ट बनाते हैं और पिक्चर बुलेट्स को सेव करना अक्षम करने के लिए SavePictureBullet प्रॉपर्टी को गलत पर सेट करते हैं।

## चरण 4: "पिक्चर बुलेट न सहेजें" सुविधा सक्षम करें

"डू नॉट सेव पिक्चर बुलेट" सुविधा को सक्षम करने के लिए, हमने पहले से ही SavePictureBullet को गलत पर सेट करके सेव विकल्पों को कॉन्फ़िगर कर लिया है। यह सुनिश्चित करता है कि छवि बुलेट अंतिम दस्तावेज़ में सहेजे नहीं गए हैं।

## चरण 5: दस्तावेज़ सहेजें

अंत में, आप दस्तावेज़ वर्ग की सेव विधि का उपयोग करके दस्तावेज़ को सहेज सकते हैं। फ़ाइल का पूरा पथ और वांछित फ़ाइल नाम निर्दिष्ट करें। उदाहरण के लिए :

```csharp
doc.Save(dataDir + "WorkingWithDocSaveOptions.DoNotSavePictureBullet.docx", saveOptions);
```

अपने दस्तावेज़ों में "dataDir" को निर्देशिका पथ से बदलना सुनिश्चित करें।

## DocSaveOptions के लिए उदाहरण स्रोत कोड, .NET के लिए Aspose.Words का उपयोग करके "डू नॉट सेव पिक्चर बुलेट" कार्यक्षमता वाले विकल्प सहेजें

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// दस्तावेज़ को छवि बुलेट्स के साथ लोड करें
Document doc = new Document(dataDir + "Image bullet points.docx");

// "डू नॉट सेव पिक्चर बुलेट" सुविधा के साथ सेव विकल्पों को कॉन्फ़िगर करें
DocSaveOptions saveOptions = new DocSaveOptions { SavePictureBullet = false };

// दस्तावेज़ को निर्दिष्ट विकल्पों के साथ सहेजें
doc.Save(dataDir + "WorkingWithDocSaveOptions.DoNotSavePictureBullet.docx", saveOptions);
```

## निष्कर्ष

इस गाइड में, हमने .NET के लिए Aspose.Words लाइब्रेरी का उपयोग करके किसी दस्तावेज़ में छवि बुलेट्स को सहेजने को अक्षम करने का तरीका बताया है। दिए गए चरणों का पालन करके और दिए गए C# स्रोत कोड का उपयोग करके, आप इस कार्यक्षमता को अपने C# एप्लिकेशन में आसानी से लागू कर सकते हैं। पिक्चर बुलेट सेविंग को अक्षम करना कुछ स्थितियों में पिक्चर बुलेट को सेव किए बिना दस्तावेज़ संरचना और फ़ॉर्मेटिंग को संरक्षित करने के लिए उपयोगी हो सकता है।
