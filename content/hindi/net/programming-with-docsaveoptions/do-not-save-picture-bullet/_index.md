कस्टम बुलेट जोड़ने के लिए Word दस्तावेज़ों में पिक्चर बुलेट एक आम तौर पर इस्तेमाल की जाने वाली सुविधा है। हालाँकि, कुछ मामलों में .NET के लिए Aspose.Words लाइब्रेरी का उपयोग करके दस्तावेज़ों में हेरफेर करते समय इमेज बुलेट पंजीकरण को अक्षम करना आवश्यक हो सकता है। इस चरण-दर-चरण मार्गदर्शिका में, हम बताएंगे कि DocSaveOptions सेव विकल्पों का उपयोग करके इमेज बुलेट सेविंग को अक्षम करने के लिए .NET के लिए Aspose.Words C# स्रोत कोड का उपयोग कैसे करें।

## Aspose.Words लाइब्रेरी को समझना

कोड में गोता लगाने से पहले, .NET के लिए Aspose.Words लाइब्रेरी को समझना महत्वपूर्ण है। Aspose.Words .NET सहित विभिन्न प्लेटफ़ॉर्म में Word दस्तावेज़ बनाने, संपादित करने, परिवर्तित करने और सुरक्षित करने के लिए एक शक्तिशाली लाइब्रेरी है। यह दस्तावेज़ों में हेरफेर करने के लिए कई सुविधाएँ प्रदान करता है, जैसे कि टेक्स्ट डालना, फ़ॉर्मेटिंग बदलना, अनुभाग जोड़ना और बहुत कुछ।

## चरण 1: दस्तावेज़ निर्देशिका सेट करना

पहला कदम उस निर्देशिका को परिभाषित करना है जहाँ आपके दस्तावेज़ स्थित हैं। आपको पूर्ण निर्देशिका पथ निर्दिष्ट करना होगा। उदाहरण के लिए:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

"आपकी दस्तावेज़ निर्देशिका" को अपनी दस्तावेज़ निर्देशिका के वास्तविक पथ से प्रतिस्थापित करना सुनिश्चित करें।

## चरण 2: छवि बुलेट के साथ दस्तावेज़ लोड करना

इसके बाद, आपको छवि बुलेट के साथ दस्तावेज़ लोड करना होगा। फ़ाइल से दस्तावेज़ लोड करने के लिए Document क्लास का उपयोग करें। उदाहरण के लिए:

```csharp
Document doc = new Document(dataDir + "Image bullet points.docx");
```

इस उदाहरण में हम "Image bullet points.docx" फ़ाइल से दस्तावेज़ लोड कर रहे हैं

  दस्तावेज़ निर्देशिका में स्थित है.

## चरण 3: रिकॉर्डिंग विकल्प कॉन्फ़िगर करें

अब आइए अपने दस्तावेज़ के लिए सेव ऑप्शन कॉन्फ़िगर करें। सेव सेटिंग निर्दिष्ट करने के लिए DocSaveOptions क्लास का उपयोग करें। उदाहरण के लिए:

```csharp
DocSaveOptions saveOptions = new DocSaveOptions { SavePictureBullet = false };
```

इस उदाहरण में, हम एक नया DocSaveOptions ऑब्जेक्ट बनाते हैं और चित्र बुलेट्स को सहेजने को अक्षम करने के लिए SavePictureBullet गुण को false पर सेट करते हैं।

## चरण 4: "पिक्चर बुलेट को न सहेजें" सुविधा सक्षम करें

"पिक्चर बुलेट को न सहेजें" सुविधा को सक्षम करने के लिए, हमने पहले से ही SavePictureBullet को false पर सेट करके सेव विकल्पों को कॉन्फ़िगर कर लिया है। यह सुनिश्चित करता है कि इमेज बुलेट को अंतिम दस्तावेज़ में सहेजा नहीं गया है।

## चरण 5: दस्तावेज़ सहेजें

अंत में, आप Document क्लास की Save विधि का उपयोग करके दस्तावेज़ को सहेज सकते हैं। फ़ाइल का पूरा पथ और वांछित फ़ाइल नाम निर्दिष्ट करें। उदाहरण के लिए :

```csharp
doc.Save(dataDir + "WorkingWithDocSaveOptions.DoNotSavePictureBullet.docx", saveOptions);
```

"dataDir" को अपने दस्तावेज़ों के निर्देशिका पथ से प्रतिस्थापित करना सुनिश्चित करें।

## .NET के लिए Aspose.Words का उपयोग करके "चित्र बुलेट को न सहेजें" कार्यक्षमता के साथ DocSaveOptions सहेजने के विकल्पों के लिए उदाहरण स्रोत कोड

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// दस्तावेज़ को छवि बुलेट से लोड करें
Document doc = new Document(dataDir + "Image bullet points.docx");

// "पिक्चर बुलेट को न सहेजें" सुविधा के साथ सेव विकल्पों को कॉन्फ़िगर करें
DocSaveOptions saveOptions = new DocSaveOptions { SavePictureBullet = false };

// निर्दिष्ट विकल्पों के साथ दस्तावेज़ को सहेजें
doc.Save(dataDir + "WorkingWithDocSaveOptions.DoNotSavePictureBullet.docx", saveOptions);
```

## निष्कर्ष

इस गाइड में, हमने .NET के लिए Aspose.Words लाइब्रेरी का उपयोग करके दस्तावेज़ में छवि बुलेट सहेजने को अक्षम करने का तरीका बताया है। दिए गए चरणों का पालन करके और दिए गए C# स्रोत कोड का उपयोग करके, आप आसानी से अपने C# एप्लिकेशन में इस कार्यक्षमता को लागू कर सकते हैं। चित्र बुलेट सहेजने को अक्षम करना कुछ स्थितियों में चित्र बुलेट सहेजे बिना दस्तावेज़ संरचना और स्वरूपण को संरक्षित करने के लिए उपयोगी हो सकता है।
