इस आलेख में, हम .NET लाइब्रेरी के लिए Aspose.Words में रिप्लेस टेक्स्ट इन फूटर फ़ंक्शन का उपयोग करने के तरीके को समझने के लिए उपरोक्त C# स्रोत कोड का पता लगाएंगे। यह सुविधा आपको Word दस्तावेज़ों के फ़ुटर में विशिष्ट टेक्स्ट को ढूंढने और बदलने की अनुमति देती है।

## आवश्यक शर्तें

- C# भाषा का बुनियादी ज्ञान।
- Aspose.Words लाइब्रेरी के साथ .NET विकास वातावरण स्थापित।

## चरण 1: दस्तावेज़ लोड करें

इससे पहले कि हम पाद लेख में टेक्स्ट प्रतिस्थापन का उपयोग शुरू करें, हमें दस्तावेज़ को .NET के लिए Aspose.Words में लोड करना होगा। इसका उपयोग करके ऐसा किया जा सकता है`Document` क्लास और दस्तावेज़ फ़ाइल पथ निर्दिष्ट करना:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document doc = new Document(dataDir + "Footer.docx");
```

## चरण 2: पादलेख तक पहुंचें

 एक बार दस्तावेज़ लोड हो जाने के बाद, हमें टेक्स्ट प्रतिस्थापन करने के लिए पादलेख तक पहुंचने की आवश्यकता होती है। हमारे उदाहरण में, हम इसका उपयोग करते हैं`HeadersFooters` शीर्षलेख/पाद लेख का संग्रह प्राप्त करने के लिए दस्तावेज़ के पहले खंड की संपत्ति। इसके बाद, हम का उपयोग करके मुख्य पादलेख का चयन करते हैं`HeaderFooterType.FooterPrimary` अनुक्रमणिका:

```csharp
HeaderFooterCollection headersFooters = doc.FirstSection.HeadersFooters;
HeaderFooter footer = headersFooters[HeaderFooterType.FooterPrimary];
```

## चरण 3: खोज कॉन्फ़िगर करें और विकल्प बदलें

 अब हम a का उपयोग करके विकल्पों को ढूंढने और बदलने को कॉन्फ़िगर करेंगे`FindReplaceOptions` वस्तु। हमारे उदाहरण में, हम सेट करते हैं`MatchCase` को`false` खोज करते समय मामले को नज़रअंदाज करना, और`FindWholeWordsOnly` को`false` शब्दों के कुछ हिस्सों को खोजने और बदलने की अनुमति देने के लिए:

```csharp
FindReplaceOptions options = new FindReplaceOptions { MatchCase = false, FindWholeWordsOnly = false };
```

## चरण 4: पाद लेख में टेक्स्ट बदलें

 हम उपयोग करते हैं`Range.Replace` पादलेख में पाठ प्रतिस्थापन करने की विधि। हमारे उदाहरण में, हम वाक्यांश "(C) 2006 Aspose Pty Ltd" को प्रतिस्थापित करते हैं। "कॉपीराइट (सी) 2020 Aspose Pty Ltd द्वारा।" :

```csharp
footer

.Range.Replace("(C) 2006 Aspose Pty Ltd.", "Copyright (C) 2020 by Aspose Pty Ltd.", options);
```

## चरण 5: संपादित दस्तावेज़ सहेजें

अंत में, हम संशोधित दस्तावेज़ को इसका उपयोग करके एक निर्दिष्ट निर्देशिका में सहेजते हैं`Save` तरीका:

```csharp
doc.Save(dataDir + "FindAndReplace.ReplaceTextInFooter.docx");
```

### .NET के लिए Aspose.Words का उपयोग करके फ़ुटर में टेक्स्ट बदलने के लिए उदाहरण स्रोत कोड

.NET के लिए Aspose.Words के साथ पाद लेख प्रतिस्थापन के उपयोग को प्रदर्शित करने के लिए यहां पूर्ण नमूना स्रोत कोड दिया गया है:

```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(MyDir + "Footer.docx");

	HeaderFooterCollection headersFooters = doc.FirstSection.HeadersFooters;
	HeaderFooter footer = headersFooters[HeaderFooterType.FooterPrimary];

	FindReplaceOptions options = new FindReplaceOptions { MatchCase = false, FindWholeWordsOnly = false };

	footer.Range.Replace("(C) 2006 Aspose Pty Ltd.", "Copyright (C) 2020 by Aspose Pty Ltd.", options);

	doc.Save(dataDir + "FindAndReplace.ReplaceTextInFooter.docx");
            
        
```
