इस उदाहरण में, हम आपको दिखाने जा रहे हैं कि .NET के लिए Aspose.Words के साथ क्षैतिज नियम सुविधा का उपयोग कैसे करें। क्षैतिज नियम का उपयोग किसी दस्तावेज़ के अनुभागों को दृष्टिगत रूप से अलग करने के लिए किया जाता है।

## चरण 1: दस्तावेज़ जनरेटर का उपयोग करना

सबसे पहले, हम अपने दस्तावेज़ में सामग्री जोड़ने के लिए एक दस्तावेज़ जनरेटर का उपयोग करेंगे।

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## चरण 2: एक क्षैतिज नियम सम्मिलित करना

 हम इसका उपयोग करके एक क्षैतिज नियम सम्मिलित कर सकते हैं`InsertHorizontalRule` दस्तावेज़ जनरेटर की विधि.

```csharp
builder. InsertHorizontalRule();
```

## .NET के लिए Aspose.Words के साथ क्षैतिज नियम के लिए नमूना स्रोत कोड

```csharp
// दस्तावेज़ में सामग्री जोड़ने के लिए दस्तावेज़ निर्माता का उपयोग करें।
DocumentBuilder builder = new DocumentBuilder();

// क्षैतिज नियम सम्मिलित करें.
builder.InsertHorizontalRule();
```

बधाई हो! अब आपने सीख लिया है कि .NET के लिए Aspose.Words के साथ क्षैतिज नियम सुविधा का उपयोग कैसे करें।


### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मैं मार्कडाउन में एक क्षैतिज रूलर कैसे बनाऊं?

उ: मार्कडाउन में एक क्षैतिज रूलर बनाने के लिए, आप एक खाली लाइन पर निम्नलिखित प्रतीकों में से एक का उपयोग कर सकते हैं: तीन तारांकन (\***), तीन डैश (\---), या तीन अंडरस्कोर (\___).

#### प्रश्न: क्या मैं मार्कडाउन में क्षैतिज रूलर के स्वरूप को अनुकूलित कर सकता हूँ?

उ: मानक मार्कडाउन में, क्षैतिज शासकों की उपस्थिति को अनुकूलित करने का कोई तरीका नहीं है। हालाँकि, कुछ उन्नत मार्कडाउन संपादक और एक्सटेंशन अतिरिक्त अनुकूलन सुविधाएँ प्रदान करते हैं।

#### प्रश्न: क्या क्षैतिज शासक सभी मार्कडाउन संपादकों द्वारा समर्थित हैं?

उत्तर: हां, अधिकांश लोकप्रिय मार्कडाउन संपादक क्षैतिज शासकों का समर्थन करते हैं। हालाँकि, यह सुनिश्चित करने के लिए कि यह समर्थित है, अपने विशिष्ट विक्रेता के दस्तावेज़ की जाँच करना हमेशा सर्वोत्तम होता है।

#### प्रश्न: मैं मार्कडाउन में और कौन से तत्व बना सकता हूं?

उ: क्षैतिज रूलर के अलावा, आप मार्कडाउन में शीर्षक, पैराग्राफ, सूचियाँ, लिंक, चित्र, तालिकाएँ और बहुत कुछ बना सकते हैं।
