इस उदाहरण में, हम आपको Aspose.Words for .NET के साथ क्षैतिज नियम सुविधा का उपयोग करने का तरीका दिखाने जा रहे हैं। क्षैतिज नियम का उपयोग दस्तावेज़ के अनुभागों को दृष्टिगत रूप से अलग करने के लिए किया जाता है।

## चरण 1: दस्तावेज़ जनरेटर का उपयोग करना

सबसे पहले, हम अपने दस्तावेज़ में सामग्री जोड़ने के लिए दस्तावेज़ जनरेटर का उपयोग करेंगे।

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## चरण 2: क्षैतिज नियम सम्मिलित करना

 हम इसका उपयोग करके एक क्षैतिज नियम सम्मिलित कर सकते हैं`InsertHorizontalRule` दस्तावेज़ जनरेटर की विधि.

```csharp
builder. InsertHorizontalRule();
```

## .NET के लिए Aspose.Words के साथ क्षैतिज नियम के लिए नमूना स्रोत कोड

```csharp
// दस्तावेज़ में सामग्री जोड़ने के लिए दस्तावेज़ बिल्डर का उपयोग करें.
DocumentBuilder builder = new DocumentBuilder();

// क्षैतिज नियम डालें.
builder.InsertHorizontalRule();
```

बधाई हो! अब आपने सीख लिया है कि .NET के लिए Aspose.Words के साथ क्षैतिज नियम सुविधा का उपयोग कैसे करें।


### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मैं मार्कडाउन में क्षैतिज रूलर कैसे बनाऊं?

उत्तर: मार्कडाउन में एक क्षैतिज रूलर बनाने के लिए, आप खाली लाइन पर निम्नलिखित प्रतीकों में से एक का उपयोग कर सकते हैं: तीन तारांकन चिह्न (\***), तीन डैश (\---), या तीन अंडरस्कोर (\___).

#### प्रश्न: क्या मैं मार्कडाउन में क्षैतिज रूलर के स्वरूप को अनुकूलित कर सकता हूँ?

उत्तर: मानक मार्कडाउन में, क्षैतिज रूलर की उपस्थिति को अनुकूलित करने का कोई तरीका नहीं है। हालाँकि, कुछ उन्नत मार्कडाउन संपादक और एक्सटेंशन अतिरिक्त अनुकूलन सुविधाएँ प्रदान करते हैं।

#### प्रश्न: क्या क्षैतिज रूलर सभी मार्कडाउन संपादकों द्वारा समर्थित हैं?

उत्तर: हां, अधिकांश लोकप्रिय मार्कडाउन संपादक क्षैतिज रूलर का समर्थन करते हैं। हालांकि, यह सुनिश्चित करने के लिए कि यह समर्थित है, अपने विशिष्ट विक्रेता के दस्तावेज़ की जांच करना हमेशा सबसे अच्छा होता है।

#### प्रश्न: मार्कडाउन में मैं अन्य कौन से तत्व बना सकता हूँ?

उत्तर: क्षैतिज रूलर के अतिरिक्त, आप मार्कडाउन में शीर्षक, पैराग्राफ, सूचियाँ, लिंक, चित्र, तालिकाएँ आदि बना सकते हैं।
