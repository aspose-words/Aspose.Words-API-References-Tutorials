इस उदाहरण में, हम बताएंगे कि .NET के लिए Aspose.Words के साथ इंडेंटेड कोड सुविधा का उपयोग कैसे करें। इंडेंटेड कोड का उपयोग विशिष्ट स्वरूपण के साथ कोड के ब्लॉक को दृश्य रूप से दर्शाने के लिए किया जाता है।

## चरण 1: दस्तावेज़ जनरेटर का उपयोग करना

सबसे पहले, हम अपने दस्तावेज़ में सामग्री जोड़ने के लिए एक दस्तावेज़ जनरेटर का उपयोग करेंगे।

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## चरण 2: इंडेंटेड कोड के लिए स्टाइल जोड़ें

 हम इसका उपयोग करके इंडेंटेड कोड के लिए एक कस्टम शैली जोड़ेंगे`Styles.Add` की विधि`Document` वस्तु। इस उदाहरण में, हम इंडेंटेड कोड के लिए "इंडेंटेडकोड" नामक एक शैली बना रहे हैं।

```csharp
Style indentedCode = builder.Document.Styles.Add(StyleType.Paragraph, "IndentedCode");
builder.ParagraphFormat.Style = indentedCode;
```

## चरण 3: इंडेंटेड कोड जोड़ें

अब हम "इंडेंटेडकोड" कस्टम शैली का उपयोग करके एक इंडेंटेड कोड ब्लॉक जोड़ सकते हैं।

```csharp
builder.Writeln("This is an indented code block");
```

### .NET के लिए Aspose.Words के साथ इंडेंटेड कोड के लिए उदाहरण स्रोत कोड

```csharp
// दस्तावेज़ में सामग्री जोड़ने के लिए दस्तावेज़ निर्माता का उपयोग करें।
DocumentBuilder builder = new DocumentBuilder();

Style indentedCode = builder.Document.Styles.Add(StyleType.Paragraph, "IndentedCode");
builder.ParagraphFormat.Style = indentedCode;
builder.Writeln("This is an indented code");
```

बधाई हो! अब आपने सीख लिया है कि .NET के लिए Aspose.Words के साथ इंडेंटेड कोड सुविधा का उपयोग कैसे करें।


### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मार्कडाउन में इंडेंटेड कोड क्या है?

ए: मार्कडाउन में इंडेंटेड कोड एक फ़ॉर्मेटिंग विधि है जिसका उपयोग मार्कडाउन दस्तावेज़ में कोड प्रदर्शित करने के लिए किया जाता है। इसमें कोड की प्रत्येक पंक्ति को रिक्त स्थान या टैब के साथ इंडेंट करना शामिल है।

#### प्रश्न: मार्कडाउन में इंडेंटेड कोड का उपयोग कैसे करें?

ए: मार्कडाउन में इंडेंटेड कोड का उपयोग करने के लिए, कोड की प्रत्येक पंक्ति को रिक्त स्थान या टैब के साथ इंडेंट करें।

#### प्रश्न: मार्कडाउन में इंडेंटेड कोड के क्या फायदे हैं?

ए: मार्कडाउन में इंडेंटेड कोड कोड पठनीयता में सुधार करता है और पाठकों के लिए इसे समझना आसान बनाता है।

#### प्रश्न: मार्कडाउन में इंडेंटेड कोड और कोड ब्लॉक के बीच क्या अंतर है?

ए: इंडेंटेड कोड का उपयोग टेक्स्ट में डाले गए छोटे कोड स्निपेट के लिए किया जाता है, जबकि कोड ब्लॉक का उपयोग अलग-अलग फ़ॉर्मेटिंग में कोड के बड़े टुकड़ों को प्रदर्शित करने के लिए किया जाता है।

#### प्रश्न: क्या मार्कडाउन में इंडेंटेड कोड सभी मार्कडाउन संपादकों द्वारा समर्थित है?

उ: मार्कडाउन में इंडेंटेड कोड के लिए समर्थन मार्कडाउन संपादकों के बीच भिन्न हो सकता है। सुनिश्चित करने के लिए अपने प्रकाशक के विशिष्ट दस्तावेज़ की जाँच करें।
