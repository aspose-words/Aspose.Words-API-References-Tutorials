इस उदाहरण में, हम बताएंगे कि Aspose के साथ उद्धरण सुविधा का उपयोग कैसे करें। .NET उद्धरण के लिए शब्दों का उपयोग पाठ के अनुभागों को एक विशेष सीमा के साथ घेरकर हाइलाइट करने के लिए किया जाता है।

## चरण 1: दस्तावेज़ जनरेटर का उपयोग करना

सबसे पहले, हम अपने दस्तावेज़ में सामग्री जोड़ने के लिए एक दस्तावेज़ जनरेटर का उपयोग करेंगे।

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## चरण 2: डिफ़ॉल्ट उद्धरण शैली का उपयोग करना

हम पाठ पर उद्धरण स्वरूपण लागू करने के लिए "उद्धरण" नामक डिफ़ॉल्ट अनुच्छेद शैली का उपयोग करेंगे।

```csharp
builder.ParagraphFormat.StyleName = "Quote";
builder.Writeln("Blockquote");
```

## चरण 3: नेस्टेड स्तरों के लिए शैलियाँ बनाना

 हम इसका उपयोग करके नेस्टेड स्तरों के लिए शैलियाँ बना सकते हैं`Styles.Add` की विधि`Document` वस्तु। इस उदाहरण में, हम नेस्टेड उद्धरण स्तर का प्रतिनिधित्व करने के लिए "उद्धरण1" नामक एक शैली बना रहे हैं।

```csharp
Style quoteLevel2 = builder.Document.Styles.Add(StyleType.Paragraph, "Quote1");
builder.ParagraphFormat.Style = quoteLevel2;
builder.Document.Styles["Quote1"].BaseStyleName = "Quote";
builder.Writeln("1. Nested blockquote");
```

### .NET के लिए Aspose.Words के साथ उद्धरणों के लिए उदाहरण स्रोत कोड


```csharp
// दस्तावेज़ में सामग्री जोड़ने के लिए दस्तावेज़ निर्माता का उपयोग करें।
DocumentBuilder builder = new DocumentBuilder();

// डिफ़ॉल्ट रूप से कोई दस्तावेज़ पहले स्तर के लिए ब्लॉककोट शैली संग्रहीत करता है।
builder.ParagraphFormat.StyleName = "Quote";
builder.Writeln("Blockquote");

// शैली वंशानुक्रम के माध्यम से नेस्टेड स्तरों के लिए शैलियाँ बनाएँ।
Style quoteLevel2 = builder.Document.Styles.Add(StyleType.Paragraph, "Quote1");
builder.ParagraphFormat.Style = quoteLevel2;
builder.Document.Styles["Quote1"].BaseStyleName = "Quote";
builder.Writeln("1. Nested blockquote");
```

बधाई हो! अब आपने सीख लिया है कि .NET के लिए Aspose.Words के साथ उद्धरण सुविधा का उपयोग कैसे करें।


### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मार्कडाउन में उद्धरण क्या है?

उत्तर: मार्कडाउन में एक उद्धरण अन्य स्रोतों से पाठ के अंशों को उजागर करने या प्रसिद्ध उद्धरणों को संदर्भित करने का एक तरीका है।

#### प्रश्न: मार्कडाउन में उद्धरण चिह्नों का उपयोग कैसे करें?

उ: मार्कडाउन में एक उद्धरण का उपयोग करने के लिए, उद्धरण के पाठ को कोण कोष्ठक में संलग्न करें (`>`). उद्धरण की प्रत्येक पंक्ति एक शेवरॉन से शुरू होनी चाहिए।

#### प्रश्न: क्या मार्कडाउन उद्धरण विशेषताओं का समर्थन करता है?

उ: मार्कडाउन उद्धरण विशिष्ट विशेषताओं का समर्थन नहीं करते हैं। उन्हें केवल उद्धृत पाठ के स्वरूपण द्वारा हाइलाइट किया जाता है।

#### प्रश्न: क्या आप मार्कडाउन में उद्धरण एम्बेड कर सकते हैं?

उ: हां, कोण कोष्ठकों का एक अतिरिक्त स्तर जोड़कर मार्कडाउन में उद्धरण चिह्नों को घोंसला बनाना संभव है (`>`).
