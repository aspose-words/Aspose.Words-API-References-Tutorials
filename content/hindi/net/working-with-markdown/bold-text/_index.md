इस उदाहरण में, हम आपको Aspose.Words for .NET के साथ टेक्स्ट को बोल्ड करने का तरीका बताने जा रहे हैं। टेक्स्ट को बोल्ड करने से यह अधिक दिखाई देता है और इसे अधिक प्रमुखता मिलती है।

## चरण 1: दस्तावेज़ जनरेटर का उपयोग करना

सबसे पहले, हम अपने दस्तावेज़ में सामग्री जोड़ने के लिए दस्तावेज़ जनरेटर का उपयोग करेंगे।

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## चरण 2: टेक्स्ट को बोल्ड करें

 हम डॉक्यूमेंट बिल्डर की सेटिंग करके टेक्स्ट को बोल्ड कर सकते हैं`Font.Bold`संपत्ति को`true`.

```csharp
builder.Font.Bold = true;
```

## चरण 3: दस्तावेज़ में सामग्री जोड़ें

 अब हम दस्तावेज़ निर्माता विधियों का उपयोग करके दस्तावेज़ में सामग्री जोड़ सकते हैं, जैसे`Writeln`, जो पाठ की एक पंक्ति जोड़ता है.

```csharp
builder.Writeln("This text will be bold");
```

## .NET के लिए Aspose.Words का उपयोग करके बोल्ड टेक्स्ट के लिए उदाहरण स्रोत कोड


```csharp
// दस्तावेज़ में सामग्री जोड़ने के लिए दस्तावेज़ बिल्डर का उपयोग करें.
DocumentBuilder builder = new DocumentBuilder();

// पाठ को बोल्ड बनाएं.
builder.Font.Bold = true;
builder.Writeln("This text will be Bold");  
```

बधाई हो! अब आपने .NET के लिए Aspose.Words के साथ टेक्स्ट को बोल्ड करना सीख लिया है।


### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मैं Aspose.Words में टेक्स्ट को बोल्ड कैसे बना सकता हूँ?

 A: Aspose.Words में टेक्स्ट को बोल्ड करने के लिए, आप इसका उपयोग कर सकते हैं`Font.Bold` की संपत्ति`Run`ऑब्जेक्ट. आप इस प्रॉपर्टी को सेट कर सकते हैं`true` विशिष्ट टेक्स्ट को बोल्ड करने के लिए। उदाहरण के लिए, आप इसका उपयोग कर सकते हैं`run.Font.Bold=true` अंदर के पाठ को बोल्ड करने के लिए`Run` वस्तु।

#### प्रश्न: क्या एक ही पैराग्राफ में कई पाठों को बोल्ड करना संभव है?

 उत्तर: हां, आप एकाधिक टूल का उपयोग करके एक ही पैराग्राफ में टेक्स्ट के कई टुकड़ों को बोल्ड कर सकते हैं।`Run` ऑब्जेक्ट्स. आप कई बना सकते हैं`Run` ऑब्जेक्ट्स और सेट करें`Font.Bold`संपत्ति को`true` प्रत्येक ऑब्जेक्ट के लिए टेक्स्ट के वांछित भागों को बोल्ड करना। फिर आप उन्हें पैराग्राफ़ में जोड़ सकते हैं`Paragraph.AppendChild(run)` तरीका।

#### प्रश्न: क्या मैं Aspose.Words में किसी तालिका या सेल में मौजूद टेक्स्ट को बोल्ड कर सकता हूँ?

 उत्तर: हाँ, आप Aspose.Words में किसी टेबल या सेल में मौजूद टेक्स्ट को बोल्ड कर सकते हैं। आप उचित तरीकों का उपयोग करके अपने इच्छित सेल या पैराग्राफ़ पर नेविगेट कर सकते हैं और फिर बोल्ड फ़ॉर्मेटिंग लागू कर सकते हैं।`Font.Bold` की संपत्ति`Run` या`Paragraph` वस्तु।
