इस उदाहरण में, हम आपको बताएंगे कि .NET के लिए Aspose.Words का उपयोग करके स्ट्राइकथ्रू टेक्स्ट शैली को कैसे लागू किया जाए। स्ट्राइकथ्रू टेक्स्ट का उपयोग यह इंगित करने के लिए किया जाता है कि टेक्स्ट हटा दिया गया है या अब मान्य नहीं है।

## चरण 1: दस्तावेज़ जनरेटर का उपयोग करना

सबसे पहले, हम अपने दस्तावेज़ में सामग्री जोड़ने के लिए एक दस्तावेज़ जनरेटर का उपयोग करेंगे।

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## चरण 2: स्ट्राइकथ्रू टेक्स्ट शैली लागू करें

हम सेट करके स्ट्राइकथ्रू टेक्स्ट शैली को सक्षम करेंगे`StrikeThrough` की संपत्ति`Font` करने के लिए वस्तु`true`.

```csharp
builder.Font.StrikeThrough = true;
```

## चरण 3: स्ट्राइकथ्रू टेक्स्ट जोड़ें

 अब हम दस्तावेज़ जनरेटर का उपयोग करके स्ट्राइकथ्रू टेक्स्ट जोड़ सकते हैं`Writeln` तरीका।

```csharp
builder.Writeln("This text will be StrikeThrough");
```


### .NET के लिए Aspose.Words के साथ स्ट्राइकथ्रू टेक्स्ट के लिए उदाहरण स्रोत कोड

```csharp
// दस्तावेज़ में सामग्री जोड़ने के लिए दस्तावेज़ निर्माता का उपयोग करें।
DocumentBuilder builder = new DocumentBuilder();

// टेक्स्ट को स्ट्राइकथ्रू बनाएं.
builder.Font.StrikeThrough = true;
builder.Writeln("This text will be StrikeThrough");
```

बधाई हो! अब आपने सीख लिया है कि .NET के लिए Aspose.Words के साथ स्ट्राइकथ्रू टेक्स्ट शैली कैसे लागू करें।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मैं Aspose.Words में स्ट्राइकथ्रू टेक्स्ट कैसे जोड़ सकता हूँ?

 उ: Aspose.Words में स्ट्राइकथ्रू टेक्स्ट जोड़ने के लिए, आप इसका उपयोग कर सकते हैं`Font.StrikeThrough` की संपत्ति`Run`वस्तु। आप इस प्रॉपर्टी को यहां सेट कर सकते हैं`true` विशिष्ट टेक्स्ट में स्ट्राइकथ्रू टेक्स्ट जोड़ने के लिए। उदाहरण के लिए, आप उपयोग कर सकते हैं`run.Font.StrikeThrough=true` में स्ट्राइकथ्रू टेक्स्ट जोड़ने के लिए`Run` वस्तु।

#### प्रश्न: क्या एक ही पैराग्राफ में पाठ के कई टुकड़ों में स्ट्राइकथ्रू पाठ जोड़ना संभव है?

 उ: हां, आप मल्टीपल का उपयोग करके एक ही पैराग्राफ में टेक्स्ट के कई हिस्सों में स्ट्राइकथ्रू टेक्स्ट जोड़ सकते हैं।`Run` वस्तुएं. आप एकाधिक बना सकते हैं`Run` ऑब्जेक्ट्स और सेट करें`Font.StrikeThrough`संपत्ति को`true` प्रत्येक ऑब्जेक्ट के लिए स्ट्राइकथ्रू टेक्स्ट को वांछित टेक्स्ट भागों में जोड़ें। फिर आप उनका उपयोग करके पैराग्राफ में जोड़ सकते हैं`Paragraph.AppendChild(run)` तरीका।

#### प्रश्न: क्या मैं Aspose.Words में किसी तालिका या सेल में मौजूद टेक्स्ट में स्ट्राइकथ्रू टेक्स्ट जोड़ सकता हूँ?

 उत्तर: हां, आप Aspose.Words में किसी तालिका या सेल में मौजूद टेक्स्ट में स्ट्राइकथ्रू टेक्स्ट जोड़ सकते हैं। आप उपयुक्त तरीकों का उपयोग करके इच्छित सेल या पैराग्राफ पर जा सकते हैं और फिर स्ट्राइकथ्रू टेक्स्ट फ़ॉर्मेटिंग लागू कर सकते हैं`Font.StrikeThrough` की संपत्ति`Run` या`Paragraph` वस्तु।
