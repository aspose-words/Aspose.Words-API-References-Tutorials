## परिचय

Aspose.Words for .NET का उपयोग करके टेक्स्ट पर स्ट्राइकथ्रू फ़ॉर्मेटिंग लागू करने के तरीके पर इस विस्तृत गाइड में आपका स्वागत है। यदि आप अपने दस्तावेज़ प्रसंस्करण कौशल को बढ़ाना चाहते हैं और अपने टेक्स्ट में एक अनूठा स्पर्श जोड़ना चाहते हैं, तो आप सही जगह पर हैं। आइए शुरू करते हैं!

आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

## नामस्थान आयात करें

आरंभ करने के लिए, आपको आवश्यक नामस्थान आयात करने होंगे। ये Aspose.Words लाइब्रेरी और इसकी विशेषताओं तक पहुँचने के लिए आवश्यक हैं।

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Saving;
```

## चरण 1: डॉक्यूमेंटबिल्डर को आरंभ करें

`DocumentBuilder` क्लास Aspose.Words में एक शक्तिशाली उपकरण है जो आपको आसानी से अपने दस्तावेज़ में सामग्री जोड़ने की अनुमति देता है।

```csharp
// एक डॉक्यूमेंटबिल्डर आरंभ करें.
DocumentBuilder builder = new DocumentBuilder();
```

## चरण 2: स्ट्राइकथ्रू प्रॉपर्टी सेट करें

अब, आइए अपने टेक्स्ट पर स्ट्राइकथ्रू प्रॉपर्टी लागू करें। इसमें सेटिंग करना शामिल है`StrikeThrough` की संपत्ति`Font` करने के लिए वस्तु`true`.

```csharp
// पाठ को स्ट्राइकथ्रू बनाएं.
builder.Font.StrikeThrough = true;
```

## चरण 3: स्ट्राइकथ्रू के साथ टेक्स्ट लिखें

 स्ट्राइकथ्रू प्रॉपर्टी सेट होने के बाद, अब हम अपना टेक्स्ट जोड़ सकते हैं।`Writeln` विधि दस्तावेज़ में पाठ जोड़ देगी.

```csharp
// स्ट्राइकथ्रू के साथ पाठ लिखें.
builder.Writeln("This text will be StrikeThrough");
```
