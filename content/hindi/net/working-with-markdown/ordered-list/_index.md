इस उदाहरण में, हम बताएंगे कि .NET के लिए Aspose.Words के साथ ऑर्डर की गई सूची कार्यक्षमता का उपयोग कैसे करें। ऑर्डर की गई सूची आपको संख्याओं के साथ क्रमिक रूप से आइटम व्यवस्थित करने की अनुमति देती है।

## चरण 1: दस्तावेज़ जनरेटर का उपयोग करना

सबसे पहले, हम एक नया दस्तावेज़ बनाने के लिए दस्तावेज़ जनरेटर का उपयोग करेंगे।

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## चरण 2: क्रमबद्ध सूची प्रारूप लागू करना

 हम दस्तावेज़ बिल्डर का उपयोग करके क्रमबद्ध सूची प्रारूप लागू करेंगे`ApplyBulletDefault`विधि। हम सूची स्तर पर जाकर और इच्छित प्रारूप सेट करके नंबरिंग प्रारूप को भी अनुकूलित कर सकते हैं।

```csharp
builder.ListFormat.ApplyBulletDefault();
builder.ListFormat.List.ListLevels[0].NumberFormat = $"{(char) 0}.";
builder.ListFormat.List.ListLevels[1].NumberFormat = $"{(char) 1}.";
```

## चरण 3: सूची में आइटम जोड़ना

 हम दस्तावेज़ जनरेटर का उपयोग करके सूची में आइटम जोड़ सकते हैं`Writeln` तरीका।

```csharp
builder. Writen("Element 1");
builder. Writen("Element 2");
```

## चरण 4: सूची को इंडेंट करें

 हम दस्तावेज़ जनरेटर का उपयोग करके सूची को इंडेंट कर सकते हैं`ListIndent` तरीका।

```csharp
builder.ListFormat.ListIndent();
builder.Writeln("Item 2a");
builder.Writeln("Item 2b");
```

## चरण 5: दस्तावेज़ को सहेजना

अंततः, हम दस्तावेज़ को वांछित प्रारूप में सहेज सकते हैं।

### .NET के लिए Aspose.Words के साथ क्रमबद्ध सूची के लिए उदाहरण स्रोत कोड

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

builder.ListFormat.ApplyBulletDefault();
builder.ListFormat.List.ListLevels[0].NumberFormat = $"{(char) 0}.";
builder.ListFormat.List.ListLevels[1].NumberFormat = $"{(char) 1}.";

builder.Writeln("Item 1");
builder.Writeln("Item 2");

builder.ListFormat.ListIndent();

builder.Writeln("Item 2a");
builder.Writeln("Item 2b");
```

बधाई हो! अब आप सीख चुके हैं कि .NET के लिए Aspose.Words के साथ ऑर्डर की गई सूची सुविधा का उपयोग कैसे करें।


### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मार्कडाउन में क्रमबद्ध सूची कैसे बनाएं?

उत्तर: मार्कडाउन में एक क्रमबद्ध सूची बनाने के लिए, प्रत्येक सूची आइटम को एक संख्या से शुरू करें जिसके बाद एक अवधि हो (`1.`, `2.`, `3.`), उसके बाद एक स्थान।

#### प्रश्न: क्या हम मार्कडाउन में क्रमबद्ध सूचियाँ बना सकते हैं?

उत्तर: हां, प्रत्येक नेस्टेड सूची आइटम के सामने चार ऑफसेट रिक्त स्थान जोड़कर मार्कडाउन में क्रमबद्ध सूचियों को नेस्ट करना संभव है।

#### प्रश्न: क्रमबद्ध सूचियों की संख्या को कैसे अनुकूलित करें?

उत्तर: मानक मार्कडाउन में, क्रमबद्ध सूची क्रमांकन स्वचालित रूप से उत्पन्न होता है। हालाँकि, कुछ मार्कडाउन संपादक आपको विशिष्ट एक्सटेंशन का उपयोग करके इसे अनुकूलित करने की अनुमति देते हैं।

#### प्रश्न: क्या मार्कडाउन में क्रमबद्ध सूचियाँ इंडेंटेशन का समर्थन करती हैं?

उत्तर: हां, मार्कडाउन में ऑर्डर की गई सूचियाँ इंडेंटेशन का समर्थन करती हैं। आप स्पेस या टैब का उपयोग करके लेफ्ट शिफ्ट जोड़ सकते हैं।

#### प्रश्न: क्या सूची आइटम में लिंक या इनलाइन टेक्स्ट जोड़ा जा सकता है?

उत्तर: हां, आप उचित मार्कडाउन सिंटैक्स का उपयोग करके सूची आइटम में लिंक या इनलाइन टेक्स्ट जोड़ सकते हैं।
