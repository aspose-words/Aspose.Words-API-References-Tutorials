इस ट्यूटोरियल में, हम आपको बताने जा रहे हैं कि .NET के लिए Aspose.Words के साथ बुलेटेड सूची कैसे बनाएं। बुलेटेड सूची का उपयोग क्रमांकन का उपयोग किए बिना वस्तुओं को सूचीबद्ध करने के लिए किया जाता है।

## चरण 1: दस्तावेज़ जनरेटर का उपयोग करना

सबसे पहले, हम अपने दस्तावेज़ में सामग्री जोड़ने के लिए एक दस्तावेज़ जनरेटर का उपयोग करेंगे।

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## चरण 2: डिफ़ॉल्ट बुलेटेड सूची लागू करना

 हम दस्तावेज़ बिल्डर का उपयोग करके एक डिफ़ॉल्ट बुलेटेड सूची लागू कर सकते हैं`ApplyBulletDefault` तरीका।

```csharp
builder.ListFormat.ApplyBulletDefault();
```

## चरण 3: बुलेट प्रारूप को अनुकूलित करना

हम के गुणों तक पहुंच कर बुलेट प्रारूप को अनुकूलित कर सकते हैं`ListFormat.List.ListLevels[0]`. इस उदाहरण में, हम डैश "-" का उपयोग बुलेट के रूप में करते हैं।

```csharp
builder.ListFormat.List.ListLevels[0].NumberFormat = "-";
```

## चरण 4: सूची में आइटम जोड़ना

 अब हम दस्तावेज़ बिल्डर का उपयोग करके बुलेटेड सूची में आइटम जोड़ सकते हैं`Writeln` तरीका।

```csharp
builder. Writen("Element 1");
builder. Writen("Element 2");
```

## चरण 5: सूची से इंडेंटेशन हटाना

 यदि हम एक उपसूची बनाना चाहते हैं, तो हम इसका उपयोग करके इंडेंटेशन बढ़ा सकते हैं`ListFormat.ListIndent()` तरीका। इस उदाहरण में, हम आइटम 2ए और 2बी में एक उपसूची जोड़ रहे हैं।

```csharp
builder.ListFormat.ListIndent();
builder. Writeln("Element 2a");
builder.Writeln("Element 2b");
```
### .NET के लिए Aspose.Words का उपयोग करके बुलेटेड सूची के लिए उदाहरण स्रोत कोड


```csharp
// दस्तावेज़ में सामग्री जोड़ने के लिए दस्तावेज़ निर्माता का उपयोग करें।
DocumentBuilder builder = new DocumentBuilder();

builder.ListFormat.ApplyBulletDefault();
builder.ListFormat.List.ListLevels[0].NumberFormat = "-";

builder.Writeln("Item 1");
builder.Writeln("Item 2");

builder.ListFormat.ListIndent();

builder.Writeln("Item 2a");
builder.Writeln("Item 2b");
```

बधाई हो! अब आपने सीख लिया है कि .NET के लिए Aspose.Words के साथ बुलेटेड सूची कैसे बनाई जाती है।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मार्कडाउन में बुलेटेड सूची कैसे बनाएं?

उ: मार्कडाउन में बुलेटेड सूची बनाने के लिए, प्रत्येक सूची आइटम को बुलेट प्रतीक के साथ प्रारंभ करें (`-`, `*` , या`+`), उसके बाद एक स्थान।

#### प्रश्न: क्या आप मार्कडाउन में बुलेटेड सूचियाँ घोंसला बना सकते हैं?

उ: हां, प्रत्येक नेस्टेड सूची आइटम के सामने चार ऑफसेट रिक्त स्थान जोड़कर मार्कडाउन में बुलेटेड सूचियों को नेस्ट करना संभव है।

#### प्रश्न: बुलेट प्रतीकों को कैसे अनुकूलित करें?

ए: मानक मार्कडाउन में, बुलेट प्रतीक पूर्वनिर्धारित होते हैं। हालाँकि, कुछ मार्कडाउन संपादक आपको विशिष्ट एक्सटेंशन का उपयोग करके उन्हें अनुकूलित करने की अनुमति देते हैं।

#### प्रश्न: क्या मार्कडाउन में बुलेटेड सूचियाँ इंडेंटेशन का समर्थन करती हैं?

उ: हाँ, मार्कडाउन में बुलेटेड सूचियाँ इंडेंटेशन का समर्थन करती हैं। आप रिक्त स्थान या टैब का उपयोग करके बाईं ओर शिफ्ट जोड़ सकते हैं।

#### प्रश्न: क्या सूची आइटम में लिंक या इनलाइन टेक्स्ट जोड़ा जा सकता है?

उ: हाँ, आप उपयुक्त मार्कडाउन सिंटैक्स का उपयोग करके आइटमों को सूचीबद्ध करने के लिए लिंक या इनलाइन टेक्स्ट जोड़ सकते हैं।
