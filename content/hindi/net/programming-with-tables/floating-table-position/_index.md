इस ट्यूटोरियल में, हम सीखेंगे कि किसी Word दस्तावेज़ में तालिका को फ़्लोटिंग स्थिति में रखने के लिए .NET के लिए Aspose.Words का उपयोग कैसे करें। हम कोड को समझने और इस सुविधा को लागू करने के लिए चरण दर चरण मार्गदर्शिका का पालन करेंगे। इस ट्यूटोरियल के अंत में, आप प्रोग्रामेटिक रूप से अपने वर्ड दस्तावेज़ों में फ़्लोटिंग टेबल की स्थिति और संरेखण को नियंत्रित करने में सक्षम होंगे।

## चरण 1: प्रोजेक्ट सेटअप
1. विज़ुअल स्टूडियो लॉन्च करें और एक नया C# प्रोजेक्ट बनाएं।
2. .NET लाइब्रेरी के लिए Aspose.Words का संदर्भ जोड़ें।

## चरण 2: दस्तावेज़ लोड करना और तालिका तक पहुँचना
तालिका के साथ वर्ड प्रोसेसिंग शुरू करने के लिए, हमें उस दस्तावेज़ को लोड करना होगा जिसमें यह शामिल है और इसे एक्सेस करना होगा। इन चरणों का पालन करें:

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// दस्तावेज़ लोड करें
Document doc = new Document(dataDir + "Table wrapped by text.docx");

// सरणी तक पहुंच
Table table = doc.FirstSection.Body.Tables[0];
```

अपने दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ "आपकी दस्तावेज़ निर्देशिका" को बदलना सुनिश्चित करें। साथ ही, सुनिश्चित करें कि दस्तावेज़ में एक तालिका है जो फ़्लोटिंग स्थिति में स्थित होगी।

## चरण 3: फ्लोटिंग बोर्ड की स्थिति निर्धारित करना
इसके बाद, हम .NET के लिए Aspose.Words द्वारा प्रदान किए गए गुणों का उपयोग करके तालिका को फ़्लोटिंग स्थिति में रखेंगे। निम्नलिखित कोड का प्रयोग करें:

```csharp
// फ़्लोटिंग टेबल की स्थिति निर्धारित करना
table. AbsoluteHorizontalDistance = 10;
table. RelativeVerticalAlignment = VerticalAlignment. Center;
```

 यहां हम इसका उपयोग करते हैं`AbsoluteHorizontalDistance` पृष्ठ के बाएँ किनारे से तालिका की पूर्ण क्षैतिज दूरी निर्धारित करने की संपत्ति। हम भी उपयोग करते हैं`RelativeVerticalAlignment` आसपास की सामग्री के साथ तालिका के सापेक्ष लंबवत संरेखण को सेट करने की संपत्ति।

## चरण 4: संशोधित दस्तावेज़ को सहेजना
अंत में, हमें संशोधित दस्तावेज़ को फ्लोटिंग स्थिति में स्थित तालिका के साथ सहेजने की आवश्यकता है। निम्नलिखित कोड का प्रयोग करें:

```csharp
// संशोधित दस्तावेज़ सहेजें
doc.Save(dataDir + "WorkingWithTables.FloatingTablePosition.docx");
```

आउटपुट दस्तावेज़ के लिए सही पथ और फ़ाइल नाम निर्दिष्ट करना सुनिश्चित करें।

### .NET के लिए Aspose.Words का उपयोग करके फ़्लोटिंग टेबल स्थिति के लिए नमूना स्रोत कोड 

```csharp
	// आपकी दस्तावेज़ निर्देशिका का पथ
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document(dataDir + "Table wrapped by text.docx");
	Table table = doc.FirstSection.Body.Tables[0];
	table.AbsoluteHorizontalDistance = 10;
	table.RelativeVerticalAlignment = VerticalAlignment.Center;
	doc.Save(dataDir + "WorkingWithTables.FloatingTablePosition.docx");
```

## निष्कर्ष
इस ट्यूटोरियल में, हमने सीखा कि .NET के लिए Aspose.Words का उपयोग करके किसी Word दस्तावेज़ में तालिका को फ़्लोटिंग स्थिति में कैसे रखा जाए। इस चरण-दर-चरण मार्गदर्शिका का पालन करके और दिए गए C# कोड को लागू करके, आप प्रोग्रामेटिक रूप से अपने Word दस्तावेज़ों में फ़्लोटिंग तालिकाओं की स्थिति और संरेखण को नियंत्रित कर सकते हैं।
