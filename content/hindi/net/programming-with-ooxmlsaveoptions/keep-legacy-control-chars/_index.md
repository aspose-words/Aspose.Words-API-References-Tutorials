इस ट्यूटोरियल में, हम Aspose.Words for .NET का उपयोग करके दस्तावेज़ सहेजते समय विरासत नियंत्रण वर्णों को संरक्षित करने के लिए प्रदान किए गए C# स्रोत कोड का पता लगाएंगे। यह सुविधा आपको दस्तावेज़ को परिवर्तित या सहेजते समय विशेष नियंत्रण वर्णों को संरक्षित करने की अनुमति देती है।

## चरण 1: वातावरण की स्थापना

आरंभ करने से पहले, सुनिश्चित करें कि आपने .NET के लिए Aspose.Words के साथ अपना विकास वातावरण सेट अप कर लिया है। सुनिश्चित करें कि आपने आवश्यक संदर्भ जोड़ दिए हैं और उचित नामस्थान आयात कर लिए हैं।

## चरण 2: दस्तावेज़ लोड करना

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

Document doc = new Document(dataDir + "Legacy control character.doc");
```

 इस चरण में, हम दस्तावेज़ को लोड करते हैं`Document` विधि और विरासत में मिले नियंत्रण वर्णों वाली फ़ाइल का पथ पास करना।

## चरण 3: OOXML बैकअप विकल्प कॉन्फ़िगर करना

```csharp
OoxmlSaveOptions saveOptions = new OoxmlSaveOptions(SaveFormat.FlatOpc) { KeepLegacyControlChars = true };
```

 इस चरण में, हम एक नया बनाकर OOXML सेव विकल्प कॉन्फ़िगर करते हैं`OoxmlSaveOptions` ऑब्जेक्ट। हम वांछित सेव फ़ॉर्मेट निर्दिष्ट करते हैं (यहाँ,`FlatOpc` ) और सक्षम करें`KeepLegacyControlChars` विरासत नियंत्रण वर्ण रखने का विकल्प.

## चरण 4: विरासत नियंत्रण वर्णों के साथ दस्तावेज़ को सहेजना

```csharp
doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.KeepLegacyControlChars.docx", saveOptions);
```

 इस अंतिम चरण में, हम दस्तावेज़ को सहेजते हैं`Save` विधि और आउटपुट फ़ाइल के पथ को पास करना`.docx` एक्सटेंशन, निर्दिष्ट सहेजें विकल्पों के साथ।

अब आप दस्तावेज़ सहेजते समय विरासत नियंत्रण वर्णों को संरक्षित करने के लिए स्रोत कोड चला सकते हैं। परिणामी फ़ाइल "WorkingWithOoxmlSaveOptions.KeepLegacyControlChars.docx" नाम से निर्दिष्ट निर्देशिका में सहेजी जाएगी।

### .NET के लिए Aspose.Words का उपयोग करके लीगेसी नियंत्रण वर्णों को रखने के लिए नमूना स्रोत कोड 
```csharp

// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY"; 

Document doc = new Document(dataDir + "Legacy control character.doc");

OoxmlSaveOptions saveOptions = new OoxmlSaveOptions(SaveFormat.FlatOpc) { KeepLegacyControlChars = true };

doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.KeepLegacyControlChars.docx", saveOptions);
            
        
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने Aspose.Words for .NET का उपयोग करके दस्तावेज़ सहेजते समय लीगेसी नियंत्रण वर्णों को संरक्षित करने की कार्यक्षमता का पता लगाया। हमने सीखा है कि उन विशेष वर्णों को कैसे संरक्षित किया जाए जो उचित दस्तावेज़ स्वरूपण या प्रदर्शन के लिए महत्वपूर्ण हो सकते हैं।

 विरासत नियंत्रण वर्णों को संरक्षित करना विशेष रूप से तब उपयोगी होता है जब वर्ड प्रोसेसिंग में ऐसे दस्तावेज़ होते हैं जो पुराने या विशिष्ट सुविधाओं का उपयोग करते हैं, जैसे कि विशेष नियंत्रण वर्ण।`KeepLegacyControlChars` दस्तावेज़ को सहेजते समय विकल्प का उपयोग करके, आप सुनिश्चित करते हैं कि ये वर्ण संरक्षित हैं।

Aspose.Words for .NET आपके दस्तावेज़ हेरफेर की ज़रूरतों को पूरा करने के लिए लचीले और शक्तिशाली बैकअप विकल्पों की एक श्रृंखला प्रदान करता है। उपयुक्त विकल्पों का उपयोग करके, आप अपने दस्तावेज़ों की विशिष्ट विशेषताओं को संरक्षित करने के लिए बैकअप प्रक्रिया को अनुकूलित कर सकते हैं।

अपने दस्तावेज़ों में विरासत नियंत्रण वर्णों की अखंडता और संरक्षण सुनिश्चित करने के लिए अपने Aspose.Words for .NET प्रोजेक्ट्स में इस कार्यक्षमता को शामिल करने के लिए स्वतंत्र महसूस करें।
