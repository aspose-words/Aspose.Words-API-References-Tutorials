इस ट्यूटोरियल में, हम .NET के लिए Aspose.Words का उपयोग करके किसी दस्तावेज़ को सहेजते समय विरासत नियंत्रण वर्णों को संरक्षित करने के लिए दिए गए C# स्रोत कोड का पता लगाएंगे। यह सुविधा आपको किसी दस्तावेज़ को परिवर्तित या सहेजते समय विशेष नियंत्रण वर्णों को संरक्षित करने की अनुमति देती है।

## चरण 1: वातावरण स्थापित करना

शुरू करने से पहले, सुनिश्चित करें कि आपने .NET के लिए Aspose.Words के साथ अपना विकास वातावरण स्थापित कर लिया है। सुनिश्चित करें कि आपने आवश्यक संदर्भ जोड़ दिए हैं और उचित नामस्थान आयात कर लिए हैं।

## चरण 2: दस्तावेज़ लोड करना

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

Document doc = new Document(dataDir + "Legacy control character.doc");
```

 इस चरण में, हम का उपयोग करके दस्तावेज़ को लोड करते हैं`Document` विधि और विरासत में मिले नियंत्रण वर्णों वाली फ़ाइल का पथ पारित करना।

## चरण 3: OOXML बैकअप विकल्पों को कॉन्फ़िगर करना

```csharp
OoxmlSaveOptions saveOptions = new OoxmlSaveOptions(SaveFormat.FlatOpc) { KeepLegacyControlChars = true };
```

 इस चरण में, हम एक नया बनाकर OOXML सेव विकल्पों को कॉन्फ़िगर करते हैं`OoxmlSaveOptions` वस्तु। हम वांछित सेव प्रारूप निर्दिष्ट करते हैं (यहां,`FlatOpc` ) और सक्षम करें`KeepLegacyControlChars` विरासत नियंत्रण वर्ण रखने का विकल्प।

## चरण 4: दस्तावेज़ को लीगेसी नियंत्रण वर्णों के साथ सहेजना

```csharp
doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.KeepLegacyControlChars.docx", saveOptions);
```

 इस अंतिम चरण में, हम दस्तावेज़ को का उपयोग करके सहेजते हैं`Save` विधि और आउटपुट फ़ाइल के लिए पथ पास करना`.docx` एक्सटेंशन, निर्दिष्ट सेव विकल्पों के साथ।

अब आप किसी दस्तावेज़ को सहेजते समय विरासत नियंत्रण वर्णों को संरक्षित करने के लिए स्रोत कोड चला सकते हैं। परिणामी फ़ाइल निर्दिष्ट निर्देशिका में "WorkingWithOoxmlSaveOptions.KeepLegacyControlChars.docx" नाम से सहेजी जाएगी।

### .NET के लिए Aspose.Words का उपयोग करके विरासत नियंत्रण वर्ण रखने के लिए नमूना स्रोत कोड 
```csharp

// आपकी दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY"; 

Document doc = new Document(dataDir + "Legacy control character.doc");

OoxmlSaveOptions saveOptions = new OoxmlSaveOptions(SaveFormat.FlatOpc) { KeepLegacyControlChars = true };

doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.KeepLegacyControlChars.docx", saveOptions);
            
        
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने .NET के लिए Aspose.Words का उपयोग करके दस्तावेज़ को सहेजते समय विरासत नियंत्रण वर्णों को संरक्षित करने की कार्यक्षमता का पता लगाया। हमने सीखा है कि उन विशेष वर्णों को कैसे संरक्षित किया जाए जो उचित दस्तावेज़ स्वरूपण या प्रदर्शन के लिए महत्वपूर्ण हो सकते हैं।

 विरासत नियंत्रण वर्णों को संरक्षित करना विशेष रूप से तब उपयोगी होता है जब दस्तावेज़ों को ऐसे दस्तावेज़ों के साथ संसाधित किया जाता है जो पुराने या विशिष्ट सुविधाओं, जैसे विशेष नियंत्रण वर्णों का उपयोग करते हैं। सक्षम करके`KeepLegacyControlChars` विकल्प दस्तावेज़ को सहेजते समय, आप सुनिश्चित करते हैं कि ये अक्षर संरक्षित हैं।

.NET के लिए Aspose.Words आपकी दस्तावेज़ हेरफेर आवश्यकताओं को पूरा करने के लिए लचीले और शक्तिशाली बैकअप विकल्पों की एक श्रृंखला प्रदान करता है। उपयुक्त विकल्पों का उपयोग करके, आप अपने दस्तावेज़ों की विशिष्ट विशेषताओं को संरक्षित करने के लिए बैकअप प्रक्रिया को अनुकूलित कर सकते हैं।

अपने दस्तावेज़ों में विरासत नियंत्रण वर्णों की अखंडता और संरक्षण सुनिश्चित करने के लिए .NET परियोजनाओं के लिए इस कार्यक्षमता को अपने Aspose.Words में शामिल करने के लिए स्वतंत्र महसूस करें।
