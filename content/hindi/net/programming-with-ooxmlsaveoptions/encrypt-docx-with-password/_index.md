इस ट्यूटोरियल में, हम .NET के लिए Aspose.Words का उपयोग करके पासवर्ड के साथ DOCX फ़ाइल को एन्क्रिप्ट करने के लिए प्रदान किए गए C# स्रोत कोड का पता लगाएंगे। यह सुविधा आपको अपने दस्तावेज़ को केवल निर्दिष्ट पासवर्ड के साथ सुलभ बनाकर सुरक्षित करने की अनुमति देती है।

## चरण 1: वातावरण की स्थापना

आरंभ करने से पहले, सुनिश्चित करें कि आपने .NET के लिए Aspose.Words के साथ अपना विकास वातावरण सेट अप कर लिया है। सुनिश्चित करें कि आपने आवश्यक संदर्भ जोड़ दिए हैं और उचित नामस्थान आयात कर लिए हैं।

## चरण 2: दस्तावेज़ लोड करना

```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

Document doc = new Document(dataDir + "Document.docx");
```

 इस चरण में, हम दस्तावेज़ को लोड करते हैं`Document` विधि और लोड करने के लिए DOCX फ़ाइल का पथ पास करना।

## चरण 3: OOXML बैकअप विकल्प कॉन्फ़िगर करना

```csharp
OoxmlSaveOptions saveOptions = new OoxmlSaveOptions { Password = "password" };
```

 इस चरण में, हम एक नया बनाकर OOXML सेव विकल्प कॉन्फ़िगर करते हैं`OoxmlSaveOptions` ऑब्जेक्ट। हम दस्तावेज़ को एन्क्रिप्ट करने के लिए वांछित पासवर्ड निर्दिष्ट करते हैं`Password` प्रॉपर्टी को अपने कस्टम पासवर्ड में जोड़ें.

## चरण 4: दस्तावेज़ को पासवर्ड से एन्क्रिप्ट करना

```csharp
doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.EncryptDocxWithPassword.docx", saveOptions);
```

 इस अंतिम चरण में, हम दस्तावेज़ को सहेजते हैं`Save` विधि और आउटपुट फ़ाइल के पथ को पास करना`.docx` एक्सटेंशन, निर्दिष्ट सहेजें विकल्पों के साथ।

अब आप अपने DOCX दस्तावेज़ को पासवर्ड से एन्क्रिप्ट करने के लिए स्रोत कोड चला सकते हैं। परिणामी फ़ाइल निर्दिष्ट निर्देशिका में "WorkingWithOoxmlSaveOptions.EncryptDocxWithPassword.docx" नाम से सहेजी जाएगी। अपना पासवर्ड सुरक्षित रखना सुनिश्चित करें, क्योंकि एन्क्रिप्ट किए गए दस्तावेज़ को खोलने के लिए इसकी आवश्यकता होगी।

### .NET के लिए Aspose.Words का उपयोग करके पासवर्ड के साथ Docx एन्क्रिप्ट करने के लिए नमूना स्रोत कोड 

```csharp

// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";  

Document doc = new Document(dataDir + "Document.docx");

OoxmlSaveOptions saveOptions = new OoxmlSaveOptions { Password = "password" };

doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.EncryptDocxWithPassword.docx", saveOptions);
            
        
```

संवेदनशील जानकारी की सुरक्षा के लिए दस्तावेज़ एन्क्रिप्शन एक आवश्यक सुरक्षा उपाय है। Aspose.Words for .NET की बदौलत, हम इस कार्यक्षमता को आसानी से अपने अनुप्रयोगों में जोड़ सकते हैं।

दिए गए चरणों का पालन करके, आप अपने Aspose.Words for .NET प्रोजेक्ट्स में पासवर्ड एन्क्रिप्शन को एकीकृत कर सकते हैं और अपने दस्तावेज़ों की गोपनीयता सुनिश्चित कर सकते हैं।

अपने अनुप्रयोगों को उन्नत दस्तावेज़ हेरफेर सुविधाओं के साथ समृद्ध करने के लिए .NET के लिए Aspose.Words द्वारा प्रदान की गई अन्य सुविधाओं के साथ प्रयोग करने के लिए स्वतंत्र महसूस करें।
