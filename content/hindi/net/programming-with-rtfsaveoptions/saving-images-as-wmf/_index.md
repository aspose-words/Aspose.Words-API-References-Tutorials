## नामस्थान आयात करें

सबसे पहले, आइए आवश्यक नेमस्पेस को आयात करें। यह Aspose.Words क्लासेस और मेथड्स तक पहुँचने के लिए महत्वपूर्ण है जिनका हम उपयोग करेंगे।

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Saving;
```

ठीक है, अब हम मज़ेदार भाग पर आ रहे हैं। चलिए इस प्रक्रिया को आसान चरणों में विभाजित करते हैं।

## चरण 1: अपना दस्तावेज़ लोड करें

सबसे पहले, आपको उस दस्तावेज़ को लोड करना होगा जिसमें वे छवियां हैं जिन्हें आप WMF के रूप में सहेजना चाहते हैं। 

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Document.docx");
```

 स्पष्टीकरण: इस चरण में, हम वह निर्देशिका निर्दिष्ट करते हैं जहाँ आपका दस्तावेज़ स्थित है। फिर, हम दस्तावेज़ को लोड करने के लिए निम्न का उपयोग करते हैं:`Document` Aspose.Words द्वारा प्रदान की गई क्लास। आसान है, है ना?

## चरण 2: सहेजें विकल्प कॉन्फ़िगर करें

इसके बाद, हमें यह सुनिश्चित करने के लिए सेव विकल्पों को कॉन्फ़िगर करना होगा कि छवियाँ WMF के रूप में सेव की गई हैं।

```csharp
RtfSaveOptions saveOptions = new RtfSaveOptions { SaveImagesAsWmf = true };
```

 स्पष्टीकरण: यहाँ, हम एक उदाहरण बनाते हैं`RtfSaveOptions` और सेट करें`SaveImagesAsWmf`संपत्ति को`true`यह Aspose.Words को दस्तावेज़ सहेजे जाने पर छवियों को WMF के रूप में सहेजने के लिए कहता है।

## चरण 3: दस्तावेज़ सहेजें

अंत में, निर्दिष्ट सहेजें विकल्पों के साथ दस्तावेज़ को सहेजने का समय आ गया है।

```csharp
doc.Save(dataDir + "WorkingWithRtfSaveOptions.SavingImagesAsWmf.rtf", saveOptions);
```

 स्पष्टीकरण: इस चरण में, हम उपयोग करते हैं`Save` की विधि`Document` दस्तावेज़ को सहेजने के लिए क्लास का उपयोग करें। हम फ़ाइल पथ और`saveOptions` पैरामीटर के रूप में। यह सुनिश्चित करता है कि छवियाँ WMF के रूप में सहेजी गई हैं।
