जब आप किसी C# एप्लीकेशन में Word डॉक्यूमेंट को OpenDocument Text (ODT) फॉर्मेट में कन्वर्ट करते हैं, तो आप मापनीय फॉर्मेटिंग और कंटेंट प्रॉपर्टी के लिए इस्तेमाल की जाने वाली माप की इकाई को निर्दिष्ट करना चाह सकते हैं। .NET के लिए Aspose.Words लाइब्रेरी के साथ, आप OdtSaveOptions सेव ऑप्शन का उपयोग करके आसानी से इस कार्यक्षमता को निर्दिष्ट कर सकते हैं। इस चरण-दर-चरण मार्गदर्शिका में, हम आपको OdtSaveOptions का उपयोग करके माप की इकाई निर्दिष्ट करके Word डॉक्यूमेंट को ODT में बदलने के लिए Aspose.Words for .NET C# स्रोत कोड का उपयोग करने का तरीका बताएंगे।

## Aspose.Words लाइब्रेरी को समझना

कोड में गोता लगाने से पहले, .NET के लिए Aspose.Words लाइब्रेरी को समझना महत्वपूर्ण है। Aspose.Words .NET सहित विभिन्न प्लेटफ़ॉर्म में Word दस्तावेज़ बनाने, संपादित करने, परिवर्तित करने और सुरक्षित करने के लिए एक शक्तिशाली लाइब्रेरी है। यह दस्तावेज़ों में हेरफेर करने के लिए कई सुविधाएँ प्रदान करता है, जैसे कि टेक्स्ट डालना, फ़ॉर्मेटिंग बदलना, अनुभाग जोड़ना और बहुत कुछ।

## Word दस्तावेज़ लोड करना

पहला चरण उस Word दस्तावेज़ को लोड करना है जिसे आप ODT में बदलना चाहते हैं। स्रोत फ़ाइल से दस्तावेज़ लोड करने के लिए Document क्लास का उपयोग करें। यहाँ एक उदाहरण दिया गया है:

```csharp
Document doc = new Document(dataDir + "Document.docx");
```

इस उदाहरण में, हम दस्तावेज़ निर्देशिका में स्थित दस्तावेज़ "Document.docx" को लोड करते हैं।

## बैकअप विकल्प कॉन्फ़िगर करना

अगला चरण ODT में कनवर्ट करने के लिए बैकअप विकल्पों को कॉन्फ़िगर करना है। OdtSaveOptions क्लास का उपयोग करें और MeasureUnit प्रॉपर्टी को वांछित मान पर सेट करें। उदाहरण के लिए, यदि आप माप की इकाई के रूप में इंच का उपयोग करना चाहते हैं, तो MeasureUnit को OdtSaveMeasureUnit.Inches पर सेट करें। इसे करने का तरीका यहां बताया गया है:

```csharp
OdtSaveOptions saveOptions = new OdtSaveOptions { MeasureUnit = OdtSaveMeasureUnit.Inches };
```

हम एक नया OdtSaveOptions ऑब्जेक्ट बनाते हैं और MeasureUnit गुण को वांछित मान पर सेट करते हैं, हमारे मामले में, माप इकाई के रूप में इंच का उपयोग करने के लिए OdtSaveMeasureUnit.Inches।

## दस्तावेज़ को ODT में बदलें

अब जब हमने सेव ऑप्शन कॉन्फ़िगर कर लिया है, तो हम डॉक्यूमेंट को ODT में बदलने के लिए आगे बढ़ सकते हैं। सेव ऑप्शन निर्दिष्ट करके परिवर्तित डॉक्यूमेंट को ODT फॉर्मेट में सेव करने के लिए डॉक्यूमेंट क्लास की सेव विधि का उपयोग करें। यहाँ एक उदाहरण दिया गया है :

```csharp
doc.Save(dataDir + "WorkingWithOdtSaveOptions.MeasureUnit.odt", saveOptions);
```

इस उदाहरण में, हम निर्दिष्ट सहेजने के विकल्पों का उपयोग करके परिवर्तित दस्तावेज़ को "WorkingWithOdtSaveOptions.MeasureUnit.odt" के रूप में सहेजते हैं।

### .NET के लिए Aspose.Words का उपयोग करके "माप की इकाई" कार्यक्षमता के साथ OdtSaveOptions के लिए उदाहरण स्रोत कोड



```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// Word दस्तावेज़ लोड करें
Document doc = new Document(dataDir + "Document.docx");

// "माप की इकाई" सुविधा के साथ बैकअप विकल्पों का कॉन्फ़िगरेशन
OdtSaveOptions saveOptions = new OdtSaveOptions { MeasureUnit = OdtSaveMeasureUnit.Inches };

// दस्तावेज़ को ODT में बदलें
doc.Save(dataDir + "WorkingWithOdtSaveOptions.MeasureUnit.odt", saveOptions);
```

## निष्कर्ष

इस गाइड में, हमने बताया है कि .NET के लिए Aspose.Words लाइब्रेरी के साथ OdtSaveOptions सेव ऑप्शन का उपयोग करके माप की इकाई निर्दिष्ट करके Word दस्तावेज़ को ODT में कैसे परिवर्तित किया जाए। दिए गए चरणों का पालन करके और दिए गए C# स्रोत कोड का उपयोग करके, आप आसानी से अपने C# एप्लिकेशन में इस कार्यक्षमता को लागू कर सकते हैं। ODT में कनवर्ट करते समय माप की इकाई निर्दिष्ट करने से आप अपनी विशिष्ट आवश्यकताओं के अनुसार परिणामी दस्तावेज़ के स्वरूपण और आयामों को नियंत्रित कर सकते हैं।
