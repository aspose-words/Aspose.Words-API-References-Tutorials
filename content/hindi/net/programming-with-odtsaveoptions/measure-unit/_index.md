जब आप किसी Word दस्तावेज़ को C# एप्लिकेशन में OpenDocument Text (ODT) प्रारूप में परिवर्तित करते हैं, तो आप मापने योग्य स्वरूपण और सामग्री गुणों के लिए उपयोग की जाने वाली माप की इकाई निर्दिष्ट करना चाह सकते हैं। .NET के लिए Aspose.Words लाइब्रेरी के साथ, आप OdtSaveOptions सेव विकल्पों का उपयोग करके इस कार्यक्षमता को आसानी से निर्दिष्ट कर सकते हैं। इस चरण-दर-चरण मार्गदर्शिका में, हम आपको बताएंगे कि OdtSaveOptions का उपयोग करके माप की इकाई निर्दिष्ट करके किसी Word दस्तावेज़ को ODT में परिवर्तित करने के लिए .NET C# स्रोत कोड के लिए Aspose.Words का उपयोग कैसे करें।

## Aspose.Words लाइब्रेरी को समझना

कोड में गोता लगाने से पहले, .NET के लिए Aspose.Words लाइब्रेरी को समझना महत्वपूर्ण है। Aspose.Words .NET सहित विभिन्न प्लेटफार्मों में Word दस्तावेज़ों को बनाने, संपादित करने, परिवर्तित करने और सुरक्षित करने के लिए एक शक्तिशाली लाइब्रेरी है। यह दस्तावेज़ों में हेरफेर करने के लिए कई सुविधाएँ प्रदान करता है, जैसे टेक्स्ट सम्मिलित करना, फ़ॉर्मेटिंग बदलना, अनुभाग जोड़ना और बहुत कुछ।

## Word दस्तावेज़ लोड हो रहा है

पहला कदम उस वर्ड दस्तावेज़ को लोड करना है जिसे आप ओडीटी में कनवर्ट करना चाहते हैं। दस्तावेज़ को स्रोत फ़ाइल से लोड करने के लिए दस्तावेज़ वर्ग का उपयोग करें। यहाँ एक उदाहरण है :

```csharp
Document doc = new Document(dataDir + "Document.docx");
```

इस उदाहरण में, हम दस्तावेज़ निर्देशिका में स्थित दस्तावेज़ "Document.docx" लोड करते हैं।

## बैकअप विकल्प कॉन्फ़िगर करना

अगला कदम ओडीटी में कनवर्ट करने के लिए बैकअप विकल्पों को कॉन्फ़िगर करना है। OdtSaveOptions क्लास का उपयोग करें और मापयूनिट प्रॉपर्टी को वांछित मान पर सेट करें। उदाहरण के लिए, यदि आप माप की इकाई के रूप में इंच का उपयोग करना चाहते हैं, तो मापयूनिट को OdtSaveMeasureUnit.Inches पर सेट करें। यह कैसे करना है यहां बताया गया है:

```csharp
OdtSaveOptions saveOptions = new OdtSaveOptions { MeasureUnit = OdtSaveMeasureUnit.Inches };
```

हम एक नया OdtSaveOptions ऑब्जेक्ट बनाते हैं और माप इकाई के रूप में इंच का उपयोग करने के लिए हमारे मामले में, OdtSaveMeasureUnit.Inches, माप इकाई संपत्ति को वांछित मान पर सेट करते हैं।

## दस्तावेज़ को ODT में कनवर्ट करें

अब जब हमने सेव विकल्प कॉन्फ़िगर कर लिया है, तो हम दस्तावेज़ को ओडीटी में बदलने के लिए आगे बढ़ सकते हैं। सेव विकल्पों को निर्दिष्ट करके परिवर्तित दस्तावेज़ को ओडीटी प्रारूप में सहेजने के लिए दस्तावेज़ वर्ग की सेव विधि का उपयोग करें। यहाँ एक उदाहरण है :

```csharp
doc.Save(dataDir + "WorkingWithOdtSaveOptions.MeasureUnit.odt", saveOptions);
```

इस उदाहरण में, हम निर्दिष्ट सेव विकल्पों का उपयोग करके परिवर्तित दस्तावेज़ को "WorkingWithOdtSaveOptions.MeasureUnit.odt" के रूप में सहेजते हैं।

### .NET के लिए Aspose.Words का उपयोग करके "माप की इकाई" कार्यक्षमता के साथ OdtSaveOptions के लिए उदाहरण स्रोत कोड



```csharp
// आपके दस्तावेज़ निर्देशिका का पथ
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// Word दस्तावेज़ लोड करें
Document doc = new Document(dataDir + "Document.docx");

// "माप की इकाई" सुविधा के साथ बैकअप विकल्पों का कॉन्फ़िगरेशन
OdtSaveOptions saveOptions = new OdtSaveOptions { MeasureUnit = OdtSaveMeasureUnit.Inches };

// दस्तावेज़ को ODT में कनवर्ट करें
doc.Save(dataDir + "WorkingWithOdtSaveOptions.MeasureUnit.odt", saveOptions);
```

## निष्कर्ष

इस गाइड में, हमने बताया है कि .NET के लिए Aspose.Words लाइब्रेरी के साथ OdtSaveOptions सेव विकल्पों का उपयोग करके माप की इकाई निर्दिष्ट करके किसी Word दस्तावेज़ को ODT में कैसे परिवर्तित किया जाए। दिए गए चरणों का पालन करके और दिए गए C# स्रोत कोड का उपयोग करके, आप इस कार्यक्षमता को अपने C# एप्लिकेशन में आसानी से लागू कर सकते हैं। ओडीटी में कनवर्ट करते समय माप की इकाई निर्दिष्ट करने से आप अपनी विशिष्ट आवश्यकताओं के अनुसार परिणामी दस्तावेज़ के स्वरूपण और आयामों को नियंत्रित कर सकते हैं।
