इस ट्यूटोरियल में, हम आपको Aspose.Words for .NET के साथ दस्तावेज़ हस्ताक्षर सुविधा का उपयोग करने के चरणों के बारे में बताएँगे। यह सुविधा आपको प्रमाणपत्र का उपयोग करके Word दस्तावेज़ पर डिजिटल रूप से हस्ताक्षर करने की अनुमति देती है। नीचे दिए गए चरणों का पालन करें:

## चरण 1: प्रमाणपत्र लोड करना

CertificateHolder वर्ग का उपयोग करके हस्ताक्षर प्रमाणपत्र लोड करके प्रारंभ करें:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
CertificateHolder certHolder = CertificateHolder.Create(dataDir + "morzal.pfx", "aw");
```

अपने प्रमाणपत्र और संबंधित पासवर्ड का सही पथ निर्दिष्ट करना सुनिश्चित करें।

## चरण 2: दस्तावेज़ पर हस्ताक्षर करना

दस्तावेज़ पर हस्ताक्षर करने के लिए DigitalSignatureUtil वर्ग का उपयोग करें:

```csharp
DigitalSignatureUtil.Sign(dataDir + "Digitally signed.docx", dataDir + "Document.Signed.docx",
	certHolder);
```

स्रोत दस्तावेज़ और हस्ताक्षरित दस्तावेज़ के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें।

### .NET के लिए Aspose.Words का उपयोग करके दस्तावेज़ पर हस्ताक्षर करने के लिए उदाहरण स्रोत कोड

.NET के लिए Aspose.Words के साथ दस्तावेज़ पर हस्ताक्षर करने के लिए पूरा स्रोत कोड यहां दिया गया है:

```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	CertificateHolder certHolder = CertificateHolder.Create(dataDir + "morzal.pfx", "aw");
	
	DigitalSignatureUtil.Sign(dataDir + "Digitally signed.docx", dataDir + "Document.Signed.docx",
		certHolder);

```

इन चरणों का पालन करके, आप आसानी से .NET के लिए Aspose.Words के साथ एक Word दस्तावेज़ पर हस्ताक्षर कर सकते हैं।

## निष्कर्ष

 इस ट्यूटोरियल में, हमने .NET के लिए Aspose.Words में दस्तावेज़ हस्ताक्षर सुविधा का पता लगाया। हस्ताक्षर प्रमाणपत्र लोड करके और उसका उपयोग करके`DigitalSignatureUtil.Sign` इस विधि से हम वर्ड दस्तावेज़ पर डिजिटल हस्ताक्षर कर सकते हैं। दस्तावेज़ हस्ताक्षर प्रमाणीकरण प्रदान करता है और दस्तावेज़ की सामग्री की अखंडता सुनिश्चित करता है, जिससे यह सुरक्षित और भरोसेमंद दस्तावेज़ प्रबंधन के लिए एक मूल्यवान सुविधा बन जाती है।

### साइन वर्ड दस्तावेज़ के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: Aspose.Words for .NET में दस्तावेज़ हस्ताक्षर क्या है?

उत्तर: Aspose.Words for .NET में दस्तावेज़ हस्ताक्षर करना प्रमाणपत्र का उपयोग करके Word दस्तावेज़ पर डिजिटल हस्ताक्षर करने की प्रक्रिया को संदर्भित करता है। यह सुविधा दस्तावेज़ में एक डिजिटल हस्ताक्षर जोड़ती है, जो दस्तावेज़ की सामग्री की प्रामाणिकता, अखंडता और गैर-अस्वीकृति प्रदान करती है।

#### प्रश्न: मैं .NET के लिए Aspose.Words में हस्ताक्षर प्रमाणपत्र कैसे लोड कर सकता हूं?

 A: .NET के लिए Aspose.Words में हस्ताक्षर प्रमाणपत्र लोड करने के लिए, आप इसका उपयोग कर सकते हैं`CertificateHolder` क्लास का एक उदाहरण बनाएँ।`CertificateHolder` प्रमाणपत्र फ़ाइल का पथ और संबंधित पासवर्ड प्रदान करके। यहाँ एक उदाहरण दिया गया है:

```csharp
CertificateHolder certHolder = CertificateHolder.Create("path/to/certificate.pfx", "password");
```

अपने प्रमाणपत्र और उससे संबंधित पासवर्ड का सही पथ प्रदान करना सुनिश्चित करें।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके Word दस्तावेज़ पर हस्ताक्षर कैसे करूँ?

 उत्तर: .NET के लिए Aspose.Words का उपयोग करके Word दस्तावेज़ पर हस्ताक्षर करने के लिए, आप इसका उपयोग कर सकते हैं`DigitalSignatureUtil` कक्षा को बुलाओ।`Sign` विधि, स्रोत दस्तावेज़ के लिए पथ, हस्ताक्षरित दस्तावेज़ (आउटपुट) के लिए पथ, और प्रदान करना`CertificateHolder` वस्तु। यहाँ एक उदाहरण है:

```csharp
DigitalSignatureUtil.Sign("path/to/source/document.docx", "path/to/signed/document.docx", certHolder);
```

सुनिश्चित करें कि आप स्रोत दस्तावेज़ और हस्ताक्षरित दस्तावेज़ (आउटपुट) के लिए सही पथ प्रदान करते हैं।

#### प्रश्न: दस्तावेज़ पर हस्ताक्षर करने का उद्देश्य क्या है?

उत्तर: दस्तावेज़ हस्ताक्षर किसी दस्तावेज़ की प्रामाणिकता और अखंडता सुनिश्चित करने की एक विधि के रूप में कार्य करता है। किसी दस्तावेज़ पर डिजिटल हस्ताक्षर करके, आप इसकी उत्पत्ति का प्रमाण प्रदान कर सकते हैं, इसकी सामग्री में कोई बदलाव नहीं किया गया है, और गैर-अस्वीकृति स्थापित कर सकते हैं। दस्तावेज़ हस्ताक्षर का उपयोग आमतौर पर कानूनी, वित्तीय और संवेदनशील दस्तावेज़ों के लिए किया जाता है।

#### प्रश्न: क्या मैं Aspose.Words for .NET में दस्तावेज़ हस्ताक्षर के लिए किसी भी प्रमाणपत्र का उपयोग कर सकता हूँ?

उत्तर: Aspose.Words for .NET में दस्तावेज़ पर हस्ताक्षर करने के लिए, आपको एक वैध X.509 प्रमाणपत्र का उपयोग करना होगा। यह प्रमाणपत्र किसी विश्वसनीय प्रमाणपत्र प्राधिकरण (CA) से प्राप्त किया जा सकता है या परीक्षण उद्देश्यों के लिए स्व-हस्ताक्षरित प्रमाणपत्र का उपयोग किया जा सकता है।

#### प्रश्न: दस्तावेज़ हस्ताक्षर के लिए Aspose.Words for .NET किस फ़ाइल प्रारूप का समर्थन करता है?

 उत्तर: Aspose.Words for .NET, DOCX फ़ाइल फ़ॉर्मेट में Word दस्तावेज़ों के लिए दस्तावेज़ हस्ताक्षर का समर्थन करता है। आप DOCX फ़ाइलों पर हस्ताक्षर करने के लिए DOCX फ़ाइल का उपयोग कर सकते हैं।`DigitalSignatureUtil` कक्षा और उपयुक्त प्रमाण पत्र।

#### प्रश्न: क्या मैं एक ही प्रमाणपत्र का उपयोग करके एकाधिक वर्ड दस्तावेज़ों पर हस्ताक्षर कर सकता हूँ?

उत्तर: हाँ, आप एक ही प्रमाणपत्र का उपयोग करके कई Word दस्तावेज़ों पर हस्ताक्षर कर सकते हैं। एक बार जब आप प्रमाणपत्र को लोड कर लेते हैं`CertificateHolder` क्लास में, आप इसे कॉल करके कई दस्तावेज़ों पर हस्ताक्षर करने के लिए पुनः उपयोग कर सकते हैं`DigitalSignatureUtil.Sign` विभिन्न स्रोत और हस्ताक्षरित दस्तावेज़ पथों के साथ विधि।

#### प्रश्न: क्या दस्तावेज़ पर हस्ताक्षर करने से मूल दस्तावेज़ में परिवर्तन हो जाता है?

उत्तर: Aspose.Words for .NET के साथ दस्तावेज़ पर हस्ताक्षर करने से मूल दस्तावेज़ में कोई बदलाव नहीं होता। इसके बजाय, यह दस्तावेज़ की डिजिटल रूप से हस्ताक्षरित प्रतिलिपि बनाता है, जिससे मूल दस्तावेज़ बरकरार रहता है। डिजिटल रूप से हस्ताक्षरित प्रतिलिपि में अतिरिक्त डिजिटल हस्ताक्षर शामिल होते हैं, जो दस्तावेज़ की सामग्री की अखंडता सुनिश्चित करता है।

#### प्रश्न: क्या मैं .NET के लिए Aspose.Words का उपयोग करके हस्ताक्षरित दस्तावेज़ के डिजिटल हस्ताक्षर को सत्यापित कर सकता हूं?

 उत्तर: हां, Aspose.Words for .NET हस्ताक्षरित दस्तावेज़ के डिजिटल हस्ताक्षर को सत्यापित करने के लिए कार्यक्षमता प्रदान करता है। आप इसका उपयोग कर सकते हैं`DigitalSignatureUtil.Verify` डिजिटल हस्ताक्षर की वैधता और प्रामाणिकता की जांच करने की विधि।
