इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.Words के साथ एन्क्रिप्टेड वर्ड दस्तावेज़ पर हस्ताक्षर करने की सुविधा का उपयोग करने के चरणों के माध्यम से मार्गदर्शन करेंगे। यह सुविधा आपको डिक्रिप्शन पासवर्ड का उपयोग करके एन्क्रिप्ट किए गए वर्ड दस्तावेज़ पर डिजिटल रूप से हस्ताक्षर करने की अनुमति देती है। नीचे दिए गए चरणों का पालन करें:

## चरण 1: हस्ताक्षर विकल्प सेट करना

SignOptions वर्ग का एक उदाहरण बनाएं और डिक्रिप्शन पासवर्ड सेट करें:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
SignOptions signOptions = new SignOptions { DecryptionPassword = "decryptionpassword" };
```

अपने एन्क्रिप्टेड दस्तावेज़ के लिए सही डिक्रिप्शन पासवर्ड निर्दिष्ट करना सुनिश्चित करें।

## चरण 2: प्रमाणपत्र लोड करना

CertificateHolder वर्ग का उपयोग करके हस्ताक्षर प्रमाणपत्र लोड करके प्रारंभ करें:

```csharp
CertificateHolder certHolder = CertificateHolder.Create(dataDir + "morzal.pfx", "aw");
```

अपने प्रमाणपत्र और संबंधित पासवर्ड का सही पथ निर्दिष्ट करना सुनिश्चित करें।

## चरण 3: एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर करना

एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर करने के लिए DigitalSignatureUtil वर्ग का उपयोग करें:

```csharp
DigitalSignatureUtil.Sign(dataDir + "Digitally signed.docx", dataDir + "Document.EncryptedDocument.docx",
	certHolder, signOptions);
```

एन्क्रिप्टेड दस्तावेज़, हस्ताक्षरित दस्तावेज़ और प्रमाणपत्र के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें।

### .NET के लिए Aspose.Words का उपयोग करके एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर करने के लिए उदाहरण स्रोत कोड

.NET के लिए Aspose.Words के साथ एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर करने के लिए पूरा स्रोत कोड यहां दिया गया है:

```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	SignOptions signOptions = new SignOptions { DecryptionPassword = "decryptionPassword" };

	CertificateHolder certHolder = CertificateHolder.Create(dataDir + "morzal.pfx", "aw");
	
	DigitalSignatureUtil.Sign(dataDir + "Digitally signed.docx", dataDir + "Document.EncryptedDocument.docx",
		certHolder, signOptions);
	

```
इन चरणों का पालन करके, आप आसानी से .NET के लिए Aspose.Words के साथ एक एन्क्रिप्टेड Word दस्तावेज़ पर हस्ताक्षर कर सकते हैं।

## निष्कर्ष

इस ट्यूटोरियल में, हमने .NET के लिए Aspose.Words का उपयोग करके एन्क्रिप्टेड Word दस्तावेज़ पर हस्ताक्षर करने की प्रक्रिया का पता लगाया। डिक्रिप्शन पासवर्ड और हस्ताक्षर प्रमाणपत्र प्रदान करके, हम एन्क्रिप्टेड दस्तावेज़ में एक डिजिटल हस्ताक्षर जोड़ सकते हैं। एन्क्रिप्टेड दस्तावेज़ों पर हस्ताक्षर करने से उनकी प्रामाणिकता और अखंडता सुनिश्चित होती है, जिससे सुरक्षा की एक अतिरिक्त परत मिलती है। .NET के लिए Aspose.Words आपको एन्क्रिप्टेड दस्तावेज़ों पर हस्ताक्षर करने और अपनी Word फ़ाइलों की सुरक्षा और विश्वसनीयता बनाए रखने में सक्षम बनाता है।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: Aspose.Words for .NET में दस्तावेज़ हस्ताक्षर क्या है?

उत्तर: Aspose.Words for .NET में दस्तावेज़ हस्ताक्षर करना किसी Word दस्तावेज़ पर डिजिटल हस्ताक्षर करने की प्रक्रिया को संदर्भित करता है ताकि इसकी प्रामाणिकता, अखंडता और गैर-अस्वीकृति सुनिश्चित की जा सके। इसमें प्रमाणपत्र का उपयोग करके दस्तावेज़ में एक डिजिटल हस्ताक्षर जोड़ना शामिल है।

#### प्रश्न: एन्क्रिप्टेड वर्ड दस्तावेज़ क्या है?

उत्तर: एन्क्रिप्टेड वर्ड दस्तावेज़ एक ऐसा दस्तावेज़ है जिसे पासवर्ड का उपयोग करके एन्क्रिप्ट किया गया है। एन्क्रिप्शन एक सुरक्षा उपाय है जो दस्तावेज़ की सामग्री को अव्यवस्थित करके और सही डिक्रिप्शन पासवर्ड के बिना इसे अपठनीय बनाकर सुरक्षित रखता है।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके एन्क्रिप्टेड Word दस्तावेज़ पर हस्ताक्षर कैसे कर सकता हूं?

उत्तर: .NET के लिए Aspose.Words का उपयोग करके एन्क्रिप्टेड Word दस्तावेज़ पर हस्ताक्षर करने के लिए, आपको हस्ताक्षर प्रमाणपत्र के साथ डिक्रिप्शन पासवर्ड प्रदान करना होगा। इन चरणों का पालन करें:
1.  डिक्रिप्शन पासवर्ड सेट करें`SignOptions` वस्तु।
2.  का उपयोग करके हस्ताक्षर प्रमाणपत्र लोड करें`CertificateHolder` कक्षा।
3.  उपयोग`DigitalSignatureUtil.Sign` एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर करने की विधि, आवश्यक पैरामीटर प्रदान करना।

#### प्रश्न: एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर करने का उद्देश्य क्या है?

उत्तर: .NET के लिए Aspose.Words के साथ एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर करने से आप एन्क्रिप्टेड होने पर भी दस्तावेज़ में डिजिटल हस्ताक्षर जोड़ सकते हैं। यह सुरक्षा की एक अतिरिक्त परत प्रदान करता है और एन्क्रिप्टेड सामग्री की प्रामाणिकता और अखंडता सुनिश्चित करता है। यह प्राप्तकर्ताओं को दस्तावेज़ की उत्पत्ति को सत्यापित करने और किसी भी छेड़छाड़ का पता लगाने की अनुमति देता है।

#### प्रश्न: क्या मैं डिक्रिप्शन पासवर्ड बताए बिना एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर कर सकता हूँ?

उत्तर: नहीं, एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर करने के लिए, आपको सही डिक्रिप्शन पासवर्ड प्रदान करना होगा। डिजिटल हस्ताक्षर लागू करने से पहले दस्तावेज़ की एन्क्रिप्टेड सामग्री तक पहुँचने और उसे संशोधित करने के लिए डिक्रिप्शन पासवर्ड की आवश्यकता होती है।

#### प्रश्न: क्या मैं किसी भी प्रमाणपत्र का उपयोग करके एन्क्रिप्टेड वर्ड दस्तावेज़ पर हस्ताक्षर कर सकता हूँ?

उत्तर: .NET के लिए Aspose.Words का उपयोग करके एन्क्रिप्टेड Word दस्तावेज़ पर हस्ताक्षर करने के लिए, आपको एक वैध X.509 प्रमाणपत्र की आवश्यकता होती है। प्रमाणपत्र किसी विश्वसनीय प्रमाणपत्र प्राधिकरण (CA) से प्राप्त किया जा सकता है या परीक्षण उद्देश्यों के लिए स्व-हस्ताक्षरित प्रमाणपत्र का उपयोग किया जा सकता है।

#### प्रश्न: क्या मैं एक ही प्रमाणपत्र का उपयोग करके एकाधिक एन्क्रिप्टेड वर्ड दस्तावेज़ों पर हस्ताक्षर कर सकता हूँ?

 उत्तर: हाँ, आप एक ही प्रमाणपत्र का उपयोग करके कई एन्क्रिप्टेड वर्ड दस्तावेज़ों पर हस्ताक्षर कर सकते हैं। एक बार जब आप प्रमाणपत्र को लोड कर लेते हैं`CertificateHolder` क्लास में, आप इसे एकाधिक एन्क्रिप्टेड दस्तावेज़ों पर हस्ताक्षर करने के लिए पुनः उपयोग कर सकते हैं।

#### प्रश्न: क्या मैं हस्ताक्षरित एन्क्रिप्टेड दस्तावेज़ के डिजिटल हस्ताक्षर को सत्यापित कर सकता हूँ?

 उत्तर: हां, Aspose.Words for .NET हस्ताक्षरित एन्क्रिप्टेड दस्तावेज़ के डिजिटल हस्ताक्षर को सत्यापित करने के लिए कार्यक्षमता प्रदान करता है। आप इसका उपयोग कर सकते हैं`DigitalSignatureUtil.Verify` डिजिटल हस्ताक्षर की वैधता और प्रामाणिकता की जांच करने की विधि।

#### प्रश्न: एन्क्रिप्टेड दस्तावेज़ों पर हस्ताक्षर करने के लिए Aspose.Words for .NET किस फ़ाइल प्रारूप का समर्थन करता है?

 उत्तर: .NET के लिए Aspose.Words DOCX फ़ाइल प्रारूप में एन्क्रिप्टेड Word दस्तावेज़ों पर हस्ताक्षर करने का समर्थन करता है। आप एन्क्रिप्टेड DOCX फ़ाइलों पर हस्ताक्षर करने के लिए DOCX फ़ाइल का उपयोग कर सकते हैं।`DigitalSignatureUtil.Sign` आवश्यक डिक्रिप्शन पासवर्ड और प्रमाण पत्र के साथ विधि।

#### प्रश्न: एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर करने से एन्क्रिप्शन पर क्या प्रभाव पड़ता है?

उत्तर: .NET के लिए Aspose.Words के साथ एन्क्रिप्टेड दस्तावेज़ पर हस्ताक्षर करने से दस्तावेज़ के एन्क्रिप्शन पर कोई प्रभाव नहीं पड़ता है। एन्क्रिप्शन बरकरार रहता है, और एन्क्रिप्टेड सामग्री में डिजिटल हस्ताक्षर जोड़ा जाता है। डिजिटल हस्ताक्षर दस्तावेज़ पर लागू एन्क्रिप्शन से समझौता किए बिना अतिरिक्त सुरक्षा और सत्यापन प्रदान करता है।
