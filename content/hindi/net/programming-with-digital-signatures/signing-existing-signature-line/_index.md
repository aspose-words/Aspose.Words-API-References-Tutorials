इस ट्यूटोरियल में, हम आपको Aspose.Words for .NET के साथ मौजूदा हस्ताक्षर लाइन की हस्ताक्षर सुविधा का उपयोग करने के चरणों के बारे में बताएँगे। यह सुविधा आपको Word दस्तावेज़ में पहले से मौजूद हस्ताक्षर लाइन पर डिजिटल रूप से हस्ताक्षर करने की अनुमति देती है। नीचे दिए गए चरणों का पालन करें:

## चरण 1: दस्तावेज़ लोड करना और हस्ताक्षर पंक्ति तक पहुँचना

मौजूदा हस्ताक्षर पंक्ति वाले दस्तावेज़ को अपलोड करके प्रारंभ करें:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Signature line.docx");

SignatureLine signatureLine = ((Shape)doc.FirstSection.Body.GetChild(NodeType.Shape, 0, true)).SignatureLine;
```

## चरण 2: हस्ताक्षर विकल्प सेट करना

SignOptions वर्ग का एक उदाहरण बनाएं और हस्ताक्षर विकल्प सेट करें, जिसमें हस्ताक्षर पंक्ति ID और हस्ताक्षर पंक्ति छवि शामिल हैं:

```csharp
SignOptions signOptions = new SignOptions
{
SignatureLineId = signatureLine.Id,
SignatureLineImage = File.ReadAllBytes(ImagesDir + "Enhanced Windows MetaFile.emf")
};
```

हस्ताक्षर पंक्ति छवि के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें।

## चरण 3: प्रमाणपत्र लोड करना

CertificateHolder वर्ग का उपयोग करके हस्ताक्षर प्रमाणपत्र लोड करके प्रारंभ करें:

```csharp
CertificateHolder certHolder = CertificateHolder.Create(dataDir + "morzal.pfx", "aw");
```

अपने प्रमाणपत्र और संबंधित पासवर्ड का सही पथ निर्दिष्ट करना सुनिश्चित करें।

## चरण 4: मौजूदा हस्ताक्षर पंक्ति पर हस्ताक्षर करना

मौजूदा हस्ताक्षर पंक्ति पर हस्ताक्षर करने के लिए DigitalSignatureUtil वर्ग का उपयोग करें:

```csharp
DigitalSignatureUtil.Sign(dataDir + "Digitally signed.docx",
	dataDir + "SignDocuments.SigningExistingSignatureLine.docx", certHolder, signOptions);
```

स्रोत दस्तावेज़, हस्ताक्षरित दस्तावेज़ और प्रमाणपत्र के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें।

### .NET के लिए Aspose.Words का उपयोग करके मौजूदा हस्ताक्षर लाइन पर हस्ताक्षर करने के लिए उदाहरण स्रोत कोड

.NET के लिए Aspose.Words के साथ मौजूदा हस्ताक्षर पंक्ति पर हस्ताक्षर करने के लिए यहां पूर्ण स्रोत कोड दिया गया है:


```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Signature line.docx");
	
	SignatureLine signatureLine =
		((Shape) doc.FirstSection.Body.GetChild(NodeType.Shape, 0, true)).SignatureLine;

	SignOptions signOptions = new SignOptions
	{
		SignatureLineId = signatureLine.Id,
		SignatureLineImage = File.ReadAllBytes(ImagesDir + "Enhanced Windows MetaFile.emf")
	};

	CertificateHolder certHolder = CertificateHolder.Create(dataDir + "morzal.pfx", "aw");
	
	DigitalSignatureUtil.Sign(dataDir + "Digitally signed.docx",
		dataDir + "SignDocuments.SigningExistingSignatureLine.docx", certHolder, signOptions);
	

```

इन चरणों का पालन करके, आप आसानी से Aspose.Words for .NET के साथ Word दस्तावेज़ में मौजूदा हस्ताक्षर पंक्ति पर हस्ताक्षर कर सकते हैं।

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा कि Aspose.Words for .NET का उपयोग करके Word दस्तावेज़ में मौजूदा हस्ताक्षर लाइन पर हस्ताक्षर कैसे करें। दिए गए चरणों का पालन करके, आप आसानी से दस्तावेज़ लोड कर सकते हैं, मौजूदा हस्ताक्षर लाइन तक पहुँच सकते हैं, हस्ताक्षर विकल्प सेट कर सकते हैं और दस्तावेज़ पर हस्ताक्षर कर सकते हैं। मौजूदा हस्ताक्षर लाइन पर हस्ताक्षर करने की क्षमता आपके Word दस्तावेज़ों में पूर्वनिर्धारित क्षेत्रों में डिजिटल हस्ताक्षर जोड़ने का एक सुविधाजनक तरीका प्रदान करती है, जिससे दस्तावेज़ की अखंडता और प्रमाणीकरण सुनिश्चित होता है। Aspose.Words for .NET डिजिटल हस्ताक्षरों के साथ Words प्रोसेसिंग के लिए एक शक्तिशाली API प्रदान करता है, जिससे आप हस्ताक्षर प्रक्रिया को अनुकूलित कर सकते हैं और अपने Word दस्तावेज़ों की सुरक्षा बढ़ा सकते हैं।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: वर्ड दस्तावेज़ में मौजूदा हस्ताक्षर पंक्ति क्या है?

उत्तर: वर्ड दस्तावेज़ में मौजूदा हस्ताक्षर रेखा एक पूर्वनिर्धारित क्षेत्र है जहाँ हस्ताक्षर रखा जा सकता है। इसे आम तौर पर दस्तावेज़ में किसी आकृति या वस्तु द्वारा दर्शाया जाता है और यह हस्ताक्षरकर्ता के लिए अपना डिजिटल हस्ताक्षर जोड़ने के लिए निर्दिष्ट स्थान के रूप में कार्य करता है।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके Word दस्तावेज़ में मौजूदा हस्ताक्षर पंक्ति पर हस्ताक्षर कैसे कर सकता हूं?

उत्तर: Aspose.Words for .NET का उपयोग करके Word दस्तावेज़ में मौजूदा हस्ताक्षर पंक्ति पर हस्ताक्षर करने के लिए, आप इन चरणों का पालन कर सकते हैं:
1.  दस्तावेज़ को लोड करने के लिए निम्न का उपयोग करें:`Document` क्लास चुनें और दस्तावेज़ फ़ाइल का पथ निर्दिष्ट करें.
2.  उचित विधि या प्रॉपर्टी का उपयोग करके मौजूदा हस्ताक्षर लाइन तक पहुँचें। उदाहरण के लिए, आप इसका उपयोग कर सकते हैं`GetChild` हस्ताक्षर लाइन आकार को पुनः प्राप्त करने की विधि।
3.  इसका एक उदाहरण बनाएं`SignOptions` वर्ग और सेट`SignatureLineId` संपत्ति को मौजूदा हस्ताक्षर पंक्ति की आईडी में बदलें।
4.  ठीक`SignatureLineImage` की संपत्ति`SignOptions` डिजिटल हस्ताक्षर का प्रतिनिधित्व करने वाली छवि के लिए वर्ग को जोड़ें।
5.  का उपयोग करके हस्ताक्षर प्रमाणपत्र लोड करें`CertificateHolder` कक्षा में जाकर आवश्यक प्रमाणपत्र और पासवर्ड प्रदान करें।
6.  उपयोग`DigitalSignatureUtil.Sign` दस्तावेज़ पर हस्ताक्षर करने की विधि, जिसमें आवश्यक पैरामीटर प्रदान करना शामिल है`SignOptions` वस्तु।

#### प्रश्न: मैं Aspose.Words for .NET का उपयोग करके Word दस्तावेज़ में मौजूदा हस्ताक्षर पंक्ति तक कैसे पहुँच सकता हूँ?

 उत्तर: Aspose.Words for .NET का उपयोग करके Word दस्तावेज़ में मौजूदा हस्ताक्षर लाइन तक पहुँचने के लिए, आप दस्तावेज़ की संरचना से हस्ताक्षर लाइन आकार को पुनः प्राप्त करने के लिए उपयुक्त विधि या गुण का उपयोग कर सकते हैं। उदाहरण के लिए, आप इसका उपयोग कर सकते हैं`GetChild` वांछित हस्ताक्षर रेखा आकार प्राप्त करने के लिए उपयुक्त मापदंडों के साथ विधि।

#### प्रश्न: क्या मैं मौजूदा हस्ताक्षर पंक्ति में डिजिटल हस्ताक्षर के स्वरूप को अनुकूलित कर सकता हूँ?

उत्तर: हां, आप हस्ताक्षर का प्रतिनिधित्व करने वाली एक छवि फ़ाइल प्रदान करके मौजूदा हस्ताक्षर पंक्ति में डिजिटल हस्ताक्षर की उपस्थिति को अनुकूलित कर सकते हैं। छवि एक लोगो, हस्तलिखित हस्ताक्षर या हस्ताक्षर का कोई अन्य ग्राफ़िकल प्रतिनिधित्व हो सकता है। आप सेट कर सकते हैं`SignatureLineImage` की संपत्ति`SignOptions` क्लास को छवि फ़ाइल के बाइट्स में बदलें।

#### प्रश्न: क्या मैं वर्ड दस्तावेज़ में एकाधिक मौजूदा हस्ताक्षर पंक्तियों पर हस्ताक्षर कर सकता हूँ?
 उत्तर: हां, आप वर्ड डॉक्यूमेंट में कई मौजूदा हस्ताक्षर लाइनों पर हस्ताक्षर कर सकते हैं। आपको प्रत्येक हस्ताक्षर लाइन के लिए अलग-अलग चरणों का पालन करना होगा, उचित सेटिंग करनी होगी`SignatureLineId` और`SignatureLineImage` मूल्यों में`SignOptions` प्रत्येक हस्ताक्षर पंक्ति के लिए एक ऑब्जेक्ट।

#### प्रश्न: मौजूदा हस्ताक्षर पंक्ति में डिजिटल हस्ताक्षर के लिए छवि फ़ाइल का प्रारूप क्या होना चाहिए?

 उत्तर: मौजूदा हस्ताक्षर लाइन में डिजिटल हस्ताक्षर के लिए छवि फ़ाइल विभिन्न स्वरूपों में हो सकती है, जैसे PNG, JPEG, BMP, या GIF। आप फ़ाइल पथ निर्दिष्ट कर सकते हैं या छवि फ़ाइल के बाइट्स को पढ़ सकते हैं और इसे असाइन कर सकते हैं`SignatureLineImage` की संपत्ति`SignOptions` कक्षा।
