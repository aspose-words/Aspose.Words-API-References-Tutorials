इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.Words की अप्रतिबंधित अनुभाग सुविधा का उपयोग करने के चरणों के माध्यम से मार्गदर्शन करेंगे। यह सुविधा आपको Word दस्तावेज़ में विशिष्ट अनुभागों को परिभाषित करने की अनुमति देती है जो सुरक्षित नहीं हैं, भले ही दस्तावेज़ का बाकी हिस्सा सुरक्षित हो। नीचे दिए गए चरणों का पालन करें:

## चरण 1: दस्तावेज़ और अनुभाग बनाना

डॉक्यूमेंट क्लास और डॉक्यूमेंटबिल्डर ऑब्जेक्ट का एक उदाहरण बनाकर प्रारंभ करें:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## चरण 2: दस्तावेज़ में सामग्री जोड़ें
दस्तावेज़ में सामग्री जोड़ने और अनुभाग विराम सम्मिलित करने के लिए DocumentBuilder ऑब्जेक्ट का उपयोग करें:

```csharp
builder.Writeln("Section 1. Unprotected.");
builder. InsertBreak(BreakType. SectionBreakContinuous);
builder.Writeln("Section 2. Protected.");
```

## चरण 3: दस्तावेज़ और अनुभागों को सुरक्षित करें

सेक्शन सुरक्षा केवल तभी काम करती है जब दस्तावेज़ सुरक्षा सक्षम हो और केवल फ़ॉर्म फ़ील्ड में संपादन की अनुमति हो। आप Document ऑब्जेक्ट की Protect() विधि का उपयोग करके दस्तावेज़ की सुरक्षा कर सकते हैं:

```csharp
doc.Protect(ProtectionType.AllowOnlyFormFields, "password");
```

सुरक्षा का सही प्रकार निर्दिष्ट करना सुनिश्चित करें और वांछित पासवर्ड सेट करें।

## चरण 4: किसी विशिष्ट अनुभाग के लिए सुरक्षा अक्षम करना

डिफ़ॉल्ट रूप से, सभी अनुभाग संरक्षित होते हैं, लेकिन आप Section ऑब्जेक्ट की ProtectedForForms संपत्ति का उपयोग करके किसी विशिष्ट अनुभाग के लिए सुरक्षा को चुनिंदा रूप से अक्षम कर सकते हैं:

```csharp
doc.Sections[0].ProtectedForForms = false;
```

इस उदाहरण में, पहले खंड के लिए सुरक्षा अक्षम है।

## चरण 5: दस्तावेज़ सहेजें

अंत में, संशोधित दस्तावेज़ को सहेजें:

```csharp
doc.Save(dataDir + "DocumentProtection.UnrestrictedSection.docx");
```

दस्तावेज़ को अप्रतिबंधित अनुभागों के साथ सहेजने के लिए सही पथ और फ़ाइल नाम निर्दिष्ट करना सुनिश्चित करें।

### .NET के लिए Aspose.Words का उपयोग करके अप्रतिबंधित अनुभाग के लिए उदाहरण स्रोत कोड

.NET के लिए Aspose.Words का उपयोग करते हुए अप्रतिबंधित अनुभाग के लिए पूर्ण स्रोत कोड यहां दिया गया है:


```csharp

// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// कुछ पाठ के साथ दो अनुभाग सम्मिलित करें.
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

builder.Writeln("Section 1. Unprotected.");
builder.InsertBreak(BreakType.SectionBreakContinuous);
builder.Writeln("Section 2. Protected.");

// अनुभाग सुरक्षा केवल तभी काम करती है जब दस्तावेज़ सुरक्षा चालू हो और केवल प्रपत्र फ़ील्ड में संपादन की अनुमति हो।
doc.Protect(ProtectionType.AllowOnlyFormFields, "password");

//डिफ़ॉल्ट रूप से, सभी अनुभाग संरक्षित होते हैं, लेकिन हम चुनिंदा रूप से सुरक्षा को बंद कर सकते हैं।
doc.Sections[0].ProtectedForForms = false;
doc.Save(dataDir + "DocumentProtection.UnrestrictedSection.docx");

doc = new Document(dataDir + "DocumentProtection.UnrestrictedSection.docx");

```

इन चरणों का पालन करके, आप आसानी से Aspose.Words for .NET के साथ अपने Word दस्तावेज़ में अप्रतिबंधित अनुभागों को परिभाषित करने में सक्षम होंगे।

## निष्कर्ष

इस ट्यूटोरियल में, हमने .NET के लिए Aspose.Words की अप्रतिबंधित अनुभाग सुविधा का पता लगाया, जो Word दस्तावेज़ में विशिष्ट अनुभागों को असुरक्षित रहने की अनुमति देता है जबकि दस्तावेज़ का बाकी हिस्सा सुरक्षित रहता है। दिए गए चरणों का पालन करके, आप आसानी से अपने दस्तावेज़ के भीतर अनुभागों को परिभाषित कर सकते हैं जहाँ उपयोगकर्ता अन्य अनुभागों के लिए सुरक्षा बनाए रखते हुए सामग्री को स्वतंत्र रूप से संपादित कर सकते हैं। .NET के लिए Aspose.Words दस्तावेज़ सुरक्षा और अनुकूलन के लिए शक्तिशाली क्षमताएँ प्रदान करता है, जिससे आपको अपने Word दस्तावेज़ों में संपादन अनुमतियों पर नियंत्रण मिलता है।

### वर्ड दस्तावेज़ में अप्रतिबंधित अनुभाग के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: Aspose.Words for .NET में अप्रतिबंधित अनुभाग क्या हैं?

उत्तर: Aspose.Words for .NET में अप्रतिबंधित अनुभाग Word दस्तावेज़ के भीतर विशिष्ट अनुभाग हैं जो सुरक्षित नहीं हैं, भले ही दस्तावेज़ का बाकी हिस्सा सुरक्षित हो। ये अनुभाग उपयोगकर्ताओं को दस्तावेज़ के शेष भागों के लिए सुरक्षा बनाए रखते हुए उनके भीतर की सामग्री को संशोधित करने की अनुमति देते हैं।

#### प्रश्न: मैं .NET के लिए Aspose.Words का उपयोग करके अप्रतिबंधित अनुभाग कैसे बना सकता हूं?

उत्तर: .NET के लिए Aspose.Words का उपयोग करके Word दस्तावेज़ में अप्रतिबंधित अनुभाग बनाने के लिए, आप इन चरणों का पालन कर सकते हैं:
1.  इसका एक उदाहरण बनाएं`Document` कक्षा और एक`DocumentBuilder` वस्तु।
2.  उपयोग`DocumentBuilder` दस्तावेज़ में सामग्री जोड़ने और अनुभाग विराम सम्मिलित करने के लिए.
3.  दस्तावेज़ को सुरक्षित करने के लिए निम्न का उपयोग करें:`Protect` की विधि`Document` ऑब्जेक्ट, वांछित सुरक्षा प्रकार और पासवर्ड निर्दिष्ट करना।
4.  किसी विशिष्ट अनुभाग के लिए सुरक्षा अक्षम करने के लिए सेटिंग करें`ProtectedForForms` संबंधित संपत्ति`Section` करने के लिए वस्तु`false`.
5. संशोधित दस्तावेज़ को सहेजें.

#### प्रश्न: क्या मैं वर्ड दस्तावेज़ में एकाधिक अप्रतिबंधित अनुभाग रख सकता हूँ?

 उत्तर: हां, आप Word दस्तावेज़ में कई अप्रतिबंधित अनुभाग रख सकते हैं। चुनिंदा अनुभागों के लिए सुरक्षा को अक्षम करके`ProtectedForForms` की संपत्ति`Section`ऑब्जेक्ट में, आप कई अनुभागों को परिभाषित कर सकते हैं जहां उपयोगकर्ता अन्य अनुभागों को सुरक्षित रखते हुए सामग्री को स्वतंत्र रूप से संशोधित कर सकते हैं।

#### प्रश्न 4. क्या मैं उस अनुभाग से सुरक्षा हटा सकता हूँ जो शुरू में संरक्षित था?
 हां, आप उस अनुभाग से सुरक्षा हटा सकते हैं जो शुरू में सेट करके संरक्षित था`ProtectedForForms` संबंधित संपत्ति`Section` करने के लिए वस्तु`false`इससे उपयोगकर्ताओं को बिना किसी प्रतिबंध के उस विशिष्ट अनुभाग में सामग्री को संपादित करने की अनुमति मिलती है।

#### प्रश्न: वर्ड दस्तावेज़ पर किस प्रकार की सुरक्षा लागू की जा सकती है?

उत्तर: Aspose.Words for .NET विभिन्न प्रकार की सुरक्षा प्रदान करता है जिन्हें Word दस्तावेज़ पर लागू किया जा सकता है, जिनमें शामिल हैं:
- NoProtection: कोई सुरक्षा लागू नहीं है.
- AllowOnlyRevisions: उपयोगकर्ता दस्तावेज़ में केवल संशोधन कर सकते हैं.
- AllowOnlyComments: उपयोगकर्ता केवल दस्तावेज़ में टिप्पणियाँ जोड़ सकते हैं.
- AllowOnlyFormFields: उपयोगकर्ता केवल दस्तावेज़ में फ़ॉर्म फ़ील्ड संपादित कर सकते हैं.
- केवल पढ़ने के लिए: दस्तावेज़ केवल पढ़ने के लिए है, और इसमें कोई संपादन की अनुमति नहीं है।
