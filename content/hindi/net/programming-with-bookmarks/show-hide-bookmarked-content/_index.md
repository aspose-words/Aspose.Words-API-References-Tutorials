## परिचय

नमस्ते! क्या आपने कभी किसी Word दस्तावेज़ में कुछ शर्तों के आधार पर विशिष्ट सामग्री की दृश्यता को नियंत्रित करना चाहा है? Aspose.Words for .NET के साथ, आप कोड की कुछ पंक्तियों के साथ बुकमार्क की गई सामग्री को गतिशील रूप से दिखा या छिपा सकते हैं। इस ट्यूटोरियल में, मैं आपको चरण-दर-चरण प्रक्रिया से गुज़रने दूँगा, यह सुनिश्चित करते हुए कि आप कोड के प्रत्येक भाग को समझते हैं। अंत तक, आप Word दस्तावेज़ों में बुकमार्क को हेरफेर करने में एक पेशेवर बन जाएँगे। चलिए शुरू करते हैं!

## आवश्यक शर्तें

इससे पहले कि हम ट्यूटोरियल में आगे बढ़ें, आइए सुनिश्चित करें कि आपके पास वह सब कुछ है जो आपको चाहिए:

1. C# का बुनियादी ज्ञान: आपको C# सिंटैक्स और अवधारणाओं से परिचित होना चाहिए।
2.  .NET के लिए Aspose.Words: इसे डाउनलोड करें[यहाँ](https://releases.aspose.com/words/net/) यदि आप खरीदने के लिए तैयार नहीं हैं, तो आप एक से शुरू कर सकते हैं[मुफ्त परीक्षण](https://releases.aspose.com/).
3. विजुअल स्टूडियो: कोई भी नवीनतम संस्करण काम करेगा, लेकिन नवीनतम संस्करण का उपयोग करना अनुशंसित है।
4. .NET फ्रेमवर्क: सुनिश्चित करें कि यह आपकी मशीन पर स्थापित है।

शुरू करने के लिए तैयार हैं? बढ़िया! चलिए आवश्यक नेमस्पेस को आयात करके शुरू करते हैं।

## नामस्थान आयात करें

.NET के लिए Aspose.Words का उपयोग करने के लिए, हमें आवश्यक नामस्थानों को आयात करना होगा। यह चरण सुनिश्चित करता है कि हमारे पास उन सभी वर्गों और विधियों तक पहुँच है जिनका हम उपयोग करेंगे।

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Fields;
```

ये नामस्थान वर्ड दस्तावेजों के साथ काम करने और उनकी सामग्री में परिवर्तन करने के लिए महत्वपूर्ण हैं।

## चरण 1: दस्तावेज़ सेट करना

सबसे पहले, आइए एक नया वर्ड डॉक्यूमेंट और एक डॉक्यूमेंट बिल्डर बनाएं। डॉक्यूमेंट बिल्डर हमें डॉक्यूमेंट के भीतर आसानी से कंटेंट जोड़ने और उसमें बदलाव करने में मदद करता है।

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

इस चरण में, हम एक नया दस्तावेज़ और एक दस्तावेज़ बिल्डर आरंभ करते हैं। यह आगे के संचालन के लिए हमारे वातावरण को सेट करता है।

## चरण 2: बुकमार्क की गई सामग्री जोड़ना

इसके बाद, हम दस्तावेज़ में कुछ सामग्री जोड़ेंगे और उसके चारों ओर एक बुकमार्क बनाएंगे। यह बुकमार्क हमें सामग्री को पहचानने और उसमें बदलाव करने में मदद करेगा।

```csharp
builder.Write("This is some text before the bookmark.");
builder.StartBookmark("MyBookmark");
builder.Write("This is the bookmarked content.");
builder.EndBookmark("MyBookmark");
builder.Write("This is some text after the bookmark.");
```

 यहां, हम बुकमार्क की गई सामग्री से पहले और बाद में कुछ पाठ जोड़ते हैं।`StartBookmark`और`EndBookmark` विधियाँ बुकमार्क की सीमाओं को परिभाषित करती हैं।

## चरण 3: सशर्त फ़ील्ड सम्मिलित करना

बुकमार्क की गई सामग्री की दृश्यता को नियंत्रित करने के लिए, हम एक सशर्त फ़ील्ड का उपयोग करेंगे। यह फ़ील्ड एक शर्त की जाँच करेगी और उसके अनुसार सामग्री को प्रदर्शित या छिपाएगी।

```csharp
builder.MoveToDocumentEnd();
Field field = builder.InsertField("IF \"", null);
builder.MoveTo(field.Start.NextSibling);
builder.InsertField("MERGEFIELD MyBookmark", null);
builder.Write("\" = \"true\" \"Visible\" \"Hidden\"");
```

इस चरण में, हम एक IF फ़ील्ड डालते हैं जो बुकमार्क के मान की जाँच करता है। यदि मान "सत्य" है, तो यह "दृश्यमान" प्रदर्शित करेगा; अन्यथा, यह "छिपा हुआ" प्रदर्शित करेगा।

## चरण 4: नोड्स को पुनः व्यवस्थित करना

इसके बाद, हमें नोड्स को पुनः व्यवस्थित करने की आवश्यकता है ताकि यह सुनिश्चित हो सके कि सशर्त तर्क बुकमार्क की गई सामग्री पर सही ढंग से लागू होता है।

```csharp
Bookmark bm = doc.Range.Bookmarks["MyBookmark"];
Node currentNode = field.Start;
bool flag = true;

while (currentNode != null && flag)
{
    if (currentNode.NodeType == NodeType.Run && currentNode.ToString(SaveFormat.Text).Trim() == "\"")
        flag = false;

    Node nextNode = currentNode.NextSibling;
    bm.BookmarkStart.ParentNode.InsertBefore(currentNode, bm.BookmarkStart);
    currentNode = nextNode;
}

Node endNode = bm.BookmarkEnd;
flag = true;

while (currentNode != null && flag)
{
    if (currentNode.NodeType == NodeType.FieldEnd)
        flag = false;

    Node nextNode = currentNode.NextSibling;
    bm.BookmarkEnd.ParentNode.InsertAfter(currentNode, endNode);
    endNode = currentNode;
    currentNode = nextNode;
}
```

यहां, हम नोड्स को इधर-उधर ले जाते हैं ताकि यह सुनिश्चित हो सके कि शर्त बुकमार्क की गई सामग्री को ठीक से शामिल करती है।

## चरण 5: मेल मर्ज निष्पादित करना

अंत में, हम बुकमार्क का मान निर्धारित करने के लिए मेल मर्ज निष्पादित करेंगे और यह निर्धारित करेंगे कि सामग्री दिखाई जानी चाहिए या छिपाई जानी चाहिए।

```csharp
doc.MailMerge.Execute(new[] { "MyBookmark" }, new object[] { "true" });
```

यह चरण बुकमार्क मान को "सत्य" पर सेट करता है, जो हमारी स्थिति के आधार पर सामग्री को दृश्यमान बना देगा।

## चरण 6: दस्तावेज़ को सहेजना

सभी जोड़-तोड़ के बाद, अंतिम चरण संशोधित दस्तावेज़ को सहेजना है।

```csharp
doc.Save("ShowHideBookmarkedContent.docx");
```

यहां, हम परिवर्तनों को इंगित करने के लिए दस्तावेज़ को वर्णनात्मक फ़ाइल नाम से सहेजते हैं।

## निष्कर्ष

 और बस! आपने सफलतापूर्वक सीख लिया है कि .NET के लिए Aspose.Words का उपयोग करके Word दस्तावेज़ में बुकमार्क की गई सामग्री को कैसे दिखाया या छिपाया जाए। इस ट्यूटोरियल में दस्तावेज़ बनाना, बुकमार्क जोड़ना, सशर्त फ़ील्ड सम्मिलित करना, नोड्स को पुनर्व्यवस्थित करना और मेल मर्ज निष्पादित करना शामिल है। Aspose.Words में बहुत सारी सुविधाएँ हैं, इसलिए इसे एक्सप्लोर करने में संकोच न करें[एपीआई दस्तावेज़ीकरण](https://reference.aspose.com/words/net/) अधिक उन्नत क्षमताओं के लिए.

## पूछे जाने वाले प्रश्न

### 1. .NET के लिए Aspose.Words क्या है?

Aspose.Words for .NET एक शक्तिशाली लाइब्रेरी है जो डेवलपर्स को प्रोग्रामेटिक रूप से Word दस्तावेज़ बनाने, संशोधित करने और परिवर्तित करने की अनुमति देती है। इसका व्यापक रूप से दस्तावेज़ स्वचालन कार्यों के लिए उपयोग किया जाता है।

### 2. क्या मैं .NET के लिए Aspose.Words का निःशुल्क उपयोग कर सकता हूँ?

 आप .NET के लिए Aspose.Words का उपयोग करके प्रयास कर सकते हैं[मुफ्त परीक्षण](https://releases.aspose.com/)दीर्घकालिक उपयोग के लिए, आपको लाइसेंस खरीदना होगा।

### 3. मैं बुकमार्क के अन्य गुणों को कैसे संशोधित कर सकता हूँ?

 Aspose.Words आपको बुकमार्क के विभिन्न गुणों, जैसे कि उसका टेक्स्ट और स्थान, में हेरफेर करने की अनुमति देता है।[एपीआई दस्तावेज़ीकरण](https://reference.aspose.com/words/net/) विस्तृत निर्देशों के लिए कृपया देखें.

### 4. मैं .NET के लिए Aspose.Words का समर्थन कैसे प्राप्त करूं?

आप यहां जाकर सहायता प्राप्त कर सकते हैं[Aspose समर्थन मंच](https://forum.aspose.com/c/words/8).

### 5. क्या मैं .NET के लिए Aspose.Words के साथ अन्य प्रकार की सामग्री में हेरफेर कर सकता हूं?

हां, .NET के लिए Aspose.Words विभिन्न प्रकार की सामग्री हेरफेर का समर्थन करता है, जिसमें पाठ, चित्र, तालिकाएं और बहुत कुछ शामिल है।
