इस लेख में, हम ऊपर दिए गए C# स्रोत कोड का पता लगाएंगे ताकि यह समझा जा सके कि Aspose.Words for .NET लाइब्रेरी में Untangle Row Bookmarks फ़ंक्शन का उपयोग कैसे किया जाए। यह फ़ंक्शन बुकमार्क की पंक्तियों के सिरों को बुकमार्क की शुरुआत के समान पंक्ति में रखना संभव बनाता है।

## आवश्यक शर्तें

- C# भाषा का मूलभूत ज्ञान.
- Aspose.Words लाइब्रेरी के साथ .NET विकास वातावरण स्थापित।

## चरण 1: दस्तावेज़ लोड करना

 हम उपयोग करते हैं`Document` फ़ाइल से मौजूदा दस्तावेज़ लोड करने के लिए क्लास:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Table column bookmarks.docx");
```

## चरण 2: बुकमार्क की लाइन खोलना

 हम उपयोग करते हैं`Untangle` बुकमार्क को पंक्तियों से अलग करने के लिए फ़ंक्शन। यह फ़ंक्शन बुकमार्क के आरंभिक भाग वाली पंक्तियों के बुकमार्क सिरों को उसी पंक्ति में रखने का कस्टम कार्य करता है:

```csharp
Untangle(doc);
```

## चरण 3: बुकमार्क द्वारा पंक्ति हटाएं

 हम उपयोग करते हैं`DeleteRowByBookmark` किसी विशिष्ट पंक्ति को उसके बुकमार्क द्वारा हटाने के लिए फ़ंक्शन:

```csharp
DeleteRowByBookmark(doc, "ROW2");
```

## चरण 4: अन्य बुकमार्क्स की अखंडता की जाँच करें

हम यह जाँच कर सत्यापित करते हैं कि अन्य बुकमार्क क्षतिग्रस्त नहीं हुए हैं, कि बुकमार्क का अंतिम सिरा अभी भी मौजूद है या नहीं:

```csharp
if (doc.Range.Bookmarks["ROW1"].BookmarkEnd == null)
throw new Exception("Wrong, the end of the bookmark was deleted.");

doc.Save(dataDir + "WorkingWithBookmarks.UntangleRowBookmarks.docx");
```

### .NET के लिए Aspose.Words का उपयोग करके Untangle Row Bookmarks के लिए उदाहरण स्रोत कोड

.NET के लिए Aspose.Words का उपयोग करके बुकमार्क को लाइनों से अलग करने के लिए यहां पूर्ण नमूना स्रोत कोड दिया गया है:


```csharp

	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document(dataDir + "Table column bookmarks.docx");

	//यह पंक्ति बुकमार्क के अंत को बुकमार्क के आरंभ वाली पंक्ति में रखने का कस्टम कार्य निष्पादित करता है।
	Untangle(doc);

	// अब हम किसी अन्य पंक्ति के बुकमार्क को नुकसान पहुंचाए बिना आसानी से एक बुकमार्क द्वारा पंक्तियों को हटा सकते हैं।
	DeleteRowByBookmark(doc, "ROW2");

	// यह केवल यह जांचने के लिए है कि अन्य बुकमार्क क्षतिग्रस्त तो नहीं हुआ है।
	if (doc.Range.Bookmarks["ROW1"].BookmarkEnd == null)
		throw new Exception("Wrong, the end of the bookmark was deleted.");

	doc.Save(dataDir + "WorkingWithBookmarks.UntangleRowBookmarks.docx");

```

#### स्रोत कोड को सुलझाएँ
```csharp

private void Untangle(Document doc)
        {
            foreach (Bookmark bookmark in doc.Range.Bookmarks)
            {
                // बुकमार्क और बुकमार्क अंत नोड दोनों की मूल पंक्ति प्राप्त करें।
                Row row1 = (Row) bookmark.BookmarkStart.GetAncestor(typeof(Row));
                Row row2 = (Row) bookmark.BookmarkEnd.GetAncestor(typeof(Row));

                // यदि दोनों पंक्तियाँ ठीक पाई जाती हैं, तथा बुकमार्क का आरंभ और अंत आसन्न पंक्तियों में समाहित है,
                // बुकमार्क अंत नोड को शीर्ष पंक्ति के अंतिम सेल में अंतिम पैराग्राफ के अंत में ले जाएं।
                if (row1 != null && row2 != null && row1.NextSibling == row2)
                    row1.LastCell.LastParagraph.AppendChild(bookmark.BookmarkEnd);
            }
        }

```

#### DeleteRowByBookmark स्रोत कोड
```csharp

 private void DeleteRowByBookmark(Document doc, string bookmarkName)
        {
            Bookmark bookmark = doc.Range.Bookmarks[bookmarkName];

            Row row = (Row) bookmark?.BookmarkStart.GetAncestor(typeof(Row));
            row?.Remove();
        }

```
## निष्कर्ष

इस लेख में, हमने C# स्रोत कोड का पता लगाया ताकि यह समझा जा सके कि .NET के लिए Aspose.Words की अनटंगल रो बुकमार्क सुविधा का उपयोग कैसे करें। हमने रो बुकमार्क को अनटंगल करने और अन्य बुकमार्क को नुकसान पहुँचाए बिना एक विशिष्ट पंक्ति को हटाने के लिए चरण-दर-चरण मार्गदर्शिका का पालन किया।

### वर्ड डॉक्यूमेंट में अनटैंगल रो बुकमार्क्स के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: क्या अनस्क्रैम्बल रो बुकमार्क्स केवल तालिकाओं में रो बुकमार्क्स के साथ ही काम करता है?

उत्तर: हां, अनटैंगल रो बुकमार्क्स सुविधा विशेष रूप से टेबल में मौजूद रो बुकमार्क्स को अनटैंगल करने के लिए डिज़ाइन की गई है। इस फ़ंक्शन का उपयोग सरणियों में लाइन बुकमार्क्स को संसाधित करने और यह सुनिश्चित करने के लिए किया जा सकता है कि बुकमार्क के अंत बुकमार्क की शुरुआत के समान पंक्ति में हों।

#### प्रश्न: क्या अनस्क्रैम्बल लाइन बुकमार्क्स फ़ंक्शन मूल दस्तावेज़ की सामग्री को संशोधित करता है?

उत्तर: हां, अनस्क्रैम्बल लाइन बुकमार्क्स फ़ंक्शन लाइन बुकमार्क्स के सिरों को बुकमार्क्स की शुरुआत के समान पंक्ति में रखने के लिए मूल दस्तावेज़ को संशोधित करता है। इस सुविधा को लागू करने से पहले दस्तावेज़ की बैकअप कॉपी सहेजना सुनिश्चित करें।

#### प्रश्न: मैं अपने वर्ड दस्तावेज़ में लाइन बुकमार्क्स की पहचान कैसे कर सकता हूँ?

उत्तर: पंक्ति बुकमार्क का उपयोग आम तौर पर तालिकाओं में विशिष्ट अनुभागों को चिह्नित करने के लिए किया जाता है। आप दस्तावेज़ में बुकमार्क ब्राउज़ करके और यह देखने के लिए जाँच करके पंक्ति बुकमार्क की पहचान कर सकते हैं कि बुकमार्क तालिका पंक्तियों में हैं या नहीं।

#### प्रश्न: क्या गैर-आसन्न तालिकाओं में पंक्ति बुकमार्क को सुलझाना संभव है?

उत्तर: इस लेख में प्रस्तुत अनटैंगल रो बुकमार्क्स फ़ंक्शन को आसन्न तालिकाओं में पंक्ति बुकमार्क्स को अनटैंगल करने के लिए डिज़ाइन किया गया है। गैर-आसन्न तालिकाओं में पंक्ति बुकमार्क्स को अलग करने के लिए, दस्तावेज़ की संरचना के आधार पर कोड में अतिरिक्त समायोजन की आवश्यकता हो सकती है।

#### प्रश्न: पंक्ति बुकमार्क्स के खुल जाने के बाद मैं उन पर और क्या हेरफेर कर सकता हूँ?

उत्तर: एक बार लाइन बुकमार्क खुल जाने के बाद, आप आवश्यकतानुसार अलग-अलग हेरफेर कर सकते हैं। इसमें बुकमार्क की गई लाइनों में संपादन, हटाना या सामग्री जोड़ना शामिल हो सकता है। दस्तावेज़ के बाकी हिस्सों पर किसी भी अवांछित प्रभाव से बचने के लिए लाइन बुकमार्क को सावधानी से संभालना सुनिश्चित करें।
