इस लेख में, हम ऊपर दिए गए C# स्रोत कोड का पता लगाएंगे ताकि यह समझ सकें कि Aspose.Words for .NET लाइब्रेरी में Delete Row By Bookmark फ़ंक्शन का उपयोग कैसे करें। यह सुविधा आपको वर्ड दस्तावेज़ में किसी विशिष्ट बुकमार्क के आधार पर टेबल रो को हटाने की अनुमति देती है।

## आवश्यक शर्तें

- C# भाषा का मूलभूत ज्ञान.
- Aspose.Words लाइब्रेरी के साथ .NET विकास वातावरण स्थापित।

## चरण 1: बुकमार्क प्राप्त करना

 हम उपयोग करते हैं`Bookmarks` दस्तावेज़ श्रेणी की संपत्ति का उपयोग उस विशिष्ट बुकमार्क को प्राप्त करने के लिए करें जिसका उपयोग हम तालिका पंक्ति को हटाने के लिए करना चाहते हैं:

```csharp
Bookmark bookmark = doc.Range.Bookmarks[bookmarkName];
```

## चरण 2: तालिका पंक्ति हटाना

 हम उपयोग करते हैं`GetAncestor` पाने की विधि`Row` बुकमार्क का पैरेंट एलिमेंट टाइप करें। इसके बाद, हम इसका उपयोग करते हैं`Remove` तालिका पंक्ति को हटाने की विधि:

```csharp
Row row = (Row)bookmark?.BookmarkStart.GetAncestor(typeof(Row));
row?.Remove();
```

### .NET के लिए Aspose.Words का उपयोग करके बुकमार्क द्वारा पंक्ति को हटाने के लिए उदाहरण स्रोत कोड

.NET के लिए Aspose.Words का उपयोग करके किसी विशिष्ट बुकमार्क के आधार पर तालिका पंक्ति को हटाने का प्रदर्शन करने के लिए यहां पूर्ण नमूना स्रोत कोड दिया गया है:

```csharp

	Bookmark bookmark = doc.Range.Bookmarks[bookmarkName];

	Row row = (Row) bookmark?.BookmarkStart.GetAncestor(typeof(Row));
	row?.Remove();
        
```

## निष्कर्ष

इस लेख में, हमने .NET के लिए Aspose.Words के Delete Row By Bookmark फ़ंक्शन का उपयोग करने के तरीके को समझने के लिए C# स्रोत कोड का पता लगाया है। हमने दस्तावेज़ में किसी विशिष्ट बुकमार्क के आधार पर तालिका पंक्ति को हटाने के लिए चरण-दर-चरण मार्गदर्शिका का पालन किया।

### वर्ड दस्तावेज़ में बुकमार्क द्वारा पंक्ति हटाने के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: क्या मैं एक ही बुकमार्क का उपयोग करके एकाधिक पंक्तियों को हटा सकता हूँ?

उत्तर: हां, आप एक ही बुकमार्क का उपयोग करके कई पंक्तियों को हटा सकते हैं। हालाँकि, आपको हटाए जाने वाली पंक्तियों की संख्या निर्धारित करने और दिए गए कोड स्निपेट में आवश्यक समायोजन करने के लिए अपने कोड में तर्क को संभालने की आवश्यकता है।

#### प्रश्न: यदि दस्तावेज़ में बुकमार्क मौजूद नहीं है तो क्या होगा?

उत्तर: यदि निर्दिष्ट बुकमार्क दस्तावेज़ में मौजूद नहीं है, तो कोड स्निपेट बुकमार्क ऑब्जेक्ट के लिए शून्य मान लौटाएगा। इसलिए, आपको टेबल पंक्ति को हटाने का प्रयास करने से पहले उचित जाँच जोड़कर अपने कोड में इस परिदृश्य को संभालने की आवश्यकता है।

#### प्रश्न: क्या Aspose.Words लाइब्रेरी का उपयोग निःशुल्क है?

 उत्तर: Aspose.Words लाइब्रेरी एक व्यावसायिक लाइब्रेरी है, और आपको इसे अपने प्रोजेक्ट में उपयोग करने के लिए वैध लाइसेंस की आवश्यकता हो सकती है। आप यहाँ जा सकते हैं[.NET API संदर्भों के लिए Aspose.Words](https://reference.aspose.com/words/net/) उनके लाइसेंसिंग विकल्पों और मूल्य निर्धारण के बारे में अधिक जानने के लिए।

#### प्रश्न: क्या मैं Word दस्तावेज़ के किसी विशिष्ट अनुभाग में तालिका से पंक्तियों को हटा सकता हूँ?

उत्तर: हाँ, आप Word दस्तावेज़ के किसी विशिष्ट अनुभाग में किसी तालिका से पंक्तियाँ हटा सकते हैं। आप उस अनुभाग के भीतर उपयुक्त श्रेणी या बुकमार्क का उपयोग करके किसी विशिष्ट अनुभाग को लक्षित करने के लिए दिए गए कोड स्निपेट को संशोधित कर सकते हैं।
