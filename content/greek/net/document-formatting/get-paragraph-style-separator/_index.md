## Βήμα 2: Εύρεση διαχωριστικών στυλ παραγράφου

Τώρα θα κάνουμε βρόχο σε όλες τις παραγράφους του εγγράφου και θα ελέγξουμε αν μια παράγραφος είναι διαχωριστικό στυλ. Δείτε πώς:

```csharp
foreach(Paragraph paragraph in doc.GetChildNodes(NodeType.Paragraph, true))
{
     if (paragraph.BreakIsStyleSeparator)
     {
         Console.WriteLine("Separator found!");
     }
}
```
