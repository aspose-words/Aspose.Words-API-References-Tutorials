Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στον τρόπο εισαγωγής ενός εγγράφου σε άλλο έγγραφο κατά την αντικατάσταση χρησιμοποιώντας τη δυνατότητα Εισαγωγή εγγράφου κατά την αντικατάσταση του Aspose.Words για .NET. Ακολουθήστε τα παρακάτω βήματα για να κατανοήσετε τον πηγαίο κώδικα και να εκτελέσετε την εισαγωγή του εγγράφου.

## Βήμα 1: Φόρτωση του κύριου εγγράφου

Για να ξεκινήσετε, καθορίστε τον κατάλογο για τα έγγραφά σας και φορτώστε το κύριο έγγραφο σε ένα αντικείμενο Document. Δείτε πώς:

```csharp
// Διαδρομή στον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document mainDoc = new Document(MyDir + "Document insert 1.docx");
```

## Βήμα 2: Διαμορφώστε τις επιλογές αναζήτησης και αντικατάστασης

Τώρα θα διαμορφώσουμε τις επιλογές εύρεσης και αντικατάστασης καθορίζοντας την κατεύθυνση αναζήτησης και την αντικατάσταση επανάκλησης για την εισαγωγή ενός εγγράφου σε άλλο έγγραφο. Δείτε πώς:

```csharp
// Διαμορφώστε τις επιλογές αναζήτησης και αντικατάστασης.
FindReplaceOptions options = new FindReplaceOptions
{
Direction = FindReplaceDirection.Backward,
ReplacingCallback = new InsertDocumentAtReplaceHandler()
};
```

## Βήμα 3: Κλήση της μεθόδου αντικατάστασης

Τώρα θα καλέσουμε τη μέθοδο αντικατάστασης για να βρούμε και να αντικαταστήσουμε το καθορισμένο κείμενο με μια κενή συμβολοσειρά, χρησιμοποιώντας τις διαμορφωμένες επιλογές. Δείτε πώς:

```csharp
mainDoc.Range.Replace(new Regex("\\[MY_DOCUMENT\\]"), "", options);
mainDoc.Save(dataDir + "CloneAndCombineDocuments.InsertDocumentAtReplace.docx");
```

### Παράδειγμα πηγαίου κώδικα για το Insert Document At Replace χρησιμοποιώντας το Aspose.Words για .NET

Ακολουθεί ο πλήρης πηγαίος κώδικας για τη δυνατότητα Εισαγωγής εγγράφου κατά την αντικατάσταση του Aspose.Words για .NET:

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document mainDoc = new Document(MyDir + "Document insertion 1.docx");

// Ορίστε τις επιλογές εύρεσης και αντικατάστασης.
FindReplaceOptions options = new FindReplaceOptions
{
	Direction = FindReplaceDirection.Backward, 
	ReplacingCallback = new InsertDocumentAtReplaceHandler()
};

// Καλέστε τη μέθοδο αντικατάστασης.
mainDoc.Range.Replace(new Regex("\\[MY_DOCUMENT\\]"), "", options);
mainDoc.Save(dataDir + "CloneAndCombineDocuments.InsertDocumentAtReplace.docx");
```

Διαμορφώνοντας τις επιλογές εύρεσης και αντικατάστασης και παρέχοντας τα απαραίτητα δεδομένα, μπορείτε να συναρμολογήσετε δυναμικά έγγραφα αντικαθιστώντας συγκεκριμένα σύμβολα κράτησης θέσης με τα περιεχόμενα άλλων προτύπων εγγράφων ή ενοτήτων. ### Συχνές ερωτήσεις

#### Ε: Ποιος είναι ο σκοπός της εισαγωγής ενός εγγράφου σε άλλο έγγραφο κατά την αντικατάσταση;

Α: Η εισαγωγή ενός εγγράφου σε άλλο έγγραφο κατά την αντικατάσταση σάς επιτρέπει να αντικαταστήσετε δυναμικά ένα συγκεκριμένο σύμβολο κράτησης θέσης με τα περιεχόμενα ενός ξεχωριστού εγγράφου. Αυτή η δυνατότητα είναι ιδιαίτερα χρήσιμη όταν θέλετε να συναρμολογήσετε ένα μεγαλύτερο έγγραφο συνδυάζοντας διάφορα προκαθορισμένα πρότυπα εγγράφων ή ενότητες σε συγκεκριμένα σύμβολα κράτησης θέσης.

#### Ε: Πώς μπορώ να εισαγάγω ένα έγγραφο σε άλλο έγγραφο κατά την αντικατάσταση χρησιμοποιώντας το Aspose.Words για .NET;

Α: Για να εισαγάγετε ένα έγγραφο σε άλλο έγγραφο κατά την αντικατάσταση χρησιμοποιώντας το Aspose.Words για .NET, ακολουθήστε τα εξής βήματα:
1. Φορτώστε το κύριο έγγραφο που περιέχει τα σύμβολα κράτησης θέσης σε ένα αντικείμενο Document.
2. Διαμορφώστε τις επιλογές εύρεσης και αντικατάστασης, συμπεριλαμβανομένης της κατεύθυνσης αναζήτησης και αντικαταστήστε την επανάκληση για να χειριστείτε την εισαγωγή εγγράφου.
3. Καλέστε τη μέθοδο αντικατάστασης με το κατάλληλο μοτίβο αναζήτησης, αντικαθιστώντας τα σύμβολα κράτησης θέσης με μια κενή συμβολοσειρά, χρησιμοποιώντας τις διαμορφωμένες επιλογές.

#### Ε: Μπορώ να προσαρμόσω τη συμπεριφορά εισαγωγής κατά την αντικατάσταση;

Α: Ναι, μπορείτε να προσαρμόσετε τη συμπεριφορά εισαγωγής κατά την αντικατάσταση εφαρμόζοντας ένα προσαρμοσμένο ReplacingCallback. Με κληρονομιά από τη διεπαφή IReplacingCallback, μπορείτε να ελέγξετε τον τρόπο εισαγωγής και συγχώνευσης των εγγράφων με βάση τις συγκεκριμένες απαιτήσεις σας κατά την αντικατάσταση των συμβόλων κράτησης θέσης.

#### Ε: Μπορώ να αντικαταστήσω πολλά σύμβολα κράτησης θέσης με διαφορετικά έγγραφα;

Α: Ναι, μπορείτε να αντικαταστήσετε πολλαπλά σύμβολα κράτησης θέσης με διαφορετικά έγγραφα, καθορίζοντας τα κατάλληλα μοτίβα αναζήτησης για κάθε σύμβολο κράτησης θέσης και παρέχοντας τα αντίστοιχα έγγραφα προς εισαγωγή.
