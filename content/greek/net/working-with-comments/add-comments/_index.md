Σε αυτό το περιεκτικό σεμινάριο, θα μάθετε πώς να προσθέτετε σχόλια σε ένα έγγραφο του Word χρησιμοποιώντας το Aspose.Words για .NET. Θα σας καθοδηγήσουμε στη διαδικασία και θα σας παρέχουμε τα απαραίτητα αποσπάσματα κώδικα C#. Μέχρι το τέλος αυτού του οδηγού, θα μπορείτε να εισάγετε σχόλια και να προσαρμόσετε το περιεχόμενό τους στα έγγραφά σας.

## Προαπαιτούμενα
Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:
- Η βιβλιοθήκη Aspose.Words for .NET είναι εγκατεστημένη στο σύστημά σας.

## Βήμα 1: Δημιουργήστε ένα νέο Document and DocumentBuilder
Για να ξεκινήσετε, δημιουργήστε ένα νέο έγγραφο χρησιμοποιώντας την κλάση Document και αρχικοποιήστε ένα αντικείμενο DocumentBuilder:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## Βήμα 2: Προσθήκη περιεχομένου στο Έγγραφο
Στη συνέχεια, προσθέστε το επιθυμητό περιεχόμενο στο έγγραφο χρησιμοποιώντας το αντικείμενο DocumentBuilder. Σε αυτό το παράδειγμα, προσθέτουμε κάποιο κείμενο:

```csharp
builder.Write("Some text is added.");
```

## Βήμα 3: Δημιουργήστε ένα σχόλιο και προσθέστε περιεχόμενο
Για να προσθέσετε ένα σχόλιο, δημιουργήστε μια παρουσία της κλάσης Comment, περνώντας το αντικείμενο Document, το όνομα του συγγραφέα, τα αρχικά του συγγραφέα και την τρέχουσα ημερομηνία:

```csharp
Comment comment = new Comment(doc, "Awais Hafeez", "AH", DateTime.Today);
```

Στη συνέχεια, προσθέστε το σχόλιο στην τρέχουσα παράγραφο:

```csharp
builder.CurrentParagraph.AppendChild(comment);
```

Προσθέστε περιεχόμενο στο σχόλιο, όπως παράγραφο και κείμενο:

```csharp
comment.Paragraphs.Add(new Paragraph(doc));
comment.FirstParagraph.Runs.Add(new Run(doc, "Comment text."));
```

## Βήμα 4: Αποθηκεύστε το έγγραφο
Αφού προσθέσετε το σχόλιο και τα περιεχόμενά του, αποθηκεύστε το έγγραφο σε ένα αρχείο χρησιμοποιώντας τη μέθοδο Save της κλάσης Document:

```csharp
doc.Save(dataDir + "WorkingWithComments.AddComments.docx");
```

## Παράδειγμα πηγαίου κώδικα για προσθήκη σχολίων χρησιμοποιώντας το Aspose.Words για .NET
Εδώ είναι ο πλήρης πηγαίος κώδικας για την προσθήκη σχολίων χρησιμοποιώντας το Aspose.Words για .NET:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

builder.Write("Some text is added.");

Comment comment = new Comment(doc, "Awais Hafeez", "AH", DateTime.Today);
builder.CurrentParagraph.AppendChild(comment);

comment.Paragraphs.Add(new Paragraph(doc));
comment.FirstParagraph.Runs.Add(new Run(doc, "Comment text."));

doc.Save(dataDir + "WorkingWithComments.AddComments.docx");
```

Ακολουθώντας τον οδηγό βήμα προς βήμα και χρησιμοποιώντας τον παρεχόμενο πηγαίο κώδικα, μπορείτε τώρα να εισάγετε σχόλια και να προσαρμόσετε το περιεχόμενό τους στα έγγραφά σας.

Τα σχόλια είναι χρήσιμα για τη συνεργασία, την παροχή πρόσθετων πληροφοριών ή τη δημιουργία σημειώσεων σε ένα έγγραφο. Πειραματιστείτε με διαφορετικά ονόματα συγγραφέων, αρχικά και περιεχόμενο σχολίων για να καλύψετε τις συγκεκριμένες απαιτήσεις σας.
