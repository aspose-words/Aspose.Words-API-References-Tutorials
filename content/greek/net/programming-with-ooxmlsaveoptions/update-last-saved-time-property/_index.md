Σε αυτό το σεμινάριο, θα εξερευνήσουμε τον παρεχόμενο πηγαίο κώδικα C# για να ενημερώσουμε την ιδιότητα τελευταίας αποθήκευσης χρόνου κατά την αποθήκευση ενός εγγράφου χρησιμοποιώντας το Aspose.Words για .NET. Αυτή η δυνατότητα σάς επιτρέπει να ενημερώνετε αυτόματα την ιδιότητα τελευταίας εξοικονόμησης χρόνου του εγγράφου που δημιουργήθηκε.

## Βήμα 1: Ρύθμιση περιβάλλοντος

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε ρυθμίσει το περιβάλλον ανάπτυξης με το Aspose.Words για .NET. Βεβαιωθείτε ότι έχετε προσθέσει τις απαραίτητες αναφορές και έχετε εισαγάγει τους κατάλληλους χώρους ονομάτων.

## Βήμα 2: Φόρτωση του εγγράφου

```csharp
// Διαδρομή στον κατάλογο των εγγράφων σας
string dataDir = "YOUR DOCUMENTS DIRECTORY";

Document doc = new Document(dataDir + "Document.docx");
```

 Σε αυτό το βήμα, φορτώνουμε το έγγραφο χρησιμοποιώντας το`Document` μέθοδο και περνώντας τη διαδρομή προς το αρχείο DOCX για φόρτωση.

## Βήμα 3: Διαμόρφωση επιλογών δημιουργίας αντιγράφων ασφαλείας OOXML

```csharp
OoxmlSaveOptions saveOptions = new OoxmlSaveOptions { UpdateLastSavedTimeProperty = true };
```

 Σε αυτό το βήμα, διαμορφώνουμε τις επιλογές αποθήκευσης OOXML χρησιμοποιώντας το`OoxmlSaveOptions` τάξη. Ενεργοποιούμε την αυτόματη ενημέρωση της τελευταίας ιδιότητας εξοικονόμησης χρόνου με ρύθμιση`UpdateLastSavedTimeProperty` προς την`true`.

## Βήμα 4: Αποθήκευση εγγράφου με ενημερωμένη ιδιότητα

```csharp
doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.UpdateLastSavedTimeProperty.docx", saveOptions);
```

 Σε αυτό το τελευταίο βήμα, αποθηκεύουμε το έγγραφο χρησιμοποιώντας το`Save` μέθοδο και περνώντας τη διαδρομή προς το αρχείο εξόδου με το`.docx` επέκταση, μαζί με τις καθορισμένες επιλογές αποθήκευσης.

Τώρα μπορείτε να εκτελέσετε τον πηγαίο κώδικα για αυτόματη ενημέρωση της τελευταίας ιδιότητας χρόνου αποθήκευσης κατά την αποθήκευση ενός εγγράφου. Το αρχείο που προκύπτει θα αποθηκευτεί στον καθορισμένο κατάλογο με το όνομα "WorkingWithOoxmlSaveOptions.UpdateLastSavedTimeProperty.docx".

### Δείγμα πηγαίου κώδικα για Ενημέρωση ιδιότητας τελευταίας αποθηκευμένης ώρας χρησιμοποιώντας το Aspose.Words για .NET 

```csharp

// Διαδρομή στον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY"; 

Document doc = new Document(dataDir + "Document.docx");

OoxmlSaveOptions saveOptions = new OoxmlSaveOptions { UpdateLastSavedTimeProperty = true };

doc.Save(dataDir + "WorkingWithOoxmlSaveOptions.UpdateLastSavedTimeProperty.docx", saveOptions);
            
        
```

Ενεργοποιώντας αυτήν τη δυνατότητα με τις επιλογές αποθήκευσης OOXML, μπορείτε να διασφαλίσετε ότι η ιδιότητα τελευταίου χρόνου αποθήκευσης ενημερώνεται αυτόματα στο έγγραφο που δημιουργείται.

Η ενημέρωση της ιδιότητας τελευταίας εξοικονόμησης χρόνου μπορεί να είναι χρήσιμη για την παρακολούθηση αλλαγών και εκδόσεων ενός εγγράφου. Επίσης, παρακολουθεί την τελευταία αποθήκευση του εγγράφου, κάτι που μπορεί να είναι χρήσιμο σε διάφορα σενάρια.
