Κατά την επεξεργασία λέξεων με έγγραφα Word σε μια εφαρμογή C#, μπορεί να είναι απαραίτητο να χρησιμοποιήσετε έναν προσωρινό φάκελο για την αποθήκευση προσωρινών αρχείων που δημιουργούνται κατά την επεξεργασία εγγράφων. Με τη βιβλιοθήκη Aspose.Words για .NET, μπορείτε εύκολα να καθορίσετε έναν προσωρινό φάκελο χρησιμοποιώντας τις επιλογές φόρτωσης LoadOptions. Σε αυτόν τον οδηγό βήμα προς βήμα, θα σας δείξουμε πώς να χρησιμοποιείτε τον πηγαίο κώδικα του Aspose.Words για .NET C# για τη φόρτωση ενός εγγράφου χρησιμοποιώντας έναν προσωρινό φάκελο που έχει καθοριστεί χρησιμοποιώντας τις επιλογές φόρτωσης LoadOptions.

## Κατανόηση της βιβλιοθήκης Aspose.Words

Πριν βουτήξετε στον κώδικα, είναι σημαντικό να κατανοήσετε τη βιβλιοθήκη Aspose.Words για το .NET. Το Aspose.Words είναι μια ισχυρή βιβλιοθήκη για τη δημιουργία, επεξεργασία, μετατροπή και προστασία εγγράφων του Word σε διαφορετικές πλατφόρμες, συμπεριλαμβανομένου του .NET. Προσφέρει πολλές δυνατότητες χειρισμού εγγράφων, όπως εισαγωγή κειμένου, αλλαγή μορφοποίησης, προσθήκη ενοτήτων και πολλά άλλα.

## Διαμόρφωση επιλογών φόρτωσης

Το πρώτο βήμα είναι να διαμορφώσουμε τις επιλογές φόρτωσης για το έγγραφό μας. Χρησιμοποιήστε την κλάση LoadOptions για να καθορίσετε τις παραμέτρους φόρτωσης. Στην περίπτωσή μας, πρέπει να ορίσουμε την ιδιότητα TempFolder στη διαδρομή του επιθυμητού προσωρινού φακέλου. Δείτε πώς να το κάνετε:

```csharp
LoadOptions loadOptions = new LoadOptions { TempFolder = ArtifactsDir };
```

Δημιουργούμε ένα νέο αντικείμενο LoadOptions και ορίζουμε την ιδιότητα TempFolder στη διαδρομή του επιθυμητού προσωρινού φακέλου.

## Μεταφορτώστε το έγγραφο χρησιμοποιώντας τον καθορισμένο προσωρινό φάκελο

Τώρα που έχουμε διαμορφώσει τις επιλογές φόρτωσης, μπορούμε να φορτώσουμε το έγγραφο χρησιμοποιώντας την κλάση Document και να καθορίσουμε τις επιλογές φόρτωσης. Εδώ είναι ένα παράδειγμα:

```csharp
Document doc = new Document(dataDir + "Document.docx", loadOptions);
```

Σε αυτό το παράδειγμα, φορτώνουμε το έγγραφο "Document.docx" που βρίσκεται στον κατάλογο εγγράφων χρησιμοποιώντας τις καθορισμένες επιλογές φόρτωσης.

### Παράδειγμα πηγαίου κώδικα για το LoadOptions με τη λειτουργία "Use Temp Folder" χρησιμοποιώντας Aspose.Words για .NET

```csharp
// Διαδρομή στον κατάλογο των εγγράφων σας
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// Διαμορφώστε τις επιλογές φόρτωσης με τη λειτουργία "Χρήση φακέλου θερμοκρασίας".
LoadOptions loadOptions = new LoadOptions { TempFolder = ArtifactsDir };

// Φορτώστε το έγγραφο χρησιμοποιώντας έναν καθορισμένο προσωρινό φάκελο
Document doc = new Document(dataDir + "Document.docx", loadOptions);
```
