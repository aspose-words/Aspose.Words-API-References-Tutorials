Οι κουκκίδες εικόνας είναι μια ευρέως χρησιμοποιούμενη δυνατότητα στα έγγραφα του Word για την προσθήκη προσαρμοσμένων κουκκίδων. Ωστόσο, σε ορισμένες περιπτώσεις μπορεί να είναι απαραίτητο να απενεργοποιήσετε την εγγραφή κουκκίδων εικόνας κατά τον χειρισμό εγγράφων χρησιμοποιώντας τη Βιβλιοθήκη Aspose.Words για .NET. Σε αυτόν τον οδηγό βήμα προς βήμα, θα εξηγήσουμε πώς να χρησιμοποιήσετε τον πηγαίο κώδικα του Aspose.Words C# για .NET για να απενεργοποιήσετε την αποθήκευση κουκκίδων εικόνας χρησιμοποιώντας τις επιλογές αποθήκευσης του DocSaveOptions.

## Κατανόηση της βιβλιοθήκης Aspose.Words

Πριν βουτήξετε στον κώδικα, είναι σημαντικό να κατανοήσετε τη βιβλιοθήκη Aspose.Words για το .NET. Το Aspose.Words είναι μια ισχυρή βιβλιοθήκη για τη δημιουργία, επεξεργασία, μετατροπή και προστασία εγγράφων του Word σε διαφορετικές πλατφόρμες, συμπεριλαμβανομένου του .NET. Προσφέρει πολλές δυνατότητες χειρισμού εγγράφων, όπως εισαγωγή κειμένου, αλλαγή μορφοποίησης, προσθήκη ενοτήτων και πολλά άλλα.

## Βήμα 1: Ρύθμιση του Καταλόγου Εγγράφων

Το πρώτο βήμα είναι να ορίσετε τον κατάλογο όπου βρίσκονται τα έγγραφά σας. Πρέπει να καθορίσετε την πλήρη διαδρομή καταλόγου. Για παράδειγμα :

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

Φροντίστε να αντικαταστήσετε τον "ΚΑΤΑΛΟΓΟ ΕΓΓΡΑΦΩΝ ΣΑΣ" με την πραγματική διαδρομή προς τον κατάλογο των εγγράφων σας.

## Βήμα 2: Φόρτωση του εγγράφου με κουκκίδες εικόνας

Στη συνέχεια, πρέπει να φορτώσετε το έγγραφο με κουκκίδες εικόνας. Χρησιμοποιήστε την κλάση Document για να φορτώσετε το έγγραφο από ένα αρχείο. Για παράδειγμα :

```csharp
Document doc = new Document(dataDir + "Image bullet points.docx");
```

Σε αυτό το παράδειγμα φορτώνουμε το έγγραφο από το αρχείο "Image bullet points.docx"

  που βρίσκεται στον κατάλογο εγγράφων.

## Βήμα 3: Διαμορφώστε τις επιλογές εγγραφής

Τώρα ας διαμορφώσουμε τις επιλογές αποθήκευσης για το έγγραφό μας. Χρησιμοποιήστε την κλάση DocSaveOptions για να καθορίσετε τις ρυθμίσεις αποθήκευσης. Για παράδειγμα :

```csharp
DocSaveOptions saveOptions = new DocSaveOptions { SavePictureBullet = false };
```

Σε αυτό το παράδειγμα, δημιουργούμε ένα νέο αντικείμενο DocSaveOptions και ορίζουμε την ιδιότητα SavePictureBullet σε false για να απενεργοποιήσουμε την αποθήκευση κουκκίδων εικόνας.

## Βήμα 4: Ενεργοποιήστε τη λειτουργία "Να μην αποθηκεύεται η κουκκίδα εικόνας".

Για να ενεργοποιήσετε τη δυνατότητα "Να μην αποθηκεύεται η κουκκίδα εικόνας", έχουμε ήδη διαμορφώσει τις επιλογές αποθήκευσης με το SavePictureBullet να έχει οριστεί σε false. Αυτό διασφαλίζει ότι οι κουκκίδες εικόνας δεν αποθηκεύονται στο τελικό έγγραφο.

## Βήμα 5: Αποθηκεύστε το έγγραφο

Τέλος, μπορείτε να αποθηκεύσετε το έγγραφο χρησιμοποιώντας τη μέθοδο Save της κλάσης Document. Καθορίστε την πλήρη διαδρομή προς το αρχείο και το επιθυμητό όνομα αρχείου. Για παράδειγμα :

```csharp
doc.Save(dataDir + "WorkingWithDocSaveOptions.DoNotSavePictureBullet.docx", saveOptions);
```

Φροντίστε να αντικαταστήσετε το "dataDir" με τη διαδρομή καταλόγου στα έγγραφά σας.

## Παράδειγμα πηγαίου κώδικα για τις επιλογές αποθήκευσης DocSaveOptions με τη λειτουργία "Να μην αποθηκεύεται Κουκκίδα εικόνας" χρησιμοποιώντας το Aspose.Words για .NET

```csharp
// Διαδρομή στον κατάλογο των εγγράφων σας
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// Φορτώστε το έγγραφο με κουκκίδες εικόνας
Document doc = new Document(dataDir + "Image bullet points.docx");

// Διαμορφώστε τις επιλογές αποθήκευσης με τη λειτουργία "Να μην αποθηκεύεται Κουκκίδα εικόνας".
DocSaveOptions saveOptions = new DocSaveOptions { SavePictureBullet = false };

// Αποθηκεύστε το έγγραφο με τις καθορισμένες επιλογές
doc.Save(dataDir + "WorkingWithDocSaveOptions.DoNotSavePictureBullet.docx", saveOptions);
```
