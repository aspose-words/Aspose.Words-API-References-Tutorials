Μέχρι το τέλος αυτού του σεμιναρίου, θα είστε επαγγελματίας στη συγχώνευση κελιών στα έγγραφα του Word μέσω προγραμματισμού. Λοιπόν, ας σηκώσουμε τα μανίκια και ας ξεκινήσουμε!

## Εισαγωγή χώρων ονομάτων

Πριν βουτήξουμε στον κώδικα, ας βεβαιωθούμε ότι έχουμε εισαγάγει τους απαραίτητους χώρους ονομάτων. Στο έργο σας C#, φροντίστε να συμπεριλάβετε:

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

Εντάξει, ας αναλύσουμε τη διαδικασία οριζόντιας συγχώνευσης κελιών πίνακα σε ένα έγγραφο του Word χρησιμοποιώντας το Aspose.Words για .NET.

## Βήμα 1: Ρύθμιση του εγγράφου σας

 Πρώτα πράγματα πρώτα, πρέπει να δημιουργήσουμε ένα νέο έγγραφο του Word και να αρχικοποιήσουμε το`DocumentBuilder`:

```csharp
// Διαδρομή στον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 Αυτό το απόσπασμα κώδικα δημιουργεί ένα νέο έγγραφο και προετοιμάζει το`DocumentBuilder` για δράση.

## Βήμα 2: Εισαγωγή του πρώτου κελιού

Στη συνέχεια, ξεκινάμε εισάγοντας το πρώτο κελί και επισημαίνοντάς το για οριζόντια συγχώνευση:

```csharp
builder.InsertCell();
builder.CellFormat.HorizontalMerge = CellMerge.First;
builder.Write("Text in merged cells.");
```

 Εδώ, εισάγουμε ένα νέο κελί και το ορίζουμε`HorizontalMerge`ιδιοκτησία σε`CellMerge.First`, υποδεικνύοντας ότι αυτό το κελί είναι η αρχή μιας συγχωνευμένης ακολουθίας κελιών.

## Βήμα 3: Εισαγωγή του συγχωνευμένου κελιού

Τώρα, εισάγουμε το κελί που θα συγχωνευθεί με το προηγούμενο:

```csharp
builder.InsertCell();
builder.CellFormat.HorizontalMerge = CellMerge.Previous;
builder.EndRow();
```

 Αυτό το κελί έχει ρυθμιστεί να συγχωνεύεται με το προηγούμενο κελί χρησιμοποιώντας`CellMerge.Previous` . Παρατηρήστε πώς τελειώνουμε τη σειρά με`builder.EndRow()`.

## Βήμα 4: Εισαγωγή μη συγχωνευμένων κελιών

Για να δείξουμε τη διαφορά, ας εισαγάγουμε μερικά μη συγχωνευμένα κελιά:

```csharp
builder.InsertCell();
builder.CellFormat.HorizontalMerge = CellMerge.None;
builder.Write("Text in one cell.");
builder.InsertCell();
builder.Write("Text in another cell.");
builder.EndRow();
```

Εδώ, εισάγουμε δύο κελιά χωρίς οριζόντια συγχώνευση. Αυτό δείχνει πώς συμπεριφέρονται τα κελιά όταν δεν αποτελούν μέρος μιας συγχωνευμένης ακολουθίας.

## Βήμα 5: Ολοκλήρωση του πίνακα

Τέλος, τελειώνουμε τον πίνακα και αποθηκεύουμε το έγγραφο:

```csharp
builder.EndTable();
doc.Save(dataDir + "WorkingWithTables.HorizontalMerge.docx");
```

Αυτό το απόσπασμα κώδικα συμπληρώνει τον πίνακα και αποθηκεύει το έγγραφο στον καθορισμένο κατάλογο.
