Όταν μετατρέπετε ένα έγγραφο του Word σε μορφή OpenDocument Text (ODT) σε μια εφαρμογή C#, ίσως θέλετε να καθορίσετε τη μονάδα μέτρησης που χρησιμοποιείται για μετρήσιμη μορφοποίηση και ιδιότητες περιεχομένου. Με τη βιβλιοθήκη Aspose.Words για .NET, μπορείτε εύκολα να καθορίσετε αυτήν τη λειτουργία χρησιμοποιώντας τις επιλογές αποθήκευσης OdtSaveOptions. Σε αυτόν τον οδηγό βήμα προς βήμα, θα σας καθοδηγήσουμε πώς να χρησιμοποιήσετε τον πηγαίο κώδικα του Aspose.Words για .NET C# για τη μετατροπή ενός εγγράφου του Word σε ODT, καθορίζοντας τη μονάδα μέτρησης χρησιμοποιώντας το OdtSaveOptions.

## Κατανόηση της βιβλιοθήκης Aspose.Words

Πριν βουτήξετε στον κώδικα, είναι σημαντικό να κατανοήσετε τη βιβλιοθήκη Aspose.Words για το .NET. Το Aspose.Words είναι μια ισχυρή βιβλιοθήκη για τη δημιουργία, επεξεργασία, μετατροπή και προστασία εγγράφων του Word σε διαφορετικές πλατφόρμες, συμπεριλαμβανομένου του .NET. Προσφέρει πολλές δυνατότητες χειρισμού εγγράφων, όπως εισαγωγή κειμένου, αλλαγή μορφοποίησης, προσθήκη ενοτήτων και πολλά άλλα.

## Φόρτωση του εγγράφου του Word

Το πρώτο βήμα είναι να φορτώσετε το έγγραφο του Word που θέλετε να μετατρέψετε σε ODT. Χρησιμοποιήστε την κλάση Document για να φορτώσετε το έγγραφο από το αρχείο προέλευσης. Εδώ είναι ένα παράδειγμα:

```csharp
Document doc = new Document(dataDir + "Document.docx");
```

Σε αυτό το παράδειγμα, φορτώνουμε το έγγραφο "Document.docx" που βρίσκεται στον κατάλογο εγγράφων.

## Διαμόρφωση επιλογών δημιουργίας αντιγράφων ασφαλείας

Το επόμενο βήμα είναι να διαμορφώσετε τις επιλογές αντιγράφων ασφαλείας για τη μετατροπή σε ODT. Χρησιμοποιήστε την κλάση OdtSaveOptions και ορίστε την ιδιότητα MeasureUnit στην επιθυμητή τιμή. Για παράδειγμα, εάν θέλετε να χρησιμοποιήσετε ίντσες ως μονάδα μέτρησης, ορίστε το MeasureUnit σε OdtSaveMeasureUnit.Inches. Δείτε πώς να το κάνετε:

```csharp
OdtSaveOptions saveOptions = new OdtSaveOptions { MeasureUnit = OdtSaveMeasureUnit.Inches };
```

Δημιουργούμε ένα νέο αντικείμενο OdtSaveOptions και ορίζουμε την ιδιότητα MeasureUnit στην επιθυμητή τιμή, στην περίπτωσή μας, OdtSaveMeasureUnit.Inches για χρήση ίντσες ως μονάδα μέτρησης.

## Μετατροπή εγγράφου σε ODT

Τώρα που έχουμε διαμορφώσει τις επιλογές αποθήκευσης, μπορούμε να προχωρήσουμε στη μετατροπή του εγγράφου σε ODT. Χρησιμοποιήστε τη μέθοδο Save της κλάσης Document για να αποθηκεύσετε το έγγραφο που έχει μετατραπεί σε μορφή ODT, καθορίζοντας τις επιλογές αποθήκευσης. Εδώ είναι ένα παράδειγμα:

```csharp
doc.Save(dataDir + "WorkingWithOdtSaveOptions.MeasureUnit.odt", saveOptions);
```

Σε αυτό το παράδειγμα, αποθηκεύουμε το έγγραφο που έχει μετατραπεί ως "WorkingWithOdtSaveOptions.MeasureUnit.odt" χρησιμοποιώντας τις καθορισμένες επιλογές αποθήκευσης.

### Παράδειγμα πηγαίου κώδικα για το OdtSaveOptions με λειτουργικότητα "Μονάδα μέτρησης" χρησιμοποιώντας Aspose.Words για .NET



```csharp
// Διαδρομή στον κατάλογο των εγγράφων σας
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// Φορτώστε το έγγραφο του Word
Document doc = new Document(dataDir + "Document.docx");

// Διαμόρφωση εφεδρικών επιλογών με τη λειτουργία «Μονάδα μέτρησης».
OdtSaveOptions saveOptions = new OdtSaveOptions { MeasureUnit = OdtSaveMeasureUnit.Inches };

// Μετατρέψτε το έγγραφο σε ODT
doc.Save(dataDir + "WorkingWithOdtSaveOptions.MeasureUnit.odt", saveOptions);
```
