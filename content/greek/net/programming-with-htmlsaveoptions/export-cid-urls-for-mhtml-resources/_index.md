## Βήμα 2: Φόρτωση του εγγράφου του Word

### Βήμα 2.1: Καθορίστε τον Κατάλογο Εγγράφων
Καθορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας. Εδώ βρίσκεται το έγγραφό σας στο Word.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τον κατάλογό σας.

### Βήμα 2.2: Φορτώστε το έγγραφο
Φορτώστε το έγγραφο του Word στο έργο.

```csharp
Document doc = new Document(dataDir + "Content-ID.docx");
```

## Βήμα 3: Διαμόρφωση επιλογών αποθήκευσης HTML

 Δημιουργήστε ένα παράδειγμα του`HtmlSaveOptions` για να προσαρμόσετε τον τρόπο με τον οποίο το έγγραφό σας θα αποθηκεύεται ως MHTML.

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.Mhtml)
{
    PrettyFormat = true,
    ExportCidUrlsForMhtmlResources = true
};
```

- `SaveFormat.Mhtml` καθορίζει ότι η μορφή εξόδου είναι MHTML.
- `PrettyFormat = true` διασφαλίζει ότι η έξοδος είναι σωστά διαμορφωμένη.
- `ExportCidUrlsForMhtmlResources = true` επιτρέπει την εξαγωγή Cid URL για πόρους MHTML.

### Βήμα 4: Αποθήκευση του Εγγράφου ως MHTML

Βήμα 4.1: Αποθηκεύστε το έγγραφο
Αποθηκεύστε το έγγραφό σας ως αρχείο MHTML χρησιμοποιώντας τις διαμορφωμένες επιλογές.

```csharp
doc.Save(dataDir + "WorkingWithHtmlSaveOptions.ExportCidUrlsForMhtmlResources.mhtml", saveOptions);
```
