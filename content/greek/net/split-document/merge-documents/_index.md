## Εισαγωγή χώρων ονομάτων

Πρώτα πράγματα πρώτα, πρέπει να εισαγάγουμε τους απαραίτητους χώρους ονομάτων για να δουλέψουμε με το Aspose.Words. Αυτό μας επιτρέπει να έχουμε πρόσβαση σε όλες τις κλάσεις και τις μεθόδους που θα χρειαστούμε.

```csharp
using System;
```

Αυτοί οι χώροι ονομάτων είναι απαραίτητοι για τη δημιουργία, τον χειρισμό και την αποθήκευση εγγράφων σε διαφορετικές μορφές.

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων

Πριν ξεκινήσουμε τη συγχώνευση εγγράφων, πρέπει να καθορίσουμε τον κατάλογο όπου αποθηκεύονται τα έγγραφά μας. Αυτό βοηθά το Aspose.Words να εντοπίσει τα αρχεία που θέλουμε να συγχωνεύσουμε.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Εδώ, ορίζουμε τη διαδρομή προς τον κατάλογο όπου βρίσκονται τα έγγραφά σας στο Word. Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή.

## Βήμα 2: Απλή συγχώνευση

 Ας ξεκινήσουμε με μια απλή συγχώνευση. Θα συγχωνεύσουμε δύο έγγραφα σε ένα χρησιμοποιώντας το`Merger.Merge` μέθοδος.

```csharp
Merger.Merge(dataDir + "MergedDocument.docx", new[] { dataDir + "Document1.docx", dataDir + "Document2.docx" });
```

 Σε αυτό το βήμα, συγχωνευόμαστε`Document1.docx` και`Document2.docx` σε ένα νέο αρχείο που ονομάζεται`MergedDocument.docx`.

## Βήμα 3: Συγχώνευση με Επιλογές Αποθήκευσης

Μερικές φορές, μπορεί να θέλετε να ορίσετε συγκεκριμένες επιλογές για το συγχωνευμένο έγγραφο, όπως προστασία με κωδικό πρόσβασης. Δείτε πώς μπορείτε να το κάνετε:

```csharp
OoxmlSaveOptions saveOptions = new OoxmlSaveOptions { Password = "Aspose.Words" };
Merger.Merge(dataDir + "MergedWithPassword.docx", new[] { dataDir + "Document1.docx", dataDir + "Document2.docx" }, saveOptions, MergeFormatMode.KeepSourceFormatting);
```

Αυτό το απόσπασμα κώδικα συγχωνεύει τα έγγραφα με προστασία κωδικού πρόσβασης, διασφαλίζοντας ότι το τελικό έγγραφο είναι ασφαλές.

## Βήμα 4: Συγχώνευση και αποθήκευση ως PDF

Εάν χρειάζεται να συγχωνεύσετε έγγραφα και να αποθηκεύσετε το αποτέλεσμα ως PDF, το Aspose.Words διευκολύνει:

```csharp
Merger.Merge(dataDir + "MergedDocument.pdf", new[] { dataDir + "Document1.docx", dataDir + "Document2.docx" }, SaveFormat.Pdf, MergeFormatMode.KeepSourceLayout);
```

 Εδώ, συγχωνευόμαστε`Document1.docx` και`Document2.docx` και αποθηκεύστε το αποτέλεσμα ως αρχείο PDF.

## Βήμα 5: Δημιουργία παρουσίας εγγράφου από συγχωνευμένα έγγραφα

 Μερικές φορές, μπορεί να θέλετε να εργαστείτε περαιτέρω με το συγχωνευμένο έγγραφο πριν από την αποθήκευση. Μπορείτε να δημιουργήσετε ένα`Document` παράδειγμα από συγχωνευμένα έγγραφα:

```csharp
Document doc = Merger.Merge(new[] { dataDir + "Document1.docx", dataDir + "Document2.docx" }, MergeFormatMode.MergeFormatting);
doc.Save(dataDir + "MergedDocumentInstance.docx");
```

 Σε αυτό το βήμα, δημιουργούμε ένα`Document` παράδειγμα από τα συγχωνευμένα έγγραφα, επιτρέποντας περαιτέρω χειρισμό πριν την αποθήκευση.
