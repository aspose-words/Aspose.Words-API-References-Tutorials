Σε αυτό το σεμινάριο, θα σας πούμε πώς να δημιουργήσετε μια λίστα με κουκκίδες με το Aspose.Words για .NET. Μια λίστα με κουκκίδες χρησιμοποιείται για τη λίστα στοιχείων χωρίς τη χρήση αρίθμησης.

## Βήμα 1: Χρήση ενός προγράμματος δημιουργίας εγγράφων

Αρχικά, θα χρησιμοποιήσουμε ένα πρόγραμμα δημιουργίας εγγράφων για να προσθέσουμε περιεχόμενο στο έγγραφό μας.

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## Βήμα 2: Εφαρμογή μιας προεπιλεγμένης λίστας με κουκκίδες

 Μπορούμε να εφαρμόσουμε μια προεπιλεγμένη λίστα με κουκκίδες χρησιμοποιώντας το πρόγραμμα δημιουργίας εγγράφων`ApplyBulletDefault` μέθοδος.

```csharp
builder.ListFormat.ApplyBulletDefault();
```

## Βήμα 3: Προσαρμογή της μορφής Bullet

Μπορούμε να προσαρμόσουμε τη μορφή κουκκίδας αποκτώντας πρόσβαση στις ιδιότητες του`ListFormat.List.ListLevels[0]`. Σε αυτό το παράδειγμα, χρησιμοποιούμε την παύλα "-" ως κουκκίδα.

```csharp
builder.ListFormat.List.ListLevels[0].NumberFormat = "-";
```

## Βήμα 4: Προσθήκη στοιχείων στη λίστα

 Τώρα μπορούμε να προσθέσουμε στοιχεία στη λίστα με κουκκίδες χρησιμοποιώντας το πρόγραμμα δημιουργίας εγγράφων`Writeln` μέθοδος.

```csharp
builder. Writen("Element 1");
builder. Writen("Element 2");
```

## Βήμα 5: Αφαίρεση εσοχής από τη λίστα

 Εάν θέλουμε να δημιουργήσουμε μια υπολίστα, μπορούμε να αυξήσουμε την εσοχή χρησιμοποιώντας το`ListFormat.ListIndent()` μέθοδος. Σε αυτό το παράδειγμα, προσθέτουμε μια υπολίστα στα στοιχεία 2α και 2β.

```csharp
builder.ListFormat.ListIndent();
builder. Writeln("Element 2a");
builder.Writeln("Element 2b");
```
### Παράδειγμα πηγαίου κώδικα για Λίστα με κουκκίδες χρησιμοποιώντας το Aspose.Words για .NET


```csharp
// Χρησιμοποιήστε ένα πρόγραμμα δημιουργίας εγγράφων για να προσθέσετε περιεχόμενο στο έγγραφο.
DocumentBuilder builder = new DocumentBuilder();

builder.ListFormat.ApplyBulletDefault();
builder.ListFormat.List.ListLevels[0].NumberFormat = "-";

builder.Writeln("Item 1");
builder.Writeln("Item 2");

builder.ListFormat.ListIndent();

builder.Writeln("Item 2a");
builder.Writeln("Item 2b");
```

### Συχνές ερωτήσεις

#### Ε: Πώς να δημιουργήσετε μια λίστα με κουκκίδες στο Markdown;

Α: Για να δημιουργήσετε μια λίστα με κουκκίδες στο Markdown, ξεκινήστε κάθε στοιχείο λίστας με ένα σύμβολο κουκκίδων (`-`, `*` , ή`+`), ακολουθούμενο από ένα κενό.

#### Ε: Μπορείτε να τοποθετήσετε λίστες με κουκκίδες στο Markdown;

Α: Ναι, είναι δυνατή η ένθεση λιστών με κουκκίδες στο Markdown προσθέτοντας τέσσερα κενά μετατόπισης μπροστά από κάθε ένθετο στοιχείο λίστας.

#### Ε: Πώς να προσαρμόσετε τα σύμβολα κουκκίδων;

Α: Στο τυπικό Markdown, τα σύμβολα κουκκίδων είναι προκαθορισμένα. Ωστόσο, ορισμένοι επεξεργαστές Markdown σάς επιτρέπουν να τους προσαρμόσετε χρησιμοποιώντας συγκεκριμένες επεκτάσεις.

#### Ε: Οι λίστες με κουκκίδες στο Markdown υποστηρίζουν την εσοχή;

Α: Ναι, οι λίστες με κουκκίδες στο Markdown υποστηρίζουν την εσοχή. Μπορείτε να προσθέσετε μια αριστερή μετατόπιση χρησιμοποιώντας κενά ή καρτέλες.

#### Ε: Μπορούν να προστεθούν σύνδεσμοι ή ενσωματωμένο κείμενο σε στοιχεία λίστας;

Α: Ναι, μπορείτε να προσθέσετε συνδέσμους ή ενσωματωμένο κείμενο σε στοιχεία λίστας χρησιμοποιώντας την κατάλληλη σύνταξη Markdown.
