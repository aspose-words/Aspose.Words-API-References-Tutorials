Σε αυτό το παράδειγμα, θα σας καθοδηγήσουμε στον τρόπο εφαρμογής του στυλ κειμένου διαγραφής χρησιμοποιώντας το Aspose.Words για .NET. Το κείμενο διαγραφής χρησιμοποιείται για να υποδείξει ότι το κείμενο έχει διαγραφεί ή δεν είναι πλέον έγκυρο.

## Βήμα 1: Χρήση ενός προγράμματος δημιουργίας εγγράφων

Αρχικά, θα χρησιμοποιήσουμε ένα πρόγραμμα δημιουργίας εγγράφων για να προσθέσουμε περιεχόμενο στο έγγραφό μας.

```csharp
DocumentBuilder builder = new DocumentBuilder();
```

## Βήμα 2: Εφαρμόστε στυλ κειμένου διαγραφής

Θα ενεργοποιήσουμε το στυλ κειμένου διαγραφής ορίζοντας το`StrikeThrough` ιδιοκτησία του`Font` αντιτίθεμαι`true`.

```csharp
builder.Font.StrikeThrough = true;
```

## Βήμα 3: Προσθέστε κείμενο διαγραφής

 Μπορούμε τώρα να προσθέσουμε κείμενο διαγραφής χρησιμοποιώντας τη δημιουργία εγγράφων`Writeln` μέθοδος.

```csharp
builder.Writeln("This text will be StrikeThrough");
```


### Παράδειγμα πηγαίου κώδικα για κείμενο διαγραφής με το Aspose.Words για .NET

```csharp
// Χρησιμοποιήστε ένα πρόγραμμα δημιουργίας εγγράφων για να προσθέσετε περιεχόμενο στο έγγραφο.
DocumentBuilder builder = new DocumentBuilder();

// Κάντε το κείμενο Διαγραφή.
builder.Font.StrikeThrough = true;
builder.Writeln("This text will be StrikeThrough");
```

συγχαρητήρια ! Τώρα μάθατε πώς να εφαρμόζετε το στυλ κειμένου διαγραφής με το Aspose.Words για .NET.

### Συχνές ερωτήσεις

#### Ε: Πώς μπορώ να προσθέσω το κείμενο διαγραφής στο Aspose.Words;

 Α: Για να προσθέσετε το κείμενο διαγραφής στο Aspose.Words, μπορείτε να χρησιμοποιήσετε το`Font.StrikeThrough` ιδιοκτησία του`Run`αντικείμενο. Μπορείτε να ορίσετε αυτήν την ιδιότητα σε`true` για να προσθέσετε κείμενο διαγραφής σε συγκεκριμένο κείμενο. Για παράδειγμα, μπορείτε να χρησιμοποιήσετε`run.Font.StrikeThrough=true` για να προσθέσετε το κείμενο διαγραφής στο`Run` αντικείμενο.

#### Ε: Είναι δυνατόν να προσθέσετε το κείμενο διαγραφής σε πολλά κομμάτια κειμένου στην ίδια παράγραφο;

 Α: Ναι, μπορείτε να προσθέσετε κείμενο διαγραφής σε πολλά μέρη κειμένου σε μία παράγραφο χρησιμοποιώντας πολλά.`Run` αντικείμενα. Μπορείτε να δημιουργήσετε πολλά`Run` αντικείμενα και ορίστε το`Font.StrikeThrough`ιδιοκτησία σε`true` για κάθε αντικείμενο για να προσθέσετε το κείμενο διαγραφής στα επιθυμητά μέρη κειμένου. Στη συνέχεια, μπορείτε να τα προσθέσετε στην παράγραφο χρησιμοποιώντας το`Paragraph.AppendChild(run)` μέθοδος.

#### Ε: Μπορώ να προσθέσω κείμενο διαγραφής σε κείμενο που βρίσκεται σε πίνακα ή κελί στο Aspose.Words;

 Α: Ναι, μπορείτε να προσθέσετε κείμενο διαγραφής σε κείμενο που βρίσκεται σε έναν πίνακα ή κελί στο Aspose.Words. Μπορείτε να μεταβείτε στο κελί ή στην παράγραφο που θέλετε χρησιμοποιώντας τις κατάλληλες μεθόδους και, στη συνέχεια, να εφαρμόσετε τη μορφοποίηση κειμένου διαγραφής χρησιμοποιώντας το`Font.StrikeThrough` ιδιοκτησία του`Run` ή`Paragraph` αντικείμενο.
