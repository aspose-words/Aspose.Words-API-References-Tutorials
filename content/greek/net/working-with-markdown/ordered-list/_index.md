Σε αυτό το παράδειγμα, θα εξηγήσουμε πώς να χρησιμοποιήσετε τη λειτουργία ταξινομημένης λίστας με το Aspose.Words για .NET. Η Ordered List σάς επιτρέπει να οργανώνετε τα στοιχεία διαδοχικά με αριθμούς.

## Βήμα 1: Χρήση ενός προγράμματος δημιουργίας εγγράφων

Αρχικά, θα χρησιμοποιήσουμε ένα πρόγραμμα δημιουργίας εγγράφων για να δημιουργήσουμε ένα νέο έγγραφο.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## Βήμα 2: Εφαρμογή της μορφής ταξινομημένης λίστας

 Θα εφαρμόσουμε τη μορφή της ταξινομημένης λίστας χρησιμοποιώντας το εργαλείο δημιουργίας εγγράφων`ApplyBulletDefault`μέθοδος. Μπορούμε επίσης να προσαρμόσουμε τη μορφή αρίθμησης πηγαίνοντας στα επίπεδα της λίστας και ορίζοντας τη μορφή που θέλουμε.

```csharp
builder.ListFormat.ApplyBulletDefault();
builder.ListFormat.List.ListLevels[0].NumberFormat = $"{(char) 0}.";
builder.ListFormat.List.ListLevels[1].NumberFormat = $"{(char) 1}.";
```

## Βήμα 3: Προσθήκη στοιχείων στη λίστα

 Μπορούμε να προσθέσουμε στοιχεία στη λίστα χρησιμοποιώντας τη δημιουργία εγγράφων`Writeln` μέθοδος.

```csharp
builder. Writen("Element 1");
builder. Writen("Element 2");
```

## Βήμα 4: Κάντε εσοχή στη λίστα

 Μπορούμε να κάνουμε εσοχή στη λίστα χρησιμοποιώντας τη δημιουργία εγγράφων`ListIndent` μέθοδος.

```csharp
builder.ListFormat.ListIndent();
builder.Writeln("Item 2a");
builder.Writeln("Item 2b");
```

## Βήμα 5: Αποθήκευση του εγγράφου

Τέλος, μπορούμε να αποθηκεύσουμε το έγγραφο στην επιθυμητή μορφή.

### Παράδειγμα πηγαίου κώδικα για ταξινομημένη λίστα με Aspose.Words για .NET

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

builder.ListFormat.ApplyBulletDefault();
builder.ListFormat.List.ListLevels[0].NumberFormat = $"{(char) 0}.";
builder.ListFormat.List.ListLevels[1].NumberFormat = $"{(char) 1}.";

builder.Writeln("Item 1");
builder.Writeln("Item 2");

builder.ListFormat.ListIndent();

builder.Writeln("Item 2a");
builder.Writeln("Item 2b");
```

Συγχαρητήρια ! Τώρα μάθατε πώς να χρησιμοποιείτε τη δυνατότητα λίστας ταξινομημένων με το Aspose.Words για .NET.


### Συχνές ερωτήσεις

#### Ε: Πώς να δημιουργήσετε μια ταξινομημένη λίστα στο Markdown;

Α: Για να δημιουργήσετε μια ταξινομημένη λίστα στο Markdown, ξεκινήστε κάθε στοιχείο λίστας με έναν αριθμό ακολουθούμενο από μια τελεία (`1.`, `2.`, `3.`), ακολουθούμενο από ένα κενό.

#### Ε: Μπορούμε να τοποθετήσουμε διατεταγμένες λίστες στο Markdown;

Α: Ναι, είναι δυνατή η ένθεση ταξινομημένων λιστών στο Markdown προσθέτοντας τέσσερα κενά μετατόπισης μπροστά από κάθε ένθετο στοιχείο λίστας.

#### Ε: Πώς να προσαρμόσετε την αρίθμηση των ταξινομημένων λιστών;

Α: Στο τυπικό Markdown, η διατεταγμένη αρίθμηση λιστών δημιουργείται αυτόματα. Ωστόσο, ορισμένοι επεξεργαστές Markdown σάς επιτρέπουν να το προσαρμόσετε χρησιμοποιώντας συγκεκριμένες επεκτάσεις.

#### Ε: Οι ταξινομημένες λίστες στο Markdown υποστηρίζουν την εσοχή;

Α: Ναι, οι ταξινομημένες λίστες στην εσοχή υποστήριξης Markdown. Μπορείτε να προσθέσετε μια αριστερή μετατόπιση χρησιμοποιώντας κενά ή καρτέλες.

#### Ε: Μπορούν να προστεθούν σύνδεσμοι ή ενσωματωμένο κείμενο σε στοιχεία λίστας;

Α: Ναι, μπορείτε να προσθέσετε συνδέσμους ή ενσωματωμένο κείμενο σε στοιχεία λίστας χρησιμοποιώντας την κατάλληλη σύνταξη Markdown.
