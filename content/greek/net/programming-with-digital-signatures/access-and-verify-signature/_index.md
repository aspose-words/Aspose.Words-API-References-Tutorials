## Βήμα 3: Πρόσβαση στις Ψηφιακές Υπογραφές

Με το έγγραφό σας φορτωμένο, ήρθε η ώρα να αποκτήσετε πρόσβαση στις ψηφιακές υπογραφές.

```csharp
foreach (DigitalSignature signature in doc.DigitalSignatures)
{
    Console.WriteLine("* Signature Found *");
    Console.WriteLine("Is valid: " + signature.IsValid);
    Console.WriteLine("Reason for signing: " + signature.Comments); 
    Console.WriteLine("Time of signing: " + signature.SignTime);
    Console.WriteLine("Subject name: " + signature.CertificateHolder.Certificate.SubjectName.Name);
    Console.WriteLine("Issuer name: " + signature.CertificateHolder.Certificate.IssuerName.Name);
    Console.WriteLine();
}
```

Αυτός ο κωδικός περνά μέσα από κάθε ψηφιακή υπογραφή στο έγγραφο και εκτυπώνει διάφορες λεπτομέρειες σχετικά με την υπογραφή. Ας αναλύσουμε τι κάνει κάθε μέρος:

1. Υπογραφή Βρέθηκε: Υποδεικνύει ότι έχει βρεθεί μια υπογραφή.
2. Είναι έγκυρο: Ελέγχει εάν η υπογραφή είναι έγκυρη.
3. Αιτία υπογραφής: Εμφανίζει τον λόγο υπογραφής, εάν υπάρχει.
4. Ώρα υπογραφής: Εμφανίζει τη χρονική σήμανση της υπογραφής του εγγράφου.
5. Όνομα θέματος: Ανακτά το όνομα θέματος από το πιστοποιητικό.
6. Όνομα εκδότη: Ανακτά το όνομα του εκδότη από το πιστοποιητικό.

## Βήμα 5: Χειρισμός πιθανών σφαλμάτων

Είναι πάντα καλή ιδέα να χειρίζεστε τυχόν πιθανά σφάλματα που μπορεί να προκύψουν. Ας προσθέσουμε κάποιο βασικό χειρισμό σφαλμάτων στον κώδικά μας.

```csharp
try
{
    // Η διαδρομή προς τον κατάλογο εγγράφων.
    string dataDir = "YOUR DOCUMENT DIRECTORY";
    Document doc = new Document(dataDir + "Digitally signed.docx");

    foreach (DigitalSignature signature in doc.DigitalSignatures)
    {
        Console.WriteLine("* Signature Found *");
        Console.WriteLine("Is valid: " + signature.IsValid);
        Console.WriteLine("Reason for signing: " + signature.Comments); 
        Console.WriteLine("Time of signing: " + signature.SignTime);
        Console.WriteLine("Subject name: " + signature.CertificateHolder.Certificate.SubjectName.Name);
        Console.WriteLine("Issuer name: " + signature.CertificateHolder.Certificate.IssuerName.Name);
        Console.WriteLine();
    }
}
catch (Exception ex)
{
    Console.WriteLine("An error occurred: " + ex.Message);
}
```

Αυτό θα εντοπίσει τυχόν εξαιρέσεις που ενδέχεται να προκύψουν και θα εκτυπώσει ένα μήνυμα σφάλματος.
