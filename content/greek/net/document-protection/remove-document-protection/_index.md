Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στα βήματα για να χρησιμοποιήσετε τη δυνατότητα μη προστασίας εγγράφων του Aspose.Words για .NET. Αυτή η δυνατότητα σάς επιτρέπει να αφαιρέσετε την προστασία σε ένα έγγραφο του Word για να το κάνετε προσβάσιμο για περαιτέρω επεξεργασία. Ακολουθήστε τα παρακάτω βήματα:

## Βήμα 1: Δημιουργία εγγράφου και προσθήκη περιεχομένου

Ξεκινήστε δημιουργώντας μια παρουσία της κλάσης Document και ένα αντικείμενο DocumentBuilder:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## Βήμα 2: Προσθέστε περιεχόμενο στο έγγραφο

Χρησιμοποιήστε το αντικείμενο DocumentBuilder για να προσθέσετε περιεχόμενο στο έγγραφο:

```csharp
builder.Writeln("Text added to a document.");
```

## Βήμα 3: Κατάργηση προστασίας εγγράφου

Για να καταργήσετε την προστασία του εγγράφου, μπορείτε να χρησιμοποιήσετε τη μέθοδο Unprotect() του αντικειμένου Document. Μπορείτε να επιλέξετε να αφαιρέσετε την προστασία χωρίς κωδικό πρόσβασης ή με σωστό κωδικό πρόσβασης. Κατάργηση προστασίας χωρίς κωδικό πρόσβασης:

```csharp
doc.Unprotect();
doc.Protect(ProtectionType.ReadOnly, "newPassword");
doc.Unprotect("newPassword");
```

Βεβαιωθείτε ότι έχετε αντικαταστήσει το "newPassword" με τον σωστό κωδικό πρόσβασης εγγράφου.

## Βήμα 4: Αποθηκεύστε το έγγραφο χωρίς προστασία

Τέλος, αποθηκεύστε το έγγραφο χωρίς προστασία χρησιμοποιώντας τη μέθοδο Save() του αντικειμένου Document:

```csharp
doc.Save(dataDir + "DocumentProtection.RemoveDocumentProtection.docx");
```

Βεβαιωθείτε ότι έχετε καθορίσει τη σωστή διαδρομή και όνομα αρχείου για να αποθηκεύσετε το έγγραφο χωρίς προστασία.

### Παράδειγμα πηγαίου κώδικα για Κατάργηση προστασίας εγγράφου χρησιμοποιώντας Aspose.Words για .NET

Ακολουθεί ο πλήρης πηγαίος κώδικας για την κατάργηση προστασίας του εγγράφου χρησιμοποιώντας το Aspose.Words για .NET:

```csharp

// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

builder.Writeln("Text added to a document.");

// Η προστασία των εγγράφων μπορεί να αφαιρεθεί είτε χωρίς κωδικό πρόσβασης είτε με τον σωστό κωδικό πρόσβασης.
doc.Unprotect();
doc.Protect(ProtectionType.ReadOnly, "newPassword");
doc.Unprotect("newPassword");

doc.Save(dataDir + "DocumentProtection.RemoveDocumentProtection.docx");

```

Ακολουθώντας αυτά τα βήματα, μπορείτε εύκολα να καταργήσετε την προστασία από το έγγραφο του Word με το Aspose.Words για .NET.
