Οι αλλαγές σελίδας μπορεί μερικές φορές να επηρεάσουν τη μορφοποίηση και τη διάταξη ενός εγγράφου και μπορεί να χρειαστεί να τις αφαιρέσετε μέσω προγραμματισμού. Για να ξεκινήσετε, δημιουργήστε ένα νέο έργο C# στο περιβάλλον ανάπτυξης που προτιμάτε. Βεβαιωθείτε ότι η βιβλιοθήκη Aspose.Words for .NET αναφέρεται σωστά στο έργο σας.

## Βήμα 2: Φόρτωση του εγγράφου

Για να αφαιρέσουμε αλλαγές σελίδας από ένα έγγραφο, πρέπει πρώτα να φορτώσουμε το έγγραφο στη μνήμη. Ο παρακάτω κώδικας δείχνει πώς να φορτώσετε ένα έγγραφο από έναν συγκεκριμένο κατάλογο:

```csharp
// Διαδρομή στον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Φορτώστε το έγγραφο
Document doc = new Document(dataDir + "your-document.docx");
```

 Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς το έγγραφό σας.

## Βήμα 3: Κατάργηση αλλαγών σελίδας

Μόλις φορτωθεί το έγγραφο, μπορούμε να αρχίσουμε να αφαιρούμε τις αλλαγές σελίδας. Το παρακάτω απόσπασμα κώδικα δείχνει πώς μπορείτε να επαναλάβετε όλες τις παραγράφους του εγγράφου, να ελέγξετε για αλλαγές σελίδας και να τις αφαιρέσετε:

```csharp
NodeCollection paragraphs = doc.GetChildNodes(NodeType.Paragraph, true);

foreach (Paragraph para in paragraphs)
{
     // Εάν η παράγραφος έχει προηγουμένως αλλαγή σελίδας, τότε διαγράψτε την
     if (para.ParagraphFormat.PageBreakBefore)
         para.ParagraphFormat.PageBreakBefore = false;

     // Ελέγξτε όλες τις εκτελέσεις στην παράγραφο για αλλαγές σελίδας και καταργήστε τις
     foreach(Run run in para.Runs)
     {
         if (run.Text.Contains(ControlChar.PageBreak))
             run.Text = run.Text.Replace(ControlChar.PageBreak, string.Empty);
     }
}
```

Το παραπάνω απόσπασμα κώδικα επαναλαμβάνεται σε όλες τις παραγράφους του εγγράφου και ελέγχει εάν κάθε παράγραφος έχει μια αλλαγή σελίδας πριν από αυτήν. Εάν εντοπιστεί αλλαγή σελίδας, διαγράφεται. Στη συνέχεια, ελέγχει κάθε εκτέλεση εντός της παραγράφου για αλλαγές σελίδας και τις καταργεί.

## Βήμα 4: Αποθήκευση του τροποποιημένου εγγράφου

Αφού αφαιρέσουμε τις αλλαγές σελίδας, πρέπει να αποθηκεύσουμε το τροποποιημένο έγγραφο. Ο παρακάτω κώδικας δείχνει πώς να αποθηκεύσετε το τροποποιημένο έγγραφο σε μια συγκεκριμένη τοποθεσία:

```csharp
doc.Save(dataDir + "modified-document.docx", SaveFormat.Docx);
```

 Αντικαθιστώ`"modified-document.docx"`με το επιθυμητό όνομα για το τροποποιημένο έγγραφό σας.
