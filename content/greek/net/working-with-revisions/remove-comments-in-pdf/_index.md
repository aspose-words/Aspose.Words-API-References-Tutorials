Γεια σας, συνάδελφοι προγραμματιστές! Βρεθήκατε ποτέ μπερδεμένοι σε ένα χάος σχολίων ενώ ασχολείστε με αρχεία PDF; Δεν είσαι μόνος. Είτε προέρχονται από αξιολογήσεις από ομοτίμους είτε από συνεργατικά έργα, τα σχόλια μπορεί μερικές φορές να γεμίσουν τα έγγραφά σας. Ευτυχώς για εμάς, το Aspose.Words για .NET παρέχει έναν απρόσκοπτο τρόπο για να αφαιρέσετε αυτούς τους ενοχλητικούς σχολιασμούς. Σήμερα, θα ακολουθήσουμε τη διαδικασία βήμα προς βήμα. Λάβετε, λοιπόν, και ας βουτήξουμε στον κόσμο του Aspose.Words!

## Εισαγωγή χώρων ονομάτων

Πρώτα πράγματα πρώτα, πρέπει να εισαγάγουμε τους απαραίτητους χώρους ονομάτων. Αυτό μας επιτρέπει να χρησιμοποιούμε τις κλάσεις και τις μεθόδους που παρέχονται από το Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Layout;
```

Αυτοί οι χώροι ονομάτων μας δίνουν πρόσβαση στις επιλογές χειρισμού εγγράφων και διάταξης που θα χρειαστούμε.

## Βήμα 1: Φορτώστε το έγγραφο

Ας ξεκινήσουμε φορτώνοντας το έγγραφο που περιέχει τα σχόλια. Αυτό το έγγραφο θα πρέπει να αποθηκευτεί σε έναν κατάλογο στον οποίο έχετε πρόσβαση.


```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Revisions.docx");
```

 Σε αυτό το απόσπασμα, αντικαταστήστε`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τον κατάλογο εγγράφων σας. Φορτώνουμε ένα έγγραφο με το όνομα`Revisions.docx`.

## Βήμα 2: Απόκρυψη σχολίων στο PDF

Στη συνέχεια, πρέπει να αποκρύψουμε τα σχόλια ώστε να μην εμφανίζονται στην έκδοση PDF του εγγράφου μας. Το Aspose.Words το κάνει απίστευτα απλό.

```csharp
// Απόκρυψη σχολίων στο PDF.
doc.LayoutOptions.CommentDisplayMode = CommentDisplayMode.Hide;
```

Αυτή η γραμμή κώδικα λέει στο Aspose.Words να κρύβει τα σχόλια κατά την απόδοση του εγγράφου.

## Βήμα 3: Αποθηκεύστε το Έγγραφο ως PDF

Τέλος, αποθηκεύουμε το τροποποιημένο έγγραφο ως PDF. Αυτό το βήμα διασφαλίζει ότι τα σχόλιά μας θα αφαιρεθούν στο αρχείο εξόδου.


```csharp
doc.Save(dataDir + "WorkingWithRevisions.RemoveCommentsInPdf.pdf");
```

Εδώ, αποθηκεύουμε το έγγραφο στον ίδιο κατάλογο με νέο όνομα, υποδεικνύοντας ότι τα σχόλια έχουν αφαιρεθεί στην έκδοση PDF.
