Αυτό το άρθρο παρέχει έναν οδηγό βήμα προς βήμα σχετικά με τον τρόπο λειτουργίας Escape URI σε έγγραφο PDF με το Aspose.Words για .NET. Θα εξηγήσουμε λεπτομερώς κάθε μέρος του κώδικα. Στο τέλος αυτού του σεμιναρίου, θα μπορείτε να κατανοήσετε πώς να εισάγετε υπερσυνδέσμους με Uri διαφυγής σε ένα έγγραφο.

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε εγκαταστήσει και διαμορφώσει τη βιβλιοθήκη Aspose.Words για .NET στο έργο σας. Μπορείτε να βρείτε τη βιβλιοθήκη και τις οδηγίες εγκατάστασης στον ιστότοπο Aspose.

## Βήμα 1: Ορίστε τον κατάλογο εγγράφων

 Για να ξεκινήσετε, πρέπει να ορίσετε τη διαδρομή προς τον κατάλογο όπου βρίσκονται τα έγγραφά σας. Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τον κατάλογο των εγγράφων σας.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Βήμα 2: Δημιουργήστε ένα έγγραφο και ένα DocumentBuilder

 Στη συνέχεια, πρέπει να δημιουργήσουμε ένα νέο`Document` αντικείμενο και α`DocumentBuilder` αντικείμενο δημιουργίας του εγγράφου.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## Βήμα 3: Εισαγάγετε υπερσυνδέσμους με Uri διαφυγής

 Χρησιμοποιήστε το`InsertHyperlink` μέθοδος του`DocumentBuilder` αντικείμενο εισαγωγής υπερσυνδέσμων στο έγγραφο. Το Uri πρέπει να διαφύγει χρησιμοποιώντας το`Uri.EscapeUriString` λειτουργία για την αποφυγή σφαλμάτων μορφής.

```csharp
builder.InsertHyperlink("Testlink",
     Uri.EscapeUriString("https://www.google.com/search?q=%2Fthe%20test"), false);
builder. Writen();
builder.InsertHyperlink(Uri.EscapeUriString("https://www.google.com/search?q=%2Fthe%20test"),
     Uri.EscapeUriString("https://www.google.com/search?q=%2Fthe%20test"), false);
```

## Βήμα 4: Αποθηκεύστε το έγγραφο ως PDF

 Τέλος, μπορούμε να αποθηκεύσουμε το έγγραφο ως PDF χρησιμοποιώντας το`Save` μέθοδος του`Document` αντικείμενο. Καθορίστε το όνομα του αρχείου εξόδου.

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.EscapeUri.pdf");
```

Αυτό είναι όλο ! Έχετε εισαγάγει με επιτυχία υπερσυνδέσμους με Uri με διαφυγή σε ένα έγγραφο χρησιμοποιώντας το Aspose.Words για .NET.

### Δείγμα πηγαίου κώδικα για το Uri που διαφεύγει με το Aspose.Words για .NET


```csharp

	// Η διαδρομή προς τον κατάλογο εγγράφων.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	
	builder.InsertHyperlink("Testlink", 
		"https://www.google.com/search?q=%2Fthe%20test", false);
	builder.Writeln();
	builder.InsertHyperlink("https:// www.google.com/search?q=%2Fthe%20test",
		"https://www.google.com/search?q=%2Fthe%20test", false);

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.EscapeUri.pdf");    
  
```

## συμπέρασμα

Σε αυτό το σεμινάριο, καλύψαμε τον τρόπο διαφυγής URI σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.Words για .NET. Με τη διαφυγή των URI, μπορείτε να αποφύγετε σφάλματα μορφοποίησης και να διασφαλίσετε ότι οι υπερσυνδέσεις ερμηνεύονται σωστά και εμφανίζονται στο έγγραφο PDF. Ακολουθήστε τα βήματα που περιγράφονται για να εισαγάγετε υπερσυνδέσμους με URI διαφυγής στο έγγραφό σας PDF. Φροντίστε να ξεφύγετε.

### Συχνές Ερωτήσεις

#### Ε: Τι είναι το URI διαφυγής σε ένα έγγραφο PDF και γιατί είναι σημαντικό;
Α: Το Escape URI σε ένα έγγραφο PDF αναφέρεται στη μέθοδο μετατροπής ειδικών χαρακτήρων σε μια διεύθυνση URL σε ακολουθίες διαφυγής για την αποφυγή σφαλμάτων μορφής. Αυτό είναι σημαντικό επειδή οι ειδικοί χαρακτήρες σε μια διεύθυνση URL μπορούν να διαταράξουν τη δομή της διεύθυνσης URL και να οδηγήσουν σε παρερμηνεία ή λανθασμένη απόδοση. Διαφεύγοντας από τους ειδικούς χαρακτήρες, εγγυόμαστε ότι η διεύθυνση URL ερμηνεύεται σωστά και εμφανίζεται στο έγγραφο PDF.

#### Ε: Πώς μπορώ να χρησιμοποιήσω το Aspose.Words για .NET για διαφυγή URI σε ένα έγγραφο PDF;
Α: Για να διαφύγετε από URI σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.Words για .NET, ακολουθήστε τα εξής βήματα:

 Ορίστε τη διαδρομή καταλόγου όπου βρίσκονται τα έγγραφά σας αντικαθιστώντας`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή του καταλόγου των εγγράφων σας.

 Δημιουργήστε μια νέα παρουσία του`Document` τάξη και α`DocumentBuilder` αντικείμενο δημιουργίας του εγγράφου.

 Χρησιμοποιήστε το`InsertHyperlink` μέθοδος του`DocumentBuilder` αντικείμενο εισαγωγής υπερσυνδέσμων στο έγγραφο. Φροντίστε να ξεφύγετε από το URI χρησιμοποιώντας το`Uri.EscapeUriString` λειτουργία για την αποφυγή σφαλμάτων μορφής.

 Χρησιμοποιήστε το`Save` μέθοδος του`Document` αντικείμενο αποθήκευσης του εγγράφου σε μορφή PDF, καθορίζοντας το όνομα του αρχείου εξόδου.

#### Ε: Ποια είναι τα οφέλη της διαφυγής URI σε ένα έγγραφο PDF;
Α: Τα οφέλη της διαφυγής URI σε ένα έγγραφο PDF είναι:

Πρόληψη σφαλμάτων μορφής: Η διαφυγή URI βοηθά στην αποτροπή σφαλμάτων μορφής που προκαλούνται από ειδικούς χαρακτήρες σε μια διεύθυνση URL, διασφαλίζοντας ότι η διεύθυνση URL ερμηνεύεται σωστά και εμφανίζεται στο έγγραφο PDF.

Συμβατότητα με προγράμματα ανάγνωσης PDF: Τα URI διαφυγής γενικά υποστηρίζονται καλά από προγράμματα ανάγνωσης PDF, διασφαλίζοντας καλύτερη συμβατότητα και συνεπή εμπειρία χρήστη.

#### Ε: Ποιοι ειδικοί χαρακτήρες πρέπει να διαφύγουν σε ένα URI;
 Α: Οι ειδικοί χαρακτήρες που πρέπει να διαγραφούν σε ένα URI είναι: space, <, >, ", #, %, {, },|, \, ^, ~, [, ], `, ;, /, ?, :, @, =, &, $, +, ,, [, ], and !.
