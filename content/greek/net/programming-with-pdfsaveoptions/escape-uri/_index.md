## Εισαγωγή

Γεια σου! Αναρωτηθήκατε ποτέ πώς να χειρίζεστε URI όταν αποθηκεύετε ένα έγγραφο του Word ως PDF χρησιμοποιώντας το Aspose.Words για .NET; Είστε στο σωστό μέρος! Σήμερα, θα εμβαθύνουμε σε URI διαφυγής σε έγγραφα PDF για να διασφαλίσουμε ότι λειτουργούν σωστά κατά τη μετατροπή του εγγράφου σας. Ας ξεκινήσουμε αυτό το ταξίδι μαζί, βήμα-βήμα, διασφαλίζοντας ότι θα κατανοήσετε κάθε έννοια στην πορεία.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε:

1.  Aspose.Words για .NET: Βεβαιωθείτε ότι διαθέτετε την πιο πρόσφατη έκδοση. Μπορείτε να το κατεβάσετε[εδώ](https://releases.aspose.com/words/net/).
2. Περιβάλλον ανάπτυξης: Visual Studio ή οποιοδήποτε άλλο περιβάλλον ανάπτυξης .NET.
3. Βασικές γνώσεις C#: Αυτό το σεμινάριο προϋποθέτει ότι έχετε βασική κατανόηση του προγραμματισμού C#.

## Εισαγωγή χώρων ονομάτων

Πρώτα πράγματα πρώτα, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων. Αυτό είναι ζωτικής σημασίας για την πρόσβαση στις λειτουργίες Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

Εντάξει, ας αναλύσουμε τη διαδικασία σε εύπεπτα βήματα. Ακολουθήστε και θα είστε επαγγελματίας στο να ξεφεύγετε από URI σε χρόνο μηδέν!

## Βήμα 1: Ρυθμίστε τον Κατάλογο Εγγράφων σας

Πριν βουτήξετε στον κώδικα, ορίστε τη διαδρομή προς τον κατάλογο των εγγράφων σας. Εδώ θα αποθηκευτούν τα αρχεία σας.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή όπου θέλετε να αποθηκεύσετε το έγγραφο PDF σας.

## Βήμα 2: Δημιουργήστε ένα νέο έγγραφο

Στη συνέχεια, θα δημιουργήσουμε ένα νέο έγγραφο χρησιμοποιώντας το Aspose.Words. Αυτό το έγγραφο θα χρησιμεύσει ως καμβάς μας για την εισαγωγή υπερσυνδέσμων.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 Εδώ,`Document` είναι το νέο σας έγγραφο Word και`DocumentBuilder` είναι μια βοηθητική κλάση που απλοποιεί τη διαδικασία δημιουργίας εγγράφων.

## Βήμα 3: Εισαγάγετε μια υπερσύνδεση

Τώρα, ας εισαγάγουμε μια υπερ-σύνδεση στο έγγραφο. Αυτή η υπερσύνδεση θα είναι το επίκεντρο της διαδικασίας διαφυγής URI.

```csharp
builder.InsertHyperlink("Testlink", "https://www.google.com/search?q= aspose", false);
```

 Σε αυτή τη γραμμή,`"Testlink"` είναι το κείμενο εμφάνισης για την υπερ-σύνδεση και`"https://www.google.com/search?q= aspose"` είναι η διεύθυνση URL. ο`false` Η παράμετρος υποδεικνύει ότι ο σύνδεσμος δεν πρέπει να μορφοποιηθεί ως υπερσύνδεσμος (δηλαδή, δεν θα είναι υπογραμμισμένος και μπλε).

## Βήμα 4: Αποθηκεύστε το Έγγραφο ως PDF

Τέλος, αποθηκεύστε το έγγραφο ως PDF. Εδώ το Aspose.Words φροντίζει να ξεφύγει από το URI στην έξοδο PDF.

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.EscapeUri.pdf");
```

 Αυτή η γραμμή αποθηκεύει το έγγραφο στον καθορισμένο κατάλογο με το όνομα αρχείου`WorkingWithPdfSaveOptions.EscapeUri.pdf`.
