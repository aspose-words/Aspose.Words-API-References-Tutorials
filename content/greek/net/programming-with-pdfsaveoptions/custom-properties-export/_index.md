Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στα βήματα για την εξαγωγή των προσαρμοσμένων ιδιοτήτων ενός εγγράφου σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.Words για .NET. Η εξαγωγή προσαρμοσμένων ιδιοτήτων σάς επιτρέπει να συμπεριλάβετε πρόσθετες πληροφορίες στο έγγραφο PDF που δημιουργείται. Ακολουθήστε τα παρακάτω βήματα:

## Βήμα 1: Δημιουργία εγγράφου και προσθήκη προσαρμοσμένων ιδιοτήτων

Ξεκινήστε δημιουργώντας μια παρουσία της κλάσης Document:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
```

## Βήμα 2: Προσθέστε προσαρμοσμένες ιδιότητες
 Στη συνέχεια, προσθέστε τις επιθυμητές προσαρμοσμένες ιδιότητες. Για παράδειγμα, για να προσθέσετε μια ιδιότητα "Εταιρεία" με την τιμή "Apose", χρησιμοποιήστε το`Add` μέθοδος της συλλογής CustomDocumentProperties:

```csharp
doc.CustomDocumentProperties.Add("Company", "Aspose");
```

Μπορείτε να προσθέσετε όσες προσαρμοσμένες ιδιότητες χρειάζεται.

## Βήμα 3: Ορίστε τις επιλογές εξαγωγής PDF

Δημιουργήστε μια παρουσία της κλάσης PdfSaveOptions και καθορίστε τον τρόπο εξαγωγής προσαρμοσμένων ιδιοτήτων:

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions { CustomPropertiesExport = PdfCustomPropertiesExport.Standard };
```

Αυτή η επιλογή ελέγχει την εξαγωγή προσαρμοσμένων ιδιοτήτων κατά τη μετατροπή σε PDF.

## Βήμα 4: Μετατροπή εγγράφου σε PDF

 Χρησιμοποιήστε το`Save` μέθοδος μετατροπής του εγγράφου σε PDF καθορίζοντας τις επιλογές μετατροπής:

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.CustomPropertiesExport.pdf", saveOptions);
```

Βεβαιωθείτε ότι έχετε καθορίσει τη σωστή διαδρομή για την αποθήκευση του PDF που έχει μετατραπεί.

### Παράδειγμα πηγαίου κώδικα για Εξαγωγή προσαρμοσμένων ιδιοτήτων με χρήση Aspose.Words για .NET

Ακολουθεί ο πλήρης πηγαίος κώδικας για την εξαγωγή προσαρμοσμένων ιδιοτήτων από ένα έγγραφο χρησιμοποιώντας το Aspose.Words για .NET:


```csharp

	// Η διαδρομή προς τον κατάλογο εγγράφων.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	Document doc = new Document();
	doc.CustomDocumentProperties.Add("Company", "Aspose");

	PdfSaveOptions saveOptions = new PdfSaveOptions { CustomPropertiesExport = PdfCustomPropertiesExport.Standard };

	doc.Save(dataDir + "WorkingWithPdfSaveOptions.CustomPropertiesExport.pdf", saveOptions);

```

Ακολουθώντας αυτά τα βήματα, μπορείτε εύκολα να εξαγάγετε προσαρμοσμένες ιδιότητες ενός εγγράφου κατά τη μετατροπή σε PDF με το Aspose.Words για .NET.

Οι προσαρμοσμένες ιδιότητες είναι μεταδεδομένα ειδικά για το έγγραφό σας, όπως ετικέτες, λέξεις-κλειδιά ή διαπιστευτήρια. #### Ε: Πώς μπορώ να εξαγάγω τις προσαρμοσμένες ιδιότητες ενός εγγράφου σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.Words για .NET;
Α: Για να εξαγάγετε τις προσαρμοσμένες ιδιότητες ενός εγγράφου σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.Words για .NET, ακολουθήστε τα εξής βήματα:

 Δημιουργήστε ένα παράδειγμα του`Document` τάξη.

 Προσθέστε τις επιθυμητές προσαρμοσμένες ιδιότητες χρησιμοποιώντας το`CustomDocumentProperties` συλλογή. Για παράδειγμα, χρησιμοποιήστε το`Add` μέθοδος για να προσθέσετε μια ιδιότητα "Εταιρεία" με την τιμή "Apose".

 Δημιουργήστε ένα παράδειγμα του`PdfSaveOptions` κλάση και καθορίστε τον τρόπο εξαγωγής προσαρμοσμένων ιδιοτήτων χρησιμοποιώντας το`CustomPropertiesExport` ιδιοκτησία. ο`PdfCustomPropertiesExport.Standard` τιμή εξάγει προσαρμοσμένες ιδιότητες σύμφωνα με τις προεπιλεγμένες ρυθμίσεις.

 Χρησιμοποιήστε το`Save` μέθοδος του`Document` κλάση για να μετατρέψετε το έγγραφο σε PDF καθορίζοντας τις επιλογές μετατροπής.

Τα περισσότερα κοινά προγράμματα ανάγνωσης PDF, όπως το Adobe Acrobat Reader, παρέχουν πρόσβαση σε μεταδεδομένα και ιδιότητες ενός εγγράφου PDF. Μπορείτε συνήθως να βρείτε αυτές τις επιλογές στο μενού "Αρχείο" ή κάνοντας δεξί κλικ στο έγγραφο και επιλέγοντας "Ιδιότητες".
