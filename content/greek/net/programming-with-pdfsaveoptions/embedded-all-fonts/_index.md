## Εισαγωγή χώρων ονομάτων

Πρώτα πράγματα πρώτα, ας εισάγουμε τους απαραίτητους χώρους ονομάτων. Αυτό το βήμα είναι ζωτικής σημασίας καθώς δημιουργεί το περιβάλλον για τη χρήση των λειτουργιών Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

Τώρα, ας αναλύσουμε τη διαδικασία σε βήματα που μπορείτε να ακολουθήσετε εύκολα. Κάθε βήμα θα σας καθοδηγήσει σε ένα συγκεκριμένο μέρος της ενσωμάτωσης γραμματοσειρών στο έγγραφο PDF σας χρησιμοποιώντας το Aspose.Words για .NET.

## Βήμα 1: Ρυθμίστε τον Κατάλογο Εγγράφων σας

Πριν βουτήξετε στον κώδικα, πρέπει να ρυθμίσετε τον κατάλογο εγγράφων σας. Εδώ είναι το δείγμα εγγράφου του Word (`Rendering.docx`) και το PDF εξόδου θα παραμείνει.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τον κατάλογο εγγράφων σας. Εδώ θα γίνει όλη η μαγεία!

## Βήμα 2: Φορτώστε το έγγραφο του Word

 Στη συνέχεια, θα φορτώσετε το έγγραφο του Word στο Aspose.Words`Document` αντικείμενο. Αυτό είναι το έγγραφο με το οποίο θα εργαστείτε.

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

 Σε αυτή τη γραμμή, δημιουργούμε ένα νέο`Document` αντικείμενο και φορτώστε το`Rendering.docx` αρχείο από τον κατάλογο εγγράφων μας.

## Βήμα 3: Διαμόρφωση επιλογών αποθήκευσης PDF

 Τώρα, ήρθε η ώρα να διαμορφώσετε τις επιλογές αποθήκευσης PDF. Συγκεκριμένα, θα ορίσουμε το`EmbedFullFonts`ιδιοκτησία σε`true` για να διασφαλίσετε ότι όλες οι γραμματοσειρές που χρησιμοποιούνται στο έγγραφο είναι ενσωματωμένες στο PDF.

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions { EmbedFullFonts = true };
```

 Αυτή η γραμμή δημιουργεί μια νέα`PdfSaveOptions` αντικείμενο και θέτει το`EmbedFullFonts`ιδιοκτησία σε`true`. Αυτό διασφαλίζει ότι το PDF που δημιουργείται θα περιλαμβάνει όλες τις γραμματοσειρές που χρησιμοποιούνται στο έγγραφο.

## Βήμα 4: Αποθηκεύστε το Έγγραφο ως PDF

Τέλος, θα αποθηκεύσετε το έγγραφο του Word ως PDF με τις καθορισμένες επιλογές αποθήκευσης. Αυτό το βήμα μετατρέπει το έγγραφο και ενσωματώνει τις γραμματοσειρές.

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.EmbeddedFontsInPdf.pdf", saveOptions);
```

Σε αυτή τη γραμμή, αποθηκεύουμε το έγγραφο ως PDF στον κατάλογο εγγράφων, ενσωματώνοντας όλες τις γραμματοσειρές που χρησιμοποιούνται στο έγγραφο του Word.
