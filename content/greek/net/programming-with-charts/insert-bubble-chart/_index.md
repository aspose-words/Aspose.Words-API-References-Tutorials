Αυτό το σεμινάριο εξηγεί πώς να χρησιμοποιήσετε το Aspose.Words για .NET για να εισαγάγετε ένα γράφημα φυσαλίδων σε ένα έγγραφο. Ο παρεχόμενος πηγαίος κώδικας δείχνει πώς να δημιουργήσετε ένα γράφημα, να προσθέσετε δεδομένα σειράς και να αποθηκεύσετε το έγγραφο.

## Βήμα 1: Ρύθμιση του έργου

Βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:

- Εγκαταστάθηκε η βιβλιοθήκη Aspose.Words για .NET. Μπορείτε να το κατεβάσετε χρησιμοποιώντας τη Διαχείριση πακέτων NuGet για να το εγκαταστήσετε.
- Μια διαδρομή καταλόγου εγγράφων όπου θα αποθηκευτεί το έγγραφο εξόδου.

## Βήμα 2: Δημιουργήστε ένα νέο έγγραφο και εισαγάγετε ένα γράφημα.

 Δημιούργησε ένα νέο`Document` αντικείμενο και α`DocumentBuilder` για τη δημιουργία του εγγράφου.

```csharp
// Διαδρομή στον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 Στη συνέχεια, χρησιμοποιήστε το`InsertChart` μέθοδος του`DocumentBuilder` για να εισαγάγετε ένα γράφημα φυσαλίδων στο έγγραφο.

```csharp
Shape shape = builder.InsertChart(ChartType.Bubble, 432, 252);
Chart chart = shape.Chart;
```

## Βήμα 3: Προσθέστε δεδομένα σειράς στο γράφημα

Προσθέστε δεδομένα σειράς στο γράφημα. Σε αυτό το παράδειγμα, θα προσθέσουμε τρία σημεία δεδομένων με τις αντίστοιχες τιμές X, Y και μεγέθους φυσαλίδων.

```csharp
chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 },
    new double[] { 10, 4, 8 });
```

## Βήμα 4: Αποθηκεύστε το έγγραφο

 Τέλος, αποθηκεύστε το έγγραφο στον καθορισμένο κατάλογο χρησιμοποιώντας το`Save` μέθοδος του`Document` αντικείμενο.

```csharp
doc.Save(dataDir + "WorkingWithCharts.InsertBubbleChart.docx");
```

Αυτό ολοκληρώνει την εφαρμογή της εισαγωγής ενός γραφήματος με συννεφάκια χρησιμοποιώντας το Aspose.Words για .NET.

### Παράδειγμα πηγαίου κώδικα για το Insert Bubble Chart χρησιμοποιώντας το Aspose.Words για .NET 

```csharp
// Διαδρομή στον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
Shape shape = builder.InsertChart(ChartType.Bubble, 432, 252);
Chart chart = shape.Chart;
chart.Series.Add("Aspose Series 1", new double[] { 0.7, 1.8, 2.6 }, new double[] { 2.7, 3.2, 0.8 },
	new double[] { 10, 4, 8 });
doc.Save(dataDir + "WorkingWithCharts.InsertBubbleChart.docx");
```

Το γράφημα φυσαλίδων είναι ένας τύπος γραφήματος που εμφανίζει τρισδιάστατα δεδομένα χρησιμοποιώντας φυσαλίδες ή σφαίρες. Κάθε σημείο δεδομένων αντιπροσωπεύεται από μια φυσαλίδα, όπου οι συντεταγμένες X και Y καθορίζουν τη θέση της φυσαλίδας στο γράφημα και το μέγεθος της φυσαλίδας αντιπροσωπεύει την τρίτη διάσταση των δεδομένων. Τα γραφήματα με φυσαλίδες είναι χρήσιμα για την οπτικοποίηση σχέσεων και μοτίβων μεταξύ πολλαπλών μεταβλητών.

Ναι, μπορείτε να προσθέσετε πολλές σειρές στο γράφημα φυσαλίδων χρησιμοποιώντας το Aspose.Words για .NET. Κάθε σειρά αντιπροσωπεύει ένα σύνολο σημείων δεδομένων με τις αντίστοιχες τιμές X, Y και μεγέθους φυσαλίδων. Προσθέτοντας πολλές σειρές, μπορείτε να συγκρίνετε και να αναλύσετε διαφορετικά σύνολα δεδομένων στο ίδιο γράφημα, παρέχοντας μια ολοκληρωμένη προβολή των δεδομένων σας.

#### Ε3. Μπορώ να προσαρμόσω την εμφάνιση του γραφήματος με συννεφάκια;
Ναι, χρησιμοποιώντας το Aspose.Words για .NET, μπορείτε να προσαρμόσετε διάφορες πτυχές της εμφάνισης του γραφήματος φυσαλίδων. Η βιβλιοθήκη παρέχει ένα πλούσιο σύνολο API για τον έλεγχο των οπτικών στοιχείων του γραφήματος και τη δημιουργία μιας προσαρμοσμένης εμφάνισης που ταιριάζει στις ανάγκες σας.

 Ναι, το Aspose.Words για .NET σάς επιτρέπει να αποθηκεύσετε το έγγραφο με το εισαγόμενο γράφημα με συννεφάκια σε διάφορες μορφές, όπως DOCX, PDF, HTML και άλλα. Το εισαγόμενο γράφημα φυσαλίδων θα διατηρηθεί στο αποθηκευμένο έγγραφο.

Ναι, μετά την εισαγωγή του γραφήματος με συννεφάκια στο έγγραφο, μπορείτε να τροποποιήσετε τα δεδομένα και την εμφάνισή του χρησιμοποιώντας τα API που παρέχονται από το Aspose.Words για .NET.
