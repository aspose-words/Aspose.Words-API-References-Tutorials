Αυτό το σεμινάριο εξηγεί πώς να χρησιμοποιήσετε το Aspose.Words για .NET για να μορφοποιήσετε τον αριθμό των ετικετών δεδομένων σε ένα γράφημα. Ο παρεχόμενος πηγαίος κώδικας δείχνει πώς μπορείτε να δημιουργήσετε ένα γράφημα, να προσθέσετε δεδομένα σειρών και να προσαρμόσετε τη μορφή αριθμών των ετικετών δεδομένων.

## Βήμα 1: Ρύθμιση του έργου

Βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:

- Εγκαταστάθηκε η βιβλιοθήκη Aspose.Words για .NET. Μπορείτε να το κατεβάσετε χρησιμοποιώντας τη Διαχείριση πακέτων NuGet για να το εγκαταστήσετε.
- Μια διαδρομή καταλόγου εγγράφων όπου θα αποθηκευτεί το έγγραφο εξόδου.

## Βήμα 2: Δημιουργήστε ένα νέο έγγραφο και εισαγάγετε ένα γράφημα

 Δημιούργησε ένα νέο`Document` αντικείμενο και α`DocumentBuilder` για τη δημιουργία του εγγράφου.

```csharp
// Διαδρομή στον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

 Στη συνέχεια, εισαγάγετε ένα γράφημα στο έγγραφο χρησιμοποιώντας το`InsertChart` μέθοδος του`DocumentBuilder`. Σε αυτό το παράδειγμα, θα εισαγάγουμε ένα γραμμικό γράφημα.

```csharp
Shape shape = builder.InsertChart(ChartType.Line, 432, 252);
Chart chart = shape.Chart;
chart.Title.Text = "Data Labels With Different Number Format";
```

## Βήμα 3: Προσθέστε δεδομένα σειράς στο γράφημα

Προσθέστε δεδομένα σειράς στο γράφημα. Σε αυτό το παράδειγμα, θα προσθέσουμε τρεις κατηγορίες και τις αντίστοιχες τιμές τους.

```csharp
chart.Series.Clear();
ChartSeries series1 = chart.Series.Add("Aspose Series 1", 
    new string[] { "Category 1", "Category 2", "Category 3" }, 
    new double[] { 2.5, 1.5, 3.5 });
series1.HasDataLabels = true;
```

## Βήμα 4: Προσαρμόστε τη μορφή αριθμών των ετικετών δεδομένων

 Για να μορφοποιήσετε τον αριθμό των ετικετών δεδομένων, μεταβείτε στο`DataLabels` συλλογή που σχετίζεται με τη σειρά.

```csharp
series1.DataLabels.ShowValue = true;
series1.DataLabels[0].NumberFormat.FormatCode = "\"$\"#,##0.00";
series1.DataLabels[1].NumberFormat.FormatCode = "dd/mm/yyyy";
series1.DataLabels[2].NumberFormat.FormatCode = "0.00%";
```

Σε αυτό το παράδειγμα, ορίζουμε διαφορετικές μορφές αριθμών για κάθε ετικέτα δεδομένων. Η πρώτη ετικέτα δεδομένων μορφοποιείται ως νόμισμα, η δεύτερη ως ημερομηνία και η τρίτη ως ποσοστό.

## Βήμα 5: Αποθηκεύστε το έγγραφο

 Τέλος, αποθηκεύστε το έγγραφο στον καθορισμένο κατάλογο χρησιμοποιώντας το`Save` μέθοδος του`Document` αντικείμενο.

```csharp
doc.Save(dataDir + "WorkingWithCharts.FormatNumberOfDataLabel.docx");
```

Αυτό ολοκληρώνει την υλοποίηση της μορφοποίησης του αριθμού των ετικετών δεδομένων σε ένα γράφημα χρησιμοποιώντας το Aspose.Words για .NET.

### Παράδειγμα πηγαίου κώδικα για το Format Number of Data Label χρησιμοποιώντας το Aspose.Words για .NET 

```csharp
	// Διαδρομή στον κατάλογο εγγράφων σας
	string dataDir = "YOUR DOCUMENT DIRECTORY";

	Document doc = new Document();
	DocumentBuilder builder = new DocumentBuilder(doc);
	Shape shape = builder.InsertChart(ChartType.Line, 432, 252);
	Chart chart = shape.Chart;
	chart.Title.Text = "Data Labels With Different Number Format";
	// Διαγραφή προεπιλεγμένων σειρών που δημιουργούνται.
	chart.Series.Clear();
	ChartSeries series1 = chart.Series.Add("Aspose Series 1", 
		new string[] { "Category 1", "Category 2", "Category 3" }, 
		new double[] { 2.5, 1.5, 3.5 });
	series1.HasDataLabels = true;
	series1.DataLabels.ShowValue = true;
	series1.DataLabels[0].NumberFormat.FormatCode = "\"$\"#,##0.00";
	series1.DataLabels[1].NumberFormat.FormatCode = "dd/mm/yyyy";
	series1.DataLabels[2].NumberFormat.FormatCode = "0.00%";
	// Ή μπορείτε να ορίσετε τον κώδικα μορφής ώστε να συνδέεται με ένα κελί προέλευσης,
	//Σε αυτήν την περίπτωση, το NumberFormat θα επαναφερθεί σε γενικό και θα κληρονομηθεί από ένα κελί προέλευσης.
	series1.DataLabels[2].NumberFormat.IsLinkedToSource = true;
	doc.Save(dataDir + "WorkingWithCharts.FormatNumberOfDataLabel.docx");
```
