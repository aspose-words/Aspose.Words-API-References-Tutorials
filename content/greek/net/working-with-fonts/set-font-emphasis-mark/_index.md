Σε αυτό το σεμινάριο, θα σας δείξουμε πώς να ορίσετε το στυλ έμφασης γραμματοσειράς σε ένα έγγραφο του Word χρησιμοποιώντας το Aspose.Words για .NET. Η έμφαση στη γραμματοσειρά χρησιμοποιείται για την επισήμανση ορισμένων λέξεων ή φράσεων στο κείμενο.

## Προαπαιτούμενα
Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε τα ακόλουθα στοιχεία:
- Γνώση εργασίας της γλώσσας προγραμματισμού C#
- Η βιβλιοθήκη Aspose.Words για .NET είναι εγκατεστημένη στο έργο σας

## Βήμα 1: Ορίστε τον κατάλογο εγγράφων
Ξεκινήστε ορίζοντας τη διαδρομή καταλόγου στη θέση του εγγράφου του Word. Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` στον κωδικό με την κατάλληλη διαδρομή.

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## Βήμα 2: Δημιουργήστε και προσαρμόστε το έγγραφο
 Δημιουργήστε ένα παράδειγμα του`Document` τάξη και μια σχετική`DocumentBuilder` για τη δημιουργία του περιεχομένου του εγγράφου. Χρησιμοποιήστε το`Font.EmphasisMark` ιδιότητα για να ορίσετε το στυλ της γραμματοσειράς`EmphasisMark.UnderSolidCircle` . Στη συνέχεια χρησιμοποιήστε το`Write` και`Writeln` μεθόδους του`DocumentBuilder` για να προσθέσετε κείμενο με έμφαση στην καθορισμένη γραμματοσειρά.

```csharp
Document document = new Document();
DocumentBuilder builder = new DocumentBuilder(document);
builder.Font.EmphasisMark = EmphasisMark.UnderSolidCircle;
builder.Write("Emphasized text");
builder. Writen();
builder.Font.ClearFormatting();
builder.Write("Simple text");
```

## Βήμα 3: Αποθηκεύστε το έγγραφο
 Αποθηκεύστε το έγγραφο χρησιμοποιώντας το`Save` μέθοδος του`Document` με την κατάλληλη διαδρομή και όνομα αρχείου.

```csharp
document.Save(dataDir + "WorkingWithFonts.SetFontEmphasisMark.docx");
```

### Δείγμα πηγαίου κώδικα για τον καθορισμό γραμματοσειράς με έμφαση στο Aspose.Words για .NET 

```csharp
// Διαδρομή στον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document document = new Document();
DocumentBuilder builder = new DocumentBuilder(document);
builder.Font.EmphasisMark = EmphasisMark.UnderSolidCircle;
builder.Write("Emphasis text");
builder.Writeln();
builder.Font.ClearFormatting();
builder.Write("Simple text");
document.Save(dataDir + "WorkingWithFonts.SetFontEmphasisMark.docx");
```
